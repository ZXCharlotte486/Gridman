# ESG 与可持续发展

> 定位：ESG 披露、碳核算、鉴证程序、与财税的交叉点
> 来源：ISSB/GRI/TCFD 框架 + 中国证监会 ESG 披露要求
> 覆盖：ESG 三维度、披露框架、碳核算（Scope 1/2/3）、ESG 鉴证、碳税、ESG 尽调
> 性质：理论

---

## 一、ESG 基础

### 三个维度

| 维度 | 英文 | 核心内容 |
|------|------|---------|
| 环境 | Environmental | 碳排放、能源使用、水资源、废弃物、生物多样性、气候变化应对 |
| 社会 | Social | 员工权益、供应链劳工标准、社区影响、产品安全、数据隐私 |
| 治理 | Governance | 董事会结构、股东权利、反腐败、高管薪酬、信息披露透明度 |

### 为什么财税人员需要懂 ESG

- **审计**：ESG 信息披露鉴证是审计师的新业务，类似财务报表审计
- **税务**：碳税、环保税是 ESG 的税务延伸；绿色债券、ESG 投资有税收优惠
- **投行/尽调**：PE/VC 投资时标配 ESG 尽调，ESG 风险影响估值
- **财务报告**：上市公司需要在年报中披露 ESG 信息，财务与非财务数据融合趋势明显

## 二、主要 ESG 披露框架

### 全球框架对比

| 框架 | 全称 | 定位 | 受众 |
|------|------|------|------|
| ISSB | 国际可持续准则理事会 | 全球基准，投资者导向 | 投资者 |
| GRI | 全球报告倡议组织 | 影响导向，双重重要性 | 多利益相关方 |
| TCFD | 气候相关财务信息披露工作组 | 气候风险，财务影响 | 投资者/监管 |
| SASB | 可持续会计准则委员会 | 行业专项，财务重要性 | 投资者 |
| CSRD/ESRS | 欧盟企业可持续发展报告指令 | 欧盟强制，双重重要性 | 欧盟监管 |

**2026年格局**：ISSB 已成为全球基准，GRI/TCFD/SASB 逐步向 ISSB 靠拢。中国、新加坡、香港、澳大利亚均已推进 ISSB 对齐的强制披露。

### ISSB 核心标准

- **IFRS S1**：可持续相关财务信息披露一般要求（治理、战略、风险管理、指标与目标）
- **IFRS S2**：气候相关披露（物理风险、转型风险、Scope 1/2/3 排放）

**四大披露要素**（TCFD 框架，ISSB 沿用）：
1. **治理**：董事会和管理层如何监督气候/可持续风险
2. **战略**：气候风险和机遇对业务、战略、财务的影响
3. **风险管理**：识别、评估、管理气候风险的流程
4. **指标与目标**：量化指标和减排目标

## 三、中国 ESG 监管要求

### 披露要求演进

| 时间 | 政策 | 要求 |
|------|------|------|
| 2022年前 | 自愿披露为主 | 鼓励但不强制 |
| 2022-2023 | 上交所/深交所指引 | 部分行业强制，其他鼓励 |
| 2024 | 沪深北三所联合发布《上市公司可持续发展报告指引》（2024年5月发布，2024年6月施行） | 沪深300、科创50等强制，其他鼓励 |
| 2026起 | 强制范围扩大 | 沪深300成分股强制披露，覆盖400+大型企业 |

### 中国可持续发展报告指引（2024版）核心要求

**强制披露内容**（适用于沪深300等重点指数成分股）：
- 应对气候变化（Scope 1、Scope 2 排放量）
- 污染物排放
- 生态保护
- 乡村振兴
- 员工权益保障

**鼓励披露内容**：
- Scope 3 排放（供应链上下游）
- 生物多样性
- 供应链管理
- 产品安全

**中国特色**：
- 双重重要性原则（财务重要性 + 影响重要性）
- 强调乡村振兴、共同富裕等中国特色议题
- 与 ISSB 对齐但有本土化调整

### 企业可持续披露准则体系（财政部发布）

> 区别于交易所层面的"上市公司可持续发展报告指引"，这是**会计准则体系层面**的文件，由财政部牵头制定，定位为中国版ISSB。

**体系结构**：

| 层级 | 文件 | 状态 |
|------|------|------|
| 基本准则 | 《企业可持续披露准则——基本准则（试行）》 | 2024年12月发布 |
| 具体准则第1号 | 《企业可持续披露准则第1号——气候相关披露（试行）》 | 2025年12月发布 |
| 具体准则第2号+ | 后续环境、社会、治理各专项准则 | 规划中，2027年前陆续出台 |
| 应用指南 | 行业应用指南（电力、钢铁、煤炭、石油、水泥、汽车等） | 开发中 |

**实施路线图**：
- 当前：自愿试行
- 逐步扩展：上市公司→非上市公司，大型企业→中小企业，定性→定量，自愿→强制
- 目标：2030年建成全国统一的企业可持续信息披露制度

#### 基本准则核心内容（2024年12月17日发布）

**发文机关**：财政部会同国资委、金融监管总局、证监会等九部门

**六大章节**：

| 章节 | 内容 |
|------|------|
| 总则 | 准则目的、适用范围、与财务报告的关系 |
| 披露目标与原则 | 重要性原则（双重重要性）、决策有用性 |
| 信息质量要求 | 可靠性、相关性、可比性、可验证性、可理解性、及时性 |
| 披露要素 | 治理、战略、风险和机遇管理、指标与目标（对标ISSB S1四大支柱） |
| 其他披露要求 | 报告期间、可比信息、合规声明、差错更正 |
| 附则 | 解释权归财政部，实施范围另行规定 |

**关键规则**：
- 适用于在中国境内设立的、被要求披露可持续信息的企业
- 在强制实施范围和要求明确之前，企业可自愿采用
- 与国际准则（ISSB）实质趋同，但保留中国本土化调整空间

#### 气候相关披露准则核心内容（2025年12月25日发布）

**对标**：ISSB IFRS S2

**四大披露支柱**：

| 支柱 | 披露要求 |
|------|----------|
| 治理 | 监督气候风险和机遇的治理结构、控制措施和程序 |
| 战略 | 气候变化对商业模式、战略和财务规划的影响；气候情景分析评估韧性 |
| 风险和机遇管理 | 识别、评估和优先排序气候风险和机遇的流程 |
| 指标与目标 | 定量和定性绩效数据，包括Scope 1、Scope 2、Scope 3温室气体排放 |

**中国特色要求**：
- **双重重要性**：既披露气候对企业的财务影响（财务重要性），也披露企业对气候的影响（影响重要性）
- **本土碳核算方法优先**：温室气体排放须使用中国主管部门制定的碳核算标准计算，明确披露计算依据；国际GHG Protocol仅在本土标准不可用时作为参考
- **对齐"双碳"目标**：披露须与中国2030年碳达峰、2060年碳中和目标及国家行业路线图衔接
- **Scope 3要求**：企业应尽合理努力收集供应链数据并披露相关风险和机遇；无法获取时应估算并说明

**与交易所指引的关系**：

| 对比项 | 交易所指引（2024年6月施行） | 财政部准则（2024年12月发布） |
|--------|---------------------------|---------------------------|
| 性质 | 监管规则（信息披露要求） | 会计准则体系（披露标准） |
| 发布方 | 沪深北交易所 | 财政部等九部门 |
| 适用对象 | 上市公司（沪深300等强制） | 所有在中国设立的企业（远期） |
| 强制时间 | 2026年起分批强制 | 当前自愿，2030年前逐步强制 |
| 关系 | 两者互补，交易所指引是"谁必须披露"，财政部准则是"怎么披露" |

## 四、碳核算基础

### 三个范围（Scope）

```
Scope 1：直接排放
  企业自有或控制的排放源
  例：自有锅炉燃烧、公司车队、生产过程排放

Scope 2：间接排放（能源）
  购买的电力、热力、蒸汽产生的排放
  例：工厂用电、办公室空调

Scope 3：其他间接排放（价值链）
  上游：原材料采购、供应商生产、员工通勤
  下游：产品使用、产品废弃处置、投资组合排放
```

**重要性**：Scope 3 通常占企业总排放的 70-90%，但最难计量。2026年后主要市场逐步要求披露。

### 碳排放计算方法

**排放因子法**（最常用）：
```
排放量（tCO₂e）= 活动数据 × 排放因子

例：
用电排放 = 用电量(kWh) × 电网排放因子(kgCO₂e/kWh)
燃油排放 = 燃油消耗量(L) × 燃油排放因子(kgCO₂e/L)
```

**排放因子来源**：
- 中国：生态环境部发布的省级电网排放因子
- 国际：IPCC 排放因子数据库、GHG Protocol

**碳排放单位**：tCO₂e（吨二氧化碳当量），将不同温室气体换算为 CO₂ 当量

### 碳中和路径

```
净零排放 = 减少排放 + 碳抵消（碳汇/碳信用）

碳抵消方式：
- 购买碳信用（CCER、VCS等）
- 植树造林
- 碳捕获与封存（CCS）
```

## 五、ESG 信息披露鉴证

### 鉴证 vs 审计

| 维度 | 财务报表审计 | ESG 鉴证 |
|------|------------|---------|
| 对象 | 财务报表 | ESG 报告/可持续发展报告 |
| 准则 | 中国审计准则（CSA） | ISAE 3000、AA1000AS |
| 保证程度 | 合理保证（高） | 有限保证（低）或合理保证 |
| 成熟度 | 高度成熟 | 快速发展中 |
| 执行主体 | 注册会计师 | 注册会计师/专业机构 |

### 鉴证程序要点

**有限保证鉴证**（目前主流）：
- 了解被鉴证方的 ESG 数据收集和报告流程
- 分析性程序（合理性检查）
- 询问关键人员
- 抽样核查原始数据
- 结论：未发现重大错报（负面保证）

**合理保证鉴证**（更高标准）：
- 更广泛的测试程序
- 实质性测试
- 结论：数据在所有重大方面公允呈现（正面保证）

### ESG 数据质量常见问题

- 数据收集分散，依赖人工填报，错误率高
- 排放因子选取不一致
- 边界界定不清（哪些子公司纳入）
- Scope 3 数据缺失或估算方法不透明
- 历史数据不可比（口径变化）

## 六、ESG 与税务的交叉

### 碳税与环保税

**中国环境保护税**（2018年起）：
- 应税污染物：大气污染物、水污染物、固体废物、噪声
- 税额：按污染当量计算，各省税额不同
- 减免：排放量低于标准的可享受减税

**碳市场与碳税**：
- 中国全国碳排放权交易市场（ETS）：目前覆盖电力行业，逐步扩大
- 碳配额的会计处理：无偿分配的配额确认为政府补助还是零成本资产，存在争议
- 欧盟碳边境调节机制（CBAM）：对出口欧盟的高碳产品征收碳关税，影响中国出口企业

### 绿色税收优惠

| 优惠类型 | 内容 |
|---------|------|
| 节能节水设备 | 购置节能节水专用设备，投资额10%抵免企业所得税 |
| 环保设备 | 购置环境保护专用设备，投资额10%抵免企业所得税 |
| 绿色债券利息 | 部分绿色债券利息收入免税 |
| 碳汇项目 | 林业碳汇相关收入有税收优惠 |

## 七、ESG 尽调

### PE/VC 投资中的 ESG 尽调

**为什么做 ESG 尽调**：
- 监管要求（LP 要求、基金管理人责任）
- 风险识别（ESG 风险可能影响估值和退出）
- 价值创造（ESG 改善可提升企业价值）

**ESG 尽调核心关注点**：

| 维度 | 关注内容 | 风险信号 |
|------|---------|---------|
| 环境 | 污染记录、环保合规、碳排放 | 重大环保处罚、高碳行业转型压力 |
| 社会 | 劳工纠纷、安全事故、供应链 | 重大工伤事故、童工/强迫劳动指控 |
| 治理 | 股权结构、关联交易、反腐 | 实控人集权、大额关联交易、行贿记录 |

**ESG 尽调输出**：
- ESG 风险评级（高/中/低）
- 重大 ESG 风险清单及影响评估
- 投后 ESG 改善建议（100天计划）

## 八、ESG 评级

### 主要评级机构

| 机构 | 特点 |
|------|------|
| MSCI ESG | 全球最广泛使用，AAA-CCC评级 |
| 华证 ESG | 中国本土，覆盖A股 |
| 商道融绿 | 中国本土，侧重社会责任 |
| Wind ESG | 数据覆盖广，与Wind数据库集成 |

### 评级逻辑

```
ESG 评级 = 行业调整后的 E/S/G 各维度得分加权

关键：同一行业内横向比较，不同行业间不可直接比较
（石油公司的高评级 ≠ 科技公司的高评级）
```

**影响评级的关键因素**：
- 信息披露质量（披露越详细，评级越高）
- 重大 ESG 事件（负面事件大幅拉低评级）
- 行业特定指标（不同行业权重不同）

## 九、重要提示

ESG 领域政策变化快，以下内容需定期核实：
- 中国强制披露范围和时间表
- 碳市场覆盖行业和配额分配方式
- 各省环保税税额标准
- 绿色税收优惠政策有效期

建议关注：生态环境部、上交所/深交所官网、财政部绿色金融相关政策。
