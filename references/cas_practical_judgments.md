# 会计准则实务判断速查

> 定位：教材不会写、但实务中经常遇到的边界判断
> 来源：审计实务经验提炼
> 覆盖：DTA确认、商誉减值、顺流逆流、资本化时点、预计负债、政策vs估计变更
> 性质：判断

---

## 一、递延所得税

### DTA 确认的关键判断

| 问题 | 判断规则 |
|------|----------|
| DTA 确认有上限吗？ | 没有固定百分比上限。关键看"未来很可能取得足够应纳税所得额"。预计未来5年盈利好→可全额确认 |
| 持续亏损企业能确认 DTA 吗？ | 谨慎确认。如果预计未来5年无法盈利，不应确认或应全额计提减值 |
| 未弥补亏损的 DTA 怎么算？ | DTA = min(未弥补亏损额, 预计未来应纳税所得额) × 税率 |
| 减值准备长期挂账（>3年），DTA 还能确认吗？ | 非常谨慎。如果收回可能性很低，可能需要全额计提 |

### 常见暂时性差异方向速查

| 项目 | 差异方向 | 确认 |
|------|----------|------|
| 计提坏账准备 | 可抵扣（账面<计税） | DTA |
| 计提存货跌价 | 可抵扣 | DTA |
| 研发费用资本化 | 可抵扣（加计摊销使计税基础>账面） | DTA |
| 会计折旧>税法折旧 | 可抵扣 | DTA |
| 会计折旧<税法折旧 | 应纳税 | DTL |
| 公允价值上升（交易性金融资产） | 应纳税（账面>计税） | DTL |
| 预收账款（增值税层面收款即产生纳税义务） | 应纳税 | DTL |
| 未弥补亏损 | 可抵扣 | DTA（视盈利预期） |
| 广告费超支 | 可抵扣 | DTA |

### 税率变化处理

- 税率上升 → DTA 增加 / DTL 增加
- 税率下降 → DTA 减少 / DTL 减少
- 差额计入当期所得税费用（除非原始差异计入 OCI/权益）

## 二、资产减值与商誉

### 商誉减值测试 S-A-B 流程

**Step 1 分摊（Allocation）：**
- 商誉分摊至能从合并协同效应中受益的资产组
- 不得晚于首年财务报告批准报出日
- 以后期间不得重新分摊（除非资产结构重大变化）
- 分摊层次不低于分部报告层次

**Step 2 比较（Compare）：**
- 每年末强制测试（不论有无减值迹象）
- 可收回金额 = max(公允价值-处置费用, 预计未来现金流现值)
- 含商誉的资产组账面价值 vs 可收回金额

**Step 3 分摊减值（Allocate）：**
- 先冲减商誉，再按比例冲减资产组内其他资产
- 但冲减后各资产账面价值不得低于其单独可收回金额

### 资产组认定判断

| 问题 | 判断规则 |
|------|----------|
| 什么是资产组？ | 能够独立产生现金流入的最小资产组合 |
| 一条生产线算一个资产组吗？ | 如果该生产线独立产生收入且可单独核算→是 |
| 总部资产要分摊吗？ | 如果能合理分摊→分摊；不能→扩大资产组范围 |
| 资产组可以跨分部吗？ | 不可以。资产组不得大于分部报告层次 |

## 三、合并报表

### 顺流/逆流交易抵销

| 类型 | 抵销规则 |
|------|----------|
| 顺流（母→子） | 全额抵销未实现损益 |
| 逆流（子→母） | 全额抵销未实现损益，但少数股东按比例分担 |

**逆流交易分录（母持股60%）：**
```
借：少数股东损益（未实现损益 × 40%）
贷：少数股东权益
借：营业收入
贷：营业成本
    存货（未实现部分）
```

### 同控合并留存收益恢复

- 同一控制下企业合并，合并日需要恢复子公司留存收益
- 恢复金额 = 子公司在合并前的留存收益 × 母公司持股比例
- 分录：借资本公积，贷盈余公积/未分配利润

### 超额亏损处理

- 子公司亏损超过少数股东权益时，超额部分仍由少数股东承担
- 少数股东权益可以为负数
- 除非少数股东有法定义务承担额外损失

## 四、借款费用资本化

### 资本化三时点判断

| 时点 | 条件（同时满足） |
|------|-----------------|
| 开始资本化 | ①资产支出已发生 ②借款费用已发生 ③为使资产达到预定可使用状态的购建活动已开始 |
| 暂停资本化 | 非正常中断且连续超过3个月 |
| 停止资本化 | 资产达到预定可使用状态或可销售状态 |

### 实务判断

| 问题 | 判断 |
|------|------|
| "达到预定可使用状态"怎么判断？ | 实体建造完成+安装调试完毕+试运行合格。不以竣工验收为准，以实际可使用为准 |
| 部分完工部分使用，利息还能资本化吗？ | 已使用部分停止资本化，未完工部分继续 |
| 一般借款资本化率怎么算？ | 加权平均利率 = Σ(各笔一般借款本金×利率) ÷ Σ(各笔一般借款本金) |

## 五、政府补助

### 总额法 vs 净额法判断

| 情形 | 方法 | 理由 |
|------|------|------|
| 补助与日常活动相关 | 总额法计入其他收益，或净额法冲减成本费用 | 企业可选择，但需一致 |
| 补助与日常活动无关 | 计入营业外收入 | 不可选择 |
| 补助退回（总额法） | 冲减递延收益余额，不足部分计入当期损益 | — |
| 补助退回（净额法） | 调整资产账面价值或恢复已冲减的费用 | — |

### 实务判断

| 问题 | 判断 |
|------|------|
| 增值税即征即退算政府补助吗？ | 算。属于与日常活动相关的政府补助 |
| 先征后返的所得税算吗？ | 算。计入其他收益或营业外收入 |
| 政府补助能确认为收入吗？ | 不能。只能计入其他收益/营业外收入/冲减成本 |

## 六、金融工具

### SPPI 测试失败的常见情形

| 情形 | SPPI 结果 | 分类 |
|------|-----------|------|
| 浮动利率贷款（基准利率+固定利差） | 通过 | AC 或 FVOCI |
| 与股票指数挂钩的结构性存款 | 失败 | FVTPL |
| 可转换债券（持有方） | 失败 | FVTPL |
| 含提前偿还权的贷款（补偿金额合理） | 通过 | AC |
| 逆浮动利率工具 | 失败 | FVTPL |

### ECL 三阶段判断

| 阶段 | 信用风险变化 | 计提范围 |
|------|-------------|----------|
| 第一阶段 | 未显著增加 | 12个月 ECL |
| 第二阶段 | 显著增加但未发生信用减值 | 整个存续期 ECL |
| 第三阶段 | 已发生信用减值 | 整个存续期 ECL（利息按摊余成本净额计算） |

**"信用风险显著增加"的判断标准：**
- 逾期超过 30 天（可推翻的推定）
- 外部信用评级下降
- 债务人经营状况恶化
- 宏观经济环境恶化

## 七、或有事项与预计负债

### 预计负债确认三条件

同时满足：
1. 现时义务（法定或推定）
2. 很可能导致经济利益流出（>50%）
3. 金额能够可靠计量

### 最佳估计数

| 情形 | 计算方法 |
|------|----------|
| 单一结果 | 最可能发生的金额 |
| 连续范围（各点等概率） | 中间值（上下限平均） |
| 多个可能结果 | 加权平均（概率×金额） |

### 实务判断

| 问题 | 判断 |
|------|------|
| 亏损合同要确认预计负债吗？ | 是。不可撤销合同的预计亏损应确认预计负债 |
| 未决诉讼"可能"败诉（50%），确认吗？ | "可能"≠"很可能"。50%不满足确认条件，只需披露 |
| 产品质量保证要确认吗？ | 是。按历史经验估计保修费用，确认预计负债 |
| 重组计划公布了但还没执行，确认吗？ | 已对外公布详细正式方案→确认；仅内部决议→不确认 |

## 八、股份支付

### 权益结算 vs 现金结算

| 类型 | 计量基础 | 后续变动 |
|------|----------|----------|
| 权益结算 | 授予日公允价值（锁定） | 不调整公允价值，只调整预计可行权数量 |
| 现金结算 | 每个资产负债表日公允价值（重估） | 每期调整公允价值 |

### 实务判断

| 问题 | 判断 |
|------|------|
| 限制性股票回购条款，算权益还是现金结算？ | 看回购是否"很可能"发生。如果员工离职才回购→权益结算 |
| 等待期内离职，已确认的费用要冲回吗？ | 权益结算：冲回（调整预计可行权数量）。现金结算：同样冲回 |
| 条款修改（降低行权价），怎么处理？ | 有利修改：按修改后公允价值重新计量增量部分。不利修改：继续按原条件确认 |

## 九、租赁

### 承租人不确认使用权资产的豁免

| 豁免情形 | 条件 |
|----------|------|
| 短期租赁 | 租赁期 ≤ 12个月（不含购买选择权） |
| 低价值资产 | 单项资产全新时价值较低（实务中通常 < 5万元） |

### 实务判断

| 问题 | 判断 |
|------|------|
| 续租选择权要不要纳入租赁期？ | 如果"合理确定"会行使→纳入。判断标准：经济激励（搬迁成本高/装修投入大） |
| 租赁变更（扩大面积），怎么处理？ | 如果增加了单独租赁→视为新租赁。否则→重新计量租赁负债 |
| 免租期怎么处理？ | 租金总额在整个租赁期内分摊（含免租期），不是免租期不确认费用 |

## 十、职工薪酬

### 辞退福利确认时点

- 企业不能单方面撤回辞退计划时确认
- 不是实际支付时确认
- 如果辞退计划跨期执行（>12个月），按折现值确认

### 实务判断

| 问题 | 判断 |
|------|------|
| 年终奖12月计提还是次年发放时确认？ | 12月计提。属于短期薪酬，在员工提供服务的期间确认 |
| 社保/公积金算职工薪酬吗？ | 算。属于短期薪酬中的"其他短期薪酬" |
| 股权激励算职工薪酬吗？ | 算。属于股份支付，在等待期内分期确认 |

## 十一、非货币性资产交换

### 25% 标准判断

| 角度 | 公式 | 结论 |
|------|------|------|
| 收到补价方 | 收到的补价 ÷ 换出资产公允价值 < 25% | 非货币性资产交换 |
| 支付补价方 | 支付的补价 ÷（换出资产公允价值 + 支付的补价）< 25% | 非货币性资产交换 |

> 比例 ≥ 25% 的，不视为非货币性资产交换，适用收入等相关准则。

### 商业实质判断

同时满足以下之一即具有商业实质：
1. 换入资产的未来现金流在风险、时间、金额方面与换出资产显著不同
2. 换入资产与换出资产的预计未来现金流现值不同，且差额与公允价值相比是重大的

## 十二、会计政策变更 vs 会计估计变更

### 区分标准

| 类型 | 定义 | 处理方法 |
|------|------|----------|
| 会计政策变更 | 对相同交易采用不同的会计政策 | 追溯调整法（调期初留存收益） |
| 会计估计变更 | 对相同交易的估计发生变化 | 未来适用法（不调以前年度） |

### 实务判断

| 问题 | 判断 |
|------|------|
| 折旧年限从10年改为5年 | 会计估计变更（未来适用） |
| 折旧方法从直线法改为加速法 | 会计估计变更（未来适用） |
| 存货计价从先进先出改为加权平均 | 会计政策变更（追溯调整） |
| 坏账计提比例从5%改为10% | 会计估计变更（未来适用） |
| 投资性房地产从成本模式改为公允价值模式 | 会计政策变更（追溯调整） |
| 无法区分是政策变更还是估计变更 | 按会计估计变更处理（未来适用） |
