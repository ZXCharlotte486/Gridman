## 一、内部控制基本框架

### 1.1 COSO内部控制整合框架（2013版）

**定义：** 内部控制是由组织的董事会、管理层和其他人员实施的，旨在为实现运营、报告和合规三类目标提供合理保证的过程。

**三大目标：**
- **运营目标**：运营的效率和效果
- **报告目标**：财务和非财务报告的可靠性、及时性、透明度
- **合规目标**：遵循适用的法律法规

**五大要素：**

| 要素 | 核心内容 |
|------|---------|
| 控制环境 | 组织对内控重要性的态度和行动。包括：诚信与道德价值观、董事会独立性与监督、组织架构与权责分配、人才吸引/培养/留用、绩效考核与问责 |
| 风险评估 | 识别和分析实现目标的相关风险。包括：明确目标、识别风险、评估风险（可能性×影响程度）、评估变化（外部环境/业务模式/领导层变化） |
| 控制活动 | 确保管理层指令得以执行的政策和程序。包括：授权审批、职责分离、实物控制、信息处理控制、绩效评价、IT一般控制与应用控制 |
| 信息与沟通 | 获取和传递相关信息。包括：内部信息系统、内部沟通（上行/下行/横向）、外部沟通（监管机构/审计师/利益相关方） |
| 监控活动 | 评估内控是否存在并有效运行。包括：持续监控（日常管理活动中嵌入）、单独评价（内审/专项评价）、缺陷报告与整改 |

**17项原则（每个要素对应的具体原则）：**

控制环境（5项）：
1. 组织对诚信和道德价值观的承诺
2. 董事会独立于管理层并对内控进行监督
3. 管理层建立适当的组织架构、报告路线和权责
4. 组织吸引、培养和留用胜任的人才
5. 组织对内控责任进行问责

风险评估（4项）：
6. 组织明确目标以识别和评估相关风险
7. 组织识别实现目标的风险并分析如何管理
8. 组织在评估风险时考虑舞弊的可能性
9. 组织识别和评估可能重大影响内控的变化

控制活动（3项）：
10. 组织选择和制定有助于降低风险的控制活动
11. 组织选择和制定技术方面的一般控制活动
12. 组织通过政策和程序部署控制活动

信息与沟通（3项）：
13. 组织获取或生成相关、优质的信息
14. 组织在内部沟通信息
15. 组织与外部沟通信息

监控活动（2项）：
16. 组织选择、制定和执行持续和/或单独评价
17. 组织评估和沟通内控缺陷

## 二、风险评估方法

### 2.1 风险识别

**常用方法：**
- 头脑风暴法
- 流程分析法（沿业务流程逐环节识别）
- 检查表法（基于历史经验的风险清单）
- 情景分析法
- SWOT分析
- PEST分析（政治/经济/社会/技术）

**风险分类：**

| 维度 | 分类 |
|------|------|
| 按来源 | 外部风险（政策、市场、自然灾害）、内部风险（管理、运营、财务） |
| 按性质 | 战略风险、财务风险、运营风险、合规风险 |
| 按可控性 | 可控风险、不可控风险 |

### 2.2 风险分析与评估

**风险矩阵（可能性 × 影响程度）：**

| | 影响程度低 | 影响程度中 | 影响程度高 |
|---|---|---|---|
| **可能性高** | 中风险 | 高风险 | 极高风险 |
| **可能性中** | 低风险 | 中风险 | 高风险 |
| **可能性低** | 极低风险 | 低风险 | 中风险 |

**风险应对策略：**
- **规避**：放弃产生风险的活动
- **降低**：采取措施降低可能性或影响程度
- **分担**：通过保险、外包等转移部分风险
- **接受**：在风险容忍度内不采取额外措施

### 2.3 舞弊风险评估（舞弊三角理论）

舞弊发生需同时具备三个条件：
- **动机/压力**：财务压力、业绩考核压力、个人贪欲
- **机会**：内控薄弱、职责分离不当、监督缺失
- **合理化**：自我辩解（"只是借用"、"公司欠我的"）

**常见舞弊类型：**
- 财务报表舞弊（虚增收入、隐瞒负债、操纵利润）
- 资产侵占（挪用资金、虚报费用、盗窃存货）
- 腐败（商业贿赂、利益冲突、非法馈赠）

## 三、关键控制活动

### 3.1 职责分离（不相容职务分离）

**核心原则：** 一项经济业务的授权、执行、记录、保管、核查五个环节不能由同一人或同一部门完成。

**常见不相容职务：**

| 业务 | 不相容职务 |
|------|-----------|
| 货币资金 | 出纳 vs 会计记账；资金审批 vs 资金执行 |
| 采购 | 请购 vs 审批；采购 vs 验收；采购 vs 付款 |
| 销售 | 合同签订 vs 发货；发货 vs 收款；收款 vs 记账 |
| 存货 | 采购 vs 保管；保管 vs 记账；盘点 vs 记账 |
| 固定资产 | 购置审批 vs 购置执行；保管 vs 记账；处置审批 vs 处置执行 |
| 投资 | 投资决策 vs 投资执行；投资执行 vs 投资记录 |

### 3.2 授权审批控制

**授权类型：**
- **一般授权**：对常规业务的授权（如日常采购限额内审批）
- **特别授权**：对非常规或重大业务的逐笔授权（如超限额采购、重大投资）

**授权审批要素：**
- 授权范围（哪些业务需要审批）
- 授权层级（不同金额对应不同审批层级）
- 授权责任（审批人对审批结果负责）
- 授权记录（审批留痕）

### 3.3 IT一般控制（ITGC）

| 控制领域 | 核心内容 |
|---------|---------|
| 程序开发 | 系统开发生命周期管理、变更管理、测试与上线审批 |
| 程序变更 | 变更申请、审批、测试、迁移的分离控制 |
| 计算机运行 | 作业调度、异常处理、备份恢复 |
| 访问安全 | 用户账号管理、密码策略、权限分配、特权账号管理、日志审计 |

### 3.4 IT应用控制

| 控制类型 | 核心内容 |
|---------|---------|
| 输入控制 | 数据完整性校验、格式校验、逻辑校验、重复检查 |
| 处理控制 | 计算准确性、数据匹配、异常处理 |
| 输出控制 | 报表完整性、分发权限、打印控制 |
| 接口控制 | 系统间数据传输的完整性、准确性、及时性 |

## 四、内控评价

### 4.1 评价流程

1. 制定评价方案（范围、标准、方法、时间）
2. 组建评价团队
3. 实施现场测试（穿行测试、控制测试）
4. 认定内控缺陷
5. 编制评价报告

### 4.2 内控缺陷分类

| 缺陷等级 | 定义 | 报告要求 |
|---------|------|---------|
| 重大缺陷 | 一个或多个控制缺陷的组合，可能导致严重偏离控制目标 | 必须在内控评价报告中披露 |
| 重要缺陷 | 严重程度低于重大缺陷，但仍应引起管理层关注 | 应在内控评价报告中披露 |
| 一般缺陷 | 除重大缺陷和重要缺陷之外的其他缺陷 | 以适当形式报告 |

**缺陷认定标准（定量+定性）：**

定量标准（示例）：
- 重大缺陷：错报金额 ≥ 税前利润的5%
- 重要缺陷：错报金额在税前利润的3%-5%之间
- 一般缺陷：错报金额 < 税前利润的3%

定性标准（直接认定为重大缺陷的情形）：
- 董事、监事和高管舞弊
- 已报告的重大缺陷经过合理时间后未得到整改
- 注册会计师发现的重大错报未被内控识别
- 审计委员会和内审部门对内控监督无效

### 4.3 内控审计（注册会计师执行）

**依据：** 《企业内部控制审计指引》

**审计目标：** 对财务报告内部控制的有效性发表审计意见。

**审计意见类型：**
- 无保留意见：内控在所有重大方面有效
- 否定意见：存在重大缺陷
- 无法表示意见：审计范围受限

> **注意：** 内控审计只对财务报告内部控制发表意见，不对非财务报告内部控制发表意见（但发现的非财务报告重大缺陷应在审计报告中增加"非财务报告内部控制重大缺陷描述段"）。

## 五、常见问题判断

**Q：内控能否完全消除风险？**
不能。内控只能提供"合理保证"，不是"绝对保证"。固有局限包括：人为判断失误、串通舞弊、管理层凌驾于控制之上、成本效益约束。

**Q：职责分离在小企业人手不足时怎么办？**
可通过补偿性控制弥补：加强管理层直接监督、增加独立核查频率、利用IT系统强制控制、定期轮岗。

**Q：IT一般控制和应用控制的关系？**
ITGC是基础，如果ITGC失效（如程序变更不受控），则依赖该系统的所有应用控制都不可信赖。审计时先测ITGC，ITGC有效才测应用控制。

**Q：内控评价报告和内控审计报告的区别？**
内控评价报告是企业自己做的（管理层责任），内控审计报告是注册会计师做的（审计责任）。两者结论可能不一致。

## 第一章 战略与战略管理概述

### 1.1 公司战略的定义

| 概念 | 核心观点 | 强调属性 |
|------|---------|---------|
| 传统概念（波特） | 战略是公司为之奋斗的终点与达到终点的途径的结合物 | 计划性、全局性、长期性 |
| 现代概念（明茨伯格） | 战略是一系列或整套的决策或行动方式，包括计划性战略和非计划性战略 | 应变性、竞争性、风险性 |
| 综合观点（汤姆森） | 战略既是预先性的（预谋战略），又是反应性的（适应性战略） | 动态调整 |

### 1.2 公司使命与目标体系

**公司使命三要素：**
1. **公司目的**：组织存在的根本性质（营利/非营利）
2. **公司宗旨**：长期战略意向，说明经营业务范围（产品/顾客/市场/技术）
3. **经营哲学**：价值观、基本信念和行为准则

**公司目标体系：**
- **财务目标**：市场占有率、收益增长率、投资回报率、股利增长率、现金流等
- **战略目标**：竞争优势、产品质量、客户服务、技术领导地位、持久竞争力等
- 两类目标均需从短期和长期两个维度体现

### 1.3 公司战略的三个层次

| 层次 | 别称 | 主要内容 | 决策主体 |
|------|------|---------|---------|
| 总体战略 | 公司层战略 | 选择经营领域、配置资源、协调各业务 | 企业最高管理层 |
| 业务单位战略 | 竞争战略 | 在各经营领域有效竞争、控制资源分配 | 业务单位主管 |
| 职能战略 | 职能层战略 | 各职能部门如何配置资源、提高效率 | 职能部门负责人 |

> 单业务公司：总体战略与业务单位战略合二为一。

### 1.4 战略管理过程

```
战略分析 → 战略选择 → 战略实施 → （循环往复）
```

**战略分析：**
- 外部环境分析（宏观环境、产业环境、竞争环境）→ 机会与威胁
- 内部环境分析（资源与能力、价值链、业务组合）→ 优势与劣势

**战略选择过程（约翰逊和施乐斯）：**
1. 制定战略选择方案（自上而下/自下而上/上下结合）
2. 评估战略备选方案（适宜性/可接受性/可行性）
3. 选择战略（根据目标/上级审批/外部专家）

**战略实施关键工作：**
- 调整组织结构适配战略
- 推进企业文化建设
- 运用财务和非财务手段监控战略进程
- 采用数字化技术支持转型
- 协调战略、组织、文化、技术的关系

### 1.5 创新与创新战略

**创新价值链：** 创意 → 发明 → 研发 → 创新（商业化价值实现）

**创新类型体系：**

| 分类维度 | 类型 | 说明 |
|---------|------|------|
| 按内容对象 | 产品创新 | 全新或显著改进的产品/服务 |
| | 流程创新 | 新的生产方法/设备，提升运营效率 |
| | 定位创新 | 不改变核心功能，重新定义市场定位 |
| | 范式创新 | 重新定义价值创造的思维模式和底层逻辑 |
| 按新颖程度 | 渐进性创新 | 小幅持续优化，风险低 |
| | 突破性创新 | 重大技术突破，质的飞跃，风险很高 |
| | 颠覆性创新 | 全新范式，从低端切入颠覆行业，风险极高 |
| 按系统层面 | 组件创新 | 单一模块技术改进 |
| | 架构创新 | 多组件系统性重构 |

**创新生命周期模型（阿伯内西-厄特巴克）：**

| 阶段 | 竞争重点 | 创新类型 | 产品线 | 生产流程 |
|------|---------|---------|--------|---------|
| 流变阶段 | 产品性能 | 产品重大变革 | 多样性/定制 | 灵活但低效 |
| 过渡阶段 | 产品差异化 | 重大流程创新 | 形成主导设计 | 逐渐规范化 |
| 成熟阶段 | 降低成本 | 渐进性改进 | 高度标准化 | 高效/资本集约 |

**精益创新五原则：** 客户导向、行动原则、试错原则、聚焦原则、迭代原则

## 第二章 战略分析

### 2.1 外部环境分析

#### 2.1.1 宏观环境分析——PESTEL模型

| 因素 | 分析内容 |
|------|---------|
| P-政治 | 政治体制、政府政策、政治稳定性、税收政策、贸易限制 |
| E-经济 | GDP增长率、利率、汇率、通货膨胀率、失业率、经济周期 |
| S-社会 | 人口结构、收入分配、生活方式、教育水平、消费习惯 |
| T-技术 | 技术变革速度、研发投入、技术转让、自动化程度 |
| E-生态 | 环保法规、碳排放、可持续发展要求、气候变化 |
| L-法律 | 反垄断法、劳动法、消费者保护法、知识产权法 |

#### 2.1.2 产业环境分析——波特五力模型

| 竞争力量 | 分析要点 |
|---------|---------|
| 现有竞争者的竞争强度 | 竞争者数量与规模、行业增长率、固定成本比例、产品差异化程度、退出壁垒 |
| 潜在进入者的威胁 | 规模经济、资本需求、转换成本、分销渠道、政府政策、品牌忠诚度 |
| 替代品的威胁 | 替代品的相对价格、转换成本、买方对替代品的倾向 |
| 供应商的议价能力 | 供应商集中度、替代投入品、转换成本、前向一体化威胁 |
| 购买者的议价能力 | 买方集中度、购买量、产品差异化、转换成本、后向一体化威胁 |

**五力模型的战略含义：**
- 五种力量越强 → 行业平均利润率越低
- 企业应选择五力较弱的行业，或在行业中建立防御壁垒

#### 2.1.3 产业生命周期

| 阶段 | 特征 | 战略重点 |
|------|------|---------|
| 导入期 | 需求有限、技术不确定、进入壁垒低 | 产品开发、市场教育 |
| 成长期 | 需求快速增长、竞争者增加 | 扩大市场份额、建立品牌 |
| 成熟期 | 增长放缓、竞争激烈、产能过剩 | 效率提升、差异化、市场细分 |
| 衰退期 | 需求下降、退出者增多 | 收割/退出/利基市场 |

### 2.2 内部环境分析

#### 2.2.1 资源与能力分析

**企业资源分类：**
- **有形资源**：财务资源（现金、融资能力）、实物资源（设备、土地、原材料）
- **无形资源**：技术资源（专利、商业秘密）、声誉资源（品牌、客户忠诚度）
- **人力资源**：员工技能、经验、判断力、适应力

**核心能力判断标准（VRIO框架）：**

| 标准 | 含义 |
|------|------|
| V-有价值 | 能够利用机会或抵御威胁 |
| R-稀缺性 | 竞争对手不具备或很少具备 |
| I-难以模仿 | 竞争对手难以复制（历史条件、因果模糊、社会复杂性） |
| O-组织化 | 企业有能力开发利用该资源 |

#### 2.2.2 价值链分析（波特）

**基本活动：**
1. 进货物流（原材料接收、储存、分配）
2. 生产经营（加工、组装、包装、测试）
3. 出货物流（成品储存、运输、配送）
4. 市场营销（广告、促销、定价、渠道）
5. 服务（安装、维修、培训、零配件供应）

**支持活动：**
1. 企业基础设施（总体管理、计划、财务、法律）
2. 人力资源管理（招聘、培训、薪酬）
3. 技术开发（研发、流程改进、产品设计）
4. 采购（原材料、设备、服务的购买）

**价值链分析的战略意义：**
- 识别哪些活动创造价值、哪些不创造价值
- 与竞争对手比较各环节的成本和差异化
- 决定哪些活动自己做、哪些外包

#### 2.2.3 SWOT分析

| | 正面因素 | 负面因素 |
|---|---|---|
| **内部** | S-优势 | W-劣势 |
| **外部** | O-机会 | T-威胁 |

**SWOT战略组合：**
- SO战略：利用优势抓住机会（增长型）
- WO战略：克服劣势利用机会（扭转型）
- ST战略：利用优势应对威胁（多元化）
- WT战略：减少劣势回避威胁（防御型）

## 第三章 战略选择

### 3.1 总体战略（公司层战略）

#### 3.1.1 三种基本总体战略

| 战略类型 | 适用情形 | 主要方式 |
|---------|---------|---------|
| 发展战略 | 行业前景好、企业有能力扩张 | 一体化、多元化、密集型增长 |
| 稳定战略 | 外部环境稳定、企业经营良好 | 维持现有战略、暂停战略 |
| 收缩战略 | 经营困难、行业衰退 | 紧缩与集中、转向、放弃 |

#### 3.1.2 发展战略的主要途径

**一体化战略：**

| 类型 | 方向 | 含义 | 适用条件 |
|------|------|------|---------|
| 前向一体化 | 向下游 | 控制分销商/零售商 | 分销成本高、分销商不可靠 |
| 后向一体化 | 向上游 | 控制供应商 | 供应不稳定、供应商议价力强 |
| 横向一体化 | 同行业 | 收购/合并竞争对手 | 规模经济显著、行业集中度低 |

**多元化战略：**

| 类型 | 含义 | 优势 | 风险 |
|------|------|------|------|
| 相关多元化 | 进入与现有业务有战略关联的新业务 | 协同效应、资源共享、范围经济 | 管理复杂性增加 |
| 非相关多元化 | 进入与现有业务无关联的新业务 | 分散风险、财务协同 | 缺乏协同、管理难度大 |

**密集型增长战略（安索夫矩阵）：**

| | 现有产品 | 新产品 |
|---|---|---|
| **现有市场** | 市场渗透 | 产品开发 |
| **新市场** | 市场开发 | 多元化 |

#### 3.1.3 收缩战略

| 方式 | 含义 | 适用情形 |
|------|------|---------|
| 紧缩与集中 | 缩减规模、削减成本、集中核心业务 | 短期困难、可以扭转 |
| 转向战略 | 重新定位、调整产品/市场组合 | 战略方向错误需纠正 |
| 放弃战略 | 出售或关闭部分业务 | 业务无法挽救、资源需重新配置 |

### 3.2 业务单位战略（竞争战略）

#### 3.2.1 波特基本竞争战略

| 战略 | 核心逻辑 | 关键资源/能力 | 风险 |
|------|---------|-------------|------|
| 成本领先 | 以低于竞争对手的成本提供产品 | 规模经济、学习曲线、流程创新、严格成本控制 | 技术变革使投资无效、模仿、忽视差异化 |
| 差异化 | 提供独特的产品/服务获取溢价 | 研发能力、品牌、质量、创新、客户服务 | 成本差距过大、模仿、客户需求变化 |
| 集中化 | 专注于特定细分市场 | 对细分市场的深入了解 | 细分市场萎缩、大企业进入 |

**成本领先战略的实现途径：**
- 规模经济与学习效应
- 供应链优化与采购议价
- 产品设计简化
- 生产流程标准化与自动化
- 严格的间接费用控制

**差异化战略的实现途径：**
- 产品特性与功能创新
- 品牌形象与声誉
- 技术领先
- 客户服务与售后支持
- 分销渠道独特性
- 定制化能力

#### 3.2.2 商业模式创新

**商业模式画布九要素：**
1. 客户细分
2. 价值主张
3. 渠道通路
4. 客户关系
5. 收入来源
6. 核心资源
7. 关键业务
8. 重要合作
9. 成本结构

**典型商业模式：**
- 平台商业模式（连接多边市场）
- 长尾商业模式（大量利基产品汇聚）
- 免费商业模式（单方免费、多方收费）
  - 增值服务收费模式
  - 广告模式（三方市场）
  - 交叉补贴模式
  - 非货币市场模式

### 3.3 职能战略

#### 3.3.1 市场营销战略

**STP营销核心：**
- **S-市场细分**：按地理/人口/心理/行为因素划分市场
- **T-目标市场选择**：无差异性/差异性/集中性营销策略
- **P-市场定位**：避强/迎头/并存/取代/重新/领先定位

**营销组合4P：**

| 要素 | 关键决策 |
|------|---------|
| 产品 | 产品组合（宽度/长度/深度/关联度）、品牌策略、新产品开发 |
| 价格 | 成本导向/需求导向/竞争导向定价；渗透/撇脂/温和/免费定价 |
| 分销 | 渠道长度（直接/间接）、宽度（独家/选择/密集）、广度（多渠道/全渠道） |
| 促销 | 广告、公关营销、营业推广、人员推销；推式/拉式/推拉结合策略 |

#### 3.3.2 研究与开发战略

**研发三层次：** 基础性研究 → 应用型研究 → 开发型研究

**研发四种类型：** 产品研发、技术研发、工艺研发、流程研究

**研发四种模式：**

| 模式 | 优点 | 缺点 | 适用企业 |
|------|------|------|---------|
| 自主研发 | 技术领先、知识产权完整 | 投入大、周期长、风险高 | 研发能力强的大企业 |
| 合作研发 | 优势互补、分散风险 | 协调难、难独家领先 | 期望快速提升的企业 |
| 委托研发 | 省精力、快速获得成果 | 自身能力提升有限 | 研发能力有限的企业 |
| 开放研发 | 知识共享、成本低、透明 | 需知识保护措施 | 互联网/平台型企业 |

**研发战略定位四种选择：**
1. 成为向市场推出新技术产品的企业（高风险高回报）
2. 成为成功产品的创新模仿者（需优秀研发+营销）
3. 成为成功产品的低成本生产者（规模营销为主）
4. 成为低成本生产者的模仿者（适合发展中国家企业）

#### 3.3.3 生产运营战略

**生产运营四大影响因素：**
- 批量（高→规模经济/低单位成本；低→灵活/高单位成本）
- 种类（多→灵活/高成本；少→标准化/低成本）
- 需求变动（大→产能利用率问题；稳定→高利用率/低成本）
- 可见性（高→需沟通技巧/高费用；低→可标准化/低费用）

#### 3.3.4 财务战略

**财务战略与企业生命周期：**

| 生命周期阶段 | 资金需求 | 融资策略 | 股利政策 |
|------------|---------|---------|---------|
| 初创期 | 大量资金需求 | 权益融资为主 | 不分配或低股利 |
| 成长期 | 持续资金需求 | 权益+债务混合 | 低股利/股票股利 |
| 成熟期 | 资金充裕 | 债务融资为主 | 高股利/回购股票 |
| 衰退期 | 资金回收 | 减少融资 | 高股利/清算分配 |

### 3.4 国际化经营战略

**国际化动因：** 寻求市场、寻求资源、寻求效率、寻求战略性资产

**国际化战略类型：**

| 战略 | 全球整合压力 | 当地响应压力 | 特点 |
|------|------------|------------|------|
| 国际战略 | 低 | 低 | 将国内能力转移到海外 |
| 多国本土化战略 | 低 | 高 | 各国独立运营、适应当地 |
| 全球化战略 | 高 | 低 | 标准化产品、集中生产 |
| 跨国战略 | 高 | 高 | 兼顾全球效率与本地响应 |

## 第四章 战略实施

### 4.1 组织结构类型

| 结构类型 | 特点 | 适用情形 | 优缺点 |
|---------|------|---------|--------|
| 创业型组织 | 老板直管、无正式结构 | 小型/初创企业 | 灵活但依赖个人 |
| 职能制 | 按职能划分部门 | 单一业务/产品线简单 | 专业化高但跨部门协调难 |
| 事业部制 | 按产品/地区/客户划分 | 多元化经营 | 自主性强但可能重复建设 |
| 矩阵制 | 职能+项目双重汇报 | 复杂项目/多产品线 | 灵活但双重领导易冲突 |
| 战略业务单位（SBU） | 将相关事业部归类管理 | 大型多元化企业 | 层次清晰但增加管理层级 |
| 网络制/虚拟组织 | 核心企业+外部合作网络 | 快速变化的环境 | 灵活低成本但控制力弱 |

**组织结构与战略的匹配（钱德勒命题）：** 结构跟随战略。战略变化时组织结构应相应调整。

### 4.2 企业文化与战略

**企业文化的层次：**
- 表层：物质文化（标识、建筑、产品外观）
- 中层：制度文化（规章制度、行为规范）
- 深层：精神文化（价值观、经营哲学、企业精神）

**文化与战略的关系：**
- 文化是战略实施的驱动力或阻力
- 战略变革需要文化变革的配合
- 文化惯性可能阻碍战略转型

### 4.3 战略控制

**战略控制的类型：**

| 控制类型 | 控制方式 | 适用情形 |
|---------|---------|---------|
| 预算控制 | 通过财务预算监控 | 成本控制、资源分配 |
| 平衡计分卡 | 财务+客户+内部流程+学习成长 | 综合绩效评价 |
| 标杆管理 | 与最佳实践对比 | 持续改进 |

**平衡计分卡四维度：**
1. **财务维度**：收入增长、成本降低、资产利用率、股东回报
2. **客户维度**：客户满意度、市场份额、客户获取/保留率
3. **内部流程维度**：运营效率、质量管理、创新流程
4. **学习与成长维度**：员工能力、信息系统、组织氛围

### 4.4 数智化转型

**数字化战略的核心要素：**
- 数据驱动决策
- 业务流程数字化
- 客户体验数字化
- 商业模式创新
- 组织与文化变革

## 第五章 公司治理

### 5.1 公司治理概述

**公司治理的定义：** 用于管理和控制公司的制度安排，涉及公司各利益相关者之间的权利、责任和利益分配。

**公司治理的核心问题：**
- 委托代理问题（所有权与经营权分离）
- 信息不对称（道德风险、逆向选择）
- 利益冲突（股东vs管理层、大股东vs小股东、股东vs债权人）

### 5.2 公司内部治理结构

| 治理机构 | 主要职责 |
|---------|---------|
| 股东大会 | 最高权力机构；选举董事/监事、审批重大事项、利润分配 |
| 董事会 | 战略决策、选聘经理层、监督经营、风险管理 |
| 监事会 | 监督董事和高管、检查财务、提议召开临时股东大会 |
| 经理层 | 日常经营管理、执行董事会决议 |

**董事会专门委员会：**
- 审计委员会（财务报告监督、内外审计协调）
- 薪酬委员会（高管薪酬方案）
- 提名委员会（董事/高管候选人）
- 战略委员会（战略规划研究）
- 风险管理委员会（风险管理策略审议）

### 5.3 公司外部治理机制

- 资本市场（股价反映经营绩效、敌意收购威胁）
- 产品市场（竞争压力迫使提高效率）
- 经理人市场（声誉机制约束管理层行为）
- 法律法规（公司法、证券法等强制性约束）
- 中介机构（审计师、律师、评级机构等外部监督）

## 第六章 风险与风险管理概述

### 6.1 风险的概念与要素

**风险定义：** 未来结果的不确定性，既包括损失的可能性，也包括收益的可能性。

**风险的构成要素：**
- 风险事件（可能发生的事项）
- 风险概率（发生的可能性）
- 风险影响（对目标的影响程度）

### 6.2 风险管理的目标与职能

**风险管理目标：**
- 确保企业战略目标的实现
- 确保经营活动的效率和效果
- 确保财务报告及管理信息的可靠性
- 确保资产安全完整
- 确保遵循法律法规

**风险管理的基本特征：**
- 全面性（覆盖所有风险类型和业务领域）
- 全员性（涉及组织各层级人员）
- 全程性（贯穿经营管理全过程）
- 系统性（各要素相互关联、协调运作）

### 6.3 风险管理理论演进

| 阶段 | 时期 | 标志性事件/成果 |
|------|------|---------------|
| 传统风险管理 | 20世纪初~70年代 | 保险管理为主；1956年《风险管理——成本控制的新时期》 |
| 现代风险管理 | 70年代~90年代末 | 1977年FCPA；1985年Treadway委员会；1992年COSO内控框架 |
| 当代全面风险管理 | 90年代末至今 | 2004年COSO-ERM；2017年COSO-ERM修订；ISO 31000 |

### 6.4 COSO企业风险管理框架演进

**ERM 2004《企业风险管理——整合框架》：**
- 定义：全面风险管理是由董事会、管理层和其他人员实施的过程
- 应用于战略制定并贯穿企业
- 旨在识别潜在事项、管理风险使其在风险容量之内
- 为目标实现提供合理保证

**ERM 2017《企业风险管理——与战略和绩效的整合》核心变化：**

| 变化要点 | 具体内容 |
|---------|---------|
| 框架结构 | 首次采用"五大要素+二十项原则"结构 |
| 风险管理含义 | 凝练为：管理风险的文化、能力和实践 |
| 风险与价值 | 不再侧重降低风险，而是侧重创造、保持和实现价值 |
| 战略整合 | 将风险管理融入所有业务流程，从战略到执行 |
| 绩效联系 | 探讨如何识别、评估影响绩效的各种风险 |
| 决策整合 | 明确纳入战略目标选择、经营目标设定、资源分配 |

**ERM 2017 五大要素：**
1. 治理与文化
2. 战略与目标设定
3. 绩效（风险识别、评估、排序、应对）
4. 审查与修订
5. 信息、沟通与报告

### 6.5 ISO 31000 风险管理标准体系

**ISO 31000:2018 三大组成：**
- **原则**：价值创造和保护
- **框架**：领导力与承诺（领导层职责）
- **流程**：风险识别 → 风险分析 → 风险评价 → 风险应对 → 风险记录和报告

**相关标准：**
- ISO/IEC 31010：风险评估技术
- ISO 31073：风险管理术语
- 中国等同标准：GB/T 24353-2022、GB/T 23694-2024

## 第七章 风险管理的流程、体系与方法

### 7.1 风险管理基本流程（中央企业全面风险管理指引）

```
收集初始信息 → 风险评估 → 制定风险管理策略 → 提出和实施解决方案 → 监督与改进
```

#### 7.1.1 收集风险管理初始信息

按风险类型分别收集：

| 风险类型 | 收集内容 |
|---------|---------|
| 战略风险 | 宏观环境、产业环境、竞争环境、内部环境 |
| 市场风险 | 市场供需、价格、竞争、经济政策 |
| 财务风险 | 财务战略、财务管理状况、财务指标数据 |
| 运营风险 | 生产运营、营销、研发、组织人员、信息系统、外部社会自然因素 |
| 法律合规风险 | 政治法律环境、内部合规因素 |

#### 7.1.2 风险评估三步骤

| 步骤 | 内容 | 方法 |
|------|------|------|
| 风险辨识 | 查找有无风险、有哪些风险 | 问卷调查、集体讨论、专家咨询、情景分析 |
| 风险分析 | 定义描述风险、分析发生条件和可能性 | 统计推论、马尔科夫分析、失效模式分析、事件树分析 |
| 风险评价 | 评估对目标的影响程度 | 风险矩阵、风险坐标图 |

#### 7.1.3 风险管理策略工具

| 策略工具 | 含义 | 适用风险类型 |
|---------|------|------------|
| 风险承担 | 在风险偏好内不采取额外措施 | 低影响/低概率风险 |
| 风险规避 | 放弃产生风险的活动 | 不可承受的风险 |
| 风险转移 | 通过保险、外包等转移风险 | 可保风险、可外包风险 |
| 风险转换 | 将一种风险转换为另一种风险 | 可对冲的风险 |
| 风险对冲 | 利用金融工具对冲风险敞口 | 市场风险、汇率风险 |
| 风险补偿 | 通过风险收益补偿承担的风险 | 战略性风险承担 |
| 风险控制 | 采取措施降低可能性或影响程度 | 运营风险、合规风险 |

**制定策略的关键环节：**
- 确定风险偏好（愿意承担哪些风险）
- 确定风险承受度（风险的最低限度和最高限度）
- 设定风险预警线
- 平衡风险与收益

#### 7.1.4 风险管理解决方案

**两种类型：**
- **外部方案**：外包给专业机构（投行、评级公司、保险公司、律所、会计师事务所）
- **内部方案**：组织职能体系 + 风险管理策略 + 金融工具 + 内控系统 + 信息系统

**关键风险指标（KRI）管理六步：**
1. 分析风险成因，找出关键成因
2. 将关键成因量化，确定度量
3. 确定导致风险事件发生的具体数值
4. 以该数值为基础设定预警指标
5. 制定预警时的风险控制措施
6. 跟踪监测，触发预警即实施控制

#### 7.1.5 风险管理的监督与改进

**监督方法：**

| 方法 | 含义 |
|------|------|
| 压力测试 | 在极端情景下分析风险管理模型/内控流程的有效性 |
| 返回测试 | 将历史数据输入模型，对比结果与预测值 |
| 穿行测试 | 将初始数据穿越全流程，对比运行结果与设计要求 |
| 风险控制自我评估（RCSA） | 定期评价自身风险管理系统的有效性 |

**监督职责分工：**
- 各部门/业务单位：定期自查 → 报送风险管理职能部门
- 风险管理职能部门：定期检查各部门实施情况 → 报送总经理
- 内部审计部门：每年至少一次监督评价 → 直接报送董事会/审计委员会

### 7.2 风险管理体系（五大系统）

| 系统 | 功能定位 |
|------|---------|
| 风险管理组织职能体系 | 治理基础和运行载体，明确各层级职责与授权 |
| 风险管理策略 | 顶层设计与决策指引，明确偏好/承受度/工具/资源 |
| 内部控制系统 | 操作载体，将策略嵌入日常经营的制度化流程 |
| 金融工具运用 | 针对内控无法消除的剩余风险提供财务保障 |
| 风险管理信息系统 | 技术基石，实现风险信息的集中、处理、传递和应用 |

#### 7.2.1 风险管理组织职能体系（三道防线）

| 防线 | 组成 | 职责 |
|------|------|------|
| 第一道 | 各职能部门和业务单位 | 执行风险管理基本流程、日常风险管理 |
| 第二道 | 风险管理职能部门+风险管理委员会 | 制定策略、协调指导、监督检查 |
| 第三道 | 内部审计部门+审计委员会 | 独立监督评价、出具审计报告 |

**董事会风险管理职责（10项）：**
1. 审议并提交全面风险管理年度工作报告
2. 确定风险管理总体目标、风险偏好、风险承受度
3. 批准风险管理策略和重大风险管理解决方案
4. 了解和掌握重大风险及管理现状
5. 批准重大决策/事件/流程的判断标准
6. 批准重大决策的风险评估报告
7. 批准内审部门的风险管理监督评价审计报告
8. 批准风险管理组织机构设置及职责方案
9. 督导风险管理文化培育
10. 批准或决定其他重大事项

#### 7.2.2 风险偏好与风险承受度

**风险偏好的决定因素：**
- 公司战略（稳健型vs进取型）
- 公司治理结构（监督方式和授权体系）
- 企业文化（保守型vs创新型）
- 风险管理能力（识别和计量能力）

**风险承受度的决定因素：**
- 战略及业务目标的重要性
- 风险偏好的整体约束
- 绩效管理的严格程度
- 组织的风险管理能力

**风险偏好表现形式：**
- 具体目标偏好（如贷款损失≤总贷款的0.5%）
- 分级范围偏好（不同目标设定不同偏好水平）
- 明确上限/下限偏好（设定不可逾越的边界）

#### 7.2.3 内部控制系统（作为风险管理体系组成部分）

**内控在风险管理体系中的定位：**
- 内控是全面风险管理的重要组成部分和基础
- 内控针对的是可控纯粹风险
- 控制对象是企业中的个人
- 控制目的是规范员工行为
- 控制范围是业务和管理流程

**内控制度体系（九项基本制度）：**
1. 内控岗位授权制度
2. 内控报告制度
3. 内控批准制度
4. 内控责任制度
5. 内控审计检查制度
6. 内控考核评价制度
7. 重大风险预警制度
8. 企业法律顾问制度
9. 重要岗位权力制衡制度（不相容职责分离）

## 第八章 企业面对的主要风险与应对

### 8.1 战略风险与应对

**战略风险来源：**
- 战略制定失误（目标不切实际、环境判断错误）
- 战略实施偏差（资源配置不当、组织能力不足）
- 外部环境剧变（技术颠覆、政策突变、竞争格局变化）

**战略风险应对：**
- 建立战略风险预警机制
- 定期进行战略回顾与调整
- 保持战略灵活性和适应性
- 加强战略执行的监控

### 8.2 市场风险与应对

**市场风险类型：**
- 产品市场风险（需求变化、竞争加剧、替代品出现）
- 要素市场风险（原材料价格波动、劳动力成本上升）
- 金融市场风险（利率风险、汇率风险、股价波动）

**市场风险应对工具：**
- 多元化经营分散风险
- 套期保值（期货、期权、远期合约）
- 灵活定价策略
- 供应链多元化

### 8.3 财务风险与应对

**财务风险分类：**

| 风险类型 | 具体表现 | 关键指标 |
|---------|---------|---------|
| 筹资风险 | 资本结构不合理、偿债能力不足 | 资产负债率、利息保障倍数 |
| 投资风险 | 投资决策失误、回报不达预期 | NPV、IRR、投资回收期 |
| 资金营运风险 | 现金流断裂、营运资金不足 | 流动比率、速动比率、现金周转期 |
| 收益分配风险 | 股利政策不当影响企业发展 | 股利支付率、留存收益率 |

**财务风险应对：**
- 优化资本结构
- 加强现金流管理
- 建立财务预警体系
- 运用金融衍生工具对冲

### 8.4 运营风险与应对

**运营风险领域：**
- 生产安全风险（设备故障、安全事故）
- 质量风险（产品缺陷、质量不达标）
- 供应链风险（供应中断、物流延误）
- 人力资源风险（关键人才流失、劳动纠纷）
- 信息系统风险（系统故障、数据泄露、网络攻击）
- 合规运营风险（违规操作、内部舞弊）

**运营风险应对：**
- 建立健全内部控制制度
- 加强流程管理和标准化
- 实施全面质量管理
- 建立业务连续性计划（BCP）
- 加强信息安全管理（参照ISO 27000系列）

### 8.5 法律风险和合规风险与应对

**法律合规风险来源：**
- 外部法律环境变化（新法规出台、执法趋严）
- 合同管理不善（条款不完善、履约纠纷）
- 知识产权风险（侵权、被侵权）
- 劳动用工风险（违法解除、社保欠缴）
- 税务合规风险（偷逃税、税务筹划失当）

**法律合规风险应对：**
- 建立企业法律顾问制度
- 加强合同全生命周期管理
- 建立合规管理体系
- 定期法律风险排查
- 加强员工法律培训

## 常见战略分析问题判断

**Q：五力模型的局限性？**
- 静态分析，未考虑行业动态变化
- 假设行业边界清晰，但实际中行业融合趋势明显
- 未充分考虑互补品的作用
- 未考虑政府和公共政策的影响

**Q：多元化战略何时适用？**
- 现有行业增长放缓或衰退
- 企业拥有可转移的核心能力
- 存在范围经济或协同效应
- 分散经营风险的需要
- 有充裕的资金和管理能力

**Q：战略与组织结构的关系？**
钱德勒命题：结构跟随战略。但实践中结构也会反过来影响战略选择（路径依赖）。战略变革时，组织结构调整往往滞后，形成"战略-结构"缺口。

**Q：风险偏好和风险承受度的区别？**
- 风险偏好：企业愿意接受的风险类型和程度（定性为主）
- 风险承受度：可接受的绩效变化界限（定量为主，作为预警指标）
- 各类风险偏好/承受度加总应≤总体风险偏好/承受度

**Q：内控与全面风险管理的关系？**
- 内控是全面风险管理的重要组成部分和基础
- 内控针对可控纯粹风险，全面风险管理覆盖所有风险
- 内控侧重流程控制，全面风险管理还包括战略层面
- 两者相辅相成，不可替代
