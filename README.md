# 古立特 Gridman — Hyper Agent for Chinese Finance & Tax

古立特（Gridman），财税领域 AI 助手。一个入口覆盖中国会计准则（CAS）、税法、审计、投行估值、内控、内审、ESG、政府会计、财务造假识别、经济法、财务BP等全领域，**35 个专业知识模块 + 45 个 MCP 计算工具 + 16 个端到端工作流**。

从底稿编制、分录判断、准则应用，到税务筹划、报表勾稽、财报研究、IPO尽调——处理中国财税场景时的专业助手。

## 快速开始

**第一步：放置知识层（零配置）**

把 `gridman-skill（公开版）/` 文件夹放到你的 AI Agent 能读到的位置（建议重命名去掉"（公开版）"后缀）：

| Agent | 放置位置 |
|-------|---------|
| Kiro | `.kiro/skills/gridman/` |
| Cursor / Claude Code / Codex | 项目根目录 |
| Windsurf / Cline | 项目根目录 |

放好后问任何财税问题，古立特就能回答了。

**第二步：安装工具层（一句话搞定，可选）**

对 AI 说一句：**"装古立特"**

AI 自动读取 INSTALL.md，完成 uv 检测 → whl 安装 → MCP 配置写入 → 提示重启。

重启后看到 `gridman Connected (45 tools)` 即成功（默认全开 45 个；按职业精简见 `INSTALL.md`）。

详细说明见 [`gridman-skill（公开版）/使用说明.txt`](gridman-skill（公开版）/使用说明.txt)

## MCP 工具一览（45 个，按 6 类分组）

| 分组 | 工具 |
|------|------|
| 数据处理（8） | 科目余额表处理 · 地址拆分 · 名称模糊匹配 · 数据概览 / 分组 / 筛选 / 透视 · 图表生成 |
| 文档处理（6） | OCR 识别 · 批量 OCR · PDF 合并 / 拆分 / 提取 · 凭证 PDF 自动拆分 |
| 底稿编制（13） | 银行调节表 · 账龄分析 · 本福特检验 · 折旧重算 · 审计抽样 · 审计调整三栏 TB · 截止测试 · 函证生成 · 底稿初始化 · 重要性计算 · 分析性程序 · 往来重分类 · 凭证巡检 |
| 财务分析（5） | 现金流量表（间接法测算）· 现金流量表（直接法）· 非经常性损益 · 财务比率 · 纳税调整 |
| 市场数据（4） | 多市场实时行情（A 股 / 港股 / 美股 / 指数）· 历史 K 线 · A 股财务报表 · 公告 / 年报下载 |
