# 证监会处罚先例与合规红线

> 定位：基于约 22,213 份证监会处罚文书（2016 至 2026 年 5 月），提供合规风险评价的先例参照
> 来源：证监会行政处罚决定、行政监管措施、市场禁入决定
> 性质：实操 + 判断

---

## 一、审计处罚四维缺陷模型

会计师事务所被处罚的审计程序缺陷归纳为四个维度：

### 维度 A：函证程序缺陷（约占 35%）

| 编号 | 缺陷 | 违反准则 | 后果 |
|------|------|---------|------|
| A1 | 未对全部银行账户函证（含零余额/已注销） | CSA 1312 第17条 | 警示函 |
| A2 | 未回函项目未实施替代程序 | CSA 1312 第21条 | 警示函 |
| A3 | 未对询证函保持控制（回函由被审计单位转交） | CSA 1312 第17-18条 | 警示函/行政处罚 |
| A4 | 未对回函异常保持警觉（地址不一致/印章异常） | CSA 1312 第19条 + CSA 1141 第13条 | 警示函/行政处罚 |

### 维度 B：审计证据缺陷（约占 25%）

| 编号 | 缺陷 | 违反准则 | 案例 |
|------|------|---------|------|
| B1 | 未获取充分适当的审计证据 | CSA 1301 第10、15条 | 大华所〔2024〕313号 |
| B2 | 异常交易未追查至底层证据 | CSA 1301 第15条 + CSA 1141 第13-14条 | 普华永道恒大案 |
| B3 | 依赖管理层声明替代审计程序 | CSA 1301 第10、15条 | 多案——管理层声明不能替代实质性程序 |
| B4 | 收入确认舞弊风险应对不足 | CSA 1141 第14条 | 普华永道恒大案 |

### 维度 C：职业判断缺陷（约占 20%）

| 编号 | 缺陷 | 核心要点 |
|------|------|---------|
| C1 | 未对重大异常形成恰当职业判断 | 毛利率异常/收入与产能不匹配/期末集中确认 |
| C2 | 未保持职业怀疑 | "已执行常规程序"不是免责理由 |
| C3 | 会计估计合理性未充分评估 | 坏账/跌价/商誉减值 |
| C4 | 持续经营能力评估不充分 | CSA 1324 |

### 维度 D：底稿与质控缺陷（约占 20%）

| 编号 | 缺陷 | 严重程度 |
|------|------|---------|
| D1 | 底稿不完整/记录与执行不一致 | 叠加性——强化其他维度处罚 |
| D2 | 质控复核流于形式 | 苏亚金诚案 |
| D3 | 伪造/篡改/毁损底稿 | **最严重**——必然行政处罚+暂停业务（天职国际〔2024〕78号） |
| D4 | 档案管理不合规 | 轻微 |

### 处罚升级路径

```
单维度缺陷（较轻）→ 警示函
多维度缺陷（一般）→ 警示函或责令改正
严重缺陷+故意行为 → 行政处罚+暂停业务+CPA罚款
连续审计失败（多年未发现造假）→ 高额罚款+暂停业务+市场禁入
```

## 三、合规红线启发式（8 条）

### H1：实质重于形式
交易经济实质与法律形式不一致，且影响信披真实性 → 已触发红线。
先例：江苏舜天〔2024〕10号——合同齐全、资金流水完整，仍认定虚增收入。

### H2：全部银行账户函证
未对全部银行账户（含零余额/已注销）函证 → 已触发最低程序红线。

### H3：关联方资金占用标尺
占净资产 >5% 且未披露 → 监管关注显著提高。>30% 且有主观故意 → 从严。
先例：金洲慈航〔2024〕1号——51.15%，终身禁入。

### H4：持续造假年限
3年以上持续造假 → 近年案例均严惩。
先例：江苏舜天13年、美尚生态9年。

### H5：收入造假"无商业实质"判定
上下游同一方控制 + 资金回流 + 不承担风险 + 缺乏商业合理性 → 触发红线。

### H6：审计"未勤勉尽责"最低门槛
"签字但未关注到明显异常"即构成未勤勉尽责。不需证明"参与造假"。
先例：钱仁勇〔2024〕6号——"不知悉、未参与"被驳回。

### H7：主动纠正的正面作用
立案调查前主动更正/追回资金/补缴税款 → 正面因素。立案后效果有限。

### H8：形式合规≠实质合规
"走了流程"（董事会签字、审批）不构成免责。必须有实质性独立判断证据。
先例：恒大〔2024〕49号——"消极不履责同样是未勤勉尽责"。

## 五、财务造假典型手法库

| 手法 | 典型案例 | 识别信号 |
|------|---------|---------|
| 提前确认收入 | 恒大〔2024〕49号 | 期末集中确认、收入与交付不匹配 |
| 无商业实质自循环 | 江苏舜天〔2024〕10号 | 上下游同一控制、资金闭环、不承担风险 |
| 虚构交易+资金闭环 | 东方金钰 | 壳公司、多层中转、资金回流 |
| 隐瞒担保/资金占用 | 三盛教育〔2023〕11号 | 未履行决策程序、占净资产比例高 |
| 伪造单据 | 华锐风电〔2015〕66号 | 签署单据与实物交付不匹配 |
| 虚增服务收入 | 金洲慈航〔2024〕1号 | 无实际服务内容、资金占用 |
| IPO 阶段造假 | 思尔芯、紫晶存储 | 上市前虚增、申报期现场检查暴露 |
| 跨期调节虚构业绩 | 金通灵 | 收入费用跨期、关联方配合、连续多年 |

**通用识别信号：**
- 收入增长与产能/人员/现金流不匹配
- 毛利率显著高于行业且不可解释
- 应收增速远超收入增速
- 关联交易占比高且对手集中
- 频繁变更会计估计或政策
- 期末集中确认大量收入

## 六、问询函编制规范

### 双角色架构

| 角色 | 职责 | 视角 |
|------|------|------|
| 审核专家（主导） | 准则适用性、信披完整性、内控有效性、会计估计合理性 | 以准则为标尺，关注"是否合规" |
| 舞弊专员（协同） | 舞弊动机（压力/机会/合理化）、收入操纵、隐性关联交易 | 以舞弊三角+处罚先例为参照，关注"是否存在异常信号" |

### 问询问题标准结构

```
【核查发现】
[公司自身财务数据异常，用表格或逐条列举]

【问询事项】
（1）请说明/论证/量化/提供……（底稿要求：①……；②……）
（2）请说明/论证/量化/提供……（底稿要求：③……；④……）

【准则/法规依据】（如适用）
【参照先例】（如适用）
```

### 核心原则

1. **基于公司自身数据异常发问，不做同业直接对比**
2. **解释责任交予企业，不前置定性结论**
3. **每个问题附带底稿要求**

### 禁止行为

- 不以同业数字对比作为问询逻辑起点
- 不在问题中预设结论（如"你公司通过XX操纵利润"）
- 不要求企业自证"未违规"——应要求"说明业务逻辑和会计处理依据"
- 不编造底稿编号或数据来源
