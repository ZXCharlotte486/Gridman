## 第十八章 政府补助

### 一、政府补助的分类

| 类别 | 定义 | 处理方式 |
|------|------|----------|
| 与资产相关 | 用于购建或形成长期资产 | 在资产使用寿命内分期计入损益 |
| 与收益相关 | 除与资产相关之外的补助 | 补偿已发生费用→直接计入损益；补偿以后期间费用→先确认递延收益 |

### 二、两种处理方法

| 方法 | 核心思路 | 适用 |
|------|----------|------|
| 总额法 | 将政府补助全额确认为收益（其他收益/营业外收入） | 一般情形 |
| 净额法 | 将政府补助冲减相关资产账面价值或成本费用 | 一般情形 |

同类或类似业务只能选用一种方法，且应一贯运用。

### 三、与资产相关的政府补助——总额法

收到补助：
```
借：银行存款
  贷：递延收益
```

在资产使用寿命内分期结转（与日常活动相关）：
```
借：递延收益
  贷：其他收益
```

资产提前处置时转销递延收益余额：
```
借：递延收益（尚未分摊余额）
  贷：营业外收入（或其他收益）
```

### 四、与资产相关的政府补助——净额法

收到补助后冲减资产账面价值：
```
借：银行存款
  贷：递延收益

借：递延收益
  贷：固定资产/无形资产
```

后续按冲减后的账面价值计提折旧/摊销。

### 五、与收益相关的政府补助——补偿以后期间费用

总额法：
```
收到时：
借：银行存款
  贷：递延收益

实际发生费用时结转：
借：递延收益
  贷：其他收益
```

净额法：
```
收到时：
借：银行存款
  贷：递延收益

实际发生费用时冲减：
借：递延收益
  贷：管理费用/研发费用等
```

### 六、与收益相关的政府补助——补偿已发生费用

总额法（与日常活动相关）：
```
借：银行存款
  贷：其他收益
```

总额法（与日常活动无关）：
```
借：银行存款
  贷：营业外收入
```

净额法：
```
借：银行存款
  贷：管理费用/营业外支出等
```

### 七、政府补助的退回

```
存在递延收益余额的：
借：递延收益（余额部分）
    其他收益/营业外支出（超出部分）
  贷：银行存款

净额法下退回（调整资产账面价值）：
借：固定资产/无形资产
  贷：银行存款
```

## 第十九章 所得税

### 一、基本原理——资产负债表债务法

采用**资产负债表债务法**核算所得税，从资产负债表出发，通过比较资产/负债的账面价值与计税基础，确定暂时性差异，确认递延所得税资产/负债，最终确定所得税费用。

**核算程序（5步）**：

1. 确定资产负债表中各项资产、负债的账面价值
2. 按照税法规定确定各项资产、负债的计税基础
3. 比较账面价值与计税基础，确定暂时性差异（应纳税/可抵扣），确定期末递延所得税资产/负债应有金额
4. 计算当期应纳税所得额和当期应交所得税（当期所得税）
5. 确定所得税费用 = 当期所得税 + 递延所得税

### 二、资产的计税基础

**资产计税基础** = 未来期间按税法规定可以税前扣除的金额

| 资产类型 | 计税基础确定规则 |
|----------|-----------------|
| 固定资产 | 取得成本 - 按税法规定已税前扣除的累计折旧（注意折旧方法、年限与会计可能不同） |
| 无形资产（外购） | 取得成本 - 按税法规定已税前扣除的累计摊销 |
| 无形资产（内部研发） | 按税法规定的200%摊销基础确定（加计扣除） |
| 使用寿命不确定的无形资产 | 会计不摊销，税法按规定年限摊销→计税基础 = 成本 - 税法累计摊销 |
| 以公允价值计量且变动计入损益的金融资产 | 取得成本（持有期间公允价值变动税法不认） |
| 以公允价值计量且变动计入OCI的金融资产 | 取得成本（同上） |
| 投资性房地产（公允价值模式） | 按固定资产/无形资产规则确定（成本 - 税法折旧/摊销） |
| 计提减值准备的资产 | 计税基础不因减值准备而变化（减值准备实质性损失前不允许税前扣除） |


### 三、负债的计税基础

**负债计税基础** = 账面价值 - 未来期间按税法规定可予税前扣除的金额

| 负债类型 | 计税基础 | 是否形成暂时性差异 |
|----------|----------|-------------------|
| 预计负债（产品保修等） | 0（未来实际发生时可税前扣除） | 是，可抵扣暂时性差异 |
| 合同负债（一般情况） | 账面价值（税法与会计确认收入时点一致） | 否 |
| 合同负债（税法要求提前纳税） | 0（已纳税，未来可全额扣除） | 是，可抵扣暂时性差异 |
| 应付职工薪酬（超标准部分） | 账面价值（超标部分当期和未来均不可扣除） | 否（永久性差异） |
| 罚款/滞纳金 | 账面价值（当期和未来均不可税前扣除） | 否（永久性差异） |

### 四、暂时性差异判断规则

| 情形 | 差异类型 | 确认 |
|------|----------|------|
| 资产账面价值 > 计税基础 | 应纳税暂时性差异 | 递延所得税负债 |
| 资产账面价值 < 计税基础 | 可抵扣暂时性差异 | 递延所得税资产 |
| 负债账面价值 > 计税基础 | 可抵扣暂时性差异 | 递延所得税资产 |
| 负债账面价值 < 计税基础 | 应纳税暂时性差异 | 递延所得税负债 |

**特殊项目产生的暂时性差异**：
- 未作为资产确认但有计税基础的项目（如超标广告费可结转扣除）：账面价值视为0，计税基础为可结转金额→可抵扣暂时性差异
- 可抵扣亏损和税款抵减：视同可抵扣暂时性差异处理


### 五、不确认递延所得税负债的三种情况

1. **商誉的初始确认**：非同一控制下企业合并中，免税合并条件下商誉计税基础为0，产生的应纳税暂时性差异不确认递延所得税负债（避免循环确认导致商誉金额增加）
2. **非企业合并中既不影响会计利润也不影响应纳税所得额的交易**：如内部研发形成无形资产的初始确认差异
3. **与子公司/联营/合营投资相关的应纳税暂时性差异**：同时满足两个条件时不确认——①投资企业能够控制暂时性差异转回的时间；②该暂时性差异在可预见的未来很可能不会转回

> 注意：对于权益法核算的长期股权投资，拟长期持有时一般不确认所得税影响；改变持有意图拟出售时，应确认相关递延所得税。

### 六、不确认递延所得税资产的情况

- 非企业合并中，交易发生时既不影响会计利润也不影响应纳税所得额，且产生的资产/负债初始确认金额与计税基础不同的（如内部研发无形资产）
- 未来期间无法产生足够应纳税所得额用以利用可抵扣暂时性差异的
- 与子公司/联营/合营投资相关的可抵扣暂时性差异，不满足"很可能转回+未来很可能获得足够应纳税所得额"两个条件的

### 七、递延所得税的计量

- **适用税率**：按照预期收回资产或清偿负债期间的适用税率计量（非当期税率）
- **不折现**：无论暂时性差异何时转回，递延所得税资产/负债均不要求折现
- **复核**：资产负债表日应复核递延所得税资产账面价值，未来无法取得足够应纳税所得额的应减记
- **税率变化**：适用税率变化时，应按新税率重新计量已确认的递延所得税资产/负债，调整金额计入当期所得税费用（直接计入所有者权益的除外）


### 八、所得税费用的计算

**公式**：
```
所得税费用 = 当期所得税 + 递延所得税

当期所得税 = 应纳税所得额 × 适用税率

应纳税所得额 = 会计利润
  + 按会计计入损益但税法不允许扣除的费用（如超标业务招待费、罚款）
  ± 计入损益的费用与税法可扣除金额的差额（如折旧差异、减值准备）
  ± 计入损益的收入与税法应税收入的差额（如公允价值变动）
  - 税法规定的不征税收入/免税收入
  ± 其他调整（如加计扣除）

递延所得税 = (期末递延所得税负债 - 期初递延所得税负债)
           - (期末递延所得税资产 - 期初递延所得税资产)
```

> 注意：递延所得税不包括计入所有者权益的交易或事项的所得税影响，也不包括企业合并中确认的递延所得税。

### 九、所得税费用确认分录

**综合分录**：
```
借：所得税费用
    递延所得税资产（本期增加额）
  贷：应交税费——应交所得税
      递延所得税负债（本期增加额）
```

**转回递延所得税负债**：
```
借：递延所得税负债
  贷：所得税费用
```

**转回递延所得税资产**：
```
借：所得税费用
  贷：递延所得税资产
```


### 十、与直接计入所有者权益的交易相关的递延所得税

与当期及以前期间直接计入所有者权益的交易或事项相关的当期所得税及递延所得税应当计入所有者权益，不计入所得税费用。

主要情形：
- 会计政策变更追溯调整/前期差错追溯重述→调整留存收益
- 以公允价值计量且变动计入OCI的金融资产公允价值变动
- 同时包含负债及权益成分的复合金融工具初始确认时计入权益的部分
- 自用房地产转为公允价值模式投资性房地产时公允价值大于账面价值的差额

```
确认与OCI相关的递延所得税负债：
借：其他综合收益
  贷：递延所得税负债

确认与权益工具相关的递延所得税负债（如可转债权益成分）：
借：其他权益工具
  贷：递延所得税负债
```

### 十一、与企业合并相关的递延所得税

- 非同一控制下企业合并中，购买方取得的可辨认资产/负债按公允价值确认，与计税基础的差异应确认递延所得税，影响商誉金额
- 购买日不符合确认条件的递延所得税资产（如被购买方未弥补亏损），购买日后12个月内取得新信息表明购买日已存在相关情况的→确认递延所得税资产，同时减少商誉（商誉不足冲减的计入当期损益）
- 购买日后12个月之外确认的→计入当期损益

```
购买日后12个月内确认（购买日已存在）：
借：递延所得税资产
  贷：商誉（先冲减）
      所得税费用（商誉不足部分）

购买日后12个月外确认（或新事实）：
借：递延所得税资产
  贷：所得税费用
```


### 十二、与股份支付相关的递延所得税

- 税法规定股份支付相关支出在可行权时方可税前扣除，可扣除金额 = 行权时股票公允价值 - 行权价格
- 等待期内，企业应根据期末股价估计未来可税前扣除金额，确认递延所得税资产
- 若预计未来可扣除金额 > 等待期内累计确认的成本费用，超出部分形成的递延所得税资产计入**资本公积**（而非所得税费用）

```
等待期内确认递延所得税资产（未超过累计费用部分）：
借：递延所得税资产
  贷：所得税费用

超出累计费用部分：
借：递延所得税资产
  贷：所得税费用（累计费用对应部分）
      资本公积——其他资本公积（超出部分）
```

### 十三、单项交易的递延所得税（租赁/弃置义务）

对于不是企业合并、交易发生时既不影响会计利润也不影响应纳税所得额，且初始确认的资产和负债导致产生**等额**应纳税暂时性差异和可抵扣暂时性差异的单项交易，**不适用豁免规定**，应在交易发生时分别确认递延所得税负债和递延所得税资产。

典型场景：
- 承租人初始确认使用权资产和租赁负债
- 因固定资产弃置义务确认预计负债并计入资产成本

```
租赁期开始日确认递延所得税：
借：递延所得税资产（租赁负债产生的可抵扣暂时性差异 × 税率）
    所得税费用（初始直接费用相关差额，如有）
  贷：递延所得税负债（使用权资产产生的应纳税暂时性差异 × 税率）

后续期间——租赁负债减少（可抵扣暂时性差异减少）：
借：所得税费用
  贷：递延所得税资产

后续期间——使用权资产折旧（应纳税暂时性差异减少）：
借：递延所得税负债
  贷：所得税费用
```

### 十四、发行方分类为权益工具的金融工具相关股利的所得税影响

对于分类为权益工具的金融工具（如永续债），相关股利支出可税前扣除的，企业应在确认应付股利时确认所得税影响：
- 所分配利润来源于以前产生损益的交易→股利的所得税影响计入当期损益
- 所分配利润来源于以前确认在所有者权益中的交易→股利的所得税影响计入所有者权益

## 第二十章 非货币性资产交换

### 一、认定标准（25%比例）

| 角度 | 公式 | 结论 |
|------|------|------|
| 收到补价方 | 收到的补价 ÷ 换出资产公允价值 < 25% | 非货币性资产交换 |
| 支付补价方 | 支付的补价 ÷（换出资产公允价值 + 支付的补价）< 25% | 非货币性资产交换 |

> 比例 ≥ 25% 的，不视为非货币性资产交换，适用收入等相关准则。


### 二、商业实质的判断

满足下列条件**之一**的，视为具有商业实质：

1. **换入资产的未来现金流量在风险、时间分布或金额方面与换出资产显著不同**（三个方面任一显著不同即可）
2. **使用换入资产所产生的预计未来现金流量现值与继续使用换出资产不同，且差额与资产公允价值相比是重大的**（体现资产对企业自身的特定价值）

判断要点：
- 不同类资产交换（如固定资产换长期股权投资）通常具有商业实质
- 同类资产交换需具体分析
- 应遵循实质重于形式原则

### 三、关联方交易注意事项

- 非货币性资产交换的一方直接或间接对另一方持股且以股东身份进行交易→构成**权益性交易**，不确认损益
- 双方均受同一方或相同多方最终控制，且交易实质是权益性分配或权益性投入→适用权益性交易处理
- 集团重组中的无代价或明显不公平代价的资产划拨/划转→权益性交易
- 企业应遵循实质重于形式原则判断是否构成权益性交易

### 四、以公允价值为基础计量

**适用条件**：同时满足①具有商业实质 + ②换入或换出资产公允价值能可靠计量

**换入资产成本公式**：

| 情形 | 换入资产成本 |
|------|-------------|
| 以换出资产公允价值为基础（一般情况） | 换出资产公允价值 + 应支付的相关税费 ± 补价 |
| 以换入资产公允价值为基础（有确凿证据更可靠） | 换入资产公允价值 + 应支付的相关税费 |

**损益确认**：
- 换出资产公允价值与账面价值的差额→计入当期损益
- 换出固定资产/无形资产/在建工程→资产处置损益
- 换出长期股权投资→投资收益
- 换出投资性房地产→其他业务收入（公允价值）/其他业务成本（账面价值）


### 五、以账面价值为基础计量

**适用条件**：不具有商业实质，或公允价值均不能可靠计量

**换入资产成本公式**：

| 角色 | 换入资产成本 |
|------|-------------|
| 支付补价方 | 换出资产账面价值 + 支付补价的账面价值 + 应支付的相关税费 |
| 收到补价方 | 换出资产账面价值 - 收到补价的公允价值 + 应支付的相关税费 |
| 不涉及补价 | 换出资产账面价值 + 应支付的相关税费 |

**不确认任何损益**。

### 六、涉及补价的处理（以公允价值为基础）

**支付补价方**：
- 换入资产成本 = 换出资产公允价值 + 支付补价的公允价值 + 应支付的相关税费
- 损益 = 换出资产公允价值 - 换出资产账面价值

**收到补价方**：
- 换入资产成本 = 换出资产公允价值 - 收到补价的公允价值 + 应支付的相关税费
- 损益 = 换出资产公允价值 - 换出资产账面价值

### 七、涉及多项资产交换的分配方法

**以公允价值为基础**：按各项换入资产的公允价值占换入资产公允价值总额的相对比例，分摊换入资产总成本

**以账面价值为基础**：按各项换入资产的公允价值相对比例（公允价值不可靠的，按原账面价值相对比例或其他合理比例）分摊

### 八、核心会计分录

**换出固定资产（以公允价值为基础）**：
```
借：固定资产清理
    累计折旧
    固定资产减值准备
  贷：固定资产（换出）

借：固定资产/无形资产等（换入，公允价值）
    应交税费——应交增值税（进项税额）
    资产处置损益（公允价值 < 账面价值时）
  贷：固定资产清理
      应交税费——应交增值税（销项税额）
      银行存款（支付补价时）
      资产处置损益（公允价值 > 账面价值时）
```

**换出存货（视同销售，适用收入准则）**：
```
借：固定资产等（换入资产）
    应交税费——应交增值税（进项税额）
  贷：主营业务收入（存货公允价值）
      应交税费——应交增值税（销项税额）

借：主营业务成本
  贷：库存商品
```

**换出长期股权投资**：
```
借：固定资产等（换入资产）
    应交税费——应交增值税（进项税额）
  贷：长期股权投资（账面价值）
      应交税费——应交增值税（销项税额）
      投资收益（差额）
```

**以账面价值为基础（不确认损益）**：
```
借：固定资产等（换入）= 换出资产账面价值 ± 补价 + 相关税费
    累计折旧
  贷：固定资产（换出）
      银行存款（支付补价时，或借方收到补价时）
```

## 第二十一章 债务重组

### 一、定义和前提条件

**定义**：在不改变交易对手方的情况下，经债权人和债务人协定或法院裁定，就清偿债务的时间、金额或方式等重新达成协议的交易。

**要点**：
- 不强调债务人发生财务困难，不论债权人是否作出让步
- 涉及的债权和债务是金融工具准则规范的（不含合同资产/负债/预计负债，但含租赁应收/应付款）
- 经法院裁定进行债务重整并按持续经营核算的适用本章；破产清算期间的不适用
- 通过债务重组形成企业合并的，适用企业合并准则

### 二、四种方式

1. **以资产清偿债务**：债务人转让资产给债权人以清偿债务
2. **将债务转为权益工具**：权益工具指按金融工具列报准则分类为"权益工具"的金融工具
3. **修改其他条款**：调整债务本金、改变利息、变更还款期限等
4. **组合方式**：上述方式中一种以上的组合

### 三、权益性交易判断（不确认损益的情况）

构成权益性交易的情形：
1. 债权人直接或间接对债务人持股（或反之），且持股方**以股东身份**进行债务重组
2. 债权人与债务人在债务重组前后均受同一方或相同多方最终控制，且交易实质是权益性分配或权益性投入

> 不属于权益性交易的部分仍应确认债务重组相关损益。如股东比其他债权人多豁免的部分为权益性交易，与其他债权人同等豁免的部分确认损益。

### 四、以资产清偿债务

#### 债务人的会计处理

**以金融资产清偿**：
```
借：应付账款/长期借款（债务账面价值）
  贷：交易性金融资产/债权投资等（金融资产账面价值）
      投资收益（差额）
```

**以非金融资产清偿（固定资产/无形资产/存货等）**：
```
借：应付账款/长期借款（债务账面价值）
    累计折旧/累计摊销
    固定资产减值准备/无形资产减值准备
  贷：固定资产/无形资产/库存商品
      应交税费——应交增值税（销项税额）
      其他收益——债务重组收益（差额）
```

> 注意：以非金融资产清偿债务时，不区分资产处置损益和债务重组损益，统一记入"其他收益——债务重组收益"。以存货清偿债务不确认收入，不视同销售。

**以长期股权投资清偿**：
```
借：应付账款/长期借款（债务账面价值）
  贷：长期股权投资（账面价值）
      投资收益（差额）
```


#### 债权人的会计处理

**受让金融资产**：按公允价值初始计量，金融资产确认金额与债权终止确认日账面价值的差额记入"投资收益"。

**受让非金融资产**：以放弃债权的公允价值 + 可直接归属的相关税费作为受让资产成本。

```
借：固定资产/无形资产/周转材料等（放弃债权公允价值 + 相关税费）
    应交税费——应交增值税（进项税额）
    坏账准备
    投资收益（差额，借方为损失/贷方为收益）
  贷：应收账款/贷款（账面余额）
      银行存款（相关税费）
```

**受让存货**：
```
借：库存商品/原材料（放弃债权公允价值 + 相关成本）
    应交税费——应交增值税（进项税额）
    坏账准备
    投资收益（差额）
  贷：应收账款（账面余额）
```

### 五、将债务转为权益工具

**债务人**：
```
借：应付账款/长期借款（债务账面价值）
  贷：股本/实收资本（面值）
      资本公积——股本溢价/资本溢价（权益工具公允价值 - 面值 - 发行费用）
      银行存款（发行费用）
      投资收益（债务账面价值 - 权益工具确认金额）
```

> 权益工具按公允价值计量；公允价值不能可靠计量的，按所清偿债务的公允价值计量。发行权益工具的相关税费依次冲减资本公积、盈余公积、未分配利润。

**债权人**（转为对联营/合营企业投资）：
```
借：长期股权投资（放弃债权公允价值 + 相关税费）
    坏账准备
    投资收益（差额）
  贷：应收账款/贷款（账面余额）
      银行存款（相关税费）
```

**债权人**（转为金融资产）：
```
借：交易性金融资产/其他权益工具投资（公允价值）
    坏账准备
    投资收益（差额）
  贷：应收账款/贷款（账面余额）
```


### 六、修改其他条款

**10%实质性修改判断标准**：重组债务未来现金流量现值与原债务剩余期间现金流量现值之间的差异超过10%（均按原实际利率折现）→实质性修改→终止确认原债务

#### 终止确认（实质性修改）

**债务人**：
```
借：应付账款/长期借款（原债务账面价值）
  贷：长期借款/应付账款（重组债务公允价值）
      投资收益（差额）
```

**债权人**：
```
借：债权投资/贷款（新金融资产公允价值）
    坏账准备
    投资收益（差额）
  贷：应收账款/贷款（原债权账面余额）
```

#### 不终止确认（非实质性修改）

**债务人**：重新计算重组债务账面价值（将修改后的合同现金流量按原实际利率折现），差额记入"投资收益"，后续按摊余成本计量。

**债权人**：重新计算重组债权账面余额（将修改后的合同现金流量按原实际利率折现），差额记入"投资收益"，后续按原分类继续计量。

### 七、组合方式

- 债权人：一般认为对全部债权作出实质性修改→终止确认全部债权，按修改后条款以公允价值确认新金融资产和受让资产
- 债务人：分别处理各部分——以资产清偿的部分终止确认该部分债务；转为权益工具的部分按公允价值确认权益工具；修改其他条款的部分判断是否终止确认

**债务人组合方式分录**：
```
借：应付账款/长期借款（原债务全部账面价值）
    累计折旧（如涉及固定资产）
  贷：固定资产/库存商品等（转让资产账面价值）
      应交税费——应交增值税（销项税额）
      股本/实收资本
      资本公积——股本溢价
      长期借款（修改条款后的重组债务确认金额）
      其他收益——债务重组收益/投资收益（差额）
```

### 八、破产重整的特殊处理

- 涉及破产重整的债务重组协议执行过程及结果存在重大不确定性
- 企业通常应在**破产重整协议履行完毕后**确认债务重组收益
- 除非有确凿证据表明重大不确定性已经消除
- 债务人只有在债务的现时义务解除时才能终止确认相关债务

## 第二十二章 外币折算

### 一、记账本位币确定的三个因素

1. **收入角度**：该货币对商品和劳务销售价格起主要作用，通常以该货币计价和结算
2. **支出角度**：该货币对人工、材料和其他费用产生主要影响，通常以该货币计价和结算
3. **融资活动**：融资活动获得的资金以及保存从经营活动中收取款项时所使用的货币（前两项难以确定时的参考因素）

> 记账本位币一经确定不得随意变更，除非经营所处的主要经济环境发生重大变化。

### 二、外币交易初始确认规则

- 采用**交易日的即期汇率**（中间价）或**即期汇率的近似汇率**（当期平均汇率/加权平均汇率）折算
- 近似汇率的方法应前后各期保持一致
- 汇率波动使得近似汇率不适当时，应采用交易发生日即期汇率

```
借：固定资产/库存商品等（外币金额 × 交易日即期汇率）
    应交税费——应交增值税（进项税额）
  贷：应付账款——外币（外币金额 × 交易日即期汇率）
```

### 三、收到外币投资的特殊规则

**必须用交易日即期汇率**折算，不得使用合同约定汇率或近似汇率。这样外币投入资本与相应货币性项目的记账本位币金额相等，不产生外币资本折算差额。

```
借：银行存款——外币（外币金额 × 交易日即期汇率）
  贷：实收资本/股本（外币金额 × 交易日即期汇率）
```

### 四、期末货币性项目调整规则

货币性项目（现金、银行存款、应收账款、应付账款、长期借款、应付债券等）期末以**资产负债表日即期汇率**折算，产生的汇兑差额计入当期损益（财务费用）。

```
汇兑损失（外币资产贬值或外币负债升值）：
借：财务费用——汇兑差额
  贷：银行存款——外币/应收账款——外币
      或
借：应付账款——外币/长期借款——外币
  贷：财务费用——汇兑差额（负债减少为收益）

汇兑收益（外币资产升值或外币负债贬值）：
借：银行存款——外币/应收账款——外币
  贷：财务费用——汇兑差额
      或
借：财务费用——汇兑差额（负债增加为损失）
  贷：应付账款——外币/长期借款——外币
```


### 五、期末非货币性项目处理规则

| 计量属性 | 折算规则 | 汇兑差额处理 |
|----------|----------|-------------|
| 历史成本计量 | 维持交易日即期汇率折算金额不变 | 不产生汇兑差额 |
| 公允价值计量且变动计入当期损益 | 公允价值确定日即期汇率折算 | 差额计入公允价值变动损益（含汇率和价格变动） |
| 公允价值计量且变动计入OCI（指定为FVOCI的非交易性权益工具） | 公允价值确定日即期汇率折算 | 差额计入其他综合收益（含汇率和价格变动） |
| 成本与可变现净值孰低（存货） | 可变现净值按资产负债表日即期汇率折算为记账本位币，再与记账本位币成本比较 | 差额计入资产减值损失 |

### 六、外币专门借款汇兑差额资本化

企业为购建或生产符合资本化条件的资产而借入的外币专门借款，在**借款费用资本化期间内**：
- 本金的汇兑差额→计入在建工程（资本化）
- 利息的汇兑差额→计入在建工程（资本化）

```
本金汇兑差额（汇率下降，负债减少）：
借：长期借款——外币
  贷：在建工程

本金汇兑差额（汇率上升，负债增加）：
借：在建工程
  贷：长期借款——外币

利息计提：
借：在建工程（外币利息 × 资产负债表日即期汇率）
  贷：应付利息——外币

利息支付时的汇兑差额：
借：应付利息——外币（原账面价值）
    在建工程（汇兑差额）
  贷：银行存款（实际支付的记账本位币金额）
```

### 七、外币财务报表折算规则

| 项目 | 折算汇率 |
|------|----------|
| 资产和负债 | 资产负债表日即期汇率 |
| 所有者权益——实收资本/股本 | 发生时即期汇率 |
| 所有者权益——资本公积 | 发生时即期汇率 |
| 所有者权益——盈余公积 | 计提时的平均汇率（累计） |
| 所有者权益——未分配利润 | 倒挤（由利润表折算结果推算） |
| 收入和费用 | 交易发生日即期汇率或近似汇率（当期平均汇率） |
| 外币报表折算差额 | 资产折算合计 - 负债折算合计 - 所有者权益各项折算合计的差额 |

折算差额在资产负债表"其他综合收益"项目列示。

合并报表中：折算差额按母公司和少数股东各自享有的份额分摊。

### 八、处置境外经营时折算差额的处理

- **全部处置**：将与该境外经营相关的外币报表折算差额全部从其他综合收益转入处置当期损益
- **部分处置**：按处置比例计算处置部分的折算差额，转入处置当期损益

```
处置境外经营：
借：其他综合收益——外币报表折算差额
  贷：投资收益

部分处置（按比例）：
借：其他综合收益——外币报表折算差额（处置比例部分）
  贷：投资收益
```

## 第二十三章 财务报告

### 一、财务报表的构成（五表一注）

资产负债表 + 利润表 + 现金流量表 + 所有者权益变动表 + 附注

分类：
- 按编报期间：中期财务报表（月报/季报/半年报）、年度财务报表
- 按编报主体：个别财务报表、合并财务报表

### 二、列报基本要求

| 要求 | 核心内容 |
|------|----------|
| 持续经营 | 以持续经营为基础编制；评估期间≥资产负债表日起12个月；非持续经营采用其他基础（如破产清算净值） |
| 权责发生制 | 除现金流量表按收付实现制外，其他报表均按权责发生制 |
| 一致性 | 列报项目名称、分类、排列顺序各期保持一致；变更需满足：准则要求或能提供更可靠相关信息 |
| 重要性 | 重要项目单独列报，不重要项目可汇总；从性质和金额两方面判断 |
| 不得抵销 | 资产与负债、收入与费用不得相互抵销（净额列示不属于抵销：类似交易净额、扣除备抵后净额、非日常活动净额） |
| 比较信息 | 至少提供上一可比期间比较数据；追溯调整/重述时至少列报三期资产负债表 |

### 三、资产负债表

**结构**：账户式，左方资产、右方负债+所有者权益，资产=负债+所有者权益。分"期末余额"和"上年年末余额"两栏。

**流动资产条件**（满足之一即为流动资产）：
1. 预计在一个正常营业周期中变现、出售或耗用
2. 主要为交易目的而持有
3. 预计在资产负债表日起一年内变现
4. 自资产负债表日起一年内交换其他资产或清偿负债的能力不受限制的现金或现金等价物

**流动负债条件**（满足之一即为流动负债）：
1. 预计在一个正常营业周期中清偿
2. 主要为交易目的而持有
3. 自资产负债表日起一年内到期应予以清偿
4. 企业在资产负债表日没有将清偿推迟至一年以上的实质性权利

> 关键规则：负债流动性划分反映资产负债表日有效的合同安排，资产负债表日后的再融资/展期不影响划分。违反长期借款协议导致贷款人可随时要求清偿的→流动负债；但贷款人在资产负债表日前同意提供一年以上宽限期的→非流动负债。

**资产负债表关键项目填列规则**：

| 项目 | 填列方法 |
|------|----------|
| 货币资金 | 库存现金+银行存款+其他货币资金+数字货币合计 |
| 应收账款 | "应收账款"期末余额 - 相关坏账准备 |
| 预付款项 | "预付账款"和"应付账款"所属明细借方余额合计 - 相关坏账准备 |
| 存货 | 材料采购+在途物资+原材料+库存商品+生产成本等合计 - 存货跌价准备等（计划成本需调整差异） |
| 固定资产 | "固定资产" - 累计折旧 - 固定资产减值准备 + 固定资产清理 |
| 在建工程 | "在建工程"+"工程物资" - 相关减值准备 |
| 应付账款 | "应付账款"和"预付账款"所属明细贷方余额合计 |
| 长期借款 | "长期借款"总账余额 - 一年内到期且无实质性展期权利部分 |
| 其他应付款 | "其他应付款"+"应付利息"+"应付股利"合计 |
| 长期应付款 | "长期应付款" - 未确认融资费用 + 专项应付款 |

> 注意：长期待摊费用剩余摊销期≤1年的仍在原项目列示，不转入"一年内到期的非流动资产"；递延收益同理不转入"一年内到期的非流动负债"。

### 四、利润表

**结构**：多步式，按功能分类费用。分"本期金额"和"上期金额"两栏。

**完整公式**：
```
营业收入
- 营业成本
- 税金及附加
- 销售费用
- 管理费用
- 研发费用
- 财务费用
+ 其他收益
+ 投资收益（损失以"-"号填列）
+ 净敞口套期收益
+ 公允价值变动收益（损失以"-"号填列）
+ 信用减值损失（损失以"-"号填列）
+ 资产减值损失（损失以"-"号填列）
+ 资产处置收益（损失以"-"号填列）
= 营业利润
+ 营业外收入
- 营业外支出
= 利润总额
- 所得税费用
= 净利润
  其中：(一)持续经营净利润
       (二)终止经营净利润
+ 其他综合收益的税后净额
  (一)不能重分类进损益的其他综合收益
  (二)将重分类进损益的其他综合收益
= 综合收益总额
```

**其他综合收益分类**：
- 不能重分类进损益：重新计量设定受益计划变动额、权益法不能转损益的OCI、其他权益工具投资公允价值变动、企业自身信用风险公允价值变动
- 将重分类进损益：权益法可转损益的OCI、其他债权投资公允价值变动、金融资产重分类计入OCI的金额、其他债权投资信用减值准备、现金流量套期储备、外币报表折算差额

### 五、现金流量表

**结构**：按收付实现制编制，直接法列报经营活动现金流量，附注中提供间接法调节信息。

**三类活动划分**：

| 活动类型 | 定义 | 主要内容 |
|----------|------|----------|
| 经营活动 | 投资和筹资以外的所有交易和事项 | 销售商品/提供劳务收到现金、购买商品/接受劳务支付现金、支付职工薪酬、支付各项税费、收到税费返还 |
| 投资活动 | 长期资产的购建和不包括在现金等价物范围内的投资及其处置 | 购建/处置固定资产无形资产、投资支付/收回现金、取得/处置子公司、取得投资收益 |
| 筹资活动 | 导致资本及债务规模和构成发生变化的活动 | 吸收投资、取得/偿还借款、发行/偿还债券、分配股利利润或偿付利息、偿还租赁负债 |

> 关键规则：应付票据/应付账款属于经营活动，不属于筹资活动。票据贴现不符合终止确认条件的→筹资活动；符合终止确认条件的→经营活动。偿还租赁负债本金和利息→筹资活动。

**间接法调节（净利润→经营活动现金流量）**：
```
净利润
+ 资产减值准备 + 信用减值准备
+ 固定资产折旧/投资性房地产折旧/油气资产折耗/生产性生物资产折旧
+ 使用权资产折旧
+ 无形资产摊销 + 长期待摊费用摊销
+ 处置固定资产、无形资产和其他长期资产的损失（收益为负）
+ 固定资产报废损失（收益为负）
+ 公允价值变动损失（收益为负）
+ 财务费用（收益为负）
+ 投资损失（收益为负）
+ 递延所得税资产减少（增加为负）
+ 递延所得税负债增加（减少为负）
+ 存货的减少（增加为负）
+ 经营性应收项目的减少（增加为负）
+ 经营性应付项目的增加（减少为负）
= 经营活动产生的现金流量净额
```

### 六、所有者权益变动表

矩阵式结构：纵向列示变动原因，横向列示各组成部分（实收资本/股本、其他权益工具、资本公积、减：库存股、其他综合收益、专项储备、盈余公积、未分配利润）。

变动原因包括：
- 综合收益总额
- 所有者投入和减少资本（投入普通股、其他权益工具持有者投入、股份支付计入权益）
- 利润分配（提取盈余公积、对所有者分配）
- 所有者权益内部结转（资本公积/盈余公积转增资本、弥补亏损、设定受益计划变动额结转留存收益、其他综合收益结转留存收益）
- 专项储备（本期提取、本期使用）

### 七、附注的主要内容

按以下顺序至少披露：
1. 企业基本情况（注册地、业务性质、母公司名称、财务报告批准报出日）
2. 财务报表编制基础（是否持续经营）
3. 遵循企业会计准则的声明
4. 重要会计政策和会计估计（确定依据、计量基础、关键假设）
5. 会计政策和会计估计变更以及差错更正的说明
6. 报表重要项目的说明
7. 其他需要说明的重要事项（或有事项、日后事项、关联方交易）
8. 有助于评价企业管理资本的目标、政策及程序的信息

### 八、中期财务报告

**构成**：至少包括资产负债表、利润表、现金流量表、附注（所有者权益变动表可选）。

**编制原则**：
- 会计政策与年度报告一致，中期视为独立会计期间
- 重要性以中期财务数据为基础判断（非预计年度数据）
- 季节性/周期性/偶然性收入在发生时确认，不得预计或递延
- 不均匀发生的费用在发生时确认，不得预提或待摊（年末允许的除外）

**比较报表要求**：
- 本中期末资产负债表 + 上年度末资产负债表
- 本中期利润表 + 年初至本中期末利润表 + 上年可比中期利润表 + 上年年初至可比中期末利润表
- 年初至本中期末现金流量表 + 上年年初至可比中期末现金流量表

## 第二十四章 会计政策、会计估计及其变更和差错更正

### 一、会计政策vs会计估计的区分

| 维度 | 会计政策 | 会计估计 |
|------|----------|----------|
| 定义 | 企业在会计确认、计量和报告中所采用的原则、基础和会计处理方法 | 企业对结果不确定性的交易或事项以最近可利用信息为基础所作的判断 |
| 涉及内容 | 会计确认标准、计量基础（属性）、列报项目 | 为取得与报表项目有关的金额或数值所采用的处理方法 |
| 举例 | 存货发出计价方法、长期股权投资核算方法、投资性房地产后续计量模式、收入确认方法、借款费用资本化、金融资产分类 | 固定资产使用寿命/净残值/折旧方法、坏账准备计提比例、存货可变现净值、公允价值确定方法、预计负债金额、履约进度 |

**划分基础**：以变更事项的会计确认、计量基础和列报项目是否发生变更为判断标准。涉及确认/计量基础/列报变更→会计政策变更；仅涉及金额或数值确定→会计估计变更。

**难以区分时**：按会计估计变更处理。

### 二、会计政策变更

**属于会计政策变更的典型情形**：
1. 发出存货计价方法的变更（如后进先出法改为先进先出法）
2. 投资性房地产后续计量模式变更（成本模式→公允价值模式）
3. 收入确认方法的变更（如执行新收入准则）
4. 借款费用由费用化改为资本化
5. 所得税核算由应付税款法改为资产负债表债务法
6. 研发支出由全部费用化改为符合条件的资本化
7. 商品进货费用由计入当期损益改为计入存货采购成本

**不属于会计政策变更的情形**：
1. 本期发生的交易与以前相比具有本质差别而采用新政策（如自用房产改为出租→投资性房地产）
2. 对初次发生的或不重要的交易采用新的会计政策

### 三、会计政策变更的条件和处理

**变更条件**（满足之一）：
1. 法律、行政法规或国家统一的会计制度等要求变更
2. 变更能够提供更可靠、更相关的会计信息

**处理方法选择**：

| 情形 | 处理方法 |
|------|----------|
| 国家发布了相关会计处理办法 | 按国家规定处理 |
| 国家未发布处理办法 | 追溯调整法 |
| 变更能提供更可靠更相关信息 | 追溯调整法 |
| 确定对列报前期影响数不切实可行 | 从可追溯的最早期间期初开始应用 |
| 当期期初确定累积影响数不切实可行 | 未来适用法 |

### 四、追溯调整法

**步骤**：
1. 计算会计政策变更的累积影响数（列报前期最早期初留存收益应有金额与现有金额之差）
2. 编制相关项目的调整分录
3. 调整列报前期最早期初财务报表相关项目及其金额
4. 附注说明

**累积影响数计算步骤**：
1. 根据新政策重新计算受影响的前期交易或事项
2. 计算两种政策下的差异
3. 计算差异的所得税影响金额
4. 确定前期中每一期的税后差异
5. 计算累积影响数

**典型分录**：
```
借：利润分配——未分配利润（或贷方，累积影响数）
    递延所得税资产（或贷方）
  贷：相关资产/负债科目（或借方）
      递延所得税负债（或贷方）

借：盈余公积（或贷方，按比例调整）
  贷：利润分配——未分配利润（或借方）
```

> 注意：不考虑由于损益变化而应补分的利润或股利。应重新计算各列报期间的每股收益。

### 五、未来适用法（会计政策变更）

将变更后的会计政策应用于变更日及以后发生的交易或事项。不计算累积影响数，不重编以前年度财务报表，在现有金额基础上按新政策核算。

### 六、会计估计变更

**属于会计估计变更的典型情形**：
1. 固定资产使用寿命、净残值的变更
2. 固定资产折旧方法的变更（如双倍余额递减法改为直线法）
3. 无形资产摊销年限、摊销方法的变更
4. 坏账准备计提比例的变更
5. 存货可变现净值的重新确定
6. 公允价值确定方法的变更（因市场情况变化）
7. 预计负债金额的重新估计

**处理方法**：未来适用法
- 仅影响变更当期的→变更当期确认
- 既影响变更当期又影响未来期间的→变更当期和未来期间确认
- 影响数计入与前期相同的项目（日常活动损益→日常活动损益）

**典型处理**：
```
变更后每年折旧 = (变更日账面净值 - 新预计净残值) ÷ 新预计剩余使用年限
不调整以前各期折旧，不计算累积影响数
```

### 七、前期差错

**定义**：由于没有运用或错误运用编报时预期能够取得的可靠信息，或前期财务报告批准报出时能够取得的可靠信息，而对前期财务报表造成的省略或错报。

**类型**：计算错误、账户分类错误、应用不允许的会计政策、疏忽或曲解事实、舞弊。

**重要性判断**：
- 重要的前期差错：足以影响报表使用者对财务状况、经营成果和现金流量作出正确判断
- 不重要的前期差错：不足以影响使用者判断
- 判断标准：综合考虑差错对差错发现当期及其比较期间财务报表的影响

### 八、前期差错更正的处理

**不重要的前期差错**：不调整期初数，调整发现当期与前期相同的相关项目。

**重要的前期差错**：追溯重述法

**追溯重述法分录**：
```
① 更正差错：
借：以前年度损益调整（多计费用则贷方）
  贷：相关资产/负债（如累计折旧、应收账款等）

② 调整所得税：
借：应交税费——应交所得税（多缴则借方）
  贷：以前年度损益调整

③ 结转至留存收益：
借：利润分配——未分配利润
  贷：以前年度损益调整

④ 调整盈余公积：
借：盈余公积（法定+任意）
  贷：利润分配——未分配利润
```

> 同时调整比较财务报表相关项目的上年金额/年初余额。

### 九、会计估计变更vs前期差错的区分

| 判断标准 | 会计估计变更 | 前期差错更正 |
|----------|-------------|-------------|
| 前期估计基础 | 以当时信息和合理假设为基础 | 未合理使用编报时已存在且能取得的可靠信息 |
| 变更原因 | 资产/负债当前状况及预期经济利益发生变化 | 前期估计结果未恰当反映当时情况 |
| 处理方法 | 未来适用法 | 追溯重述法 |

> 不应简单将会计估计与实际结果对比认定存在差错。

### 十、三种变更/更正方法对比

| 方法 | 适用情形 | 是否调整以前期间 | 核心操作 |
|------|----------|-----------------|----------|
| 追溯调整法 | 会计政策变更 | 是，调整期初留存收益和比较报表 | 视同一直采用新政策，计算累积影响数 |
| 未来适用法 | 会计估计变更；政策变更无法追溯时 | 否 | 从变更日起按新估计/政策处理 |
| 追溯重述法 | 重要的前期差错更正 | 是，调整期初留存收益和比较报表 | 视同差错从未发生，通过"以前年度损益调整"更正 |

## 第二十五章 资产负债表日后事项

### 一、日后事项期间与定义

- **定义**：资产负债表日至财务报告批准报出日之间发生的有利或不利事项
- **涵盖期间**：资产负债表日次日起 → 财务报告批准报出日止
  - 财务报告批准报出日 = 董事会（或类似机构）批准财务报告对外公布的日期
  - 若批准报出后、实际报出前又发生相关事项需调整，以再次批准报出日为截止日

### 二、调整事项 vs 非调整事项

| 类型 | 判断标准 | 处理方式 |
|------|----------|----------|
| 调整事项 | 对资产负债表日**已经存在**的情况提供了新的或进一步证据 | 调整报告期财务报表 |
| 非调整事项 | 表明资产负债表日**后才发生**的新情况 | 重要的在附注中披露 |

**核心判断原则**：该事项表明的情况在资产负债表日或之前是否已经存在。已存在→调整事项；日后才发生→非调整事项。

### 三、典型调整事项（至少6项）

1. 日后诉讼案件结案，法院判决证实了资产负债表日已存在现时义务，需调整预计负债或确认新负债
2. 日后取得确凿证据，表明某项资产在资产负债表日发生了减值或需调整原先确认的减值金额
3. 日后进一步确定了资产负债表日前购入资产的成本或售出资产的收入
4. 日后发现了财务报表舞弊或差错
5. 资产负债表日后发生的销售退回（因资产负债表日已存在的质量问题导致）
6. 资产负债表日后债务人被宣告破产（资产负债表日已存在财务恶化事实）

### 四、典型非调整事项（至少6项）

1. 资产负债表日后发生重大诉讼、仲裁、承诺
2. 资产负债表日后资产价格、税收政策、外汇汇率发生重大变化
3. 资产负债表日后因自然灾害导致资产发生重大损失
4. 资产负债表日后发行股票和债券以及其他巨额举债
5. 资产负债表日后资本公积转增资本
6. 资产负债表日后发生企业合并或处置子公司
7. 资产负债表日后发生巨额亏损
8. 企业利润分配方案中拟分配的以及经审议批准宣告发放的现金股利或利润


### 五、调整事项的会计处理原则

年度资产负债表日后发生的调整事项，分以下情况处理：
1. **涉及损益**：通过"以前年度损益调整"科目核算（损益类科目已结转无余额）
2. **涉及所得税**：调整"应交税费——应交所得税"或"递延所得税资产/负债"
3. **涉及利润分配**：直接在"利润分配——未分配利润"科目核算
4. **不涉及损益及利润分配**：调整相关科目
5. 最后调整财务报表相关项目的期末数/本年发生数，以及比较报表的期初数/上年数

### 六、诉讼结案的完整分录（含所得税和盈余公积调整）

```
（1）补提预计负债差额（假定原已确认预计负债，判决金额更大）：
借：以前年度损益调整（差额部分）
  贷：其他应付款

（2）调整所得税（假定损失实际发生时才允许税前抵扣）：
借：应交税费——应交所得税（判决金额 × 25%）
  贷：以前年度损益调整

同时冲销原确认的递延所得税资产：
借：以前年度损益调整
  贷：递延所得税资产（原预计负债 × 25%）

（3）将预计负债转入其他应付款：
借：预计负债
  贷：其他应付款

（4）支付赔偿款：
借：其他应付款
  贷：银行存款

（5）结转以前年度损益调整：
借：利润分配——未分配利润
  贷：以前年度损益调整（净额）

（6）调整盈余公积：
借：盈余公积（净利润减少额 × 10%）
  贷：利润分配——未分配利润
```

### 七、销售退回的完整分录（含所得税调整）

```
（1）冲减收入：
借：以前年度损益调整
    应交税费——应交增值税（销项税额）
  贷：应收账款

（2）冲减成本：
借：库存商品
  贷：以前年度损益调整

（3）调整所得税（退回发生在汇算清缴前）：
借：应交税费——应交所得税（利润减少额 × 25%）
  贷：以前年度损益调整

（4）结转以前年度损益调整：
借：利润分配——未分配利润
  贷：以前年度损益调整（净额）

（5）调整盈余公积：
借：盈余公积
  贷：利润分配——未分配利润
```

### 八、坏账确认的分录

```
资产负债表日后取得证据表明应收账款发生减值（需补提坏账准备）：
借：以前年度损益调整（补提的信用减值损失）
  贷：应收账款——坏账准备

调整所得税：
借：递延所得税资产（补提金额 × 25%）
  贷：以前年度损益调整

结转：
借：利润分配——未分配利润
  贷：以前年度损益调整

调整盈余公积：
借：盈余公积
  贷：利润分配——未分配利润
```

## 第二十六章 企业合并

### 一、企业合并的界定与类型

#### （一）按合并方式划分

| 方式 | 特征 | 结果 |
|------|------|------|
| 控股合并 | 取得被合并方控制权，被合并方保留法人资格 | 形成母子公司关系 |
| 吸收合并 | 取得被合并方全部净资产，注销被合并方法人资格 | 合并方持有全部资产负债继续经营 |
| 新设合并 | 参与合并各方法人资格均被注销，新设企业 | 新企业持有全部资产负债 |

#### （二）按控制关系划分

| 类型 | 判断标准 | 会计处理方法 |
|------|----------|--------------|
| 同一控制下企业合并 | 合并前后均受同一方或相同多方最终控制，且该控制非暂时性（≥1年） | 类似权益结合法（账面价值） |
| 非同一控制下企业合并 | 合并前后不受同一方或相同多方最终控制 | 购买法（公允价值） |


**同一控制下的判断要点**：
- 能够对参与合并各方在合并前后均实施最终控制的一方通常为企业集团的母公司
- 控制时间要求：合并前后均≥1年
- 同受国家控制的企业之间的合并，不应仅因均为国有企业即作为同一控制下企业合并，应按实质重于形式原则判断

### 二、业务的判断标准

- **业务定义**：企业内部某些生产经营活动或资产负债的组合，具有投入、加工处理过程和产出能力
- **构成业务的条件**：至少同时具有一项投入和一项**实质性**加工处理过程，且二者结合对产出能力有显著贡献
- 有无实际产出不是判断构成业务的必要条件

**实质性加工处理过程的判断**：

| 情形 | 实质性标准 |
|------|-----------|
| 合并日无产出 | ①加工处理过程对投入转化为产出至关重要；②具备执行该过程所需技能、知识或经验的有组织员工及必要投入 |
| 合并日有产出 | 满足之一：①对持续产出至关重要且有组织员工；②对产出能力有显著贡献且该过程独有、稀缺或难以取代 |

### 三、集中度测试（非同一控制下可选用）

- **适用范围**：非同一控制下企业合并的购买方判断取得的组合是否构成业务时可选择采用
- **测试方法**：取得的总资产公允价值是否几乎相当于其中某一单独可辨认资产或一组类似可辨认资产的公允价值
- **通过测试**→不构成业务，无须进一步判断
- **未通过测试**→仍需按构成业务的判断条件进行判断

总资产公允价值计算时不包括：现金及现金等价物、递延所得税资产、由递延所得税负债影响形成的商誉

### 四、同一控制下企业合并的处理原则

1. 合并方取得的被合并方资产、负债按**账面价值**（最终控制方合并报表中的账面价值）确认
2. **不产生新的商誉**（被合并方原已确认的商誉除外）
3. 合并对价账面价值与取得净资产入账价值的差额→调整**资本公积（资本溢价/股本溢价）**，不足冲减的依次冲减盈余公积和未分配利润
4. 合并方与被合并方会计政策不一致的，应先统一会计政策再合并
5. 处置原同一控制下取得的子公司时，原合并日调整的资本公积**不得转出**至当期损益或留存收益

### 五、同一控制下控股合并分录

#### 发行股份方式：
```
借：长期股权投资（被合并方净资产在最终控制方合并报表中的账面价值 × 持股比例）
  贷：股本（发行股份面值）
      资本公积——股本溢价（差额，可能在借方）
```

#### 支付现金/非现金资产方式：
```
借：长期股权投资（被合并方净资产在最终控制方合并报表中的账面价值 × 持股比例）
    累计折旧/累计摊销（如涉及）
  贷：固定资产/无形资产/银行存款等（对价账面价值）
      资本公积——资本溢价（差额，可能在借方）
```

#### 合并日合并报表抵销分录：
```
借：股本（子公司）
    资本公积（子公司）
    盈余公积（子公司）
    未分配利润（子公司）
  贷：长期股权投资
      少数股东权益（如非全资）
```

恢复被合并方留存收益（归属于合并方的部分，以资本公积余额为限）：
```
借：资本公积
  贷：盈余公积
      未分配利润
```

### 六、同一控制下吸收合并分录

```
借：各项资产（被合并方各项资产账面价值）
  贷：各项负债（被合并方各项负债账面价值）
      股本（如发行股份，面值部分）
      资本公积——股本溢价（差额）
```

### 七、非同一控制下企业合并的处理原则

1. 采用**购买法**，以**公允价值**为基础
2. 合并成本 = 购买方支付的现金/非现金资产/发行权益性证券等在**购买日的公允价值**
3. 取得的被购买方各项可辨认资产、负债按**购买日公允价值**确认
4. 合并成本 > 可辨认净资产公允价值份额 → 确认**商誉**
5. 合并成本 < 可辨认净资产公允价值份额 → 复核后计入当期**营业外收入**
6. 以非现金资产为对价的，公允价值与账面价值差额计入**资产处置损益**


### 八、非同一控制下控股合并分录

#### 以现金为对价：
```
借：长期股权投资（合并对价公允价值）
  贷：银行存款
```

#### 以发行股权为对价：
```
借：长期股权投资（发行股份公允价值）
  贷：股本（面值）
      资本公积——股本溢价（公允价值与面值差额）
```

#### 以非现金资产为对价（混合对价）：
```
借：长期股权投资（合并对价公允价值合计）
  贷：银行存款（现金部分）
      股本（发行股份面值）
      资本公积——股本溢价
      固定资产清理/无形资产等（账面价值）
      资产处置收益（非现金资产公允价值与账面价值差额）
```

#### 购买日合并报表抵销分录：
```
先将子公司资产负债调整为公允价值：
借：存货（公允价值增值）
    固定资产（公允价值增值）
    无形资产（公允价值增值）
  贷：应收账款（公允价值减值）
      资本公积（差额）

再抵销：
借：股本（子公司）
    资本公积（子公司，含公允价值调整）
    盈余公积（子公司）
    未分配利润（子公司）
    商誉（合并成本 - 可辨认净资产公允价值 × 持股比例）
  贷：长期股权投资
      少数股东权益（可辨认净资产公允价值 × 少数股东比例）
```

### 九、非同一控制下吸收合并分录

```
借：各项资产（被购买方各项可辨认资产公允价值）
    商誉（合并成本 > 可辨认净资产公允价值时的差额）
  贷：各项负债（被购买方各项负债公允价值）
      银行存款/股本/资本公积等（合并对价）

若合并成本 < 可辨认净资产公允价值（廉价购买）：
借：各项资产（公允价值）
  贷：各项负债（公允价值）
      银行存款/股本等（合并对价）
      营业外收入（差额）
```

### 十、合并费用的处理

| 费用类型 | 处理方式 |
|----------|----------|
| 审计费、评估费、法律咨询费等直接相关费用 | 计入当期损益（管理费用） |
| 发行权益性证券的佣金、手续费 | 冲减资本公积（股本溢价），不足冲减的冲减留存收益 |
| 发行债券的佣金、手续费 | 计入债券初始确认金额（按金融工具准则处理） |

### 十一、分步实现企业合并的处理

**非同一控制下分步合并**：
- 购买日之前持有的被购买方股权，按购买日公允价值重新计量
- 公允价值与账面价值的差额计入当期投资收益
- 原计入其他综合收益的金额转入当期损益

**同一控制下分步合并**：
- 合并日，长期股权投资初始投资成本 = 被合并方净资产在最终控制方合并报表中的账面价值 × 合并后持股比例
- 初始投资成本与原投资账面价值 + 合并日新支付对价账面价值之和的差额，调整资本公积

## 第二十七章 合并财务报表

### 一、合并范围——控制的三要素

投资方控制被投资方必须**同时**具备以下三个要素：

| 要素 | 含义 |
|------|------|
| 权力 | 投资方拥有对被投资方的权力（主导相关活动的现时能力） |
| 可变回报 | 因参与被投资方的相关活动而享有可变回报 |
| 权力与回报的联系 | 有能力运用对被投资方的权力影响其回报金额 |

### 二、权力的来源

| 来源 | 说明 |
|------|------|
| 表决权 | 最常见来源；通常持有半数以上表决权即拥有权力 |
| 合同安排 | 结构化主体中，权力可能来自合同约定的决策权而非表决权 |
| 潜在表决权 | 可转换工具、认股权证、远期股权购买合同等产生的权利（须为实质性权利） |
| 其他权利组合 | 表决权+委派关键管理人员+合同安排等组合 |

### 三、实质性权利 vs 保护性权利

| 类型 | 特征 | 在控制判断中的作用 |
|------|------|-------------------|
| 实质性权利 | 持有人有实际能力行使的可执行权利 | 应予考虑 |
| 保护性权利 | 旨在保护持有人权益，不赋予对主体的权力 | 不予考虑 |

**保护性权利举例**：贷款方限制借款方高风险活动的权利、少数股东批准超常规资本性支出的权利、违约时扣押资产的权利

### 四、持有半数以下表决权仍可能控制的情况

综合考虑以下因素判断：
1. 投资方持有的表决权相对于其他方的份额大小及其他方的分散程度
2. 与其他表决权持有人的协议（使投资方可控制足够表决权）
3. 其他合同安排产生的权利（如指派权力机构成员的权利）
4. 其他事实或情况：能否任命关键管理人员、能否决定重大交易、与被投资方管理层的关联关系等

### 五、投资性主体豁免

- 母公司为投资性主体的，只将为其投资活动提供相关服务的子公司纳入合并范围
- 其他子公司按以公允价值计量且其变动计入当期损益的金融资产核算
- 投资性主体的母公司如果本身不是投资性主体，应将全部主体（包括通过投资性主体间接控制的主体）纳入合并范围


### 六、编制步骤概述

1. 统一母子公司会计政策和会计期间
2. 对外币报表进行折算
3. 设置合并工作底稿，将母子公司个别报表数据过入并加总
4. 编制调整分录（非同一控制下：将子公司资产负债调整为公允价值基础）
5. 将成本法调整为权益法
6. 编制抵销分录（长期股权投资与所有者权益、投资收益与利润分配、内部交易等）
7. 计算合并数并填列合并财务报表

### 七、成本法调整为权益法的分录

#### 投资当年：
```
按子公司当年净利润中母公司享有的份额调增：
借：长期股权投资（子公司调整后净利润 × 持股比例）
  贷：投资收益

按子公司当年分派现金股利中母公司享有的份额调减：
借：投资收益
  贷：长期股权投资

按子公司其他综合收益变动中母公司享有的份额：
借：长期股权投资
  贷：其他综合收益

按子公司除净损益、其他综合收益和利润分配以外的所有者权益其他变动：
借：长期股权投资
  贷：资本公积
```

#### 连续编制（第二年及以后）：
```
按子公司以前年度净利润累计调整：
借：长期股权投资（以前年度累计调整后净利润 × 持股比例 - 以前年度累计分派股利 × 持股比例）
  贷：未分配利润——年初（以前年度投资收益净额）

按子公司本年净利润调整：
借：长期股权投资（本年调整后净利润 × 持股比例）
  贷：投资收益

按子公司本年分派股利调整：
借：投资收益
  贷：长期股权投资
```

### 八、长期股权投资与所有者权益的抵销分录

#### 全资子公司：
```
借：股本（子公司）
    资本公积（子公司）
    其他综合收益（子公司）
    盈余公积（子公司）
    未分配利润——年末（子公司）
    商誉（借方差额，非同一控制下）
  贷：长期股权投资（经权益法调整后的金额）
```

#### 非全资子公司：
```
借：股本（子公司）
    资本公积（子公司）
    其他综合收益（子公司）
    盈余公积（子公司）
    未分配利润——年末（子公司）
    商誉（借方差额，非同一控制下）
  贷：长期股权投资（经权益法调整后的金额）
      少数股东权益（子公司所有者权益 × 少数股东比例）
```

### 九、投资收益与利润分配的抵销分录

```
借：投资收益（子公司调整后净利润 × 母公司持股比例）
    少数股东损益（子公司调整后净利润 × 少数股东比例）
    未分配利润——年初（子公司年初未分配利润）
  贷：提取盈余公积（子公司本年提取数）
      对所有者（或股东）的分配（子公司本年分配数）
      未分配利润——年末（子公司年末未分配利润，倒挤）
```

### 十、内部商品交易的抵销

#### （一）当期内部销售，期末存货未实现利润

```
借：营业收入（内部销售收入全额）
  贷：营业成本（差额，倒挤）
      存货（期末未售出部分包含的未实现内部利润）
```

#### （二）连续编制——上期存货本期仍未售出

```
借：未分配利润——年初（上期末未实现利润）
  贷：存货
```

#### （三）连续编制——上期存货本期已全部售出

```
借：未分配利润——年初
  贷：营业成本
```

#### （四）连续编制——上期存货本期部分售出

```
借：未分配利润——年初（上期末全部未实现利润）
  贷：营业成本（本期已售出部分的未实现利润）
      存货（本期仍未售出部分的未实现利润）
```

#### （五）存货跌价准备的抵销

```
当期计提（内部购入存货的可变现净值低于其取得成本但高于集团成本）：
借：存货——存货跌价准备
  贷：资产减值损失

连续编制：
借：存货——存货跌价准备
  贷：未分配利润——年初

当期补提/转回的调整：
借/贷：存货——存货跌价准备
  贷/借：资产减值损失
```

抵销存货跌价准备的限额：以未实现内部利润为限（即抵销后存货跌价准备余额不能为负）

#### （六）递延所得税的处理

```
因抵销未实现利润导致合并报表中存货账面价值 < 计税基础，产生可抵扣暂时性差异：
借：递延所得税资产（未实现利润 × 购买方税率）
  贷：所得税费用

连续编制：
借：递延所得税资产
  贷：未分配利润——年初

本期变动调整：
借/贷：递延所得税资产
  贷/借：所得税费用
```


### 十一、内部固定资产交易的抵销

#### （一）交易当期（卖方为生产商品的企业）

```
抵销未实现利润：
借：营业收入（内部售价）
  贷：营业成本（内部成本）
      固定资产——原价（未实现内部利润 = 售价 - 成本）

抵销当年多计提的折旧：
借：固定资产——累计折旧（未实现利润 ÷ 剩余使用年限 × 当年使用月数/12）
  贷：管理费用/制造费用
```

#### （二）交易当期（卖方为使用该资产的企业，即固定资产→固定资产）

```
借：资产处置收益（内部交易利润）
  贷：固定资产——原价

借：固定资产——累计折旧（当年多提折旧）
  贷：管理费用/制造费用
```

#### （三）以后各期（连续编制）

```
抵销原价中的未实现利润：
借：未分配利润——年初
  贷：固定资产——原价

抵销以前年度累计多计提的折旧：
借：固定资产——累计折旧（以前年度累计多提）
  贷：未分配利润——年初

抵销本年多计提的折旧：
借：固定资产——累计折旧（本年多提）
  贷：管理费用/制造费用
```

#### （四）清理期间（使用期满报废清理）

```
借：未分配利润——年初（原未实现利润）
  贷：管理费用/制造费用（最后一年应抵销的折旧，即未实现利润 - 以前年度累计已抵销折旧）

注：清理当期无需再抵销固定资产原价和累计折旧（资产已不存在），
只需将剩余未抵销的未实现利润通过调整费用的方式消除。
```

#### （五）清理期间（提前清理/出售）

```
借：未分配利润——年初（原未实现利润）
  贷：固定资产——累计折旧（以前年度累计多提）
      资产处置收益/营业外收入（差额）
```

### 十二、内部无形资产交易的抵销

原理与内部固定资产交易相同：

```
交易当期：
借：资产处置收益（内部交易利润）
  贷：无形资产——原价

借：无形资产——累计摊销（当年多摊销额）
  贷：管理费用

连续编制：
借：未分配利润——年初
  贷：无形资产——原价

借：无形资产——累计摊销（以前年度累计多摊销）
  贷：未分配利润——年初

借：无形资产——累计摊销（本年多摊销）
  贷：管理费用
```

### 十三、内部债权债务的抵销

#### （一）基本抵销

```
借：应付账款/应付票据/其他应付款
  贷：应收账款/应收票据/其他应收款
```

#### （二）坏账准备的抵销——当期

```
当期对内部应收款计提坏账准备：

  贷：信用减值损失
```

```
  贷：递延所得税资产
```

## 第二十八章 每股收益

### 一、基本每股收益

**公式**：

$$基本每股收益 = \frac{归属于普通股股东的当期净利润}{发行在外普通股的加权平均数}$$

#### （一）分子的确定

归属于普通股股东的当期净利润 = 当期净利润 - 其他权益工具的股利或利息

**优先股股利的扣除规则**：
- **不可累积优先股**：仅扣除当期宣告发放的股利
- **累积优先股**：无论当期是否宣告发放，均应扣除归属于该期间的股利

以合并报表为基础计算时，分子 = 归属于母公司普通股股东的当期合并净利润（扣减少数股东损益后的余额）

#### （二）分母的确定

$$发行在外普通股加权平均数 = 期初发行在外普通股股数 + 当期新发行普通股股数 × \frac{已发行时间}{报告期时间} - 当期回购普通股股数 × \frac{已回购时间}{报告期时间}$$

- 时间权数通常按天数计算，简化时可按月数计算
- 库存股不属于发行在外普通股，应从分母中扣除

**新发行普通股计算起点**：
- 收取现金发行 → 自应收现金之日起
- 债务转资本 → 自停计债务利息之日或结算日起
- 非同一控制下企业合并 → 自购买日起
- 同一控制下企业合并 → 计入各列报期间加权平均数

### 二、稀释每股收益

#### （一）基本概念

稀释每股收益 = 假设所有稀释性潜在普通股均已转换为普通股后计算的每股收益

**稀释性判断标准**：假定转换后会减少**持续经营**每股收益或增加每股亏损 → 具有稀释性

#### （二）可转换公司债券

**假设转换法**：假设债券在期初（或发行日）即已转换为普通股

- **分子调整**：加回当期已确认为费用的利息的税后影响额
  - 调整金额 = 负债成分摊余成本 × 实际利率 × (1 - 所得税税率)
  - 注意：仅调整属于金融负债部分的利息（含权益成分拆分的债券按负债部分实际利率计算）
- **分母调整**：加上假定转换为普通股的股数加权平均数
  - 以前期间发行的 → 假设期初转换
  - 当期发行的 → 假设发行日转换

**稀释性判断**：增量股每股收益 = 增加的净利润 ÷ 增加的股数 < 基本每股收益 → 具有稀释性

#### （三）认股权证和股份期权（库存股法）

**适用条件**：行权价格 < 当期普通股平均市场价格时具有稀释性（仅适用于盈利企业）

- **分子**：不调整（假设行权不影响净利润）
- **分母**：加上增量股数

$$增量股数 = 拟行权时转换的普通股股数 - \frac{行权价格 × 拟行权时转换的普通股股数}{当期普通股平均市场价格}$$

**含义**：行权发行的股份 = 按市价发行部分（不稀释）+ 无对价发行部分（稀释），仅无对价部分计入分母

**亏损企业**：认股权证/股份期权假设行权会减少每股亏损金额（反稀释），不应计算稀释每股收益

#### （四）限制性股票

等待期内基本每股收益：
- 现金股利可撤销：分子扣除当期分配给预计未来可解锁限制性股票持有者的现金股利；分母不含限制性股票
- 现金股利不可撤销：分子扣除归属于预计未来可解锁限制性股票的净利润；分母不含限制性股票

等待期内稀释每股收益：
- **行权价格** = 限制性股票的发行价格 + 资产负债表日尚未取得的职工服务的公允价值
- 行权价格 < 当期普通股平均市场价格时 → 考虑稀释性
- 分子加回计算基本EPS时已扣除的现金股利或归属净利润
- 分母增量股数 = 限制性股票股数 - 行权价格 × 限制性股票股数 ÷ 当期普通股平均市场价格

**解锁条件包含业绩条件的**：资产负债表日实际业绩满足条件 → 按上述方法计算；不满足 → 不考虑稀释性

#### （五）企业承诺回购股份

**稀释条件**：回购价格 > 当期普通股平均市场价格时具有稀释性

$$增加的普通股股数 = \frac{回购价格 × 承诺回购的普通股股数}{当期普通股平均市场价格} - 承诺回购的普通股股数$$

**思路**：与认股权证相反——假设按市价发行足够股份募集回购资金，再按约定价格回购，净增加的股数即为稀释部分

#### （六）多项潜在普通股的排序规则

**步骤**：
1. 计算各潜在普通股的增量股每股收益（增加的净利润 ÷ 增加的股数）
2. 按增量股每股收益从小到大排序（稀释程度从大到小）
3. 依次计入稀释每股收益，每步检验：若计入后EPS下降 → 具有稀释性，保留；若计入后EPS上升 → 反稀释，不计入
4. 最终得到的最小每股收益即为稀释每股收益

**注意**：认股权证/股份期权通常排在最前面（增量股每股收益为零或极小）

### 三、每股收益的列报——需要重新计算的情形

#### （一）股票股利、公积金转增资本、拆股和并股

- 不影响所有者权益金额，仅改变股份规模
- 应按调整后的股数**重新计算各列报期间**的每股收益
- 发生于资产负债表日至财务报告批准报出日之间的 → 以调整后股数重新计算

#### （二）配股

配股 = 按市价发行 + 无对价送股的混合体，送股因素视同列报最早期间期初已发行在外

$$每股理论除权价格 = \frac{行权前发行在外普通股的公允价值总额 + 配股收到的款项}{行权后发行在外的普通股股数}$$

$$调整系数 = \frac{行权前每股公允价值}{每股理论除权价格}$$

- 因配股重新计算的上年度基本每股收益 = 上年度基本每股收益 ÷ 调整系数
- 本年度基本每股收益 = 当期净利润 ÷（配股前股数 × 调整系数 × 配股前时间权重 + 配股后股数加权平均数）

**注意**：向特定对象以低于市价发行股票（如战略投资者），若综合考虑其他补偿因素后可视为不存在送股因素，则视同发行新股处理，不适用配股调整

#### （三）追溯调整或追溯重述

按会计政策变更和差错更正准则对以前年度损益进行追溯调整/重述的 → 重新计算各列报期间的每股收益

## 第二十九章 公允价值计量

### 一、公允价值的定义

公允价值 = 市场参与者在计量日发生的有序交易中，出售一项资产所能收到或者转移一项负债所需支付的价格（**脱手价格**）

**关键特征**：
- 是脱手价格（出售/转移），非进入价格（取得/承担）
- 基于市场参与者角度，非企业自身角度
- 假定计量日发生有序交易

### 二、公允价值计量的基本要求

#### （一）相关资产或负债

- 应考虑资产或负债的特征（状况、位置、使用限制等）
- 限制针对资产本身（会传递给受让方）→ 计量时应考虑
- 限制针对资产持有者（不会传递）→ 计量时不应考虑
- 计量单元：以单独或组合方式计量的最小单位

#### （二）有序交易

- 当前市场情况下具有惯常市场活动的交易
- 不包括被迫清算和抛售
- 非有序交易的价格不应作为公允价值的主要依据

#### （三）主要市场或最有利市场

- **主要市场**：交易量最大、交易活跃程度最高的市场 → 优先使用
- **最有利市场**：不存在主要市场时，考虑交易费用和运输费用后能以最高金额出售资产/最低金额转移负债的市场
- 应从企业自身角度判定，但不要求计量日实际在该市场交易

**交易费用 vs 运输费用**：
- 交易费用：直接归属于资产出售/负债转移的不可避免费用 → **不调整**公允价值
- 运输费用：使资产从当前位置转移到主要市场/最有利市场的费用 → **应调整**公允价值（从市场价格中扣除）

#### （四）市场参与者

市场参与者特征：①相互独立；②熟悉情况；③有能力并自愿交易（非被迫）

企业应从市场参与者角度计量公允价值，不考虑企业自身持有/使用意图

### 三、估值技术

#### （一）市场法

利用相同或类似资产、负债或其组合的价格及其他相关市场交易信息进行估值

- 直接使用公开报价
- 市场乘数法（市盈率、市净率、EV/EBITDA等）

**适用**：存在活跃市场报价或可比交易的情形

#### （二）收益法

将未来金额转换为单一现值的估值技术

- **现金流量折现法**：传统法（最可能现金流量+风险调整折现率）、期望现金流量法（概率加权现金流量）
- **期权定价模型**：布莱克—斯科尔斯模型、二叉树模型、蒙特卡洛模拟法

**适用**：能够合理预测未来现金流量的情形

#### （三）成本法

反映当前要求重置相关资产服务能力所需金额（现行重置成本）

- 应考虑折旧贬值：实体性损耗、功能性贬值、经济性贬值

**适用**：主要用于与其他资产组合使用的有形资产

#### （四）估值技术的选择与变更

- 存在活跃市场公开报价 → 优先使用该报价
- 其他情况下三种方法无优先顺序，可使用一种或多种
- 估值技术一经确定不得随意变更（除非变更能更好代表公允价值）
- 变更按会计估计变更处理

### 四、输入值与公允价值层次

#### （一）三个层次的定义

| 层次 | 输入值 | 优先级 | 说明 |
|------|--------|--------|------|
| 第一层次 | 活跃市场中相同资产/负债未经调整的报价 | 最高 | 交易对象同质、可随时找到自愿交易方、价格信息公开 |
| 第二层次 | 除第一层次外直接或间接可观察的输入值 | 中 | 包括：活跃市场类似资产报价、非活跃市场相同/类似资产报价、可观察利率和收益率曲线、市场验证的输入值 |
| 第三层次 | 不可观察输入值 | 最低 | 包括：不能直接观察的利率、股票波动率、弃置义务未来现金流量、企业自身财务预测等 |

#### （二）层次确定规则

- 公允价值计量结果所属层次 = 对计量整体而言**重要的输入值所属的最低层次**
- 取决于输入值层次，而非估值技术本身
- 使用不可观察输入值对可观察输入值进行重大调整 → 计量结果划入第三层次

#### （三）第一层次的调整例外

以下情况可对活跃市场报价进行调整（调整后划入第二或第三层次）：
1. 持有大量类似资产难以获得单独定价信息
2. 发生影响公允价值的重大事件导致报价不代表计量日公允价值

### 五、非金融资产的公允价值计量

#### （一）最佳用途

企业以公允价值计量非金融资产，应考虑市场参与者将该资产用于**最佳用途**产生经济利益的能力

**最佳用途判断条件**（同时满足）：
1. **法律上允许**：未被法律禁止的用途
2. **实物上可能**：考虑资产实物特征
3. **财务上可行**：能产生足够收益补偿成本并满足投资回报要求

通常情况下，企业对非金融资产的当前用途可视为最佳用途（除非有证据表明其他用途可实现价值最大化）

#### （二）估值前提

- 单独使用实现最佳用途 → 公允价值 = 出售给同样单独使用的市场参与者的价格
- 与其他资产/负债组合使用实现最佳用途 → 公允价值 = 出售给以同样组合方式使用的市场参与者的价格（假定其可取得组合中其他资产/负债）

### 六、负债和自身权益工具的公允价值计量

**基本假设**：假定计量日将负债转移给市场参与者，负债转移后继续存在，由受让方履行义务

**确定方法优先顺序**：
1. 存在相同/类似负债或权益工具可观察市场报价 → 以该报价为基础
2. 被其他方作为资产持有 → 以对应资产的公允价值为基础（需调整不适用特征）
3. 未被其他方作为资产持有 → 采用估值技术（如未来现金流量现值）

**不履约风险**：
- 以公允价值计量负债应考虑不履约风险（包括企业自身信用风险）
- 假定不履约风险在转移前后保持不变
- 附有不可分割的第三方信用增级且单独会计处理的 → 不考虑该信用增级影响

**具有可随时要求偿还特征的金融负债**：公允价值不应低于债权人要求偿还时的应付金额（从可要求偿还第一天起折现的现值）

### 七、公允价值披露要求

企业应当披露：
- 持续和非持续以公允价值计量的资产和负债在各层次的金额
- 第三层次公允价值计量的估值过程、输入值信息
- 各层次之间的转换情况及原因
- 估值技术和输入值的变更
- 第三层次计量的未实现利得或损失对当期损益/其他综合收益的影响

## 第三十章 政府及民间非营利组织会计

### 一、政府会计概述

#### （一）核心特征："双功能、双基础、双报告"

| 维度 | 预算会计 | 财务会计 |
|------|----------|----------|
| 功能 | 反映预算收入、预算支出、预算结余 | 反映资产、负债、净资产、收入、费用 |
| 基础 | 收付实现制 | 权责发生制 |
| 报告 | 决算报告 | 财务报告 |

#### （二）平行记账的触发条件

对**纳入年度部门预算管理的现金收支业务**，在采用财务会计核算的同时应当进行预算会计核算；对于其他业务，仅需进行财务会计核算。

**现金范围**：库存现金、银行存款、其他货币资金、零余额账户用款额度、财政应返还额度、财政直接支付方式支付的款项

**不需预算会计处理的**：受托代理的现金、不纳入部门预算管理的暂收暂付款项

#### （三）政府会计要素

**预算会计要素**（3个）：预算收入、预算支出、预算结余

**财务会计要素**（5个）：资产、负债、净资产、收入、费用

### 二、政府单位特定业务的会计核算

#### （一）取得财政拨款收入

**财政直接支付方式**：

财务会计：
```
借：库存物品/固定资产/业务活动费用/单位管理费用等
  贷：财政拨款收入
```

预算会计：
```
借：行政支出/事业支出
  贷：财政拨款预算收入
```

**财政授权支付方式**：

收到授权支付到账通知书——财务会计：
```
借：零余额账户用款额度
  贷：财政拨款收入
```

预算会计：
```
借：资金结存——零余额账户用款额度
  贷：财政拨款预算收入
```

支用额度——财务会计：
```
借：库存物品/固定资产/业务活动费用等
  贷：零余额账户用款额度
```

预算会计：
```
借：行政支出/事业支出
  贷：资金结存——零余额账户用款额度
```

#### （二）发生日常支出

财务会计：
```
借：业务活动费用/单位管理费用
  贷：银行存款/零余额账户用款额度
```

预算会计：
```
借：行政支出/事业支出
  贷：资金结存——货币资金/零余额账户用款额度
```

#### （三）购入固定资产

财务会计：
```
借：固定资产
  贷：财政拨款收入/零余额账户用款额度/银行存款
```

预算会计：
```
借：行政支出/事业支出
  贷：财政拨款预算收入/资金结存——零余额账户用款额度/货币资金
```

#### （四）计提折旧（仅财务会计，预算会计不处理）

```
借：业务活动费用/单位管理费用
  贷：固定资产累计折旧
```

#### （五）取得事业收入

**财政专户返还方式**：

收到应上缴收入——财务会计：
```
借：银行存款
  贷：应缴财政款
```

上缴财政专户——财务会计：
```
借：应缴财政款
  贷：银行存款
```

收到财政专户返还——财务会计：
```
借：银行存款
  贷：事业收入
```

预算会计：
```
借：资金结存——货币资金
  贷：事业预算收入
```

**应收款方式**：

确认应收——仅财务会计：
```
借：应收账款
  贷：事业收入
```

收到款项——财务会计：
```
借：银行存款
  贷：应收账款
```

预算会计：
```
借：资金结存——货币资金
  贷：事业预算收入
```

#### （六）发放职工薪酬

财务会计——计提：
```
借：业务活动费用/单位管理费用
  贷：应付职工薪酬
```

财务会计——发放：
```
借：应付职工薪酬
  贷：银行存款/零余额账户用款额度
```

预算会计（发放时）：
```
借：行政支出/事业支出
  贷：资金结存——货币资金/零余额账户用款额度
```

#### （七）年末结转

**财务会计结转**：

收入结转本期盈余：
```
借：财政拨款收入/事业收入/其他收入等
  贷：本期盈余
```

费用结转本期盈余：
```
借：本期盈余
  贷：业务活动费用/单位管理费用等
```

本期盈余转入本年盈余分配：
```
借：本期盈余（或贷方）
  贷：本年盈余分配（或借方）
```

本年盈余分配转入累计盈余：
```
借：本年盈余分配（或贷方）
  贷：累计盈余（或借方）
```

**预算会计结转**：

财政拨款预算收入结转：
```
借：财政拨款预算收入
  贷：财政拨款结转——本年收支结转
```

财政拨款支出结转：
```
借：财政拨款结转——本年收支结转
  贷：行政支出/事业支出（财政拨款支出）
```

非财政非专项收入结转：
```
借：事业预算收入/其他预算收入等（非专项）
  贷：其他结余
```

非财政非专项支出结转：
```
借：其他结余
  贷：事业支出/其他支出等（非财政非专项）
```

事业单位将其他结余转入非财政拨款结余分配：
```
借：其他结余
  贷：非财政拨款结余分配
```

#### （八）确认应收款项（仅财务会计）

```
借：应收账款
  贷：事业收入/租金收入/其他收入
```

#### （九）收到应收款项（双分录）

财务会计：
```
借：银行存款
  贷：应收账款
```

预算会计：
```
借：资金结存——货币资金
  贷：事业预算收入/经营预算收入等
```

#### （十）接受捐赠

**接受现金捐赠**（双分录）：

财务会计：
```
借：银行存款
  贷：捐赠收入
```

预算会计：
```
借：资金结存——货币资金
  贷：其他预算收入——捐赠预算收入
```

**接受非现金资产捐赠**（仅财务会计，除非支付相关税费）：

财务会计：
```
借：库存物品/固定资产等                XX（确定的成本）
  贷：银行存款                          XX（相关税费、运输费）
      捐赠收入                          XX（差额）
```

预算会计（仅对支付的相关税费）：
```
借：其他支出
  贷：资金结存——货币资金
```

### 三、民间非营利组织会计

#### （一）基本特征

民间非营利组织同时具备三个特征：
1. **不以营利为目的**：为公益目的或其他非营利目的成立（不排除因提供服务获取收入）
2. **资源提供者不取得经济回报**：出资者不能从组织中获取回报
3. **资源提供者不保留或享有财产权利**：投入后不再保留资产出售、转让、处置权及清算时剩余财产分配权

**会计特点**：
- 以权责发生制为核算基础
- 引入公允价值计量基础（解决捐赠资产计量问题）
- 会计要素不包括所有者权益和利润，设置净资产要素
- 净资产分为**限定性净资产**和**非限定性净资产**

**限定性净资产**：资产或其经济利益的使用受到资产提供者或法律法规设置的时间限制或（和）用途限制
**非限定性净资产**：除限定性净资产以外的其他净资产

#### （二）接受捐赠

**接受非限定性捐赠**：
```
借：银行存款/固定资产/无形资产等
  贷：捐赠收入——非限定性收入
```

**接受限定性捐赠**：
```
借：银行存款/固定资产/无形资产等
  贷：捐赠收入——限定性收入
```

**限定性捐赠收入的限制在确认收入当期解除的**：
```
借：捐赠收入——限定性收入
  贷：捐赠收入——非限定性收入
```

**期末结转捐赠收入**：
```
借：捐赠收入——限定性收入
  贷：限定性净资产

借：捐赠收入——非限定性收入
  贷：非限定性净资产
```

#### （三）限定性净资产转为非限定性净资产

当限制已完全解除时（时间限制到期且用途限制解除）：
```
借：限定性净资产
  贷：非限定性净资产
```

#### （四）业务活动成本

发生业务活动成本：
```
借：业务活动成本——限定性费用/非限定性费用
  贷：银行存款/存货/应付账款等
```

期末结转：
```
借：限定性净资产
  贷：业务活动成本——限定性费用

借：非限定性净资产
  贷：业务活动成本——非限定性费用
```

#### （五）商品销售收入

**确认条件**（同时满足）：
1. 已将商品所有权上的主要风险和报酬转移给购货方
2. 既没有保留通常与所有权相联系的继续管理权，也没有对已售出商品实施控制
3. 与交易相关的经济利益能够流入
4. 相关的收入和成本能够可靠计量

**会计分录**：
```
借：银行存款/应收账款
  贷：商品销售收入——非限定性收入
```

同时结转成本：
```
借：业务活动成本——非限定性费用
  贷：存货
```

期末结转：
```
借：商品销售收入——非限定性收入
  贷：非限定性净资产
```

#### （六）收入分类规则

| 收入类型 | 通常属性 | 说明 |
|----------|----------|------|
| 捐赠收入 | 视是否设限 | 有时间/用途限制→限定性；无限制→非限定性 |
| 政府补助收入 | 视是否设限 | 同上 |
| 会费收入 | 非限定性 | 除非资产提供者设置了限制 |
| 提供服务收入 | 非限定性 | 除非资产提供者设置了限制 |
| 商品销售收入 | 非限定性 | 除非资产提供者设置了限制 |
| 投资收益 | 非限定性 | 除非资产提供者设置了限制 |

> 本章无传统意义上的会计分录考核，重点在于平行记账规则和民间非营利组织净资产分类。

## 五大心智模型

### 1. 实质重于形式

会计处理应反映交易的经济实质，而非仅依据法律形式。

**判断标准**：
- 是否符合资产/负债/权益的定义？
- 控制权/风险报酬是否真正转移？
- 交易是否具有商业实质？
- 价格是否公允，是否反映正常市场交易条件？

**应用场景**：收入确认总额法vs净额法、租赁主要责任人vs代理人、企业合并同一控制vs非同一控制、债务重组重组收益vs权益性交易。

### 2. 监管导向

**优先级**：《监管规则适用指引——会计类》> 《企业会计准则解释》> 准则原文 > 理论推断。

### 3. 权益性交易边界识别

**两个条件**（同时满足才是权益性交易）：
- 价格不公允（缺乏商业实质）
- 交易方属于同一控制下的关联方

**应用场景**：集团内无偿划转、母公司豁免子公司债务、超低价转让股权、承债式收购中的债务承担。

### 4. 准则适用优先级

多个准则可能适用时，判断交易的核心性质，确定主要适用准则。

**判断原则**：分析交易主要目的和经济实质 → 识别主要风险和报酬 → 考虑准则排除条款 → 参考监管指引分类标准。

### 5. 经济实质穿透判断

透过交易安排的表面形式，分析其真实的经济后果和会计影响。

**穿透方法**：识别关键条款 → 分析各方权利义务 → 评估最终经济后果 → 判断是否存在规避准则的安排。

**应用场景**：结构化融资安排、明股实债分类、资产证券化出表判断。

## 十二条决策启发式

1. **谨慎性底线**：不确定是否满足确认条件时，倾向于不确认。收益确认从严，损失确认从快。
2. **分层判断**：复杂问题先明确问题本质 → 分情况讨论 → 给出操作性建议。
3. **实务可操作性**：理论与实务冲突时，基于重要性原则处理。优先给出具体账务处理方案（科目、借贷方向），而非仅给原则性判断。强调一贯性应用。
4. **分层补偿处理**：涉及违约补偿、价格调整时：先冲减已确认的直接损失 → 超出部分再按合同变更或其他准则处理。
5. **收入确认-控制权转移**：是否承担主要责任和风险？是否拥有自主定价权？是否主导履约过程？
6. **权益性交易-两个条件**：价格不公允 + 同一控制下关联方，两个条件同时满足。
7. **准则交叉-核心准则优先**：分析交易的主要目的 → 识别主要风险和报酬 → 参考监管指引分类标准。
8. **租赁业务-风险报酬承担**：是否承担存货风险？是否承担空置风险和市场租金涨跌风险？能否主导租赁业务？
9. **递延所得税-确认顺序**：优先确认可抵扣亏损 → 剩余额度按"转回时间早的优先，转回不确定性小的优先"分配。
10. **合并范围-实质控制**：超过50%表决权通常推定控制。50%以下需判断是否有实质控制权（协议安排、董事会席位等）。
11. **会计估计-避免后见之明**：不得用后续信息推翻前期合理估计。会计估计变更影响计入当期损益，前期差错需追溯调整。
12. **权益法核算-普通股股东视角**：权益法基础是投资方在"归属普通股股东的净利润和净资产"中的份额。永续债/优先股等"其他权益工具"不属于普通股股东权益。

## 回答准则问题的工作流

1. **判断问题类型**：准则解释类 / 实务判断类 / 复杂案例类 / 信息不足类
2. **查阅准则原文**：按上方准则查阅规则，知识库优先，需要原文佐证时联网
3. **分析实质**：法律形式 vs 经济实质，判断适用准则
4. **给出结论**：先结论，再理由，再补充说明。信息不足时明确说明所需信息，不强行给确定结论
