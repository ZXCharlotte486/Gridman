## 第六章 城市维护建设税法和烟叶税法

### 一、城市维护建设税

### （一）基本规则

| 要素       | 内容                                                                 |
| ---------- | -------------------------------------------------------------------- |
| 性质       | 增值税、消费税的附加税                                               |
| 纳税人     | 缴纳增值税、消费税的单位和个人                                       |
| 不征收情形 | 进口货物或境外单位和个人向境内销售服务、无形资产缴纳的增值税、消费税 |

### （二）税率

| 纳税人所在地       | 税率 |
| ------------------ | ---- |
| 市区               | 7%   |
| 县城、镇           | 5%   |
| 不在市区、县城或镇 | 1%   |

### （三）计税依据与应纳税额

```
计税依据 = 依法实际缴纳的增值税 + 依法实际缴纳的消费税

依法实际缴纳的增值税 = 应当缴纳的增值税(不含进口) + 增值税免抵税额 - 直接减免的增值税 - 留抵退税额

应纳税额 = 计税依据 × 适用税率
```

**注意：**

- 先征后返、先征后退、即征即退退还的增值税/消费税，不退还城建税（另有规定除外）
- 出口退还增值税/消费税的，不退还城建税
- 留抵退税额仅允许在一般计税方法确定的计税依据中扣除

### （四）主要优惠

- 2023.1.1~2027.12.31：增值税小规模纳税人、小型微利企业和个体工商户减半征收

### 二、教育费附加和地方教育附加

| 项目         | 征收比率 | 计征依据                    |
| ------------ | -------- | --------------------------- |
| 教育费附加   | 3%       | 依法实际缴纳的增值税+消费税 |
| 地方教育附加 | 2%       | 依法实际缴纳的增值税+消费税 |

计征依据与城建税一致。进口不征收。2023.1.1~2027.12.31小规模纳税人、小型微利企业和个体工商户减半征收。

### 三、烟叶税

| 要素             | 内容                                      |
| ---------------- | ----------------------------------------- |
| 纳税人           | 收购烟叶的单位                            |
| 征税范围         | 烤烟叶、晾晒烟叶                          |
| 税率             | 20%                                       |
| 计税依据         | 实际支付价款总额 = 收购价款 ×（1 + 10%） |
| 纳税义务发生时间 | 收购烟叶的当日                            |
| 纳税期限         | 按月计征，次月15日内申报缴纳              |

```
应纳税额 = 收购价款 × (1 + 10%) × 20%
```

## 第七章 关税法和船舶吨税法

### 一、关税基本要素

| 要素     | 内容                                                   |
| -------- | ------------------------------------------------------ |
| 征税对象 | 准许进出口的货物、进境物品                             |
| 纳税人   | 进口货物收货人、出口货物发货人、进境物品携带人或收件人 |
| 征收机关 | 海关                                                   |

### 二、进口关税税率分类

| 税率类型   | 适用范围                                                                              |
| ---------- | ------------------------------------------------------------------------------------- |
| 最惠国税率 | WTO成员国、与我国签订最惠国待遇条约的国家/地区、原产于我国境内的进口货物              |
| 协定税率   | 与我国签订含关税优惠条款国际条约/协定的国家/地区（如RCEP）                            |
| 特惠税率   | 我国给予特殊关税优惠安排的最不发达国家（2024.12.1起对建交最不发达国家100%税目零关税） |
| 普通税率   | 上述以外的其他国家/地区，或原产地不明的货物                                           |
| 暂定税率   | 在最惠国税率基础上对特定商品实施的更优惠税率（按年度制定）                            |
| 配额税率   | 实行关税配额管理的进口货物，配额内适用                                                |

**税率适用优先规则：**

- 适用最惠国税率的有暂定税率→用暂定税率
- 适用协定税率的有暂定税率→从低适用
- 适用特惠税率的有暂定税率→从低适用
- 适用普通税率的→不适用暂定税率

### 三、进口货物完税价格

### （一）成交价格法（CIF价格）

```
完税价格 = 成交价格 + 运抵境内输入地点起卸前的运输及相关费用 + 保险费
```

**应计入完税价格的项目：**

- 买方负担的佣金（购货佣金除外）和经纪费
- 买方负担的容器费用、包装材料和劳务费用
- 买方免费或低价提供的材料/部件/工具/模具/工程设计等
- 买方应付的特许权使用费
- 卖方从买方进口后销售/处置/使用所得中获得的收益

**不计入完税价格的项目：**

- 进口后的建设、安装、装配、维修费用（保修费除外）
- 起卸后的运输费、保险费
- 进口关税及进口环节代征税
- 境内复制费用、境内外技术培训及境外考察费用
- 符合条件的利息费用

**保险费确定：** 无法确定时按 (货价 + 运费) × 3‰ 计算

### （二）海关估价方法（依次适用）

1. 相同货物成交价格法
2. 类似货物成交价格法
3. 倒扣价格法（可与第4项互换顺序）
4. 计算价格法
5. 合理方法

### 四、出口货物完税价格（FOB价格）

```
完税价格 = 成交价格（含运至境内输出地点装载前的运输费、保险费）
```

**不计入：** 出口关税；单独列明的装载后运输费和保险费

### 五、关税应纳税额计算

```
从价计征：应纳税额 = 完税价格 × 关税税率
从量计征：应纳税额 = 货物数量 × 定额税率
复合计征：应纳税额 = 完税价格 × 比例税率 + 货物数量 × 定额税率
```

### 六、关税减免税货物补税

```
补税的完税价格 = 原进口时海关审定价格 × [1 - 已使用月数 ÷ (监管年限 × 12)]
```

监管年限：船舶/飞机8年；机动车辆6年；其他货物3年。

### 七、跨境电商零售进口税收政策

| 要素                              | 规定                               |
| --------------------------------- | ---------------------------------- |
| 单次交易限值                      | 5,000元                            |
| 年度交易限值                      | 26,000元                           |
| 限值内关税                        | 暂设为0                            |
| 限值内增值税/消费税               | 按法定应纳税额的70%征收            |
| 超5,000元但≤26,000元且仅一件商品 | 按货物税率全额征收关税和进口环节税 |

### 八、进口税收优惠延续政策（"十五五"期间2026-2030）

一揽子进口税收优惠政策延续至2030年12月31日，覆盖以下领域：

| 领域 | 优惠内容 |

| 跨境电商出口退货 | 跨境电商出口退运商品税收减免政策延续 |

### 八、关税征收管理要点

- 纳税期限：完成申报之日起15日内缴纳
- 滞纳金：按日加收万分之五
- 补征/追征期限：自缴纳税款之日起3年内（漏征的自货物放行之日起3年内）
- 退税申请：多缴税款自缴纳之日起3年内申请
- 税率适用日期：完成申报之日实施的税率

### 九、船舶吨税

### （一）基本规则

| 要素     | 内容                                      |
| -------- | ----------------------------------------- |
| 征税范围 | 自境外港口进入境内港口的船舶              |
| 计税依据 | 船舶净吨位                                |
| 税率类型 | 优惠税率（中国籍/最惠国待遇国）和普通税率 |
| 执照期限 | 30日、90日、1年（每次申报时选择）         |
| 征收机关 | 海关                                      |

### （二）税率表（元/净吨）

| 净吨位        | 普通税率(1年/90日/30日) | 优惠税率(1年/90日/30日) |
| ------------- | ----------------------- | ----------------------- |
| ≤2,000       | 12.6 / 4.2 / 2.1        | 9.0 / 3.0 / 1.5         |
| 2,000~10,000  | 24.0 / 8.0 / 4.0        | 17.4 / 5.8 / 2.9        |
| 10,000~50,000 | 27.6 / 9.2 / 4.6        | 19.8 / 6.6 / 3.3        |
| >50,000       | 31.8 / 10.6 / 5.3       | 22.8 / 7.6 / 3.8        |

**特殊规则：**

- 拖船：按发动机功率每千瓦折合0.67净吨，按相同净吨位税率的50%计征
- 非机动驳船：按相同净吨位税率的50%计征
- 游艇（无净吨位证明）：按发动机功率每千瓦折合0.05净吨

```
应纳税额 = 船舶净吨位 × 适用定额税率
```

### （三）免税船舶

1. 应纳税额在50元以下的船舶
2. 初次进口到港的空载船舶（购买/受赠/继承取得）
3. 执照期满后24小时内不上下客货的船舶
4. 非机动船舶（不含非机动驳船）
5. 捕捞、养殖渔船
6. 避难、防疫隔离、修理、改造、终止运营或拆解且不上下客货的船舶
7. 军队、武装警察部队专用或征用的船舶
8. 警用船舶
9. 外国驻华使领馆、国际组织驻华代表机构的船舶

### （四）征收管理

- 纳税义务发生时间：应税船舶进入港口的当日
- 缴纳期限：海关填发缴款凭证之日起15日内
- 补征期限：应缴税款之日起1年内；因违规造成的追征3年

## 第八章 资源税法和环境保护税法

### 第一节 资源税

#### 一、纳税人

- 在中国领域及管辖的其他海域**开发应税资源**的单位和个人
- 进口矿产品和盐**不征**资源税；出口也**不免退**
- 自用应税产品用于非货币性资产交换、捐赠、偿债、投资等→**视同销售纳税**
- 自采原矿用于连续生产应税产品→移送环节**不缴**资源税

#### 二、税目与税率

资源税5大类税目：能源矿产、金属矿产、非金属矿产、水气矿产、盐。

| 序号 | 税目                       | 征税对象   | 税率                    |
| ---- | -------------------------- | ---------- | ----------------------- |
| 1    | 原油                       | 原矿       | **6%**（固定）    |
| 2    | 天然气/页岩气/天然气水合物 | 原矿       | **6%**（固定）    |
| 3    | 煤                         | 原矿或选矿 | 2%~10%                  |
| 4    | 煤成(层)气                 | 原矿       | 1%~2%                   |
| 5    | 铀、钍                     | 原矿       | **4%**（固定）    |
| 8    | 铁、锰、铬、钒、钛         | 原矿或选矿 | 1%~9%                   |
| 11   | 钨                         | 选矿       | **6.5%**（固定）  |
| 12   | 钼                         | 选矿       | **8%**（固定）    |
| 15   | 轻稀土                     | 选矿       | 7%~12%                  |
| 16   | 中重稀土                   | 选矿       | **20%**（固定）   |
| 27   | 砂石                       | 原矿或选矿 | 1%~5%或0.1~5元/吨(m³) |
| 31   | 钠盐/钾盐/镁盐/锂盐        | 选矿       | 3%~15%                  |

> 战略资源（原油、天然气、中重稀土、钨、钼）实行**固定税率**，其余为幅度税率。

#### 三、计税依据与应纳税额

**从价定率（主）：**

```
应纳税额 = 销售额 × 适用税率
```

- 销售额 = 全部价款（不含增值税），可扣除运杂费（有合法凭据）

**从量定额（辅）：**

```
应纳税额 = 课税数量 × 单位税额
```

- 适用：地热、石灰岩、其他粘土、砂石、矿泉水、天然卤水

**组成计税价格（无销售额时）：**

```
组成计税价格 = 成本 × (1 + 成本利润率) ÷ (1 - 资源税税率)
```

**外购应税产品扣减：**

```
准予扣减的外购应税产品购进金额 = 外购原矿购进金额 × (本地区原矿适用税率 ÷ 本地区选矿产品适用税率)
```

#### 四、减免税

| 优惠类型           | 具体情形                                                          |
| ------------------ | ----------------------------------------------------------------- |
| **免征**     | 开采原油及油田范围内加热用原油/天然气；安全生产抽采的煤成(层)气   |
| **减征20%**  | 低丰度油气田                                                      |
| **减征30%**  | 高含硫天然气（≥30g/m³）；三次采油；深水油气田（水深>300m）      |
| **减征40%**  | 稠油（黏度≥50毫帕/秒或密度≥0.92g/cm³）；高凝油（凝固点>40°C） |
| **减征30%**  | 衰竭期矿山（设计开采>15年，剩余储量<20%或剩余年限≤5年）          |
| **减征50%**  | 充填开采置换出来的煤炭（至2027.12.31）                            |
| **减征30%**  | 页岩气（按6%税率减征30%，至2027.12.31）                           |
| **减半征收** | 增值税小规模纳税人/小型微利企业/个体工商户（至2027.12.31）        |

- 省级政府可决定减免：意外事故/自然灾害重大损失；共伴生矿/低品位矿/尾矿
- 同一应税产品符合两项以上减征优惠的，只能选择一项执行

#### 五、征收管理

- **纳税义务发生时间**：收讫销售款或取得索取销售款凭据的当日；自用为移送当日
- **纳税期限**：按月或按季，自期满15日内申报缴纳；也可按次
- **纳税地点**：矿产品开采地或海盐生产地

#### 六、水资源税（2024.12.1起全面实施）

- **纳税人**：直接取用地表水或地下水的单位和个人
- **计征方式**：从量计征

```
应纳税额 = 实际取用水量 × 适用税额
```

- 地下水税额 > 地表水税额
- 水力发电：应纳税额 = 实际发电量 × 适用税额（最高不超过0.008元/千瓦时）
- 城市公共供水：实际取用水量 × (1 - 合理漏损率) × 适用税额

**免征/减征：**

- 免征：规定限额内农业生产取用水；军队/武警/消防非公共管网取水；抽水蓄能发电；采油(气)排水回注
- 减征20%：工业用水前一年度效率达国家先进值

### 第二节 环境保护税

#### 一、纳税人

- 直接向环境排放应税污染物的企业事业单位和其他生产经营者
- **不缴纳**情形：向污水/垃圾集中处理场所排放；在合规设施贮存/处置固废

#### 二、税目税额表

| 税目                             | 计税单位   | 税额             |
| -------------------------------- | ---------- | ---------------- |
| 大气污染物                       | 每污染当量 | 1.2~12元         |
| 水污染物                         | 每污染当量 | 1.4~14元         |
| 固体废物-煤矸石                  | 每吨       | 5元              |
| 固体废物-尾矿                    | 每吨       | 15元             |
| 固体废物-危险废物                | 每吨       | **1000元** |
| 固体废物-冶炼渣/粉煤灰/炉渣/其他 | 每吨       | 25元             |
| 噪声-超标1~3分贝                 | 每月       | 350元            |
| 噪声-超标4~6分贝                 | 每月       | 700元            |
| 噪声-超标7~9分贝                 | 每月       | 1400元           |
| 噪声-超标10~12分贝               | 每月       | 2800元           |
| 噪声-超标13~15分贝               | 每月       | 5600元           |
| 噪声-超标16分贝以上              | 每月       | 11200元          |

#### 三、计税依据与计算

```
污染当量数 = 该污染物排放量 ÷ 该污染物的污染当量值
应纳税额 = 污染当量数 × 适用税额
```

- 大气污染物：每一排放口按污染当量数从大到小排序，对**前三项**征税
- 水污染物：第一类按**前五项**征税，其他类按**前三项**征税
- 固体废物排放量 = 产生量 - 综合利用量 - 贮存量 - 处置量

#### 四、税收减免

- **暂免**：农业生产排放；流动污染源（机动车/船舶/航空器）；城乡污水/垃圾集中处理场所不超标排放
- **减按75%**：浓度值低于排放标准30%
- **减按50%**：浓度值低于排放标准50%

#### 五、征收管理

- 按月计算，**按季**申报缴纳，季度终了15日内
- 纳税地点：应税污染物排放地

## 第九章 城镇土地使用税法和耕地占用税法

### 第一节 城镇土地使用税

#### 一、基本规定

- **纳税人**：城市/县城/建制镇/工矿区内使用土地的单位和个人
- **计税依据**：实际占用的土地面积（平方米）
- **地下建筑**：暂按应征税款的**50%**征收

#### 二、税率（定额税率）

| 地区               | 年税额（元/㎡） |
| ------------------ | --------------- |
| 大城市             | 1.5~30          |
| 中等城市           | 1.2~24          |
| 小城市             | 0.9~18          |
| 县城/建制镇/工矿区 | 0.6~12          |

```
全年应纳税额 = 实际占用应税土地面积(㎡) × 适用税额
```

#### 三、主要优惠

- 免征：国家机关/军队/人民团体自用；财政拨付事业经费单位自用；宗教寺庙/公园/名胜古迹自用；市政公共用地；直接用于农林牧渔生产用地
- 开山填海整治/改造废弃土地：免征5~10年
- 物流企业大宗商品仓储设施用地：减按50%（至2027.12.31）
- 小规模纳税人/小型微利企业/个体工商户：减半征收（至2027.12.31）

#### 四、纳税义务发生时间

- 购置新建商品房：交付使用之**次月**起
- 购置存量房：权属登记签发证书之**次月**起
- 新征耕地：批准征收之日起满**1年**开始
- 新征非耕地：批准征收**次月**起

### 第二节 耕地占用税

#### 一、基本规定

- **纳税人**：占用耕地建设建筑物/构筑物或从事非农业建设的单位和个人
- **征税范围**：耕地、园地、林地、草地、农田水利用地、养殖水面、渔业水域滩涂
- **一次性征收**

#### 二、税率（地区差别定额税率）

| 人均耕地    | 税额（元/㎡） |
| ----------- | ------------- |
| ≤1亩       | 10~50         |
| >1亩且≤2亩 | 8~40          |
| >2亩且≤3亩 | 6~30          |
| >3亩        | 5~25          |

- 占用**基本农田**：加按**150%**征收
- 人均耕地<0.5亩地区：可适当提高，但不超过适用税额的50%

```
应纳税额 = 应税土地面积(㎡) × 适用税额
占用基本农田：应纳税额 = 应税土地面积 × 适用税额 × 150%
```

#### 三、减免

- **免征**：军事设施；学校/幼儿园/社会福利机构/医疗机构；农村烈士遗属/因公牺牲军人遗属/残疾军人/低保户新建自用住宅（规定标准内）
- **减按2元/㎡**：铁路/公路线路、飞机场跑道/停机坪、港口/航道/水利工程
- **减半**：农村居民规定标准内新建自用住宅

## 第十章 房产税法、契税法和土地增值税法

### 第一节 房产税

#### 一、纳税人与征税范围

- 产权所有人缴纳；出典的由承典人缴纳
- 征税范围：城市/县城/建制镇/工矿区的房产
- 房地产开发企业商品房出售前不征；已使用/出租/出借的应征

#### 二、税率与计税依据

| 计征方式                                    | 计税依据              | 税率           |
| ------------------------------------------- | --------------------- | -------------- |
| **从价计征**                          | 房产原值×(1-10%~30%) | **1.2%** |
| **从租计征**                          | 租金收入              | **12%**  |
| 个人出租住房                                | 租金收入              | **4%**   |
| 企事业单位向个人/规模化住房租赁企业出租住房 | 租金收入              | **4%**   |

```
从价：应纳税额 = 房产原值 × (1 - 扣除比例) × 1.2%
从租：应纳税额 = 租金收入 × 12%（或4%）
```

**房产原值注意事项：**

- 应包含地价（为取得土地使用权支付的价款、开发成本等）
- 宗地容积率<0.5的：按房产建筑面积的2倍计算土地面积确定计入地价
- 地下建筑自用：工业用途按原价50%~60%作为应税原值；商业用途按70%~80%

#### 三、主要优惠

- 免征：国家机关/军队/人民团体自用；财政拨付事业经费单位自用；宗教寺庙/公园/名胜古迹自用；个人非营业用房产
- 公租房免征；小规模纳税人/小型微利企业/个体工商户减半（至2027.12.31）

### 第二节 契税

#### 一、纳税人与征税范围

- **纳税人**：承受方（买方/受赠方）
- 征税范围：土地使用权出让/转让；房屋买卖/赠与/互换

#### 二、税率与计税依据

- 税率幅度：**3%~5%**
- 计税依据：成交价格（不含增值税）
- 互换：差额为计税依据，由支付差额方缴纳；等价互换计税依据为零

```
应纳税额 = 计税依据 × 税率
```

#### 三、个人住房优惠（重要）

| 情形           | 面积    | 税率           |
| -------------- | ------- | -------------- |
| 家庭唯一住房   | ≤140㎡ | **1%**   |
| 家庭唯一住房   | >140㎡  | **1.5%** |
| 家庭第二套住房 | ≤140㎡ | **1%**   |
| 家庭第二套住房 | >140㎡  | **2%**   |

#### 四、其他免征情形

- 婚姻关系存续期间夫妻之间变更权属；离婚分割共同财产
- 法定继承人继承
- 承受荒山/荒地/荒滩用于农林牧渔
- 企业改制重组（至2027.12.31）：整体改制（原投资主体>75%）、合并、分立、破产等

### 第三节 土地增值税（重点）

#### 一、纳税人与征税范围

- **纳税人**：转让国有土地使用权、地上建筑物及附着物并取得收入的单位和个人
- **征税范围**：转让国有土地使用权；房地产开发销售；存量房地产买卖
- **不征**：出租（未转让权属）；抵押期间；继承；赠与直系亲属或公益事业；代建行为；重新评估

#### 二、四级超率累进税率表

| 级数 | 增值额与扣除项目金额的比率 | 税率          | 速算扣除系数 |
| ---- | -------------------------- | ------------- | ------------ |
| 1    | 不超过50%                  | **30%** | 0            |
| 2    | 超过50%~100%               | **40%** | 5%           |
| 3    | 超过100%~200%              | **50%** | 15%          |
| 4    | 超过200%                   | **60%** | 35%          |

```
应纳税额 = 增值额 × 适用税率 - 扣除项目金额 × 速算扣除系数
增值额 = 转让收入(不含增值税) - 扣除项目金额
```

#### 三、扣除项目（房地产开发企业）

| 项目                         | 内容                                                                                   |
| ---------------------------- | -------------------------------------------------------------------------------------- |
| 1.取得土地使用权所支付的金额 | 地价款+有关费用                                                                        |
| 2.房地产开发成本             | 土地征用及拆迁补偿费、前期工程费、建安工程费、基础设施费、公共配套设施费、开发间接费用 |
| 3.房地产开发费用             | 能分摊利息并有金融机构证明：利息+(1+2)×5%以内；不能：(1+2)×10%以内                   |
| 4.与转让有关的税金           | 城建税、教育费附加、印花税                                                             |
| 5.加计扣除                   | (1+2)×**20%**（仅限房地产开发企业）                                             |

**旧房转让扣除：**

- 评估价格 = 重置成本价 × 成新度折扣率
- 无评估价但有购房发票：发票金额从购买年度起至转让年度止每年加计5%

#### 四、清算条件

**应清算：**

1. 项目全部竣工、完成销售
2. 整体转让未竣工决算项目
3. 直接转让土地使用权

**可要求清算：**

1. 已竣工验收，已转让面积占可售面积≥85%（或剩余已出租/自用）
2. 取得预售许可证满3年仍未销售完毕
3. 申请注销税务登记但未清算

#### 五、税收优惠

- 建造普通标准住宅出售，增值额未超过扣除项目金额**20%**的→**免征**
- 因国家建设需要依法征用/收回的房地产→免征
- 个人销售住房→**暂免**
- 企业改制重组（至2027.12.31）：整体改制/合并/分立/以房地产作价入股→暂不征收（不适用于房地产开发企业）

#### 六、土地增值税计算示例

**示例**：某房地产开发企业销售商品房，取得收入10,000万元（不含增值税），相关扣除项目：

- 取得土地使用权支付金额：2,000万元
- 房地产开发成本：3,000万元
- 房地产开发费用：能分摊利息200万元（有金融机构证明），当地规定其他费用扣除比例5%
- 与转让有关的税金：100万元

```
扣除项目计算：
① 取得土地使用权支付金额 = 2,000万
② 房地产开发成本 = 3,000万
③ 房地产开发费用 = 利息200万 + (2,000+3,000)×5% = 200 + 250 = 450万
④ 与转让有关的税金 = 100万
⑤ 加计扣除 = (2,000+3,000)×20% = 1,000万

扣除项目金额合计 = 2,000 + 3,000 + 450 + 100 + 1,000 = 6,550万元
增值额 = 10,000 - 6,550 = 3,450万元
增值率 = 3,450 ÷ 6,550 = 52.67%（适用第二级：税率40%，速算扣除系数5%）

应纳土地增值税 = 3,450 × 40% - 6,550 × 5% = 1,380 - 327.5 = 1,052.5万元
```

#### 六、预征管理

- 东部地区预征率下限**1.5%**，中部和东北**1%**，西部**0.5%**
- 预征计征依据 = 预收款 ÷ (1 + 增值税适用税率或征收率)

## 第十一章 车辆购置税法、车船税法和印花税法

### 第一节 车辆购置税

#### 一、基本规定

- **纳税人**：在中国境内购置汽车、有轨电车、汽车挂车、排气量>150ml摩托车的单位和个人
- **税率**：**10%**（统一比例税率）
- **一次性征收**

#### 二、计税依据

| 取得方式           | 计税价格                                                 |
| ------------------ | -------------------------------------------------------- |
| 购买自用           | 发票电子信息中不含增值税价                               |
| 进口自用           | 关税完税价格+关税+消费税                                 |
| 自产自用           | 同类应税车辆销售价格（不含增值税）；无同类→组成计税价格 |
| 受赠/获奖          | 相关凭证载明价格（不含增值税）                           |
| 换电模式新能源汽车 | 不含动力电池的不含税价                                   |

```
应纳税额 = 计税价格 × 10%
进口：应纳税额 = (关税完税价格 + 关税 + 消费税) × 10%
```

#### 三、税收优惠

- 免税：外国使领馆/国际组织自用车辆；军队/武警装备订货计划车辆；应急救援专用号牌消防车辆；设有固定装置的非运输专用作业车辆；城市公交企业公共汽电车辆
- **新能源汽车**：2024.1.1~2025.12.31免征（每辆乘用车免税额≤3万元）；2026.1.1~2027.12.31减半征收（每辆减税额≤1.5万元）
- 挂车：减半征收（至2027.12.31）

#### 四、征收管理

- 纳税义务发生时间：购置应税车辆的当日
- 申报期限：纳税义务发生之日起**60日**内
- 退税：应退税额 = 已纳税额 × (1 - 使用年限 × 10%)

### 第二节 车船税

#### 一、基本规定

- **纳税人**：车船的所有人或管理人
- **按年申报**，分月计算，一次性缴纳
- 扣缴义务人：从事机动车交强险业务的保险机构

#### 二、税目税额表（定额税率）

| 税目           | 计税单位     | 年基准税额                |
| -------------- | ------------ | ------------------------- |
| 乘用车≤1.0L   | 每辆         | 60~360元                  |
| 乘用车1.0~1.6L | 每辆         | 300~540元                 |
| 乘用车1.6~2.0L | 每辆         | 360~660元                 |
| 乘用车2.0~2.5L | 每辆         | 660~1200元                |
| 乘用车2.5~3.0L | 每辆         | 1200~2400元               |
| 乘用车3.0~4.0L | 每辆         | 2400~3600元               |
| 乘用车>4.0L    | 每辆         | 3600~5400元               |
| 商用客车       | 每辆         | 480~1440元                |
| 商用货车       | 整备质量每吨 | 16~120元                  |
| 挂车           | 整备质量每吨 | 按货车税额的**50%** |
| 摩托车         | 每辆         | 36~180元                  |
| 机动船舶       | 净吨位每吨   | 3~6元                     |
| 游艇           | 艇身长度每米 | 600~2000元                |

```
新购车船当年应纳税额 = (年应纳税额 ÷ 12) × 应纳税月份数
```

#### 三、税收优惠

- 免征：捕捞/养殖渔船；军队/武警专用车船；警用车船；消防车辆/船舶
- **减半**：节能汽车（排量≤1.6L符合燃料消耗标准的乘用车等）
- **免征**：新能源汽车（纯电动商用车、插电式混合动力汽车、燃料电池商用车）；纯电动乘用车和燃料电池乘用车**不属于征税范围**

### 第三节 印花税

#### 一、纳税人

- 在境内书立应税凭证、进行证券交易的单位和个人
- 证券交易印花税：对**出让方**征收，不对受让方征收

#### 二、税目税率表

| 税目                                | 税率                                                                      |
| ----------------------------------- | ------------------------------------------------------------------------- |
| 借款合同                            | 借款金额的**万分之0.5**                                             |
| 融资租赁合同                        | 租金的**万分之0.5**                                                 |
| 买卖合同                            | 价款的**万分之3**（不含个人书立的动产买卖合同）                     |
| 承揽合同                            | 报酬的**万分之3**                                                   |
| 建设工程合同                        | 价款的**万分之3**                                                   |
| 运输合同                            | 运输费用的**万分之3**（货运/多式联运，不含管道运输）                |
| 技术合同                            | 价款/报酬/使用费的**万分之3**                                       |
| 租赁合同                            | 租金的**千分之1**                                                   |
| 保管合同                            | 保管费的**千分之1**                                                 |
| 仓储合同                            | 仓储费的**千分之1**                                                 |
| 财产保险合同                        | 保险费的**千分之1**（不含再保险）                                   |
| 土地使用权出让书据                  | 价款的**万分之5**                                                   |
| 土地使用权/房屋等转让书据           | 价款的**万分之5**                                                   |
| 股权转让书据                        | 价款的**万分之5**                                                   |
| 商标/著作权/专利权/专有技术转让书据 | 价款的**万分之3**                                                   |
| 营业账簿                            | 实收资本+资本公积合计的**万分之2.5**                                |
| 证券交易                            | 成交金额的**千分之1**（自2023.8.28起**减半**→实际千分之0.5） |

#### 三、计税依据

- 应税合同：合同所列金额，**不包括列明的增值税税款**
- 未列明金额：按实际结算金额确定
- 同一凭证载有两个以上税目：分别列明→分别计税；未分别列明→从高适用

#### 四、主要优惠

- 免征：副本/抄本；农民/家庭农场/农民专业合作社购买农业生产资料或销售农产品的买卖合同；无息/贴息借款合同；财产赠与政府/学校/社会福利机构/慈善组织的产权转移书据；个人出租/承租住房的租赁合同；个人销售或购买住房
- 小规模纳税人/小型微利企业/个体工商户：减半征收（至2027.12.31，不含证券交易印花税）

#### 五、征收管理

- 纳税义务发生时间：书立应税凭证或完成证券交易的当日
- 按季/按年/按次计征；证券交易印花税按周解缴
- 未履行的合同/产权转移书据，已缴印花税**不予退还**

## 第十二章 国际税收税务管理实务（重点）

### 第一节 国际税收协定

#### 一、两大范本

- **《经合组织范本》（OECD范本）**：强调居民税收管辖权，促进发达国家间协定签订
- **《联合国范本》（UN范本）**：注重扩大收入来源国税收管辖权，促进发达国家与发展中国家间协定签订

#### 二、我国缔约情况

- 截至2025年7月底：与**111个**国家（地区）正式签署避免双重征税协定，108个已生效
- 与香港/澳门签署税收安排，与台湾签署税收协议

#### 三、典型条款

**（一）居民身份判定（双重居民身份加比规则）：**

1. 永久性住所 → 2. 重要利益中心 → 3. 习惯性居处 → 4. 国籍 → 5. 相互协商

- 非个人（公司等）：以**实际管理机构**所在国确定

**（二）常设机构判定：**

- 定义：企业进行全部或部分营业的**固定营业场所**
- 包括：管理场所、分支机构、办事处、工厂、作业场所、矿场/油井/气井等
- 建筑工地/安装工程：连续**6个月以上**构成常设机构
- 劳务活动：任何12个月中连续或累计超过**183天**构成常设机构
- **不构成**常设机构：仅从事储存/陈列/交付/采购/搜集情报等准备性或辅助性活动
- **非独立代理人**：有权并经常以企业名义签订合同→构成常设机构
- **独立代理人**：法律上和经济上独立于被代理企业，按常规进行自身业务→不构成

**（三）劳务所得：**

- 独立个人劳务：一般仅在居民国征税；但在来源国有固定基地或停留≥183天的，来源国可征税
- 非独立个人劳务（受雇所得）：同时满足以下三条件的，仅在居民国征税：
  1. 任何12个月中在来源国停留≤183天
  2. 报酬由非来源国居民的雇主支付
  3. 报酬不由雇主在来源国的常设机构/固定基地负担

#### 四、受益所有人认定

- 定义：对所得或所得据以产生的权利或财产具有所有权和支配权的人
- 不利因素：12个月内将50%以上所得支付给第三国居民；不构成实质性经营活动；来源国不征税或实际税率极低

### 第二节 非居民企业税收管理

#### 转让定价方法

1. **可比非受控价格法**（CUP）
2. **再销售价格法**
3. **成本加成法**
4. **交易净利润法**（TNMM）
5. **利润分割法**

### 第三节 境外所得税收管理

#### 境外所得税额抵免

```
抵免限额 = 中国境内外所得应纳税总额 × (来源于某国的应纳税所得额 ÷ 中国境内外应纳税所得总额)
```

- 分国不分项计算
- 超过抵免限额的部分：可在以后连续**5个纳税年度**结转抵免
- 简易办法：实际有效税率明显高于我国的→直接以25%计算的抵免限额作为可抵免税额

#### 一·拓展、BEPS 2.0 双支柱方案（2021-2026实施中）

> OECD/G20 包容性框架在 BEPS 1.0 基础上推出的国际税改方案，重塑跨国企业税收规则。

**支柱一（Pillar One）——征税权重新分配**

- **适用范围**：年合并收入 ≥ €200亿、利润率 > 10% 的最大型跨国企业（约 100 家）
- **核心规则**：将剩余利润（Amount A）的 25% 重新分配到市场国（用户/消费者所在国）
- **进展**：多边公约（MLC）签署进度缓慢，全球落地节奏滞后于支柱二

**支柱二（Pillar Two）——全球最低税 15%**

- **适用范围**：年合并收入 ≥ **€7.5亿** 的跨国企业集团（MNE Group）
- **核心规则——GloBE规则三层抽税顺序**：
  1. **QDMTT**（合格的国内最低补足税）：低税辖区先收齐补足税到15%
  2. **IIR**（收入纳入规则）：母公司辖区按"自上而下"补征
  3. **UTPR**（低税支付规则）：兄弟辖区/集团其他成员补征兜底
- **有效税率（ETR）计算**：按辖区计算，ETR = 调整后税负 / GloBE 收入；不足 15% 即触发补足税
- **实质性活动剔除（SBIE）**：有形资产账面 + 工资费用按一定比例（10年过渡期由 8%/10% 降至 5%）从 GloBE 税基中剔除

**全球落地进展（2024-2026）**：

| 辖区 | 实施时点 | 实施方式 |
|------|---------|---------|
| 欧盟成员国 | 2024.1.1 起 | IIR 全面施行，UTPR 2025.1.1 起 |
| 英国 | 2024.1.1 起 | IIR + DMTT 同步 |
| 韩国、日本、越南、澳大利亚、加拿大、新加坡 | 2024 起陆续施行 | IIR 优先 |
| 香港 | 2025.1.1 起 | IIR + DMTT |
| **中国大陆** | **2025.1.1 起** | **居民跨国集团需对其低税境外子公司适用 IIR**；同步引入 QDMTT 防止税收外流；财政部、税务总局已发布配套立法 |
| 美国 | 暂未直接采纳 | 现行 GILTI 体系与 GloBE 不完全等效，OECD 与美国仍在协调 |

**对中国跨境企业的影响**：

- **走出去企业**：在低税辖区（如新加坡、香港、爱尔兰、BVI、开曼等）的子公司，若 ETR < 15%，将在中国母公司层面被 IIR 补征至 15%；以往通过低税地结构享受的"组合税收红利"基本失效
- **外资在华企业**：中国法定税率 25%，但若享受高新 15%、西部大开发 15%、海南自贸港 15% 等优惠，加上加计扣除、研发费用加计、税收返还等，部分企业**实际 ETR 可能低于 15%**，会触发 QDMTT 补足税；优惠的实际价值需重新评估
- **合规要求**：在范围内集团需准备 GloBE 信息申报表（GIR）、辖区 ETR 计算底稿、SBIE 计算明细，工作量与转让定价同期资料相当
- **重要性阈值**：**€7.5亿 ≈ 56亿人民币**（按 2026 年汇率），低于此规模不在范围内，但接近阈值的集团应建立预警机制

**实务关注点**：

1. 范围内集团的合并财务报表准备工作量将显著加大（GloBE 调整与 IFRS/CAS 差异）
2. 税收激励政策可能从"税率优惠"转向"现金返还/补贴"形式（避免被 QDMTT 抵消）
3. 海南自贸港、西部大开发等区域优惠的有效性需结合企业整体规模重新评估
4. 出海架构（如新加坡控股）需重新设计——单纯的"低税地控股"模型已失效

**官方依据查询**：

- OECD GloBE 模型规则及行政指引：https://www.oecd.org/tax/beps/
- 中国财政部、税务总局支柱二配套法规：fgk.chinatax.gov.cn 搜索"全球反税基侵蚀"或"GloBE"
- 各辖区落地清单：建议查询 BDO/KPMG/EY 公开的 Pillar Two Tracker

#### 二、预约定价安排（APA）

- 定义：税务机关与纳税人就关联交易的定价原则和计算方法达成的事先约定
- 类型：单边APA、双边APA、多边APA
- 适用期限：一般3~5年，可申请续签

#### 三、受控外国企业（CFC）规则

- 适用条件：中国居民企业/居民个人控制的设立在实际税负明显偏低（<12.5%）的国家/地区的企业
- 后果：对无合理经营需要而不分配或减少分配的利润，视同分配，计入中国居民企业/个人当期收入

#### 四、资本弱化规则

- 关联方债资比例限制：金融企业**5:1**；其他企业**2:1**
- 超过部分的利息支出不得在计算应纳税所得额时扣除

#### 五、一般反避税规则

- 适用条件：企业实施的不具有合理商业目的的安排而减少其应纳税收入或所得额
- 税务机关有权按合理方法进行纳税调整
- 调整期限：自该业务发生的纳税年度起**10年**内

### 多辖区合规与跨境税务（实务指引）

#### 转让定价实务要点

- 关联交易类型：货物购销、劳务提供、无形资产使用、融通资金
- 同期资料准备：**主体文档+本地文档+国别报告**
- 可比性分析五步法：功能分析→可比对象筛选→可比性调整→方法选择→结果验证

#### 跨境架构常见税务风险

- 常设机构认定风险
- 受益所有人认定
- 间接转让中国应税财产
- 协定滥用（主要目的测试）

## 第十三章 税收征收管理法

### 一、税务登记管理

- 设立登记：自领取营业执照之日起**30日**内申报办理
- 变更登记：自变更之日起**30日**内申报办理
- 注销登记：应在注销工商登记**前**办理注销税务登记
- 外出经营报验登记：外出经营活动税收管理证明

### 二、账簿凭证管理

- 设置账簿时限：自领取营业执照或发生纳税义务之日起**15日**内
- 账簿/凭证保管期限：一般**10年**（另有规定除外）
- 发票管理：增值税专用发票由国务院税务主管部门指定企业印制

### 三、纳税申报管理

- 申报方式：直接申报、邮寄申报、数据电文申报（网络/APP等）
- 延期申报：经核准可延期，但应在纳税期内按上期或核定税额预缴

### 四、税款征收方式

| 方式         | 适用对象                                                             |
| ------------ | -------------------------------------------------------------------- |
| 查账征收     | 账簿/凭证/财务核算制度健全的纳税人                                   |
| 查定征收     | 生产经营规模较小、产品零星、税源分散、会计账册不健全的小型厂矿和作坊 |
| 查验征收     | 经营品种单一、经营地点/时间/来源不固定的纳税单位                     |
| 定期定额征收 | 无完整考核依据的小型纳税单位                                         |

### 五、税款征收制度

#### （一）滞纳金

```
滞纳金 = 滞纳税款 × 滞纳天数 × 万分之五
```

- 起止时间：缴纳期限届满次日起至实际缴纳之日止
- 可先缴欠税再缴滞纳金

#### （二）延期缴纳

- 条件：不可抗力导致较大损失；当期货币资金扣除应付工资/社保后不足以缴税
- 审批：省级税务局批准
- 期限：最长不超过**3个月**，同一笔税款不得滚动审批
- 批准延期内免予加收滞纳金

#### （三）税收保全措施

- 前提：从事生产经营的纳税人有逃避纳税义务行为
- 审批：经县以上税务局（分局）局长批准
- 措施：①冻结存款（相当于应纳税款金额）；②扣押/查封商品货物或其他财产
- 不适用：个人及其所扶养家属维持生活必需的住房和用品（不含机动车/金银饰品/古玩字画/豪华住宅/一处以外住房）
- 单价5000元以下其他生活用品不采取保全/强制措施

#### （四）税收强制执行措施

- 适用对象：从事生产经营的纳税人、扣缴义务人、纳税担保人
- 措施：①从存款中扣缴税款；②扣押/查封/拍卖/变卖抵缴税款
- 滞纳金同时强制执行

#### （五）税额核定

适用情形：不设账簿/未设账簿/擅自销毁账簿/账目混乱/逾期不申报/计税依据明显偏低无正当理由

#### （六）关联企业税收调整

- 独立交易原则（arm's length principle）
- 调整期限：一般**3年**内；特殊情况**10年**内
- 调整方法：可比非受控价格法、再销售价格法、成本加成法、其他合理方法

### 六、法律责任

| 违法行为               | 处罚                                                                                    |
| ---------------------- | --------------------------------------------------------------------------------------- |
| **偷税**         | 追缴税款/滞纳金，并处不缴或少缴税款**50%以上5倍以下**罚款；构成犯罪的追究刑事责任 |
| **逃避追缴欠税** | 追缴欠税/滞纳金，并处欠缴税款**50%以上5倍以下**罚款；构成犯罪的追究刑事责任       |
| **骗税**         | 追缴骗取的退税款，并处骗取税款**1倍以上5倍以下**罚款；构成犯罪的追究刑事责任      |
| **抗税**         | 追缴拒缴税款/滞纳金，并处拒缴税款**1倍以上5倍以下**罚款；构成犯罪的追究刑事责任   |

**偷税手段**：伪造/变造/隐匿/擅自销毁账簿凭证；在账簿上多列支出或不列/少列收入；经通知申报而拒不申报或虚假申报

### 七、纳税担保

- 方式：保证、抵押、质押
- 担保范围：税款+滞纳金
- 纳税担保人：以保证方式提供担保的保证人；以财产提供担保的第三人

### 税务风险识别（实务指引）

#### 高风险信号

- 收入确认时点与增值税纳税义务发生时间不一致
- 关联交易定价偏离市场水平
- 研发费用归集口径不符合加计扣除要求
- 发票管理不规范（虚开风险）
- 税负率异常波动

#### 风险应对

- 建立税务风险内控制度
- 定期进行税务健康检查
- 重大交易事前税务论证
- 保留完整的商业目的证据链

## 第十四章 税务行政法制

### 第一节 税务行政处罚

#### 一、处罚种类

1. 罚款
2. 没收违法所得/没收非法财物
3. 停止出口退税权
4. 吊销税务行政许可证件（如发票领购资格）

#### 二、处罚程序

| 程序类型           | 适用情形                                                   | 要点                                       |
| ------------------ | ---------------------------------------------------------- | ------------------------------------------ |
| **简易程序** | 对公民处以200元以下罚款；对法人/其他组织处以3000元以下罚款 | 当场作出处罚决定                           |
| **一般程序** | 其他情形                                                   | 调查→告知→听证（符合条件时）→决定→送达 |

#### 三、听证程序

- 适用条件：对公民罚款**>2000元**；对法人/其他组织罚款**>10000元**
- 申请期限：被告知之日起**5个工作日**内提出
- 税务机关应在收到听证申请后**15日**内举行听证

### 第二节 税务行政复议

#### 一、复议范围

**必经复议（复议前置）：**

- 对税务机关作出的**征税行为**不服（包括确认纳税主体/征税对象/征税范围/减免税/适用税率/计税依据/纳税环节/纳税期限/纳税地点/征收方式/税款征收/加收滞纳金/扣缴义务等）
- 纳税人必须先依照税务机关的纳税决定缴纳或者解缴税款及滞纳金或者提供相应的担保，然后可以依法申请行政复议

**可选复议：**

- 行政处罚、行政强制措施、行政强制执行、税务行政许可、资格认定等→可直接复议或直接诉讼

#### 二、复议管辖

- 对各级税务局作出的具体行政行为不服→向**其上一级税务局**申请复议
- 对税务所/稽查局作出的具体行政行为不服→向**其所属税务局**申请复议
- 对国家税务总局作出的具体行政行为不服→向**国家税务总局**申请复议

#### 三、复议程序与期限

| 环节                 | 期限                                           |
| -------------------- | ---------------------------------------------- |
| 申请复议             | 知道具体行政行为之日起**60日**内         |
| 复议机关受理         | 收到申请之日起**5日**内决定是否受理      |
| 作出复议决定         | 受理之日起**60日**内（经批准可延长30日） |
| 不服复议决定提起诉讼 | 收到复议决定书之日起**15日**内           |
| 复议期间不停止执行   | 原则上不停止具体行政行为的执行                 |

#### 四、复议决定类型

1. **维持**：具体行政行为认定事实清楚、证据确凿、适用依据正确、程序合法、内容适当
2. **撤销/变更/确认违法**：主要事实不清/证据不足/适用依据错误/违反法定程序/超越或滥用职权/明显不当
3. **责令履行**：被申请人不履行法定职责

### 第三节 税务行政诉讼

#### 一、受案范围

- 对税务机关作出的行政处罚不服
- 对税务机关作出的税收保全措施/强制执行措施不服
- 认为税务机关侵犯其合法经营自主权
- 对复议决定不服（征税行为须先经复议）

#### 二、管辖

- **一般管辖**：由最初作出具体行政行为的税务机关所在地人民法院管辖
- **级别管辖**：基层人民法院管辖第一审税务行政案件

#### 三、诉讼程序与期限

| 环节                   | 期限                                                                                           |
| ---------------------- | ---------------------------------------------------------------------------------------------- |
| 直接起诉（非复议前置） | 知道或应当知道作出行政行为之日起**6个月**内                                              |
| 经复议后起诉           | 收到复议决定书之日起**15日**内；复议机关逾期不作决定的，自复议期满之日起**15日**内 |
| 法院立案               | 收到起诉状之日起**7日**内决定是否立案                                                    |
| 一审判决               | 立案之日起**6个月**内（经批准可延长）                                                    |
| 上诉期限               | 判决书送达之日起**15日**内；裁定书送达之日起**10日**内                             |

#### 四、诉讼原则

- **被告负举证责任**：税务机关对其作出的具体行政行为负举证责任
- **不适用调解**：税务行政诉讼案件不适用调解（行政赔偿/补偿案件除外）
- **诉讼期间不停止执行**：原则上不停止具体行政行为的执行
- **司法变更权有限**：法院一般只作维持/撤销判决，对行政处罚显失公正的可以判决变更

## 2025年中下旬重要政策补充（知识库增补）

> 以下政策为2025年中下旬发布，CPA教材截稿时尚未纳入，经联网核实后补充。

### 一、外商利润再投资10%税收抵免

**文件依据**：
- 财政部 税务总局 商务部公告[2025]第2号（2025年6月27日发布）
- 国家税务总局公告[2025]第18号（2025年7月31日发布，实施细则）

**有效期**：2025年1月1日至2028年12月31日

**核心内容**：

境外投资者将从中国居民企业取得的利润直接再投资于境内，可享受再投资金额10%的企业所得税抵免，未用完的抵免额可结转至用完为止。

**适用条件**：
- 利润用于认缴出资、增加实缴资本或转增资本公积
- 投资持有期以商务部门出具的"利润再投资声明"载明日期为准

**抵免计算规则**：

| 要素 | 规定 |
|------|------|
| 可抵免税额来源 | 同一利润分配企业产生的应纳企业所得税（股息、利息、特许权使用费所得） |
| 收入时间要求 | 再投资日之后取得的所得 |
| 抵免比例 | 再投资金额×10%，或按税收协定更低税率（但投资收回时不再适用协定优惠税率） |
| 多次再投资 | 按分配企业分别核算 |
| 外币折算 | 按付款日人民币汇率中间价折算 |

**投资收回规则**（FIFO顺序）：
1. 已享受抵免的再投资
2. 符合条件但尚未使用抵免的再投资
3. 享受递延纳税但不符合抵免条件的再投资

同一企业多次再投资的，按再投资日期先后顺序收回。

**申报要求**：
- 申请抵免：提交"外商投资者再投资税收抵免信息报告"、"利润再投资声明"、"扣缴企业所得税报告"
- 投资收回申报：提交"利润再投资声明"、"外商投资者再投资税收抵免信息报告"
- 税务机关建立抵免台账，不当享受的追缴税款+加收滞纳金

**过渡规则**：
- 2028年12月31日前未用完的抵免余额可继续使用至用完
- 2025年1月1日至公告发布日之间的合规再投资可追溯适用（抵免仅适用于公告发布后的应纳税额）

### 二、互联网平台企业涉税信息报送规定

**文件依据**：国务院令第810号（2025年6月20日发布，即日施行）

**核心要求**：

| 要素 | 规定 |
|------|------|
| 适用主体 | 互联网平台企业（电商平台、网约车平台、直播平台、外卖平台等） |
| 报送义务 | 向税务机关报送平台内经营者和从业人员的涉税信息 |
| 代扣代缴 | 平台需为从业人员代扣代缴个人所得税 |
| 信息范围 | 经营者身份信息、交易金额、收入支付信息等 |

**对实务的影响**：
- 平台经济从业者（主播、骑手、网约车司机等）的个税征管将显著加强
- 平台企业需建立涉税信息采集和报送系统
- 与增值税法施行后"互联网平台内从业人员由平台代办申报"的规定形成配套

### 三、海南自贸港封关运作税收政策

**文件依据**：财关税[2025]12号（财政部、海关总署、税务总局联合发布）

**封关日期**：2025年12月18日

**政策框架**：

| 政策领域 | 主要规定 |
|----------|----------|
| "一线"进口（境外→海南） | 零关税政策：免征进口关税、进口环节增值税和消费税；适用主体为海南注册的独立法人企业和公共机构；零关税清单从约1,900个税目扩大至约6,600个；企业可自愿放弃免税但12个月内不得对同类货物重新申请 |
| 应税进口目录内货物 | 仍需缴纳进口关税、增值税和消费税 |
| "二线"出区（海南→内地） | 零关税货物及其加工制品进入内地需补缴进口关税、增值税和消费税（按进口料件计算）；已在进口或岛内流通环节缴纳过的不再重复征收 |
| 岛内流通 | 零关税进口货物在岛内销售、加工、使用不再额外征收进口环节税 |

**对企业的影响**：
- 在海南设立实体的企业可享受大幅扩大的零关税进口范围
- 加工贸易企业需建立"一线"/"二线"货物分账管理体系
- 内地企业从海南采购含进口料件的产品需关注补税义务

### 四、七部门鼓励外商投资企业境内再投资措施

**文件依据**：国家发改委、财政部、人民银行等七部门联合发布（2025年7月7日）

**主要措施**：
- 建立再投资项目数据库，符合条件的纳入外资重点项目清单
- 提供工业用地优惠价格、简化行业准入、税收优惠、进口设备免税
- 优化外汇管理：允许外币利润境内划转、简化登记手续
- 支持融资渠道：允许发行"熊猫债"（境外主体在中国发行的人民币债券）
