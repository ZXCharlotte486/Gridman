# 财务BP与经营分析

> 定位：帮助财务BP做经营分析、业务归因、决策支持
> 来源：实务经验提炼 + 行业研究
> 覆盖：经营分析、量价分析、预算差异、滚动预测、业务归因、行业指标、合同风控
> 性质：实操+判断

---

## 一、财务BP vs 传统财务

| 维度 | 传统财务 | 财务BP |
|------|---------|--------|
| 定位 | 事后核算、监督合规 | 事前预测 + 事中控制 + 事后复盘闭环 |
| 时间视角 | 过去（历史数据） | 未来（滚动预测、场景模拟） |
| 核心产出 | 报表、凭证、纳税申报 | 经营分析报告、ROI测算、定价建议、行动方案 |
| 沟通对象 | 税务局、审计师、外部监管 | 业务总监、产品经理、销售团队、CEO |
| 语言 | 会计语言（借贷/科目） | 业务语言（量价/渠道/客户/产品） |
| 价值体现 | 数字准确 | 影响决策 |

**BP 价值金字塔**（从基础到战略）：
1. 基础级：数据清洗、报表出具、流程标准化
2. 业务级：合同审核、定价支持、成本优化
3. 经营级：预算编制、滚动预测、经营分析会主持
4. 战略级：参与中长期规划、新业务可行性、并购财务尽调

## 二、经营分析框架

### 月度经营分析标准结构

```
1. 经营结果概览（1页）
   - 收入/利润/现金流 vs 预算 vs 去年同期
   - 红绿灯标识：达标/预警/未达标

2. 收入分析（量价拆分）
   - 总收入变动 = 量变动 + 价变动 + 结构变动
   - 按产品线/客户/渠道/区域拆分

3. 毛利分析（归因到驱动因素）
   - 毛利率变动 = 售价变动 + 成本变动 + 产品结构变动
   - 标准成本 vs 实际成本差异（量差/价差/效率差）

4. 费用分析（对标+归因）
   - 费用率 vs 预算 vs 行业benchmark
   - 超支归因：量的问题 or 单价问题 or 一次性

5. 现金流与营运资本
   - DSO/DPO/DIO 变动趋势
   - 回款预警、逾期账款风险

### 营运资本周转天数计算（统一口径：365天）

```
DSO（应收账款周转天数）= 应收账款余额 ÷ (年化收入 ÷ 365)
DPO（应付账款周转天数）= 应付账款余额 ÷ (年化采购额 ÷ 365)
DIO（存货周转天数）= 存货余额 ÷ (年化营业成本 ÷ 365)
CCC（现金转换周期）= DSO + DIO - DPO
```

⚠️ **口径规则：默认使用365天。** 以下场景使用360天：银行利息计算（银行惯例按360天计息）、审计底稿中沿用被审计单位已有口径时、题目/客户明确要求360天时。

### DSO异常诊断与BP跟进框架

**当DSO环比/同比恶化超过5天时，触发以下跟进动作：**

```
1. 数据穿透排查
   - 按客户/渠道/账龄拆分应收：区分正常账期、逾期1-30天、逾期30天以上占比
   - 核对大额应收客户：逐一核查合同约定账期 vs 实际回款天数
   - 对比营收结构：判断是新客户（长账期）还是老客户（逾期）导致应收激增

2. 业务协同管控
   - 联动销售：梳理逾期客户清单，制定分层催款计划，明确回款节点
   - 规范信用政策：重新评估新客户授信额度与账期，杜绝无规则放宽
   - 对长期逾期客户：暂停新发货、逐步缩减合作规模，控制坏账风险

3. 风险与财务影响评估
   - 测算资金占用增加额、资金成本上升幅度，评估对现金流压力
   - 计提坏账准备：对长逾期、回款困难客户评估坏账风险，足额计提
   - 滚动预测未来应收、回款、现金流，预判季度末资金缺口

4. 流程优化与常态化监控
   - 梳理开票、对账、结算流程，消除内部低效导致的回款延迟
   - 建立DSO、应收余额、逾期占比周/月度看板，设置预警阈值
   - 定期向管理层汇报应收风险、回款进度、改善效果

5. 专项改善跟进
   - 设定阶段性压降目标（如每月压降DSO 3-5天、清理逾期账款）
   - 跟踪落地效果，同步考核关联业务团队
   - 区分一次性因素（大单延迟）vs 趋势性恶化（信用政策失控），对症下药
```

**DSO恶化常见原因清单：**
- 销售端放宽信用政策（为冲收入给客户更长账期）
- 客户回款能力/意愿变差（下游资金紧张）
- 营收结构变化（新增长账期客户/渠道占比提升）
- 内部流程问题（开票滞后、对账不及时、催款执行不到位）
- 季节性/业务节奏（集中铺货发货，收入增长跟不上应收增速）
- 期末突击确认收入但尚未回款

6. 重点项目追踪
   - 新产品上市进度及预算执行
   - 降本增效项目效果

7. 风险与行动计划
   - 下月/下季度关键风险点
   - 可行动的改善建议（不超过3条）
```

### 量价分析（Revenue Bridge）

```
收入变动 = 量变动贡献 + 价变动贡献 + 结构变动贡献

量变动贡献 = (本期销量 - 上期销量) × 上期均价
价变动贡献 = (本期均价 - 上期均价) × 本期销量
结构变动贡献 = 总变动 - 量变动 - 价变动（交叉项）
```

**应用**：老板问"收入为什么下降了"→ 不回答"因为收入少了"，而是"量增5%但价降8%，净效果收入降3%，主要是A产品降价促销导致"。

### 成本差异分析（制造业重点）

```
总成本差异 = 实际成本 - 标准成本

拆解为：
├── 材料差异
│   ├── 价格差异 = (实际单价 - 标准单价) × 实际用量
│   └── 用量差异 = (实际用量 - 标准用量) × 标准单价
├── 人工差异
│   ├── 工资率差异 = (实际工资率 - 标准工资率) × 实际工时
│   └── 效率差异 = (实际工时 - 标准工时) × 标准工资率
└── 制造费用差异
    ├── 预算差异 = 实际制造费用 - 弹性预算
    └── 产量差异 = (实际产量 - 预算产量) × 标准分摊率
```

### 费用分析方法

```
费用超支归因：
- 收入超预期 → 变动费用自然增长（合理，不需要解释）
- 费用率上升 → 效率下降（需关注原因）
- 固定费用超支 → 新增投入或失控（需解释并提方案）
```

**费用对标四维度**：同比 / 环比 / 预算比 / 行业比

**费用分摊原则**：谁受益谁承担；可量化用实际数据（人头/面积/收入占比）；不可量化用协商比例，年初锁定年中不改。

## 三、预算管理

### 预算编制方法

| 方法 | 适用场景 | 特点 |
|------|---------|------|
| 增量预算 | 业务稳定、变化小 | 简单但容易固化 |
| 零基预算 | 费用管控严格、需重新审视每笔支出 | 精确但耗时 |
| 滚动预测 | 业务变化快、需动态调整 | 灵活但需持续投入 |
| 弹性预算 | 收入波动大 | 按收入档位设定费用上限 |
| 作业基础预算 | 成本动因复杂 | 精确但需要完善的作业数据 |

### 滚动预测（Rolling Forecast）

```
每月更新未来12个月预测：
- 已过去月份：用实际数替换
- 当月：用最新预估
- 未来月份：基于最新假设更新

关键假设需明确标注：
- 销量假设（来自销售pipeline）
- 价格假设（来自定价策略）
- 成本假设（来自采购合同/市场价格）
- 人员假设（来自HC计划）
```

### 预算差异分析

```
预算差异 = 实际 - 预算

归因拆解：
1. 量差 = (实际量 - 预算量) × 预算单价
2. 价差 = (实际单价 - 预算单价) × 实际量
3. 结构差 = 产品/渠道mix变化带来的影响

报告原则：
- 差异 > ±5% 需解释原因
- 差异 > ±10% 需提出行动方案
- 有利差异也要解释（防止预算松弛）
```

## 四、业务决策支持

### 定价决策

```
成本加成法：售价 = 成本 × (1 + 目标加成率)
目标毛利法：售价 = 成本 ÷ (1 - 目标毛利率)
竞争定价法：参考竞品价格，倒推可接受成本
边际贡献法：只要售价 > 变动成本，短期可接受（产能闲置时）

BP的角色：
- 不是定价格（那是业务的事）
- 是告诉业务"这个价格下，毛利是多少，要卖多少量才能覆盖固定成本"
- 是给出"价格每降1%，需要多卖X%才能保持利润不变"的量化测算
```

**降价换量决策公式**：
```
维持原利润所需销量增幅 = 原边际贡献率 / 新边际贡献率 - 1

例：原售价100，变动成本60，边际贡献率40%
    降价10%后：售价90，变动成本60，边际贡献率33.3%
    需增量 = 40%/33.3% - 1 = 20%
    即：降价10%需要销量增加20%才能维持原利润
```

### 投资决策（新项目/新门店/新产线）

```
输入假设：
- 投资额（设备/研发/市场/装修）
- 预计销量曲线（爬坡期→稳定期）
- 单价 & 单位变动成本
- 新增固定成本

输出：
- 盈亏平衡点（月/年销量）
- 投资回收期（Payback Period）
- NPV（净现值）& IRR（内部收益率）
- 敏感性分析（销量±20%、价格±10%对IRR的影响）

决策标准：
- IRR > 公司加权资本成本（WACC）→ 可投
- 回收期 < 行业惯例（制造业3-5年，零售2-3年）→ 可接受
- NPV > 0 → 创造价值
```

#### ⚠️ 投资测算关键假设检查清单（古立特必须在计算前确认）

**通用规则：用户给出的假设如果缺少以下关键项，古立特必须先反问再计算，不能用100%/理想值默认填充。**

| 行业 | 必须确认的关键假设 | 常见陷阱（不能默认100%） | 合理区间参考 |
|------|-------------------|------------------------|-------------|
| **餐饮/零售门店** | 上座率/客流转化率 | 不能假设100%满座 | 餐饮60-80%，零售客流转化率15-30% |
| | 爬坡期时长 | 不能假设开业即满负荷 | 餐饮3-6个月，零售6-12个月 |
| | 其他运营费用（水电/物业/损耗） | 不能只算租金+人工+食材 | 通常占收入5-10% |
| **制造业/产线** | 产能爬坡曲线 | 不能假设投产即满产 | 通常6-12个月达到设计产能的80% |
| | 良品率 | 不能假设100%良品 | 初期70-85%，稳定期90-98% |
| | 设备调试期/停机维护 | 不能假设365天×24小时运转 | 有效生产天数通常300-330天 |
| **互联网/SaaS** | 获客成本（CAC）变化趋势 | 不能假设CAC恒定 | 通常逐年上升15-30% |
| | 用户留存曲线 | 不能假设用户永不流失 | 月留存率85-95%，年NRR见上方 |
| | 服务器/带宽成本弹性 | 不能假设边际成本为零 | 用户量翻倍，基础设施成本通常增60-80% |
| **金融/保险** | 不良率/赔付率 | 不能假设零坏账/零赔付 | 银行不良率1-3%，保险赔付率60-75% |
| | 资本充足率约束 | 不能假设无限放贷 | 监管红线：核心一级≥7.5% |
| | 利率敏感性 | 不能假设利差恒定 | 净息差波动±20-50bp |
| **医药/研发** | 研发成功概率 | 不能假设100%成功 | 临床I期→上市成功率约10-15% |
| | 审批周期 | 不能假设即时获批 | 新药审批通常2-5年 |
| | 专利悬崖 | 不能假设永续收入 | 专利到期后收入通常下降60-80% |
| **房地产** | 去化周期 | 不能假设开盘即清盘 | 住宅6-18个月，商业24-36个月 |
| | 融资成本变动 | 不能假设利率恒定 | 开发贷利率波动±100bp |
| | 预售资金监管 | 不能假设预售款自由使用 | 监管账户通常冻结30-50% |
| **物流/运输** | 装载率/空驶率 | 不能假设100%满载 | 整车85-92%，零担60-75% |
| | 油价波动 | 不能假设油价恒定 | 年波动±15-25% |
| **服务/咨询** | 人员利用率 | 不能假设100%计费 | 咨询65-75%，IT外包70-85% |
| | 人员流失与招聘成本 | 不能假设团队稳定 | 年流失率15-25%，招聘成本约3个月薪资 |
| **跨境电商** | 新品日均单量爬坡 | 不能假设上架即爆单 | 前3个月日均5-20单（非爆品类） |
| | ACoS（广告成本占比） | 不能假设低ACoS | 新品期30-60%，成熟期15-25% |
| | 退货率 | 不能假设零退货 | 标品5-10%，服装15-30% |
| | 库存滞销比例 | 不能假设全部售罄 | 首批备货20-30%可能滞销 |
| | 平台费率变动 | 不能假设费率恒定 | FBA费年均涨5-10% |
| | 汇率波动 | 不能假设汇率恒定 | 美元兑人民币年波动±5-8% |

**古立特行为规则：**
- 用户给出的假设缺少上述关键项 → 反问（最多问2-3个最关键的）
- 用户明确说"就按理想情况算" → 可以计算，但必须在结果中标注"⚠️ 以下为理想假设，实际需打折"并给出敏感性分析
- 用户给了完整假设 → 直接计算，不废话

### 客户盈利分析

```
客户级损益表：
  收入（该客户）
- 直接成本（产品成本、运输、包装）
- 可归属费用（专属销售人员、售后服务、账期资金成本）
= 客户边际贡献

按客户排序后通常呈现"鲸鱼曲线"：
- 前20%客户贡献150%利润
- 后30%客户实际在亏损

行动：淘汰/提价/降低服务等级
```

### 场景分析（Scenario Analysis）

```
三种情景：
- 乐观（Upside）：量+20%，价不变
- 基准（Base）：按预算假设
- 悲观（Downside）：量-20%，价-5%

每种情景下给出：
- 收入/利润/现金流预测
- 关键风险点
- 应对措施建议（提前准备的Plan B）
```

## 五、合同与风险控制

BP 审核合同时关注的财务条款：

| 条款 | 关注点 |
|------|--------|
| 付款条件 | 预付比例、账期长短、是否有提前付款折扣 |
| 价格条款 | 是否含税、汇率锁定、价格调整机制 |
| 折扣与返利 | 阶梯折扣触发条件、返利计提时点 |
| 违约金 | 金额是否合理、触发条件是否清晰 |
| 验收条款 | 收入确认时点的影响 |
| 知识产权 | 研发成果归属（影响资本化判断） |

**风险识别清单**：
- 信用风险：客户账期过长、集中度过高
- 汇率风险：外币收入/成本敞口
- 税务风险：关联交易定价、跨境架构
- 合规风险：销售返利是否合规（医药行业尤其敏感）

## 六、行业差异（BP必须掌握的行业特征）

| 行业 | 收入驱动因子 | 核心成本 | 关键KPI | 典型BP分析场景 |
|------|-------------|---------|---------|---------------|
| **互联网/SaaS** | 用户数、ARPU、留存率 | 研发、营销、服务器 | LTV/CAC、MRR/ARR、流失率、NRR | 获客效率、版本定价、用户分层盈利 |
| **制造业** | 销量、产品结构、单价 | 原材料、人工、制造费用 | 产能利用率、单位成本、良品率、库存周转 | 成本差异（量/价/效率）、BOM分析、自制vs外购 |
| **零售/快消** | 同店销售、客单价、客流 | 商品成本、租金、营销 | 坪效、库存周转、售罄率、同店增长 | 促销ROI、商品组合优化、新店回收期 |
| **金融** | 净息差、手续费、保费 | 资金成本、风险准备金 | ROA、不良率、资本充足率、RAROC | 风险调整回报、客户生命周期价值 |
| **医药** | 药品销量、医保覆盖 | 研发、销售费用 | 研发投入占比、管线进度、医院覆盖率 | 研发项目NPV（含成功概率）、医保谈判影响 |
| **房地产** | 销售面积、均价、去化 | 土地、建安、利息 | 净负债率、去化周期、IRR | 项目现金流、土地测算、融资方案 |
| **能源/化工** | 产品价格（大宗品）、产销量 | 原材料、折旧、环保 | 单位加工费、开工率 | 套期保值效果、产能扩张ROI |
| **物流** | 货运量、单价 | 燃油、人力、车辆折旧 | 装载率、单位成本、准时率 | 线路盈利、自有vs外包决策 |
| **服务/项目制** | 人天单价、项目数 | 人力成本 | 利用率、项目毛利率、人均产值 | 人效分析、项目报价测算 |
| **跨境电商** | 订单量、客单价、转化率、ASIN数 | 采购/头程/FBA/广告/平台佣金 | ACoS、TACoS、ASIN毛利、库存周转、退货率、IPI | 广告ROI、ASIN盈利分析、库存健康度、新品爬坡 |

### 行业深度要点

**互联网/SaaS**：
- LTV/CAC > 3 才健康；CAC回收期 < 12个月
- ARR波动拆解：新增 + 扩增 - 流失 - 降级 = 净变动
- 研发费用资本化判断（CAS 6 五条件）
- **NRR（净收入留存率）标准定义**：
  ```
  NRR = (期初ARR - 流失ARR - 降级ARR + 扩展ARR) ÷ 期初ARR
  ```
  - 分母是期初全量ARR（包括未到期合同），不是只看当期到期合同
  - NRR衡量的是"同一批客户一年后还贡献多少收入"
  - NRR > 110% = 优秀（扩展超过流失）；100-110% = 健康；< 100% = 存量萎缩
  - ⚠️ 不要与"续签率/Gross Renewal Rate"混淆：续签率 = 到期续签ARR ÷ 到期合同ARR，口径不同
- **GRR（毛收入留存率）**：= (期初ARR - 流失ARR - 降级ARR) ÷ 期初ARR（不含扩展）
- **MRR = ARR ÷ 12**（月度经常性收入）

**制造业**：
- 产能利用率每下降10%，单位固定成本上升约X%（具体看固定成本占比）
- 库存呆滞料计提：超过6个月未动的物料需评估减值
- 经济订货批量（EOQ）= √(2×年需求量×单次订货成本 / 单位持有成本)

**零售**：
- 租售比（租金/收入）> 15% 需预警
- 促销评估：增量毛利 > 促销投入 才值得做
- 新店模型：通常第1年亏损（爬坡期），第2年盈亏平衡，第3年贡献利润
- **促销补贴计算注意**：满减类促销（如满300减50），不能假设每单都触发满减。正确做法：先算活动期间实际客单价（GMV÷订单量），如果客单价<满减门槛，说明部分订单未触发。补贴总额应按实际核销数据或用GMV÷门槛金额×补贴金额估算，不能用订单数×补贴金额

**跨境电商（亚马逊/独立站/Temu/TikTok Shop等）**：

#### 商品层级体系（必须搞清楚的基础概念）

```
SPU（Standard Product Unit）= 标准产品单元
  → 一个"产品概念"，不区分规格
  → 例：某款蓝牙耳机

SKU（Stock Keeping Unit）= 库存最小管理单位
  → 一个具体可售卖的规格组合
  → 例：蓝牙耳机-黑色-带充电仓

PID（Product ID）= 平台商品ID的通称
  → 各平台叫法不同，本质是商品唯一标识

ASIN（Amazon Standard Identification Number）= 亚马逊标准识别号
  → 亚马逊给每个商品页面的唯一10位编码（如 B09V3KXJPB）
  → 一个ASIN = 一个可购买的商品详情页

父ASIN（Parent ASIN）= 变体组的"壳"
  → 不可直接购买，是一组变体的容器/聚合页
  → 作用：把同一产品的不同颜色/尺码/规格聚合在一个商品页面
  → 类似SPU的概念

子ASIN（Child ASIN）= 具体可购买的变体
  → 每个颜色/尺码/规格对应一个子ASIN
  → 买家在商品页面切换颜色/尺码，实际是在不同子ASIN之间切换
  → 类似SKU的概念

层级关系：
  1个父ASIN → N个子ASIN（变体关系）
  1个SPU → N个SKU（规格关系）
  父ASIN ≈ SPU，子ASIN ≈ SKU（概念对应，不完全等同）
```

**为什么BP必须理解这套体系？**
- 利润分析必须到**子ASIN/SKU级别**，父ASIN级别的毛利率是假象（可能A颜色赚钱B颜色亏钱）
- 广告投放按ASIN维度，广告费归属必须到子ASIN
- 库存管理按SKU维度，滞销判断必须到SKU
- 退货率按子ASIN差异巨大（某个颜色/尺码退货率可能是其他的3倍）

#### 跨境电商成本结构（以亚马逊FBA为例）

```
售价（Selling Price）
├── 平台佣金（Referral Fee）：通常8-15%（品类不同）
├── FBA费用（Fulfillment Fee）：按件/按重计费
│   ├── 配送费（Pick & Pack）
│   ├── 仓储费（Monthly Storage Fee）：旺季(10-12月)翻倍
│   └── 长期仓储费（Aged Inventory Surcharge）：超365天额外收费
├── 商品成本（COGS）
│   ├── 采购/生产成本
│   ├── 头程物流（海运/空运/快递到FBA仓）
│   └── 关税（进口关税+可能的反倾销税）
├── 广告费（PPC/DSP/品牌推广）
├── 退货成本（退货率×退货处理费+商品损耗）
├── 其他费用
│   ├── 促销折扣/Coupon
│   ├── Vine测评费
│   ├── 品牌注册/A+页面制作
│   └── 仓储移除/销毁费
└── 净利润

单位经济模型（Unit Economics）：
  毛利 = 售价 - 佣金 - FBA费 - COGS - 头程 - 关税
  净利 = 毛利 - 广告费分摊 - 退货损失 - 其他费用分摊
```

#### 核心KPI体系

```
【流量与转化】
- Sessions（访问量）：商品页面被访问次数
- CVR（转化率）= 订单数 ÷ Sessions，通常10-20%算健康
- 页面浏览量（Page Views）vs 独立访客（Unique Visitors）

【广告效率】
- ACoS（Advertising Cost of Sales）= 广告花费 ÷ 广告带来的销售额
  → ACoS < 毛利率 才不亏钱（广告维度）
  → 新品期ACoS高（30-50%）可接受，成熟期应<20%
- TACoS（Total ACoS）= 广告花费 ÷ 总销售额（含自然流量）
  → 衡量广告对整体销售的依赖度
  → TACoS持续下降 = 自然流量占比提升 = 健康信号
  → 成熟产品TACoS应<10%
- ROAS（Return on Ad Spend）= 广告销售额 ÷ 广告花费 = 1/ACoS
- CPC（Cost Per Click）= 广告花费 ÷ 点击数

【库存健康】
- IPI（Inventory Performance Index）：亚马逊库存绩效指标（0-1000分）
  → <400分会被限制仓容
  → 影响因素：售罄率、冗余库存、无在售库存、有货率
- 库存周转天数 = 平均库存 ÷ 日均销量
  → 亚马逊FBA建议保持30-60天库存
  → <30天有断货风险，>90天面临长期仓储费
- 断货率：断货天数 ÷ 总天数
  → 断货不只是损失当天销售，还会丢失排名（恢复需2-4周）
- 售罄率（Sell-through Rate）= 过去90天出货量 ÷ 平均库存

【盈利能力】
- ASIN级毛利率 = (售价 - 佣金 - FBA费 - COGS - 头程) ÷ 售价
- ASIN级净利率 = 扣除广告、退货、促销后的真实利润率
- 贡献利润（Contribution Profit）= 收入 - 所有可归属变动成本
- 退货率：通常5-15%（服装可达20-30%）

【运营效率】
- BSR（Best Sellers Rank）：类目排名，越小越好
- Review数量与评分：影响转化率的核心因素
- Buy Box占有率：多卖家竞争同一ASIN时的购物车占比
```

#### 跨境电商BP经营分析框架

```
月度经营分析（跨境电商版）：

1. GMV与订单概览
   - GMV vs 预算 vs 去年同期
   - 订单量、客单价、退货率变动
   - 按站点/品类/ASIN拆分

2. ASIN级盈利分析（核心）
   - Top 20 ASIN的毛利/净利排名
   - 亏损ASIN清单（广告费吃掉利润的）
   - 新品ASIN爬坡进度 vs 预期

3. 广告效率分析
   - ACoS/TACoS趋势（周维度看）
   - 广告花费占比 vs 自然订单占比
   - 高ACoS关键词/ASIN识别 → 是否暂停或降价

4. 库存健康度
   - IPI分数变动
   - 即将断货的SKU预警（<14天库存）
   - 滞销SKU（>90天未动）→ 清货/移除决策
   - 在途库存 vs 到货预计

5. 现金流特殊性
   - 亚马逊回款周期：14天一结（新卖家可能更长）
   - 头程物流预付（海运提前30-45天付款）
   - 库存占用资金 = 采购成本 × 平均库存天数
   - 实际现金周转周期 = 采购付款→亚马逊回款，通常60-90天

6. 汇率影响
   - 收入端：美元/欧元/英镑收入 vs 人民币成本
   - 汇率波动对毛利的影响测算
   - 是否需要锁汇
```

#### 跨境电商特有决策场景

**场景1：新品上架投入测算**
```
投入：
- 首批备货（MOQ × 单位成本 + 头程）
- 上架推广预算（广告+Vine+Coupon，通常前3个月亏损）
- 品牌内容（A+页面、视频、摄影）

回收期测算：
- 月1-3：亏损期（高ACoS + 低销量）
- 月4-6：爬坡期（ACoS下降，自然流量起来）
- 月7+：盈利期（TACoS<10%，稳定出单）
- 典型回收期：6-12个月（快消品短，高客单价长）

关键假设必须确认：
- 日均单量预期（参考竞品BSR反推）
- 目标ACoS（基于品类竞争程度）
- 退货率假设（参考品类平均）
- 是否有季节性（Q4旺季备货需提前3个月）
```

**场景2：是否清货/移除滞销库存**
```
决策公式：
  继续持有成本 = 月仓储费 × 预计剩余月数 + 长期仓储附加费
  清货回收 = 清货售价 × (1 - 佣金 - FBA费) × 预计可售数量
  移除成本 = 移除费/件 × 数量

  如果：继续持有成本 > 清货回收 → 立即清货
  如果：清货回收 < 移除成本 → 直接销毁（止损）
```

**场景3：多站点扩张决策**
```
评估维度：
- 市场容量（该品类在目标站点的搜索量/竞品数量）
- 合规成本（CE认证/FDA/EPR注册/VAT注册）
- 物流成本差异（头程到欧洲vs北美vs日本）
- 语言/本地化投入
- 汇率风险敞口增加

通常优先级：美国 → 欧洲五国 → 日本 → 其他
```

|-------------------|------------------------|-------------|

#### 跨境税务合规（2026年更新）

**欧盟：2026年7月1日起取消150€免税政策**
```
政策变化：
- 取消非欧盟进口货物≤150欧元的免税待遇
- 过渡期（2026.7.1-2028.7.1）：≤150€包裹按关税税号行征收€3/行
  → 注意：不是按包裹收€3，是按包裹内不同HS编码的商品种类数收
  → 例：10双同款袜子（同一HS编码）= €3；5双棉袜+5双毛袜（两个HS编码）= €6
  → 部分成员国可能额外加收约€2/包裹处理费
- 过渡期后（2028.7起）：按品类征收从价关税（具体税率待定）
- 强制使用IOSS（进口一站式服务）进行增值税申报

对卖家影响：
- 低客单价（<20€）产品利润被严重压缩 → 考虑提价或退出
- 多品类混装包裹成本翻倍 → 尽量同品类合并发货
- 考虑VAT递延（PVA）模式，推迟增值税缴纳到商品售出后
- 欧盟境内设仓可避免每票清关的€3关税
```

**企业所得税架构要点**
```
常见架构：中国母公司 + 香港中间控股 + 美国/欧洲销售子公司

注意事项：
- 中国主体：跨境出口免增值税（出口退税），所得税25%（小微5%）
- 受控外国企业（CFC）规则：利润留在低税区可能被中国税务机关调整补税
- 转移定价：关联交易需准备主体文档+本地文档，利润率（EBIT/收入）保持3-8%
- 美国de minimis：⚠️ 行政暂停 + 立法永久废除双重终结
  → 行政暂停：2025.7.30行政令（2025.8.29全球生效）→ 2026.2.20行政令延续暂停
  → 立法终点：2025.7.4 签署的《One Big Beautiful Bill Act》（OBBBA）已修改美国《1930年关税法》第321条，**$800免税额自2027.7.1起永久废除**，且对违规使用豁免新增民事处罚
  → 含义：行政暂停是过渡，2027.7.1是终点。即便未来政府更迭，要恢复需重新立法
  → $800免税额目前对所有国家均不适用
  → 所有进口包裹（无论价值）均需正式报关并缴纳关税
  → 对直邮小包模式影响巨大，需转向海外仓备货模式
```

- 利用自贸协定（如RCEP）享受降税

- 调整HS编码归类选择低税率税号（需合规）

#### 多币种核算与汇率管理

```
记账规则：
- 境内主体：人民币记账
- 境外主体（美国LLC/香港公司）：美元或港币
- 交易日汇率：按当日央行中间价或平台锁汇汇率
```

#### 现金流管理（跨境电商特有）

```
各平台回款周期：
- Amazon：14天（新卖家可能更长）
- Temu：7天
- 速卖通：15-30天
- 独立站（Stripe/PayPal）：2-7天

资金占用特点：
- 采购预付：通常30%定金 + 70%发货前付清
- 广告预充值：Google/Facebook需提前充值
- 退货准备金：平台暂扣销售额5-10%
- 头程物流：海运提前30-45天付款

现金流预警线：
- 现金余额 < 3个月固定运营成本 → 启动融资或压缩开支
- 库存资金占比 > 总资产60% → 库存过重，需加速周转

融资选择：
- 平台借贷（Amazon Lending）：10-20%，适合短期周转
- 应收账款保理：费率1-3%/月，适合大账期B2B
- 库存质押贷款：年化8-15%，适合滞销少的企业
```

#### 选品盈亏平衡实例

```
假设：
  售价 = $29.99
  采购成本 = $7.00
  头程物流 = $2.00/件
  平台佣金 = 15% × $29.99 = $4.50
  FBA配送费 = $5.00
  广告预估 = $4.00/件（按ACoS 25%×$16售价估算）
  
单位变动成本 = $7 + $2 + $4.5 + $5 + $4 = $22.50
单位边际贡献 = $29.99 - $22.50 = $7.49

月固定成本 = 店铺月租$39.99 + 运营人力分摊$1500 + 软件$200 ≈ $1740
盈亏平衡销量 = $1740 ÷ $7.49 ≈ 233件/月 ≈ 8件/天

判断：如果竞品BSR反推日均销量<8件 → 这个品不值得做
```

#### 年度预算编制节奏

```
Q4上旬：总部下达下年度GMV、利润目标
Q4中旬：各站点/品类制定销售计划、广告预算
Q4下旬：财务汇总，测算现金流、库存、人力
12月底：预算终版报批，分解至月度
每月5日：滚动更新未来3个月预测

预算差异处理规则：
- 差异 < ±5% → 仅记录

```

## 七、BP沟通与汇报

### 经营分析会汇报结构（5分钟版）

```
1. 结论先行（30秒）
   "本月收入达标，但利润低于预算8%，主要是A产品降价促销拉低毛利率2个点。"

2. 数据支撑（2分钟）
   用图表展示量价拆分、费用对标（瀑布图最佳）

3. 归因分析（1分钟）
   "根本原因是竞品X在本月推出低价产品，我们被迫跟进。"

4. 行动建议（1分钟）
   "建议：①下月减少A产品促销 ②加大B产品推广 ③预计毛利率可回升1个点"

5. 下月预测（30秒）
   "基于当前pipeline，下月收入预计达标，利润风险可控。"
```

### 与业务沟通原则

- **用业务语言**：不说"管理费用超支"，说"你们团队差旅费比上月多30%，主要是XX项目出差频次增加"
- **带着方案去**：不只说"超了"，而是"超了，但如果下月控制在XX以内，季度整体可达标"
- **数据可视化**：一张图胜过一页表。趋势图 > 数字表
- **区分可控与不可控**：原材料涨价不可控，但采购策略（锁价/替代/集采）可控
- **不当警察当军师**：目标是让业务主动来找你要分析，而不是躲着你

### 向上汇报原则

- **老板关心三件事**：收入能不能完成？利润有没有风险？现金流够不够？
- **坏消息早说**：预警比事后解释有价值100倍
- **给选择题不给问答题**："方案A省钱但慢，方案B快但贵，我建议选B因为时间窗口只有2周"
- **一页纸原则**：核心结论一页纸能说清楚，详细数据放附件

## 九、BP常见场景问答

**Q1：销售要求降价10%换大订单，怎么分析？**

计算当前边际贡献率 → 模拟降价后新边际贡献率 → 算出维持原利润需增加的销量（公式见第四章）→ 评估销量增幅是否现实 → 同时考虑：竞争对手反应、客户长期价值、产能约束、是否形成价格锚点。

**Q2：产品线毛利下降，如何快速定位？**

三步拆解：①价格差异（实际售价 vs 上期/预算）②成本差异（材料价格、用量、人工效率）③结构变化（高毛利产品占比是否下降）。用瀑布图可视化，一眼看出最大影响因素。

**Q3：新开门店投资回报怎么测？**

预测投资额（装修+设备+前期租金+铺货）→ 预测未来5年收入/成本/费用（含爬坡期）→ 计算NPV/IRR/回收期 → 敏感性分析（客流±20%、客单价±10%、租金+15%）→ 给出"在什么条件下这个店值得开"的结论。

**Q4：如何向业务解释预算刚性？**

用"固定vs变动"概念：固定成本（房租、基本工资）短期无法调整，这部分预算是刚性的；变动成本（促销、差旅）可随业务量弹性调整。建议业务优先优化变动部分，固定部分需提前申报。

**Q5：研发费用该资本化还是费用化？**

CAS 6 五条件：技术可行 + 有意完成 + 有市场 + 资源充足 + 支出可靠计量。BP 协助研发部门整理立项文件和技术可行性报告，判断是否满足。不确定时费用化更稳健。（详见 accounting_core.md 无形资产章节）

**Q6：跨境电商某个ASIN广告ACoS很高，要不要停？**

不能只看ACoS。完整判断链：①看TACoS趋势——如果TACoS在下降，说明广告在带动自然流量，高ACoS可能是"投资期"；②看ASIN生命周期——新品前3个月高ACoS正常（在抢排名），成熟品高ACoS才是问题；③算盈亏平衡ACoS = 毛利率（扣除佣金+FBA+COGS后），ACoS超过这个值就是每单亏钱；④看竞品动态——如果竞品也在猛投，停了可能丢排名，恢复成本更高。结论：新品期容忍高ACoS但设止损线（如连续4周无改善），成熟品ACoS>盈亏线则优化关键词或降价。

**Q7：跨境电商库存备多少合适？**

核心公式：安全库存 = 日均销量 × (补货周期 + 安全天数)。补货周期包括：生产周期(15-30天) + 头程物流(海运30-45天/空运7-10天) + FBA入仓(5-10天)。安全天数通常加7-14天。旺季(Q4)需提前3个月备货，备货量按去年同期×增长系数。⚠️ 备货不是越多越好：超90天库存面临长期仓储费，超365天面临高额附加费。建议保持30-60天库存为健康区间。

**Q8：跨境电商怎么判断一个产品该不该继续卖？**

ASIN级损益表判断：①算清单位经济模型（售价-佣金-FBA-COGS-头程-广告分摊-退货损失=净利）；②如果净利为负且连续3个月无改善趋势→考虑清货；③如果毛利为正但广告吃掉利润→先优化广告（降CPC/精准否词），给2-4周观察期；④如果连毛利都为负（售价<佣金+FBA+COGS）→立即停止补货，清完现有库存退出。
