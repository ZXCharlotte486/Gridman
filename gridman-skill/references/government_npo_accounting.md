# 政府与非营利组织会计

> 定位：财政总会计、行政事业单位会计、民间非营利组织会计三大体系
> 来源：《政府与非营利组织会计》（陈平主编，广东高等教育出版社，2024年9月第1版）
> 覆盖：平行记账、财政拨款、预算会计、国库支付、事业收支、民非收入费用、报表编制
> 性质：理论+实操

## 回答格式要求

回答政府与非营利组织会计的账务处理题时，**必须**使用以下左右对照表格式，财务会计和预算会计并列呈现：

```
| 财务会计 | 预算会计 |
|---------|---------|
| （1）业务描述 ||
| 借：科目　　　金额 | 借：科目　　　金额 |
| 　贷：科目　　金额 | 　贷：科目　　金额 |
```

**规则：**
- 每个业务步骤单独一行标题，横跨两栏
- 财务会计在左，预算会计在右
- 预算会计不做处理时，右栏写"不做处理"
- 金额对齐，借贷方向缩进清晰

## 体系全景

### 基础理论框架（第1-2章）

#### 政府与非营利组织的六大特征

| 特征 | 含义 |
|------|------|
| 公权性 | 政府特有。国家立法机关代表国民意志依法进行公权设计，行政机关具体执行 |
| 公益性与服务性 | 最主要特征。承担社会责任，为公众提供服务，满足社会需求和福利 |
| 非营利性 | 宗旨在于维护国家安全、政治经济文化稳定、社会和谐，不谋求经济利益 |
| 受托责任的多层次性 | 政府受托于人民；非营利组织受托于政府、社会单位、自然人等多方 |
| 资源获得的无偿性 | 政府收入主要来自税收（强制性、无偿性）；非营利组织资源提供者不追求对应经济利益 |
| 资源使用的限定性 | 获得的资源被限制用于国家事务管理、公益与社会服务等非营利性活动 |

#### 四项会计假设

| 假设 | 含义 | 实务意义 |
|------|------|---------|
| 会计主体 | 会计工作为之服务的特定对象。政府会计主体=各级政府+各部门+各单位；民间非营利组织=社会团体+基金会+民办非企业单位 | 已纳入企业财务管理体系的单位和执行《民非制度》的社会团体**不是**政府会计主体 |
| 持续运行 | 业务活动将无限期延续，不会被解散或清算 | 因此可采用历史成本计量、权责发生制；若不成立则需用清算价格 |
| 会计分期 | 将持续不断的业务活动划分为年度、半年度、季度、月度 | 分期结算账目、编制报告、对外信息披露；便于预算执行监控 |
| 货币计量 | 统一以人民币为记账本位币 | 发生外币业务时折算为人民币，同时登记外币金额 |

#### 会计要素确认与计量原则

**财务会计要素（5个）：**

| 要素 | 确认条件 | 计量属性 |
|------|---------|---------|
| 资产 | ①服务潜力/经济利益实现可能性很高 ②成本或价值能可靠计量 | 历史成本（一般）、重置成本、现值、公允价值、名义金额（1元） |
| 负债 | ①履行义务可能导致资源流出 ②金额能可靠计量 | 历史成本（一般）、现值、公允价值 |
| 净资产 | 资产扣除负债后的净额 | — |
| 收入 | ①经济资源很可能流入且有服务潜力/经济利益 ②导致资产增加或负债减少 ③金额能可靠计量 | — |
| 费用 | ①经济资源很可能流出且有服务潜力/经济利益 ②导致资产减少或负债增加 ③金额能可靠计量 | — |

**预算会计要素（3个）：**

| 要素 | 确认原则 | 计量原则 |
|------|---------|---------|
| 预算收入 | 实际收到时确认 | 实际收到的金额 |
| 预算支出 | 实际支付时确认 | 实际支付的金额 |
| 预算结余 | 预算收入 - 预算支出 | — |

**资产五种计量属性详解：**
- **历史成本**：取得时支付的现金或对价的公允价值（最常用）
- **重置成本**：现在购买相同或相似资产所需支付的现金
- **现值**：预计持续使用和最终处置产生的未来净现金流量折现
- **公允价值**：有序交易中出售资产能获得的价格
- **名义金额**：人民币1元（无法按上述属性计量时使用）

#### 八项会计信息质量要求

| 要求 | 含义 |
|------|------|
| 可靠性 | 以实际发生的经济业务为依据，如实反映各要素情况和结果 |
| 全面性 | 将发生的各项经济业务统一纳入会计核算 |
| 相关性 | 与公共受托责任履行情况及报告使用者决策/监督需要相关 |
| 及时性 | 已发生的业务应及时核算，不得提前或延后。《预算法》第14条：批复后20日内向社会公开 |
| 可比性 | 同一主体不同时期采用一致的会计政策；不同主体之间口径一致 |
| 明晰性 | 清晰明了，便于报告使用者理解和使用 |
| 实质重于形式 | 以经济实质为依据，不拘泥于法律形式。如：PPP项目虽由社会资本运营但政府承担主要风险→确认为政府资产；"暂存款"实质已形成支出责任→按实质确认 |
| 谨慎性 | 资产计价从低、负债计量从高、收入确认从低、费用计量从高。不能作为调节盈余的手段 |

#### 会计恒等式

**财务会计：**
```
资产 = 负债 + 净资产
收入 - 费用 = 本期盈余（净资产变动）
合并：资产 = 负债 + 净资产 + 收入 - 费用
```

**预算会计：**
```
预算收入 - 预算支出 = 预算结余
```

#### 政府决算报告 vs 财务报告

| 对比项 | 决算报告 | 财务报告 |
|--------|---------|---------|
| 编制基础 | 收付实现制 | 权责发生制 |
| 核算数据来源 | 预算会计 | 财务会计 |
| 目标 | 反映预算收支年度执行结果 | 反映财务状况、运行情况、现金流量 |
| 使用者 | 各级人大及常委会、各级政府、会计主体自身、社会公众 | 各级人大常委会、债权人、各级政府、会计主体自身 |
| 内容 | 决算报表 + 相关信息资料 | 会计报表（资产负债表、收入费用表等）+ 附注 |

## 体系全景图

```
政府与非营利组织会计
├── 政府会计（广义）
│   ├── 财政总会计（会计主体：各级人民政府）
│   │   ├── 财务会计（权责发生制）
│   │   └── 预算会计（收付实现制）
│   └── 行政事业单位会计（会计主体：行政事业单位）
│       ├── 财务会计（权责发生制）
│       └── 预算会计（收付实现制）
└── 非营利组织会计
    └── 民间非营利组织会计（社会团体、基金会、民办非企业单位）
        └── 单一基础：权责发生制（无预算会计）
```

**三双特点（行政事业单位 & 财政总会计共有）：**
- **双基础**：财务会计用权责发生制，预算会计用收付实现制
- **双功能**：同时具备财务会计和预算会计两种核算功能
- **双报告**：财务报告 + 预算执行情况报告（决算报告）

**法规依据：**
- 财政总会计：《财政总会计制度》财库〔2022〕41号，2023年1月1日起施行
- 行政事业单位：《政府会计制度》财会〔2017〕25号，2019年1月1日起施行
- 民间非营利组织：《民间非营利组织会计制度》财会〔2004〕7号

## 核心判断规则

### R1：平行记账触发条件

涉及**预算管理的现金收支**，必须同时做财务会计和预算会计记录。

**触发平行记账的业务：**
- 财政拨款收到/支出
- 政府债券发行/偿还
- 借款取得/还本
- 实际收到投资收益（股利、利息）
- 财政专户资金收支

**不触发预算会计的业务（只做财务会计）：**
- 计提折旧/摊销
- 计提工资（应付职工薪酬）
- 计提利息（应付利息）
- 权益法确认投资收益（账面调整）
- 资产盘盈/盘亏处理
- 坏账准备计提
- PPP项目资产折旧（部分情形）

### R2：财务会计 vs 预算会计确认时点差异

| 事项 | 财务会计（权责发生制） | 预算会计（收付实现制） |
|------|---------------------|---------------------|
| 工资 | 计提时确认费用 | 实际发放时确认支出 |
| 利息 | 按期计提确认费用 | 实际支付时确认支出 |
| 折旧 | 每期计提确认费用 | 购入固定资产时已记支出，后续不再做 |
| 债券收入 | 发行时确认负债 | 发行时确认预算收入 |
| 投资收益 | 权益法调整时确认 | 实际收到股利时确认预算收益 |
| 财政拨款 | 收到通知书时确认收入 | 收到通知书时确认预算收入 |

### R3：行政单位 vs 事业单位科目差异

**行政单位没有的科目（事业单位特有）：**
- 短期投资、长期股权投资、长期债券投资
- 应收票据、应收股利、应收利息、坏账准备
- 预收账款、应付票据、应付利息、短期借款、长期借款
- 事业收入、上级补助收入、附属单位上缴收入、经营收入、投资收益
- 单位管理费用、经营费用、上缴上级费用、对附属单位补助费用、所得税费用
- 专用基金、权益法调整

**事业单位没有的科目（行政单位特有）：**
- 应付政府补贴款

### R4：民间非营利组织与政府会计的核心区别

| 对比项 | 行政事业单位 | 民间非营利组织 |
|--------|------------|--------------|
| 核算基础 | 双基础（财务+预算） | 单一权责发生制 |
| 净资产分类 | 累计盈余等 | 限定性净资产 + 非限定性净资产 |
| 预算会计 | 有 | 无 |
| 报表体系 | 财务报告 + 决算报告 | 资产负债表 + 业务活动表 + 现金流量表 |
| 收入确认 | 财政拨款为主 | 捐赠收入、会费、服务收入等 |

## 第一篇：财政总会计

### 一、财政总会计概述

**会计主体：** 各级人民政府（中央、省、市、县、乡镇）

**核算范围：** 一般公共预算、政府性基金预算、国有资本经营预算、财政专户管理资金、专用基金、地方债券等资金活动

**双基础特点：**
- 财务会计：权责发生制，核算资产、负债、净资产、收入、费用
- 预算会计：收付实现制，核算预算收入、预算支出、预算结余

#### 1.1 财政总会计的组织管理与职责

**组织形式：** 各级政府财政部门设置财政总会计岗位，负责本级政府财政资金的会计核算和管理。

**主要职责：**
1. 办理本级政府财政资金的会计核算业务
2. 编制本级政府财务报告和决算报告
3. 参与本级政府预算的编制和执行
4. 管理本级政府财政资金的收付和结算
5. 监督本级政府财政资金的使用情况
6. 保管会计档案

**内部控制要求：**
- 不相容岗位分离（出纳与记账分离、审批与执行分离）
- 凭证审核制度（以审核无误的国库入库凭证、预算收入日报表和其他合法凭证为依据）
- 定期对账制度（与国库、代理银行、上下级财政定期核对）
- 会计档案管理制度

### 二、财政总会计科目体系（第4章）

#### 2.1 资产类科目详解

| 科目 | 核算内容 | 余额方向 | 关键规则 |
|------|---------|---------|---------|
| 国库存款 | 存放国库单一账户的款项 | 借 | 财政总会计最核心的资金科目 |
| 国库现金管理资产 | 闲置库款存放商业银行或投资货币市场 | 借 | 属于国库现金管理操作形成的资产 |
| 其他财政存款 | 存放财政专户、汇缴账户等的款项 | 借 | 与国库存款区分，专户资金用此科目 |
| 有价证券 | 按规定取得并持有的有价证券 | 借 | 包括国债、地方债等 |
| 应收非税收入 | 应收但未缴入国库的非税收入 | 借 | 权责发生制确认，实收时冲销 |
| 应收地方政府债券转贷款 | 转贷给下级政府的债券资金本息 | 借 | 含本金和应收利息 |
| 股权投资 | 政府持有的各类股权 | 借 | 权益法核算，含投资成本/损益调整/其他权益变动三个明细 |
| 与下级往来 | 本级与下级政府财政往来待结算款 | 借/贷 | 年终结算后转入补助收入/费用或上解收入/费用 |
| 借出款项 | 财政向预算单位等借出的款项 | 借 | 区别于股权投资，属于债权 |

**国库单一账户体系（重要）：**
```
国库单一账户体系
├── 国库存款账户（实存，中国人民银行）
│   → 与财政部门零余额账户和预算单位零余额账户清算
├── 零余额账户（虚存）
│   ├── 财政部门零余额账户 → 财政直接支付和与国库单一账户清算
│   ├── 预算单位零余额账户 → 财政授权支付和与国库单一账户支出清算
│   └── 汇缴零余额账户 → 预算单位收取应汇国库的财政性资金
└── 财政专户
    ├── 一般财政专户 → 财政专户管理收支
    └── 特设专户 → 特殊过渡性专项支出
```

#### 2.2 负债类科目详解

| 科目 | 核算内容 | 关键规则 |
|------|---------|---------|
| 应付短期政府债券 | 期限1年以内的国债和地方政府债券 | 按实际发行额确认 |
| 应付长期政府债券 | 期限超过1年的国债和地方政府债券 | 含一般债券和专项债券 |
| 应付利息 | 应付未付的债券利息 | 按期计提，实付时冲销 |
| 与上级往来 | 本级与上级政府财政往来待结算款 | 年终结算后转入上解费用/补助收入 |
| 应付地方政府债券转贷款 | 下级从上级取得的转贷款形成的债务 | 对应资产方的应收地方政府债券转贷款 |
| 借入款项 | 向国际金融组织等借入的主权外债 | 含本金，利息单独计提 |
| 其他负债 | 政府财政承担支出责任的其他负债 | 如政府承担的拆迁安置等支出责任 |

#### 2.3 净资产类科目

**财务会计净资产：**
- 累计盈余（含预算管理资金/财政专户管理资金/专用基金三个明细）
- 本期盈余（含三个明细，年末结转后无余额）
- 预算稳定调节基金（储备性资金）
- 预算周转金（季节性周转资金）
- 权益法调整（股权投资他权益变动份额）
- 以前年度盈余调整（前期差错更正，年末结转后无余额）
- 资产基金（对应政府储备物资、文物文化资产等无对应负债的资产价值）

**预算会计净资产（资金结存）：**
- 资金结存——库款资金结存（国库存款对应）
- 资金结存——财政专户资金结存（财政专户存款对应）

**资产基金的特殊性：**
政府储备物资、文物文化资产、公共基础设施等资产取得时，若无对应负债，则贷记"资产基金"而非收入科目。
```
取得政府储备物资（财政直接支付）：
【财务会计】
借：政府储备物资
  贷：财政拨款收入（若有对应拨款）
  或贷：资产基金（若无对应负债）
```

### 三、财政总会计收入核算

#### 3.1 税收收入与非税收入

**税收收入分类（按央地划分）：**

| 类型 | 税种 |
|------|------|
| 中央固定收入 | 消费税、车辆购置税、关税、船舶吨税、海关代征增值税 |
| 地方固定收入 | 城镇土地使用税、耕地占用税、土地增值税、房产税、车船税、契税、烟叶税 |
| 共享收入 | 增值税、企业所得税、个人所得税、资源税、城市维护建设税、印花税 |

**账务处理：**

收到国库报来预算收入日报表时：
```
【财务会计】
借：国库存款
  贷：税收收入 / 非税收入

【预算会计】
借：资金结存——库款资金结存
  贷：一般公共预算收入
```

年终结转：
```
【财务会计】
借：税收收入 / 非税收入
  贷：本期盈余——预算管理资金本期盈余

【预算会计】
借：一般公共预算收入
  贷：一般公共预算结转结余
```

#### 3.2 政府债券发行与偿还

**发行（双分录标准模板）：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：国库存款（实收款）<br>　有关费用科目（发行费用）<br>贷：应付长期/短期政府债券（实际发行额） | 借：资金结存——库款资金结存<br>贷：债务预算收入 |

**偿还本金：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：应付长期/短期政府债券<br>贷：国库存款 | 借：债务还本预算支出<br>贷：资金结存——库款资金结存 |

**支付利息：**

| 财务会计 | 预算会计 |
|---------|---------|
| 计提：借：财务费用——利息费用<br>　　　贷：应付利息 | 不做处理 |
| 支付：借：应付利息<br>　　　贷：国库存款 | 借：债务付息预算支出<br>贷：资金结存——库款资金结存 |

**示例（例5-3）：** 某省发行2年期地方政府一般债券，票面利率3%，发行金额80,000,000元，每年付息一次，到期还本。

| 步骤 | 财务会计 | 预算会计 |
|------|---------|---------|
| 收到债券资金 | 借：国库存款 80,000,000<br>贷：应付长期政府债券 80,000,000 | 借：资金结存——库款 80,000,000<br>贷：债务预算收入 80,000,000 |
| 每月计提利息（200,000元） | 借：财务费用——利息费用 200,000<br>贷：应付利息 200,000 | 不做处理 |
| 第一年偿还利息（2,400,000元） | 借：应付利息 2,400,000<br>贷：国库存款 2,400,000 | 借：债务付息预算支出 2,400,000<br>贷：资金结存——库款 2,400,000 |
| 到期还本付息 | 借：应付长期政府债券 80,000,000<br>　　应付利息 2,200,000<br>　　财务费用——利息费用 200,000<br>贷：国库存款 82,400,000 | 借：债务还本预算支出 80,000,000<br>　　债务付息预算支出 2,400,000<br>贷：资金结存——库款 82,400,000 |


#### 3.3 股权投资（权益法）

```
【预算会计】不做处理
```

**宣告发放股利：**
```
【财务会计】
借：应收股利
  贷：股权投资——损益调整
【预算会计】不做处理
```

**实际收到股利：**
```
【财务会计】
借：国库存款
  贷：应收股利

【预算会计】
借：资金结存——库款资金结存
  贷：投资预算收益（或相关预算收入科目）
```

**被投资主体发生其他权益变动：**
```
【财务会计】
借：股权投资——其他权益变动
  贷：权益法调整
【预算会计】不做处理
```

### 四、财政总会计净资产核算

#### 4.1 本期盈余

本期盈余核算政府财政各类收入、费用相抵后的余额，分三个明细：
- 预算管理资金本期盈余
- 财政专户管理资金本期盈余
- 专用基金本期盈余

**年终结转流程：**

第一步：将各类收入科目余额转入本期盈余（贷方）
```
借：税收收入 / 非税收入 / 投资收益 / 补助收入 / 上解收入 / 地区间援助收入 / 其他收入
  贷：本期盈余——预算管理资金本期盈余
```

第二步：将各类费用科目余额转入本期盈余（借方）
```
借：本期盈余——预算管理资金本期盈余
  贷：政府机关商品和服务拨款费用 / 工资福利拨款费用 / 对事业单位补助拨款费用 /
      对企业补助拨款费用 / 对个人和家庭补助拨款费用 / 资本性拨款费用 /
      财务费用 / 补助费用 / 上解费用 / 地区间援助费用 / 其他费用
```

第三步：设置或补充预算稳定调节基金
```
借：本期盈余——预算管理资金本期盈余
  贷：预算稳定调节基金
```

第四步：将本期盈余余额转入累计盈余
```
如为贷方余额（盈余）：
借：本期盈余——预算管理资金本期盈余
  贷：累计盈余——预算管理资金累计盈余

如为借方余额（亏损）：
借：累计盈余——预算管理资金累计盈余
  贷：本期盈余——预算管理资金本期盈余
```

#### 4.2 预算稳定调节基金

用于平衡年度间预算衔接的储备性资金。

```
设置或补充：
借：本期盈余——预算管理资金本期盈余
  贷：预算稳定调节基金

动用：
借：预算稳定调节基金
  贷：本期盈余——预算管理资金本期盈余
```

#### 4.3 预算周转金

用于调剂预算年度内季节性收支差额的周转资金。

```
设置或补充：
借：本期盈余——预算管理资金本期盈余
  贷：预算周转金

将预算周转金调入预算稳定调节基金：
借：预算周转金
  贷：预算稳定调节基金
```

#### 4.4 权益法调整

核算政府财政按持股比例享有的被投资主体除净损益和利润分配以外的所有者权益变动份额。

```
被投资主体发生其他权益变动：
借：股权投资——其他权益变动
  贷：权益法调整

处置股权投资时：
借：权益法调整
  贷：股权投资——其他权益变动
```

**权益法转成本法示例（例5-11）：**
某市财政持有某企业60%股权，权益法核算。年末该企业盈利660,000元，其他权益变动420,000元。次年改为成本法。

| 步骤 | 分录 |
|------|------|
| 取得投资 | 借：股权投资（投资成本）12,000,000 / 贷：国库存款 12,000,000 |
| 确认净利润（660,000×60%=396,000） | 借：股权投资（损益调整）396,000 / 贷：投资收益 396,000 |
| 确认其他权益变动（420,000×60%=252,000） | 借：股权投资（其他权益变动）252,000 / 贷：权益法调整 252,000 |
| 权益法转成本法 | 借：权益法调整 252,000 / 贷：股权投资（其他权益变动）252,000；借：股权投资（投资成本）12,396,000 / 贷：股权投资（投资成本）12,000,000、股权投资（损益调整）396,000 |

#### 4.5 以前年度盈余调整

核算调整以前年度盈余的事项（前期差错更正等）。

```
调整增加以前年度收入：
借：有关资产或负债科目
  贷：以前年度盈余调整

调整增加以前年度费用：
借：以前年度盈余调整
  贷：有关资产或负债科目

年终结转：
借/贷：累计盈余
  贷/借：以前年度盈余调整
```

#### 5.0 政府收支分类科目体系

#### 5.0b 一般公共预算收入的收缴方式和程序

**直接缴库：** 缴款单位或缴款人按照法律法规，将应缴款项直接缴入财政国库。
- 纳税人提出纳税申报 → 征收机关审核 → 纳税人通过开户银行直接将税款缴入国库存款账户

**集中汇缴：** 征收机关按照法律法规，将所收取的应缴款项汇总缴入财政国库。
- 适用于小额零散税收和非税现金收入
- 征收机关在收到应缴款项当日将其汇总缴入国库存款账户

> **重要规则：** 无论直接缴库还是集中汇缴，征收机关都**不需要设立缴款项的过渡账户**。收到的款项不必先存入征收机关的专门账户再缴入国库。

#### 5.1 预算收入体系

财政总会计预算收入主要包括：

| 科目 | 核算内容 |
|------|---------|
| 一般公共预算收入 | 税收收入 + 非税收入（本级） |
| 政府性基金预算收入 | 土地出让金等专项基金收入 |
| 国有资本经营预算收入 | 国企利润上缴 |
| 财政专户管理资金收入 | 教育收费等专户资金 |
| 专用基金收入 | 粮食风险基金等 |
| 债务预算收入 | 发行政府债券取得的资金 |
| 债务转贷预算收入 | 上级转贷给本级的债券资金 |
| 补助预算收入 | 上级补助本级的款项 |
| 上解预算收入 | 下级上解本级的款项 |
| 地区间援助预算收入 | 受援方收到的援助资金 |
| 调入预算资金 | 从其他预算调入的资金 |
| 动用预算稳定调节基金 | 动用储备基金 |

**账务处理通则：**
```
收到款项时：
借：资金结存——库款资金结存
  贷：对应预算收入科目

年终结转：
借：对应预算收入科目
  贷：对应预算结转结余科目
```

| 科目 | 核算内容 |
|------|---------|

#### 5.3 预算结余体系

| 科目 | 核算内容 |
|------|---------|
| 一般公共预算结转结余 | 一般公共预算收支差额 |
| 政府性基金预算结转结余 | 政府性基金预算收支差额 |
| 国有资本经营预算结转结余 | 国有资本经营预算收支差额 |
| 财政专户管理资金结余 | 专户资金结余 |
| 专用基金结余 | 专用基金收支差额 |
| 预算稳定调节基金 | 储备性资金 |
| 预算周转金 | 季节性周转资金 |
| 资金结存 | 库款资金结存 + 财政专户资金结存 |

**年终结转流程（以一般公共预算为例）：**
```
第一步：将一般公共预算收入转入结转结余
借：一般公共预算收入
  贷：一般公共预算结转结余

第二步：将一般公共预算支出转入结转结余
借：一般公共预算结转结余
  贷：一般公共预算支出

第三步：安排预算稳定调节基金（如有）
借：一般公共预算结转结余
  贷：预算稳定调节基金

第四步：结余与资金结存核对
一般公共预算结转结余余额 = 资金结存——库款资金结存余额（理论上应相等）
```


### 六、财政总会计报告体系

#### 6.1 财务会计报告

| 报表 | 编制基础 | 主要内容 |
|------|---------|---------|
| 资产负债表 | 权责发生制 | 资产、负债、净资产规模及构成 |
| 收入费用表 | 权责发生制 | 收入、费用及盈余 |
| 现金流量表 | 权责发生制 | 现金流入流出（选编） |
| 本年预算结余与本期盈余调节表 | 双基础 | 两套报表差异调整 |

**资产负债表关键项目填列规则：**
- 国库存款：期末借方余额
- 股权投资：期末借方余额（权益法下含损益调整）
- 应付政府债券：期末贷方余额（短期+长期分别列示）
- 累计盈余：期末贷方余额
- 预算稳定调节基金：期末贷方余额（列于净资产类）
- 资产基金：与特定资产对应，反映政府储备物资、文物文化资产等无对应负债的资产价值

**收入费用表关键项目：**
- 收入合计 = 税收收入 + 非税收入 + 投资收益 + 补助收入 + 上解收入 + 地区间援助收入 + 其他收入 + 财政专户管理资金收入 + 专用基金收入
- 费用合计 = 各类拨款费用 + 财务费用 + 补助费用 + 上解费用 + 地区间援助费用 + 其他费用 + 财政专户管理资金支出 + 专用基金支出
- 本期盈余 = 收入合计 - 费用合计

#### 6.2 预算执行情况报告

| 报表 | 内容 |
|------|------|
| 预算收入支出表 | 各类预算收支执行结果（含年初结转结余、年末结转结余） |
| 一般公共预算执行情况表 | 按《政府收支分类科目》列示一般公共预算收支 |
| 政府性基金预算执行情况表 | 政府性基金预算收支 |
| 国有资本经营预算执行情况表 | 国有资本经营预算收支 |
| 财政专户管理资金收支情况表 | 专户资金收支 |
| 专用基金收支情况表 | 各类专用基金收支 |

**预算收入支出表主要项目：**
- 年初结转结余：上年末各预算结转结余科目余额
- 收入合计：本年各预算收入科目发生额
- 支出合计：本年各预算支出科目发生额
- 年末结转结余：年末各预算结转结余科目余额
- 线下项目（不计入收支合计）：动用预算稳定调节基金、安排预算稳定调节基金、增设预算周转金

#### 6.3 本年预算结余与本期盈余调节表

该表衔接预算会计和财务会计，说明两套报表差异原因。

**主要差异项：**

| 差异项 | 影响方向 |
|--------|---------|
| 当期确认收入但未确认预算收入（如应收未缴库非税收入） | 本期盈余 > 预算结余 |
| 当期确认预算收入但未确认收入（如收到上期应收非税收入） | 预算结余 > 本期盈余 |
| 当期确认费用但未确认预算支出（如计提未拨付债务利息） | 本期盈余 < 预算结余 |
| 当期确认预算支出但未确认费用（如资本性支出、债务还本支出） | 预算结余 < 本期盈余 |

## 第二篇：行政事业单位会计

### 一、概述与特点

**会计主体：** 行政单位（国家机关、政党组织等）+ 事业单位（教育、科技、文化、卫生等）

**四大特点：**
- **双功能**：财务会计（资产/负债/净资产/收入/费用）+ 预算会计（预算收入/预算支出/预算结余）
- **双基础**：财务会计用权责发生制，预算会计用收付实现制
- **双报告**：财务报告 + 决算报告
- **平行记账**：同一原始凭证、同一记账凭证号，同时做财务会计和预算会计分录

**行政单位 vs 事业单位关键区别：**

| 对比项 | 行政单位 | 事业单位 |
|--------|---------|---------|
| 投资科目 | 无 | 有（短期投资、长期股权投资等） |
| 借款科目 | 无 | 有（短期借款、长期借款） |
| 收入科目 | 财政拨款收入为主 | 财政拨款 + 事业收入 + 经营收入等 |
| 费用科目 | 业务活动费用、资产处置费用等 | 增加单位管理费用、经营费用、所得税费用等 |
| 净资产 | 无专用基金、权益法调整 | 有专用基金、权益法调整 |
| 预算支出 | 行政支出 | 事业支出、经营支出等 |


### 二、平行记账标准模板

| 业务类型 | 财务会计 | 预算会计 |
|---------|---------|---------|
| 财政直接支付购资产/服务 | 借：资产/费用 / 贷：财政拨款收入 | 借：行政/事业支出——财政拨款支出 / 贷：财政拨款预算收入 |
| 财政授权支付（零余额账户） | 借：费用/资产 / 贷：零余额账户用款额度 | 借：行政/事业支出——财政拨款支出 / 贷：资金结存——零余额账户用款额度 |
| 自有资金支出 | 借：费用/资产 / 贷：银行存款 | 借：行政/事业支出——其他资金支出 / 贷：资金结存——货币资金 |
| 借款取得（事业单位） | 借：银行存款 / 贷：短期/长期借款 | 借：资金结存——货币资金 / 贷：债务预算收入 |
| 还借款本金（事业单位） | 借：短期/长期借款 / 贷：银行存款 | 借：债务还本支出 / 贷：资金结存——货币资金 |
| 计提折旧/摊销 | 借：费用 / 贷：累计折旧/累计摊销 | **不做** |
| 计提工资 | 借：费用 / 贷：应付职工薪酬 | **不做** |
| 计提利息 | 借：费用 / 贷：应付利息 | **不做** |
| 权益法确认投资收益 | 借：长期股权投资 / 贷：投资收益 | **不做** |
| 实收股利/利息 | 借：银行存款 / 贷：应收股利/应收利息 | 借：资金结存 / 贷：投资预算收益 |


### 三、资产核算

#### 3.1 货币资金与财政应返还额度

**零余额账户用款额度：** 财政授权支付时，单位收到额度通知书，借记本科目；实际支付时贷记本科目。

**财政应返还额度：** 年末财政收回未用额度或未下达额度时，借记本科目。

> **关键规则：财政授权支付额度确认**
> 收到"财政授权支付额度到账通知书"时，财政拨款收入按**预算指标数**（全年批准总额）确认，不是按已下达额度。已下达额度记零余额账户用款额度，未下达部分记财政应返还额度。

**年末财政授权支付额度处理（三种情形）：**

先明确三个数字的关系：
- **预算指标数**：财政批准的全年预算总额
- **已下达额度**：财政实际下达到零余额账户的金额（≤ 预算指标数）
- **实际使用额度**：单位实际支付出去的金额（≤ 已下达额度）

**情形一：年末注销已下达但未用的额度**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：财政应返还额度——财政授权支付（未用额度）<br>贷：零余额账户用款额度（未用额度） | 借：资金结存——财政应返还额度（未用额度）<br>贷：资金结存——零余额账户用款额度（未用额度） |

**情形二：年末确认未下达的额度**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：财政应返还额度——财政授权支付（未下达额度）<br>贷：财政拨款收入（未下达额度） | 借：资金结存——财政应返还额度（未下达额度）<br>贷：财政拨款预算收入（未下达额度） |

**情形三：次年财政恢复额度**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：零余额账户用款额度（恢复金额）<br>贷：财政应返还额度——财政授权支付（恢复金额） | 借：资金结存——零余额账户用款额度（恢复金额）<br>贷：资金结存——财政应返还额度（恢复金额） |

**完整示例（预算指标600,000，已下达500,000，实际使用470,000）：**

| 步骤 | 事项 | 财务会计 | 预算会计 |
|------|------|---------|---------|
| ①年中 | 收到已下达额度500,000 | 借：零余额账户用款额度 500,000<br>贷：财政拨款收入 500,000 | 借：资金结存—零余额账户 500,000<br>贷：财政拨款预算收入 500,000 |
| ②年中 | 实际使用470,000购物品 | 借：库存物品 470,000<br>贷：零余额账户用款额度 470,000 | 借：事业支出 470,000<br>贷：资金结存—零余额账户 470,000 |
| ③年末 | 注销未用额度30,000（情形一） | 借：财政应返还额度 30,000<br>贷：零余额账户用款额度 30,000 | 借：资金结存—财政应返还额度 30,000<br>贷：资金结存—零余额账户 30,000 |
| ④年末 | 确认未下达额度100,000（情形二） | 借：财政应返还额度 100,000<br>贷：财政拨款收入 100,000 | 借：资金结存—财政应返还额度 100,000<br>贷：财政拨款预算收入 100,000 |
| ⑤次年 | 恢复全部额度130,000（情形三） | 借：零余额账户用款额度 130,000<br>贷：财政应返还额度 130,000 | 借：资金结存—零余额账户 130,000<br>贷：资金结存—财政应返还额度 130,000 |

> 年末财政拨款收入合计 = 已下达500,000 + 未下达100,000 = **600,000**（等于预算指标数）

#### 3.1b 应收及预付款项

**应收账款：** 单位因提供服务、出租资产等应收取的款项。

```
确认应收：
借：应收账款
  贷：事业收入/租金收入等

收回款项：
借：银行存款
  贷：应收账款

坏账核销（报经批准，逾期3年以上且有确凿证据无法收回）：
借：业务活动费用/单位管理费用（坏账损失）
  贷：应收账款
（事业单位有坏账准备的，先冲准备，不足部分计入费用）

收回已核销坏账：
借：应收账款
  贷：业务活动费用/单位管理费用（冲回）
借：银行存款
  贷：应收账款
```

**坏账准备（事业单位特有）：**
```
计提：
借：业务活动费用/单位管理费用
  贷：坏账准备
（预算会计不做）

转回：
借：坏账准备
  贷：业务活动费用/单位管理费用
```

**预付账款：**
```
预付款项：
借：预付账款
  贷：零余额账户用款额度/银行存款
预算会计：
借：行政/事业支出
  贷：资金结存

收到货物/服务，补付余款：
借：库存物品/固定资产/费用（总价款）
  贷：预付账款（已付部分）
      零余额账户用款额度/银行存款（余款）
预算会计（余款部分）：
借：行政/事业支出
  贷：资金结存
```

> **注意：** 预付账款在支付时预算会计即确认支出，不等到收到货物。对于暂付款项，也可在支付时不做预算会计，待结算报销时再做。

#### 3.1c 长期股权投资（事业单位特有）

**取得方式与初始计量：**

| 取得方式 | 初始计量 |
|---------|---------|
| 货币资金购入 | 实际支付价款（含相关税费） |
| 非货币资产置换 | 换出资产公允价值 + 相关税费 |
| 接受捐赠 | 凭证金额或公允价值 |

**权益法核算（持股≥20%或有重大影响）：**

```
取得投资：
借：长期股权投资——投资成本
  贷：银行存款
预算会计：
借：投资支出
  贷：资金结存——货币资金

被投资单位盈利（按持股比例）：
借：长期股权投资——损益调整
  贷：投资收益
（预算会计不做）

被投资单位亏损：
借：投资收益（亏损额）
  贷：长期股权投资——损益调整
（预算会计不做）

被投资单位宣告分红：
借：应收股利
  贷：长期股权投资——损益调整
（预算会计不做）

实际收到股利：
借：银行存款
  贷：应收股利
预算会计：
借：资金结存——货币资金
  贷：投资预算收益

被投资单位其他权益变动：
借：长期股权投资——其他权益变动
  贷：权益法调整
（预算会计不做）
```

**成本法核算（持股<20%且无重大影响）：**
```
取得投资：
借：长期股权投资——投资成本
  贷：银行存款

宣告分红（按持股比例）：
借：应收股利
  贷：投资收益
（预算会计不做）

实际收到股利：
借：银行存款
  贷：应收股利
预算会计：
借：资金结存——货币资金
  贷：投资预算收益
```

**处置长期股权投资：**
```
借：银行存款（实收款）
    长期股权投资减值准备（已提准备）
  贷：长期股权投资（账面余额）
      投资收益（差额，或借方）
预算会计：
借：资金结存——货币资金（实收款）
  贷：投资支出（原投资支出金额）
      投资预算收益（差额，或借方）
```

#### 3.1d 长期债券投资（事业单位特有）

**取得（持有至到期）：**
```
借：长期债券投资——债券面值（面值）
              ——债券溢价/折价（差额）
  贷：银行存款（实付款）
预算会计：
借：投资支出
  贷：资金结存——货币资金
```

**按期确认利息收入（实际利率法）：**
```
借：应收利息（票面利率×面值）
    长期债券投资——债券折价（折价摊销）
  贷：投资收益（实际利率×账面价值）
      长期债券投资——债券溢价（溢价摊销）
（预算会计不做）

实际收到利息：
借：银行存款
  贷：应收利息
预算会计：
借：资金结存——货币资金
  贷：投资预算收益
```

**到期收回本金：**
```
借：银行存款
  贷：长期债券投资——债券面值
预算会计：
借：资金结存——货币资金
  贷：投资支出（原投资支出金额）
      投资预算收益（差额）
```

#### 3.2 固定资产

**不计提折旧的固定资产：** 文物、图书档案、不单独核算的土地。

**折旧方法：** 年限平均法（直线法）为主，特殊情况用工作量法。

**取得固定资产：**

| 取得方式 | 财务会计 | 预算会计 |
|---------|---------|---------|
| 财政直接支付 | 借：固定资产<br>贷：财政拨款收入 | 借：行政/事业支出——财政拨款支出<br>贷：财政拨款预算收入 |
| 财政授权支付 | 借：固定资产<br>贷：零余额账户用款额度 | 借：行政/事业支出——财政拨款支出<br>贷：资金结存——零余额账户用款额度 |
| 自有资金购入 | 借：固定资产<br>贷：银行存款 | 借：行政/事业支出——其他资金支出<br>贷：资金结存——货币资金 |
| 无偿调入 | 借：固定资产（调出方账面价值+调入方相关费用）<br>贷：无偿调拨净资产（差额）、银行存款（费用） | 不做处理 |
| 接受捐赠 | 借：固定资产（凭证金额或公允价值）<br>贷：捐赠收入、银行存款（相关费用） | 不做处理 |

**计提折旧：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：业务活动费用/单位管理费用等<br>贷：固定资产累计折旧 | **不做处理** |

**处置固定资产（报废/出售）：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：资产处置费用（账面价值）<br>　　固定资产累计折旧（已提折旧）<br>贷：固定资产（原值） | 不做处理 |
| 收到处置收入：<br>借：银行存款<br>贷：应缴财政款——资产处置款项 | 不做处理 |

**无偿调出：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：无偿调拨净资产（账面价值）<br>　　固定资产累计折旧<br>贷：固定资产（原值）<br>借：资产处置费用（调出相关费用）<br>贷：银行存款 | 不做处理 |

**固定资产盘盈：**
```
发现时：借：固定资产 / 贷：待处理财产损溢
批准后（本年取得）：按新取得资产处理
批准后（以前年度）：借：待处理财产损溢 / 贷：以前年度盈余调整
```

**固定资产盘亏/报废：**
```
发现时：
  借：待处理财产损溢——待处理财产价值（账面价值）
      固定资产累计折旧
    贷：固定资产（原值）
批准后：
  借：资产处置费用
    贷：待处理财产损溢——待处理财产价值
```

**后续支出判断：**
- 改建/扩建/大修 → 计入资产成本
- 日常维修 → 当期费用

#### 3.2b 工程物资与在建工程

**工程物资：** 核算单位为在建工程购入的各种物资（建筑材料、设备等）。

```
购入工程物资：
借：工程物资
  贷：银行存款/零余额账户用款额度/财政拨款收入
预算会计：
借：行政/事业支出
  贷：资金结存/财政拨款预算收入

领用工程物资：
借：在建工程
  贷：工程物资
（预算会计不做）

工程完工后剩余物资退库：
借：库存物品
  贷：工程物资
（预算会计不做）
```

**在建工程：** 核算单位已经发生必要支出但尚未达到交付使用状态的各项建筑工程和安装工程。

```
发生建设支出（自营工程）：
借：在建工程
  贷：银行存款/工程物资/应付账款等
预算会计（实际支付时）：
借：行政/事业支出
  贷：资金结存

发生建设支出（出包工程，按进度付款）：
借：在建工程
  贷：银行存款/零余额账户用款额度
预算会计：
借：行政/事业支出
  贷：资金结存

工程完工交付使用：
借：固定资产/公共基础设施/保障性住房
  贷：在建工程
（预算会计不做）

已交付使用但尚未办理竣工决算：
按暂估价值入账，待办理竣工决算后按实际成本调整。
```

> **注意：** 为在建工程借入的专门借款利息，在工程达到预定可使用状态前，应计入在建工程成本（资本化）；达到预定可使用状态后的利息计入当期费用（费用化）。


#### 3.3 无形资产

**确认规则：**
- 购入软件（不构成硬件不可缺少组成部分）→ 无形资产
- 研究阶段支出 → 当期费用；开发阶段支出 → 先归集研发支出，形成无形资产则确认
- 自创商誉、品牌、报纸期刊名称 → **不确认**为无形资产
- 摊销：按月平均（直线法）

**取得无形资产：**

| 取得方式 | 财务会计 | 预算会计 |
|---------|---------|---------|
| 财政直接支付 | 借：无形资产<br>贷：财政拨款收入 | 借：行政/事业支出<br>贷：财政拨款预算收入 |
| 委托开发（预付款） | 借：预付账款<br>贷：零余额账户用款额度 | 借：行政/事业支出<br>贷：资金结存——零余额账户 |
| 委托开发（验收完成） | 借：无形资产<br>贷：预付账款、零余额账户用款额度 | 借：行政/事业支出（余款部分）<br>贷：资金结存——零余额账户 |

**摊销无形资产：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：业务活动费用/单位管理费用<br>贷：无形资产累计摊销 | **不做处理** |

**升级改造无形资产（扩展功能）：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：无形资产（升级改造支出）<br>贷：零余额账户用款额度/银行存款 | 借：行政/事业支出<br>贷：资金结存 |

#### 3.4 存货（库存物品）

**存货包括：** 在途物品、库存物品、加工物品。

**在途物品：** 已付款或已开出商业汇票但尚未验收入库的物品。
```
付款时：
借：在途物品
  贷：零余额账户用款额度/银行存款
预算会计：
借：行政/事业支出
  贷：资金结存

验收入库时：
借：库存物品
  贷：在途物品
（预算会计不做，付款时已记支出）
```

**加工物品：** 单位自制或委托外单位加工的各种物品。
```
委托加工——发出材料：
借：加工物品
  贷：库存物品
（预算会计不做）

委托加工——支付加工费：
借：加工物品
  贷：银行存款/零余额账户用款额度
预算会计：
借：行政/事业支出
  贷：资金结存

委托加工——收回物品：
借：库存物品
  贷：加工物品
（预算会计不做）
```

**库存物品：**

**计价：** 实际成本（购价+税费+运杂费）

**发出计价方法：**
- 行政单位：加权平均法
- 事业单位：先进先出法或加权平均法

**购入存货：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：库存物品（实际成本）<br>贷：财政拨款收入/零余额账户用款额度/银行存款 | 借：行政/事业支出<br>贷：财政拨款预算收入/资金结存 |

**领用存货：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：业务活动费用/单位管理费用<br>贷：库存物品 | **不做处理**（购入时已记支出） |

**存货盘盈：**
```
发现时：借：库存物品 / 贷：待处理财产损溢
批准后（流动资产）：借：待处理财产损溢 / 贷：单位管理费用（事业）或业务活动费用（行政）
```

**存货盘亏：**
```
发现时：借：待处理财产损溢 / 贷：库存物品
批准后：借：资产处置费用 / 贷：待处理财产损溢
```

#### 3.5 待摊费用与长期待摊费用

**待摊费用：** 已经支出但应由本期和以后各期负担的分摊期在**1年以内（含1年）**的各项费用。

```
支付时：
借：待摊费用
  贷：银行存款/零余额账户用款额度
预算会计：
借：行政/事业支出
  贷：资金结存

按期摊销：
借：业务活动费用/单位管理费用
  贷：待摊费用
（预算会计不做，支付时已记支出）
```

**长期待摊费用：** 已经支出但应由本期和以后各期负担的分摊期在**1年以上（不含1年）**的各项费用。具体包括经营租赁方式租入固定资产的改良支出等。

```
发生时：
借：长期待摊费用
  贷：银行存款/零余额账户用款额度
预算会计：
借：行政/事业支出
  贷：资金结存

按受益期间摊销：
借：业务活动费用/单位管理费用/经营费用
  贷：长期待摊费用
（预算会计不做）

不能使单位受益时（一次性转入费用）：
借：业务活动费用/单位管理费用/经营费用
  贷：长期待摊费用
（预算会计不做）
```

**示例（例9-91）：** 某单位4月10日租入设备一台，租期2年，租赁费240,000元，以财政授权方式支付，自下月开始摊销。

| 步骤 | 财务会计 | 预算会计 |
|------|---------|---------|
| 4月租入设备 | 借：长期待摊费用 240,000<br>贷：零余额账户用款额度 240,000 | 借：事业支出——财政拨款支出 240,000<br>贷：资金结存——零余额账户用款额度 240,000 |
| 5月起每月摊销（240,000÷24=10,000） | 借：业务活动费用——商品和服务费用 10,000<br>贷：长期待摊费用 10,000 | 不做处理 |

#### 3.6 公共基础设施

公共基础设施是指行政事业单位为满足社会公共需求而控制的，同时具有以下特征的有形资产：直接用于提供公共产品和服务；一般不产生直接现金流入；通常不会转让或处置。

包括：市政基础设施（道路、桥梁、隧道）、水利基础设施、交通基础设施、文化基础设施等。

**取得：** 参照固定资产取得的账务处理。
**折旧：** 参照固定资产计提折旧，折旧不触发预算会计。
**处置：** 参照固定资产处置。

#### 3.6 政府储备物资

政府储备物资是指政府为应对自然灾害、事故灾难、公共卫生事件、社会安全事件等突发事件，以及为保障国家安全和国民经济平稳运行而储备的各类物资。

**取得：**
```
【财务会计】
借：政府储备物资
  贷：财政拨款收入/零余额账户用款额度/银行存款

【预算会计】
借：行政/事业支出
  贷：财政拨款预算收入/资金结存
```

**动用（发出）：**
```
【财务会计】
借：业务活动费用
  贷：政府储备物资
【预算会计】不做处理（购入时已记支出）
```

**盘亏处理：**
```
发现时：
  借：待处理财产损溢——待处理财产价值
    贷：政府储备物资

批准处理时：
  借：资产处置费用
    贷：待处理财产损溢——待处理财产价值

取得变价收入：
  借：银行存款
    贷：待处理财产损溢——处理净收入

支付清理费用：
  借：待处理财产损溢——处理净收入
    贷：银行存款

尾款上缴财政：
  借：待处理财产损溢——处理净收入
    贷：应缴财政款——应缴国库款
```

#### 3.7 文物文化资产

文物文化资产是指用于展览、教育或研究等的历史文物、艺术品以及其他具有文化或历史价值并长期或永久保存的典藏等。

**注意：** 为满足自身开展业务活动需要而控制的文物和陈列品，通过"固定资产"科目核算，不通过本科目核算。

**取得：**
```
外购：
  借：文物文化资产
    贷：银行存款/应付账款

接受捐赠（有凭据）：
  借：文物文化资产（凭证金额）
    贷：捐赠收入、银行存款（相关费用）

无偿调入：
  借：文物文化资产（调出方账面价值+调入方费用）
    贷：无偿调拨净资产（差额）、银行存款（费用）
```

**处置（对外捐赠）：**
```
借：资产处置费用（账面余额+捐赠相关费用）
  贷：文物文化资产（账面余额）、银行存款（费用）
```

**处置（无偿调出）：**
```
借：无偿调拨净资产（账面余额）
  贷：文物文化资产
借：资产处置费用（调出相关费用）
  贷：银行存款
```

#### 3.8 保障性住房

保障性住房是指行政事业单位为满足社会公共需求而控制的用于居住保障目的的住房（廉租房、公租房、人才公寓等）。

**取得、折旧、后续支出：** 参照固定资产相关规定处理。

**出租收入上缴财政：**
```
借：银行存款
  贷：应缴财政款
```

#### 3.9 PPP项目资产

PPP（政府和社会资本合作）项目资产，适用于同时满足"双特征"（社会资本代表政府提供公共服务并获得补偿）和"双控制"（政府控制服务类型/对象/价格，且合同终止时政府控制重大剩余权益）的项目合同。

**取得（社会资本方投资建造，BOT模式）：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：PPP项目资产（确定的成本）<br>贷：PPP项目净资产 | **不做处理** |

**计提折旧（PPP项目净资产初始金额 = PPP项目资产初始金额时）：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：PPP项目净资产<br>贷：PPP项目资产累计折旧（摊销） | **不做处理** |

**计提折旧（PPP项目净资产初始金额 < PPP项目资产初始金额时）：**
```
设净资产/资产比例为X%
借：PPP项目净资产（折旧额×X%）
    业务活动费用（折旧额×(1-X%)）
  贷：PPP项目资产累计折旧（摊销）（折旧额）
```

**合同终止移交：**
```
借：公共基础设施（账面价值）
    PPP项目资产累计折旧（摊销）（已提折旧）
  贷：PPP项目资产（账面余额）
借：PPP项目净资产（尚未冲减完的余额）
  贷：累计盈余
```

#### 3.10 受托代理资产

受托代理资产是指行政事业单位接受委托方委托管理的各项非现金资产，包括受托指定转赠的物资、受托存储保管的物资、罚没物资等。

**收到受托物资：**
```
借：受托代理资产
  贷：受托代理负债
```

**转赠/发出/处置：**
```
借：受托代理负债
  贷：受托代理资产
```

**委托人取消转赠且不收回（转为单位资产）：**
```
借：受托代理负债
  贷：受托代理资产
借：库存物品/固定资产
  贷：其他收入
```

**罚没物资取得变价收入：**
```
借：银行存款
  贷：应缴财政款——应缴国库款
```


### 四、负债核算

#### 4.1 流动负债

**短期借款（事业单位特有）：**
```
借入：
  借：银行存款
    贷：短期借款

计提利息：
  借：其他费用——利息费用
    贷：应付利息
  （预算会计不做）

支付利息：
  借：应付利息
    贷：银行存款
  预算会计：
  借：其他支出——其他资金支出
    贷：资金结存——货币资金

归还本金：
  借：短期借款
    贷：银行存款
  预算会计：
  借：债务还本支出
    贷：资金结存——货币资金
```

**应付账款：**

核算单位因购买物资、接受服务等应付而未付的款项。

```
收到物资/服务，尚未付款：
借：库存物品/固定资产/业务活动费用等
  贷：应付账款

偿付应付账款：
借：应付账款
  贷：银行存款/零余额账户用款额度
预算会计：
借：行政/事业支出
  贷：资金结存

开出商业汇票抵付（事业单位）：
借：应付账款
  贷：应付票据

确实无法偿付的应付账款（报经批准核销）：
借：应付账款
  贷：其他收入
（预算会计不做）
```

> **注意：** 核销后应在备查簿中保留登记，以备日后查询。

**预收账款（事业单位特有）：**

核算事业单位按合同约定预收的款项。

```
收到预收款项：
借：银行存款
  贷：预收账款
预算会计：
借：资金结存——货币资金
  贷：事业预算收入/经营预算收入

确认收入（提供服务/交付商品后）：
借：预收账款
  贷：事业收入/经营收入
（预算会计不做，预收时已确认预算收入）

需退回预收款项：
借：预收账款
  贷：银行存款
预算会计：
借：事业预算收入/经营预算收入（冲回）
  贷：资金结存——货币资金

确实无法偿付的预收账款（报经批准核销）：
借：预收账款
  贷：其他收入
（预算会计不做，预收时已确认预算收入）
```

**应付职工薪酬：**
```
计提：
  借：业务活动费用/单位管理费用
    贷：应付职工薪酬
  （预算会计不做）

实际发放（财政直接支付）：
  借：应付职工薪酬
    贷：财政拨款收入
  预算会计：
  借：行政/事业支出——财政拨款支出
    贷：财政拨款预算收入
```

**应缴财政款：** 核算单位应上缴财政的各项款项（资产处置收入、罚没收入等）。
```
确认应缴：
  借：银行存款/资产处置费用等
    贷：应缴财政款

实际上缴：
  借：应缴财政款
    贷：银行存款/零余额账户用款额度
```

**应付政府补贴款（行政单位特有）：** 核算行政单位代财政向有关单位或个人发放的政府补贴款。

**预提费用：** 核算已计入费用但尚未支付的各项费用（如预提租金、保险费等）。
```
预提时：
借：业务活动费用/单位管理费用
  贷：预提费用
（预算会计不做）

实际支付时：
借：预提费用
  贷：银行存款
预算会计：
借：行政/事业支出
  贷：资金结存
```

**应付票据（事业单位特有）：**

核算事业单位因购买物资等开出的商业汇票。

```
开出承兑汇票：
借：库存物品/固定资产等
  贷：应付票据

到期支付票款：
借：应付票据
  贷：银行存款
预算会计：
借：事业支出/经营支出
  贷：资金结存——货币资金

到期无力支付（转为短期借款）：
借：应付票据
  贷：短期借款
（预算会计不做，无现金流出）
```

> **注意：** 应付票据应设置备查簿，逐笔登记每笔票据的种类、号数、签发日期、到期日、票面金额、收款人等。

#### 4.2 应交增值税（详细规则）

行政事业单位按纳税主体分为**增值税一般纳税人**和**增值税小规模纳税人**。

**一般纳税人明细科目设置：**
- 应交税金（含进项税额、已交税金、销项税额、进项税额转出、转出未交增值税等专栏）
- 未交税金
- 预交税金
- 待抵扣进项税额
- 待认证进项税额
- 待转销项税额
- 简易计税
- 转让金融商品应交增值税
- 代扣代缴增值税

**小规模纳税人：** 只需设置"转让金融商品应交增值税"和"代扣代缴增值税"明细科目。

**进项税额可抵扣（一般纳税人购入用于应税项目）：**
```
借：库存物品/固定资产/费用等（不含税金额）
    应交增值税——应交税金（进项税额）（当月已认证）
    应交增值税——待认证进项税额（未认证）
  贷：应付账款/银行存款/零余额账户用款额度等（含税金额）
```

**进项税额不得抵扣（用于免税项目、集体福利等）：**
```
取得专用发票时：
借：库存物品/费用等（不含税）
    应交增值税——待认证进项税额（税额）
  贷：银行存款等

税务机关认证为不可抵扣时：
借：应交增值税——应交税金（进项税额）
  贷：应交增值税——待认证进项税额
借：库存物品/费用等（税额转入成本）
  贷：应交增值税——应交税金（进项税额转出）
```

**行政单位购入行政办公用品（一般纳税人，财政直接支付）：**
```
增值税专用发票：材料价款20,000元，增值税2,600元
借：库存物品 22,600
  贷：财政拨款收入 22,600
预算会计：
借：行政支出——财政拨款支出 22,600
  贷：财政拨款预算收入 22,600
```
> 行政单位购入行政办公用品，进项税额不得抵扣，直接计入库存物品成本。

**事业单位购入经营用品（一般纳税人，银行存款支付）：**
```
增值税专用发票：材料价款10,000元，增值税1,300元
借：库存物品 10,000
    应交增值税——应交税金（进项税额）1,300
  贷：银行存款 11,300
预算会计：
借：经营支出 10,000
  贷：资金结存——货币资金 10,000
```

**事业单位购入经营用品（小规模纳税人）：**
```
借：库存物品 11,300（含税全额入账）
  贷：银行存款 11,300
预算会计：
借：经营支出 11,300
  贷：资金结存——货币资金 11,300
```

**销售货物/提供服务（一般纳税人）：**
```
借：银行存款/应收账款
  贷：经营收入/事业收入（不含税）
      应交增值税——应交税金（销项税额）
```

**月末结转未交增值税：**
```
当月销项税额 > 进项税额（应补缴）：
借：应交增值税——应交税金（转出未交增值税）
  贷：应交增值税——未交税金

实际缴纳：
借：应交增值税——未交税金
  贷：银行存款
```

#### 4.3 非流动负债

**长期借款（事业单位特有）：**
```
借入：
  借：银行存款
    贷：长期借款
  预算会计：
  借：资金结存——货币资金
    贷：债务预算收入

归还本金：
  借：长期借款
    贷：银行存款
  预算会计：
  借：债务还本支出
    贷：资金结存——货币资金
```

**长期应付款：** 核算偿还期限超过1年的应付款项，主要是融资租赁形成的应付租赁款。

```
融资租入固定资产（分期付款）：
借：固定资产（合同总价款）
  贷：长期应付款（合同总价款）
（预算会计不做，无现金流出）

分期支付款项：
借：长期应付款
  贷：银行存款
预算会计：
借：事业支出/其他支出
  贷：资金结存——货币资金

确实无法偿付的长期应付款（报经批准核销）：
借：长期应付款
  贷：其他收入
（预算会计不做）
```

**示例（例10-23）：** 某事业单位分期付款购入设备6,000,000元，分3年支付，每年支付2,000,000元。

| 步骤 | 财务会计 | 预算会计 |
|------|---------|---------|
| 购入设备 | 借：固定资产 6,000,000<br>贷：长期应付款 6,000,000 | 不做处理 |
| 每年支付2,000,000 | 借：长期应付款 2,000,000<br>贷：银行存款 2,000,000 | 借：事业支出 2,000,000<br>贷：资金结存——货币资金 2,000,000 |

**预计负债：** 核算因或有事项等确认的预计负债。

确认条件（同时满足）：
1. 该义务是单位承担的现时义务
2. 履行该义务很可能导致经济资源流出
3. 该义务的金额能够可靠计量

```
确认预计负债：
借：业务活动费用/其他费用
  贷：预计负债
（预算会计不做）

实际支付时：
借：预计负债
  贷：银行存款
预算会计：
借：其他支出
  贷：资金结存——货币资金

预计负债不再需要（冲回）：
借：预计负债
  贷：业务活动费用/其他费用
（预算会计不做）
```

#### 4.3 受托代理负债

与受托代理资产对应，收到受托物资时确认，转出时消除。

```
收到：借：受托代理资产 / 贷：受托代理负债
转出：借：受托代理负债 / 贷：受托代理资产
```

#### 4.4 现金短缺与溢余处理

| 情形 | 财务会计 | 预算会计 |
|------|---------|---------|
| 发现短缺 | 借：待处理财产损溢 / 贷：库存现金 | 借：其他支出 / 贷：资金结存——货币资金 |
| 认定责任赔偿 | 借：其他应收款 / 贷：待处理财产损溢 | 不做 |
| 收到赔偿款 | 借：库存现金 / 贷：其他应收款 | 借：资金结存——货币资金 / 贷：其他支出（冲） |
| 无法查明核销 | 借：资产处置费用 / 贷：待处理财产损溢 | 不做 |
| 发现溢余（应付） | 借：待处理财产损溢 / 贷：其他应付款 | 借：其他预算收入 / 贷：资金结存（冲） |
| 溢余无法查明 | 借：待处理财产损溢 / 贷：其他收入 | 不做 |


### 五、收入与预算收入核算

#### 5.1 财政拨款收入——财政直接支付

财政直接向供应商付款，单位收到"财政直接支付入账通知书"。

| 财务会计 | 预算会计 |
|---------|---------|
| 借：库存物品/固定资产/业务活动费用等<br>贷：财政拨款收入 | 借：行政/事业支出——财政拨款支出<br>贷：财政拨款预算收入 |

#### 5.2 财政拨款收入——财政授权支付

财政向单位零余额账户拨付额度，单位用额度付款，收到"额度到账通知书"。

**收到额度通知书：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：零余额账户用款额度<br>贷：财政拨款收入 | 借：资金结存——零余额账户用款额度<br>贷：财政拨款预算收入 |

**实际使用额度支付：**

| 财务会计 | 预算会计 |
|---------|---------|
| 借：费用/资产<br>贷：零余额账户用款额度 | 借：行政/事业支出——财政拨款支出<br>贷：资金结存——零余额账户用款额度 |

#### 5.3 事业收入（事业单位特有）

事业单位开展专业业务活动及其辅助活动取得的收入。

```
【财务会计】
借：银行存款/应收账款
  贷：事业收入

【预算会计】
借：资金结存——货币资金
  贷：事业预算收入
```

#### 5.4 经营收入（事业单位特有）

事业单位在专业业务活动及其辅助活动之外开展非独立核算经营活动取得的收入。

```
【财务会计】
借：银行存款/应收账款
  贷：经营收入

【预算会计】
借：资金结存——货币资金
  贷：经营预算收入
```

#### 5.5 非同级财政拨款收入

单位取得的非同级财政部门拨付的财政性资金。

```
【财务会计】
借：银行存款/零余额账户用款额度
  贷：非同级财政拨款收入

【预算会计】
借：资金结存
  贷：非财政拨款预算收入（专项）或其他预算收入
```

#### 5.6 捐赠收入、利息收入、租金收入、其他收入

```
【财务会计】
借：银行存款/固定资产等
  贷：捐赠收入/利息收入/租金收入/其他收入

【预算会计】
借：资金结存——货币资金
  贷：其他预算收入（或投资预算收益）
```

#### 5.7 上级补助收入（事业单位特有）

事业单位从主管部门和上级单位取得的非财政补助收入。

```
【财务会计】
借：银行存款
  贷：上级补助收入

【预算会计】
借：资金结存——货币资金
  贷：其他预算收入
```

#### 5.8 投资收益（事业单位特有）

**权益法确认（账面调整，不触发预算会计）：**
```
【财务会计】
借：长期股权投资——损益调整
  贷：投资收益
【预算会计】不做处理
```

**实际收到股利/利息：**
```
【财务会计】
借：银行存款
  贷：应收股利/应收利息

【预算会计】
借：资金结存——货币资金
  贷：投资预算收益
```


### 六、费用与预算支出核算

#### 6.1 费用科目体系

| 科目 | 适用范围 | 核算内容 |
|------|---------|---------|
| 业务活动费用 | 行政+事业 | 履行职能目标所发生的费用（工资福利、商品服务、折旧摊销等） |
| 单位管理费用 | 事业单位特有 | 管理活动发生的费用 |
| 经营费用 | 事业单位特有 | 非独立核算经营活动发生的费用 |
| 资产处置费用 | 行政+事业 | 处置各类资产的账面价值及相关费用，含盘亏核销净损失 |
| 上缴上级费用 | 事业单位特有 | 按规定上缴上级单位款项 |
| 对附属单位补助费用 | 事业单位特有 | 用非财政资金对附属单位补助 |
| 所得税费用 | 事业单位特有 | 应缴企业所得税 |
| 其他费用 | 行政+事业 | 利息支出、捐赠支出、现金盘亏、罚没支出等 |

**费用 vs 预算支出的核心差异：**

| 事项 | 预算支出时点 | 费用确认时点 |
|------|------------|------------|
| 购固定资产 | 购入时一次性记支出 | 后续每期计提折旧 |
| 工资 | 实际发放时 | 计提时 |
| 利息 | 实际支付时 | 按期计提时 |
| 购存货 | 购入时记支出 | 领用时转费用 |

#### 6.2 预算支出科目体系

| 科目 | 适用范围 | 核算内容 |
|------|---------|---------|
| 行政支出 | 行政单位 | 行政单位开展业务活动实际发生的各项现金流出 |
| 事业支出 | 事业单位 | 事业单位开展专业业务活动及辅助活动实际发生的各项现金流出 |
| 经营支出 | 事业单位 | 非独立核算经营活动实际发生的各项现金流出 |
| 上缴上级支出 | 事业单位 | 上缴上级单位款项的现金流出 |
| 对附属单位补助支出 | 事业单位 | 对附属单位补助的现金流出 |
| 投资支出 | 事业单位 | 以货币资金对外投资的现金流出 |
| 债务还本支出 | 事业单位 | 偿还借款本金的现金流出 |
| 其他支出 | 行政+事业 | 利息支出、捐赠、现金盘亏、罚没等现金流出 |

#### 6.3 行政支出明细核算维度

行政支出按以下维度设置明细科目：
1. 支出类型：财政拨款支出、非财政专项资金支出、其他资金支出
2. 资金来源：一般公共预算财政拨款、政府性基金预算财政拨款
3. 支出性质：基本支出、项目支出
4. 功能分类：按《政府收支分类科目》项级科目
5. 经济分类：按部门预算支出经济分类款级科目

**年末结转规则：**
```
财政拨款支出 → 财政拨款结转——本年收支结转
非财政专项资金支出 → 非财政拨款结转——本年收支结转
其他资金支出 → 其他结余
```

### 七、净资产与预算结余核算（第13章）

#### 7.1 财务会计净资产科目

| 科目 | 适用范围 | 核算内容 |
|------|---------|---------|
| 累计盈余 | 行政+事业 | 历年实现的盈余扣除盈余分配后滚存的金额，以及无偿调入调出资产产生的净资产变动 |
| 专用基金 | 事业单位特有 | 按规定提取或设置的有专门用途的资金（职工福利基金、医疗基金等） |
| 权益法调整 | 事业单位特有 | 按持股比例享有的被投资单位除净损益和利润分配以外的所有者权益变动份额 |
| 本期盈余 | 行政+事业 | 本期各项收入、费用相抵后的余额，年末结转后无余额 |
| 本年盈余分配 | 行政+事业 | 本年度盈余分配情况，年末结转后无余额 |
| 无偿调拨净资产 | 行政+事业 | 无偿调入或调出非现金资产引起的净资产变动，年末转入累计盈余 |
| 以前年度盈余调整 | 行政+事业 | 调整以前年度盈余的事项，年末转入累计盈余后无余额 |

#### 7.2 本期盈余年末结转流程

**第一步：收入类科目余额转入本期盈余（贷方）**
```
借：财政拨款收入
    事业收入
    经营收入
    非同级财政拨款收入
    投资收益
    捐赠收入
    利息收入
    租金收入
    其他收入
  贷：本期盈余
```

**第二步：费用类科目余额转入本期盈余（借方）**
```
借：本期盈余
  贷：业务活动费用
      单位管理费用（事业单位）
      经营费用（事业单位）
      资产处置费用
      上缴上级费用（事业单位）
      对附属单位补助费用（事业单位）
      所得税费用（事业单位）
      其他费用
```

**第三步：本期盈余转入本年盈余分配**
```
借：本期盈余
  贷：本年盈余分配
```

**第四步：本年盈余分配转入累计盈余**
```
借：本年盈余分配
  贷：累计盈余
```

**第五步：无偿调拨净资产转入累计盈余**
```
借/贷：无偿调拨净资产
  贷/借：累计盈余
```

**第六步：以前年度盈余调整转入累计盈余**
```
借/贷：以前年度盈余调整
  贷/借：累计盈余
```

**示例（例13-1、13-2、13-3）：**

某行政单位12月财务会计收入科目贷方发生额：财政拨款收入2,000,000元，非同级财政拨款收入400,000元，捐赠收入20,000元，利息收入10,000元，租金收入30,000元，其他收入550,000元。费用科目借方发生额：业务活动费用195,000元，资产处置费用56,000元，其他费用54,000元。1—11月本期盈余贷方累计500,000元。

| 步骤 | 财务会计 | 预算会计 |
|------|---------|---------|
| 结转收入 | 借：财政拨款收入 2,000,000<br>　　非同级财政拨款收入 400,000<br>　　捐赠收入 20,000<br>　　利息收入 10,000<br>　　租金收入 30,000<br>　　其他收入 550,000<br>贷：本期盈余 3,010,000 | 不做处理 |
| 结转费用 | 借：本期盈余 305,000<br>贷：业务活动费用 195,000<br>　　资产处置费用 56,000<br>　　其他费用 54,000 | 不做处理 |
| 本期盈余转本年盈余分配（含1-11月） | 借：本期盈余 3,205,000<br>贷：本年盈余分配 3,205,000 | 不做处理 |
| 本年盈余分配转累计盈余 | 借：本年盈余分配 3,205,000<br>贷：累计盈余 3,205,000 | 不做处理 |

#### 7.3 以前年度盈余调整

核算本年度发生的调整以前年度盈余的事项（重要前期差错更正）。

**调整增加以前年度费用（例13-4）：**
某行政单位发现上年11月领用库存物品成本少记4,500元。

| 财务会计 | 预算会计 |
|---------|---------|
| 借：以前年度盈余调整 4,500<br>贷：库存物品 4,500 | 不做处理 |

年末结转：
```
借：累计盈余 4,500
  贷：以前年度盈余调整 4,500
```

#### 7.4 预算结余科目体系

| 科目 | 核算内容 |
|------|---------|
| 资金结存 | 货币资金结存 + 零余额账户用款额度结存 + 财政应返还额度结存 |
| 财政拨款结转 | 财政拨款资金中尚未完成、需延续使用的支出 |
| 财政拨款结余 | 财政拨款资金中已完成任务后的剩余资金 |
| 非财政拨款结转 | 非财政专项资金中尚未完成的支出 |
| 非财政拨款结余 | 非财政拨款资金中的结余 |
| 经营结余 | 经营收入扣除经营支出后的余额 |
| 其他结余 | 其他资金收支相抵后的余额 |

#### 7.5 财政拨款结转与结余

**财政拨款结转：** 当年预算已执行但尚未完成，或因故未执行，需延续到下年按原用途继续使用的财政拨款资金。

**财政拨款结余：** 当年预算工作目标已完成，或因故终止后剩余的财政拨款资金。

**年末结转流程：**

第一步：将财政拨款预算收入转入财政拨款结转
```
借：财政拨款预算收入
  贷：财政拨款结转——本年收支结转
```

第二步：将财政拨款支出转入财政拨款结转
```
借：财政拨款结转——本年收支结转
  贷：行政/事业支出——财政拨款支出
```

第三步：将本年收支结转余额分配为结转或结余
```
未完成项目（结转）：
借：财政拨款结转——本年收支结转
  贷：财政拨款结转——基本支出结转/项目支出结转

已完成项目（结余）：
借：财政拨款结转——本年收支结转
  �dat：财政拨款结余
```

#### 7.6 非财政拨款结转与结余

**年末结转流程：**
```
第一步：将非财政专项资金预算收入转入非财政拨款结转
借：非财政拨款预算收入（专项）
  贷：非财政拨款结转——本年收支结转

第二步：将非财政专项资金支出转入非财政拨款结转
借：非财政拨款结转——本年收支结转
  贷：行政/事业支出——非财政专项资金支出

第三步：分配结转或结余
未完成：借：非财政拨款结转——本年收支结转 / 贷：非财政拨款结转——项目结转
已完成：借：非财政拨款结转——本年收支结转 / 贷：非财政拨款结余
```

#### 7.7 其他结余与经营结余

**其他结余年末结转：**
```
将其他资金预算收入转入其他结余：
借：其他预算收入/投资预算收益
  贷：其他结余

将其他资金支出转入其他结余：
借：其他结余
  贷：行政/事业支出——其他资金支出/其他支出——其他资金支出/
      上缴上级支出/对附属单位补助支出/投资支出/债务还本支出

将其他结余转入非财政拨款结余：
借：其他结余
  贷：非财政拨款结余
```

**经营结余年末结转：**
```
将经营预算收入转入经营结余：
借：经营预算收入
  贷：经营结余

将经营支出转入经营结余：
借：经营结余
  贷：经营支出

经营结余如为贷方余额（盈利），转入非财政拨款结余：
借：经营结余
  贷：非财政拨款结余
```

2. **合计填列法**：根据几个总账科目期末余额合计填列（如货币资金=库存现金+银行存款+其他货币资金）

#### 8.3b 现金流量表编制方法

现金流量表反映单位在一定会计期间内现金及现金等价物流入和流出的情况。行政事业单位可根据实际情况需要，以收付实现制为基础，自行选择编制。

**三类现金流量：**

**一、业务活动产生的现金流量**

| 项目 | 填列依据 |
|------|---------|
| 财政拨款收到的现金 | "财政拨款收入"科目本期发生额中实际收到现金的部分 |
| 事业活动收到的现金 | "事业收入""上级补助收入"等科目中实际收到现金的部分 |
| 经营活动收到的现金 | "经营收入"科目中实际收到现金的部分 |
| 其他业务活动收到的现金 | 利息收入、租金收入、捐赠收入等实际收到的现金 |
| 购买商品/接受劳务支付的现金 | 根据"库存物品""应付账款""预付账款"等科目分析 |
| 支付给职工的现金 | "应付职工薪酬"科目借方发生额中实际支付的部分 |
| 支付的各项税费 | "应交增值税""其他应交税费"科目借方发生额 |
| 支付的其他与业务活动有关的现金 | 差旅费、会议费、培训费等实际支付的现金 |

**二、投资活动产生的现金流量**

| 项目 | 填列依据 |
|------|---------|
| 收回投资收到的现金 | 出售短期投资、长期股权投资、长期债券投资收到的现金 |
| 取得投资收益收到的现金 | 实际收到的股利、利息等 |
| 处置固定资产等收到的现金 | 出售固定资产、无形资产等收到的现金净额 |
| 购建固定资产等支付的现金 | 购买固定资产、无形资产、在建工程等支付的现金 |
| 对外投资支付的现金 | 购买短期投资、长期股权投资、长期债券投资支付的现金 |

**三、筹资活动产生的现金流量**

| 项目 | 填列依据 |
|------|---------|
| 借款收到的现金 | 取得短期借款、长期借款收到的现金 |
| 偿还借款本金支付的现金 | 偿还短期借款、长期借款本金支付的现金 |
| 偿还借款利息支付的现金 | 实际支付的借款利息 |
| 其他筹资活动支付的现金 | 融资租赁支付的租金等 |

**现金流量表勾稽关系：**
```
现金及现金等价物净增加额 = 业务活动产生的现金流量净额
                         + 投资活动产生的现金流量净额
                         + 筹资活动产生的现金流量净额

期末现金及现金等价物余额 = 期初余额 + 净增加额
```

#### 8.4 预算执行情况表体系

| 报表 | 内容 |
|------|------|
| 预算收入支出表 | 预算收入、预算支出、预算结余 |
| 财政拨款预算收入支出表 | 财政拨款预算收支及结转结余 |

#### 8.5 本年盈余与预算结余差异说明

财务报告中需编制"本年盈余与预算结余差异情况说明"，解释两套报表差异原因。

**主要差异项：**

| 差异项 | 影响 |
|--------|------|
| 当期计提折旧/摊销（只有财务会计） | 本期盈余 < 预算结余 |
| 当期计提工资（只有财务会计） | 本期盈余 < 预算结余 |
| 当期购入固定资产（预算支出一次性记，财务会计分期折旧） | 本期盈余 > 预算结余 |
| 当期计提利息（只有财务会计） | 本期盈余 < 预算结余 |
| 当期实际支付利息（只有预算会计） | 预算结余 < 本期盈余 |
| 权益法确认投资收益（只有财务会计） | 本期盈余 > 预算结余 |
| 实际收到股利（只有预算会计） | 预算结余 > 本期盈余 |

#### 8.6 财政资金绩效评价（第14章第三节）

**概念：** 财政资金绩效评价是指财政部门和预算部门（单位）根据设定的绩效目标，运用科学合理的绩效评价指标、评价标准和评价方法，对财政支出的经济性、效率性和效益性进行客观、公正的评价。

**评价对象：** 纳入政府预算管理的资金和纳入部门预算管理的资金。

**评价内容：**
- 绩效目标的设定情况
- 资金投入和使用情况
- 为实现绩效目标制定的制度、采取的措施等
- 绩效目标的实现程度及效果
- 绩效评价的其他内容

**评价指标体系（四维度）：**

| 维度 | 主要指标 |
|------|---------|
| 投入 | 目标设定（绩效目标合理性、指标明确性）、预算配置（资金到位率、预算执行率） |
| 过程 | 资金管理（资金使用合规性、财务监控有效性）、组织实施（管理制度健全性、制度执行有效性） |
| 产出 | 产出数量、产出质量、产出时效、产出成本 |
| 效果 | 经济效益、社会效益、生态效益、可持续影响、服务对象满意度 |

**评价方法：**
- 成本效益分析法：将投入与产出进行比较
- 比较法：与计划标准、历史标准、行业标准等进行比较
- 因素分析法：综合分析影响绩效目标实现的内外因素
- 最低成本法：在保证功能前提下选择成本最低方案
- 公众评判法：通过问卷调查等方式征求公众意见

**评价结果应用：**
- 作为改进预算管理和安排以后年度预算的重要依据
- 作为对预算部门和单位实施行政问责的重要依据
- 评价结果向同级政府报告，并向社会公开

#### 8.7 行政事业单位划转撤并相关会计处理

依据：财会〔2022〕29号《行政事业单位划转撤并相关会计处理规定》

**单位撤销（清算）：**
- 清算期间继续按原会计制度核算
- 清算结束后，将剩余资产和负债移交接收单位
- 移交资产：借：无偿调拨净资产 / 贷：各资产科目（净值）
- 移交负债：借：各负债科目 / 贷：无偿调拨净资产

**单位合并：**
- 合并方接收被合并方资产负债，按账面价值入账
- 接收资产：借：各资产科目 / 贷：无偿调拨净资产（差额）、各负债科目

**单位分立：**
- 分立后各单位按分配的资产负债建账
- 处理方式参照合并，按账面价值划转

## 第三篇：民间非营利组织会计

### 一、概述与特点（第15章）

**适用范围：** 依照国家法律、行政法规登记的社会团体、基金会、民办非企业单位和寺院、宫观、清真寺、教堂等。

**三个基本特征：**
1. 不以营利为宗旨和目的
2. 资源提供者向该组织投入资源不以取得经济回报为目标
3. 资源提供者不享有该组织的所有权

**会计特点：**
- 单一核算基础：权责发生制（无预算会计）
- 计量属性：历史成本为主，引入公允价值
- 会计要素：资产、负债、净资产、收入、费用（无所有者权益和利润）
- 净资产分类：**限定性净资产**（有时间或用途限制）+ **非限定性净资产**（无限制）

**与行政事业单位的核心区别：**

| 对比项 | 行政事业单位 | 民间非营利组织 |
|--------|------------|--------------|
| 核算基础 | 双基础（财务+预算） | 单一权责发生制 |
| 净资产分类 | 累计盈余等 | 限定性 + 非限定性净资产 |
| 预算会计 | 有 | 无 |
| 报表体系 | 财务报告 + 决算报告 | 资产负债表 + 业务活动表 + 现金流量表 |
| 收入来源 | 财政拨款为主 | 捐赠、会费、服务收入、政府补助等 |

### 三、资产核算（第16章）

#### 3.1 资产确认与计量原则

- 取得时按**实际成本**计量
- 定期（至少每年年末）对短期投资、应收账款、存货、长期投资等进行减值测试
- 发生减值 → 计提减值准备，计入当期费用
- 减值回升 → 在已计提准备范围内转回，冲减当期费用

**接受捐赠资产的计价规则：**
- 捐赠方提供了发票等凭证 → 按凭证金额入账（若凭证金额与公允价值相差较大，按公允价值）
- 捐赠方未提供凭证 → 按公允价值入账
- 公允价值无法可靠计量 → 设辅助账登记，待能可靠计量时再确认

**公允价值确定顺序：**
1. 同类或类似资产存在活跃市场 → 按市场价格
2. 无活跃市场 → 采用合理计价方法

#### 3.2 捐赠取得的资产

**接受捐赠现金：**
```
借：银行存款（实际收到金额）
  贷：捐赠收入——限定性收入 或 非限定性收入
```

**接受捐赠非现金资产（有凭据）：**
```
借：固定资产/存货/长期股权投资等（凭证金额）
  贷：捐赠收入——限定性收入 或 非限定性收入（差额）
      银行存款（相关税费、运输费等）
```

**接受捐赠非现金资产（无凭据，按公允价值）：**
```
借：固定资产/存货等（公允价值）
  贷：捐赠收入——限定性收入 或 非限定性收入（差额）
      银行存款（相关费用）
```

**示例（例16-1）：** 某民办医院接受捐赠设备一台，市场价值20,000元，运费900元以银行存款支付。
```
借：固定资产 20,900
  贷：捐赠收入——非限定性收入 20,000
      银行存款 900
```

#### 3.3 受托代理资产

受托代理资产是指民间非营利组织接受委托方委托从事受托代理业务而收到的资产。组织本身只起中介作用，无权改变用途或变更受益人。

**收到受托代理资产：**
```
借：受托代理资产（按捐赠资产计价原则确认）
  贷：受托代理负债
```

**转赠或转出：**
```
借：受托代理负债
  贷：受托代理资产
```

**受托代理资产为现金时：** 可不通过"受托代理资产"科目，在"银行存款"下设"受托代理资产"明细核算。

**示例（例16-2）：** 甲民间非营利组织受丙企业委托，将60台电子钢琴（每台2,000元，共120,000元）转赠给乙下属10家儿童福利院。

收到电子钢琴时：
```
借：受托代理资产——电子钢琴 120,000
  贷：受托代理负债 120,000
```

将电子钢琴送至各福利院时：
```
借：受托代理负债 120,000
  贷：受托代理资产——电子钢琴 120,000
```

#### 3.4 文物文化资产

**取得：**

外购（一次购入多项，按公允价值比例分摊总成本）：
```
借：文物文化资产——陶器类/古字画/编钟等
  贷：银行存款/应付账款
```

接受捐赠（有凭据）：
```
借：文物文化资产（凭证金额）
  贷：捐赠收入
      银行存款（相关费用）
```

接受捐赠（无凭据，按公允价值）：
```
借：文物文化资产（公允价值）
  贷：捐赠收入
```

**处置（出售）：**
```
转入清理：
借：固定资产清理（账面余额）
  贷：文物文化资产

收到出售收入：
借：银行存款
  贷：固定资产清理

支付相关费用：
借：固定资产清理
  贷：应交税金/银行存款

结转净收益：
借：固定资产清理（净收益）
  贷：其他收入——非限定性收入
```

**盘盈：**
```
借：文物文化资产（公允价值）
  贷：其他收入——非限定性收入
```

**盘亏：**
```
借：其他应收款——保险赔偿（可收回部分）
    其他应收款——责任人（可收回部分）
    管理费用（净损失）
  贷：文物文化资产（账面余额）
```

#### 3.5 固定资产

民间非营利组织固定资产的核算与企业基本一致，主要差异在于：
- 取得时按实际成本入账
- 接受捐赠的固定资产按捐赠资产计价规则确认
- 折旧计入管理费用或业务活动成本

**取得（外购）：**
```
借：固定资产
  贷：银行存款/应付账款
```

**计提折旧：**
```
借：管理费用/业务活动成本
  贷：累计折旧
```

**处置：**
```
借：固定资产清理（账面价值）
    累计折旧（已提折旧）
  贷：固定资产（原值）

收到价款：
借：银行存款
  贷：固定资产清理

净收益：
借：固定资产清理
  贷：其他收入——非限定性收入

净损失：
借：管理费用
  贷：固定资产清理
```


### 四、负债核算（第16章）

#### 4.1 流动负债

**短期借款：**
```
借入：
  借：银行存款
    贷：短期借款

计提利息：
  借：筹资费用
    贷：预提费用

到期还本付息：
  借：短期借款
      筹资费用（最后一期利息）
    贷：银行存款
```

**示例（例16-8）：** 民间非营利组织向银行借款40,000元，到期偿还本金和利息160元。
```
借入：
借：银行存款 40,000
  贷：短期借款 40,000

到期还本付息：
借：短期借款 40,000
    筹资费用 160
  贷：银行存款 40,160
```

**应付工资：**
```
计提：
借：管理费用/业务活动成本
  贷：应付工资

发放：
借：应付工资
  贷：银行存款/现金
```

**预提费用：** 按月预提的租金、保险费、借款利息等。
```
借：管理费用/筹资费用
  贷：预提费用

实际支付：
借：预提费用
  贷：银行存款
```

**预计负债：** 因担保、诉讼等或有事项确认的预计负债。
```
确认：
借：管理费用
  贷：预计负债

实际支付：
借：预计负债
  贷：银行存款
```

#### 4.2 受托代理负债

与受托代理资产对应，按相对应的受托代理资产金额确认和计量。

```
收到受托代理资产：
借：受托代理资产
  贷：受托代理负债

转出受托代理资产：
借：受托代理负债
  贷：受托代理资产
```

### 五、收入核算（第17章）

#### 5.1 收入分类

**按来源：** 捐赠收入、会费收入、提供服务收入、政府补助收入、投资收益、商品销售收入、其他收入

**按是否受限制：**
- **限定性收入**：资产提供者设置了时间限制或用途限制
- **非限定性收入**：无限制

**按是否为交换交易：**
- 交换交易收入：商品销售收入、提供服务收入、投资收益
- 非交换交易收入：捐赠收入、政府补助收入、会费收入

> **重要规则：** 各类收入科目均需设置"限定性收入"和"非限定性收入"明细科目。期末将各明细科目余额分别转入限定性净资产或非限定性净资产。

#### 5.2 捐赠收入

**接受捐赠（非限定性）：**
```
借：银行存款/固定资产/存货等
  贷：捐赠收入——非限定性收入
```

**接受捐赠（限定性）：**
```
借：银行存款/固定资产/存货等
  贷：捐赠收入——限定性收入
```

**接受附条件捐赠（存在需偿还义务）：**
```
借入时：
借：银行存款
  贷：捐赠收入——限定性收入

确认可能需偿还部分：
借：管理费用
  贷：其他应付款
```

**限制解除（限定性转非限定性）：**
```
借：捐赠收入——限定性收入
  贷：捐赠收入——非限定性收入
```

**期末结转：**
```
借：捐赠收入——非限定性收入
  贷：非限定性净资产

借：捐赠收入——限定性收入
  贷：限定性净资产
```

**示例（例17-1）：** 某基金会收到华侨捐赠汽车一辆（发票价值120,000元，运费4,000元以银行存款支付），另收到捐款300,000元（限定1年内用于救助留守儿童，余额退还，预计可用250,000元）。

```
接受汽车捐赠：
借：固定资产 124,000
  贷：捐赠收入——非限定性收入 120,000
      银行存款 4,000

接受限定性捐款：
借：银行存款 300,000
  贷：捐赠收入——限定性收入 300,000

确认可能需偿还的50,000元：
借：管理费用 50,000
  贷：其他应付款 50,000
```

#### 5.3 会费收入

会费收入一般为非限定性收入，按权责发生制确认。

```
收取会费：
借：银行存款/应收账款
  贷：会费收入——非限定性收入

期末结转：
借：会费收入——非限定性收入
  贷：非限定性净资产
```

#### 5.4 提供服务收入

提供服务收入一般为非限定性收入（学杂费、医疗费、培训收入等）。

```
确认收入：
借：银行存款/应收账款
  贷：提供服务收入——非限定性收入

预收款项：
借：银行存款
  贷：预收账款

后续确认：
借：预收账款
  贷：提供服务收入——非限定性收入

期末结转：
借：提供服务收入——非限定性收入
  贷：非限定性净资产
```

**示例（例17-2）：** 某民间医疗组织本月收到医疗费收入8,000元存入银行；年末将医疗费收入40,000元转入非限定性净资产；上年收到的政府补助9,000元（限定性），现可动用。

```
收到医疗费：
借：银行存款 8,000
  贷：提供服务收入——非限定性收入——医疗费收入 8,000

年末结转：
借：提供服务收入——非限定性收入——医疗费收入 40,000
  贷：非限定性净资产 40,000

政府补助限制解除：
借：政府补助收入——限定性收入 9,000
  贷：政府补助收入——非限定性收入 9,000
```

#### 5.5 政府补助收入

政府补助收入按是否存在限制区分为限定性和非限定性。

```
接受政府补助（限定性）：
借：银行存款
  贷：政府补助收入——限定性收入

限制解除：
借：政府补助收入——限定性收入
  贷：政府补助收入——非限定性收入

期末结转：
借：政府补助收入——非限定性收入
  贷：非限定性净资产
借：政府补助收入——限定性收入
  贷：限定性净资产
```

> **注意：** 承接政府购买服务取得的收入属于交换交易，应记入"提供服务收入"，不应记入"政府补助收入"。（财会〔2020〕9号）

#### 5.6 投资收益

```
出售短期投资（盈利）：
借：银行存款（实收款）
    短期投资跌价准备（已提准备）
  贷：短期投资（账面余额）
      投资收益——非限定性收入（差额）

出售短期投资（亏损）：
借：银行存款
    短期投资跌价准备
    投资收益（亏损额）
  贷：短期投资

期末结转：
借：投资收益——非限定性收入
  贷：非限定性净资产
```

**示例（例17-3）：** 某民间非营利组织出售账面余额54,000元（已提跌价准备2,000元）的短期债券，收到价款55,000元。
```
借：银行存款 55,000
    短期投资跌价准备 2,000
  贷：短期投资——债券投资 54,000
      投资收益——非限定性收入 3,000
```

#### 5.7 商品销售收入

**收入确认条件：**
1. 商品所有权上的主要风险和报酬已转移给购买方
2. 组织既没有保留通常与所有权相联系的继续管理权，也没有对已售出商品实施有效控制
3. 收入金额能够可靠计量
4. 相关经济利益很可能流入组织
5. 相关已发生或将发生的成本能够可靠计量

```
销售商品：
借：银行存款/应收账款/应收票据
  贷：商品销售收入——非限定性收入
      预收账款（预收部分）

结转销售成本：
借：业务活动成本
  贷：存货

期末结转：
借：商品销售收入——非限定性收入
  贷：非限定性净资产
```

**销售退回处理：**

（1）未确认收入的已发出商品退回：不需要进行会计处理。

（2）已确认收入的销售退回（一般情况，直接冲减退回当月）：
```
借：商品销售收入（应冲减的收入）
  贷：银行存款/应收账款/应收票据（已收或应收金额）
借：存货（退回商品成本）
  贷：业务活动成本
如该项销售发生现金折扣，应在退回当月一并冲减筹资费用。
```

（3）资产负债表日后事项（报告期间资产负债表日至财务报告批准报出日之间发生的退回）：
```
借：非限定性净资产（或限定性净资产）（应冲减的收入）
  贷：银行存款/应收账款/应收票据
借：存货（退回商品成本）
  贷：非限定性净资产
如该项销售已发生现金折扣，应一并处理。
```

**现金折扣处理：**

现金折扣是指民间非营利组织为尽快回笼资金而给予购买方的理财优惠（如2/10, 1/20, n/30）。

**销售方（给予折扣）：**
```
确认收入时按总价法（不扣除折扣）：
借：应收账款（全额）
  贷：商品销售收入

购买方在折扣期内付款：
借：银行存款（实收款）
    筹资费用（折扣额）
  贷：应收账款（全额）
```

**购买方（获得折扣）：**
```
借：应付账款/应付票据（全额）
  贷：筹资费用（折扣额）
      银行存款（实付款）
```

**销售折让：** 在商品销售时直接给予购买方的折让，直接从当期实现的销售收入中抵减。

**综合示例（例17-4）：** 某民间非营利组织4月1日销售商品70,000元（成本50,000元），现金折扣条件2/10, 1/20, n/30。购货方4月20日付款（享受1%折扣=700元）。5月5日发现其中20%存在质量问题需退货，5月15日退回。

```
4月1日销售：
借：应收账款 70,000
  贷：商品销售收入 70,000
借：业务活动成本 50,000
  贷：存货 50,000

4月20日收款（享受1%折扣）：
借：银行存款 69,300
    筹资费用 700
  贷：应收账款 70,000

5月15日退货（退回20%，应冲减现金折扣700×20%=140元）：
借：商品销售收入 14,000
  贷：筹资费用 140
      银行存款 13,860
借：存货 10,000
  贷：业务活动成本 10,000
```

#### 5.8 其他收入

包括无法支付的应付款项、存货盘盈、固定资产盘盈、固定资产处置净收入、无形资产处置净收入等。

```
固定资产处置净收入：
借：固定资产清理（净收益）
  贷：其他收入——非限定性收入

存货盘盈：
借：存货
  贷：其他收入——非限定性收入

期末结转：
借：其他收入——非限定性收入
  贷：非限定性净资产
```


### 六、费用核算（第17章）

#### 6.1 费用分类

| 科目 | 核算内容 |
|------|---------|
| 业务活动成本 | 为实现业务活动目标、开展项目活动或提供服务所发生的费用 |
| 管理费用 | 为组织和管理业务活动所发生的各种费用（董事会/理事会经费、行政人员工资、办公费、折旧、减值损失、预计负债损失等） |
| 筹资费用 | 为筹集业务活动所需资金而发生的费用（募款活动费、借款利息、汇兑损失等） |
| 其他费用 | 上述以外的其他费用 |

**费用确认原则：** 权责发生制，本期应发生的费用不论是否支付均确认；不属于本期的费用即使已支付也不确认。

#### 6.2 业务活动成本

```
发生业务活动成本：
借：业务活动成本（按项目/服务/业务大类明细）
  贷：银行存款/存货/应付账款等

期末结转：
借：非限定性净资产
  贷：业务活动成本
```

**示例（例17-7）：** 某基金会将上年收到的限定性捐款80,000元，按捐赠人要求购买学生教材并赠送给希望小学。

```
购买教材（限制解除，限定性转非限定性）：
借：存货 80,000
  贷：银行存款 80,000
借：限定性净资产 80,000
  贷：非限定性净资产 80,000

捐赠书籍：
借：业务活动成本 80,000
  贷：存货 80,000
```

#### 6.3 管理费用

```
计提折旧：
借：管理费用
  贷：累计折旧

购买办公用品：
借：管理费用
  贷：银行存款

发生维修费：
借：管理费用
  贷：银行存款

确认预计负债损失：
借：管理费用
  贷：预计负债

计提资产减值准备：
借：管理费用
  贷：短期投资跌价准备/坏账准备/长期投资减值准备等

摊销无形资产：
借：管理费用
  贷：无形资产

期末结转：
借：非限定性净资产
  贷：管理费用
```

**示例（例17-8）：** 某慈善机构12月：①计提折旧4,000元，购买办公用品2,000元；②固定资产维修费4,000元；③因担保确认预计负债120,000元；④期末结转管理费用。

```
①借：管理费用 6,000
    贷：累计折旧 4,000
        银行存款 2,000

②借：管理费用 4,000
    贷：银行存款 4,000

③借：管理费用 120,000
    贷：预计负债 120,000

④借：非限定性净资产 130,000
    贷：管理费用 130,000
```

#### 6.4 筹资费用

```
发生募款活动费：
借：筹资费用
  贷：银行存款

计提借款利息：
借：筹资费用
  贷：预提费用

发生汇兑损失：
借：筹资费用
  贷：银行存款/长期借款等

冲减筹资费用的利息收入/汇兑收益：
借：银行存款/长期借款等
  贷：筹资费用

现金折扣（购货方获得）：
借：应付账款/应付票据
  贷：筹资费用（折扣额）
      银行存款（实付款）

期末结转：
借：非限定性净资产
  贷：筹资费用
```

**示例（例17-9）：** 某慈善机构12月：①为取得捐赠收入发生费用3,000元；②计提当月利息1,500元；③期末结转筹资费用7,500元。

```
①借：筹资费用 3,000
    贷：银行存款 3,000

②借：筹资费用 1,500
    贷：预提费用 1,500

③借：非限定性净资产 7,500
    贷：筹资费用 7,500
```

### 七、净资产核算（第17章）

#### 7.1 净资产分类

- **非限定性净资产**：无时间或用途限制的净资产
- **限定性净资产**：有时间限制或用途限制的净资产

> **注意：** 民间非营利组织内部管理层对资产使用所做的限制（如董事会决议），**不属于**限定性净资产，属于内部管理限制。

#### 7.2 非限定性净资产

期末，将各收入类科目"非限定性收入"明细余额转入：
```
借：捐赠收入——非限定性收入
    政府补助收入——非限定性收入
    提供服务收入——非限定性收入
    会费收入——非限定性收入
    商品销售收入——非限定性收入
    投资收益——非限定性收入
    其他收入——非限定性收入
  贷：非限定性净资产
```

期末，将各费用类科目余额转出：
```
借：非限定性净资产
  贷：业务活动成本
      管理费用
      筹资费用
      其他费用
```

限定性净资产限制解除时：
```
借：限定性净资产
  贷：非限定性净资产
```

#### 7.3 限定性净资产

期末，将各收入类科目"限定性收入"明细余额转入：
```
借：捐赠收入——限定性收入
    政府补助收入——限定性收入
    （其他有限制的收入）
  贷：限定性净资产
```

限制解除时（转为非限定性）：
```
借：限定性净资产
  贷：非限定性净资产
```

**综合示例（例17-10）：** 某民间非营利组织年度收入费用累计数如下：

| 收入 | 金额 | 费用 | 金额 |
|------|------|------|------|
| 捐赠收入——限定性 | 200,000 | 业务活动成本 | 650,000 |
| 捐赠收入——非限定性 | 800,000 | 管理费用 | 800,000 |
| 政府补助——限定性 | 160,000 | 筹资费用 | 80,000 |
| 政府补助——非限定性 | 210,000 | 其他费用 | 90,000 |
| 提供服务收入（非限定） | 600,000 | | |
| 其他收入（非限定） | 100,000 | | |

```
结转非限定性收入：
借：捐赠收入——非限定性收入 800,000
    政府补助收入——非限定性收入 210,000
    提供服务收入 600,000
    其他收入 100,000
  贷：非限定性净资产 1,710,000

结转各项费用：
借：非限定性净资产 1,620,000
  贷：业务活动成本 650,000
      管理费用 800,000
      筹资费用 80,000
      其他费用 90,000

结转限定性收入：
借：捐赠收入——限定性收入 200,000
    政府补助收入——限定性收入 160,000
  贷：限定性净资产 360,000
```

本年非限定性净资产净增加 = 1,710,000 - 1,620,000 = **90,000元**
本年限定性净资产净增加 = **360,000元**


### 八、会计报告（第18章）

#### 8.1 报表体系

民间非营利组织应编制以下财务会计报告：

| 报表 | 编制基础 | 主要内容 |
|------|---------|---------|
| 资产负债表 | 权责发生制 | 资产、负债、净资产（限定性+非限定性） |
| 业务活动表 | 权责发生制 | 收入、费用、净资产变动（分限定性/非限定性列示） |
| 现金流量表 | 收付实现制 | 业务活动、投资活动、筹资活动现金流量 |

> 年度和中期财务报告至少应反映**两个年度或两个相关会计期间**的比较数据。

#### 8.2 资产负债表关键项目

**资产类：**

| 项目 | 填列依据 |
|------|---------|
| 现金及银行存款 | 现金+银行存款+其他货币资金期末余额 |
| 短期投资净额 | 短期投资期末余额 - 短期投资跌价准备期末余额 |
| 应收账款净额 | 应收账款期末余额 - 坏账准备期末余额 |
| 存货净额 | 存货期末余额 - 存货跌价准备期末余额 |
| 长期投资净额 | 长期股权投资+长期债券投资期末余额 - 长期投资减值准备期末余额 |
| 固定资产净值 | 固定资产期末余额 - 累计折旧期末余额 |
| 文物文化资产 | 文物文化资产期末借方余额 |
| 无形资产 | 无形资产期末借方余额 |
| 受托代理资产 | 受托代理资产期末借方余额 |

**净资产类：**

| 项目 | 填列依据 |
|------|---------|
| 非限定性净资产 | 非限定性净资产期末贷方余额 |
| 限定性净资产 | 限定性净资产期末贷方余额 |
| 净资产合计 | 非限定性净资产 + 限定性净资产 |

#### 8.3 业务活动表

业务活动表反映民间非营利组织在某一会计期间内净资产的变动情况，分**限定性**和**非限定性**两栏列示。

**主要结构：**

```
一、收入
  捐赠收入（限定性/非限定性）
  会费收入
  提供服务收入
  政府补助收入（限定性/非限定性）
  商品销售收入
  投资收益
  其他收入
  限制解除转入非限定性净资产（正数）
  限制解除转出限定性净资产（负数）
  收入合计

二、费用
  业务活动成本
  管理费用
  筹资费用
  其他费用
  费用合计

三、本期净资产变动额 = 收入合计 - 费用合计
四、期初净资产余额
五、期末净资产余额 = 期初余额 + 本期变动额
```

#### 8.4 现金流量表编制方法（民间非营利组织）

按业务活动、投资活动、筹资活动三类列示现金流量。

**一、业务活动现金流量**

| 项目 | 填列依据 |
|------|---------|
| 接受捐赠收到的现金 | "捐赠收入"中实际收到现金的部分（含限定性和非限定性） |
| 收取会费收到的现金 | "会费收入"中实际收到的现金 |
| 提供服务收到的现金 | "提供服务收入"中实际收到的现金+预收账款增加额 |
| 销售商品收到的现金 | "商品销售收入"中实际收到的现金 |
| 政府补助收到的现金 | "政府补助收入"中实际收到的现金 |
| 收到的其他与业务活动有关的现金 | 利息收入、租金收入等实际收到的现金 |
| 提供捐赠或资助支付的现金 | 对外捐赠实际支付的现金 |
| 购买商品/接受劳务支付的现金 | 根据"存货""应付账款""预付账款"等科目分析 |
| 支付给职工的现金 | "应付工资"借方发生额中实际支付的部分 |
| 支付的税费 | "应交税金"借方发生额 |
| 支付的其他与业务活动有关的现金 | 管理费用中实际支付的现金（办公费、差旅费等） |

**二、投资活动现金流量**

| 项目 | 填列依据 |
|------|---------|
| 收回短期投资/长期投资收到的现金 | 出售投资实际收到的现金 |
| 取得投资收益收到的现金 | 实际收到的股利、利息 |
| 处置固定资产收到的现金 | 出售固定资产收到的现金净额 |
| 购建固定资产支付的现金 | 购买固定资产、在建工程支付的现金 |
| 对外投资支付的现金 | 购买短期投资、长期投资支付的现金 |

**三、筹资活动现金流量**

| 项目 | 填列依据 |
|------|---------|
| 借款收到的现金 | 取得短期借款、长期借款收到的现金 |
| 偿还借款本金支付的现金 | 偿还借款本金支付的现金 |
| 偿还借款利息支付的现金 | 实际支付的借款利息 |
| 融资租赁支付的现金 | 支付的融资租赁款 |

**勾稽关系：**
```
现金及现金等价物净增加额 = 业务活动净额 + 投资活动净额 + 筹资活动净额
期末现金余额 = 期初现金余额 + 净增加额
期末现金余额 应等于 资产负债表中"现金及银行存款"项目期末数
```

## 常见问题判断

**Q：某笔业务只做了财务会计，没做预算会计，对不对？**
先判断是否涉及预算管理的现金收支。计提折旧、计提工资、权益法调整——只做财务会计，正确。财政拨款支出、借款取得——必须平行记账，漏做预算会计是错误。

**Q：行政单位能不能用权益法核算长期股权投资？**
不能。长期股权投资是事业单位特有科目，行政单位没有这个科目。

**Q：固定资产购入时预算会计记了支出，后续折旧还要记预算支出吗？**
不要。折旧只在财务会计做，预算会计不做处理。这是双基础最典型的差异。

**Q：财政直接支付和财政授权支付的区别？**
直接支付：财政直接向供应商付款，单位收到"财政直接支付入账通知书"，贷记财政拨款收入，借记资产或费用。
授权支付：财政向单位零余额账户拨付额度，单位用额度付款，收到"额度到账通知书"时借记零余额账户用款额度，贷记财政拨款收入。

**Q：事业单位年末盈余如何处理？**
财务会计：收入→本期盈余→本年盈余分配→累计盈余（或专用基金）。
预算会计：预算收入→结转结余科目→资金结存。
两套报表通过"本年盈余与预算结余差异说明"衔接。

**Q：授权支付年末有未用额度和未下达额度，怎么处理？**
两件事要分开做，不能混：
- 已下达但未用的额度（零余额账户还有余额）→ 注销，借：财政应返还额度 / 贷：零余额账户用款额度
- 预算指标数 > 已下达额度的差额（根本没下达的部分）→ 补确认财政拨款收入，借：财政应返还额度 / 贷：财政拨款收入
次年财政恢复额度时，反向做：借：零余额账户用款额度 / 贷：财政应返还额度。
预算会计对应做资金结存内部两个明细之间的转换。

**Q：民间非营利组织的限定性净资产和非限定性净资产如何区分？**
限制来自**外部资源提供者**（捐赠方、政府）对资产使用设置的时间或用途限制。组织内部管理层（董事会、理事会）的限制决议**不构成**限定性净资产。

**Q：民间非营利组织承接政府购买服务，收入记哪个科目？**
记"提供服务收入"，不记"政府补助收入"。承接政府购买服务属于交换交易，不是无偿补助。（财会〔2020〕9号）

**Q：PPP项目资产折旧怎么处理？**
政府方按月计提折旧。若PPP项目净资产初始金额等于PPP项目资产初始金额，折旧全额冲减PPP项目净资产；若净资产金额小于资产金额，按比例冲减净资产，差额计入业务活动费用。折旧均不触发预算会计。

**Q：行政事业单位划转撤并时资产如何处理？**
按账面价值划转，不确认损益。移交方：借：无偿调拨净资产 / 贷：各资产科目净值。接收方：借：各资产科目 / 贷：无偿调拨净资产（差额）、各负债科目。
