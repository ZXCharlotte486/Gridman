# 企业会计准则应用案例（财政部官方）

> 定位：财政部官方权威案例，具有准则解释效力
> 来源：财政部会计准则委员会（2018-2026）
> 覆盖：收入确认、主要责任人、股份支付、金融工具分类、预期信用损失、PPP、售后租回
> 性质：判断

---

## 一、收入准则应用案例（CAS 14）

### 案例1：运输服务——是否构成单项履约义务（2018）

**场景A**：控制权在出库时转移 → 出库后的运输 = 单项履约义务（运输服务），按分摊的交易价格确认收入

**场景B**：控制权在送达指定地点时转移 → 运输活动在控制权转移前发生 → 不构成单项履约义务，是履行合同的必要活动

**判断规则**：运输是否构成单项履约义务，取决于控制权转移时点与运输发生时点的先后关系。

---

### 案例2：合同履约成本——酒店等服务行业（2018）

酒店为履行合同发生的成本（如客房布置、清洁等），如果满足以下条件可确认为合同履约成本资产：
1. 与一份当前或预期取得的合同直接相关
2. 增加了企业未来用于履行履约义务的资源
3. 预期能够收回

---

### 案例3：合同负债——电商平台预售购物卡（2018）

电商平台预售购物卡收到的款项 → 确认为合同负债（非预收账款），在客户使用购物卡消费时确认收入。

---

### 案例4：合同负债——涉及不同增值税率的储值卡（2018）

储值卡可用于购买不同税率商品时，收到预付款不含增值税（因为此时无法确定适用税率），全额确认为合同负债。消费时按实际商品税率计算增值税。

---

### 案例5：亏损合同（2018）

当合同预计总成本超过合同总收入时，对亏损部分确认预计负债。

---

### 案例6：定制软件开发服务——时段vs时点（2020）

**关键判断**：定制软件是否属于"在某一时段内履行的履约义务"

三个条件逐一分析：
1. ❌ 客户未在企业履约同时取得并消耗经济利益（被更换后新供应商需重新执行）
2. ❌ 客户虽拥有所有权/知识产权，但不能合理利用开发过程中的程序获得经济利益 → 不能控制在建商品
3. ❌ 商品具有不可替代用途，但付款安排（预付5%+里程碑验收65%+终验30%，违约金仅10%）→ 不能在任一时点收取补偿已发生成本和合理利润的款项

**结论**：属于在某一时点履行的履约义务（终验交付时确认收入）

**实务启示**：定制软件≠一定是时段确认。关键看付款安排是否能补偿已发生成本+合理利润。

---

### 案例7：药品实验服务的收入确认（2020）

CRO（合同研究组织）为药企提供临床试验服务：
- 如果客户能在企业履约同时取得并消耗经济利益 → 时段确认
- 判断依据：如果更换供应商，新供应商是否需要重新执行已完成的工作

---

### 案例8：保荐服务的收入确认（2020）

保荐服务通常包含多个阶段（辅导期、申报期、持续督导期），需判断是否为单项履约义务或多项履约义务。

---

### 案例9：主要责任人和代理人的判断（2020）

**核心规则**：判断标准是"在向客户转让商品前是否拥有对该商品的控制权"

**三个迹象仅为辅助**（不能取代控制权评估）：
1. 谁承担主要责任（退换货/赔偿）
2. 谁承担存货风险
3. 谁有权自主定价

**案例1（百货联营-品牌专柜）**：
- 乙公司（品牌方）：负责商品保管/出售/调配/下架，承担丢失毁损风险，拥有未售商品所有权，负责实际定价 → **主要责任人**，确认收入1000元
- 甲公司（百货）：统一收款开票，扣除10% → **代理人**，确认收入100元

**案例2（百货直营）**：
- 甲公司（百货）：委派营业员，负责保管/出售/调配/下架，承担风险，拥有所有权，统一定价 → **主要责任人**

**案例3（联名款-五五分成）**：
- 甲公司（零售商）：能主导商品使用（出售/调配/下架），30日未售可退回但退回前乙公司不得取回 → **主要责任人**
- 乙公司（供应商）：按附有销售退回条款的销售处理

---

### 案例10：合同变更与可变对价的判断（2020）

合同变更 vs 可变对价的区分：
- 合同变更：双方批准的对合同范围或价格的变更
- 可变对价：合同中已约定的、金额可能变动的对价（如折扣、返利、奖励金）

---

### 案例11：基于客户销售额的可变对价（2020）

基于客户后续销售额的特许权使用费 → 适用"销售或使用基础的特许权使用费"例外规则 → 在客户后续销售发生时确认收入

---

### 案例12：标准化软件产品的收入确认时点（2023）

**案例1（无实质性障碍）**：
- 软件交付+授权书移交 → 客户可随时自行安装并在线获取自动生成的激活码
- 激活码生成无实质性障碍 → **交付日即确认收入**（即使客户实际3月才激活）

**案例2（有实质性障碍）**：
- 合同约定需付款后才能申请激活码，且需经A公司审核批准
- 交付日客户不能主导软件使用 → **付款并获取激活码日确认收入**

**判断规则**：激活码获取是否存在实质性障碍（是否需要卖方额外审批/授权）

---

### 案例13：预售商品房的收入确认（2023）

**结论**：预售商品房通常属于在某一时点履行的履约义务（交付时确认收入）

**三个条件分析**：
1. ❌ 购房人未在建造同时取得并消耗经济利益
2. ❌ 购房人不能控制在建商品（无法出售/抵押/主导建设/改变设计）
3. ❌ 虽有不可替代用途，但违约金（20%）不能补偿已发生成本和合理利润

**重要提示**：除非房企有权保留预售房款全额且有法律支持，否则不满足"有权收取款项"条件。在中国法律下，购房人违约金通常不能补偿房企已发生成本+合理利润。

### 案例14：充（供）电业务的收入确认（2025）

> 财政部 2025-04-17 发布

**核心问题**：充电/供电业务，按总额法还是净额法确认收入？

**判断逻辑**：电具有"瞬时转移"特性，企业在向客户转让前是否实质取得对"电"的**控制权**——是判断主要责任人/代理人的关键。

**情形一：港区供电公司**

- 业务：从国家电网购入高压电 → 自建变电站变压（110kV → 35/10/0.4kV） → 供应港区企业
- 不发电、不储电，仅做"分流"和电力调配
- 电价受政府指导价/市场化交易电价约束，自主定价空间有限
- **结论：净额法**（变压、调配只是分流，未实质取得电的控制权）

**情形二：第三方快充充电桩**

- 业务：自建快充桩为电动车充电，向用户收取"电费 + 服务费"两项
- 电费按国家电价政策执行，服务费自主定价
- 电力来源全部由电力公司提供，转换损耗（5%-10%）由企业承担
- 设备折旧约占总成本 30%，技术整合程度不显著
- **结论：净额法**（充电技术整合不显著，电费部分本质是代收代付，企业未取得电的控制权）

**实务启示**：

- 充电桩、港区供电、商业地产分时供电等业务，**按服务费净额确认收入**，不能按电费总额做收入
- 这条对充电运营商（特来电、星星充电等）的报表呈现影响巨大——同样的现金流入，按总额法可能 100 元收入、按净额法只有 30 元
- 对企业估值（P/S 倍数）、毛利率、增长率指标都会产生重大影响
- 审计/IPO 场景：被审计单位充电业务收入按总额法核算，需直接按本案例调整为净额法

**类似业务的判断思路**：

- 商业综合体的"二次供水/供电"——若不储电、按指导价收费 → 净额法
- 园区/产业地产的"电力转供" → 净额法
- 自建电站发电后销售（如分布式光伏）→ 总额法（发电=取得控制权）

**分析依据**：CAS 14 第九条、第三十四条；《企业会计准则应用指南汇编2024》第372-379页、第425-432页。原文：https://docs.maoyanqing.com/accounting/ent/casc/14/14.html

## 二、股份支付准则应用案例（CAS 11）

### 案例1：以IPO成功为可行权条件（2021）

IPO成功属于**非市场条件**（业绩条件），在估计可行权数量时考虑，不影响授予日公允价值的计量。

### 案例2："大股东兜底式"股权激励（2021）

大股东承诺兜底（如股价低于某价格则补偿差额）→ 不影响上市公司层面的股份支付会计处理，上市公司仍按正常股份支付确认费用。

### 案例3：授予限制性股票（2021）

限制性股票的授予日 = 满足所有批准程序且双方就条款达成一致的日期。

### 案例4：授予日的确定（2021）

授予日的确定需同时满足：(1)经股东大会/董事会批准；(2)双方就条款和条件达成一致。

### 案例5：实际控制人受让股份是否构成新的股份支付（2021）

实际控制人从员工处受让股份（如员工离职退出）→ 通常不构成新的股份支付，除非存在新的服务条件。

### 案例6：现金结算修改为权益结算（2023）

修改日：终止确认现金结算的股份支付负债，按权益结算重新计量，差额计入资本公积。

### 案例7：修改导致公允价值增加（2024）

修改使所授予权益工具公允价值增加的 → 按修改后的公允价值确认增量费用。

## 三、金融工具准则应用案例（CAS 22/37）

### 资金集中管理会计处理（2022）

集团内部资金集中管理（资金池）的会计处理：
- 子公司存入资金池 → 确认其他应收款（对集团财务公司/母公司）
- 不能简单确认为银行存款

### 金融负债与权益工具的区分（2022，5个案例）

**核心判断**：发行人是否承担交付现金或其他金融资产的合同义务

| 情形 | 结论 |
|------|------|
| 对减资程序的考虑 | 如果减资需经法定程序且发行人有权拒绝 → 不构成金融负债 |
| 中止和恢复回售权 | 回售权中止期间 → 权益工具；恢复后 → 重新评估 |
| 补充协议导致义务变化 | 补充协议使发行人新增交付现金义务 → 重分类为金融负债 |
| 发行人作为合同一方承担的义务 | 关注义务的实质而非形式 |
| 投资者保护条款 | 保护条款如果不构成发行人的合同义务 → 不影响权益工具分类 |

### 预期信用损失法应用案例（2023，2个）

**案例1（以内部评级为基础）**：
- 将金融资产按内部评级分组
- 基于历史违约率+前瞻性调整计算预期信用损失
- 三阶段模型：12个月ECL → 整个存续期ECL（未减值）→ 整个存续期ECL（已减值）

**案例2（简化方法-不以评级为基础）**：
- 适用于应收账款/合同资产
- 使用迁徙率矩阵（账龄分析法的升级版）
- 基于历史损失率+前瞻性调整

### 金融工具列报——FVOCI权益工具投资披露（2026）

指定为以公允价值计量且其变动计入其他综合收益的权益工具投资，需披露：
- 指定原因
- 各项投资的公允价值
- 期间确认的股利收入
- 累计利得或损失及期间转入留存收益的金额

## 四、PPP会计处理应用案例（2021）

### 三种模式

| 模式 | 确认资产 | 适用条件 |
|------|----------|----------|
| 金融资产模式 | 长期应收款 | 政府付费，社会资本方有无条件收取现金的权利 |
| 无形资产模式 | 无形资产（特许经营权） | 使用者付费，社会资本方承担需求风险 |
| 混合模式 | 部分金融资产+部分无形资产 | 政府保底+超额部分使用者付费 |

## 五、所得税准则应用案例（CAS 18）

### 单项交易不适用初始确认豁免（2023）

**背景**：准则解释第16号明确，对于不是企业合并、交易发生时既不影响会计利润也不影响应纳税所得额、且初始确认的资产和负债导致产生等额应纳税暂时性差异和可抵扣暂时性差异的单项交易（如租赁/弃置义务），不适用初始确认豁免。

**结论**：承租人在租赁开始日确认使用权资产和租赁负债时，应分别确认相应的递延所得税负债和递延所得税资产。

## 六、租赁准则应用案例（CAS 21）

### 售后租回——含非指数/比率的可变付款额（2023）

卖方兼承租人对包含非取决于指数或比率的可变租赁付款额的售后租回交易：
- 可变付款额不纳入租赁负债计量
- 售后租回中转让资产的对价需按公允价值调整

## 七、企业会计准则解释第18号（财会〔2024〕24号）

**生效日期**：2024-12-06

### 议题一：浮动收费法下投资性房地产的后续计量

**适用对象**：执行新保险合同准则（CAS 25-2020）的企业

**规则**：
- 浮动收费法下作为基础项目持有的投资性房地产，可选择全部公允价值模式或全部成本模式
- 不得对该部分投资性房地产同时采用两种计量模式
- 选择公允价值模式后不得转为成本模式
- 该选择不影响企业对其余投资性房地产的计量模式选择

**新旧衔接**：由成本模式转为公允价值模式的，作为会计政策变更追溯调整

### 议题二：不属于单项履约义务的保证类质量保证

**核心变化**：保证类质量保证的预计负债，计入科目从"销售费用"改为"主营业务成本/其他业务成本"

**会计处理**：
```
借：主营业务成本/其他业务成本
  贷：预计负债
```

**报表列示**：利润表"营业成本"项目；资产负债表"其他流动负债"/"预计负债"项目

**新旧衔接**：作为会计政策变更追溯调整

## 八、企业会计准则解释第19号（财会〔2025〕32号）

**生效日期**：2026-01-01

### 议题一：非同一控制下企业合并中补偿性资产

**定义**：出售方针对被购买方某些或有事项/特定不确定性结果给予购买方的补偿权利

**合并报表层面**：
- 确认时点：与被补偿项目同时确认
- 初始计量：与被补偿项目相同基础，考虑可收回性
- 后续计量：被补偿项目账面价值变动 → 相应调整补偿性资产，差额计入投资收益
- 终止确认：收回/出售/丧失权利时，差额计入投资收益

**个别报表层面**：
- 确认条件：基本确定能够收到且金额能够可靠计量
- 确认时：借记"补偿性资产"，贷记"长期股权投资"（冲减初始投资成本）

**科目设置**：新设"补偿性资产"科目，按流动性分别列示于"其他流动资产"/"其他非流动资产"

### 议题二：处置同一控制下合并取得子公司时资本公积的处理

**核心规则**：处置并丧失控制权时，原合并日因初始投资成本与合并对价差额调整的资本公积 → **不得转出至当期损益或留存收益**（无论交易对手是否为关联方）

### 议题三：电子支付系统结算的金融负债终止确认

**可选择提前终止确认的条件**（同时满足）：
1. 企业没有实际能力撤回/停止/取消付款指令
2. 企业没有实际能力支取将用于结算的现金
3. 与电子支付系统相关的结算风险不重大

**注意**：该选择应用于通过同一电子支付系统的所有结算

### 议题四：金融资产合同现金流量特征评估（SPPI测试）

**关于或有特征**：
- 或有事项性质与基本借贷风险直接相关 + 变动方向相同 → 通过SPPI测试
- 或有事项性质与基本借贷风险不直接相关（如碳减排挂钩利率）→ 需评估所有情景下现金流量是否与不含该特征的工具"不存在显著差异"

**新增披露要求**：对受或有事项影响的合同条款，需披露定性描述+定量信息+账面余额

### 议题五：FVOCI权益工具投资披露

至少按类别披露：期末公允价值、期间公允价值变动（区分终止确认的和继续持有的）、累计利得或损失转出情况

## 九、可持续信息鉴证业务准则第6101号——基本准则（试行）

**文号**：财会〔2026〕1号 | **生效日期**：2026-01-09

### 准则定位

- 中国首部可持续信息鉴证准则
- 规范注册会计师对企业可持续发展信息（ESG报告等）的鉴证业务
- 与财务报表审计准则（CSA）并列，属于独立的准则体系

### 核心框架

| 要素 | 内容 |
|------|------|
| 鉴证对象 | 企业依据可持续披露准则编制的可持续信息（全部或部分） |
| 保证程度 | 合理保证（积极方式）/ 有限保证（消极方式）/ 混合 |
| 鉴证者 | 注册会计师或其他具备资质的鉴证者 |
| 准则体系 | 基本准则 + 具体准则 + 应用指南 |

### 与财务报表审计的关系

- 如果可持续信息包含在财务报表中（如会计准则要求的气候相关披露）→ 适用审计准则（CSA），不适用本准则
- 独立的ESG/可持续发展报告 → 适用本准则

### 鉴证业务前提条件（四项同时满足）

1. 职业道德要求（含独立性）能够遵守
2. 项目组整体具备必要胜任能力和专业素质
3. 鉴证业务前提条件存在
4. 鉴证业务约定条款已达成一致

### 质量管理要求

- 鉴证机构层面：遵守会计师事务所质量管理准则（5101/5102）
- 项目层面：项目负责人承担总体责任
- 项目负责人需同时具备：鉴证技能 + 职业道德 + 可持续信息领域胜任能力

### 工作底稿要求

- 归档期限：鉴证报告日后60天内
- 保存期限：鉴证报告日起至少10年
- 归档后不得删除或废弃

### 实务影响

- 2026年起沪深300成分股强制披露ESG → 鉴证需求大增
- 会计师事务所需要培养ESG鉴证团队
- 与ISSB/GRI等国际框架对接
