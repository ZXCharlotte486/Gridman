# 报表审查与财务异常识别

> 定位：报表勾稽关系验证、税务风险识别、财务异常检测
> 来源：CAS 准则 + 证监会监管实务 + 上市公司真实案例
> 覆盖：勾稽关系、关键比率、税务风险、收入操纵、成本操纵
> 性质：判断+实操

财务造假与财务洗澡的识别（含恒大/五粮液案例）详见：financial_fraud.md

---

## 一、报表合规性审查

### 审查维度

#### 1. 准则合规性

- 报表格式是否符合财政部规定
- 会计政策选择是否合理且一致
- 会计估计变更是否充分披露
- 附注披露是否完整

#### 2. 勾稽关系验证

**报表内勾稽**：

| 检查项         | 公式                             | 说明     |
| -------------- | -------------------------------- | -------- |
| 资产负债表平衡 | 资产 = 负债 + 所有者权益         | 基本等式 |
| 利润表结构     | 营业利润 + 营业外收支 = 利润总额 | 利润构成 |
| 净利润         | 利润总额 - 所得税费用 = 净利润   | 税后利润 |

**报表间勾稽**：

| 检查项     | 关系                               | 说明     |
| ---------- | ---------------------------------- | -------- |
| 净利润一致 | 利润表净利润 = 权益变动表净利润    | 跨表核对 |
| 未分配利润 | 期初 + 净利润 - 分配 = 期末        | 连续性   |
| 现金流量表 | 期末现金 - 期初现金 = 现金净增加额 | 现金变动 |

#### 3. 关键比率分析

| 类别     | 指标         | 计算公式           | 关注点     |
| -------- | ------------ | ------------------ | ---------- |
| 偿债能力 | 流动比率     | 流动资产/流动负债  | < 1 警示   |
| 偿债能力 | 资产负债率   | 负债/资产          | > 70% 关注 |
| 盈利能力 | 毛利率       | (收入-成本)/收入   | 异常波动   |
| 盈利能力 | 净利率       | 净利润/收入        | 与行业对比 |
| 营运能力 | 应收周转天数 | 应收/日均收入×365 | 持续延长   |
| 营运能力 | 存货周转天数 | 存货/日均成本×365 | 持续延长   |
| 现金质量 | 收现比       | 经营现金流/净利润  | 长期 < 1   |
| 现金质量 | 净利润含金量 | (CFO + 投资收益实收现金 + 利息收入实收现金 + 处置长期资产溢余现金) / 合并净利润 × 100% | ≤ 30% 极高风险 |
| 利润归属 | 归母偏离度 | 归母净利润 / 合并净利润 × 100% | 超出 90%~110% 需核查 |
| 利润归属 | 少数股东损益/权益匹配度 | \|少数股东损益占比 - 少数股东权益占比\| | > 15% 异常 |
| 供应链 | 供应链净占款差额 | (应付票据+应付账款+合同负债) - (应收票据+应收账款+应收款项融资+预付款项+合同资产) | 持续为负 = 话语权弱 |
| 现金流画像 | 现金流四象限 | CFO±/CFI±/CFF± 组合 | 见下方分类表 |
| 偿债能力 | 现金利息保障倍数 | CFO / 实际利息支付现金 | < 1 极高风险 |
| 负债结构 | 核心经营性负债占比 | (应付账款+应付票据+合同负债) / 总负债 × 100% | 持续下降需关注 |

### 净利润含金量（增强版）

> 类型：公式
> 来源：反舞弊实务
> 置信度：实务经验

[场景：判断净利润背后有多少真金白银——比单纯"净现比 = CFO/净利润"更全面，覆盖了投资收益、利息收入和资产处置的现金回收]

**公式**：

```
净利润含金量 = (CFO + 取得投资收益收到的现金 + 利息收入实收现金 + 处置长期资产收回的现金净额) / 合并净利润 × 100%
```

**取数来源**：
- CFO = 现金流量表「经营活动产生的现金流量净额」
- 投资收益实收 = 现金流量表「取得投资收益收到的现金」
- 利息收入实收 = 现金流量表「收到其他与经营活动有关的现金」附注中的利息收入部分（或利润表「利息收入」作近似）
- 处置长期资产 = 现金流量表「处置固定资产、无形资产和其他长期资产收回的现金净额」
- 分母 = 利润表「净利润」（合并口径，非归母，确保分子分母口径一致）

**判定标准**：
- 含金量 ≥ 80%：利润现金支撑极强
- 含金量 50%~80%：利润基本有现金支撑，部分被应收/存货占用
- 含金量 30%~50%：利润水分较大
- 含金量 ≤ 30%：利润几乎无现金支撑，极高风险

⚠️ 分母口径注意：用合并净利润（不是归母净利润），因为分子 CFO 是合并口径。如果分母用归母净利润而少数股东权益占比大，含金量会被虚高。

⚠️ 与简版"净现比"的关系：净现比（CFO/净利润）是简版，适合快速筛查；含金量增强版覆盖面更全，适合深度分析和底稿留痕。

### 供应链议价能力差额

> 类型：公式
> 来源：反舞弊实务 + 产业链分析
> 置信度：实务经验

[场景：一个数字判断企业在供应链中是"占别人钱"还是"被别人占钱"——正 = 强势，负 = 弱势]

**公式**：

```
供应链净占款差额 = (应付票据 + 应付账款 + 合同负债) - (应收票据 + 应收账款 + 应收款项融资 + 预付款项 + 合同资产)
```

**判定标准**：

| 差额 | 含义 | 判定 |
|---|---|---|
| > 0 | 应付预收 > 应收预付，企业占用上下游资金 | 供应链话语权强 |
| < 0 | 应收预付 > 应付预收，企业资金被上下游占用 | 供应链话语权弱 |
| 持续 > 0 且稳定 | 产业链地位牢固 | 正常优秀 |
| 持续 < 0 且恶化 | 资金链风险累积 | 需警惕 |

**异常判定**：
- 单期差额 < 0：标记【供应链话语权弱势】
- 持续 2 期+ 为负且绝对值扩大：标记【资金被占用高风险】
- 4 期全周期为负且占营收比 < -10%：极高风险，资金链断裂隐患

**典型参照**：茅台、海天味业等消费品龙头差额持续为正（预收 > 应收）；制造业代工企业差额常为负（垫款生产 + 下游赊销）。

### 现金流四象限分类

> 类型：判断表
> 来源：财务分析实务
> 置信度：实务经验

[场景：用 CFO/CFI/CFF 三项的正负组合快速给企业现金流画像——确定企业处于哪个生命周期阶段]

| 组合 | 类型 | 含义 | 投资判断 |
|---|---|---|---|
| CFO+ / CFI- / CFF- | 优质现金牛 | 经营赚钱、投资扩张、还债分红 | 可长期持有 |
| CFO+ / CFI- / CFF+ | 扩张成长型 | 经营赚钱、大幅投资、借债扩张 | 谨慎跟踪，关注扩张合理性和回报 |
| CFO+ / CFI+ / CFF- | 收缩变现型 | 经营赚钱、变卖资产、偿债缩表 | 谨慎——可能在去杠杆或瘦身 |
| CFO- / CFI- / CFF+ | 烧钱续命型 | 经营亏现金、继续投资、靠融资续命 | 高风险，优先排除 |
| CFO- / CFI+ / CFF- | 困境变卖型 | 经营亏现金、变卖资产回血、偿债 | 谨慎回避——可能已进入财务困境 |
| CFO- / CFI+ / CFF+ | 初创/转型期 | 经营未正轨、变卖旧资产、融资支撑 | 视行业和阶段判断 |

**使用规则**：
1. 先看 4 期全周期的组合是否一致——一致说明企业处于稳定状态
2. 组合发生切换时关注切换方向：现金牛 → 烧钱续命 = 恶化信号
3. 单期异常（如年末集中分红导致 CFF 为负）需排除偶发因素

⚠️ 现金流类型只是"画像"，不是定论。CFO- 不一定是坏事（如高速增长期的提前备货），CFO+ 不一定是好事（如压供应商尾款导致的短期虚高）。必须结合利润表和资产负债表交叉验证。

### 现金利息保障倍数

> 类型：公式
> 来源：Wall Street Prep / CFA 教材（Cash Interest Coverage）
> 置信度：实务经验

[场景：传统利息保障倍数（EBIT/利息费用）是权责发生制——有利润不等于有现金。现金版用实际经营现金流除以实际利息支出，更保守、更真实]

**公式**：

```
现金利息保障倍数 = 经营活动产生的现金流量净额（CFO）/ 实际利息支付现金
```

**取数说明**：
- 分子：现金流量表「经营活动产生的现金流量净额」
- 分母：现金流量表附注中"偿付利息支付的现金"（需从"分配股利、利润或偿付利息支付的现金"中拆分出利息部分）；如附注未单列，可用利润表「利息费用」近似

**判定标准**：

| 倍数 | 判定 |
|---|---|
| ≥ 5 | 利息偿付能力极强 |
| 3~5 | 安全 |
| 1~3 | 需关注 |
| < 1 | 经营现金流不足以覆盖利息支出——极高风险 |

**与传统利息保障倍数的区别**：

| 维度 | 传统版（EBIT/利息费用） | 现金版（CFO/实际利息支付） |
|---|---|---|
| 口径 | 权责发生制 | 收付实现制 |
| 优点 | 数据易取（利润表直接有） | 更保守，反映"实际能付多少" |
| 局限 | 利润可能没收到钱（应收大 = 虚高） | CFO 波动大（季节性、回款节奏） |
| 适用 | 日常筛查 | 偿债能力深度评估、高杠杆公司 |

⚠️ 当传统利息保障倍数很高但现金版 < 3 时——说明"有利润但没现金"，是造假或回款异常的信号。两个版本配合使用效果最好。

### 核心经营性负债占比

> 类型：公式
> 来源：CPA《财务成本管理》教材（经营性负债 vs 有息负债分类框架）
> 置信度：实务经验

[场景：负债结构分析——同样是负债，"欠供应商/预收客户"和"欠银行"的性质完全不同。经营性负债不花利息，有息负债要付利息]

**公式**：

```
核心经营性负债占比 = (应付账款 + 应付票据 + 合同负债) / 负债总计 × 100%
```

⚠️ 此处"核心经营性负债"仅取供应链直接相关的三项（应付账款+应付票据+合同负债），用于衡量供应链话语权对负债结构的贡献。完整的经营性负债还包含应付职工薪酬、应交税费、其他应付款等，但这些与供应链议价能力无关，故不纳入本指标。

**分类逻辑**：

| 类型 | 科目 | 本质 |
|---|---|---|
| 经营性负债 | 应付账款、应付票据、合同负债、应付职工薪酬、应交税费 | 经营活动自然形成，不付利息，是企业话语权/经营能力的体现 |
| 有息负债 | 短期借款、长期借款、应付债券、租赁负债 | 主动融资形成，需付利息，是杠杆风险的来源 |

**判定标准**：

| 经营性负债占比 | 含义 |
|---|---|
| > 50% | 负债结构健康——靠经营能力而非借钱支撑 |
| 30%~50% | 正常 |
| < 30% | 负债以有息为主——杠杆驱动，利息负担重 |

**异常判定**：
- 占比持续下降（3 期+）且无合理理由：标记【产业链地位下滑或杠杆替代】
- 占比 < 30% 且利息保障倍数 < 3：标记【负债结构恶化高风险】

⚠️ 占比下降不一定是负面——也可能是企业主动缩短付款周期（提前付供应商争取折扣），或合同负债减少（预收转收入正常结转）。需结合具体原因判断方向。

⚠️ 行业差异大：消费品/白酒行业经营性负债占比天然高（先款后货）；资本密集型行业（电力/航空/房地产）有息负债占比天然高，不代表不健康。

## 二、策略库系统

### 内置策略列表

| 策略名称     | 策略ID              | 功能描述                                 | 适用税种           |
| ------------ | ------------------- | ---------------------------------------- | ------------------ |
| 税款比对分析 | tax_reconciliation  | 比对已纳税款与财务报表，识别少缴税款疑点 | 增值税、企业所得税 |
| 收入确认审查 | revenue_recognition | 基于 CAS 14 审查收入确认合规性           | 增值税、企业所得税 |
| 成本操纵识别 | cost_manipulation   | 识别通过成本费用调节利润的行为           | 企业所得税         |
| 税收优惠审查 | tax_preference      | 审查税收优惠政策适用条件和计算准确性     | 企业所得税         |
| 关联交易审查 | related_party       | 审查关联方交易税务合规性                 | 企业所得税、增值税 |

### 策略一：税款比对分析 (tax_reconciliation)

**功能**：通过比对财务报表与纳税申报数据，自动识别少缴税款疑点

**比对维度**：

1. **收入比对**：财务收入 vs 增值税/所得税申报收入
2. **税负率分析**：实际税负率 vs 行业参考值
3. **限额扣除检查**：福利费、招待费、广宣费等超标识别
4. **理论税额测算**：基于财务数据测算理论应纳税额

**预警规则**：

- 财务收入 > 增值税申报收入 × 1.05 → 高风险
- 实际增值税税负率 < 行业平均 × 0.7 → 中风险
- 业务招待费 > MIN(发生额×60%, 收入×0.5%) → 超标

### 策略二：收入确认审查 (revenue_recognition)

**功能**：基于 CAS 14 审查收入确认合规性，识别提前/虚构收入

**审查要点**：

- 收入确认是否满足五步法（识别合同→识别履约义务→确定交易价格→分摊→确认收入）
- 是否存在提前确认（发货未签收即确认）
- 是否存在虚构收入（无真实交易背景）
- 退货率是否异常
- 收入与现金流是否匹配

**异常信号**：

- 季末/年末收入占比异常高（>40%）
- 新客户贡献大额收入但无后续交易
- 收入增长但经营现金流未同步增长
- 应收账款增速远超收入增速

### 策略三：成本操纵识别 (cost_manipulation)

**功能**：识别通过成本费用调节利润的行为

**审查要点**：

- 毛利率异常波动（同比变动 > 5个百分点需关注）
- 成本结转方法变更
- 期间费用突然大幅变动
- 研发费用资本化比例异常
- 折旧政策变更

**异常信号**：

- 毛利率逆行业趋势上升
- 管理费用/销售费用突然大幅下降
- 在建工程长期不转固
- 研发资本化率远高于同行

### 策略四：税收优惠审查 (tax_preference)

**功能**：审查企业享受的税收优惠政策是否符合条件，关注政策时效性和业务实质

**审查维度**：

1. **小型微利企业优惠**：应纳税所得额≤300万、从业人数≤300人、资产总额≤5000万
2. **高新技术企业**：研发费用占比、高新技术产品收入占比、科技人员占比、证书有效性
3. **研发费用加计扣除**：负面清单行业限制、研发费用归集准确性、委托研发比例
4. **固定资产加速折旧/一次性扣除**：500万以下设备器具、资产类型合规性
5. **区域性优惠**：西部大开发（鼓励类产业、收入占比60%）、海南自贸港
6. **政策时效性**：检查优惠政策是否已过期或即将到期

### 策略五：关联交易审查 (related_party)

**功能**：审查关联方交易的税务合规性，防范资本弱化、转让定价和隐性利润分配风险

**审查维度**：

1. **资本弱化审查**：债资比2:1（一般企业）或5:1（金融企业），超标利息不得扣除
2. **关联方利息审查**：利率合理性（基准利率区间）、非金融企业借款凭证合规性
3. **关联方资金占用**：长期占用未收利息、大额资金往来异常
4. **转让定价审查**：关联方销售/采购毛利率差异、定价公允性
5. **关联方担保**：未收担保费、担保实际履行损失
6. **向关联方发放福利**：视同销售处理
7. **同期资料义务**：本地文档（关联交易2亿+）、主体文档（营收55亿+）

## 三、审查深度配置

### 1. 快速审查（30分钟）

- 报表格式合规性检查
- 主要科目异常变动识别
- 重大税务风险点识别
- 执行策略：税款比对（基础模式）
- 适用于：初步筛查、内部汇报

### 2. 标准审查（2-4小时）

- 全科目余额变动分析
- 会计政策适用性审查
- 主要税种风险识别
- 报表间勾稽关系验证

## 四、税务风险识别

### 1. 收入类风险

| 风险点           | 表现                           | 法规依据                     |
| ---------------- | ------------------------------ | ---------------------------- |
| 收入确认时点不当 | 提前/延迟确认收入              | CAS 14、增值税暂行条例第19条 |
| 价外费用未计税   | 违约金、延期付款利息未计增值税 | 增值税暂行条例第6条          |
| 视同销售未处理   | 无偿赠送、投资、分配未确认收入 | 增值税暂行条例实施细则第4条  |
| 关联方收入偏低   | 关联交易定价不公允             | 企业所得税法第41条           |

### 2. 成本费用类风险

| 风险点             | 表现                       | 法规依据                      |
| ------------------ | -------------------------- | ----------------------------- |
| 发票不合规         | 白条入账、虚开发票         | 企业所得税法第8条             |
| 限额扣除超标       | 招待费、广宣费、福利费超标 | 企业所得税法实施条例第43-44条 |
| 资本化与费用化混淆 | 应资本化的支出直接费用化   | CAS 4、CAS 6                  |
| 跨期费用           | 费用未在正确期间确认       | 权责发生制原则                |

### 3. 资产类风险

| 风险点          | 表现                 | 法规依据                  |
| --------------- | -------------------- | ------------------------- |
| 减值/跌价不充分 | 资产已减值但未计提   | CAS 8                     |
| 折旧方法不当    | 折旧年限或方法不合理 | CAS 4、企业所得税法第32条 |
| 资产虚增        | 虚构资产或高估价值   | CAS 基本准则              |

## 九、财务造假与财务洗澡识别

> 类型：流程
> 来源：见 financial_fraud.md
> 置信度：实务经验

财务造假识别（动机、高风险科目、造假技术、掩护机制、反舞弊方法、Red Flags、洗澡 vs 造假、恒大/五粮液案例）已统一存放，本文件不重复。

> 详见：financial_fraud.md

报表审查中触发以下情形时，转 financial_fraud.md 做深度判断：

- 利润高但经营现金流持续为负 / 收现比、净现比长期 < 1 → 造假方向核查
- 利润断崖式下跌但现金流基本正常、且伴随换帅/被查/行业下行 → 洗澡方向核查
- 收入确认口径变更但不追溯调整、新增大额过渡科目 → 口径切换洗澡核查
- 应收/存货增速显著超过收入增速、毛利率显著偏离同行 → 收入造假核查

本文件第四章"税务风险识别"中的收入类、成本费用类风险是从税务合规角度切入，与 financial_fraud.md 的造假识别角度互补。

## 十一、重要提示

1. **仅供专业参考**：审查意见基于预设规则和法规库生成，需由持证会计师或税务师复核
2. **法规时效性**：法规依据需确认是否有最新修订
3. **专业判断**：风险识别和会计处理建议需结合具体业务实质判断
4. **保密要求**：财务数据涉及商业秘密，请确保在安全环境中使用
5. **洗澡 ≠ 违法**：财务洗澡是盈余管理行为，形式上合规但损害信息可比性；财务造假是违法行为。两者的识别方法不同，后果不同，但都需要投资者警惕
