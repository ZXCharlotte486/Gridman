# 报表审查与财务异常识别

> 定位：报表勾稽关系验证、税务风险识别、财务异常检测
> 来源：CAS 准则 + 证监会监管实务 + 上市公司真实案例
> 覆盖：勾稽关系、关键比率、税务风险、收入操纵、成本操纵
> 性质：判断+实操

财务造假与财务洗澡的识别（含恒大/五粮液案例）详见：financial_fraud.md

## 一、报表合规性审查

### 审查维度

#### 1. 准则合规性

- 报表格式是否符合财政部规定
- 会计政策选择是否合理且一致
- 会计估计变更是否充分披露
- 附注披露是否完整

#### 2. 勾稽关系验证

**报表内勾稽**：

| 检查项         | 公式                             | 说明     |
| -------------- | -------------------------------- | -------- |
| 资产负债表平衡 | 资产 = 负债 + 所有者权益         | 基本等式 |
| 利润表结构     | 营业利润 + 营业外收支 = 利润总额 | 利润构成 |
| 净利润         | 利润总额 - 所得税费用 = 净利润   | 税后利润 |

**报表间勾稽**：

| 检查项     | 关系                               | 说明     |
| ---------- | ---------------------------------- | -------- |
| 净利润一致 | 利润表净利润 = 权益变动表净利润    | 跨表核对 |
| 未分配利润 | 期初 + 净利润 - 分配 = 期末        | 连续性   |
| 现金流量表 | 期末现金 - 期初现金 = 现金净增加额 | 现金变动 |

#### 3. 关键比率分析

| 类别     | 指标         | 计算公式           | 关注点     |
| -------- | ------------ | ------------------ | ---------- |
| 偿债能力 | 流动比率     | 流动资产/流动负债  | < 1 警示   |
| 偿债能力 | 资产负债率   | 负债/资产          | > 70% 关注 |
| 盈利能力 | 毛利率       | (收入-成本)/收入   | 异常波动   |
| 盈利能力 | 净利率       | 净利润/收入        | 与行业对比 |
| 营运能力 | 应收周转天数 | 应收/日均收入×365 | 持续延长   |
| 营运能力 | 存货周转天数 | 存货/日均成本×365 | 持续延长   |
| 现金质量 | 收现比       | 经营现金流/净利润  | 长期 < 1   |

## 二、策略库系统

### 内置策略列表

| 策略名称     | 策略ID              | 功能描述                                 | 适用税种           |
| ------------ | ------------------- | ---------------------------------------- | ------------------ |
| 税款比对分析 | tax_reconciliation  | 比对已纳税款与财务报表，识别少缴税款疑点 | 增值税、企业所得税 |
| 收入确认审查 | revenue_recognition | 基于 CAS 14 审查收入确认合规性           | 增值税、企业所得税 |
| 成本操纵识别 | cost_manipulation   | 识别通过成本费用调节利润的行为           | 企业所得税         |
| 税收优惠审查 | tax_preference      | 审查税收优惠政策适用条件和计算准确性     | 企业所得税         |
| 关联交易审查 | related_party       | 审查关联方交易税务合规性                 | 企业所得税、增值税 |

### 策略一：税款比对分析 (tax_reconciliation)

**功能**：通过比对财务报表与纳税申报数据，自动识别少缴税款疑点

**比对维度**：

1. **收入比对**：财务收入 vs 增值税/所得税申报收入
2. **税负率分析**：实际税负率 vs 行业参考值
3. **限额扣除检查**：福利费、招待费、广宣费等超标识别
4. **理论税额测算**：基于财务数据测算理论应纳税额

**预警规则**：

- 财务收入 > 增值税申报收入 × 1.05 → 高风险
- 实际增值税税负率 < 行业平均 × 0.7 → 中风险
- 业务招待费 > MIN(发生额×60%, 收入×0.5%) → 超标

### 策略二：收入确认审查 (revenue_recognition)

**功能**：基于 CAS 14 审查收入确认合规性，识别提前/虚构收入

**审查要点**：

- 收入确认是否满足五步法（识别合同→识别履约义务→确定交易价格→分摊→确认收入）
- 是否存在提前确认（发货未签收即确认）
- 是否存在虚构收入（无真实交易背景）
- 退货率是否异常
- 收入与现金流是否匹配

**异常信号**：

- 季末/年末收入占比异常高（>40%）
- 新客户贡献大额收入但无后续交易
- 收入增长但经营现金流未同步增长
- 应收账款增速远超收入增速

### 策略三：成本操纵识别 (cost_manipulation)

**功能**：识别通过成本费用调节利润的行为

**审查要点**：

- 毛利率异常波动（同比变动 > 5个百分点需关注）
- 成本结转方法变更
- 期间费用突然大幅变动
- 研发费用资本化比例异常
- 折旧政策变更

**异常信号**：

- 毛利率逆行业趋势上升
- 管理费用/销售费用突然大幅下降
- 在建工程长期不转固
- 研发资本化率远高于同行

### 策略四：税收优惠审查 (tax_preference)

**功能**：审查企业享受的税收优惠政策是否符合条件，关注政策时效性和业务实质

**审查维度**：

1. **小型微利企业优惠**：应纳税所得额≤300万、从业人数≤300人、资产总额≤5000万
2. **高新技术企业**：研发费用占比、高新技术产品收入占比、科技人员占比、证书有效性
3. **研发费用加计扣除**：负面清单行业限制、研发费用归集准确性、委托研发比例
4. **固定资产加速折旧/一次性扣除**：500万以下设备器具、资产类型合规性
5. **区域性优惠**：西部大开发（鼓励类产业、收入占比60%）、海南自贸港
6. **政策时效性**：检查优惠政策是否已过期或即将到期

### 策略五：关联交易审查 (related_party)

**功能**：审查关联方交易的税务合规性，防范资本弱化、转让定价和隐性利润分配风险

**审查维度**：

1. **资本弱化审查**：债资比2:1（一般企业）或5:1（金融企业），超标利息不得扣除
2. **关联方利息审查**：利率合理性（基准利率区间）、非金融企业借款凭证合规性
3. **关联方资金占用**：长期占用未收利息、大额资金往来异常
4. **转让定价审查**：关联方销售/采购毛利率差异、定价公允性
5. **关联方担保**：未收担保费、担保实际履行损失
6. **向关联方发放福利**：视同销售处理
7. **同期资料义务**：本地文档（关联交易2亿+）、主体文档（营收55亿+）

## 三、审查深度配置

### 1. 快速审查（30分钟）

- 报表格式合规性检查
- 主要科目异常变动识别
- 重大税务风险点识别
- 执行策略：税款比对（基础模式）
- 适用于：初步筛查、内部汇报

### 2. 标准审查（2-4小时）

- 全科目余额变动分析
- 会计政策适用性审查
- 主要税种风险识别
- 报表间勾稽关系验证
- 执行策略：税款比对 + 收入确认审查
- 适用于：年报审计、税务自查

### 3. 深度审查（1-3天）

- 详细凭证抽查
- 完整税务合规审查
- 内部控制评价
- 舞弊风险专项分析
- 执行策略：全部策略组合
- 适用于：投资尽调、IPO审计、税务稽查应对

## 四、税务风险识别

### 1. 收入类风险

| 风险点           | 表现                           | 法规依据                     |
| ---------------- | ------------------------------ | ---------------------------- |
| 收入确认时点不当 | 提前/延迟确认收入              | CAS 14、增值税暂行条例第19条 |
| 价外费用未计税   | 违约金、延期付款利息未计增值税 | 增值税暂行条例第6条          |
| 视同销售未处理   | 无偿赠送、投资、分配未确认收入 | 增值税暂行条例实施细则第4条  |
| 关联方收入偏低   | 关联交易定价不公允             | 企业所得税法第41条           |

### 2. 成本费用类风险

| 风险点             | 表现                       | 法规依据                      |
| ------------------ | -------------------------- | ----------------------------- |
| 发票不合规         | 白条入账、虚开发票         | 企业所得税法第8条             |
| 限额扣除超标       | 招待费、广宣费、福利费超标 | 企业所得税法实施条例第43-44条 |
| 资本化与费用化混淆 | 应资本化的支出直接费用化   | CAS 4、CAS 6                  |
| 跨期费用           | 费用未在正确期间确认       | 权责发生制原则                |

### 3. 资产类风险

| 风险点          | 表现                 | 法规依据                  |
| --------------- | -------------------- | ------------------------- |
| 减值/跌价不充分 | 资产已减值但未计提   | CAS 8                     |
| 折旧方法不当    | 折旧年限或方法不合理 | CAS 4、企业所得税法第32条 |
| 资产虚增        | 虚构资产或高估价值   | CAS 基本准则              |

## 五、审查报告输出

### 1. 报告结构

1. 审查范围和依据
2. 总体评价和意见类型
3. 具体问题清单（按重要性分级）
4. 调整分录建议
5. 管理建议书

### 2. 风险等级定义

| 等级   | 标识 | 含义                       | 行动     |
| ------ | ---- | -------------------------- | -------- |
| 高风险 | 🔴   | 可能导致重大错报或税务处罚 | 立即处理 |
| 中风险 | 🟡   | 存在合规隐患，需要关注     | 限期整改 |
| 低风险 | 🟢   | 轻微问题，不影响整体判断   | 建议改进 |

## 九、财务造假与财务洗澡识别

> 类型：流程
> 来源：见 financial_fraud.md
> 置信度：实务经验

财务造假识别（动机、高风险科目、造假技术、掩护机制、反舞弊方法、Red Flags、洗澡 vs 造假、恒大/五粮液案例）已统一存放，本文件不重复。

> 详见：financial_fraud.md

报表审查中触发以下情形时，转 financial_fraud.md 做深度判断：

- 利润高但经营现金流持续为负 / 收现比、净现比长期 < 1 → 造假方向核查
- 利润断崖式下跌但现金流基本正常、且伴随换帅/被查/行业下行 → 洗澡方向核查
- 收入确认口径变更但不追溯调整、新增大额过渡科目 → 口径切换洗澡核查
- 应收/存货增速显著超过收入增速、毛利率显著偏离同行 → 收入造假核查

本文件第四章"税务风险识别"中的收入类、成本费用类风险是从税务合规角度切入，与 financial_fraud.md 的造假识别角度互补。

## 十一、重要提示

1. **仅供专业参考**：审查意见基于预设规则和法规库生成，需由持证会计师或税务师复核
2. **法规时效性**：法规依据需确认是否有最新修订
3. **专业判断**：风险识别和会计处理建议需结合具体业务实质判断
4. **保密要求**：财务数据涉及商业秘密，请确保在安全环境中使用
5. **洗澡 ≠ 违法**：财务洗澡是盈余管理行为，形式上合规但损害信息可比性；财务造假是违法行为。两者的识别方法不同，后果不同，但都需要投资者警惕
