## 第一章 税法总论

### 税法基本原则

**税法基本原则**（统领所有税收规范的根本准则）：

1. **税收法定原则**（核心原则）：税法主体的权利义务必须由法律规定，税法各类构成要素皆必须且只能由法律予以明确
   - 税收要件法定原则：纳税人、课税对象、课税标准等必须以法律形式规定
   - 税务合法性原则：税务机关按法定程序依法征税，不得随意减征、停征或免征
2. **税收公平原则**：税收负担根据纳税人负担能力分配（横向公平+纵向公平）
3. **税收效率原则**：经济效率（有利于资源配置）+ 行政效率（节约征管成本）
4. **实质课税原则**：根据客观事实和真实负担能力决定税负，不仅考虑外观和形式

**税法适用原则**：

- 法律优位原则：法律效力高于行政立法
- 法律不溯及既往原则
- 新法优于旧法原则
- 特别法优于普通法原则
- 实体从旧、程序从新原则
- 程序优于实体原则

### 税收法律关系

**构成三要素**：

- **主体**：征税方（税务机关、海关）+ 纳税方（法人、自然人、其他组织，含外国企业和外籍个人）
- **客体**：征税对象（所得、财产、流转额等）
- **内容**：双方权利义务（税务机关：征税权、检查权、处罚权；纳税人：退税权、延期权、减免权、复议诉讼权）

**产生、变更与消灭**：由税收法律事实决定（法律事件如自然灾害→减免；法律行为如开业→产生、停业→消灭）

### 税法与其他法律的关系

| 法律部门 | 与税法的关系                                                                             |
| -------- | ---------------------------------------------------------------------------------------- |
| 宪法     | 税法的立法依据；宪法第56条"公民有依照法律纳税的义务"                                     |
| 民法     | 税法可援引民法条款（如合同关系、产权认定）；但涉及征纳关系时以税法为准（如关联交易调整） |
| 刑法     | 违反税法不一定构成犯罪；情节严重才追究刑事责任（逃避缴纳税款罪等）                       |
| 行政法   | 税法具有行政法一般特性（国家单方面意志、命令服从）；但税法还具有经济分配性质             |

### 税法要素

| 要素             | 含义                                 | 要点                                                     |
| ---------------- | ------------------------------------ | -------------------------------------------------------- |
| 纳税义务人       | 税法规定直接负有纳税义务的单位和个人 | 自然人/法人；代扣代缴义务人/代收代缴义务人               |
| 征税对象         | 对什么征税，区别税种的重要标志       | 按性质分：流转额、所得额、财产、资源、特定行为           |
| 税目             | 征税对象的具体项目                   | 明确征税范围；并非所有税种都设税目（如企业所得税无税目） |
| 税基（计税依据） | 据以计算应纳税款的直接数量依据       | 从价计征（价值形态）或从量计征（物理形态）               |
| 税率             | 征收比例或程度                       | 比例税率/累进税率/定额税率（详见下方）                   |
| 纳税环节         | 应当缴纳税款的环节                   | 生产/流通/消费/分配等环节                                |
| 纳税期限         | 纳税义务发生时间+申报期限+缴库期限   | 各税种不同                                               |
| 减税免税         | 对特定纳税人和征税对象的减免规定     | 法定减免/特定减免/临时减免                               |

### 税率类型详解

#### （一）比例税率

对同一征税对象不分数额大小，规定相同征收比例。

- 单一比例税率：所有纳税人同一税率（如企业所得税25%）
- 差别比例税率：按产品/行业/地区不同适用不同税率（如增值税13%/9%/6%）
- 幅度比例税率：规定最低和最高税率，各地在幅度内确定（如契税3%-5%）

#### （二）累进税率

**超额累进税率**（我国个人所得税采用）：

- 将征税对象按数额大小分成若干等级，每一等级规定一个税率
- 每一纳税人的征税对象依所属等级同时适用几个税率分别计算
- 用速算扣除数简化计算

```
应纳税额 = 应纳税所得额 × 适用税率 - 速算扣除数

速算扣除数 = 按全额累进计算的税额 - 按超额累进计算的税额
```

**计算示例**：某人月应纳税所得额6,000元，适用三级超额累进税率（5000以下10%，5000-20000为20%）：

- 分级计算：5000×10% + 1000×20% = 500 + 200 = 700元
- 速算法：6000×20% - 500（速算扣除数）= 700元

**超率累进税率**（土地增值税采用）：

- 以征税对象数额的相对率（增值率）为累进依据
- 四级超率累进：增值率≤50%税率30%，50%-100%税率40%，100%-200%税率50%，>200%税率60%

#### （三）定额税率

按征税对象的计量单位直接规定固定税额。如城镇土地使用税按每平方米定额征收。

## 第二章 增值税法

> 《中华人民共和国增值税法》2024年12月25日通过，2026年1月1日起施行。增值税为我国第一大税种，2024年国内增值税收入约6.57万亿元，占全部税收收入38%。

### 纳税人分类

| 类别         | 标准                          | 计税方法                       |
| ------------ | ----------------------------- | ------------------------------ |
| 一般纳税人   | 年应征增值税销售额 > 500万元  | 一般计税方法（销项-进项）      |
| 小规模纳税人 | 年应征增值税销售额 ≤ 500万元 | 简易计税方法（销售额×征收率） |

- 自然人属于小规模纳税人
- 年应征增值税销售额：连续不超过12个月或4个季度的累计销售额
- 偶然发生的销售无形资产、转让不动产的销售额不计入
- 小规模纳税人会计核算健全的，可申请登记为一般纳税人
- 登记为一般纳税人后，不得转为小规模纳税人

### 征税范围

**应税交易**：在境内销售货物、服务、无形资产、不动产

**货物**：有形动产、电力、热力、气体等

**服务**（六大类）：

1. 交通运输服务（陆路/水路/航空/管道运输）
2. 邮政服务
3. 电信服务（基础电信/增值电信）
4. 建筑服务（工程/安装/修缮/装饰/其他）
5. 金融服务（贷款/直接收费金融/保险/金融商品转让）
6. 生产生活服务（研发和技术/信息技术/文化创意/物流辅助/租赁/鉴证咨询/广播影视/商务辅助/文化体育/教育医疗/旅游娱乐/餐饮住宿/居民日常/加工修理修配/其他）

**视同应税交易**：

1. 单位和个体工商户将自产或委托加工的货物用于集体福利或个人消费
2. 单位和个体工商户无偿转让货物
3. 单位和个人无偿转让无形资产、不动产或金融商品

**非应税交易**（不征收增值税）：

1. 员工为受雇单位提供取得工资薪金的服务
2. 收取行政事业性收费、政府性基金
3. 依照法律规定被征收、征用而取得补偿
4. 取得存款利息收入
5. 符合条件的资产重组

### 税率表

| 税率          | 适用范围                                                                                                                                                                         |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **13%** | 销售货物、加工修理修配服务、有形动产租赁服务、进口货物（除适用9%/0%外）                                                                                                          |
| **9%**  | 交通运输、邮政、基础电信、建筑服务、不动产租赁、销售不动产、转让土地使用权；农产品、食用植物油、食用盐、自来水、暖气、煤气、天然气、图书报纸杂志、饲料、化肥、农药、农机、农膜等 |
| **6%**  | 增值电信服务、金融服务、现代服务（不动产租赁除外）、生活服务、销售无形资产（转让土地使用权除外）                                                                                 |
| **0%**  | 出口货物（另有规定除外）；境内单位和个人跨境销售国务院规定范围内的服务、无形资产                                                                                                 |

### 征收率

| 征收率         | 适用情形                                           |
| -------------- | -------------------------------------------------- |
| **3%**   | 小规模纳税人基本征收率；一般纳税人特定简易计税项目 |
| **5%**   | 不动产相关（销售/出租2016.4.30前取得的不动产等）   |
| **2%**   | 销售自己使用过的固定资产（3%减按2%）；销售旧货     |
| **1.5%** | 个人出租住房（3%减按1.5%）                         |
| **1%**   | 小规模纳税人3%征收率减按1%（至2027.12.31）         |
| **0.5%** | 二手车经销（至2027.12.31）                         |

### 一般计税方法

```
当期应纳税额 = 当期销项税额 - 当期进项税额
销项税额 = 销售额 × 适用税率
含税销售额换算：销售额 = 含税销售额 ÷ (1 + 税率)
```

### 销售额确认规则

**一般规定**：纳税人发生应税交易取得的全部价款（不含销项税额），不包括：

- 政府性基金或行政事业性收费
- 受托加工应征消费税的消费品所产生的消费税
- 车辆购置税、车船税
- 以委托方名义开具发票代委托方收取的款项

**特殊销售方式**：

| 方式       | 销售额确定                                                                                            |
| ---------- | ----------------------------------------------------------------------------------------------------- |
| 折扣销售   | 价款和折扣额在同一张发票"金额"栏分别注明→按折扣后销售额；否则不得扣减                                |
| 以旧换新   | 按新货物同期销售价格（金银首饰除外，按实际收取价款）                                                  |
| 还本销售   | 按货物销售价格，不得扣减还本支出                                                                      |
| 以物易物   | 双方均作购销处理，各自按发出货物核算销售额                                                            |
| 包装物押金 | 单独记账、1年内未过期→不并入；逾期/超1年→并入（换算为不含税价）；酒类（除啤酒黄酒外）→收取时即并入 |

**视同应税交易及非货币形式销售额**：按市场价格确定

**税务机关核定销售额**（销售额明显偏低且无正当理由）：

1. 纳税人最近时期同类平均价格
2. 其他纳税人最近时期同类平均价格
3. 组成计税价格 = 成本 × (1 + 成本利润率) + 消费税税额（成本利润率10%）

### 进项税额抵扣规则

**准予抵扣的进项税额**（凭合法扣税凭证）：

1. 增值税专用发票上注明的税额
2. 海关进口增值税专用缴款书上注明的税额
3. 自境外购进服务/无形资产/不动产取得的完税凭证上注明的税额
4. 购进农产品：
   - 取得一般纳税人专票/海关缴款书→按票面税额
   - 从小规模纳税人取得3%专票→按金额×9%计算
   - 取得农产品销售发票/收购发票→买价×9%
5. 国内旅客运输服务（本单位员工）：
   - 铁路电子客票/航空电子客票行程单→发票上列明的税额
   - 公路水路客票→票面金额÷(1+3%)×3%
6. 道路桥闸通行费：
   - 通行费电子发票→发票上列明的税额
   - 桥闸通行费发票→金额÷(1+5%)×5%

**不得抵扣的进项税额**：

1. 适用简易计税方法计税项目对应的进项税额
2. 免征增值税项目对应的进项税额
3. 购进并用于集体福利的货物/服务/无形资产/不动产
4. 购进并用于个人消费的（含交际应酬消费）
5. 用于不得抵扣非应税交易的
6. 非正常损失（管理不善被盗/丢失/霉烂变质；违法被没收/销毁/拆除）
7. 购进并直接用于消费的餐饮服务、居民日常服务和娱乐服务
8. 购进贷款服务及相关投融资顾问费、手续费、咨询费（暂不得抵扣）

**混合用途长期资产进项税额处理**：

- 原值≤500万元：进项税额可全额抵扣
- 原值>500万元：购进时先全额抵扣，此后按调整年限逐年调整
  - 不动产/土地使用权：调整年限20年
  - 飞机/火车/轮船：调整年限10年
  - 其他长期资产：调整年限5年

### 简易计税方法

```
应纳税额 = 销售额 × 征收率
销售额 = 含税销售额 ÷ (1 + 征收率)
```

- 不得抵扣进项税额
- 一般纳税人选择简易计税后36个月内不得变更

### 混合销售与兼营

- **一项应税交易涉及两个以上税率**：按主要业务适用税率
- **两项以上应税交易涉及不同税率**：应分别核算；未分别核算的从高适用税率

### 出口退（免）税政策

**三种政策**：

1. 免税并退税（免抵退税/免退税）
2. 免税不退税
3. 不免税不退税（适用出口禁止/限制类货物）

#### 免抵退税法（生产企业自营或委托出口自产货物）

```
当期应纳税额 = 当期销项税额 - (当期进项税额 - 当期不得免征和抵扣税额)
当期不得免征和抵扣税额 = 当期出口货物离岸价 × 外汇人民币折合率 × (出口货物适用税率 - 出口退税率)

当期免抵退税额 = 当期出口货物离岸价 × 外汇人民币折合率 × 出口退税率
```

**比较判断**：

- 当期应纳税额 ≥ 0：当期应退税额 = 0，当期免抵税额 = 当期免抵退税额
- 当期期末留抵税额 ≤ 当期免抵退税额：当期应退税额 = 当期期末留抵税额，当期免抵税额 = 当期免抵退税额 - 当期应退税额
- 当期期末留抵税额 > 当期免抵退税额：当期应退税额 = 当期免抵退税额，当期免抵税额 = 0

#### 免退税法（外贸企业购进货物出口）

```
增值税应退税额 = 增值税退(免)税计税依据 × 出口货物退税率
```

- 退税计税依据 = 购进出口货物的增值税专用发票注明的金额（不含税价）
- 退税率低于适用税率的差额部分计入出口货物成本

#### 出口退税率

- 一般货物：与适用税率相同（13%/9%）
- 部分货物：降低退税率（如10%/6%/0%等）
- 退税率为0%的：视同内销征税

> **政策更新（财政部 税务总局公告2026年第2号，2026.4.1起施行）**：
> - 自2026年4月1日起，**取消光伏产品增值税出口退税**（包括太阳能电池、组件、逆变器及相关部件）
> - 上述产品出口应视同内销货物征税（适用"出口不免税也不退税"政策）
> - 背景：2024年12月已将光伏产品出口退税率从13%降至9%，此次进一步完全取消

### 境外旅客购物离境退税

**定义**：境外旅客在离境口岸离境时，对其在退税商店购买的退税物品退还增值税。

**法规依据**：财政部公告2015年第3号《关于实施境外旅客购物离境退税政策的公告》

**适用条件**：

| 要素 | 规定 |
|------|------|
| 适用对象 | 在中国境内连续居住不超过183天的外国人和港澳台同胞 |
| 退税物品 | 境外旅客本人在退税商店购买且符合退税条件的个人物品 |
| 不适用物品 | 禁止/限制出境物品；适用增值税免税政策的物品；财政部等规定的其他物品 |
| 起退金额 | 同一境外旅客同一日在同一退税商店购买退税物品金额达到500元人民币 |
| 退税率 | 11%（增值税税率13%，扣除退税代理机构手续费后实退11%） |
| 退税方式 | 现金退税（限10,000元以下）或银行转账 |
| 时间要求 | 购买退税物品之日起90天内离境 |

**退税计算公式**：

```
应退增值税额 = 退税物品销售发票金额（含增值税） ÷ (1 + 增值税税率) × 退税率
```

#### 离境退税优化2.0版（2026年5月18日发布）

**发文机关**：商务部、财政部、国家税务总局、海关总署等六部门联合发布

**背景**：2025年实施1.0版优化措施后，全国使用离境退税的境外旅客同比增长305%。2.0版在此基础上进一步升级。

**八项核心措施**：

| 序号 | 措施 | 要点 |
|------|------|------|
| 1 | 扩大退税商店覆盖 | 鼓励更多符合条件的商店登记为退税商店，科学优化布局；在外国旅客集中的重点商圈、景区、市场、口岸实现退税商店近全覆盖 |
| 2 | 小额退税随机抽查制 | 2026年7月1日起，销售额＜10,000元的退税申请按一定比例随机抽查实物验核；≥10,000元的仍逐笔实物验核 |
| 3 | 无纸化办理 | 退税商店通过电子发票直接向海关传输数据，实现机场退税柜台当日数字化验核 |
| 4 | "即买即退"服务优化 | 实现跨口岸互认——旅客在A城市享受"即买即退"后可在B城市口岸完成离境退税手续 |
| 5 | "即买即退"离境期限统一延长 | 所有地区"即买即退"离境期限统一延长至**28天** |
| 6 | 数字化退税系统 | 2026年7月1日起，北京、上海、深圳等19个试点城市的退税商店向海关传输电子发票，实现当日数字化验核 |
| 7 | 退税服务网点优化 | 在主要离境口岸增设退税服务窗口，延长服务时间 |
| 8 | 信息化监管升级 | 加强退税数据共享和风险监控，防范骗税风险 |

**对企业的影响**：

- **零售企业（退税商店）**：需对接电子发票传输系统，配合无纸化流程改造
- **退税代理机构**：业务量预计持续增长，需升级系统处理能力
- **税务机关**：从逐笔验核转向风险导向的随机抽查模式（小额），降低征管成本

#### "即买即退"服务模式

**定义**：境外旅客在提供"即买即退"服务的退税商店购物时，签订协议并办理信用卡预授权后，现场即可领取与退税款等额的预付金。

**流程**：
1. 旅客在"即买即退"商店购买退税物品，取得增值税发票和退税申请单
2. 旅客扫码上传退税资料，签订协议并办理信用卡预授权，现场领取退税预付金
3. 旅客在规定期限内（统一28天）从指定口岸离境，向海关申报并接受验核
4. 海关验核签章后，旅客将退税申请单和发票投递至退税柜台信箱
5. 退税代理机构审核无误→解除预授权；不符合规定→通过预授权扣回预付金

**适用条件**（在一般离境退税条件基础上增加）：
- 购物商店属于"即买即退"商店名单
- 旅客持有本人可操作预授权担保的信用卡
- 退税物品由旅客本人随身携带或随行托运从指定口岸离境

### 差额征税

特定行业以全部价款和价外费用扣除规定项目后的余额为销售额：

| 行业/业务                  | 可扣除项目                                                                                                           |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| 金融商品转让               | 买入价                                                                                                               |
| 经纪代理服务               | 向委托方收取并代为支付的政府性基金或行政事业性收费                                                                   |
| 融资租赁                   | 借款利息、发行债券利息、车辆购置税                                                                                   |
| 旅游服务                   | 向旅游服务购买方收取并支付给其他单位或个人的住宿费、餐饮费、交通费、签证费、门票费和支付给其他接团旅游企业的旅游费用 |
| 建筑服务（简易计税）       | 支付的分包款                                                                                                         |
| 房地产开发企业（一般计税） | 当期允许扣除的土地价款                                                                                               |
| 劳务派遣（简易计税）       | 代用工单位支付给劳务派遣员工的工资、福利和社保、住房公积金                                                           |
| 人力资源外包               | 向委托方收取并代为发放的工资和代理缴纳的社保、住房公积金                                                             |

```
差额征税销售额 = (全部价款和价外费用 - 允许扣除项目金额) ÷ (1 + 税率或征收率)
```

### 增值税计算综合示例

**示例**：某一般纳税人本月销售货物不含税收入100万元（税率13%），购进原材料取得专票注明税额8万元，购进办公用品取得专票注明税额0.5万元，招待客户餐饮消费取得普票1万元。

```
销项税额 = 100 × 13% = 13万元
准予抵扣进项税额 = 8 + 0.5 = 8.5万元
（餐饮消费不得抵扣，且普票也无法抵扣）
当期应纳增值税 = 13 - 8.5 = 4.5万元
```

| 交易类型                                  | "在境内"的判断标准                                 |
| ----------------------------------------- | -------------------------------------------------- |

**服务、无形资产在境内消费**的具体情形：

1. 境外单位或个人向境内单位或个人销售服务、无形资产（在境外现场消费的服务除外）
2. 境外单位或个人销售的服务、无形资产与境内的货物、不动产、自然资源直接相关
3. 国务院财政、税务主管部门规定的其他情形

### 增值税即征即退政策

| 适用情形     | 即征即退规则                                                                                                               |
| ------------ | -------------------------------------------------------------------------------------------------------------------------- |
| 软件产品     | 一般纳税人销售自行开发生产的软件产品，按13%征税后，实际税负超过3%的部分即征即退（进口软件本地化改造后同样适用）            |
| 管道运输服务 | 一般纳税人提供管道运输服务，实际税负超过3%的部分即征即退（至2027.12.31）                                                   |
| 海上风力发电 | 销售自产利用海上风力生产的电力产品，即征即退50%（2025.11.1至2027.12.31）                                                   |
| 安置残疾人   | 月应退税额 = 本月安置残疾人员人数 × 本月月最低工资标准的4倍（至2027.12.31）                                               |
| 资源综合利用 | 销售自产综合利用产品和劳务，退税比例分30%/50%/70%/90%/100%五档（按《资源综合利用产品和劳务增值税优惠目录(2022年版)》执行） |

**软件产品即征即退计算：**

```
即征即退税额 = 当期软件产品增值税应纳税额 - 当期软件产品销售额 × 3%
当期软件产品增值税应纳税额 = 当期软件产品销项税额 - 当期软件产品可抵扣进项税额
```

**即征即退与留抵退税的衔接：**

- 自2019年4月1日起已享受即征即退、先征后返（退）政策的，一次性将已退还税款全部缴回后，可自全部缴回次月起申请留抵退税
- 自全部缴回次月起36个月内不得变更

### 农产品进项税额加计扣除

纳税人购进农产品用于生产或委托加工**13%税率**货物的，按照**10%扣除率**计算进项税额（在9%基础上加计1%）。

```
进项税额 = 买价 × 10%（用于生产13%税率货物时）
进项税额 = 买价 × 9%（用于生产9%/6%税率货物或直接销售时）
```

**各种取得方式下的农产品进项税额：**

| 取得方式                             | 扣除率/抵扣依据                                |
| ------------------------------------ | ---------------------------------------------- |
| 取得一般纳税人开具的专票或海关缴款书 | 按票面税额抵扣                                 |
| 从小规模纳税人取得3%专票             | 按票面金额 × 9%计算（用于13%税率货物的按10%） |
| 取得农产品销售发票或收购发票         | 按买价 × 9%计算（用于13%税率货物的按10%）     |
| 从批发零售环节购进免税蔬菜/鲜活肉蛋  | 取得的普通发票不得抵扣                         |

**农产品核定扣除（试点行业：液体乳及乳制品、酒及酒精、植物油）：**

- 投入产出法、成本法、参照法
- 购进农产品直接销售：`进项税额 = 当期销售农产品数量 ÷ (1-损耗率) × 平均购买单价 × 9% ÷ (1+9%)`

### 进项税额转出的计算方法

已抵扣进项税额的购进货物、服务等改变用途（用于不得抵扣项目），应从当期进项税额中扣减。

#### 一、一般货物/服务的进项税额转出

**无法划分不得抵扣进项税额的，按销售额占比计算：**

```
当期不得抵扣的进项税额 = 当期无法划分的全部进项税额
  × (简易计税项目销售额 + 免税项目销售额 + 不得抵扣非应税交易收入)
  ÷ 当期全部销售额和非应税交易收入
```

次年1月纳税申报期内进行全年汇总清算。

#### 二、长期资产（固定资产/无形资产/不动产）的进项税额转出

**情形一：已抵扣后专用于五类不允许抵扣项目或发生非正常损失**

```
不得抵扣的进项税额 = 长期资产对应的进项税额 × 净值率
净值率 = 当月期初长期资产净值 ÷ 长期资产原值 × 100%
```

**情形二：原专用于不得抵扣项目后改为一般计税项目或混合用途**

```
可抵扣进项税额 = 长期资产对应的进项税额 × 净值率
```

**混合用途长期资产（原值>500万元）分期调整：**

- 调整年限：不动产/土地使用权20年；飞机/火车/轮船10年；其他5年
- 购进时先全额抵扣，此后逐年调整五类不允许抵扣项目对应的进项税额

#### 三、非正常损失的进项税额转出

非正常损失（因管理不善被盗/丢失/霉烂变质，因违法被没收/销毁/拆除）涉及：

- 损失的购进货物及相关加工修理修配服务和交通运输服务
- 损失的在产品、产成品所耗用的购进货物、加工修理修配服务和交通运输服务
- 损失的不动产及其所耗用的购进货物和建筑服务
- 损失的不动产在建工程所耗用的购进货物和建筑服务

#### 四、平销返利的进项税额转出

```
当期应冲减进项税金 = 当期取得的返还资金 ÷ (1 + 所购货物适用增值税税率) × 所购货物适用增值税税率
```

### 增值税专用发票管理

#### 不得开具增值税专用发票的情形

1. 应税交易的购买方为自然人
2. 应税交易免征增值税
3. 国务院财政、税务主管部门规定的其他情形（如金融商品转让、差额计税中扣除部分等）

#### 虚开增值税专用发票的认定与处理

| 情形                                     | 处理方式                                                                       |
| ---------------------------------------- | ------------------------------------------------------------------------------ |
| 虚开方未就虚开金额申报缴纳增值税         | 按虚开金额补缴增值税                                                           |
| 虚开方已就虚开金额申报缴纳增值税         | 不再补缴，但按规定给予处罚                                                     |
| 受票方取得虚开专票                       | 不得作为合法扣税凭证抵扣进项税额                                               |
| 善意取得虚开专票（存在真实交易且不知情） | 如能重新取得合法专票可抵扣；不能取得的不准抵扣或追缴已抵扣税款（不加收滞纳金） |

#### 异常凭证的认定范围

1. 纳税人丢失、被盗税控设备中未开具或已开具未上传的专票
2. 非正常户未申报或未缴税的专票
3. 稽核比对发现"比对不符""缺联""作废"的专票
4. 大数据分析发现涉嫌虚开、未按规定缴纳消费税等情形的专票
5. 走逃（失联）企业开具的符合特定情形的专票

**异常凭证处理：**

- 尚未抵扣的：暂不允许抵扣
- 已经抵扣的：一律作进项税额转出（纳税信用A级可在10个工作日内申请核实）
- 尚未办理出口退税的：暂不允许办理

#### 数电发票（全面数字化电子发票）要点

- 2024年12月1日起全国推广，单一联次，数字化形态
- 号码20位（年度2位+省级代码2位+渠道1位+顺序编码15位）
- 发票总额度：一个自然月内开具总金额（不含税）的上限，动态调整
- 红字发票：发生销售退回、开票有误、服务中止、销售折让时开具
- 受票方通过税务数字账户确认用途后方可抵扣

### 房地产开发企业增值税特殊规定

#### 预缴税款

```
应预缴税款 = 当期取得的预收款 ÷ (1 + 适用税率或征收率) × 3%
```

- 预缴时间：收到预收款的次月纳税申报期内
- 预缴地点：机构所在地主管税务机关

#### 土地价款扣除（一般计税方法）

房地产开发企业销售适用一般计税方法的房地产项目，可从含税销售额中扣除的土地价款包括：

- 向政府部门支付的土地出让金、征地和拆迁补偿费用、土地前期开发费用和土地出让收益
- 向其他单位或个人支付的货币形式拆迁补偿费用

```
当期允许扣除的土地价款 = (当期销售房地产项目建筑面积 ÷ 房地产项目可供销售建筑面积) × 支付的土地价款
```

#### 老项目简易计税

房地产老项目（开工日期在2016.4.30前）可选择简易计税方法，按5%征收率计税。

#### 项目公司扣除土地价款的条件

房地产开发企业受让土地后设立项目公司开发，同时符合以下条件的，项目公司可扣除土地价款：

1. 三方签订变更协议，将土地受让人变更为项目公司
2. 土地价款总额不变
3. 项目公司存续期间全部股权由受让土地的房地产开发企业持有

### 金融服务增值税特殊规定

#### 贷款服务

- 以全部利息及利息性质的收入为销售额
- **贷款服务进项税额暂不得抵扣**（包括向贷款方支付的投融资顾问费、手续费、咨询费等）
- 金融机构贷款利息：结息日起90天内应收未收利息正常纳税，90天后暂不纳税待实际收到时纳税
- 以货币资金投资收取的固定利润或保底利润，按贷款服务缴纳增值税

#### 金融商品转让

```
销售额 = 卖出价 - 买入价（盈亏相抵后的余额）
```

- 正负差按盈亏相抵后的余额为销售额
- 年末负差不得转入下一会计年度
- 买入价可选加权平均法或移动加权平均法（选择后36个月内不得变更）
- **不得开具增值税专用发票**
- 个人从事金融商品转让免征增值税

#### 保险服务

- 一年期以上人身保险产品保费收入免征增值税
- 保险公司以实物赔付方式承担机动车辆保险责任的，自行向修理方购进的修理劳务进项税额可抵扣
- 以现金赔付方式直接支付给修理方的，不属于保险公司购进修理劳务，进项税额不得抵扣

#### 金融同业往来利息收入免税

包括：金融机构与人民银行资金往来、银行联行往来、金融机构间同业拆借、同业存款/借款/代付/买入返售/同业存单等

### 建筑服务增值税特殊规定

#### 预缴税款规则

| 情形                                 | 预缴地点       | 预征率（一般计税） | 预征率（简易计税） |
| ------------------------------------ | -------------- | ------------------ | ------------------ |
| 跨地级行政区提供建筑服务             | 建筑服务发生地 | 2%                 | 3%                 |
| 采取预收款方式提供建筑服务（非跨区） | 机构所在地     | 2%                 | 3%                 |
| 跨区收取预收款                       | 建筑服务发生地 | 2%                 | 3%                 |

```
应预缴税款 = (全部含税价款或预收款 - 支付的分包款) ÷ (1 + 适用税率或征收率) × 预征率
```

- 按工程项目分别计算、分别预缴
- 扣除分包款后余额为负数的，可结转下期继续扣除
- 预缴时间：纳税义务发生之日起至次月纳税申报期结束前（预收款为收到之日起至次月申报期结束前）

#### 简易计税方法下分包款扣除

纳税人提供建筑服务适用简易计税方法的，支付的分包款允许从含税销售额中扣除。

#### 建筑工程老项目认定

《建筑工程施工许可证》注明的合同开工日期在2016年4月30日前的建筑工程项目（未注明或未取得许可证的，以承包合同注明的开工日期为准）。

### 增值税发票的认证和抵扣期限

- **数电发票**：无认证期限限制，通过税务数字账户确认用途即可抵扣
- **纸质增值税专用发票**：自2020年3月1日起取消认证确认期限（360天）限制，已超期的也可按规定抵扣
- **海关进口增值税专用缴款书**：按规定进行稽核比对后抵扣
- **国内旅客运输服务凭证**：取得电子发票（铁路/航空电子客票）按票面税额抵扣；公路水路客票按 `票面金额÷(1+3%)×3%` 计算抵扣

### 增值税期末留抵退税的详细条件和计算

#### 分行业政策（2025年7号公告）

| 纳税人类型                                                                                  | 适用条件                                                            | 退税计算                                                           |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------- | ------------------------------------------------------------------ |
| 制造业等4个行业（制造业、科学研究和技术服务业、软件和信息技术服务业、生态保护和环境治理业） | 相关业务销售额占比>50%，可按月申请                                  | 允许退还 = 期末留抵税额 × 进项构成比例 × 100%                    |

#### 共同条件

1. 纳税缴费信用级别为A级或B级
2. 申请退税前36个月未发生骗取留抵退税、骗取出口退税或虚开增值税专用发票情形
3. 申请退税前36个月未因偷税被税务机关处罚两次及以上
4. 2019年4月1日起未享受增值税即征即退、先征后返（退）政策（另有规定除外）

#### 进项构成比例

**制造业等4个行业和房地产开发经营业：**

```
进项构成比例 = 2019年4月至申请退税前一税款所属期已抵扣的七类增值税扣税凭证注明的增值税额 ÷ 同期全部已抵扣进项税额
```

**其他纳税人：**

```
进项构成比例 = 申请退税前一税款所属期当年1月至申请退税前一税款所属期已抵扣的七类增值税扣税凭证注明的增值税额 ÷ 同期全部已抵扣进项税额
```

七类增值税扣税凭证：增值税专用发票、海关进口增值税专用缴款书、完税凭证、机动车销售统一发票、收费公路通行费增值税电子普通发票、电子发票（航空运输电子客票行程单）、电子发票（铁路电子客票）。

#### 与其他政策的衔接

- 既有欠税又有留抵的：按最近一期申报表"期末留抵税额"抵减欠税后的余额确定允许退还的留抵税额
- 适用免抵退税的：应先办理免抵退税，办理完毕后仍符合条件的可办理留抵退税
- 纳税人取得退还的留抵退税款后，应相应调减当期留抵税额

### 计税方法

**从价计征**：应纳税额 = 销售额 × 比例税率

**从量计征**：应纳税额 = 销售数量 × 单位税额

**复合计征**（卷烟、白酒）：应纳税额 = 销售数量 × 定额税率 + 销售额 × 比例税率

```
含增值税销售额换算：
销售额 = 含增值税销售额 ÷ (1 + 增值税税率或征收率)
```

### 自产自用应税消费品

- 用于连续生产应税消费品的：不纳税（移送环节不征，最终销售环节征）
- 用于其他方面的（生产非应税消费品、在建工程、管理部门、馈赠、赞助、集资、广告、样品、职工福利、奖励等）：于移送使用时纳税

```
组成计税价格 = (成本 + 利润) ÷ (1 - 比例税率)
组成计税价格（复合计征）= (成本 + 利润 + 自产自用数量×定额税率) ÷ (1 - 比例税率)
```

### 委托加工应税消费品

- 由受托方在向委托方交货时代收代缴消费税（受托方为个人的除外）
- 委托方收回后直接出售（不高于受托方计税价格）：不再缴纳消费税
- 委托方收回后以高于受托方计税价格出售：按规定申报缴纳消费税，准予扣除受托方已代收代缴的消费税

```
组成计税价格 = (材料成本 + 加工费) ÷ (1 - 比例税率)
组成计税价格（复合计征）= (材料成本 + 加工费 + 委托加工数量×定额税率) ÷ (1 - 比例税率)
```

### 进口应税消费品

```
从价计征：
组成计税价格 = (关税完税价格 + 关税) ÷ (1 - 消费税比例税率)
应纳税额 = 组成计税价格 × 消费税税率

从量计征：
应纳税额 = 进口应税消费品数量 × 单位税额

复合计征：
组成计税价格 = (关税完税价格 + 关税 + 进口数量×定额税率) ÷ (1 - 比例税率)
应纳税额 = 进口数量×定额税率 + 组成计税价格×比例税率
```

### 已纳消费税的扣除

外购或委托加工收回的应税消费品用于连续生产应税消费品的，已纳消费税准予从应纳消费税税额中扣除。

可抵扣的税目组合（主要）：

- 外购已税烟丝→生产卷烟
- 外购已税高档化妆品→生产高档化妆品
- 外购已税珠宝玉石→生产贵重首饰及珠宝玉石
- 外购已税鞭炮焰火→生产鞭炮焰火
- 外购已税杆头/杆身/握把→生产高尔夫球杆
- 外购已税木制一次性筷子→生产木制一次性筷子
- 外购已税实木地板→生产实木地板
- 外购已税石脑油/润滑油/燃料油→生产成品油
- 外购已税汽油/柴油→连续生产应税成品油

**不得扣除**：用于生产非应税消费品的已纳消费税

### 出口退（免）税

- 出口应税消费品：免征消费税（有出口经营权的自营出口/委托外贸企业代理出口）
- 外贸企业受其他企业委托代理出口：退还已纳消费税

### 征收管理

- 纳税义务发生时间：与增值税基本一致
- 纳税地点：纳税人核算地（进口为报关地海关）
- 纳税期限：1日/3日/5日/10日/15日/1个月/1个季度

### 消费税改革方向（2026年政府工作报告）

> **政策动态（2026年3月5日，第十四届全国人大四次会议）**：
> 2026年《政府工作报告》明确提出：**调整优化消费税征税范围、税率，并推进部分品目征收环节后移。**
>
> **解读要点**：
> - **征税范围调整**：预计将部分高端消费品/高污染产品纳入，同时可能将不再属于"奢侈品"的品目移出
> - **税率优化**：根据消费引导目标差异化调整
> - **征收环节后移**：从生产环节后移至批发或零售环节，有利于增加地方财政收入（消费税目前为中央税）
> - **立法进程**：消费税法草案已列入国务院2026年度立法工作计划
>
> 此项改革尚在推进中，具体实施细则待正式文件发布。企业应关注后续财政部、税务总局公告。

## 第四章 企业所得税法

> 《中华人民共和国企业所得税法》2007年3月16日通过，2008年1月1日起施行。

### 纳税人

| 类别       | 定义                                                                 | 纳税义务                                                                              |
| ---------- | -------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| 居民企业   | 依法在中国境内成立，或依照外国法律成立但实际管理机构在中国境内的企业 | 就来源于中国境内、境外的全部所得纳税                                                  |
| 非居民企业 | 依照外国法律成立且实际管理机构不在中国境内的企业                     | 境内设机构场所→机构场所所得+与之有实际联系的境外所得；未设机构场所→来源于境内的所得 |

- 个人独资企业、合伙企业不适用企业所得税法
- 实际管理机构：对企业生产经营、人员、账务、财产等实施实质性全面管理和控制的机构

### 税率

| 税率                     | 适用对象                                                       |
| ------------------------ | -------------------------------------------------------------- |
| **25%**            | 基本税率，居民企业和在境内设机构场所且所得与之有关的非居民企业 |
| **20%**（实际10%） | 非居民企业（未设机构场所/所得与机构场所无实际联系）的境内所得  |
| **15%**            | 高新技术企业、技术先进型服务企业、西部大开发、海南自贸港等     |
| **20%**            | 小型微利企业优惠税率                                           |

### 应纳税所得额计算

```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 允许弥补的以前年度亏损
```

### 收入总额

包括货币形式和非货币形式从各种来源取得的收入：

- 销售货物收入、提供劳务收入、转让财产收入
- 股息红利等权益性投资收益（按被投资方作出利润分配决定的日期确认）
- 利息收入（按合同约定应付利息日期确认）
- 租金收入（按合同约定应付租金日期确认；跨年度提前一次性支付的可分期均匀计入）
- 特许权使用费收入、接受捐赠收入、其他收入

**视同销售**：将资产用于市场推广、交际应酬、职工福利、股息分配、对外捐赠等→按公允价值确认收入

### 不征税收入

1. 财政拨款
2. 依法收取并纳入财政管理的行政事业性收费、政府性基金
3. 国务院规定的其他不征税收入（专项用途财政性资金，需同时满足：有资金拨付文件+有专门管理办法+单独核算）

> 注意：不征税收入用于支出形成的费用/资产折旧摊销不得税前扣除；5年内未支出且未缴回的应计入第6年应税收入

### 免税收入

1. 国债利息收入
2. 符合条件的居民企业之间的股息、红利等权益性投资收益（不含连续持有居民企业公开发行并上市流通的股票不足12个月取得的投资收益）
3. 在中国境内设立机构场所的非居民企业从居民企业取得与该机构场所有实际联系的股息红利等
4. 符合条件的非营利组织的收入

**扣除原则**：实际发生、与取得收入相关、合理

- 向金融企业借款：据实扣除
- 向非金融企业借款：不超过按金融企业同期同类贷款利率计算的数额部分准予扣除
- 关联方债资比：金融企业5:1，其他企业2:1

**不得扣除的项目**：

1. 向投资者支付的股息、红利等权益性投资收益款项
2. 企业所得税税款
3. 税收滞纳金
4. 罚金、罚款和被没收财物的损失
5. 超过规定标准的捐赠支出
6. 赞助支出（非广告性质）
7. 未经核定的准备金支出
8. 与取得收入无关的其他支出

### 资产的税务处理

**固定资产最低折旧年限**：

| 资产类别                                   | 最低折旧年限 |
| ------------------------------------------ | ------------ |
| 房屋、建筑物                               | 20年         |
| 飞机、火车、轮船、机器、机械和其他生产设备 | 10年         |
| 与生产经营活动有关的器具、工具、家具等     | 5年          |
| 飞机、火车、轮船以外的运输工具             | 4年          |
| 电子设备                                   | 3年          |

**无形资产摊销**：直线法，≥10年；外购商誉不得摊销，在整体转让或清算时扣除

**长期待摊费用**：≥3年分期摊销

### 税收优惠

**免税收入**：国债利息、符合条件的居民企业间股息红利、非营利组织收入

**减免税所得**：

- 农林牧渔业项目所得（免征/减半）
- 国家重点扶持的公共基础设施项目投资经营所得（自取得第一笔生产经营收入起，第1-3年免征，第4-6年减半，即"三免三减半"）
- 符合条件的环境保护、节能节水项目所得（三免三减半）
- 符合条件的技术转让所得（≤500万元免征，>500万元减半征收）

**加计扣除**：

- 研发费用：未形成无形资产的，加计扣除**100%**；形成无形资产的，按200%摊销
- 安置残疾人员工资：加计扣除100%

**加速折旧**：

- 技术进步/强烈振动高腐蚀的固定资产：缩短折旧年限（≥最低年限60%）或加速折旧
- 制造业等行业：新购进设备器具单位价值≤500万元的，允许一次性计入当期成本费用扣除

**抵免税额**：

- 企业购置并实际使用环境保护/节能节水/安全生产等专用设备：投资额的**10%**可从当年应纳税额中抵免，不足抵免的可结转以后5个纳税年度

**优惠税率**：

| 企业类型           | 优惠税率            | 条件                                                   |
| ------------------ | ------------------- | ------------------------------------------------------ |
| 高新技术企业       | 15%                 | 核心自主知识产权、研发费用占比、高新收入占比等         |
| 小型微利企业       | 20%（实际税负约5%） | 应纳税所得额≤300万、从业人数≤300人、资产总额≤5000万 |
| 技术先进型服务企业 | 15%                 | 服务贸易类                                             |
| 西部大开发企业     | 15%                 | 至2030年，鼓励类产业收入占比≥60%                      |
| 海南自贸港企业     | 15%                 | 实质性运营+鼓励类产业                                  |

### 应纳税额计算

```
应纳税额 = 应纳税所得额 × 适用税率 - 减免税额 - 抵免税额
```

**境外所得税收抵免**：

- 直接抵免：企业直接缴纳的境外所得税
- 间接抵免：境外子公司已纳税额中属于该企业间接负担的部分
- 抵免限额 = 中国境内外应纳税所得总额 × 25% × (来源于某国应纳税所得额 ÷ 中国境内外应纳税所得总额)
- 超过抵免限额的部分可结转以后5个年度抵补

### 源泉扣缴

非居民企业取得的下列所得，以支付人为扣缴义务人，按10%税率扣缴：

- 股息、红利等权益性投资收益
- 利息、租金、特许权使用费所得
- 转让财产所得
- 其他所得

### 特别纳税调整

1. **关联交易与转让定价**：独立交易原则，可比非受控价格法/再销售价格法/成本加成法/交易净利润法/利润分割法
2. **资本弱化**：关联方债资比超标（金融5:1/其他2:1）的利息支出不得扣除
3. **受控外国企业（CFC）**：设立在实际税负明显偏低（<12.5%）的国家/地区，无合理经营需要对利润不作分配或减少分配的，应将归属于居民企业的部分计入当期收入
4. **一般反避税**：不具有合理商业目的的安排，税务机关有权进行纳税调整
5. 特别纳税调整补征税款：加收利息（基准利率+5个百分点）

### 亏损弥补

| 企业类型                           | 弥补年限 |
| ---------------------------------- | -------- |
| 一般企业                           | 5年      |
| 高新技术企业/科技型中小企业        | 10年     |
| 受疫情影响困难行业企业（2020年度） | 8年      |

- 亏损：企业依照税法规定将每一纳税年度的收入总额减除不征税收入、免税收入和各项扣除后小于零的数额
- 境外亏损不得抵减境内盈利
- 弥补亏损后仍有余额的才计算应纳税所得额

### 企业所得税计算综合示例

**示例**：某居民企业2025年度会计利润1000万元，其中：

- 国债利息收入20万元
- 从居民企业取得股息50万元（持股超12个月）
- 业务招待费60万元，营业收入8000万元
- 广告费1300万元
- 公益性捐赠150万元
- 税收滞纳金5万元
- 研发费用200万元（未形成无形资产）

```
纳税调整：
① 国债利息20万→免税收入，调减20万
② 居民企业股息50万（持股>12个月）→免税收入，调减50万
③ 业务招待费：实际发生60万×60%=36万；收入8000万×5‰=40万；取较低值36万
   调增 = 60 - 36 = 24万
④ 广告费：限额 = 8000×15% = 1200万；实际1300万
   调增 = 1300 - 1200 = 100万（超出部分结转以后年度）
⑤ 公益性捐赠：限额 = 1000×12% = 120万；实际150万
   调增 = 150 - 120 = 30万（超出部分结转以后3年）
⑥ 税收滞纳金5万→不得扣除，调增5万
⑦ 研发费用加计扣除：200×100% = 200万，调减200万

应纳税所得额 = 1000 - 20 - 50 + 24 + 100 + 30 + 5 - 200 = 889万元
应纳税额 = 889 × 25% = 222.25万元
```

### 税务策略规划（实务指引）

#### 核心原则

1. 合法性优先：所有筹划方案必须有明确法律依据
2. 商业实质：交易安排须具备合理商业目的
3. 成本效益：筹划收益应大于实施成本（含合规成本和风险成本）
4. 动态调整：政策变化时及时更新方案

#### 企业所得税优化方向

- 研发加计扣除（100%加计，2026年继续执行）
- 高新技术企业15%优惠税率
- 小型微利企业优惠（应纳税所得额≤300万，实际税负5%）
- 西部大开发15%优惠（至2030年）
- 海南自贸港15%优惠（封关后继续执行）
- 固定资产加速折旧/一次性扣除
- 技术转让所得减免（≤500万免征，>500万减半）

#### ETR分析框架

实际税负率(ETR) = 所得税费用 ÷ 利润总额

- ETR显著低于法定税率→分析优惠政策贡献
- ETR显著高于法定税率→排查永久性差异（罚款、超标费用等）
- 行业对标：与同行业上市公司ETR比较

### 收入确认的具体时点规则

#### 销售商品收入确认的一般条件

同时满足以下条件时确认收入实现：

1. 商品销售合同已签订，主要风险和报酬已转移给购货方
2. 企业对已售商品无继续管理权，也未实施有效控制
3. 收入金额能够可靠计量
4. 已发生或将发生的销售方成本能够可靠核算

#### 不同销售方式的收入确认时点

| 销售方式           | 收入确认时点                                               |
| ------------------ | ---------------------------------------------------------- |
| 托收承付           | 办妥托收手续时                                             |
| 预收款             | 发出商品时                                                 |
| 需安装和检验       | 购买方接受商品且安装检验完毕时（安装简单的可在发出商品时） |
| 支付手续费委托代销 | 收到代销清单时                                             |
| 分期收款           | 合同约定的收款日期                                         |
| 售后回购           | 按售价确认收入，回购作为购进处理（融资性质的确认为负债）   |
| 以旧换新           | 销售按收入确认条件确认，回收作为购进处理                   |
| 买一赠一           | 按各项商品公允价值比例分摊总销售金额                       |

#### 特殊收入确认时点

| 收入类型                   | 确认时点                                                               |
| -------------------------- | ---------------------------------------------------------------------- |
| 股息、红利等权益性投资收益 | 被投资方作出利润分配决定的日期                                         |
| 利息收入                   | 合同约定的债务人应付利息的日期                                         |
| 租金收入                   | 合同约定的承租人应付租金的日期（跨年度提前一次性支付的可分期均匀计入） |
| 特许权使用费收入           | 合同约定的应付特许权使用费的日期                                       |
| 接受捐赠收入               | 实际收到捐赠资产的日期                                                 |
| 产品分成方式               | 分得产品的日期，按公允价值确定收入额                                   |
| 持续超过12个月的劳务       | 按纳税年度内完工进度或完成工作量确认                                   |

### 不征税收入的详细条件

#### 三类法定不征税收入

| 类别                       | 内容                                                             |
| -------------------------- | ---------------------------------------------------------------- |
| 财政拨款                   | 各级人民政府对纳入预算管理的事业单位、社会团体等拨付的财政资金   |
| 行政事业性收费、政府性基金 | 依法收取并纳入财政管理的，上缴财政的当年可从收入总额中减除       |
| 国务院规定的其他不征税收入 | 由国务院财政、税务主管部门规定专项用途并经国务院批准的财政性资金 |

#### 专项用途财政性资金作为不征税收入的三个条件

企业从县级以上各级人民政府财政部门及其他部门取得的财政性资金，**同时**符合以下条件的，可作为不征税收入：

1. **企业能够提供规定资金专项用途的资金拨付文件**
2. **财政部门或其他拨付资金的政府部门对该资金有专门的资金管理办法或具体管理要求**
3. **企业对该资金以及以该资金发生的支出单独进行核算**

#### 不征税收入的后续处理

- 不征税收入用于支出所形成的**费用**，不得在计算应纳税所得额时扣除
- 不征税收入用于支出所形成的**资产**，其折旧、摊销不得在计算应纳税所得额时扣除
- **5年（60个月）内未发生支出且未缴回**的部分，应计入取得该资金**第6年**的应税收入总额
- 计入应税收入总额的财政性资金发生的支出，允许在计算应纳税所得额时扣除
- 凡未按规定进行管理的，应作为企业应税收入计入应纳税所得额

### 各项扣除的更多细节

#### 工资薪金的合理性判断标准

税务机关对工资、薪金进行合理性确认时，依据以下原则：

1. 企业制订了较为规范的员工工资、薪金制度
2. 企业所制订的工资、薪金制度符合行业及地区水平
3. 企业在一定时期所发放的工资、薪金是相对固定的，调整是有序进行的
4. 企业对实际发放的工资、薪金，已依法履行了代扣代缴个人所得税义务
5. 有关工资、薪金的安排，不以减少或逃避税款为目的

> 国有性质企业的工资、薪金不得超过政府有关部门给予的限定数额。

#### 保险费扣除

| 保险类型                       | 扣除规则                                                                               |
| ------------------------------ | -------------------------------------------------------------------------------------- |
| 基本社会保险费（五险一金）     | 按规定范围和标准缴纳的，**据实扣除**                                             |
| 补充养老保险费、补充医疗保险费 | 在国务院财政、税务主管部门规定的范围和标准内准予扣除（分别不超过工资总额**5%**的部分） |
| 特殊工种人身安全保险费         | 依照国家有关规定为特殊工种职工支付的，准予扣除                                         |
| 财产保险                       | 按规定缴纳的保险费，准予扣除                                                           |
| 商业保险（为投资者或职工支付） | **不得扣除**（符合国务院财政、税务主管部门规定可以扣除的除外）                   |

#### 借款费用扣除

**费用化 vs 资本化：**

| 情形                                                                                   | 处理方式                                               |
| -------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| 生产经营活动中发生的合理的不需要资本化的借款费用                                       | 准予在发生当期扣除                                     |
| 为购置、建造固定资产/无形资产/经过12个月以上建造才能达到预定可销售状态的存货发生的借款 | 在资产购置、建造期间应予**资本化**，计入资产成本 |
| 有关资产交付使用后发生的借款利息                                                       | 可在发生当期扣除                                       |

**利息费用扣除标准：**

| 借款类型                           | 扣除标准                                                                             |
| ---------------------------------- | ------------------------------------------------------------------------------------ |
| 非金融企业向金融企业借款           | **据实扣除**                                                                   |
| 金融企业各项存款利息、同业拆借利息 | **据实扣除**                                                                   |
| 企业经批准发行债券的利息           | **据实扣除**                                                                   |
| 非金融企业向非金融企业借款         | 不超过按照金融企业**同期同类贷款利率**计算的数额部分据实扣除，超过部分不得扣除 |

**关联方借款利息限制：**

- 债权性投资与权益性投资比例限制：**金融企业 5:1，其他企业 2:1**
- 超过比例的利息支出不得在发生当期和以后年度扣除
- 例外：能证明相关交易符合独立交易原则的，或实际税负不高于境内关联方的，准予扣除

#### 资产损失税前扣除

**实际资产损失 vs 法定资产损失：**

| 类型         | 定义                                                     | 扣除年度                           |
| ------------ | -------------------------------------------------------- | ---------------------------------- |
| 实际资产损失 | 企业在实际处置、转让资产过程中发生的合理损失             | 实际发生且会计上已作损失处理的年度 |
| 法定资产损失 | 企业虽未实际处置、转让资产，但符合规定条件计算确认的损失 | 申报年度                           |

**申报管理要求：**

- 企业向税务机关申报扣除资产损失，仅需填报《资产损失税前扣除及纳税调整明细表》
- 不再报送资产损失相关资料，相关资料由企业**留存备查**
- 未经申报的损失，不得在税前扣除

**以前年度资产损失追补扣除：**

- 实际资产损失：准予追补至该项损失发生年度扣除，追补确认期限一般**不得超过5年**
- 法定资产损失：应在申报年度扣除
- 追补扣除后多缴的企业所得税，可在追补确认年度应纳税款中抵扣，不足抵扣的向以后年度递延

### 企业重组的税务处理

#### 企业重组的类型

| 类型             | 定义                                                       |
| ---------------- | ---------------------------------------------------------- |
| 企业法律形式改变 | 注册名称、住所及组织形式等的简单改变                       |
| 债务重组         | 债务人发生财务困难时，债权人按协议或法院裁定对债务作出让步 |
| 股权收购         | 收购企业购买被收购企业的股权，以实现控制                   |
| 资产收购         | 受让企业购买转让企业实质经营性资产                         |
| 合并             | 被合并企业将全部资产和负债转让给合并企业                   |
| 分立             | 被分立企业将部分或全部资产分离转让给分立企业               |

#### 一般性税务处理 vs 特殊性税务处理

| 对比项        | 一般性税务处理                 | 特殊性税务处理                               |
| ------------- | ------------------------------ | -------------------------------------------- |
| 计税基础      | 以**公允价值**为基础确定 | 以**原有计税基础**确定（股权支付部分） |
| 所得/损失确认 | 交易发生时确认转让所得或损失   | 股权支付部分暂不确认所得或损失               |
| 亏损结转      | 被合并/分立企业亏损不得结转    | 可按限额弥补被合并企业亏损                   |
| 税收优惠      | 不承继                         | 符合条件的可承继                             |

#### 适用特殊性税务处理的5个条件（须同时满足）

1. 具有合理的商业目的，且不以减少、免除或推迟缴纳税款为主要目的
2. 被收购、合并或分立部分的资产或股权比例符合规定（≥50%）
3. 企业重组后的连续**12个月**内不改变重组资产原来的实质性经营活动
4. 股权支付金额不低于交易支付总额的**85%**
5. 取得股权支付的原主要股东，在重组后连续**12个月**内不得转让所取得的股权

#### 特殊性税务处理的关键比例

| 重组类型 | 资产/股权比例要求                           | 股权支付比例要求              |
| -------- | ------------------------------------------- | ----------------------------- |
| 股权收购 | 收购股权≥被收购企业全部股权的**50%** | ≥交易支付总额的**85%** |
| 资产收购 | 收购资产≥转让企业全部资产的**50%**   | ≥交易支付总额的**85%** |
| 合并     | —                                          | ≥交易支付总额的**85%** |
| 分立     | —                                          | ≥交易支付总额的**85%** |

#### 非股权支付对应的资产转让所得计算

```
非股权支付对应的资产转让所得 = (被转让资产公允价值 - 被转让资产计税基础) × 非股权支付金额 ÷ 被转让资产公允价值
```

### 居民企业核定征收

#### 核定征收改为查账征收后有关资产的税务处理

1. 企业能够提供资产购置发票的，以发票载明金额为计税基础
2. 不能提供发票的，可凭购置资产的合同（协议）、资金支付证明、会计核算资料等记载金额作为计税基础
3. 核定征税期间投入使用的资产，改为查账征税后，按税法规定的折旧/摊销年限，扣除该资产已投入使用年限后，就**剩余年限**继续计提折旧/摊销额并在税前扣除

> 注：企业不能提供完整、准确的收入及成本、费用凭证，不能正确计算应纳税所得额的，由税务机关核定其应纳税所得额。

### 房地产开发企业企业所得税特殊规定

- 房地产开发企业采取预售方式销售自行开发的房地产项目，以当期取得的预收款和3%的预征率计算应预缴增值税
- 应预缴税款 = 当期取得的预收款 ÷ (1 + 适用税率或征收率) × 3%
- 应当在收到预收款的次月纳税申报期内预缴税款

> 注：房地产开发企业预售收入的预计毛利率、完工后的税务处理等具体企业所得税规定，在本教材中未作详细展开。

### 高新技术企业认定条件（8个条件）

国家需要重点扶持的高新技术企业减按**15%**的税率征收企业所得税，须拥有核心自主知识产权，并**同时**符合以下条件：

| 2    | 知识产权     | 通过自主研发、受让、受赠、并购等方式获得对主要产品（服务）发挥核心支持作用的知识产权所有权                                                                                                       |
| 3    | 技术领域     | 主要产品（服务）的核心技术属于《国家重点支持的高新技术领域》规定范围                                                                                                                             |
| 4    | 科技人员比例 | 从事研发和相关技术创新活动的科技人员占当年职工总数≥**10%**                                                                                                                                |

**亏损弥补延长：** 具备高新技术企业或科技型中小企业资格的企业，其具备资格年度之前5个年度发生的尚未弥补完的亏损，最长结转年限由5年延长至**10年**。

### 征收管理

**纳税年度**：公历1月1日至12月31日

**预缴与汇算清缴**：

- 按月或按季预缴（自月份或季度终了之日起15日内）
- 年度终了之日起5个月内汇算清缴，结清应缴应退税款

**综合所得应纳税额计算公式：**

```
应纳税额 = (全年收入额 - 60,000元 - 专项扣除 - 专项附加扣除 - 依法确定的其他扣除) × 适用税率 - 速算扣除数
```

**收入额确定规则：**

- 工资薪金：全额计入
- 劳务报酬：收入 × 80%
- 稿酬：收入 × 80% × 70%（即收入 × 56%）
- 特许权使用费：收入 × 80%

### 综合所得年度汇算计算示例

**示例**：张某2025年度收入情况：

- 工资薪金：月薪25,000元（全年300,000元）
- 劳务报酬：一次性取得50,000元
- 稿酬：一次性取得20,000元
- 专项扣除（三险一金）：每月4,500元（全年54,000元）
- 专项附加扣除：子女教育2,000元/月 + 住房贷款利息1,000元/月 + 赡养老人3,000元/月 = 6,000元/月（全年72,000元）
- 全年一次性奖金60,000元（选择单独计税）

```
一、综合所得年度汇算：
收入额 = 工资薪金300,000 + 劳务报酬50,000×80% + 稿酬20,000×80%×70%
       = 300,000 + 40,000 + 11,200 = 351,200元

应纳税所得额 = 351,200 - 60,000(基本减除) - 54,000(专项扣除) - 72,000(专项附加扣除)
             = 165,200元

应纳税额 = 165,200 × 20% - 16,920 = 16,120元

二、全年一次性奖金单独计税：
60,000 ÷ 12 = 5,000元 → 适用10%税率，速算扣除数210元
应纳税额 = 60,000 × 10% - 210 = 5,790元

三、全年个人所得税合计 = 16,120 + 5,790 = 21,910元
```

**对比**：若奖金并入综合所得：

```
应纳税所得额 = 165,200 + 60,000 = 225,200元
应纳税额 = 225,200 × 20% - 16,920 = 28,120元
差额 = 28,120 - 21,910 = 6,210元（单独计税更优）
```

### 十、非居民个人计税方法

非居民个人取得工资薪金、劳务报酬、稿酬、特许权使用费所得，按月/按次计税，适用上述月度税率表，不办理汇算清缴。

- 工资薪金：每月收入 - 5,000元 = 应纳税所得额
- 劳务报酬/稿酬/特许权使用费：每次收入额为应纳税所得额（收入额计算规则同居民个人）

### 十一、财产转让所得计算

```
应纳税额 = (收入总额 - 财产原值 - 合理费用) × 20%
```

### 股权转让所得要点：

- 纳税人：股权转让方；扣缴义务人：受让方
- 股权转让收入 = 现金 + 实物 + 有价证券 + 其他经济利益（含违约金、补偿金等）
- 股权原值确认：现金出资按实际支付价款+合理税费；非货币资产按税务机关认可价格+合理税费
- 多次取得同一企业股权转让时，采用"加权平均法"确定原值

### 十二、个人所得税主要免税项目

1. 省级以上政府、国务院部委、军以上单位及外国组织/国际组织颁发的科教文卫体环保奖金
2. 国债和国家发行的金融债券利息
3. 按国家统一规定发给的政府特殊津贴、院士津贴
4. 福利费、抚恤金、救济金
5. 保险赔款
6. 军人转业费、复员费、退役金
7. 安家费、退职费、基本养老金、退休费、离休费
8. 个人转让自用达5年以上且唯一家庭生活用房所得
9. 个人转让上市公司股票所得（暂免）
10. 上市公司股息红利：持股超1年免税；1个月~1年减按50%；1个月内全额征税
11. 按规定缴付的住房公积金、医疗保险、养老保险、失业保险（免税）
12. 自2008.10.9起，储蓄存款利息暂免征收

14. 解除劳动合同一次性补偿收入：当地上年职工平均工资3倍以内免税

### 十三、无住所个人特殊规则（六年规则）

- 无住所居民个人在境内居住累计满183天的年度连续不满6年：境外工作期间且由境外单位支付的部分免税
- 连续满6年（且无任何一年单次离境超30天）：全球所得均须纳税
- 此前6年起始年度自2019年开始计算

### 十四、个人养老金税收政策（2024.1.1起全国实施）

| 环节 | 税收政策                                           |
| ---- | -------------------------------------------------- |
| 缴费 | 按12,000元/年限额在综合所得或经营所得中据实扣除    |
| 投资 | 计入个人养老金资金账户的投资收益暂不征收个人所得税 |
| 领取 | 单独按3%税率计算缴纳，计入"工资、薪金所得"项目     |

### 十五、境外所得税额抵免

```
来源于一国(地区)所得的抵免限额 = 综合所得抵免限额 + 经营所得抵免限额 + 其他分类所得抵免限额

综合所得抵免限额 = 境内外全部综合所得应纳税额 × (来源于该国综合所得收入额 ÷ 境内外综合所得收入额合计)
```

- 实际缴纳 < 抵免限额：按实际缴纳额抵免，补缴差额
- 实际缴纳 > 抵免限额：超过部分可在以后5个纳税年度结转抵免

### 十六、股权激励个人所得税

### （一）上市公司股权激励计税方法

居民个人在纳税年度内取得股票期权、股票增值权所得和限制性股票所得的，在**2027年12月31日前**：

- 不并入当年综合所得
- 全额单独适用综合所得税率表（7级超额累进税率3%~45%）

**计算公式：**

```
应纳税额 = 股权激励收入 × 适用税率 - 速算扣除数
```

> 居民个人一个纳税年度内取得两次以上股权激励的，应合并计算纳税。

### （二）股票期权所得的确认

| 环节           | 税务处理                                                                     |
| -------------- | ---------------------------------------------------------------------------- |
| 授予日         | 一般不征税（可公开交易的股票期权除外）                                       |
| 行权日         | 按"工资、薪金所得"计税，应纳税所得额 = (行权日股票市价 - 施权价) × 股票数量 |
| 行权后转让     | 按"财产转让所得"计税                                                         |
| 行权前转让期权 | 以转让净收入作为工资、薪金所得征税                                           |

### （三）限制性股票所得的确认

应纳税所得额 = (股票登记日股票市价 + 本批次解禁股票当日市价) ÷ 2 × 本批次解禁股票份数 - 被激励对象实际支付的资金总额 × (本批次解禁股票份数 ÷ 被激励对象获取的限制性股票总份数)

### （四）非上市公司股权激励递延纳税

- 符合条件的非上市公司授予员工的股票期权、股权期权、限制性股票和股权奖励，可实行**递延纳税**
- 员工在取得股权激励时暂不纳税，递延至**转让该股权时**纳税
- 转让时按"财产转让所得"项目，适用**20%**税率
- 股票（权）期权取得成本按**行权价**确定，限制性股票按**实际出资额**确定，股权奖励取得成本为**零**

### （五）上市公司股权激励延长纳税期限

上市公司授予个人的股票期权、限制性股票和股权奖励，经向主管税务机关备案，个人可自行权/解禁/取得之日起，在不超过**12个月**的期限内缴纳个人所得税。

### 十七、限售股转让个人所得税

- 个人转让上市公司限售股取得的所得，按"财产转让所得"适用20%税率
- 应纳税所得额 = 限售股转让收入 - 限售股原值 - 合理税费
- 纳税人未能提供完整、真实的限售股原值凭证的，主管税务机关一律按限售股转让收入的**15%**核定为限售股原值和合理税费
- 个人转让境内公开发行和转让市场上市公司股票（非限售股）取得的所得，**暂免**征收个人所得税

### 十八、非货币性资产投资个人所得税

### （一）基本规定

- 个人以非货币性资产投资 = 转让非货币性资产 + 投资同时发生
- 按"财产转让所得"项目计税
- 应纳税所得额 = 非货币性资产评估后的公允价值 - 该资产原值 - 合理税费

### （二）分期缴纳政策

- 纳税人一次性缴税有困难的，可合理确定分期缴纳计划并报主管税务机关备案
- 自发生应税行为之日起不超过**5个公历年度内（含）**分期缴纳
- 取得现金补价的，现金部分应**优先用于缴税**
- 分期缴税期间转让股权并取得现金收入的，该现金收入应优先用于缴纳尚未缴清的税款

### （三）非货币性资产范围

包括股权、不动产、技术发明成果以及其他形式的非货币性资产。投资行为包括出资设立新企业、参与增资扩股、定向增发股票、股权置换、重组改制等。

### 十九、个人转让住房的个人所得税

### （一）计税方法

**据实征收：**
应纳税所得额 = 转让收入 - 房屋原值 - 转让过程中缴纳的税金 - 合理费用

**核定征收：**
纳税人未提供完整、准确的房屋原值凭证的，按住房转让收入的**1%~3%**核定征收（具体比例由省级税务局确定）。

### （二）免税规定

- 个人转让自用**5年以上**且是家庭**唯一**生活用房取得的所得，**免征**个人所得税
- 通过离婚析产方式分割房屋产权，办理房屋产权过户手续的，**不征收**个人所得税

### （三）房屋原值的确定

| 房屋类型     | 原值确定方法                                                                                               |
| ------------ | ---------------------------------------------------------------------------------------------------------- |
| 商品房       | 购置时实际支付的房价款及缴纳的相关税费                                                                     |
| 自建住房     | 实际发生的建造费用及建造和取得产权时缴纳的相关税费                                                         |
| 经济适用房   | 原购房人实际支付的房价款及相关税费，加上按规定缴纳的土地出让金                                             |
| 已购公有住房 | 标准面积按当地经济适用房价格计算的房价款 + 超标准面积实际支付的房价款 + 向财政部门缴纳的所得收益及相关税费 |

### （四）合理费用

- 住房装修费用：已购公有住房、经济适用房最高扣除限额为房屋原值的**15%**；商品房及其他住房最高扣除限额为房屋原值的**10%**
- 住房贷款利息：凭贷款银行出具的有效证明据实扣除
- 手续费、公证费等：凭有关部门出具的有效证明据实扣除

### 二十、财产租赁所得的详细计算

### （一）费用扣除标准

| 每次（月）收入 | 扣除方式                |
| -------------- | ----------------------- |
| ≤4000元       | 定额减除**800元** |
| >4000元        | 定率减除**20%**   |

### （二）扣除项目及顺序

依次扣除：

1. 财产租赁过程中缴纳的税金和教育费附加
2. 由纳税人负担的该出租财产实际开支的修缮费用（每次**800元**为限，一次扣不完的准予下次继续扣除）
3. 税法规定的费用扣除标准（800元或20%）

### （三）计算公式

**每次（月）收入≤4000元：**

```
应纳税所得额 = 每次收入 - 准予扣除项目 - 修缮费用(800元为限) - 800元
```

**每次（月）收入>4000元：**

```
应纳税所得额 = [每次收入 - 准予扣除项目 - 修缮费用(800元为限)] × (1-20%)
```

### （四）适用税率

- 一般财产租赁所得：**20%**
- 个人按市场价格出租居民住房：暂减按**10%**

### （五）房屋转租

个人将承租房屋转租取得的租金收入，按"财产租赁所得"计税。扣除次序调整为：

1. 财产租赁过程中缴纳的税费
2. **向出租方支付的租金**（凭房屋租赁合同和合法支付凭据）
3. 由纳税人负担的修缮费用
4. 税法规定的费用扣除标准

### 二十一、劳务报酬预扣预缴的具体计算

### （一）居民个人劳务报酬所得预扣预缴

**减除费用：**

- 每次收入≤4000元：减除费用按**800元**计算
- 每次收入>4000元：减除费用按收入的**20%**计算

**预扣预缴应纳税所得额** = 每次收入额（收入减除20%费用后）

**适用三级预扣率表：**

| 预扣预缴应纳税所得额 | 预扣率(%) | 速算扣除数(元) |
| -------------------- | --------- | -------------- |
| 不超过20000元        | 20        | 0              |
| 超过20000元至50000元 | 30        | 2000           |
| 超过50000元          | 40        | 7000           |

**计算公式：**

```
劳务报酬所得应预扣预缴税额 = 预扣预缴应纳税所得额 × 预扣率 - 速算扣除数
```

### （二）计算示例

某人一次取得劳务报酬30000元：

- 收入额 = 30000 × (1-20%) = 24000元
- 应预扣预缴税额 = 24000 × 30% - 2000 = 5200元

> 注：年度汇算清缴时，劳务报酬所得以收入减除20%费用后的余额为收入额，并入综合所得按年计算。

### 二十二、公益性捐赠扣除

### （一）限额扣除（30%）

- 个人将其所得通过中国境内的公益性社会组织、国家机关向教育、扶贫、济困等公益慈善事业的捐赠
- 捐赠额未超过纳税人申报的**应纳税所得额30%**的部分，可以从其应纳税所得额中扣除
- "应纳税所得额"是指计算扣除捐赠额**之前**的应纳税所得额

### （二）全额扣除的情形

国务院规定对公益慈善事业捐赠实行全额税前扣除的，从其规定。例如：

- 个人捐赠住房作为公租房，符合税法规定的公益性捐赠支出

### （三）扣除规则

**综合所得：**

- 居民个人在预扣预缴工资、薪金所得税时，可选择在预扣预缴时扣除，也可选择在汇算清缴时扣除
- 取得劳务报酬所得、稿酬所得、特许权使用费所得的，预扣预缴时不扣除，统一在汇算清缴时扣除

**分类所得：**

- 居民个人发生的公益捐赠支出，可在捐赠当月取得的分类所得中扣除

**全年一次性奖金、股权激励等单独计税所得：**

- 比照分类所得的扣除规定处理

### 二十三、境外所得税额抵免的详细计算

### （一）抵免原则

- 居民个人从中国境外取得的所得，可以从其应纳税额中抵免已在境外缴纳的个人所得税税额
- 抵免额不得超过该纳税人境外所得依照税法规定计算的**抵免限额**
- 来源于一个国家（地区）的综合所得抵免限额、经营所得抵免限额以及其他所得抵免限额之和，为来源于该国家（地区）所得的抵免限额

### （二）分项计算规则

| 所得类型                                                       | 计算方式                                         |
| -------------------------------------------------------------- | ------------------------------------------------ |
| 综合所得                                                       | 应当与境内综合所得**合并**计算应纳税额     |
| 经营所得                                                       | 应当与境内经营所得**合并**计算应纳税额     |
| 其他分类所得（利息、股息、红利、财产租赁、财产转让、偶然所得） | **不与境内所得合并**，分别单独计算应纳税额 |

### （三）抵免限额的计算

- 按国别（地区）分别计算
- 实际已缴税额 < 抵免限额：应在中国缴纳差额部分
- 实际已缴税额 > 抵免限额：超过部分不得在本纳税年度抵免，但可在以后纳税年度来源于该国家（地区）所得的抵免限额余额中**补扣**
- 补扣期限最长不得超过**5年**

### （四）境外经营所得亏损处理

- 境外经营所得的亏损，**不得**抵减其境内或他国（地区）的应纳税所得额
- 但可以用来源于**同一国家（地区）**以后年度的经营所得按中国税法规定弥补

### 二十四、个人所得税反避税条款

### （一）税务机关有权纳税调整的三种情形

| 情形                | 具体内容                                                                                                                                               |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 独立交易原则        | 个人与其关联方之间的业务往来不符合独立交易原则而减少本人或其关联方应纳税额，且无正当理由                                                               |
| 受控外国企业（CFC） | 居民个人控制的，或居民个人和居民企业共同控制的设立在实际税负明显偏低的国家（地区）的企业，无合理经营需要，对应当归属于居民个人的利润不作分配或减少分配 |
| 不合理商业安排      | 个人实施其他不具有合理商业目的的安排而获取不当税收利益                                                                                                 |

### （二）补税及加征利息

- 税务机关依照上述规定作出纳税调整，需要补征税款的，应当补征税款并**依法加收利息**
- 利息按照税款所属纳税申报期最后一日中国人民银行公布的与补税期间同期的**人民币贷款基准利率**计算
- 自税款纳税申报期满次日起至补缴税款期限届满之日止**按日加收**
- 纳税人在补缴税款期限届满前补缴税款的，利息加收至补缴税款之日

## 附：国家税务总局法规库查询入口

> 回答税务问题时，如涉及具体政策有效期、最新文件号或政策是否仍有效，应联网查询以下页面确认。

| 分类 | 网址 |
|------|------|
| 工作通知 | https://fgk.chinatax.gov.cn/zcfgk/c102424/listflfg.html |
| 税收规范性文件（公告） | https://fgk.chinatax.gov.cn/zcfgk/c100012/listflfg.html |
| 税务部门规章 | https://fgk.chinatax.gov.cn/zcfgk/c100011/list_guizhang.html |
| 财政部 税务总局公告 | https://fgk.chinatax.gov.cn/zcfgk/c100013/listflfg.html |
| 国务院文件 | https://fgk.chinatax.gov.cn/zcfgk/c102416/listflfg.html |
| 税收协定 | https://fgk.chinatax.gov.cn/zcfgk/c102440/listflfg.html |
| 法律（税收相关） | https://fgk.chinatax.gov.cn/zcfgk/c100010/listflfg_fg.html |
| 行政法规 | https://fgk.chinatax.gov.cn/zcfgk/c100009/listflfg_fg.html |

**使用方法**：
- 查最新公告/政策变动 → 查"税收规范性文件"和"财政部 税务总局公告"
- 查税收协定/国际税收 → 查"税收协定"
- 查法律原文（增值税法、企业所得税法等）→ 查"法律"
- 查实施条例 → 查"行政法规"
