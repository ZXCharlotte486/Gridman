# 会计监管风险提示与监管报告

> 定位：上市公司/拟上市公司会计处理的监管红线与典型错误
> 来源：中国证券监督管理委员会（2012-2025）
> 覆盖：风险提示1-9号、年报监管报告（2024）、监管规则适用指引、准则案例解析
> 性质：判断

## 第1号——政府补助（2012）

**文号**：证监办发〔2012〕22号

### 会计处理常见问题

1. 将政府有偿投入（政府作为所有者投入的资本）作为政府补助
2. 控股股东/关联方通过政府渠道转移资产伪装为政府补助
3. 与资产相关的补助误分类为与收益相关
4. 未达确认条件即确认补助收入（未满足附条件/未收到金额/费用尚未发生）
5. 已达确认条件的补助推迟确认
6. 将不符合经常性损益的政府补助披露为经常性损益

### 审计关注要点

- 分析舞弊动机（避免ST/退市、满足融资条件、股权激励行权、重组承诺等8项）
- 检查政府补助文件原件 + 公司已获得资产的凭据
- 核对补助资产来源单位与政府文件的一致性
- 核对分类恰当性（与收益相关 vs 与资产相关）
- 核对列报与披露的准确性、完整性

## 第2号——通过未披露关联方实施的舞弊风险（2012）

**文号**：证监办发〔2012〕22号

### 舞弊常见形式

1. 以显失公允价格与未披露关联方交易
2. 利用未披露关联方资金循环虚构交易（控制客户+供应商，伪造合同+收发货+资金流）
3. 未披露关联方分担公司成本费用
4. 接受未披露关联方捐赠（直接捐赠/豁免欠款计入损益）
5. 利用未披露关联方占用公司资金（预付账款长期不结算）

### 未披露关联方关系的9大特征

1. 曾经与公司存在关联关系
2. 注册/办公地址相同或接近
3. 网站/IP/邮箱域名相同或接近
4. 名称与公司或集团成员相似
5. 主要控制人/关键管理人员姓名结构相近
6. 交易与其经营范围不相关
7. 互联网难以检索到相关资料
8. 长期拖欠款项但仍继续交易
9. 当年新增的重要客户或供应商

### 未披露关联方交易的13大特征

1. 金额大，带来大额利润
2. 接近资产负债表日，频次少
3. 价格/交付/付款条件与正常客户明显不同
4. 现金交易或多方抵销结算
5. 付款人与合同/发票客户名称不一致
6. 与自然人的大额交易
7. 与同一客户同时发生销售和采购
8. 交易对象与对方经营范围明显不符
9. 交易规模与对方业务规模明显不符
10. 合同条款不符合商业惯例
11. 实际履行与合同约定明显不符
12. 款项长期以债权债务形式存在
13. 商业理由明显不充分

### 审计应对

- 分析舞弊动机（同第1号8项）
- 了解并评价关联方识别的内部控制
- 分析程序：产能/水电能耗/劳动生产率等非财务信息与财务数据关系
- 关注重大异常交易是否具备未披露关联方交易特征
- 核实手段：调查交易对方背景、询问基层员工、实地走访、查询互联网/商业数据库、聘请信用调查机构

## 第3号——审计项目复核（2012）

**文号**：证监办发〔2012〕22号

### 复核层级

1. **项目组内部复核**
   - 外勤实施阶段：现场负责人逐张复核助理人员底稿
   - 外勤临近结束：复核重要账项审计、重要程序执行、建议调整事项
   - 审计意见形成阶段：复核重大会计审计问题、重大调整事项

2. **项目质量控制复核**
   - 独立于项目组的合伙人实施
   - 对重大判断和审计结论进行客观评价
   - 复核财务报表和拟出具的审计报告

### 常见问题

- 制度上未明确复核层级和内容
- 复核人员薪酬与项目组经济利益一致（独立性问题）
- 重报告复核、轻审计证据复核
- 无复核记录或记录不完整
- 复核意见未及时回复

## 第4号——首次公开发行股票公司审计（2012）

**文号**：证监办发〔2012〕89号

### IPO审计九大关注领域

1. **质量控制**：承接时了解上市动机/高风险领域/管理层诚信；关注更换注册会计师原因
2. **内部控制**：评价内控缺陷严重程度；实质性测试发现的错报是否与内控缺陷相关
3. **财务与非财务信息印证**：产能/原材料/能耗与产量匹配；经营模式与收入/成本/应收匹配
4. **盈利异常增长**：多维度分析毛利率/期间费用率/周转率/产能利用率；关注申报期前1-2年数据
5. **异常交易**：偶发性+无实物形态+价格偏离→保持职业怀疑；考虑是否为非经常性损益
6. **关联方认定**：调查申报期内重大异常交易对手背景；关注关联方注销/非关联化动机
7. **收入确认与成本核算**：不同销售模式（经销商/直销/完工百分比）的收入确认；毛利率与同行业对比
8. **主要客户和供应商**：核查前十名客户/供应商工商资料；实地走访或电话访谈
9. **资产盘点与货币资金**：异地存货监盘；大额现金收支真实性；银行对账单核对

## 第5号——上市公司股权交易资产评估（2013）

### 评估全流程关注要点

| 环节 | 核心要求 |
|------|----------|
| 业务承接 | 不以预先设定价格作为评估结论；独立性核查；内幕信息管理 |
| 评估对象调查 | 股权≠资产负债简单组合；关注章程特殊约定；关注权属瑕疵 |
| 价值类型 | 市场价值不应体现收购后协同效应 |
| 评估方法 | 持续经营下原则上采用两种以上方法；不得只选更接近预设价格的方法 |
| 资产基础法 | 关注账外无形资产；关注经济性贬值；引用其他报告需判断可靠性 |
| 收益法 | 盈利预测需有依据；关注周期性波动；关联方定价公允性；税收优惠可持续性；折现率匹配风险 |
| 市场法 | 可比企业选择；交易背景分析；流动性和控制权溢价/折价 |
| 报告披露 | 产权瑕疵、抵押质押、基准日后重大事项、与审计报告对应关系 |

## 第8号——商誉减值（2018）

### 减值迹象（7项）

1. 现金流/经营利润持续恶化或明显低于形成商誉时预期（特别是未实现业绩承诺）
2. 行业产能过剩、产业政策/市场竞争明显不利变化
3. 技术壁垒低或技术快速进步，盈利难以维持
4. 核心团队明显不利变化且短期难恢复
5. 相关资质（行政许可/特许经营/特定合同）市场惯例变化
6. 市场投资报酬率明显提高且短期不会下降
7. 经营所处国家/地区风险突出（外汇管制/恶性通胀/宏观恶化）

### 资产组认定规则

- 一个会计核算主体 ≠ 一个资产组
- 应充分考虑企业合并的协同效应
- 按公允价值比例分摊商誉账面价值
- 归属于少数股东的商誉需调整计入
- 重组导致构成变化时需重新认定和分摊
- 无合理理由不得随意变更资产组

### 减值测试关键参数

- 未来现金流：以当前状况为基础、税前口径、预测期原则上≤5年
- 折现率：与宏观/行业/地域/特定风险匹配、税前口径
- 预测期增长率/稳定期增长率：需有可靠数据来源
- 若前期预测与实际存在重大偏差→本期需调整预测思路

### 业绩承诺与商誉减值的关系

- **不得以有业绩补偿承诺为由不进行减值测试**
- 应合理区分并分别处理商誉减值事项和业绩补偿事项

### 信息披露要求

- 资产组构成、账面金额、确定方法
- 减值测试过程与方法、重要假设、关键参数及确定依据
- 业绩承诺完成情况及对减值测试的影响
- 商誉金额重大时，无论是否减值均应详细披露

## 第9号——上市公司控股股东资金占用及其审计（2019）

### 资金占用两大模式

#### 余额模式（虚增货币资金/隐瞒受限）

1. 虚假银行对账单/回单直接虚增货币资金
2. 虚构定期存款/保证金账户间转移掩盖挪用
3. 集团资金管理协议/资金池归集并挪用
4. 以定期存款/理财产品为控股股东融资提供质押担保但未披露受限

#### 发生额模式（通过交易转移资金）

1. 直接/间接拆借资金
2. 无商业实质购销业务支付采购款/开票贴现
3. 应收账款回款被代收占用/保理资金提供给控股股东
4. 汇票交易/虚构背书掩盖占用
5. 代垫付/承担支出/偿还债务
6. 投资信托/私募/资管计划但实际被占用
7. 向控股股东控制主体投资转移资金
8. 通过关联财务公司/小贷/保理公司提供资金
9. 支付工程款/虚构在建工程
10. 以上市公司名义对外借款
11. 其他特殊模式（如贵金属委托加工后变现）

### 审计核心程序

**货币资金函证**：
- 发生额大但余额小的账户必须函证
- 亲自到人民银行查询《已开立银行结算账户清单》
- 函证集团资金管理协议/资金池安排
- 对可靠性存疑的回函验证真实性

**货币资金检查**：
- 核对银行对账单大额异常交易（相同金额一收一付）
- 必要时现场观察网上银行操作
- 定期存款凭据监盘
- "存贷双高"情形需对比利息收支/资金使用计划/同行业情况

**担保审计**：
- 获取企业信用报告核对质押/抵押/担保信息
- 关注企业信用报告的局限性
- 通过中国裁判文书网查询法院公告

### 内部控制关注（10个维度）

有效治理、独立性管理、资金及筹资管理、担保业务、关联交易、采购及付款管理、销售及收款管理、票据管理、投资管理、工程管理

## 上市公司2024年年度财务报告会计监管报告（证监会，2025.08.15）

> A股5,413家上市公司，192家被出具非标准审计意见（保留72家、无法表示意见21家）

### 一、收入相关问题

**1. 时段法确认收入——履约进度不恰当**
- 土地整理开发业务：不能以"无法合理确定履约进度"为由不确认收入也不结转成本
- 规则：履约进度不能合理确定时，已发生成本预计能得到补偿的 → 按已发生成本金额确认收入

**2. 点价模式——不是可变对价**
- 金属商品点价结算：合同定价挂钩商品指数价格变动 ≠ 可变对价
- 正确处理：视为嵌入衍生工具。控制权转移时以指数价格确认收入；之后指数变动计入公允价值变动损益

**3. 销售返利——实物返利是重大权利**
- 经销商达到采购量后获得实物返利 → 构成"重大权利"（额外购买选择权）
- 正确处理：作为单项履约义务分摊交易价格；计提的未交付返利确认为合同负债（非预计负债）

**4. 合同履约成本——产出法下超支不能资本化**
- 产出法下实际成本超过按产出法确定的成本 → 超出部分不增加未来履约资源 → 发生时计入当期损益
- 投入法下期末仍有大额合同履约成本 → 应及时摊销计入成本

### 二、长期股权投资和企业合并

**1. 合并范围判断错误（高频）**
- 持股65%但章程约定重大事项需2/3通过 → 投融资/关联交易/运营定价属于日常经营活动，不是保护性权利 → 可能不构成控制
- 可分割部分：中国法律下同一法律主体通常不满足可分割条件
- 不当调整合并范围：不能仅以收购方逾期未付款为由重新纳入；不能仅以其他股东未实缴出资为由纳入

**2. 一揽子交易分步合并——商誉确认错误**
- 一揽子交易下：应在购买日一次性确认商誉（合并成本含已付+未付对价 vs 购买日可辨认净资产公允价值份额）
- 不得分步计算确认商誉

**3. 或有对价后续计量**
- 非同一控制合并中或有对价形成金融负债 → 以FVTPL后续计量
- 业绩承诺期内每个资产负债表日都要重新估计公允价值，不能等承诺期结束一次性确认

**4. 少数股东业绩承诺——重分类时点**
- 业绩不达标当期资产负债表日（非签订回购协议日）→ 确定可收回股份数量 → 按当日股价重分类为权益工具

**5. 内部交易抵销——不得转回减值**
- 子公司间按评估价出售工程设备 → 合并报表抵销未实现内部损益 + 恢复原计提的减值准备

**6. 不丧失控制权处置——少数股东权益调整**
- 应按比例把"包含商誉在内的"归属母公司权益调整至少数股东权益（非仅子公司净资产）

### 三、金融工具

**1. 预期信用损失——组合划分**
- 有抵押物 vs 无抵押物的应收账款不能划入同一组合
- 财务担保合同：未按保险合同处理的 → 适用金融工具准则计提ECL（非或有事项准则）

**2. 金融负债 vs 权益工具**
- 少数股东增资+约定保底收益率+回购义务 → 不可无条件避免交付现金 → 金融负债
- 应将回购义务确认为金融负债，按约定收益率计提利息费用

**3. 购买少数股东权益——金融负债确认时点**
- 购买协议经股东大会批准生效时 → 确认金融负债 + 终止确认对应少数股东权益

**4. 存货采购延迟定价条款——嵌入衍生工具**
- 取得存货控制权后的延迟定价条款 → 嵌入衍生工具（与应付款项主合同不紧密相关）
- 应拆分单独核算或整体指定FVTPL，后续变动计入损益（非存货成本）

**5. 其他权益工具投资不能重分类为持有待售**
- 单项金融资产处置适用金融工具准则
- FVOCI权益工具终止确认时：累计利得损失转入留存收益（非当期损益）

### 四、资产减值

**1. 存货跌价——定制化产品**
- 已交付未验收的发出商品：以合同价格为基础计提（非库龄法）
- 超期未验收的定制化产品：分析原因，充分计提

**2. 预付账款 vs 其他应收款**
- 合同未解除、尚未取得收取现金的合同权利 → 继续作为预付账款（适用资产减值准则）
- 不能简单因提起诉讼就转为金融资产

**3. 可收回金额——不应使用重置成本法**
- 重置成本 ≠ 可收回金额（重置成本是重新取得资产的成本，非未来经济利益）

**4. 在建工程减值——不以子公司净资产为零为限**
- 应单独估计在建工程可收回金额并计提减值

### 五、其他确认与计量

**1. 定制化产品研发支出**
- 研发成果仅可用于该合同 → 合同履约成本（收入准则）→ 计入营业成本
- 能控制研发成果+预期带来经济利益 → 无形资产准则资本化

**2. 固定资产修复支出**
- 高速公路大额修复 → 固定资产后续支出（非营业外支出）
- 符合确认条件 → 资本化；不符合 → 当期损益

**3. 解除租赁违约金**
- 合同约定终止选择权 → 重新计量租赁负债+调整使用权资产
- 合同未约定终止选择权 → 租赁变更处理，差额计入资产处置损益（非营业外支出）

**4. 转让净资产为负子公司——预计负债**
- 未实缴出资的子公司股权转让 → 转让人对受让人未缴出资承担补充责任
- 应在转让时确认预计负债，冲减投资收益

**5. 股份支付递延所得税**
- 等待期内每个资产负债表日估计预计行权数量×当日股价 → 确认递延所得税资产
- 员工放弃行权 → 冲减前期已确认的递延所得税资产

**6. 会计估计变更 vs 前期差错**
- 前期未合理使用已存在的可靠信息（如未估计直接人工/材料）→ 前期差错（非估计变更）

### 六、列报与披露

- 重要性标准：应按科目/交易分别披露，不能统一用一个金额
- 稀释每股收益：不能高于基本每股收益（反稀释不考虑）
- 报告分部：收入占比≥10%的经营分部应单独披露

### 七、非经常性损益

- 固定资产一次性加计扣除/进项税加计抵减 → 与正常经营密切相关 → **不是**非经常性损益
- 非金融企业持有金融资产的公允价值变动 → **是**非经常性损益
- 非房企投资性房地产公允价值变动 → **是**非经常性损益

## 监管规则适用指引——会计类第1号（2020）

> 证监会对准则实务争议问题的权威解答，共11个专题

### 1-1 特殊股权投资的确认与分类

**附回售条款的股权投资**：

| 情形 | 持股 | 影响力 | 投资方处理 |
|------|------|--------|-----------|
| 情形1 | 3% | 无重大影响 | 金融工具准则 → FVTPL |
| 情形2 | 30% | 有重大影响 | 判断风险报酬是否与普通股明显不同：是→FVTPL；否→长期股权投资+分拆回售权为嵌入衍生工具 |

**认缴制下未出资的股权投资**：
- 合同明确约定认缴时间和金额+按认缴比例享有权利 → 确认金融负债+相应资产
- 合同无明确约定 → 未来出资承诺，不确认

### 1-2 重大影响的判断

- 核心：是否有"参与决策的权力"（非"正在行使的权力"）
- 有权委派董事 → 一般认为有重大影响（除非有明确证据不能参与）
- 持股比例等未变化 → 不应在不同期间作出不同判断

### 1-3 权益法特殊事项

**联营企业发生同一控制下合并**：
- 投资方不调整比较信息
- 以合并日为界分段处理：合并日前按原比例确认损益 → 计算稀释影响计入资本公积 → 合并日后按新比例

**被动稀释时"内含商誉"**：
- 应按比例结转（比照直接处置）

**被动稀释产生损失**：
- 先做减值测试 → 确认减值损失 → 再计算稀释影响计入资本公积借方

**联营企业未实缴出资时已亏损**：
- 即使未实缴，如果需承担亏损 → 当期确认（不等到实缴后）

### 1-4 子公司未分配利润转增资本

- 无现金选择权 → 仅为子公司权益结构重分类，母公司不确认投资收益
- 有等值现金选择权 → 实质为宣告分配现金股利，母公司确认投资收益

### 1-5 同一控制下企业合并的认定

**家族成员间转让**：
- 一般不能直接认定为同一控制下合并
- 除非能认定为"代持还原"（需充分证据：资金来源、决策权行使、交易价格）

**新设主体取得集团内公司控制权**：
- 重组由原控股股东主导且不可撤销 → 同一控制下合并
- 重组与后续出售构成一揽子交易 → 非同一控制下合并

### 1-6 同一控制下企业合并的会计处理

**同时向控股股东和第三方购买**：
- 自控股股东购买 → 同一控制下合并
- 自第三方购买 → 购买少数股东权益
- 比较期间只合并自控股股东购买的份额

**少数股东业绩承诺**：
- 合并日以公允价值确认为少数股东权益购买对价的一部分
- 后续按金融工具准则以FVTPL计量

### 1-7 非同一控制下企业合并的或有对价

**或有对价公允价值**：
- 考虑因素：业绩预测、信用风险、连带担保、货币时间价值
- 股份补偿：以确认时市价（非协议发行价）× 补偿股份数
- 购买日后12个月内变动 ≠ 计量期间调整，不调整商誉

**以自身股份结算的或有对价**：
- 购买日：不满足"固定换固定" → 金融资产（FVTPL）
- 业绩确定后股份数确定 → 重分类为权益工具（以重分类日股价计量）→ 不再核算后续公允价值变动

### 1-8 反向购买

**被购买上市公司不构成业务的三种情形**：
1. 空壳上市公司
2. 仅有现金和金融资产
3. 重大资产置换（置出全部+置入非上市公司股权）

→ 按权益性交易处理，不确认商誉或当期损益

**涉及现金对价**：上市公司支付的现金 → 作为合并主体对购买方原股东的利润分配

### 1-9 控制的判断

**委托受托经营**：
- 受托方是否有权力：需考虑重大资产购建/处置/投融资等决策权是否需委托方同意
- 可变回报：不仅是损益分配，还包括整体价值变动的报酬和风险
- 委托期限短/不确定 → 不应认定控制

**有固定期限的一致行动协议**：
- 期限结束后无控制权 → 不应因一致行动协议认定控制

**非营利性组织**：
- 无法分红 ≠ 无可变回报（可能有管理费、技术许可费等）

**清算阶段子公司**：
- 主动清算 → 仍纳入合并范围
- 破产清算（决策权移交破产管理人）→ 丧失控制，不再合并

### 1-10 集团内部交易的抵销

**内部转让房地产的土地增值税**：
- 内部交易期间：合并报表中未实现增值 → 土地增值税作为资产列示
- 出售给第三方时：转入当期损益

**集团内交易中单方计提的增值税**：
- 出售方免税+购入方可抵扣进项 → 进项税权利已成立，不应抵销
- 内部交易未实现利润前：增值税差额确认为递延收益
- 产品出售给第三方时：转入当期损益

### 1-11 不丧失控制权处置子公司部分股权——商誉的考虑

- 处置价款 vs 处置股权对应的子公司净资产份额（含商誉）→ 差额调整资本公积
- 子公司净资产份额应包含归属于母公司的商誉

## 监管规则适用指引——会计类第2号（2021）

> 14个专题，侧重收入确认、金融工具、企业合并实务争议

### 2-1 高度关联性判断（识别履约义务）

- 高度关联性 = 各单项商品/服务之间**互相**受到彼此重大影响（非单方面功能依赖）
- 模具+样品：仅因生产需要模具 ≠ 高度关联；需持续修正模具再生产样品 = 高度关联 → 一项履约义务

### 2-2 客户控制在建商品的判断

- 客户拥有在建商品法定所有权 + 更换供应商后新供应商无需重新执行前期工作 → 客户能控制在建商品 → 时段确认

### 2-3 应付客户对价

- 企业能主导推广服务的使用（上架区域/堆放位置/展示时间等）→ 作为购买推广服务处理
- 超过推广服务公允价值的部分 → 冲减销售收入

### 2-4 暂定价格中的可变对价 vs 嵌入衍生工具

| 变动原因 | 性质 | 处理 |
|----------|------|------|
| 与履约情况相关（数量/质量调整） | 可变对价 | 按可变对价原则 |
| 与挂钩的商品/原材料价格指数相关 | 嵌入衍生工具 | 控制权转移日按指数价格确认收入；后续变动计入公允价值变动损益 |

### 2-5 销售返利

- 现金返利 → 可变对价
- 实物返利/仅适用于未来采购的价格折扣 → 评估是否构成重大权利 → 单项履约义务

### 2-6 运输费用

- 控制权转移前的运输 → 不构成单项履约义务 → 合同履约成本 → 营业成本
- 控制权转移后的运输 → 构成单项履约义务 → 确认运输服务收入+运输服务成本

### 2-7 知识产权许可收入确认时点

- 实物交付（如母带）≠ 控制权转移
- 合同限制初始播放时间 → 客户尚不能主导使用 → 不确认收入

### 2-8 定制化产品研发支出

- 研发成果仅可用于该合同 → 合同履约成本 → 营业成本
- 能控制研发成果+预期带来经济利益 → 无形资产准则资本化

### 2-9 应收账款预期信用损失

- 同一账龄组合中客户应具有共同信用风险特征
- 客户信用风险显著不同/发生显著变化 → 不应继续纳入原组合
- 应收账款转为股权投资/非货币资产偿还 → 表明信用风险特征已变，不应纳入原组合

### 2-10 金融资产业务模式中"出售"的判断

- "出售"必须是满足会计终止确认条件的出售
- 出售但未终止确认（如保理但保留风险）→ 仍通过收取合同现金流实现经济利益 → 不属于"出售"

### 2-11 业绩承诺期内修订业绩补偿条款

- 已履行必要内部决策流程 → 或有对价公允价值变动计入当期损益

### 2-12 一揽子交易分步合并——取得控制权前的付款

- 若最终未取得控制权则一揽子交易整体撤销 → 取得控制权前已付款项作为"预付投资款项"（非长期股权投资）

### 2-13 购买少数股东权益后商誉减值测试

- 收购少数股东权益 ≠ 新的企业合并 → 商誉仍为前期取得控制权时的金额
- 减值测试时：按前期取得控制权时的持股比例恢复为全部商誉（100%）→ 比较可收回金额
- 减值损失：按前期取得控制权时的持股比例计算归属于母公司的部分

### 2-14 递延所得税适用税率变动与非经常性损益

- 因国家政策调整导致高新资质变化 → 一次性调整计入**非经常性损益**
- 因公司自身经营变化导致资质变化 → 一次性调整计入**经常性损益**

## 监管规则适用指引——会计类第3号（2023）

> 11个专题，侧重权益法、租赁、股份支付、搬迁补偿

### 3-1 权益法下顺流交易——长投已减至零

- 长投账面价值不足以抵销的未实现内部交易损益 → 确认为**递延收益**
- 后续相关损益实现时再结转至损益

### 3-2 权益法下未确认投资净损失的后续弥补

- 被投资单位净资产增加时：先按分享比例确认投资损益/OCI/资本公积 → 再按"当期应享有净额 vs 前期未确认损失"孰低确认前期损失

### 3-3 丧失控制权时应收原子公司款项

- 无法收回的应收原子公司款项：按金融工具准则确认计量 → 与原账面余额的差额**抵减处置投资收益**

### 3-4 应收超额亏损子公司债权——归属于母公司净利润

- 债权实质性损失 → 全部计入"归属于母公司所有者的净利润"
- 扣除债权损失后的超额亏损 → 按母公司/少数股东分配比例分别计入

### 3-5 少数股权远期收购义务

- 合并日确认金融负债（回购义务现值）
- 少数股东实质上仍享有普通股权利 → 继续确认少数股东权益，金融负债冲减资本公积
- 少数股东不具有普通股权利 → 不确认少数股东权益，金融负债视为合并成本

### 3-6 租赁负债利息费用

- 使用权资产于租赁期开始日即达到预定可使用状态 → 租赁负债利息费用**不应资本化**
- 租赁期开始日后，租赁负债可视同一般借款

### 3-7 承租人运输/安装费用

- 为使租赁资产达到计划用途的运输/安装费用 ≠ 初始直接费用 → 不计入使用权资产成本
- 判断是否形成其他准则规定的资产或长期待摊费用

### 3-8 租赁到期前购买租赁资产

- 无论是否约定购买选择权 → 购买导致租赁终止 = 补充购买选择权并立即行权
- 终止确认使用权资产和租赁负债的差额 → 调整固定资产初始确认成本（非当期损益）

### 3-9 职工提前离职回售股份

- 回售收益与企业权益工具价值相关 → 股份支付
- 回售收益与权益工具价值无关（如固定回报率/每股净资产）→ 职工薪酬（服务期内分期确认）

### 3-10 搬迁补偿

- 不满足专项应付款条件（非公共利益/非财政预算直接拨付）→ 通常整体作为**资产处置对价**
- 有确凿证据表明存在政府补助成分（如提前搬迁奖励）且能明确区分 → 政府补助部分按政府补助准则处理
- 搬迁费用/停产损失：预计能通过处置对价补偿的 → 暂计入其他资产，处置时转入损益

### 3-11 试运行产品成本

- 试运行产出的存货成本 = 正常设计生产能力下的必要合理支出
- 试运行实际投入超出存货成本的部分 → 计入在建工程

## 监管规则适用指引——会计类第4号（2024）

> 8个专题，侧重REITs、研发、租赁、企业合并

### 4-1 基础设施REITs其他方持有份额的列报

- REITs可选择终止上市以避免现金分配义务 → 不存在不可避免的支付义务
- REITs可通过扩募延长期限 → 不存在到期强制清算义务
- **结论**：并表原始权益人在合并报表中将REITs其他方份额列报为**权益**（非负债）
- 其他投资方角度：持有的份额属于权益工具投资

### 4-2 研发服务合同中排他条款

- 排他服务可单独购买/研发结束后较长时间持续存在/非行业惯例 → 单项履约义务（服务类质保）
- 排他性属于研发服务属性/与履约期限相同/行业惯例 → 不单独识别（保证类质保）

### 4-3 外购研发项目

- 外购研发项目用于自行研发 → 遵守内部研发支出资本化政策
- 用于研究阶段或未达资本化时点的开发阶段 → **费用化**
- 除非有确凿证据表明可通过对外出售等方式带来经济利益

### 4-4 销售合同附最低转售价担保

- 客户取得商品控制权 → 确认收入
- 最低转售价担保 → 作为**可变对价**处理

### 4-5 权益法下顺流交易——处置股权时实现

- 处置/视同处置联营企业股权导致未实现内部交易损益实现 → 计入当期投资收益
- 前期抵销的收入/成本/资产处置损益**不予恢复**

### 4-6 合并报表层面特殊金融工具利息费用资本化

- 子公司报表分类为权益工具、合并报表分类为金融负债 → 合并报表层面产生的借款费用**可以资本化**

### 4-7 非同控合并中被购买方政府补助递延收益

- 政府补助款项不存在需要返还的现时义务 → 购买方**不应**将该递延收益单独识别为可辨认负债
- 无论被购买方采用总额法还是净额法，购买方确认的可辨认净资产公允价值应一致

### 4-8 租赁负债相关递延所得税资产

- 即使预计未来应纳税所得额整体为负 → 仍应考虑使用权资产等产生的应纳税暂时性差异
- 据此确认与租赁负债相关的递延所得税资产

## 监管规则适用指引——会计类第5号（2025）

> 8个专题，侧重收入确认、预期信用损失、权益法、可转债

### 5-1 废弃电器电子产品拆解政府补贴

- 拆解废弃电器电子产品是企业日常经营活动
- 政府补贴实质上是企业向政府提供拆解服务的**对价**（非无偿性）
- 应按收入准则确认收入（非政府补助准则）

### 5-2 合作安排中是否包含客户合同关系

- 判断标准：公司是否基于可单独区分的安排将日常活动产出的商品控制权转移给另一合作方
- 医药研发企业授予制药企业特定区域许可 + 收取许可费 → 构成客户合同关系 → 适用收入准则

### 5-3 履约完毕后减让合同对价

| 减让原因 | 处理 |
|----------|------|
| 与履约活动相关（退回/瑕疵/市场价格波动）或符合行业惯例 | 合同变更或可变对价重新估计 |
| 与履约活动无关 + 符合债务重组定义 | 债务重组 |

### 5-4 知识产权许可合同后续变动

- 延长授予期限 = 新授权合同
- 确认时点：延长期限开始时（非新约定达成时）

### 5-5 计量预期信用损失时信用保险的考虑

- 针对债权组合投保的信用保险（组合共享赔付上限/免赔额）→ **不是**单项债权合同条款的组成部分
- 计量单项债权ECL时**不得**考虑该保险
- 保险理赔款：按或有事项准则，基本确定能收到时确认为资产

### 5-6 直接支付给联营企业的业绩补偿

- 公司无法直接取得业绩补偿 → 不得确认金融资产
- 联营企业收到控股股东业绩补偿 → 视同资本性投入 → 计入所有者权益
- 公司权益法核算：确认联营企业权益变动 → 计入所有者权益

### 5-7 处置子公司时逆流交易未实现损益

- 处置/视同处置原子公司 → 逆流交易未实现损益已实现
- 合并报表：计入当期处置子公司损益 + 调整相关资产账面价值

### 5-8 可转换债券递延所得税

- 可转债金融负债成分计税基础=票面金额 → 与账面价值产生应纳税暂时性差异
- 该差异来自权益工具成分 → **不满足**豁免条件 → 应确认递延所得税负债
- 初始确认：计入所有者权益
- 后续折价摊销：递延所得税负债变动计入当期损益

## 非经常性损益（证监会公告〔2023〕65号）

> 2023年12月22日发布，替代2008年版

### 定义

与公司正常经营业务无直接关系，以及虽与正常经营业务相关但由于其性质特殊和偶发性，影响报表使用人对公司经营业绩和盈利能力做出正常判断的各项交易和事项产生的损益。

### 界定原则

1. 基于交易和事项的**经济性质**判断
2. 基于**行业特点和业务模式**判断
3. 遵循**重要性原则**

### 非经常性损益22项清单

| 序号 | 项目 |
|------|------|
| 1 | 非流动性资产处置损益（含已计提减值准备的冲销部分） |
| 2 | 计入当期损益的政府补助（**除外**：与正常经营密切相关+符合国家政策+确定标准+持续影响） |
| 3 | 非金融企业持有/处置金融资产和金融负债的公允价值变动损益及处置损益（有效套期除外） |
| 4 | 非金融企业收取的资金占用费 |
| 5 | 委托他人投资或管理资产的损益 |
| 6 | 对外委托贷款取得的损益 |
| 7 | 不可抗力（自然灾害）导致的各项资产损失 |
| 8 | 单独减值测试的应收款项减值准备转回 |
| 9 | 取得子公司/联营/合营投资成本小于可辨认净资产公允价值份额的收益（负商誉） |
| 10 | 同一控制下企业合并产生的子公司期初至合并日的当期净损益 |
| 11 | 非货币性资产交换损益 |
| 12 | 债务重组损益 |
| 13 | 因相关经营活动不再持续而发生的一次性费用（如安置职工支出） |
| 14 | 因税收/会计等法律法规调整对当期损益的一次性影响 |
| 15 | 因取消/修改股权激励计划一次性确认的股份支付费用 |
| 16 | 现金结算股份支付可行权日后应付职工薪酬公允价值变动 |
| 17 | 投资性房地产公允价值变动损益（公允价值模式） |
| 18 | 交易价格显失公允的交易产生的收益 |
| 19 | 与正常经营无关的或有事项产生的损益 |
| 20 | 受托经营取得的托管费收入 |
| 21 | 除上述各项之外的其他营业外收入和支出 |
| 22 | 其他符合定义的损益项目 |

### 关键规则

- 将列举项目界定为经常性损益的 → 需在附注单独披露项目名称、金额及原因
- 将未列举项目认定为非经常性损益的 → 金额重大时单列项目名称和金额
- 计算扣非指标时 → 应扣除少数股东损益和所得税影响
- 注册会计师应对非经常性损益的真实性、准确性、完整性及合理性进行核实

## 审计准则问题解答第18号——识别和应对第三方配合实施财务舞弊（2025.01.07）

> 来源：中国注册会计师协会
> 背景：第三方配合舞弊比单独舞弊更隐蔽，常规审计程序（如函证）可能失效

### 六大舞弊领域及第三方配合手段

**1. 收入确认**
- 第三方配合签订"真合同假交易"
- 配合虚假销售的存货实物、单证票据和资金流转（资金闭环）
- 配合隐瞒"抽屉协议"（退货条款/价格保护/售后回购）
- 配合显失公允交易（高价销售/调节各期价格）
- 配合构建无商业实质的贸易业务
- 配合高估履约进度（产出法）

**2. 成本费用**
- 虚增/虚减采购单价套取资金
- 第三方代为承担成本费用
- 延迟/提前确认费用

**3. 关联方资金占用和违规担保**
- 通过虚增长期资产采购价格套取资金
- 投资私募基金后从基金转出资金

**4. 货币资金**
- 定期存款质押为关联方担保但未披露
- 现金管理账户协议归集资金
- 金融机构工作人员配合提供虚假询证函回函

**5. 资产减值**
- 第三方配合提供虚假订单支持高估的未来现金流量
- 虚假货款回笼/虚假保理降低应收账款余额

**6. 资产处置**
- 虚假处置或虚增处置价格

### 配合舞弊的第三方16大特征

1. 经营时间较短
2. 缴纳社保人数较少
3. 为个人或个体工商户
4. 注册资本与交易规模不匹配
5. 交易规模与所处行业不匹配
6. 经营范围与交易性质不匹配
7. 注册地址与被审计单位接近
8. 不同第三方工商信息相同/相近（股东/董监高/地址/联系方式）
9. 既是客户又是供应商，或受同一实际控制人控制
10. 对被审计单位存在重大依赖
11. 销售规模与其需求不匹配
12. 连续审计期间对常规程序配合度持续较高（如连续三年回函相符）
13. 曾配合其他方实施财务舞弊
14. 近三年因违法违规受过行政处罚
15. 股东/关键管理人员与被审计单位存在特殊关系
16. 存在做大收入等特定目的需求

### 可能存在第三方配合舞弊的迹象

- 与不符合自身规定条件的第三方交易且无合理理由
- 交易价格/方式/条款明显不同于与其他方的交易
- 交易商业逻辑不合理或无商业实质
- 交易时间敏感（临近期末/业绩承诺期）
- 对有误的询证函作出"相符"回函
- 被审计单位刻意引导走访对象的选择
- 询问时被审计单位坚持派员陪同
- 被询问人员神情紧张/回复含糊/前后矛盾
- 第三方拒绝确认"不存在抽屉协议"

### 应对措施

1. **增加不可预见性**：不同时间/不同方式/扩大范围/突击检查
2. **深入分析商业实质**：不轻信管理层和第三方解释，向下游客户取证
3. **延伸检查**：
   - 实地走访（选择多名/不同层级人员访谈相同问题）
   - 核实被询问人员是否与被审计单位存在特殊关系
   - 观察客户是否实际使用/消耗采购的产品
   - 检查经销商进销存记录
   - 要求第三方书面承诺信息真实完整
   - 跟函方式函证（观察处理过程）
4. **事务所层面**：纳入高风险项目管控、配备反舞弊专家
5. **与治理层沟通**：告知监督责任和法律后果
6. **考虑向监管机构报告**
