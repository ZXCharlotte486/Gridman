## 一、尽职调查（Due Diligence）

### 尽调范围确认

开始尽调前，先明确：
- 目标公司：名称、行业、商业模式
- 交易类型：平台收购、Add-on、成长期股权、重组、剥离
- 交易规模/复杂度：决定尽调深度
- 已知关注点：客户集中度、监管、环保等
- 时间节点：LOI / 交割目标日期

### 七大尽调工作流

**财务尽调（Financial DD）**
- 收益质量（QoE）：收入和 EBITDA 调整项
- 营运资本分析：正常化 vs 实际
- 债务及类债务项目
- 资本支出（维护性 vs 成长性）
- 税务结构和风险敞口
- 审计历史和会计政策
- 备考调整（run-rate、协同效应）

**商业尽调（Commercial DD）**
- 市场规模和增长（TAM/SAM/SOM）
- 竞争定位和市场份额
- 客户分析：集中度、留存率、NPS
- 定价能力和合同结构
- 销售管道和积压订单
- 销售效率

**法律尽调（Legal DD）**
- 公司架构和组织图
- 重要合同（客户、供应商、合作）
- 诉讼历史和待决索赔
- 知识产权组合和保护
- 监管合规
- 雇佣协议和竞业禁止

**运营尽调（Operational DD）**
- 管理团队评估
- 组织架构和关键人员风险
- IT 系统和基础设施
- 供应链和供应商依赖
- 设施和不动产
- 保险覆盖

**人力资源尽调（HR DD）**
- 组织架构和人员趋势
- 薪酬基准
- 福利和养老金义务
- 关键员工留任风险
- 工会/劳动协议

**IT/技术尽调**（科技型企业）
- 技术栈和架构
- 技术债评估
- 网络安全状况
- 数据隐私合规（GDPR、个人信息保护法）
- 产品路线图和研发投入
- 可扩展性评估

**ESG 尽调**（适用时）
- 环境负债
- 监管合规历史
- ESG 风险和机会

### 行业专项补充

| 行业 | 额外关注点 |
|------|-----------|
| 软件/SaaS | ARR 质量、客户留存分析、SOC2、CAC/LTV |
| 医疗健康 | 监管审批、报销风险、支付方结构 |
| 工业制造 | 设备状况、环境修复、安全记录 |
| 金融服务 | 监管资本、合规历史、信用质量 |
| 消费品 | 品牌健康度、渠道结构、季节性、库存管理 |

### 红旗清单

尽调中发现的问题按严重程度分级：
- **交易终止级**：无法通过谈判解决的根本性问题
- **重大级**：影响估值或交易条款
- **可管理级**：有缓解路径，不影响交易

## 二、估值方法

### 四种核心方法

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| DCF（现金流折现） | 稳定盈利企业 | 理论最严谨，反映内在价值 | 假设敏感性高，终值占比大 |
| 可比公司法（Comps） | 有上市可比公司 | 市场化、直观 | 可比性判断主观 |
| 可比交易法（Precedents） | 有近期并购案例 | 反映控制权溢价 | 案例可得性有限 |
| 资产基础法 | 重资产企业、清算 | 简单直接 | 忽略无形价值和盈利能力 |

实务中通常用"足球场图"（Football Field）展示多种方法的估值区间，取交集作为参考范围。

## 三、DCF 模型

### 核心步骤

**Step 1：历史分析（3-5年）**
- 收入增长趋势（CAGR、驱动因素）
- 利润率演变（毛利率、EBIT 利润率、FCF 利润率）
- 资本密集度（D&A 和 CapEx 占收入比）
- 营运资本效率（NWC 变动占收入增量比）

**Step 2：收入预测**
- 基于历史增长率和行业趋势设定假设
- 三情景框架：悲观/基准/乐观
- 增长率应逐步向终值增长率收敛

**Step 3：自由现金流构建**
```
EBIT
(-) 税收（EBIT × 税率）
= NOPAT（税后净营业利润）
(+) D&A（非现金费用）
(-) CapEx（资本支出）
(-) △NWC（营运资本变动）
= 无杠杆自由现金流（UFCF）
```

**Step 4：WACC 计算**
```
股权成本（CAPM）= 无风险利率 + β × 股权风险溢价
税后债务成本 = 税前债务成本 × (1 - 税率)
WACC = 股权成本 × 股权权重 + 税后债务成本 × 债务权重
```

典型 WACC 区间：
- 大型稳定企业：7-9%
- 成长型企业：9-12%
- 高成长/高风险：12-15%

**Step 5：终值计算**

永续增长法（首选）：
```
终值 = 最后一年 FCF × (1 + 终值增长率) / (WACC - 终值增长率)
约束：终值增长率 < WACC，通常不超过长期 GDP 增长率（2-3%）
```

退出倍数法（备选）：
```
终值 = 最后一年 EBITDA × 退出倍数
```

终值健康性检查：终值占企业价值比例通常在 50-70%，超过 75% 说明过度依赖终值假设。

**Step 6：企业价值 → 股权价值桥接**
```
PV（预测期 FCF 之和）
+ PV（终值）
= 企业价值（EV）
(-) 净债务（= 总债务 - 现金）
= 股权价值
÷ 稀释后股数
= 每股隐含价值
```

**Step 7：敏感性分析**

标准三张敏感性表：
1. WACC vs 终值增长率 → 企业价值/每股价值
2. 收入增长率 vs EBIT 利润率 → 企业价值
3. Beta vs 无风险利率 → WACC 变化

敏感性表要求：
- 使用奇数维度（5×5 或 7×7），确保有真正的中心格
- 中心格 = 基准情景，其输出必须等于模型实际结果
- 中心格高亮显示

## 四、LBO 模型

### 核心结构

**来源与用途（Sources & Uses）**
```
用途：
  收购价格（EV）
  + 交易费用（通常 2-4% of EV）
  = 总用途

来源：
  高级债务（Senior Debt）
  + 次级债务/夹层（Sub Debt / Mezz）
  + 股权（Equity Check）
  = 总来源

来源 = 用途（必须平衡）
```

**债务结构典型参数**

| 层级 | 利率 | 摊销 | 优先级 |
|------|------|------|--------|
| 高级有担保（TLB） | SOFR + 300-500bps | 1% 年摊销 | 最高 |
| 高级无担保 | 固定 6-9% | 到期一次性偿还 | 次之 |
| 夹层/PIK | 固定 10-15% | PIK 或到期偿还 | 最低 |

**债务偿还瀑布**
1. 强制摊销（按合同）
2. 现金扫荡（Cash Sweep）：超额现金流按优先级偿还
3. 余额到期偿还

**关键约束**：债务余额不能为负，使用 MAX(0, ...) 函数。

### 回报分析

**核心指标**
- **MOIC**（投资倍数）= 退出股权价值 / 初始股权投入
- **IRR**（内部收益率）= 使 NPV = 0 的折现率

**回报归因分解**
```
总回报 = EBITDA 增长贡献 + 估值倍数扩张/收缩 + 债务偿还贡献 + 费用拖累
```

**敏感性分析标准格式**

进入倍数 vs 退出倍数（同时展示 IRR 和 MOIC）：

| | 退出 6x | 退出 7x | 退出 8x | 退出 9x | 退出 10x |
|---|---------|---------|---------|---------|----------|
| 进入 7x | | | | | |
| 进入 8x | | | | | |
| 进入 9x | | | | | |
| 进入 10x | | | | | |

其他常用敏感性：EBITDA 增长率 vs 退出倍数、杠杆倍数 vs 退出倍数、持有期 vs 退出倍数。

**三情景分析**

| | 乐观 | 基准 | 悲观 |
|---|------|------|------|
| 收入 CAGR | | | |
| 退出 EBITDA 利润率 | | | |
| 退出倍数 | | | |
| MOIC | | | |
| IRR | | | |

**注意事项**
- 现金流符号：投入为负，退出为正
- 管理层跟投和联合投资会改变股权投入金额
- 分红重组（Dividend Recap）会显著影响 IRR
- 交易费用（通常 2-4% of EV）会降低第一天的股权价值

## 五、投委会备忘录（IC Memo）

### 标准结构

**I. 执行摘要**（1页）
- 公司描述、交易逻辑、关键条款
- 推荐意见和核心回报指标
- 前三大风险及缓解措施

**II. 公司概况**（1-2页）
- 业务描述、产品/服务
- 客户群和市场推广
- 竞争定位
- 管理团队

**III. 行业与市场**（1页）
- 市场规模和增长
- 竞争格局
- 长期趋势/顺风
- 监管环境

**IV. 财务分析**（2-3页）
- 历史业绩（收入、EBITDA、利润率、现金流）
- 收益质量调整
- 营运资本分析
- CapEx 需求

**V. 投资逻辑**（1页）
- 为什么这是好投资（3-5个支柱）
- 价值创造杠杆（有机增长、利润率提升、并购、倍数扩张）
- 前100天优先事项

**VI. 交易条款与结构**（1页）
- 企业价值和隐含倍数
- 来源与用途
- 资本结构/杠杆
- 关键法律条款

**VII. 回报分析**（1页）
- 基准、乐观、悲观情景
- 各情景 IRR 和 MOIC
- 驱动回报的关键假设
- 敏感性分析

**VIII. 风险因素**（1页）
- 按严重程度和可能性排序的关键风险
- 每项风险的缓解措施
- 交易终止风险（如有）

**IX. 推荐意见**
- 明确推荐：推进 / 放弃 / 有条件推进
- 关键条件或后续步骤

### 编写原则

- IC 备忘录应客观平衡——同时呈现多空两面
- 不要淡化风险，投委会成员会自己发现；可信度比乐观更重要
- 财务表格必须勾稽——EBITDA 桥接、来源用途平衡、回报数学一致
- 缺少关键信息时，追问而不是假设

## 六、可比公司分析（Comps）

### 关键倍数

| 倍数 | 计算 | 适用场景 |
|------|------|----------|
| EV/EBITDA | 企业价值 / EBITDA | 最常用，跨行业可比 |
| EV/EBIT | 企业价值 / EBIT | 资本密集度差异大时 |
| EV/Revenue | 企业价值 / 收入 | 高增长但未盈利企业 |
| P/E | 市值 / 净利润 | 盈利稳定的成熟企业 |
| P/B | 市值 / 净资产 | 金融、重资产行业 |
| EV/FCF | 企业价值 / 自由现金流 | 现金流质量评估 |

### 可比公司筛选标准

- 行业/子行业相似
- 商业模式相近（B2B vs B2C、产品 vs 服务）
- 规模相近（收入、市值）
- 增长阶段相近
- 地理市场相近

### 数据标准化

- 统一使用 LTM（过去12个月）或 NTM（未来12个月）
- 剔除非经常性项目
- 调整净债务（包括租赁负债、养老金等类债务项目）
- 使用稀释后股数

## 七、IPO 财务准备

### 7.0 中国 A 股全面注册制（2023 年起）

> **政策节点**：2023 年 2 月 17 日证监会发布全面实行股票发行注册制配套规则，2023 年 4 月 10 日全面注册制下首批主板新股上市。**A 股全市场实行注册制**。

**注册制 vs 核准制核心差异**：

| 维度 | 核准制（已退出） | 注册制（现行） |
|------|----------------|---------------|
| 发行决定权 | 证监会实质审核+核准 | 交易所审核+证监会注册（形式审核为主） |
| 审核重点 | 持续盈利能力、合规性 | 信息披露真实、准确、完整 |
| 定价机制 | 23 倍市盈率隐性约束 | 市场化询价 |
| 上市门槛 | 单一财务指标 | 多套上市标准（含未盈利通道） |
| 退市力度 | 弱 | 强（财务类、规范类、交易类、重大违法类） |

**A 股各板块定位与上市标准**：

| 板块 | 定位 | 主要上市标准（以财务指标为例） | 未盈利上市通道 |
|------|------|------------------------------|--------------|
| 主板 | 大型蓝筹、行业龙头、稳定盈利 | 三套标准，最低：近3年净利润累计≥1.5亿元，最近1年≥6,000万元 | 无 |
| 科创板 | "硬科技"——新一代信息技术、高端装备、新材料、新能源、节能环保、生物医药 | 五套标准，覆盖盈利型、未盈利型、红筹型 | **第五套**：预计市值≥40亿+核心产品阶段性成果 |
| 创业板 | "三创四新"——成长型创新创业企业 | 三套标准 | 不再保留未盈利通道（注册制改革后调整） |
| 北交所 | "更早、更小、更新"——专精特新中小企业 | 四套标准，规模门槛最低 | 第四套：预计市值≥15亿+研发占比 |

**港股上市路径**：

| 章节 | 适用 | 关键要求 |
|------|------|---------|
| 主板 | 一般企业 | 三套标准（盈利/市值+收入/市值+收入+现金流） |
| **18A** | 未盈利生物科技 | 预计市值≥15亿港元+核心产品通过临床I期 |
| **18C** | 特专科技公司 | 已商业化≥60亿港元/未商业化≥100亿港元，含五大领域（新一代信息技术/先进硬件软件/先进材料/新能源及节能环保/新食品及农业技术） |
| 19C | 大中华区二次上市 | 已在合资格交易所上市≥5年（创新产业公司≥2年）+市值≥30亿港元 |

**审核重点变化（注册制下）**：

- 信息披露：从"是否披露"转向"是否说清楚"——重大事项的逻辑链条、关键假设、敏感性
- 持续经营能力：仍是关注重点，但不再作为否决性条件（科创板第五套等明确允许未盈利）
- 财务真实性：穿透核查供应商/客户、关联交易、收入确认时点等
- 同业竞争：注册制下从"必须解决"转为"充分披露+解决路径"
- 历史合规：股权清晰、税务合规、社保合规、环保合规

**关键监管文件**：

- 证监会发行注册管理办法（2023 年 2 月发布，沪深主板/科创板/创业板分别立法）
- 沪深北各交易所股票上市规则（2024 年陆续修订）
- 监管规则适用指引——发行类（编号 1-7 号，持续更新）

> 详细规则参见 https://docs.maoyanqing.com/securities/garr/

### 核心要求

| 要求 | 说明 | 常见问题 |
|------|------|----------|
| 会计基础规范 | 严格执行企业会计准则 | 收入确认不规范、关联交易未充分披露 |
| 内部控制健全 | 建立完善的内控体系 | 审批流程缺失、职责分离不到位 |
| 税务合规 | 历史税务问题清理 | 补税、优惠政策不合规 |
| 独立性 | 资产、人员、业务、财务、机构独立 | 与控股股东存在大额往来 |
| 持续盈利能力 | 证明业务可持续 | 依赖单一客户/产品 |

### 审核关注重点

- 收入真实性和可持续性
- 毛利率合理性（与行业对比）
- 关联交易的必要性和公允性
- 税收优惠的合规性和持续性
- 现金流与利润的匹配度
- 重大客户和供应商依赖
- 同业竞争和关联交易解决方案

### 报告期要求

| 市场 | 报告期 |
|------|--------|
| A 股主板/创业板 | 最近 3 年 1 期 |
| 科创板 | 最近 3 年 1 期（亏损企业可申报） |
| 北交所 | 最近 2 年 1 期 |
| 港股 | 最近 3 个会计年度（或 2 年，视情况） |

### 7.5 IPO 问询预测方法论（发散诊断 + 回复评估）

> 类型：流程
> 来源：实务保荐/审计 IPO 问询经验提炼（古立特化重写）
> 置信度：实务经验
> 关联：regulatory_risk_alerts.md - 监管风险提示/IPO审计红线；penalty_cases.md - IPO问询/处罚

[场景：拟上市企业预判可能被交易所问询的问题、准备回复材料、评估回复质量。]

#### 六条监管主线

所有 IPO 问询归根结底围绕六条主线，预测时用来补漏：

| 主线 | 监管核心关注 |
|------|------------|
| 业务是否真实 | 收入确认、客户真实性、终端销售、合同履约、安装验收 |
| 业绩是否可持续 | 市场空间、行业地位、客户稳定性、在手订单、期后业绩 |
| 利润是否合理 | 成本归集、采购价格公允性、毛利率波动、供应商稳定性 |
| 主体是否清晰独立 | 关联交易、独立性、控制权、历史沿革、同业竞争 |
| 治理是否规范 | 内控、资金流水、财务规范、合规经营、诉讼处罚 |
| 中介是否查到位 | 核查范围、样本选择、函证/访谈/走访、穿行测试、结论依据 |

#### 发散诊断透镜表（信号 → 监管假设 → 问法方向 → 证据需求）

> 类型：判断表
> 来源：IPO审核问询实务

先找项目特异性异常，再归入监管主题。不因为某主题高频就机械提问——没有事实支撑的问题只能做补漏。

| 异常信号 | 监管假设 | 发散问法方向 | 证据需求 |
|---------|---------|------------|---------|
| 收入在期末/Q4集中确认 | 跨期/突击验收/客户配合 | 履约节点、验收实质、期后回款、截止测试 | 合同、发货、安装、验收、发票、回款流水 |
| 收入增长但现金流转弱 | 收入质量/回款可持续性 | 客户付款能力、信用政策变化、逾期、坏账 | 应收账龄、期后回款、客户预算/付款审批、函证 |
| 毛利率显著高于同行或波动大 | 成本归集/定价公允性 | 单价、单位成本、项目毛利、可比公司差异 | 成本明细、采购价格、工时、可比产品、项目台账 |
| 客户集中/单一客户占比高 | 重大依赖/关联关系/收入真实性 | 合作背景、获客方式、终端需求、定价和回款 | 招投标、合同、访谈、工商穿透、终端使用证明 |
| 供应商集中/核心部件外采 | 供应链稳定性/采购公允性/技术独立性 | 是否依赖供应商、替代性、自制进度、采购价格 | 采购合同、比价、供应商访谈、技术分工 |
| 存货/发出商品/合同资产金额大 | 资产真实性/跌价/确认前置不足 | 对应订单、库龄、期后结转、跌价测试 | 订单、盘点、期后发货/验收、可变现净值测算 |
| 募投规模显著大于现有体量 | 产能消化/投资必要性 | 市场容量、在手订单、达产节奏、设备合理性 | 募投可研、产能测算、订单储备、同行扩产 |
| 报告期内股权频繁变动/突击入股 | 利益输送/股份支付/锁定合规 | 入股背景、价格公允、资金来源、穿透核查 | 增资/转让协议、流水、估值报告、访谈 |
| 历史代持/对赌/特殊权利 | 权属争议/控制权稳定 | 形成和解除真实性、恢复条款影响、纠纷风险 | 解除协议、流水、承诺函、律师核查 |
| 研发投入/科创属性叙事过强 | 技术先进性/研发真实性/商业化能力 | 核心技术来源、专利贡献、研发费用归集、产品收入 | 研发项目台账、专利、人员工时、产品收入映射 |

#### 回复质量评估五步闭环

好的问询回复**应当**按以下顺序展开：

```
正面回答问题 → 量化解释原因 → 外部证据佐证 → 中介核查程序 → 补充披露修改
```

缺任何一环都容易被追问。

#### 二轮追问触发条件

当首轮回复出现以下情形，二轮/三轮追问概率极高：

- 只解释不量化（"主要因为市场拓展"但没有数据）
- 自证多第三方证据少（"发行人说明"没有客户/供应商/行业确认）
- 核查程序写得泛（"我们执行了核查"但没说查了什么、查到什么）
- 前后口径不一致（招股书与回复、首轮与二轮、发行人与中介结论矛盾）

#### 预测问题三分法与排序

| 问题类型 | 含义 | 优先级 |
|---------|------|--------|
| 项目特异性 | 由该项目事实直接推导——预测重点 | 最高 |
| 板块高频 | 科创板/创业板/主板/北交所历史问询经验 | 次高 |
| 补漏型 | 六条主线扫描发现、但当前资料证据不足 | 标记"待补充资料" |

每个预测问题标注三个维度：**概率**（高/中/低）× **杀伤力**（影响上市条件/收入利润真实性为高）× **准备难度**（需第三方证据/穿透核查为高）。

## 八、早期/硬科技公司估值

### 传统 DCF 的局限

早期公司现金流为负或高度不确定，传统 DCF 不适用。需要叠加以下方法：

### 情景加权 DCF

不做单一基准预测，构建三个情景并赋予概率权重：

| 情景 | 概率 | 核心假设 |
|------|------|----------|
| 技术突破+市场顺利 | 20-30% | 商业化周期短，市场快速渗透 |
| 基准（渐进发展） | 40-50% | 商业化周期中等，竞争正常 |
| 技术停滞/市场受阻 | 20-30% | 商业化延迟或失败 |

加权估值 = Σ（各情景 DCF × 概率权重）

### 实物期权估值

将技术商业化视为看涨期权：
- 标的资产价值 = 技术成功后的市场规模现值
- 执行价格 = 从当前到商业化所需的总投入
- 期权期限 = 预计商业化时间窗口
- 波动率 = 技术路线的不确定性

实物期权价值 + DCF 现有业务价值 = 完整企业价值

### WACC 的非财务风险溢价

对于供应链高度依赖海外、技术路线竞争激烈的硬科技公司，在标准 CAPM 基础上叠加：

```
Ke = Rf + β × ERP + 供应链风险溢价 + 技术迭代风险溢价 + 合规风险溢价
```

| 风险类型 | 量化思路 | 参考区间 |
|----------|----------|----------|
| 供应链断裂 | 断供概率 × 断供影响（占企业价值比） | 1-5% |
| 技术路线被颠覆 | 技术替代概率 × 资产贬值幅度 | 1-4% |
| 数据主权/合规 | 合规改造成本现值 / 企业价值 | 0.5-3% |

### 关键提示

- 上述溢价的具体数字需基于对特定公司的深度尽调
- 框架的价值在于让决策委员会在同一语言体系下讨论风险，把"地缘政治风险很大"变成可被质疑和修正的具体假设
- 建议由独立的技术专家和行业分析师复核地缘政治和技术路线的评分

## 九、资产评估准则与执业要点

> 本章是"持证资产评估师执业准则"视角，与前面"投行估值"（DCF/可比公司/LBO）是同一件事的两套话语：投行估值服务于投资决策，法定资产评估服务于合规与权属定价。两者方法同源（收益法≈DCF、市场法≈可比公司、成本法≈资产基础法），但评估有法定程序、法定情形、法定报告要求。
> 原文不搬，按需查：MaoDocs 评估板块 https://docs.maoyanqing.com/appraisal/aas/

### 资产评估在财税体系中的定位

> 类型：规则
> 来源：资产评估法（2016）+ 中评协准则体系
> 置信度：准则原文

[场景：搞清评估师和注会、税务师的分工，避免把评估业务当成审计或税务业务来处理]

资产评估是与注册会计师、税务师**平行的独立持证专业**，不是会计或审计的子集：

| 专业 | 持证 | 核心产出 | 解决的问题 |
|---|---|---|---|
| 注册会计师 | CPA | 审计报告 | 财务报表是否公允反映 |
| 资产评估师 | CPV（资产评估师） | 资产评估报告 | 资产/权益值多少钱 |
| 税务师 | CTA | 涉税鉴证/筹划 | 税负是否准确合规 |

⚠️ 评估报告由**资产评估机构**出具、**资产评估师签字**，注会的审计报告不能替代评估报告，反之亦然。涉及法定评估的场景（见下），必须用持证评估机构。

准则体系五层：资产评估法（人大）→ 基本准则（财政部财资〔2017〕43号）→ 职业道德准则 → 执业准则（程序性+实体性）→ 指南与指导意见。

### 法定必须评估的情形（合规红线）

> 类型：判断表
> 来源：资产评估法 + 国资监管规定 + 公司法
> 置信度：准则原文

[场景：判断一笔交易是不是"法律强制要求先评估"，漏评估会导致交易无效或被追责]

| 判断顺序 | 情形 | 是否强制评估 |
|---|---|---|
| 1（国资优先） | 国有资产转让、出售、置换、对外投资、改制 | **强制**，未评估或低于评估值成交无效，可追责 |
| 2 | 以非货币资产出资设立公司 | **强制**（公司法要求作价评估） |
| 3 | 上市公司重大资产重组、发行股份购买资产 | **强制**（证监会要求评估或估值） |
| 4 | 企业破产清算、司法拍卖处置 | **强制**（法院委托评估） |
| 5 | 抵押贷款、担保 | 视债权人要求，通常需要 |
| 6 | 一般民商事主体间的非国资交易 | 不强制，可自愿委托 |

⚠️ 核心红线：**只要一方涉及国有资产，几乎都强制评估**，且成交价不得低于评估结果（确需折价的要履行审批）。这是实务中最容易踩的合规点——国资交易没评估直接定价，交易可被认定无效。

### 三种基本评估方法的选择判断

> 类型：判断表
> 来源：资产评估执业准则——资产评估方法（中评协〔2019〕35号）
> 置信度：准则原文

[场景：拿到一个评估对象，先判断该用哪种方法、为什么不能用某种方法]

三种法定基本方法，对应投行估值的三类：

| 方法 | 原理 | 对应投行估值 | 适用判断 |
|---|---|---|---|
| 收益法 | 预期收益折现 | ≈ DCF | 收益可预测、可量化（经营性资产、企业价值、无形资产） |
| 市场法 | 参照可比交易/可比公司 | ≈ 可比公司法/可比交易法 | 有活跃市场、有可比标的（不动产、设备、上市公司股权） |
| 成本法 | 重置成本 − 各项贬值 | ≈ 资产基础法 | 单项资产、无收益或收益难预测、专用资产（机器设备、在建工程） |

选择规则（执业准则要求）：
1. **应当根据评估对象、价值类型、资料收集情况，分析三种方法的适用性。**
2. **满足条件的应优先考虑两种以上方法**，相互验证，差异大要分析原因。
3. 不能因为"算起来方便"就只用一种。

⚠️ 常见误用：
- 对**持续经营的企业整体**只用成本法（资产加总）→ 漏掉商誉和盈利能力，低估。企业价值应优先收益法/市场法。
- 对**无活跃市场的专用设备**用市场法 → 找不到真正可比标的，结论不可靠，应用成本法。
- 对**早期无收益的技术**用收益法做单一点估计 → 收益假设极不确定，应做情景/期权处理（详见本文件第八章 早期/硬科技估值）。

### 评估与会计的接口（以财务报告为目的的评估）

> 类型：规则
> 来源：以财务报告为目的的评估指南（中评协〔2017〕45号）+ CAS 39 公允价值计量
> 关联：accounting_advanced.md - 公允价值计量；accounting_core.md - 资产减值
> 置信度：准则原文

[场景：会计上要确认公允价值时（企业合并、资产减值、投资性房地产公允模式），评估结论怎么用]

会计准则要求公允价值的地方，常需要评估支撑：

| 会计场景 | 评估对象 | 准则接口 |
|---|---|---|
| 非同一控制企业合并 | 被购买方可辨认资产负债公允价值 | CAS 20，差额计入商誉 |
| 商誉减值测试 | 资产组可收回金额 | CAS 8，需估计未来现金流现值 |
| 投资性房地产公允价值模式 | 房地产公允价值 | CAS 3 |
| 非货币性资产交换 | 换入换出资产公允价值 | CAS 7 |
| 债务重组 | 抵债资产公允价值 | CAS 12 |

⚠️ "以财务报告为目的的评估"有特殊要求：价值类型通常是**公允价值**（不是市场价值），且要与会计准则的计量要求一致。评估师出具的是为会计服务的专项评估，不能直接套用一般市场价值评估的结论。

### 评估与税务的接口

> 类型：规则
> 来源：企业所得税法及实施条例 + 财税〔2009〕59号等
> 关联：tax_core.md - 股权转让/资产划转；tax_advanced.md - 转让定价
> 置信度：准则原文

[场景：资产/股权交易计税时，评估值如何影响计税基础和应纳税额]

| 税务场景 | 评估的作用 |
|---|---|
| 股权转让所得 | 转让价明显偏低且无正当理由的，税务机关按评估/核定价调整（防止平价/低价转让避税） |
| 资产划转、重组 | 特殊性税务处理需以评估确认公允价值，确定计税基础 |
| 非货币资产投资 | 按评估公允价值确认转让所得，可分 5 年递延（财税〔2014〕116号） |
| 关联交易转让定价 | 评估/独立交易价格作为是否公允的判断基准 |
| 房地产、土地增值税清算 | 评估值作为扣除项目/收入确认的依据之一 |

⚠️ 税务上"计税基础"以税法规定为准，评估值是重要参考但不必然等于计税基础——要看具体税种规则。涉及重组、划转的特殊性税务处理，评估是前置条件。

### 新兴评估对象（数据资产、关键核心技术）

> 类型：规则
> 来源：数据资产评估指导意见（中评协〔2023〕17号）、关键核心技术资产评估指导意见（中评协〔2025〕9号）
> 置信度：准则原文

[场景：企业有数据资产入表、或硬科技技术作价时的评估，这是 2023 年后的新领域]

- **数据资产评估（2023〕17号）**：配合"数据资产入表"（财政部数据资产暂行规定，2024.1.1 起），数据资源可作为无形资产或存货入表。评估方法仍是收益法/市场法/成本法三选，但数据资产的特殊性在于：可复制、价值依场景、权属复杂（持有权/加工使用权/经营权分置）。评估前先确认数据权属和合规来源。
- **关键核心技术评估（2025〕9号）**：针对"卡脖子"技术、专精特新企业的核心技术作价。因技术迭代快、收益不确定，强调情景分析和风险溢价（与本文件第八章硬科技估值方法相通）。

> 关联：accounting_core.md - 无形资产确认（数据资产入表的会计处理）；esg.md / financial_bp.md 不涉及，评估归本模块。

# CPA财务成本管理——公司金融部分（第1-11章）

## 第1章 财务管理概述

### 财务管理目标

| 目标 | 含义 | 局限性 |
|------|------|--------|
| 利润最大化 | 利润越多越好 | 未考虑时间、资本额、风险 |
| 每股收益最大化 | 利润/股数 | 未考虑时间、风险；不同公司不可比 |
| **股东财富最大化**（本教材采纳） | 股东权益市场价值最大化 | 综合考虑时间、风险、资本 |

- 股东财富 = 股东权益的市场价值
- 股东权益市场增加值 = 股东权益市场价值 − 股东投资资本
- 股价最大化 ≈ 股东财富最大化（前提：资本市场有效、投资资本不变）
- 公司价值最大化 ≈ 股东财富最大化（前提：债务价值不变）

### 代理问题与协调

| 代理关系 | 冲突表现 | 协调措施 |
|----------|----------|----------|
| 股东 vs 经营者 | 道德风险（不尽力）、逆向选择（损害股东利益） | 监督（审计、治理）+ 激励（股票期权、奖金） |
| 股东 vs 债权人 | 投资高风险项目、发行新债稀释旧债 | 限制性条款、拒绝合作/提前收回贷款 |

### 金融市场类型

| 分类标准 | 类型 | 说明 |
|----------|------|------|
| 期限 | 货币市场（≤1年）/ 资本市场（>1年） | 短期流动性 vs 长期融通 |
| 属性 | 债务市场 / 股权市场 | 固定收益 vs 剩余权益 |
| 发行 | 一级市场（发行）/ 二级市场（流通） | 筹资 vs 交易 |
| 程序 | 场内（交易所）/ 场外（OTC） | 集中竞价 vs 协商定价 |

**有效市场假说三个层次**：弱式（历史价格）、半强式（公开信息）、强式（所有信息）

## 第2章 财务报表分析和财务预测

### 财务比率分析体系

**短期偿债能力比率**

| 指标 | 公式 | 判断标准 |
|------|------|----------|
| 营运资本 | 流动资产 − 流动负债 | 越多越安全，但过多说明资金闲置 |
| 流动比率 | 流动资产 ÷ 流动负债 | 无统一标准，行业差异大；营业周期越短合理值越低 |
| 速动比率 | 速动资产 ÷ 流动负债 | 速动资产 = 货币资金+交易性金融资产+应收款项 |
| 现金比率 | (货币资金+交易性金融资产) ÷ 流动负债 | 最保守指标 |
| 现金流量比率 | 经营活动现金流量净额 ÷ 流动负债 | 流量视角，更具说服力 |

- 营运资本配置比率 = 营运资本 ÷ 流动资产 = 1 − 1/流动比率
- 表外增强因素：银行授信额度、可快速变现的非流动资产、偿债声誉
- 表外削弱因素：担保或有负债、未决诉讼

**长期偿债能力比率**

| 指标 | 公式 | 说明 |
|------|------|------|
| 资产负债率 | 总负债 ÷ 总资产 | 越低越安全，但过低说明未充分利用杠杆 |
| 产权比率 | 总负债 ÷ 股东权益 | = 资产负债率 ÷ (1−资产负债率) |
| 权益乘数 | 总资产 ÷ 股东权益 | = 1 + 产权比率 = 1/(1−资产负债率) |
| 长期资本负债率 | 非流动负债 ÷ (非流动负债+股东权益) | 反映长期资本结构 |
| 利息保障倍数 | EBIT ÷ 利息费用 | <1表示经营收益不能覆盖利息；越大越安全 |
| 现金流量利息保障倍数 | 经营活动现金流量净额 ÷ 利息费用 | 比利息保障倍数更可靠 |
| 现金流量与负债比率 | 经营活动现金流量净额 ÷ 负债总额 | 流量视角的长期偿债能力 |
| 资产有息负债率 | 有息负债 ÷ 资产总额 | 衡量付息债务压力，比资产负债率更聚焦 |
| 有形净值债务率 | 负债总额 ÷（股东权益 - 无形资产净值） | 剔除无形资产后的真实偿债能力 |
| 偿债保障比率 | 负债总额 ÷ 经营活动现金流量净额 | 用当前现金流还清全部负债需要几年 |

**营运能力比率**

| 指标 | 公式 | 说明 |
|------|------|------|
| 应收账款周转率 | 营业收入 ÷ 平均应收账款 | 周转天数 = 365/周转率 |
| 存货周转率 | 营业收入(或营业成本) ÷ 平均存货 | 评估管理业绩用成本，分解周转率用收入 |
| 流动资产周转率 | 营业收入 ÷ 平均流动资产 | |
| 非流动资产周转率 | 营业收入 ÷ 平均非流动资产 | |
| 总资产周转率 | 营业收入 ÷ 平均总资产 | 综合营运效率 |

**盈利能力比率**

| 指标 | 公式 |
|------|------|
| 营业净利率 | 净利润 ÷ 营业收入 |
| 总资产净利率(ROA) | 净利润 ÷ 平均总资产 |
| 权益净利率(ROE) | 净利润 ÷ 平均股东权益 |

**市价比率**

| 指标 | 公式 | 说明 |
|------|------|------|
| 市盈率(P/E) | 每股市价 ÷ 每股收益 | 反映市场对盈利的定价 |
| 市净率(P/B) | 每股市价 ÷ 每股净资产 | 反映市场对净资产的溢价 |
| 市销率(P/S) | 每股市价 ÷ 每股营业收入 | 适用于亏损企业 |

### 传统杜邦分析体系

$$\text{权益净利率(ROE)} = \text{营业净利率} \times \text{总资产周转次数} \times \text{权益乘数}$$

$$= \frac{\text{净利润}}{\text{营业收入}} \times \frac{\text{营业收入}}{\text{总资产}} \times \frac{\text{总资产}}{\text{股东权益}}$$

### 改进的杜邦分析体系（管理用财务报表）

**核心思想**：区分经营活动与金融活动

| 管理用报表项目 | 公式 |
|----------------|------|
| 净经营资产 | 经营资产 − 经营负债 |
| 净负债 | 金融负债 − 金融资产 |
| 净经营资产 | 净负债 + 股东权益 |
| 税后经营净利润 | 净利润 + 税后利息费用 |
| 税后利息费用 | 利息费用 ×（1 − 税率） |

**改进的核心公式**：

$$\text{权益净利率} = \text{净经营资产净利率} + (\text{净经营资产净利率} - \text{税后利息率}) \times \text{净财务杠杆}$$

其中：
- 净经营资产净利率 = 税后经营净利润 ÷ 净经营资产
- 税后利息率 = 税后利息费用 ÷ 净负债
- 净财务杠杆 = 净负债 ÷ 股东权益
- 经营差异率 = 净经营资产净利率 − 税后利息率
- 杠杆贡献率 = 经营差异率 × 净财务杠杆

### 财务预测——销售百分比法

**步骤**：
1. 确定经营资产和经营负债的销售百分比
2. 预计各项经营资产和经营负债 = 预计营业收入 × 销售百分比
3. 融资总需求 = 预计净经营资产 − 基期净经营资产
4. 外部融资额 = 融资总需求 − 可动用金融资产 − 增加留存收益

$$\text{增加留存收益} = \text{预计营业收入} \times \text{营业净利率} \times (1 - \text{股利支付率})$$

### 可持续增长率

**定义**：不增发新股/回购股票，不改变经营效率和财务政策时的最大销售增长率。

**根据期初股东权益**：

$$g = \text{营业净利率} \times \text{期末总资产周转次数} \times \text{期末总资产/期初权益} \times \text{利润留存率}$$

**根据期末股东权益**：

$$g = \frac{\text{营业净利率} \times \text{总资产周转次数} \times \text{权益乘数} \times \text{利润留存率}}{1 - \text{营业净利率} \times \text{总资产周转次数} \times \text{权益乘数} \times \text{利润留存率}}$$

**内含增长率**（仅靠留存收益增长，外部融资=0）：

$$\text{内含增长率} = \frac{\text{净利润/净经营资产} \times \text{利润留存率}}{1 - \text{净利润/净经营资产} \times \text{利润留存率}}$$

## 第3章 价值评估基础

### 货币时间价值核心公式

| 类型 | 终值(F) | 现值(P) |
|------|---------|---------|
| 复利 | $F = P(1+i)^n$ | $P = F(1+i)^{-n}$ |
| 普通年金 | $F = A \cdot \frac{(1+i)^n - 1}{i}$ | $P = A \cdot \frac{1-(1+i)^{-n}}{i}$ |
| 预付年金 | $F = A[(F/A,i,n+1)-1]$ | $P = A[(P/A,i,n-1)+1]$ |
| 递延年金(递延m期) | 同普通年金 | $P = A(P/A,i,n)(P/F,i,m)$ |
| 永续年金 | 无终值 | $P = A/i$ |

**有效年利率**：$i_{eff} = (1 + r/m)^m - 1$（r为报价利率，m为年复利次数）

**连续复利**：$i_{eff} = e^r - 1$

### 风险与报酬

**单项资产风险度量**：
- 期望报酬率：$\bar{K} = \sum P_i K_i$
- 方差：$\sigma^2 = \sum P_i(K_i - \bar{K})^2$
- 标准差：$\sigma = \sqrt{\sigma^2}$
- 变异系数（标准离差率）：$CV = \sigma / \bar{K}$

### 投资组合理论

- 组合期望报酬率：$\bar{R}_p = \sum w_i \bar{R}_i$（加权平均）
- 两资产组合标准差：$\sigma_p = \sqrt{w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\rho_{12}\sigma_1\sigma_2}$
- 相关系数ρ=1时无分散效应；ρ<1时有分散效应；ρ=−1时可完全消除风险

### 资本资产定价模型（CAPM）

$$R_i = R_f + \beta_i(R_m - R_f)$$

- $R_f$：无风险利率
- $R_m$：市场组合期望报酬率
- $R_m - R_f$：市场风险溢价
- $\beta_i$：第i项资产的系统风险系数

**β系数**：$\beta_i = \frac{Cov(R_i, R_m)}{\sigma_m^2} = \frac{\rho_{im}\sigma_i}{\sigma_m}$

**资本市场线（CML）**：描述有效组合的期望报酬率与总风险（标准差）的线性关系

**证券市场线（SML）**：描述所有资产/组合的期望报酬率与系统风险（β）的线性关系，即CAPM

### 债券估值

$$V_B = \sum_{t=1}^{n}\frac{I}{(1+r)^t} + \frac{M}{(1+r)^n} = I \cdot (P/A,r,n) + M \cdot (P/F,r,n)$$

### 股票估值

**零增长模型**：$P_0 = D/r_s$

**固定增长模型（Gordon模型）**：$P_0 = \frac{D_1}{r_s - g} = \frac{D_0(1+g)}{r_s - g}$

**两阶段模型**：高速增长期逐年折现 + 稳定期用Gordon模型折现

## 第4章 资本成本

### 债务资本成本（税后）

$$r_d(1-T)$$

估计方法：
1. **到期收益率法**：使债券市价等于未来现金流现值的折现率
2. **可比公司法**：参照同风险等级公司债券的到期收益率
3. **风险调整法**：$r_d = r_f + \text{企业信用风险补偿率}$

### 权益资本成本

| 方法 | 公式 |
|------|------|
| CAPM | $r_s = r_f + \beta(r_m - r_f)$ |
| 股利增长模型 | $r_s = D_1/P_0 + g$ |
| 债券收益率加风险溢价 | $r_s = r_d + RP_s$（RP通常3%~5%） |

### 加权平均资本成本（WACC）

$$WACC = \sum w_j \cdot r_j = w_d \cdot r_d(1-T) + w_p \cdot r_p + w_s \cdot r_s$$

权重确定方法：账面价值权重、市场价值权重（推荐）、目标资本结构权重

## 第5章 投资项目资本预算

### 项目评价方法

| 方法 | 公式/规则 | 决策标准 |
|------|-----------|----------|
| **净现值(NPV)** | $NPV = \sum\frac{CF_t}{(1+r)^t} - C_0$ | NPV > 0 接受 |
| **内含报酬率(IRR)** | 令NPV=0的折现率 | IRR > 资本成本 接受 |
| **现值指数(PI)** | 未来现金流现值 ÷ 初始投资 | PI > 1 接受 |
| **回收期** | 累计现金流=0的时间 | 越短越好（辅助指标） |
| **折现回收期** | 累计折现现金流=0的时间 | 越短越好（辅助指标） |

### 项目现金流量估计原则

- **增量原则**：只考虑有无该项目的差异现金流
- **税后原则**：使用税后现金流
- **沉没成本不考虑**
- **机会成本要考虑**
- **关联效应要考虑**（如蚕食效应）

**营业现金流量**：

$$OCF = \text{税后营业收入} - \text{税后付现成本} + \text{折旧抵税}$$
$$= \text{收入}(1-T) - \text{付现成本}(1-T) + \text{折旧} \times T$$
$$= (\text{收入} - \text{付现成本} - \text{折旧})(1-T) + \text{折旧}$$
$$= \text{税后净利润} + \text{折旧}$$

### 项目折现率估计

**使用类比法估计项目β**：

当项目的系统风险与公司整体不同时，需要单独估计项目折现率：

1. 找到经营该类业务的上市可比公司
2. 卸载可比公司的财务杠杆：$\beta_{资产} = \frac{\beta_{权益}}{1 + (1-T) \times (D/E)}$
3. 加载目标公司的财务杠杆：$\beta_{权益(目标)} = \beta_{资产} \times [1 + (1-T) \times (D/E)_{目标}]$
4. 用CAPM计算项目权益资本成本，再计算项目WACC

**项目特有风险调整**：
- 如果项目风险高于公司平均风险，折现率上调1%~3%
- 如果项目风险低于公司平均风险，折现率下调1%~3%
- 调整幅度依赖主观判断，缺乏理论依据

### 互斥项目决策

- 寿命相同：直接比较NPV
- 寿命不同：**共同年限法**（最小公倍数）或**等额年金法**（NPV÷年金现值系数）

### 通货膨胀处理

- 名义现金流用名义折现率折现
- 实际现金流用实际折现率折现
- $(1+\text{名义利率}) = (1+\text{实际利率})(1+\text{通胀率})$

## 第6章 期权价值评估

### 期权基本概念

| 要素 | 看涨期权(Call) | 看跌期权(Put) |
|------|---------------|---------------|
| 权利 | 买入标的资产 | 卖出标的资产 |
| 到期日价值 | Max(S−X, 0) | Max(X−S, 0) |
| 内在价值 | Max(S−X, 0) | Max(X−S, 0) |
| 时间溢价 | 期权价值 − 内在价值 | 期权价值 − 内在价值 |

**看涨-看跌平价关系**（欧式）：$C - P = S - PV(X)$

**影响期权价值的因素**：

| 因素 | 看涨期权 | 看跌期权 |
|------|----------|----------|
| 股价↑ | ↑ | ↓ |
| 执行价格↑ | ↓ | ↑ |
| 到期时间↑ | ↑ | ↑ |
| 波动率↑ | ↑ | ↑ |
| 无风险利率↑ | ↑ | ↓ |
| 股利↑ | ↓ | ↑ |

### 二叉树模型

**单期模型核心公式**：

$$C_0 = \frac{(1+r-d)}{(u-d)} \cdot \frac{C_u}{1+r} + \frac{(u-1-r)}{(u-d)} \cdot \frac{C_d}{1+r}$$

- 套期保值比率（Delta）：$H = \frac{C_u - C_d}{S_0(u-d)}$
- 上行概率（风险中性）：$p = \frac{1+r-d}{u-d}$
- 期权价值 = 购买股票支出 − 借款 = $H \cdot S_0 - B$

**多期模型**：从后向前逐级推进

- $u = e^{\sigma\sqrt{t/n}}$，$d = 1/u$

### BS模型

$$C_0 = S_0 N(d_1) - Xe^{-r_c t} N(d_2)$$

$$d_1 = \frac{\ln(S_0/X) + (r_c + \sigma^2/2)t}{\sigma\sqrt{t}}$$

$$d_2 = d_1 - \sigma\sqrt{t}$$

- $N(d)$：标准正态分布累积概率
- $r_c$：连续复利无风险利率
- $\sigma$：股票报酬率标准差（连续复利）

### 实物期权

| 类型 | 含义 | 对应金融期权 |
|------|------|-------------|
| 扩张期权 | 未来扩大投资规模的权利 | 看涨期权 |
| 放弃期权 | 中途放弃项目获得残值的权利 | 看跌期权 |
| 延迟期权 | 推迟投资等待更多信息的权利 | 看涨期权 |

项目价值 = NPV（传统）+ 实物期权价值

## 第7章 企业价值评估

### 现金流折现法

| 模型 | 现金流定义 | 折现率 |
|------|-----------|--------|
| **FCFF（实体自由现金流）** | 税后经营净利润 + 折旧 − 资本支出 − 营运资本增加 | WACC |
| **FCFE（股权自由现金流）** | FCFF − 利息(1−T) − 偿还本金 + 新借债务 | 权益资本成本 |

**FCFF详细公式**：

$$FCFF = \text{税后经营净利润} - \text{净经营资产增加}$$
$$= \text{税后经营净利润} \times (1 - \text{净经营资产增长率/净经营资产净利率})$$

**两阶段模型**：

$$V = \sum_{t=1}^{n}\frac{FCFF_t}{(1+WACC)^t} + \frac{FCFF_{n+1}/(WACC-g)}{(1+WACC)^n}$$

### 相对价值法

| 指标 | 公式 | 适用 | 驱动因素 |
|------|------|------|----------|
| **市盈率(P/E)** | 股价/EPS | 盈利稳定企业 | 增长率、股利支付率、风险 |
| **市净率(P/B)** | 股价/每股净资产 | 重资产企业 | ROE、增长率、风险 |
| **市销率(P/S)** | 股价/每股营业收入 | 亏损企业 | 营业净利率、增长率 |
| **EV/EBITDA** | 企业价值/息税折旧摊销前利润 | 资本密集型 | 消除资本结构差异 |

**本股价值 = 可比公司平均乘数 × 目标公司对应指标**

修正方法：修正市盈率 = 实际P/E ÷ (预期增长率×100)

## 第8章 资本结构

### MM理论

| 命题 | 无税MM | 有税MM |
|------|--------|--------|
| 命题I（价值） | $V_L = V_U$（资本结构无关） | $V_L = V_U + T \times D$（负债有税盾价值） |
| 命题II（资本成本） | $r_s = r_0 + (r_0-r_d)(D/E)$ | $r_s = r_0 + (r_0-r_d)(1-T)(D/E)$ |

- $r_0$：无杠杆权益成本 = 无负债公司的WACC
- 有税时：WACC随负债增加而降低

### 权衡理论

$$V_L = V_U + PV(\text{税盾}) - PV(\text{财务困境成本})$$

最优资本结构：税盾边际收益 = 财务困境边际成本

### 优序融资理论

融资顺序：内部融资（留存收益）→ 债务融资 → 股权融资

原因：信息不对称导致外部融资成本高于内部融资

### 杠杆系数

| 杠杆 | 公式 | 含义 |
|------|------|------|
| **经营杠杆系数(DOL)** | $\frac{\Delta EBIT/EBIT}{\Delta Q/Q} = \frac{Q(P-V)}{Q(P-V)-F} = \frac{M}{EBIT}$ | 销量变动→EBIT变动的放大倍数 |
| **财务杠杆系数(DFL)** | $\frac{\Delta EPS/EPS}{\Delta EBIT/EBIT} = \frac{EBIT}{EBIT-I}$ | EBIT变动→EPS变动的放大倍数 |
| **联合杠杆系数(DTL)** | $DOL \times DFL = \frac{M}{EBIT-I}$ | 销量变动→EPS变动的总放大倍数 |

- M = 边际贡献 = 收入 − 变动成本
- F = 固定成本，I = 利息费用

## 第9章 长期筹资

### 主要筹资方式比较

| 方式 | 优点 | 缺点 |
|------|------|------|
| 普通股 | 无固定股利负担、无到期日、增强信誉 | 资本成本高、分散控制权、信息披露 |
| 优先股 | 无表决权不稀释控制、固定股利可税前扣除（部分） | 资本成本高于债务、股利不可抵税 |
| 长期借款 | 成本低（利息抵税）、速度快 | 限制条件多、财务风险大 |
| 公司债券 | 成本低于股权、不稀释控制权 | 财务风险大、限制条件 |

### 可转换债券

- 转换比率 = 债券面值 ÷ 转换价格
- 价值下限 = Max(纯债券价值, 转换价值)
- 实质：债券 + 看涨期权（认股权证）

### 认股权证

- 实质：长期看涨期权
- 筹资时附带发行，降低债券利率
- 行权时公司增发新股（与普通期权的区别）

### 租赁决策

| 类型 | 经营租赁 | 融资租赁 |
|------|----------|----------|
| 期限 | 短期 | 接近资产寿命 |
| 维护 | 出租人 | 承租人 |
| 期满 | 退还 | 留购/续租/退还 |
| 实质 | 租用 | 分期购买 |

**融资租赁决策**：比较"借款购买"与"租赁"的现金流现值，选择成本低者

**租赁净现值（NAL）**：
$$NAL = \text{租赁相对于借款购买的净现值}$$
$$= \text{避免的资产购置支出} - \text{租赁付款现值} - \text{丧失的折旧抵税现值} + \text{避免的维护费现值}$$

- NAL > 0：租赁有利
- NAL < 0：借款购买有利
- 折现率 = 有担保借款的税后利率（因为租赁付款的风险类似于有担保债务）

### 优先股资本成本

$$r_p = \frac{D_p}{P_0(1-f)}$$

- $D_p$：优先股每年股利
- $P_0$：优先股发行价格
- $f$：发行费用率
- 优先股股利不可税前扣除，无需税后调整

## 第10章 股利分配

### 股利理论

| 理论 | 核心观点 |
|------|----------|
| **股利无关论（MM）** | 在完美市场中，股利政策不影响公司价值 |
| **税差理论** | 资本利得税率<股利所得税率，少发股利有利 |
| **信号理论** | 股利传递公司前景信息，增加股利=利好信号 |
| **代理理论** | 股利减少管理层可支配现金，降低代理成本 |

### 四种股利政策

| 政策 | 特点 | 适用 |
|------|------|------|
| 剩余股利政策 | 先满足投资需要，剩余利润分配 | 初创期/高增长期 |
| 固定股利政策 | 每股股利固定不变 | 成熟稳定企业 |
| 固定股利支付率 | 按固定比例分配利润 | 较少采用 |
| 低正常股利加额外股利 | 维持低水平+好年景额外发放 | 盈利波动大的企业 |

### 股票分割与回购

- **股票分割**：增加股数、降低每股价格，不改变股东权益总额
- **股票回购**：减少流通股数，提高EPS和股价；可替代现金股利（税收优势）

## 第11章 营运资本管理

### 营运资本策略

| 策略 | 流动资产水平 | 短期融资比例 | 风险 | 收益 |
|------|-------------|-------------|------|------|
| 激进型 | 低 | 高 | 高 | 高 |
| 保守型 | 高 | 低 | 低 | 低 |
| 适中型 | 中 | 中 | 中 | 中 |

**易变现率** = (股东权益 + 长期债务 + 自发性流动负债 − 长期资产) ÷ 经营流动资产
- 易变现率 > 1：保守型
- 易变现率 < 1：激进型

### 现金管理模型

**鲍莫尔模型**（确定性模型）：

$$C^* = \sqrt{\frac{2TF}{K}}$$

- C*：最佳现金持有量
- T：一定期间现金总需求量
- F：每次转换固定成本
- K：持有现金的机会成本率

**米勒-奥尔模型**（随机模型）：

$$R = \sqrt[3]{\frac{3b\sigma^2}{4i}} + L$$

- R：回归线（目标现金余额）
- H = 3R − 2L（上限）
- L：下限（管理层设定）
- b：每次转换固定成本
- σ²：每日现金流量方差
- i：日利率

### 应收账款信用政策

**信用政策五要素**：信用标准、信用条件（信用期限、折扣期限、现金折扣）、收账政策

**信用决策**：比较放宽信用的边际收益与边际成本

- 边际收益 = 增加的边际贡献
- 边际成本 = 增加的应收账款机会成本 + 坏账损失 + 收账费用 + 现金折扣成本

$$\text{应收账款机会成本} = \frac{\text{年赊销额}}{360} \times \text{平均收现期} \times \text{变动成本率} \times \text{资本成本}$$

### 存货管理——经济订货量(EOQ)模型

$$Q^* = \sqrt{\frac{2KD}{K_c}}$$

- Q*：经济订货批量
- K：每次订货固定成本
- D：年需求量
- $K_c$：单位存货年储存成本

**相关总成本**：$TC = \sqrt{2KDK_c}$

**最佳订货次数**：$N = D/Q^*$

**再订货点**：$R = L \times d$（L=交货时间，d=日需求量）

**考虑保险储备**：$R = L \times d + B$（B=保险储备量）

### 短期债务管理

**商业信用成本**：

放弃现金折扣的信用成本（年化）：
$$\text{放弃折扣成本} = \frac{\text{折扣百分比}}{1 - \text{折扣百分比}} \times \frac{365}{\text{信用期} - \text{折扣期}}$$

例：条件"2/10, n/30"（10天内付款享2%折扣，30天为信用期）：
$$= \frac{2\%}{1-2\%} \times \frac{365}{30-10} = 37.24\%$$

**决策规则**：如果短期借款利率 < 放弃折扣成本 → 应借款享受折扣

**短期借款实际利率**：

| 方式 | 实际利率 |
|------|----------|
| 收款法（到期付息） | 名义利率 |
| 贴现法（预扣利息） | 名义利率 ÷ (1 − 名义利率) |
| 补偿性余额 | 名义利率 ÷ (1 − 补偿性余额比例) |

### 资本结构决策方法

**每股收益无差别点法（EBIT-EPS分析）**：

$$\frac{(EBIT^* - I_1)(1-T) - D_1}{N_1} = \frac{(EBIT^* - I_2)(1-T) - D_2}{N_2}$$

- $EBIT^*$：每股收益无差别点的息税前利润
- 当预期EBIT > $EBIT^*$时，选择负债筹资方案（财务杠杆有利）
- 当预期EBIT < $EBIT^*$时，选择股权筹资方案

## 三表联动模型（3-Statement Model）

### 概述

三表联动模型是投行、PE、FP&A 的基础工具：利润表、资产负债表、现金流量表通过内在逻辑相互勾稽，改变一个假设，三张表同步联动。

### 搭建顺序

```
假设表（Assumptions）
    ↓
利润表（Income Statement）
    ↓
资产负债表（Balance Sheet）
    ↓
现金流量表（Cash Flow Statement）
    ↓
校验表（Checks）
```

### 假设表（驱动因子）

| 类别 | 典型假设 |
|------|----------|
| 收入 | 收入增长率、产品单价、销量 |
| 成本 | 毛利率、COGS 占收入比 |
| 费用 | 销售费用率、管理费用率、研发费用率 |
| 营运资本 | DSO（应收周转天数）、DIO（存货周转天数）、DPO（应付周转天数） |
| 资本支出 | CapEx 占收入比、折旧年限 |
| 融资 | 利率、还款计划、股权融资 |
| 税务 | 有效税率 |

### 利润表搭建

```
营业收入
- 营业成本
= 毛利
- 销售费用
- 管理费用
- 研发费用
= EBITDA
- 折旧与摊销
= EBIT（营业利润）
- 财务费用（利息）
+ 投资收益
= 利润总额（EBT）
- 所得税费用
= 净利润
```

**规则：** 预测期每个科目必须由假设表驱动（公式引用），不能硬编码数字。

### 资产负债表搭建

**资产侧：**
- 货币资金 = 现金流量表期末现金（联动）
- 应收账款 = 收入 × DSO ÷ 365
- 存货 = 营业成本 × DIO ÷ 365
- 固定资产净值 = 期初 + CapEx - 折旧

**负债侧：**
- 应付账款 = 营业成本 × DPO ÷ 365
- 短期借款 / 长期借款 = 按还款计划
- 应付利息 = 借款余额 × 利率（可能产生循环引用）

**权益侧：**
- 未分配利润 = 期初 + 净利润 - 分红

### 现金流量表搭建

**经营活动：**
```
净利润
+ 折旧与摊销（加回非现金）
+ 股份支付（加回非现金）
- 应收账款增加（营运资本变动）
- 存货增加
+ 应付账款增加
= 经营活动现金流量净额
```

**投资活动：**
```
- 购建固定资产（CapEx）
- 长期投资
= 投资活动现金流量净额
```

**筹资活动：**
```
+ 借款收到的现金
- 偿还借款
- 分配股利
+ 吸收投资
= 筹资活动现金流量净额
```

**期末现金 = 期初现金 + 经营 + 投资 + 筹资**

### 核心校验规则（必须全部通过）

| 校验项 | 公式 | 期望结果 |
|--------|------|----------|
| 资产负债表平衡 | 资产 - 负债 - 权益 | = 0 |
| 现金勾稽 | 现金流量表期末现金 - 资产负债表货币资金 | = 0 |
| 净利润联动 | 利润表净利润 - 现金流量表起始净利润 | = 0 |
| 未分配利润滚动 | 期初未分配利润 + 净利润 - 分红 - 期末未分配利润 | = 0 |

**如果校验不通过：**
1. 先检查资产负债表是否平衡
2. 再检查现金是否勾稽
3. 最后检查未分配利润滚动
4. 常见原因：漏了某个营运资本变动项、折旧没加回、利息循环引用未收敛

### 循环引用处理

利息费用 → 净利润 → 现金 → 借款余额 → 利息费用（循环）

**解决方法：**
- Excel：开启迭代计算（文件 → 选项 → 公式 → 启用迭代计算，最大迭代次数 100）
- 加一个"断路器"开关：在假设表放一个 TRUE/FALSE 单元格，FALSE 时利息用上期余额计算（打断循环）

### 情景分析

| 情景 | 说明 |
|------|------|
| 基准情景（Base） | 管理层指引或市场一致预期 |
| 乐观情景（Upside） | 收入增长超预期、毛利率扩张 |
| 悲观情景（Downside） | 收入下滑、毛利率压缩 |

用 CHOOSE 函数或 INDEX/MATCH 实现情景切换，一个下拉框控制三张表同时变化。

**情景校验：** 乐观 > 基准 > 悲观（净利润、EBITDA、自由现金流、各项利润率）。如果不满足这个层级关系，说明假设设置有逻辑问题。
