# W9: 报表勾稽校验

**触发场景：** "看看报表对不对"、"三表勾稽"

**步骤：**

```
[输入] 资产负债表、利润表、现金流量表
   ↓
Step 1: 资产负债表平衡：资产 = 负债 + 所有者权益
   ↓
Step 2: 现金勾稽：现金流量表期末现金 = 资产负债表货币资金
   ↓
Step 3: 净利润联动：利润表净利润 = 现金流量表起始净利润
   ↓
Step 4: 未分配利润滚动：期初 + 净利润 - 分红 = 期末
   ↓
Step 5: 投资性现金流 vs 固定资产/在建工程变动
   ↓
Step 6: 筹资性现金流 vs 借款/股本变动
   ↓
[输出] 勾稽校验报告（含通过/不通过项 + 差异分析）
```

参考详细规则：`investment.md` 三表联动模型 + `statement.md` 报表审查框架。

## 验证清单

> 从本工作流关键判断 + 步骤检查点提取，交付前逐条核对。

- [ ] 净利润联动：利润表净利润 = 现金流量表起始净利润
- [ ] 未分配利润滚动：期初 + 净利润 − 分红 = 期末
- [ ] 投资性现金流 vs 固定资产/在建工程变动已核对
- [ ] 筹资性现金流 vs 借款/股本变动已核对
- [ ] 不通过项已如实标出差异（不假装勾稽通过）
- [ ] 校验报告含通过/不通过 + 差异分析
