# W6: 月结闭环

**触发场景：** "帮我做月结"、"月度关账"、"月末结账"

**可单独执行每一步，也可以说"帮我跑完整个月结"按顺序执行。**

**步骤：**

```
[输入] 当月科目余额表 + 序时账 + 银行对账单 + 各业务模块数据
   ↓
Step 1: 银行对账
        工具：bank_reconciliation
        输入：银行对账单 + 银行日记账
        输出：调节表 + 未达账项
        判断：调节后余额必须相等
   ↓
Step 2: 往来核对
        工具：aging_analysis + data_filter
        输入：应收/应付明细
        输出：账龄分析 + 长期挂账清单
        判断：超 1 年的往来需说明原因
   ↓
Step 3: 计提处理
        工具：知识层（accounting_core.md）
        输入：当月业务数据
        输出：计提分录清单（折旧/摊销/工资/社保/税金/坏账）
        判断：与上月对比，异常波动需解释
   ↓
Step 4: 截止检查
        工具：cutoff_test
        输入：序时账 + 月末日期
        输出：跨期交易清单
        判断：月末最后几笔是否属于下月
   ↓
Step 5: 试算平衡
        工具：balance_sheet_process
        输入：当月科目余额表
        输出：末级科目 TB + 借贷平衡校验
        判断：借贷必须平衡，不平衡则有错
   ↓
Step 6: 波动分析
        工具：analytical_procedures
        输入：本月 TB vs 上月 TB
        输出：各科目波动分析表 + 异常标记
        判断：波动 > 10% 的科目需写差异说明
   ↓
Step 7: 现金流质量核查
        工具：cashflow_test
        输入：三大报表（资产负债表 + 利润表 + 现金流量表）+ 当月 TB
        输出：间接法现金流测算表 + 与报表现金流对比差异
        判断：
          ✓ 测算经营性现金流 vs 报表现金流，差异 > 5% 需关注
          ✓ 经营性现金流 < 净利润 → 应收/预收/存货吸收了利润
          ✓ 投资/筹资活动现金流与当月重大业务是否匹配
        注意：cashflow_test 含行业假设（VAT 13%/折旧比例等），输出 message 已声明
   ↓
Step 8: 报表生成
        工具：知识层（audit_procedures.md 报表调整体系）
        输入：审定后 TB
        输出：资产负债表 + 利润表（本月）
        验证：表内平衡 + 跨表勾稽
   ↓
Step 9: 差异说明
        工具：data_group_summary + analytical_procedures
        输入：波动分析结果
        输出：差异说明报告（每个异常科目的原因解释）
        格式：科目 | 本月 | 上月 | 变动 | 原因
   ↓
[输出]
   ▸ 银行余额调节表.xlsx
   ▸ 往来账龄分析.xlsx
   ▸ 计提分录清单.xlsx
   ▸ 截止检查报告.xlsx
   ▸ 试算平衡表.xlsx
   ▸ 波动分析表.xlsx
   ▸ 现金流测算表.xlsx（cashflow_test 输出）
   ▸ 月度资产负债表.xlsx
   ▸ 月度利润表.xlsx
   ▸ 差异说明报告.xlsx
```

**关键判断：**

| 项目 | 判断标准 |
|------|---------|
| 银行调节 | 调节后余额必须相等 |
| 往来长账龄 | > 1 年需说明原因 |
| 计提异常 | 与上月差异 > 20% 需解释 |
| 截止 | 月末最后 3 天的大额交易重点检查 |
| 波动阈值 | > 10% 需写差异说明 |
| 报表勾稽 | 资产=负债+权益，必须平衡 |

## 验证清单

> 从本工作流关键判断 + 步骤检查点提取，交付前逐条核对。

- [ ] 银行调节后余额两边相等
- [ ] 往来账龄完成，超 1 年长账龄已说明原因
- [ ] 计提分录（折旧/摊销/工资/社保/税金/坏账）齐全，与上月差异 > 20% 已解释
- [ ] 截止检查完成，月末最后几笔跨期已判断
- [ ] 试算平衡：借贷必须平衡
- [ ] 波动分析：> 10% 的科目已写差异说明
- [ ] 现金流质量核查：经营性现金流 vs 净利润已比对（< 0.5 预警）
- [ ] 报表生成后表内平衡 + 跨表勾稽已校验
- [ ] 差异说明报告格式：科目|本月|上月|变动|原因
- [ ] 全部输出已存入 gridman-mind/outputs/
