# W14: 财务比率分析

**触发场景：** "帮我算一下这家公司的财务指标"、"ROE/ROA/流动比率/资产周转"、"财务比率分析"、"比率底稿"

**业务背景：**

财务比率分析是财务分析的基础工具，覆盖偿债能力、营运能力、盈利能力、成长能力和每股指标五大维度。`financial_ratios` 工具以科目余额表或三大报表为输入，自动计算、标记异常并生成带趋势对比的底稿。

**步骤：**

```
[输入] 科目余额表（含本期/上期余额）或 三大报表（资产负债表+利润表+现金流量表）
   ↓
Step 1: data_file_overview（确认输入文件列名和数据范围）
   ↓
Step 2: 与用户确认
        - 分析期间（年度/季度/月度）
        - 是否有上期数据做趋势对比
        - 是否需要行业均值参照（如有，提供行业名称）
   ↓
Step 3: financial_ratios
        - 输入：财报文件 + 分析期间
        - 输出：五类比率分析底稿（含公式、计算值、同比变动）
          ✓ 变现能力：流动比率、速动比率、现金比率
          ✓ 营运能力：应收周转率/天数、存货周转率/天数、资产周转率
          ✓ 偿债能力：资产负债率、利息保障倍数、产权比率
          ✓ 盈利能力：毛利率、净利率、ROE（净资产收益率）、ROA（总资产报酬率）
          ✓ 每股指标：EPS、每股净资产、每股现金流（上市公司适用）
        - 自动标记：同比变动幅度 > 20% 的指标标橙，> 50% 标红
   ↓
Step 4: 业务结论
        - 报告各维度核心异常项（最多 3 个）
        - 给出初步判断：如"应收周转天数从 45 天上升至 78 天，应收管理明显恶化"
        - 建议下一步：关注应收/审查坏账/核查存货积压等
   ↓
[输出] 财务比率分析底稿.xlsx（五类比率 + 同比变动 + 异常标记）
```

**关键判断：**

| 指标 | 正常范围 | 警示信号 |
|------|---------|---------|
| 流动比率 | 1.5–2.0 | < 1.0 → 短期偿债风险 |
| 资产负债率 | 40%–60%（制造业） | > 80% → 财务风险高 |
| 应收周转天数 | 行业均值 ±30% | 持续上升 → 应收管理恶化 |
| ROE | > 10%（盈利企业） | 连续下降 → 盈利能力衰退 |
| 毛利率 | 行业均值 ±5% | 异常高 → 可能存在收入水分 |

**与其他工作流的关系：**

- **上游**：W7 上市公司财报分析（Step 3）调用本工作流
- **平行**：W9 报表勾稽校验（先勾稽通过再做比率分析更可靠）
- **下游**：比率分析发现异常 → W1/W2/W3 深入审计
