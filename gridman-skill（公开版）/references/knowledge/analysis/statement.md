## 一、报表合规性审查

### 审查维度

#### 1. 准则合规性

- 报表格式是否符合财政部规定
- 会计政策选择是否合理且一致
- 会计估计变更是否充分披露
- 附注披露是否完整

#### 2. 勾稽关系验证

**报表内勾稽**：

| 检查项         | 公式                             | 说明     |
| -------------- | -------------------------------- | -------- |
| 资产负债表平衡 | 资产 = 负债 + 所有者权益         | 基本等式 |
| 利润表结构     | 营业利润 + 营业外收支 = 利润总额 | 利润构成 |
| 净利润         | 利润总额 - 所得税费用 = 净利润   | 税后利润 |

**报表间勾稽**：

| 检查项     | 关系                               | 说明     |
| ---------- | ---------------------------------- | -------- |
| 净利润一致 | 利润表净利润 = 权益变动表净利润    | 跨表核对 |
| 未分配利润 | 期初 + 净利润 - 分配 = 期末        | 连续性   |
| 现金流量表 | 期末现金 - 期初现金 = 现金净增加额 | 现金变动 |

#### 3. 关键比率分析

| 类别     | 指标         | 计算公式           | 关注点     |
| -------- | ------------ | ------------------ | ---------- |
| 偿债能力 | 流动比率     | 流动资产/流动负债  | < 1 警示   |
| 偿债能力 | 资产负债率   | 负债/资产          | > 70% 关注 |
| 盈利能力 | 毛利率       | (收入-成本)/收入   | 异常波动   |
| 盈利能力 | 净利率       | 净利润/收入        | 与行业对比 |
| 营运能力 | 应收周转天数 | 应收/日均收入×365 | 持续延长   |
| 营运能力 | 存货周转天数 | 存货/日均成本×365 | 持续延长   |
| 现金质量 | 收现比       | 经营现金流/净利润  | 长期 < 1   |
| 现金质量 | 净利润含金量 | (CFO + 投资收益实收现金 + 利息收入实收现金 + 处置长期资产溢余现金) / 合并净利润 × 100% | ≤ 30% 极高风险 |
| 利润归属 | 归母偏离度 | 归母净利润 / 合并净利润 × 100% | 超出 90%~110% 需核查 |
| 利润归属 | 少数股东损益/权益匹配度 | \|少数股东损益占比 - 少数股东权益占比\| | > 15% 异常 |
| 供应链 | 供应链净占款差额 | (应付票据+应付账款+合同负债) - (应收票据+应收账款+应收款项融资+预付款项+合同资产) | 持续为负 = 话语权弱 |
| 现金流画像 | 现金流四象限 | CFO±/CFI±/CFF± 组合 | 见下方分类表 |
| 偿债能力 | 现金利息保障倍数 | CFO / 实际利息支付现金 | < 1 极高风险 |
| 负债结构 | 核心经营性负债占比 | (应付账款+应付票据+合同负债) / 总负债 × 100% | 持续下降需关注 |

### 净利润含金量（增强版）

> 类型：公式
> 来源：反舞弊实务
> 置信度：实务经验

[场景：判断净利润背后有多少真金白银——比单纯"净现比 = CFO/净利润"更全面，覆盖了投资收益、利息收入和资产处置的现金回收]

**公式**：

```
净利润含金量 = (CFO + 取得投资收益收到的现金 + 利息收入实收现金 + 处置长期资产收回的现金净额) / 合并净利润 × 100%
```

**取数来源**：
- CFO = 现金流量表「经营活动产生的现金流量净额」
- 投资收益实收 = 现金流量表「取得投资收益收到的现金」
- 利息收入实收 = 现金流量表「收到其他与经营活动有关的现金」附注中的利息收入部分（或利润表「利息收入」作近似）
- 处置长期资产 = 现金流量表「处置固定资产、无形资产和其他长期资产收回的现金净额」
- 分母 = 利润表「净利润」（合并口径，非归母，确保分子分母口径一致）

**判定标准**：
- 含金量 ≥ 80%：利润现金支撑极强
- 含金量 50%~80%：利润基本有现金支撑，部分被应收/存货占用
- 含金量 30%~50%：利润水分较大
- 含金量 ≤ 30%：利润几乎无现金支撑，极高风险

⚠️ 分母口径注意：用合并净利润（不是归母净利润），因为分子 CFO 是合并口径。如果分母用归母净利润而少数股东权益占比大，含金量会被虚高。

⚠️ 与简版"净现比"的关系：净现比（CFO/净利润）是简版，适合快速筛查；含金量增强版覆盖面更全，适合深度分析和底稿留痕。

### 供应链议价能力差额

> 类型：公式
> 来源：反舞弊实务 + 产业链分析
> 置信度：实务经验

[场景：一个数字判断企业在供应链中是"占别人钱"还是"被别人占钱"——正 = 强势，负 = 弱势]

**公式**：

```
供应链净占款差额 = (应付票据 + 应付账款 + 合同负债) - (应收票据 + 应收账款 + 应收款项融资 + 预付款项 + 合同资产)
```

**判定标准**：

| 差额 | 含义 | 判定 |
|---|---|---|
| > 0 | 应付预收 > 应收预付，企业占用上下游资金 | 供应链话语权强 |
| < 0 | 应收预付 > 应付预收，企业资金被上下游占用 | 供应链话语权弱 |
| 持续 > 0 且稳定 | 产业链地位牢固 | 正常优秀 |
| 持续 < 0 且恶化 | 资金链风险累积 | 需警惕 |

**异常判定**：
- 单期差额 < 0：标记【供应链话语权弱势】
- 持续 2 期+ 为负且绝对值扩大：标记【资金被占用高风险】
- 4 期全周期为负且占营收比 < -10%：极高风险，资金链断裂隐患

**典型参照**：茅台、海天味业等消费品龙头差额持续为正（预收 > 应收）；制造业代工企业差额常为负（垫款生产 + 下游赊销）。

### 现金流四象限分类

> 类型：判断表
> 来源：财务分析实务
> 置信度：实务经验

[场景：用 CFO/CFI/CFF 三项的正负组合快速给企业现金流画像——确定企业处于哪个生命周期阶段]

| 组合 | 类型 | 含义 | 投资判断 |
|---|---|---|---|
| CFO+ / CFI- / CFF- | 优质现金牛 | 经营赚钱、投资扩张、还债分红 | 可长期持有 |
| CFO+ / CFI- / CFF+ | 扩张成长型 | 经营赚钱、大幅投资、借债扩张 | 谨慎跟踪，关注扩张合理性和回报 |
| CFO+ / CFI+ / CFF- | 收缩变现型 | 经营赚钱、变卖资产、偿债缩表 | 谨慎——可能在去杠杆或瘦身 |
| CFO- / CFI- / CFF+ | 烧钱续命型 | 经营亏现金、继续投资、靠融资续命 | 高风险，优先排除 |
| CFO- / CFI+ / CFF- | 困境变卖型 | 经营亏现金、变卖资产回血、偿债 | 谨慎回避——可能已进入财务困境 |
| CFO- / CFI+ / CFF+ | 初创/转型期 | 经营未正轨、变卖旧资产、融资支撑 | 视行业和阶段判断 |

**使用规则**：
1. 先看 4 期全周期的组合是否一致——一致说明企业处于稳定状态
2. 组合发生切换时关注切换方向：现金牛 → 烧钱续命 = 恶化信号
3. 单期异常（如年末集中分红导致 CFF 为负）需排除偶发因素

⚠️ 现金流类型只是"画像"，不是定论。CFO- 不一定是坏事（如高速增长期的提前备货），CFO+ 不一定是好事（如压供应商尾款导致的短期虚高）。必须结合利润表和资产负债表交叉验证。

### 现金利息保障倍数

> 类型：公式
> 来源：Wall Street Prep / CFA 教材（Cash Interest Coverage）
> 置信度：实务经验

[场景：传统利息保障倍数（EBIT/利息费用）是权责发生制——有利润不等于有现金。现金版用实际经营现金流除以实际利息支出，更保守、更真实]

**公式**：

```
现金利息保障倍数 = 经营活动产生的现金流量净额（CFO）/ 实际利息支付现金
```

**取数说明**：
- 分子：现金流量表「经营活动产生的现金流量净额」
- 分母：现金流量表附注中"偿付利息支付的现金"（需从"分配股利、利润或偿付利息支付的现金"中拆分出利息部分）；如附注未单列，可用利润表「利息费用」近似

**判定标准**：

| 倍数 | 判定 |
|---|---|
| ≥ 5 | 利息偿付能力极强 |
| 3~5 | 安全 |
| 1~3 | 需关注 |
| < 1 | 经营现金流不足以覆盖利息支出——极高风险 |

**与传统利息保障倍数的区别**：

| 维度 | 传统版（EBIT/利息费用） | 现金版（CFO/实际利息支付） |
|---|---|---|
| 口径 | 权责发生制 | 收付实现制 |
| 优点 | 数据易取（利润表直接有） | 更保守，反映"实际能付多少" |
| 局限 | 利润可能没收到钱（应收大 = 虚高） | CFO 波动大（季节性、回款节奏） |
| 适用 | 日常筛查 | 偿债能力深度评估、高杠杆公司 |

⚠️ 当传统利息保障倍数很高但现金版 < 3 时——说明"有利润但没现金"，是造假或回款异常的信号。两个版本配合使用效果最好。

### 核心经营性负债占比

> 类型：公式
> 来源：CPA《财务成本管理》教材（经营性负债 vs 有息负债分类框架）
> 置信度：实务经验

[场景：负债结构分析——同样是负债，"欠供应商/预收客户"和"欠银行"的性质完全不同。经营性负债不花利息，有息负债要付利息]

**公式**：

```
核心经营性负债占比 = (应付账款 + 应付票据 + 合同负债) / 负债总计 × 100%
```

⚠️ 此处"核心经营性负债"仅取供应链直接相关的三项（应付账款+应付票据+合同负债），用于衡量供应链话语权对负债结构的贡献。完整的经营性负债还包含应付职工薪酬、应交税费、其他应付款等，但这些与供应链议价能力无关，故不纳入本指标。

**分类逻辑**：

| 类型 | 科目 | 本质 |
|---|---|---|
| 经营性负债 | 应付账款、应付票据、合同负债、应付职工薪酬、应交税费 | 经营活动自然形成，不付利息，是企业话语权/经营能力的体现 |
| 有息负债 | 短期借款、长期借款、应付债券、租赁负债 | 主动融资形成，需付利息，是杠杆风险的来源 |

**判定标准**：

| 经营性负债占比 | 含义 |
|---|---|
| > 50% | 负债结构健康——靠经营能力而非借钱支撑 |
| 30%~50% | 正常 |
| < 30% | 负债以有息为主——杠杆驱动，利息负担重 |

**异常判定**：
- 占比持续下降（3 期+）且无合理理由：标记【产业链地位下滑或杠杆替代】
- 占比 < 30% 且利息保障倍数 < 3：标记【负债结构恶化高风险】

⚠️ 占比下降不一定是负面——也可能是企业主动缩短付款周期（提前付供应商争取折扣），或合同负债减少（预收转收入正常结转）。需结合具体原因判断方向。

⚠️ 行业差异大：消费品/白酒行业经营性负债占比天然高（先款后货）；资本密集型行业（电力/航空/房地产）有息负债占比天然高，不代表不健康。

## 1.5 行业财务指标参考区间

> 类型：判断表
> 来源：上市公司公开财报统计（A股各行业中位数口径，2023-2024年度）+ 实务经验
> 置信度：实务经验
> 关联：tax_risk_indicators.md - 7.6 分行业预警税负率参考（税负率视角）

[场景：分析报表时判断"这个毛利率/净利率/周转率正不正常"——给出行业对标基准，而不是只说"偏高/偏低"]

⚠️ **以下数据为行业中位数口径的经验参考值，非官方标准。** 同一行业内细分差异巨大（如医药行业创新药 vs 仿制药毛利率差 40 个百分点），使用时必须结合企业具体业务模式判断。数据来源为 A 股上市公司公开财报统计，非上市企业（尤其中小企业）可能偏低。

|----------|----------|----------|------|

| 商业银行 | 拨备覆盖率 | 150%-300% | 监管红线 150% |
| 商业银行 | ROE | 8%-14% | 近年持续下行 |
| 证券 | ROE | 5%-12% | 牛市可达 15%+ |

| 保险 | 新业务价值率（寿险） | 20%-40% | 衡量保单质量 |

| 细分行业 | 毛利率 | 净利率 | 关键特征指标 |
|----------|--------|--------|-------------|

|----------|--------|--------|-------------|

### 使用规则

1. **先定行业**：确认企业所属细分行业（不是大类），再对标具体区间
2. **容差判断**：企业指标在区间内 ±20% 范围内属正常；超出 30% 以上需查原因
3. **趋势优先**：单期偏离不一定异常，连续 3 期偏离同方向才是信号
4. **结合业务模式**：同一行业内，代工 vs 品牌、轻资产 vs 重资产差异巨大
5. **周期性行业**：煤炭/钢铁/航运/生猪等强周期行业，看周期位置比看绝对值重要
6. **非上市企业折扣**：中小民企毛利率通常比上市公司低 5-10 个百分点（规模效应差异）

### 异常判定标准

| 偏离程度 | 判定 | 行动 |
|----------|------|------|
| 指标在行业区间内 | 正常 | 无需特别关注 |
| 偏离区间 20%-30% | 需解释 | 查看是否有合理业务原因（产品结构/区域/客户变化） |
| 偏离区间 30%-50% | 中度异常 | 深入分析，结合趋势和同业判断 |
| 偏离区间 > 50% | 高度异常 | 高风险信号，需重点核查（可能是错报/造假/业务模式根本不同） |
| 毛利率显著高于行业但现金流差 | 强烈造假信号 | 转 financial_fraud.md 深度分析 |

## 1.6 财务分析方法论与判断框架

> 来源：《财务报表分析》（199页教材蒸馏）+ 实务经验
> 性质：方法+判断

### 同型分析（结构分析 / 共同比）

> 类型：方法
> 来源：《财务报表分析》教材蒸馏
> 置信度：通用方法

[场景：把绝对金额变成占比，看清"钱从哪来、到哪去、每步损耗多少"——比直接看金额直观]

**算法**：
- 利润表同型：每个项目 ÷ 营业收入 × 100%（看从收入到利润每一步损耗）
- 资产负债表同型：每个项目 ÷ 资产总计 × 100%（看资产结构和资金来源结构）

**三步叠加才有结论**：
1. 单期结构：本期各项占比
2. 趋势分析：与企业自身历史期对比，看变化方向
3. 比较分析：与竞争对手 / 行业标杆 / 行业均值对比

**反推示例**：某企业固定资产占总资产 60%、应收账款占 14% → 资金密集型制造业，且应收占比偏高说明产品不好卖、面临竞争压力、对下游议价弱。

⚠️ 单期结构本身不说明问题，必须叠加趋势或比较才能判断是否异常。

### 杜邦分析（ROE 三层分解）

> 类型：公式
> 来源：《财务报表分析》教材蒸馏
> 置信度：通用方法

[场景：ROE 上升了，要拆清楚是靠经营变好还是靠加杠杆——两者风险天差地别]

**公式**：

```
ROE（净资产收益率）= 净利率 × 总资产周转率 × 权益乘数
                  = ROA（总资产报酬率）× 权益乘数

权益乘数 = 资产总计 / 股东权益 = 1 / (1 - 资产负债率)
```

**三因子含义**：
- 净利率 = 效益（每元收入赚多少）
- 总资产周转率 = 效率（资产转几圈）
- 权益乘数 = 杠杆（负债放大倍数）

**判断规则**：
- ROE 上升靠净利率/周转率提升 → 经营改善，健康
- ROE 上升靠权益乘数（加负债）→ 加杠杆驱动，风险累积

⚠️ 靠多借钱抬高权益乘数来提 ROE 是无效的——除非借来的钱投资回报 ≥ 原 ROA。把借款放银行（存款利率 < 贷款利率）必然拉低净利率，反而降 ROE。决定股东回报的根本是 ROA（整体投资回报），不是负债水平。

**示例**：ROA 10% × 权益乘数 2（资产负债率 50%）= ROE 20%；若资产负债率升到 67%，权益乘数变 3，但若新增负债回报跟不上导致 ROA 降到 7%，则 ROE = 21%，几乎没改善还更脆弱。

### 利息保障倍数（EBIT 版）

> 类型：公式
> 来源：《财务报表分析》教材蒸馏
> 置信度：通用方法
> 关联：本文件「现金利息保障倍数」（现金版，互补使用）

[场景：长期偿债能力的本金没法直接算，但利息能算——看息税前盈利够还几次利息]

**公式**：

```
EBIT（息税前收益）= 净利润 + 所得税 + 利息费用（实务常用财务费用近似）
利息保障倍数 = EBIT / 利息费用
```

**为什么分子要加回所得税**：净利润已扣利息和所得税；若扣息后不盈利则不缴税，所以能用于还息的钱包含了那部分待缴所得税，必须加回。

**判定**：倍数越高偿息能力越强；与现金版配合使用。

⚠️ EBIT 版是权责发生制——有利润不等于收到现金。当 EBIT 版很高但现金利息保障倍数（CFO/实际付息）< 3 时，说明"有利润没现金"，优先信现金版。

**示例**：净利润 520 + 所得税 100 + 财务费用 130 = EBIT 750；750 / 130 = 5.8 倍。

### 流动比率 / 速动比率的真实判断标准

> 类型：判断
> 来源：《财务报表分析》教材蒸馏
> 置信度：实务经验

[场景：修正"流动比率 > 1 就安全""= 2 最理想"的误区——尤其中国企业]

**公式**：

```
流动比率 = 流动资产 / 流动负债
速动比率 =（流动资产 - 存货）/ 流动负债
剔除短期借款的流动比率 = 流动资产 /（流动负债 - 短期借款）
```

**判断标准修正**：

⚠️ 流动比率 = 1：营运资金不足。流动资产身兼两职（还债 + 维持日常运营），全拿去还债企业就停摆。

⚠️ 流动比率 = 2 不等于安全：美国破产企业在破产前一年的流动比率恰好≈2；美国健康企业多在 3~4。

⚠️ 中国企业流动比率普遍 1~2 仍健康：靠"借新还旧"滚动短期借款，短期借款实质是长期借款的替代，不需要靠变现流动资产偿还。

**实操**：分析中国企业时，算「剔除短期借款的流动比率」更真实——它反映真正必须靠变现流动资产偿还的压力。示例：流动资产 3670、流动负债 5900（含短期借款 5100），表面流动比率 0.6，剔除短期借款后 = 3670 /（5900-5100）= 4.6，并不低。

### 波特五力 + 行业烙印反推法

> 类型：方法
> 来源：《财务报表分析》教材蒸馏（Porter 五力模型）
> 置信度：通用方法

[场景：反过来用——从财报数字倒推企业所处的行业竞争格局和产业链地位]

**五力**：现有竞争者、新进入者威胁、替代品威胁、供应商议价能力、购买方议价能力。

**财报指标 → 行业格局反推映射**：

| 财报指标 | 反映什么 | 判读 |
|---|---|---|
| 毛利率 | 行业竞争激烈程度 | 竞争越烈毛利越低；毛利逆势走低 = 竞争加剧 |
| 应收账款 + 存货占比 | 与下游购买方的议价关系 | 占比高 = 下游强势、企业被迫赊销/压货 |
| 应付账款占比 | 对上游供应商的议价能力 | 占比高 = 对上游强势、能免费占用供应商资金 |
| 固定资产占比 | 行业资本密集度 | 占比高 = 重资产制造业 |
| 总资产周转率 | 管理/运营效率 | 持续走低 = 存货积压 + 应收难收 |

**判断动作**：先用这 5 个指标给企业"画行业像"，再验证是否与已知行业特征一致；不一致则深挖原因。

**示例（造纸 vs 彩电，2002 年）**：造纸毛利率 28%、应付账款仅 8%（上游木浆供应商强势）、固定资产占 62%（每万吨产能投资 1 亿）；彩电毛利率 15%（竞争白热化）、应收 23% + 存货 39%（下游渠道商强势 + 产品滞销）。财报数字精准刻画了两个行业的竞争格局。

### 战略定位判断（成本领先 vs 差异化）

> 类型：判断
> 来源：《财务报表分析》教材蒸馏
> 置信度：通用方法

[场景：用净利率和周转率的组合，判断企业走的是哪条战略路线，以及执行得好不好]

**用「净利率 × 周转率」组合判断**：

| 战略 | 净利率/毛利率 | 周转率 | 本质 |
|---|---|---|---|
| 成本领先（效率型） | 低 | 高 | 薄利多销 |
| 差异化（效益型） | 高 | 低 | 高附加值、慢周转 |

**判断逻辑**：两条路都能到高 ROA（ROA = 净利率 × 周转率），关键看战略执行是否到位。

**示例（苹果 vs 华为，2017 财年）**：苹果净利率 21.2% / 总资产周转 0.6 次 = 典型差异化；华为净利率约 7% / 周转 1.2 次 = 效率型。两者毛利率都约 40%，但苹果销售管理费用率仅 6.7%（to C 直营 + 强品牌），华为 15.4%（to B 需长期客户维护）；最终苹果 ROA 12.9% > 华为 9.4%。

⚠️ 战略错配是执行失败信号：低毛利行业却铺大额研发/广告、或高毛利定位却拼命压价冲量，导致 ROA 上不去——说明既没拿到效益也没拿到效率。

### 经济利润判断"真正赚钱"（ROIC / WACC）

> 类型：公式
> 来源：《财务报表分析》教材蒸馏（EVA 经济增加值思想）
> 置信度：通用方法

[场景：净利润 > 0 不代表真赚钱——只有跑赢股东的机会成本才算为股东创造价值]

**公式**：

```
ROIC（投资资本回报率）= 税后营业净利润 / 投资资本
投资资本 = 有息负债 + 股东权益
WACC（加权平均资本成本）=（有息负债/投资资本）× 债务成本 ×（1 - 所得税率）
                       +（股东权益/投资资本）× 权益成本
经济利润 =（ROIC - WACC）× 投资资本
```

**关键参数**：
- 权益成本用「所处行业平均盈利水平」近似（股东的机会成本——这笔钱投同行能赚多少）
- 利息税盾：实际债务成本 = 贷款利率 ×（1 - 所得税率）。例：利率 7%、税率 25% → 实际成本 5.25%

**判定**：
- 经济利润 > 0 → 真正赚钱，为股东创造了价值
- 经济利润 < 0 → 即使净利润为正，也跑输行业平均，没为股东赚到钱

⚠️ "净利润 > 0 但经济利润 < 0"很常见：净利润只扣了债权人成本（利息），没扣股东的机会成本。扣掉股东资本成本后为负，说明股东把钱投别处更划算，这家公司处于行业下游水平。

### 三表勾稽核心恒等式（利润-现金背离的根因）

> 类型：公式
> 来源：《财务报表分析》教材蒸馏
> 置信度：通用方法
> 关联：本文件「净利润含金量」「现金流四象限分类」

[场景：一句话解释"为什么赚钱却没钱/有钱却不赚钱"——差额必然落在某个资产负债科目上]

**恒等式**：

```
净利润 - 现金净流量 = 非现金资产及负债的变化

经营层细化：
经营利润 - 经营活动现金流 = 经营性非现金资产负债的变化
                          =（应收账款↑ + 存货↑ + 预付↑）-（应付↑ + 预收↑）
```

**用途**：净利润高但 CFO 低时，差额一定能在资产负债表上找到落脚点。

**判断动作**：拆解差额落在哪个科目——
- 落在应收账款大增 → 收入收不回来，收入质量存疑
- 落在存货大增 → 产品卖不动，减值风险
- 落在应付/预收减少 → 占用上下游资金能力下降

**示例**：花 50 万买 10 年期设备 → 现金流量表当年 -50 万，利润表不体现（资本化进资产负债表），差额 50 万 = 固定资产增加；此后每年折旧 5 万进利润表但不动现金，差额 -5 万 = 非现金资产减少。若改为花 50 万投广告 → 现金流量表和利润表同步 -50 万，资产负债表不变，差额为 0。

### 利润 vs 现金流：谁更重要的判断规则

> 类型：判断
> 来源：《财务报表分析》教材蒸馏
> 置信度：实务经验

[场景：两家利润和现金流数字几乎一样的公司，结论可能完全相反——取决于风险]

**硬规则**：
- 企业面临巨大风险（行业被替代、竞争白热化、宏观下行）→ 现金流优先，先看能不能活下来
- 经营风险可控（寡头垄断、订单锁定、定制化有合同支撑）→ 利润优先，关注收益
- 同一企业不同周期会切换：经济差时关注现金流，经济好时关注利润

**底层逻辑**：利润表 + 资产负债表的价值前提是「资产信息准确」。当应收收不回、存货卖不掉、固定资产过时 → 资产减值 → 利润信息失真 → 此时现金流量表更可信。

**对照示例**：
- A 公司（电子书阅读器）：净利润 8000 万但 CFO -3.2 亿，应收 +1.5 亿、存货 +3.5 亿。行业遭智能手机替代，资产面临减值 → 利润不可信，信现金流。结果次年亏损 5 亿。
- B 公司（通信设备）：净利润 9 亿但 CFO 为负十几亿，应收 +30.6 亿、存货 +3.7 亿。但客户是电信运营商（赖账概率低）、定制化生产有订单支撑（存货不愁卖）→ 资产价值有保证，利润可信。结果次年利润增至 14.5 亿、CFO 转正。

⚠️ 现金流相似的两家公司，必须结合行业风险特征和资产质量判断，不能只看现金流数字下结论。

### 现金流类型分析的关注重点（补充八象限）

> 类型：判断
> 来源：《财务报表分析》教材蒸馏
> 置信度：实务经验
> 关联：本文件「现金流四象限分类」（补全其余两种组合 + 各类型关注重点）

[场景：补全 CFO/CFI/CFF 的 8 种正负组合，并明确每种类型该重点盯什么]

**补全前表未列的两种组合**：

| 组合 | 类型 | 含义 | 关注重点 |
|---|---|---|---|
| CFO+ / CFI+ / CFF+ | 三正囤现金 | 经营赚钱、变卖资产、还在融资 | 钱用不出去（无投资计划）会稀释盈利；警惕融资动机（是否转关联方掏空） |
| CFO- / CFI- / CFF- | 三负危急 | 经营失血、还在投资、还在还债 | 离破产不远，无外部输血撑不住，优先排除 |

**各类型关注重点规则**：
- 投资活动现金流为负（扩张型）→ 重点盯投资的规模和节奏，防资金链断裂（投资有周期，效益未出前持续流出现金）。即便投资方向正确，扩张过快也会断链（某非油炸方便面案例：主业正常、市占率不降，仍因扩张太快资金链断裂破产）。
- 投资活动现金流为正 → 区分来源：处置资产（不可持续，可能在变卖差资产）vs 投资收益（再分金融性收益 = 运气不可持续 / 实业分红 = 可持续）。
- 经营活动现金流为负 → 多见于初创期（竞争力不足）或衰退期（产品滞销/收不回款）。
- 净现金流（CFO+CFI+CFF 之和）为负 → 企业在消耗现金存量，资金链吃紧，需重审投资规模节奏。

⚠️ 转型要趁早：衰退企业（CFO- / CFI- / CFF+）已难获银行和股东支持；应在经营现金流还为正（CFO+ / CFI- / CFF+）时就启动转型，路才好走。

### 历史成本计价的两层含义与虚增资产识别

> 类型：判断
> 来源：《财务报表分析》教材蒸馏
> 置信度：实务经验
> 关联：financial_fraud.md（虚增资产造假深度分析）

[场景：理解历史成本原则，是尽调中倒查大额资产真实性的核心武器]

**两层含义**：
1. 只有实际花出去的钱才能入账——成本以历史支出为限
2. 历史成本下，提高资产账面价值的唯一合法途径是发生一笔新交易（按公允价值计量的金融资产、投资性房地产除外，普通资产只减不增——升值当没看见，减值要记下来）

**识别动作**：
- 资产账面金额 >> 可核实的实际投入 → 虚增嫌疑（某林浆纸案例：账面 6 亿元林木资产，实际种树仅花 3000 万元，5.7 亿元为虚增；土地系政府无偿划拨本不该计入大额成本）
- 同一资产"先卖出、隔期再买回" → 借新交易把账面价值抬上去的操纵手法

⚠️ 收购/尽调中，对大额资产务必用历史成本原则倒查真实投入凭证；账面价值与实际支出严重背离的，按虚增资产高风险处理。

## 二、策略库系统

### 内置策略列表

| 策略名称     | 策略ID              | 功能描述                                 | 适用税种           |
| ------------ | ------------------- | ---------------------------------------- | ------------------ |
| 税款比对分析 | tax_reconciliation  | 比对已纳税款与财务报表，识别少缴税款疑点 | 增值税、企业所得税 |
| 收入确认审查 | revenue_recognition | 基于 CAS 14 审查收入确认合规性           | 增值税、企业所得税 |
| 成本操纵识别 | cost_manipulation   | 识别通过成本费用调节利润的行为           | 企业所得税         |
| 税收优惠审查 | tax_preference      | 审查税收优惠政策适用条件和计算准确性     | 企业所得税         |
| 关联交易审查 | related_party       | 审查关联方交易税务合规性                 | 企业所得税、增值税 |

### 策略一：税款比对分析 (tax_reconciliation)

**功能**：通过比对财务报表与纳税申报数据，自动识别少缴税款疑点

**比对维度**：

1. **收入比对**：财务收入 vs 增值税/所得税申报收入
2. **税负率分析**：实际税负率 vs 行业参考值
3. **限额扣除检查**：福利费、招待费、广宣费等超标识别
4. **理论税额测算**：基于财务数据测算理论应纳税额

**预警规则**：

- 财务收入 > 增值税申报收入 × 1.05 → 高风险
- 实际增值税税负率 < 行业平均 × 0.7 → 中风险
- 业务招待费 > MIN(发生额×60%, 收入×0.5%) → 超标

### 策略二：收入确认审查 (revenue_recognition)

**功能**：基于 CAS 14 审查收入确认合规性，识别提前/虚构收入

**审查要点**：

- 收入确认是否满足五步法（识别合同→识别履约义务→确定交易价格→分摊→确认收入）
- 是否存在提前确认（发货未签收即确认）
- 是否存在虚构收入（无真实交易背景）
- 退货率是否异常
- 收入与现金流是否匹配

**异常信号**：

- 季末/年末收入占比异常高（>40%）
- 新客户贡献大额收入但无后续交易
- 收入增长但经营现金流未同步增长
- 应收账款增速远超收入增速

### 策略三：成本操纵识别 (cost_manipulation)

**功能**：识别通过成本费用调节利润的行为

**审查要点**：

- 毛利率异常波动（同比变动 > 5个百分点需关注）
- 成本结转方法变更
- 期间费用突然大幅变动
- 研发费用资本化比例异常
- 折旧政策变更

**异常信号**：

- 毛利率逆行业趋势上升
- 管理费用/销售费用突然大幅下降
- 在建工程长期不转固
- 研发资本化率远高于同行

### 策略四：税收优惠审查 (tax_preference)

**功能**：审查企业享受的税收优惠政策是否符合条件，关注政策时效性和业务实质

**审查维度**：

1. **小型微利企业优惠**：应纳税所得额≤300万、从业人数≤300人、资产总额≤5000万
2. **高新技术企业**：研发费用占比、高新技术产品收入占比、科技人员占比、证书有效性
3. **研发费用加计扣除**：负面清单行业限制、研发费用归集准确性、委托研发比例
4. **固定资产加速折旧/一次性扣除**：500万以下设备器具、资产类型合规性
5. **区域性优惠**：西部大开发（鼓励类产业、收入占比60%）、海南自贸港
6. **政策时效性**：检查优惠政策是否已过期或即将到期

### 策略五：关联交易审查 (related_party)

**功能**：审查关联方交易的税务合规性，防范资本弱化、转让定价和隐性利润分配风险

**审查维度**：

1. **资本弱化审查**：债资比2:1（一般企业）或5:1（金融企业），超标利息不得扣除
2. **关联方利息审查**：利率合理性（基准利率区间）、非金融企业借款凭证合规性
3. **关联方资金占用**：长期占用未收利息、大额资金往来异常
4. **转让定价审查**：关联方销售/采购毛利率差异、定价公允性
5. **关联方担保**：未收担保费、担保实际履行损失
6. **向关联方发放福利**：视同销售处理
7. **同期资料义务**：本地文档（关联交易2亿+）、主体文档（营收55亿+）

## 九、财务造假与财务洗澡识别

> 类型：流程
> 来源：见 financial_fraud.md
> 置信度：实务经验

财务造假识别（动机、高风险科目、造假技术、掩护机制、反舞弊方法、Red Flags、洗澡 vs 造假、恒大/五粮液案例）已统一存放，本文件不重复。

> 详见：financial_fraud.md

报表审查中触发以下情形时，转 financial_fraud.md 做深度判断：

- 利润高但经营现金流持续为负 / 收现比、净现比长期 < 1 → 造假方向核查
- 利润断崖式下跌但现金流基本正常、且伴随换帅/被查/行业下行 → 洗澡方向核查
- 收入确认口径变更但不追溯调整、新增大额过渡科目 → 口径切换洗澡核查
- 应收/存货增速显著超过收入增速、毛利率显著偏离同行 → 收入造假核查

本文件第四章"税务风险识别"中的收入类、成本费用类风险是从税务合规角度切入，与 financial_fraud.md 的造假识别角度互补。

## 十一、重要提示

1. **仅供专业参考**：审查意见基于预设规则和法规库生成，需由持证会计师或税务师复核
2. **法规时效性**：法规依据需确认是否有最新修订
3. **专业判断**：风险识别和会计处理建议需结合具体业务实质判断
4. **保密要求**：财务数据涉及商业秘密，请确保在安全环境中使用
5. **洗澡 ≠ 违法**：财务洗澡是盈余管理行为，形式上合规但损害信息可比性；财务造假是违法行为。两者的识别方法不同，后果不同，但都需要投资者警惕
