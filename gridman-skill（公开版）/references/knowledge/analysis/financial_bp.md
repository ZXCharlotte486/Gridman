## 一、财务BP定位与能力模型

| 维度 | 传统财务 | 财务BP |
|------|---------|--------|
| 定位 | 事后核算、监督合规 | 事前预测 + 事中控制 + 事后复盘闭环 |
| 时间视角 | 过去（历史数据） | 未来（滚动预测、场景模拟） |
| 核心产出 | 报表、凭证、纳税申报 | 经营分析报告、ROI测算、定价建议、行动方案 |
| 沟通对象 | 税务局、审计师、外部监管 | 业务总监、产品经理、销售团队、CEO |
| 语言 | 会计语言（借贷/科目） | 业务语言（量价/渠道/客户/产品） |
| 价值体现 | 数字准确 | 影响决策 |

**BP 价值金字塔**（从基础到战略）：
1. 基础级：数据清洗、报表出具、流程标准化
2. 业务级：合同审核、定价支持、成本优化
3. 经营级：预算编制、滚动预测、经营分析会主持
4. 战略级：参与中长期规划、新业务可行性、并购财务尽调

## 二、经营分析方法论

### 2.1 月度经营分析标准结构

```
1. 经营结果概览（1页）
   - 收入/利润/现金流 vs 预算 vs 去年同期
   - 红绿灯标识：达标/预警/未达标

2. 收入分析（量价拆分）
   - 总收入变动 = 量变动 + 价变动 + 结构变动
   - 按产品线/客户/渠道/区域拆分

3. 毛利分析（归因到驱动因素）
   - 毛利率变动 = 售价变动 + 成本变动 + 产品结构变动
   - 标准成本 vs 实际成本差异（量差/价差/效率差）

4. 费用分析（对标+归因）
   - 费用率 vs 预算 vs 行业benchmark
   - 超支归因：量的问题 or 单价问题 or 一次性

5. 现金流与营运资本
   - DSO/DPO/DIO 变动趋势
   - 回款预警、逾期账款风险

6. 重点项目追踪
   - 新产品上市进度及预算执行
   - 降本增效项目效果

7. 风险与行动计划
   - 下月/下季度关键风险点
   - 可行动的改善建议（不超过3条）
```

### 2.2 Revenue Bridge（收入桥）

> 类型：公式
> 来源：FP&A 通用方法论（CFI/Wall Street Prep/实务经验综合）
> 置信度：实务经验

[场景：老板问"收入为什么比上月/预算多了500万？"——拆解出每个驱动因素贡献了多少]

#### 基础版：量 × 价

```
收入变动 = 量变动 + 价变动

量变动 = (本期销量 - 基期销量) × 基期单价
价变动 = (本期单价 - 基期单价) × 本期销量

⚠️ 交叉项处理：量差用基期价、价差用本期量（Laspeyres-Paasche 混合法）
   关键是全公司统一口径，不要A产品线用这种B产品线用那种。
```

#### 进阶版：量 × 价 × 结构（PVM分析）

```
收入变动 = 量效应 + 价格效应 + 结构效应（+ 新品/退出效应）

量效应（Volume）= 总销量变动 × 基期加权平均单价
价格效应（Price）= Σ(各产品价格变动 × 各产品本期销量)
结构效应（Mix）= 总收入变动 - 量效应 - 价格效应

新品效应 = 本期有、基期无的产品收入
退出效应 = 基期有、本期无的产品收入（负数）
```

#### SaaS/订阅制 Revenue Bridge

```
ARR变动 = 新客ARR + 扩展ARR - 流失ARR - 降级ARR

瀑布：期初ARR → +新客 → +扩展 → -降级 → -流失 → 期末ARR
```

#### Revenue Bridge 汇报模板

```
"本月收入 1,200万，较上月增长 200万（+20%），拆解如下：
 ├── 量效应 +150万：A产品销量增加 2,000 件（大客户B新增采购）
 ├── 价格效应 +30万：C产品调价 +5%（覆盖原材料涨价）
 ├── 结构效应 +50万：高毛利D产品占比从25%升至32%
 └── 退出效应 -30万：E产品停产（库存清完）

→ 结论：增长健康，核心来自量的增长（非靠降价换量），结构优化是额外加分项。"
```

### 2.3 毛利桥（Gross Margin Bridge）

> 类型：公式
> 来源：FP&A 实务
> 置信度：实务经验

```
毛利变动 = 量效应 + 售价效应 + 成本效应 + 结构效应

量效应 = (本期量 - 基期量) × 基期单位毛利
售价效应 = (本期单价 - 基期单价) × 本期销量
成本效应 = (基期单位成本 - 本期单位成本) × 本期销量  ← 成本下降为正
结构效应 = 实际毛利 - (量效应 + 售价效应 + 成本效应 + 基期毛利)
```

### 2.4 费用归因三步法

> 类型：流程
> 来源：FP&A 实务
> 置信度：实务经验

**第一步：分类——固定 vs 变动 vs 半变动**

| 类型 | 特征 | 示例 | 与收入的关系 |
|------|------|------|------------|
| 固定费用 | 短期内不随业务量变化 | 房租、基本工资、折旧、保险 | 费用率随收入增长而下降 |
| 变动费用 | 随业务量线性变化 | 佣金、物流费、包装费、云服务按量计费 | 费用率相对稳定 |
| 半变动费用 | 有基础+弹性部分 | 差旅、招待、研发（有底线投入+项目弹性） | 阶梯式变化 |

**第二步：拆解——超支是"量"还是"单价"**

```
费用超支 = 量超 + 单价超 + 一次性

量超 = (实际业务量 - 预算业务量) × 预算单位费用
单价超 = (实际单位费用 - 预算单位费用) × 实际业务量
一次性 = 预算外的非经常项目（如搬家费、诉讼费）
```

**第三步：判断——可控 vs 不可控 → 行动建议**

| 类型 | 可控？ | BP建议 |
|------|--------|--------|
| 量超（业务增长带来） | 半可控 | 验证费用增速是否<收入增速（经营杠杆） |
| 单价超（效率下降/采购涨价） | 可控 | 找原因：供应商涨价 or 内部浪费 |
| 一次性（非经常） | 不可控 | 标注非经常，不纳入趋势分析 |
| 结构变化（新业务带来） | 半可控 | 对应收入是否已起来？ |

### 2.5 成本差异分析（标准成本法）

> 类型：公式
> 来源：CPA《财务成本管理》教材 + 制造业实务
> 置信度：实务经验

#### 直接材料差异

```
材料价差 = (实际单价 - 标准单价) × 实际用量     ← 采购部门负责
材料量差 = (实际用量 - 标准用量) × 标准单价     ← 生产部门负责
```

#### 直接人工差异

```
工资率差异 = (实际工资率 - 标准工资率) × 实际工时    ← HR/薪酬
效率差异 = (实际工时 - 标准工时) × 标准工资率       ← 生产管理
```

#### 固定制造费用差异（三差异法）

```
支出差异 = 实际固定制造费用 - 预算固定制造费用
产能差异 = (预算产能 - 实际产能) × 标准分配率    ← 产能闲置的代价
效率差异 = (实际工时 - 标准工时) × 标准分配率    ← 生产效率
```

#### 差异归责表

| 差异类型 | 归责部门 | 常见原因 |
|----------|---------|---------|
| 材料价差 | 采购 | 供应商涨价/未按合同采购/紧急采购溢价 |
| 材料量差 | 生产 | 废品率高/BOM不准/物料浪费 |
| 工资率差 | HR/生产 | 加班过多/用高级工做简单活 |
| 人工效率差 | 生产 | 设备故障/工艺问题/培训不足 |
| 产能差异 | 销售/生产 | 订单不足/排产不合理/设备维修 |

### 2.6 Driver-Based Model（驱动因子建模）

> 类型：流程
> 来源：CFI FP&A Modeling Best Practices + IBM + Metapraxis
> 置信度：实务经验

**核心思想**：财务数字 = f(业务驱动因子)。先建立因果关系，再通过预测驱动因子推导财务预测。

| 行业 | 收入驱动公式 | 关键驱动因子 |
|------|-------------|-------------|
| SaaS | ARR = 客户数 × ARPU × 留存率 | 新签客户、流失率、扩展率 |
| 零售 | 收入 = 门店数 × 单店客流 × 转化率 × 客单价 | 同店增长、新开店计划 |
| 制造 | 收入 = 产能 × 利用率 × 良品率 × 单价 | 订单量、开工率、产品结构 |
| 广告 | 收入 = DAU × 广告加载率 × eCPM | 用户增长、变现效率 |
| 物流 | 收入 = 票数 × 单票收入 | 市场份额、定价策略 |
| 教育 | 收入 = 学员数 × 单价 × 续费率 | 获客、转化、续费 |
| 酒店 | 收入 = 客房数 × 入住率 × ADR | 季节性、渠道占比 |

### 2.7 营运资本与 DSO 诊断

```
DSO = 应收账款余额 ÷ (年化收入 ÷ 365)
DPO = 应付账款余额 ÷ (年化采购额 ÷ 365)
DIO = 存货余额 ÷ (年化营业成本 ÷ 365)
CCC = DSO + DIO - DPO
```

⚠️ 默认使用365天。银行利息计算/审计底稿沿用被审计单位口径时用360天。

**DSO恶化超5天时触发跟进**：数据穿透排查 → 业务协同管控 → 风险评估 → 流程优化 → 专项改善。

## 三、预算与预测

### 3.1 预算编制方法

| 方法 | 适用场景 | 优点 | 缺点 | 典型行业 |
|------|---------|------|------|---------|
| 增量预算 | 业务稳定 | 简单快速 | 容易固化 | 公用事业、成熟制造业 |
| 零基预算（ZBB） | 费用管控/组织瘦身 | 每笔支出必须证明价值 | 极耗时 | 快消、互联网（优化期） |
| 滚动预测 | 业务变化快 | 持续更新不过时 | 需纪律 | SaaS、电商、初创 |
| 弹性预算 | 收入波动大 | 按收入档位设费用上限 | 设计复杂 | 广告、大宗贸易 |
| 驱动因子预算 | 业务逻辑清晰 | 可解释"为什么" | 需准确数据 | SaaS、零售连锁 |

### 3.2 滚动预测 12+3 实操

```
每月5日前更新：
├── 已完成月份：替换为实际数
├── 当月：最新估计
├── 未来2个月：按月预测（高颗粒度）
├── 未来3-12个月：按月预测（可适度粗放）
└── 未来13-15个月：按季预测（始终多看3个月）
```

**预测准确率考核**：提前1月≥95%、提前3月≥85%、提前6月≥75%、提前12月≥60%。

**组织纪律**：1-3日数据收集 → 3-5日预测更新 → 5-7日差异分析 → 8-10日管理层汇报。

### 3.3 预算差异分析

**差异分类**：量差 / 价差 / 结构差 / 时间差 / 一次性

**"CFO 真正想看的"格式**：

```
"管理费用超预算 120万（+15%）：
 ├── +80万（量驱动）：新招15人超HC计划
 ├── +50万（一次性）：办公室搬迁
 └── -10万（有利）：差旅节约

 → 剔除一次性后实际超 70万（+8.7%），费用增速<收入增速 → 经营杠杆正向。"
```

**决策阈值**：±5%仅记录 / ±5-10%一句话解释 / ±10-20%归因+行动 / ±20%+专题汇报

## 四、业务决策支持

### 4.1 定价决策

```
成本加成法：售价 = 成本 × (1 + 目标加成率)
目标毛利法：售价 = 成本 ÷ (1 - 目标毛利率)
边际贡献法：只要售价 > 变动成本，短期可接受（产能闲置时）
```

**降价换量公式**：
```
维持原利润所需销量增幅 = 原边际贡献率 / 新边际贡献率 - 1

例：原售价100，变动成本60，边际贡献率40%
    降价10%后：售价90，边际贡献率33.3%
    需增量 = 40%/33.3% - 1 = 20%
```

### 4.2 投资决策

输出：盈亏平衡点 / 回收期 / NPV / IRR / 敏感性分析

决策标准：IRR > WACC → 可投；回收期 < 行业惯例 → 可接受；NPV > 0 → 创造价值

### 4.3 投资测算关键假设检查清单

**通用规则：假设缺少关键项 → 先反问再计算，不能用理想值默认填充。**

| 行业 | 必须确认的假设 | 不能默认100% | 合理区间 |
|------|---------------|-------------|---------|
| 餐饮/零售 | 上座率/转化率/爬坡期 | 不能假设满座 | 餐饮60-80%，零售15-30% |
| 制造业 | 产能爬坡/良品率/有效生产天数 | 不能假设满产 | 6-12月达80%，良品率90-98% |
| SaaS | CAC趋势/留存曲线/基础设施弹性 | 不能假设恒定 | CAC年升15-30% |
| 金融 | 不良率/赔付率/资本约束 | 不能假设零坏账 | 不良1-3% |
| 医药 | 成功概率/审批周期/专利悬崖 | 不能假设100%成功 | I期→上市10-15% |
| 房地产 | 去化周期/融资成本/预售监管 | 不能假设即清盘 | 住宅6-18月 |
| 物流 | 装载率/油价波动 | 不能假设满载 | 整车85-92% |
| 跨境电商 | 日均单量/ACoS/退货率/滞销比 | 不能假设爆单 | 新品日均5-20单 |

### 4.4 客户盈利分析

按客户排序通常呈现"鲸鱼曲线"：前20%客户贡献150%利润，后30%客户实际亏损。行动：淘汰/提价/降低服务等级。

### 4.5 场景分析

三种情景（乐观/基准/悲观），每种给出收入/利润/现金流预测 + 风险点 + Plan B。

## 五、非财务指标 → 财务影响桥接

> 类型：公式+判断表
> 来源：FP&A 实务
> 置信度：实务经验

[场景：业务说"留存率降了1个点"，老板问"所以影响多少钱？"]

**通用公式**：财务影响 = 受影响基数 × 变动幅度 × 传导系数 × 持续时间

### 5.1 用户留存率下降（SaaS）

```
年度收入损失 ≈ 期初MRR × 12 × 留存率降幅 × 6（平均影响半年）

示例：MRR 500万，月留存率从95%降到94%
年度损失 ≈ 500万 × 12 × 1% × 6 = 360万
```

### 5.2 员工流失率上升

```
年度额外成本 = 员工总数 × 流失率增量 × 单人流失成本
单人流失成本 = 招聘成本(1-3月薪) + 培训期损失 + 空窗期产出损失

示例：200人公司，流失率增5个点 = 多流失10人 × 10万/人 = 100万
```

### 5.3 NPS/满意度下降

```
NPS 每降10分 → 复购率降5-8个百分点（消费品）/ 续约率降3-5个百分点（B2B）
收入影响 = 存量客户收入 × 复购率降幅 × 剩余生命周期

示例：存量8000万，NPS降15分→复购率降10%，3年累计损失≈2000万
```

### 5.4 产品上市延迟

```
延迟损失 = 日均预期收入 × 延迟天数 × 市场窗口系数
  蓝海：0.3-0.5 / 红海：0.8-1.2 / 季节性：1.5-3.0
```

### 5.5 产能利用率下降（制造业）

```
单位固定成本增加 = 固定制造费用 × (1/新利用率 - 1/原利用率)

示例：固定费用2000万，利用率从85%降到70%
年增成本 = (28.6-23.5)元/件 × 70万件 = 357万
```

### 桥接沟通模板

```
"[非财务指标]从X变到Y，对[财务指标]影响约[金额]，
 路径是[因果链]，持续[时间]。建议[措施]+预期挽回[金额]。"
```

## 六、数据驱动决策方法论

> 类型：流程+判断表
> 来源：FP&A 实务 + 数据分析方法论
> 置信度：实务经验

### 6.1 多源数据口径打架

**口径对账四步法**：明确各系统定义 → 画合同到现金漏斗 → 找不可解释差异 → 建月度对账机制

| 差异类型 | 常见原因 | 解决方案 |
|----------|---------|---------|
| CRM > ERP | 合同已签未启动/跨期 | 按履约进度折算 |
| ERP > 台账 | 已确认未回款 | 正常时间差 |
| 台账 > ERP | 预收到账未确认收入 | 检查合同负债 |
| 三个都不对 | 截止日不一致 | 统一口径 |

### 6.2 从100个指标找核心驱动因子

```
第一轮（100→30）：排除滞后/不可归因/纯结果指标
第二轮（30→10）：相关性>0.5，去共线
第三轮（10→3-5）：业务逻辑验证 + 时间领先性 + 可干预性
```

### 6.3 异常检测

| 方法 | 适用 | 标准 |
|------|------|------|
| 历史趋势法 | 稳定业务 | 均值±2σ |
| 同业对标法 | 有行业数据 | IQR×1.5 |
| 预算基准法 | 有预算体系 | ±10-20% |

**分层阈值**：收入±5-10% / 毛利率±2-3pp / 费用率±10-15% / 现金流±20%

### 6.4 相关性 vs 因果性

三个验证方法：时间滞后检验 / 反事实检验 / 控制变量。至少两种方法指向同一方向才作为决策依据。

### 6.5 数据质量自检清单

```
- [ ] 量级合理？币种统一？含税/不含税？
- [ ] 时间覆盖完整？截止日一致？
- [ ] 无明显缺失/断点？金额方向正确？
```

## 八、行业BP深度指南

> 本章为各行业 BP 提供同等深度的经营分析指南。每个行业统一格式：成本结构 → 核心KPI → 经营分析框架 → 关键决策场景。


### 8.1 制造业

#### 成本结构

```
售价
├── 直接材料（通常占40-70%）
│   ├── 主料（BOM标准用量 × 单价）
│   └── 辅料/包装
├── 直接人工（10-20%）
│   ├── 计件工资 or 计时工资
│   └── 加班/夜班津贴
├── 制造费用（15-25%）
│   ├── 折旧（设备/厂房）
│   ├── 能源（水电气）
│   ├── 维修保养
│   └── 车间管理人员
├── 期间费用
│   ├── 销售费用（运输/佣金/售后）
│   ├── 管理费用（行政/研发分摊）
│   └── 财务费用（贷款利息）
└── 净利润
```

#### 核心KPI

| 指标 | 公式 | 健康区间 | BP关注点 |
|------|------|---------|---------|
| 产能利用率 | 实际产量/设计产能 | 75-90% | <70%固定成本摊薄不够 |
| OEE（设备综合效率） | 可用率×性能率×良品率 | >75% | 世界级85%+ |
| 良品率 | 良品数/投入数 | 95%+ | 每降1%直接吃利润 |
| 单位制造成本 | 制造总成本/产出量 | 逐年下降 | 对标竞品 |
| 库存周转天数 | 存货/日均COGS | 30-60天 | 原材料/在制品/成品分开看 |
| 人均产值 | 收入/生产员工数 | 逐年提升 | 自动化替代人工的ROI |
| 订单交付准时率 | 准时交付/总订单 | >95% | 影响客户续单 |
| BOM准确率 | 实际用量/标准用量 | 98%+ | 量差的根源 |

#### 经营分析框架（月度）

```
1. 产销概览：产量/销量/库存变动/产销率
2. 成本差异：材料价差+量差、人工率差+效率差、制造费用支出差+产能差
3. 产线对比：各产线毛利率/利用率/良品率对比排名
4. 库存健康：原材料/在制品/成品周转 + 呆滞料占比（>6个月未动）
5. 关键设备：OEE趋势 + 计划外停机时长 + 维修成本
```

#### 关键决策场景

**自制 vs 外购（Make or Buy）**：
```
自制成本 = 变动成本 + 专属固定成本摊销
外购成本 = 采购价 + 来料检验 + 物流 + 质量风险溢价
如果：外购 < 自制且产能有更高价值用途 → 外购
如果：核心工艺/客户要求自制 → 即使贵也自制（战略考量）
```

**产能扩张决策**：现有利用率>85%且订单持续增长 → 评估扩产。投入=设备+厂房+人员培训，回收期=增量边际贡献/投资额。

### 8.2 互联网/SaaS

#### 成本结构

```
收入（ARR/MRR）
├── 获客成本（CAC）
│   ├── 营销费用（广告/内容/SEO/SEM）
│   ├── 销售人员薪酬+佣金
│   └── 售前支持/Demo成本
├── 服务交付成本
│   ├── 云服务器/带宽（按用量弹性）
│   ├── 客户成功团队
│   └── 技术支持/工单处理
├── 研发成本（通常占收入20-40%）
│   ├── 产品研发人员薪酬
│   ├── 基础设施/工具
│   └── 外包/外协
├── 管理费用
└── 净利润（成熟SaaS净利率15-30%）
```

#### 核心KPI

| 指标 | 公式 | 健康区间 | BP关注点 |
|------|------|---------|---------|
| ARR/MRR | 年度/月度经常性收入 | 增速>30%为高增长 | 是收入基本盘 |
| NRR | (期初-流失-降级+扩展)/期初 | >110%优秀 | <100%存量萎缩 |
| GRR | (期初-流失-降级)/期初 | >85% | 反映产品粘性 |
| LTV/CAC | 客户终身价值/获客成本 | >3健康 | <1在亏钱获客 |
| CAC回收期 | CAC/月度毛利贡献 | <12个月 | >18个月现金流压力大 |
| Logo Churn | 流失客户数/期初客户数 | <2%/月 | 与Revenue Churn区分 |
| 人均ARR | ARR/全员人数 | 20-50万/人 | 规模效率 |

#### 经营分析框架（月度）

```
1. MRR瀑布：新签 + 扩展 - 降级 - 流失 = 净增MRR
2. 获客效率：CAC趋势（按渠道拆）、CAC回收期、Pipeline转化率
3. 留存健康：NRR/GRR趋势、客户健康度分布（绿/黄/红）、大客户预警
4. 人效：人均ARR、研发投入产出比、客户成功人均服务客户数
5. 现金流：Runway月数、预收款消耗节奏、是否需要融资
```

#### 关键决策场景

**定价调整**：提价5% → 测算NRR变化（部分客户可能降级/流失） → 净收入影响 = 提价收益 - 流失损失

**产品投入优先级**：按"预计减少流失ARR"或"预计增加扩展ARR"排序功能需求，而不是按"用户请求数"

### 8.3 零售/快消

#### 成本结构

```
收入
├── 商品成本/COGS（零售50-70%，快消40-60%）
│   ├── 采购成本
│   ├── 物流到仓
│   └── 损耗/过期/退货
├── 门店运营成本
│   ├── 租金（通常占收入8-15%）
│   ├── 人工（店员/店长）
│   ├── 水电物业
│   └── 装修摊销
├── 营销费用
│   ├── 促销补贴
│   ├── 广告/会员运营
│   └── 陈列/堆头费
├── 总部费用（管理/IT/供应链）
└── 净利润（零售2-8%，快消5-15%）
```

#### 核心KPI

| 指标 | 公式 | 健康区间 | BP关注点 |
|------|------|---------|---------|
| 同店增长（SSSG） | (本期同店-去年同店)/去年 | >3%良好 | 剔除新开店的真实增长 |
| 坪效 | 门店收入/面积 | 行业差异大 | 效率核心 |
| 库存周转天数 | 存货/日均COGS | 超市15-30，服装60-90 | 资金效率 |
| 售罄率 | 售出量/进货量 | >70% | 选品精度 |
| 租售比 | 月租/月收入 | <15% | 超过预警 |
| 客单价 | 收入/交易笔数 | 逐步提升 | 组合销售/升单 |
| 会员复购率 | 复购会员/总活跃会员 | >40% | 私域价值 |

#### 经营分析框架（月度）

```
1. 销售概览：总收入/同店/新店拆分，vs预算vs去年
2. 商品分析：品类销售排名、毛利贡献排名、滞销品清单
3. 门店排名：坪效/人效/毛利率Top/Bottom 10门店
4. 促销复盘：促销期间增量销售 vs 促销投入，真正增量 vs 透支未来
5. 库存：周转天数趋势、滞销品占比、缺货率
6. 新店追踪：爬坡进度vs预期、盈亏平衡时间点预测
```

#### 关键决策场景

**促销ROI**：
```
真实增量 = 促销期销售 - 正常基线销售 - 促销后回落
促销ROI = 增量毛利 / 促销投入
⚠️ 满减类促销：不能假设每单都触发，按实际核销数据算
```

**新开店**：通常第1年亏损（爬坡）、第2年盈亏平衡、第3年贡献利润。关键假设：客流爬坡曲线、竞争环境、选址人流量。

### 8.4 物流/快递

#### 成本结构

```
收入（单票收入 × 票数）
├── 运输成本（40-55%）
│   ├── 燃油/能源
│   ├── 车辆折旧/租赁
│   ├── 司机人工
│   ├── 路桥费/过路费
│   └── 外协运力
├── 中转/分拣成本（15-25%）
│   ├── 分拣中心运营
│   ├── 设备折旧
│   └── 临时工
├── 末端配送成本（15-25%）
│   ├── 快递员人工+件量提成
│   ├── 电动车/站点租金
│   └── 破损赔付
├── 管理/IT/客服
└── 净利润（快递2-6%，供应链5-10%）
```

#### 核心KPI

| 指标 | 公式 | 健康区间 | BP关注点 |
|------|------|---------|---------|
| 单票收入 | 总收入/票数 | 行业逐年下降 | 定价策略 |
| 单票成本 | 总成本/票数 | 必须<单票收入 | 成本竞争力 |
| 装载率 | 实际载重/额定载重 | 整车>85%，零担>65% | 运力利用 |
| 时效达成率 | 准时送达/总量 | >95% | 服务品质 |
| 破损率 | 破损件/总件 | <0.1% | 赔付成本 |
| 线路盈利 | 各线路收入-成本 | 正毛利 | 亏损线路砍/调价 |
| 人均票量 | 总票/末端人数 | 逐步提升 | 自动化替代效果 |

#### 经营分析框架

```
1. 量价概览：票量增速/单票收入变动/市场份额
2. 线路分析：各线路毛利排名/装载率/时效达成
3. 成本拆解：运输/中转/末端三段成本占比变动
4. 运力效率：自有vs外协占比，外协议价能力
5. 旺季准备：产能缺口预测，临时运力预案成本
```

#### 关键决策场景

**自有车辆 vs 外协**：自有固定成本高但单位成本低（满载时），外协灵活但单价高。当票量稳定且装载率>80% → 自有更划算。

### 8.5 餐饮/酒店

#### 成本结构（餐饮）

```
收入
├── 食材成本（30-40%）← 毛利率的决定因素
├── 人工成本（20-30%）
│   ├── 前厅服务
│   ├── 后厨
│   └── 管理人员
├── 租金（8-15%）
├── 水电能源（3-5%）
├── 折旧/装修摊销（3-5%）
├── 营销/外卖平台佣金（5-10%）
└── 净利润（5-12%）

"三高红线"：食材>40% + 人工>25% + 租金>15% = 加总>80%几乎不可能盈利
```

#### 核心KPI

| 指标 | 健康区间 | BP关注点 |
|------|---------|---------|
| 翻台率 | 快餐>5次，正餐2-3次 | 坪效核心驱动 |
| 人效（月收入/人数） | 3-5万/人·月 | 人力配置 |
| 食材成本率 | 30-40% | 采购管控+菜品结构 |
| 外卖占比 | 30-50% | 外卖毛利低于堂食 |
| 新店爬坡期 | 3-6个月 | 扩张节奏 |
| 会员复购率 | >30% | 私域运营效果 |

#### 酒店核心KPI

| 指标 | 公式 | 健康区间 |
|------|------|---------|
| RevPAR | 收入/可用客房数 | 同级酒店对标 |
| ADR | 收入/已售客房数 | 定价能力 |
| 入住率 | 已售/可用 | 平日60-70%，旺季90%+ |
| GOP（营业毛利） | 收入-运营成本 | 占收入30-45% |

### 8.6 医药

#### 成本结构

```
收入
├── 生产成本（创新药5-15%，仿制药30-50%）
├── 销售费用（创新药30-50%，仿制药40-60%）← 最大费用项
│   ├── 学术推广
│   ├── 医院覆盖/商业化团队
│   └── 渠道/经销商激励
├── 研发费用（创新药15-30%，仿制药5-10%）
│   ├── 临床前
│   ├── 临床I/II/III期
│   └── 注册申报
├── 管理费用
└── 净利润（创新药10-25%，仿制药5-15%）
```

#### 核心KPI

| 指标 | 公式 | BP关注点 |
|------|------|---------|
| 研发费用率 | 研发/收入 | 创新投入力度 |
| 管线进度 | 各阶段品种数 | 未来收入储备 |
| 销售费用率 | 销售费用/收入 | >30%有合规风险 |
| 集采品种占比 | 集采收入/总收入 | 政策风险暴露 |
| 单品生命周期 | 上市→专利到期年限 | 收入可持续性 |

#### 关键决策场景

**研发项目NPV（含概率调整）**：
```
NPV = Σ(各阶段成功概率 × 该阶段现金流折现)
临床I期成功率约70% → II期约40% → III期约60% → 审批约90%
综合成功率 ≈ 15%（从I期到上市）
风险调整NPV = 未调整NPV × 15%
```

**集采影响测算**：中标后降价60-90%，但量可能增3-5倍。BP需算：降价后边际贡献是否为正 → 为正就值得保。

### 8.7 金融（银行/信贷）

#### 收入结构

```
收入
├── 净利息收入（NII）= 生息资产 × 净息差（NIM）← 占60-80%
├── 手续费/佣金（15-25%）
│   ├── 支付结算
│   ├── 财富管理/代销
│   └── 信用卡/POS
├── 投资收益
└── 其他

成本
├── 资金成本（付息负债 × 平均付息率）
├── 信用成本（拨备计提 = 贷款余额 × 计提比率）← 利润波动的最大来源
├── 运营成本（人/网点/IT）
└── 税务
```

#### 核心KPI

| 指标 | 公式 | 健康区间 | BP关注点 |
|------|------|---------|---------|
| NIM | (利息收入-利息支出)/生息资产 | 1.5-2.5% | 利率环境影响 |
| 不良率 | 不良余额/贷款总额 | <2% | 资产质量 |
| 拨备覆盖率 | 拨备/不良余额 | >150% | 监管红线 |
| 成本收入比 | 营业费用/营业收入 | <35%优秀 | 运营效率 |
| 资本充足率 | 资本/风险加权资产 | 核心一级≥7.5% | 约束放贷能力 |
| RAROC | 风险调整后收益/经济资本 | >WACC | 真实回报 |

#### 经营分析框架

```
1. NIM趋势：本月NIM vs 上月/去年，拆解为资产端收益率变动+负债端成本变动
2. 资产质量：新增不良/核销/迁徙率/关注类贷款占比
3. 业务增长：贷款投放/存款吸收/中间业务收入
4. 资本消耗：各业务线RWA消耗 vs 利润贡献 → RAROC排名
5. 合规：监管指标距红线的安全垫
```

### 8.8 房地产

#### 成本结构

```
销售收入（均价 × 面积）
├── 土地成本（30-50%）← 最大成本项
├── 建安成本（20-35%）
│   ├── 结构/装修/设备
│   └── 前期/配套
├── 资本化利息（5-15%）
├── 营销费用（2-4%）
├── 管理费用
├── 税金（土增税/增值税/所得税——合计占收入15-25%）
└── 净利润（5-15%）
```

#### 核心KPI

| 指标 | 公式 | 健康区间 |
|------|------|---------|
| 去化率 | 已售面积/可售面积 | 首开>50% |
| 净负债率 | (有息负债-现金)/权益 | <100% |
| 现金短债比 | 现金/一年内到期债务 | >1 |
| 项目IRR | 现金流折现率 | >15% |
| 回款率 | 实际回款/合同销售 | >85% |
| 融资成本 | 加权平均融资利率 | <7% |

#### 经营分析框架

```
1. 销售：签约金额/面积/均价 vs 目标，去化率
2. 回款：回款率/逾期按揭/监管账户余额
3. 项目进度：在建项目施工进度/竣工计划/交付风险
4. 资金：现金短债比/净负债率/融资到期日历
5. 拿地：新增土储/地价/未来结转收入
```

### 8.9 教育/培训

#### 成本结构

```
收入（学费/课时费——通常预收）
├── 教师成本（30-45%）← 最大成本项
│   ├── 全职教师薪酬
│   └── 兼职/外教课时费
├── 场地成本（10-15%）
├── 营销获客（15-25%）
├── 教研/内容（5-10%）
├── 运营/管理
└── 净利润（8-18%）

⚠️ 收入确认特殊性：预收学费按课时消耗确认收入，不是收到就确认。
   合同负债（预收）大 ≠ 利润高——只是现金流好。
```

#### 核心KPI

| 指标 | 公式 | 健康区间 |
|------|------|---------|
| 续费率 | 续费学员/到期学员 | >70% |
| 获客成本 | 营销费/新增付费学员 | <3个月学费 |
| 到课率 | 实际到课/排课人次 | >85% |
| 班均人数 | 学生/班级数 | 越接近满班越好 |
| 预收消耗率 | 已确认收入/预收余额 | 匀速消耗 |
| 退费率 | 退费金额/新签金额 | <15% |

#### 关键决策场景

**班型定价**：固定成本（教室+教师底薪）/ 盈亏平衡人数 = 最低开班人数。低于此人数开班就亏。

**扩科/扩地**：现有学员的交叉销售成本远低于新获客。优先让现有学员多报科目（扩科），再考虑新校区（扩地）。

### 8.10 跨境电商

> 关联：industry_accounting.md - 8.5 跨境电商会计核算（收入确认/出口退税/海外仓/监管方式）。本节侧重经营分析与决策。

#### 商品层级体系

```
SPU → SKU → ASIN（亚马逊）/ PID（其他平台）
父ASIN ≈ SPU（变体容器），子ASIN ≈ SKU（可购买单位）
利润分析必须到子ASIN/SKU级别。
```

#### 成本结构（以亚马逊FBA为例）

```
售价（Selling Price）
├── 平台佣金（Referral Fee）：8-15%
├── FBA费用：配送费 + 月仓储费 + 长期仓储附加费
├── 商品成本（COGS）= 采购 + 头程物流 + 关税
├── 广告费（PPC/DSP）
├── 退货成本（退货率×处理费+商品损耗）
├── 其他（促销/Vine/品牌内容/移除费）
└── 净利润

单位经济模型：
  毛利 = 售价 - 佣金 - FBA费 - COGS - 头程 - 关税
  净利 = 毛利 - 广告分摊 - 退货损失 - 其他
```

#### 核心KPI

| 类别 | 指标 | 健康区间 |
|------|------|---------|
| 流量 | CVR（转化率） | 10-20% |
| 广告 | ACoS | 新品30-50%，成熟期<20% |
| 广告 | TACoS | 成熟产品<10% |
| 库存 | IPI | >400（否则限仓） |
| 库存 | 库存周转天数 | 30-60天 |
| 盈利 | ASIN级净利率 | >10% |
| 运营 | 退货率 | 标品5-10%，服装15-30% |

#### 经营分析框架

```
1. GMV/订单/客单价/退货率 vs 预算 vs 去年
2. ASIN级盈利分析：Top20净利排名 + 亏损ASIN清单
3. 广告效率：ACoS/TACoS周趋势 + 自然流量占比
4. 库存健康：IPI + 断货预警(<14天) + 滞销(>90天)
5. 现金流：回款周期14天 + 头程预付30-45天 + 实际CCC 60-90天
6. 汇率：收入端外币 vs 人民币成本 + 锁汇情况
```

#### 关键决策场景

**新品上架投入测算**：
- 投入 = 首批备货 + 推广预算（前3个月亏损）+ 品牌内容
- 回收期：月1-3亏损 → 月4-6爬坡 → 月7+盈利。典型6-12个月。
- 关键假设：日均单量、ACoS、退货率、季节性

**清货决策**：继续持有成本 > 清货回收 → 清货；清货回收 < 移除成本 → 销毁

**选品盈亏平衡**：
```
单位边际贡献 = 售价 - 佣金 - FBA - COGS - 头程 - 广告
盈亏平衡量 = 月固定成本 / 单位边际贡献
如果竞品BSR反推日均销量 < 盈亏平衡日均 → 这个品不值得做
```

#### 跨境税务合规（2026年更新）

**欧盟**：2026.7.1起取消150€免税，过渡期按税号行征收€3/行，2028.7起从价关税。强制IOSS。

**各市场速查**：欧盟VAT 17-27%（库存所在地注册）/ 英国20% / 美国Sales Tax 0-10%（经济关联$10万）/ 日本10%

**企业所得税架构**：

> 关联：tax_advanced.md - 国际税收/关税（税种体系视角）

中国母公司+香港中间控股+海外销售子公司。注意CFC规则、转让定价（利润率3-8%）。

**美国de minimis**：⚠️ 行政暂停（2025.7.30行政令）+ 立法永久废除（OBBBA 2025.7.4签署，2027.7.1起$800免税额永久取消）。所有进口包裹均需正式报关缴税。

#### 汇率管理

> 详见：industry_accounting.md - 8.5.4 多币种核算与汇兑损益（记账规则）

**汇率对冲策略（BP视角）**：
- 月收入超50万美元 → 锁定30-50%预期汇率
- 利用Payoneer/Airwallex锁汇功能
- 将汇兑损益纳入定价模型（动态调价）
- 收支币种匹配（自然对冲）

#### 现金流管理

各平台回款：Amazon 14天 / Temu 7天 / 速卖通15-30天 / 独立站2-7天

预警线：现金<3个月运营成本 → 启动融资；库存资金>总资产60% → 库存过重

#### 风险清单（2026年）

| 风险 | BP应对 |
|------|--------|
| 汇率 | 按月锁远期，调定价 |
| 库存 | 安全库存模型，库龄监控 |
| 税务 | 定期检查VAT阈值 |
| 平台 | 分散多平台，备现金流 |
| 物流 | 最优运输组合，买保险 |
| 供应商 | 多家备选，长协锁价 |
| 关税 | 调定价，使用IOSS，境内设仓 |
| 合规 | 品牌备案，账号隔离 |

## 十、BP沟通、汇报与影响力

### 经营分析会汇报（5分钟版）

```
1. 结论先行（30秒）
2. 数据支撑（2分钟）← 瀑布图最佳
3. 归因分析（1分钟）
4. 行动建议（1分钟）
5. 下月预测（30秒）
```

### 与业务沟通原则

- 用业务语言（不说"管理费用超支"，说"团队差旅比上月多30%"）
- 带着方案去（不只说"超了"，说"超了，但控制在XX以内季度可达标"）
- 数据可视化（趋势图 > 数字表）
- 区分可控与不可控
- 不当警察当军师

### 向上汇报原则

- 老板关心三件事：收入完不完？利润有没有风险？现金流够不够？
- 坏消息早说
- 给选择题不给问答题
- 一页纸原则

## 十一、常见场景FAQ

**Q1：销售要求降价10%换大订单？**
算降价后边际贡献率 → 维持利润需增量 = 原边际贡献率/新边际贡献率 - 1

**Q2：产品线毛利下降怎么定位？**
三步：价格差异 → 成本差异 → 结构变化，用瀑布图可视化

**Q3：新开门店回报？**
投资额 → 5年收入/成本预测（含爬坡）→ NPV/IRR/回收期 → 敏感性分析

**Q4：研发费用资本化还是费用化？**
CAS 6五条件。不确定时费用化更稳健。详见 accounting_core.md。

**Q5：跨境电商ACoS高要不要停？**
看TACoS趋势 + ASIN生命周期 + 盈亏平衡ACoS + 竞品动态。新品期容忍，成熟品超盈亏线则优化。

**Q6：库存备多少？**
安全库存 = 日均销量 × (补货周期+安全天数)。保持30-60天为健康。

**Q7：某产品该不该继续卖？**
算ASIN级损益：净利为负连续3月→清货；毛利正但广告吃掉→优化广告2-4周；连毛利都负→立即退出。

## 十四、财务三支柱组织模型（COE/BP/SSC）

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

[场景：企业财务组织如何从"核算职能"升级为"业务伙伴+能力中心+共享中心"三支柱？BP在其中扮演什么角色？]

财务组织成熟后通常演进为三支柱架构，与HR三支柱同构：

| 支柱 | 全称 | 主要职能 | 定位 |
|------|------|---------|------|
| BP | 业务伙伴 | 理解业务需求，快速整合并实施解决方案 | 实现作战能力协同化，贴近一线 |
| COE | 能力中心 | 制定政策、流程、规则，提供税务/资金等专业解决方案 | 支撑能力专业化 |
| SSC | 共享服务中心 | 账务处理、财务报告、销售与交易核算 | 标准化作业、规模效应（详见 shared_services_rpa.md） |

### 14.1 财务BP的四种类型

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

按服务对象给BP分型，避免"BP不知道自己定位"：

| BP类型 | 服务对象 | 核心职责 |
|--------|---------|---------|
| 投资型BP | 投资密集型新业务线 | 落实投资战略，实现投资与经营闭环，管理财务与运营风险 |
| 业务群型BP | 各事业群(BG) | 协助BG负责人完成收入规划、经营效益与绩效管理，共同对经营指标负责 |
| 平台型BP | 研发/制造/供应链/行政等平台部门 | 提供专业财经解决方案，开展风险管理，促进业务目标达成 |
| 区域型BP | 全球各区域一线 | 作为价值整合者，确保面向客户的经营目标完成，提升运营资产效率 |

### 14.2 COE能力中心的两类九部门

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

COE按专业领域设置，分两类：
- **财经业务类**：定价中心、资金管理、销售融资、账务管理、税务管理
- **财经风控类**：内控与企业风险管理、财务风险控制、经营管理、子公司财经管理

⚠️ 大型跨国企业可在关键金融城市设**财务风控中心**：一个做"仪表盘"（研判全球经济/政治/汇率/贸易对企业的影响），一个做财务策略/架构独立评估（账务+税务+资金），一个做项目审视与风险控制。

### 14.3 财务人才结构的倒三角趋势

> 类型：示例
> 来源：企业财务管理实务
> 置信度：实务经验

传统财务组织是正三角形：底部大量会计核算人员，顶部少数支持商业决策的人。**随自动化/智能化发展，将倒置为倒三角**——底层基础核算人员被替代，留下的是高技能人才（MBA、会计/金融硕士）。

财务人员应当在夯实会计核算的同时，把重心转向：决策支持、预算预测、资金统筹、财务规划、税务筹划、控制评价——最终为战略与经营决策提供依据。

## 十五、项目"概预核决"四算管理

> 类型：流程
> 来源：企业财务管理实务
> 置信度：实务经验

[场景：以项目为经营单元时，怎样用财务工具拉通项目全生命周期、让项目经理对经营结果负责？]

把项目视为**独立的利润中心**，"四算"是项目最基础的财务工具，缺一不可：

| 算 | 环节 | 做什么 | 责任人 |
|----|------|--------|--------|
| 概算 | 设计项目利润 | 估算项目收入、成本费用、人工/材料/服务成本，倒推项目利润 | 概算经理（通常由一线代表/系统主任担任） |
| 预算 | 管理增收节支 | 项目报价结果作为交付预算，贯穿全程 | 项目经理 |
| 核算 | 管理增收节支 | 跟踪实际执行，校验预算 | 项目财务经理 |
| 决(结)算 | 验证+经验传承 | 看是否完成当初概算，沉淀经验供下个项目复用 | 项目财务经理 |

**关键规则：**
- 概算是**合同谈判的基础**：报价时先有成本线，报价结果即交付预算。
- 项目财务经理是项目预算、预测、核算准确性的**第一责任人**——出问题首先被追责，倒逼财务融入业务。
- 责权对等：把项目的计划权、预算权、核算权、销售决策权授予一线（让听得见炮声的人决策），同时把激励建立在项目结果上。
- 项目奖励设**两部分**：①按预算准时优质完成的奖励（达标即得）②对项目经营改善（增收/节支）的额外奖励。例：成本预算100万，实际90万完成 → 节省的10万拿一部分与团队分享。

## 十六、财务BP如何"懂业务、做伙伴"

> 类型：规则
> 来源：企业财务管理实务
> 置信度：实务经验

**BP做业务伙伴的硬要求：**
- 财务若不懂业务，只能提供低价值的会计服务。BP**至少要能读懂合同**——读懂商务条款、验收标准、产品配置。
- "计划是龙头，制定计划的人一定要懂业务"——计划、预算、核算部门的负责人应当由懂业务的人担任。
- 财务分析要"突破财务范畴"：找出数字背后的业务原因 → 指出问题 → 找出对策 → 落实责任 → 到期考核，做成一把手工程而非财务小闭环。
- 称职的CFO标准：**随时可以接任CEO**。没有项目经营管理经验的财务人员难以成长为CFO，应积极参与项目管理、经营分析、预算决策，不局限于专业技能。

⚠️ 财务的两个监控边界规则：①财务是"大管家"，没有任何一件事是不可以管的；②业务对风险负责，财务负责向业务**提示**风险的存在。监控重于服务，但不替业务做经营决策。
