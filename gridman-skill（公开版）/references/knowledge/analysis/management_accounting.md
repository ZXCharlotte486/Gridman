## 一、核心公式体系

### 毛利与毛利率

```
毛利 = 营业额 - 产品成本（变动成本）
毛利率 = 毛利 ÷ 营业额 × 100%
```

**易错点**：除数永远是营业额，不是成本。

| 错误算法 | 正确算法 |
|---------|---------|
| (140-100) ÷ 100 = 40%（加成率，不是毛利率） | (140-100) ÷ 140 ≈ 28.6%（毛利率） |

**目标定价反推**：
```
目标售价 = 成本 ÷ (1 - 目标毛利率)
```
例：成本100元，目标毛利率40% → 售价 = 100 ÷ (1-40%) ≈ 167元

---

### 成本结构：固定成本 vs 变动成本

| 类型 | 定义 | 典型例子 |
|------|------|---------|
| 固定成本 | 不随销量/营收变动的支出 | 房租、固定人员底薪、折旧、软件订阅费 |
| 变动成本 | 随销量/营收同步变动的支出 | 产品直接成本、业务提成、销售绩效工资、包装物料 |
| 半变动成本 | 有固定部分+变动部分 | 水电费（基础费+用量费）、阶梯式人工 |

**变动费用率** = 变动费用 ÷ 营业收入

---

### 盈亏平衡分析

**公式推导**：
```
利润 = 收入 - 固定成本 - 变动成本
     = 收入 - 固定成本 - 收入 × 变动费用率
     = 收入 × (1 - 变动费用率) - 固定成本
     = 收入 × 毛利率 - 固定成本

盈亏平衡时利润 = 0：
保本营业额 = 固定成本 ÷ 毛利率
```

**两种计算方法**：

| 方法 | 适用场景 | 特点 |
|------|---------|------|
| 顺算法（试错法） | 财务逻辑弱的业务人员 | 搭利润表模板，手动调收入直到利润→0 |
| 倒算法（公式法） | 财务自用、批量核算 | 精确、可批量，适合多项目统一测算 |

**案例**：
- 月固定成本：房租2万 + 人工5万 + 水电0.5万 + 办公0.5万 = 8万
- 整体毛利率：40%
- 保本营业额 = 8万 ÷ 40% = **20万**
- 解读：月营业额 > 20万才盈利，< 20万就亏损

---

### 边际贡献分析

```
边际贡献 = 营业额 - 变动成本
边际贡献率 = 边际贡献 ÷ 营业额
           = 1 - 变动费用率
           = 毛利率（当变动成本 = 产品成本时）

安全边际 = 实际营业额 - 保本营业额
安全边际率 = 安全边际 ÷ 实际营业额
```

安全边际率越高，企业抗风险能力越强。

## 二、经营分析框架

### 利润驱动因素分解

利润变动可以分解为三个驱动因素：

```
利润变动 = 量的变动 + 价的变动 + 本的变动

量：销量/营业额变化
价：单价/毛利率变化
本：成本结构变化（固定成本增减、变动费用率变化）
```

**分析步骤**：
1. 对比本期 vs 上期（或预算 vs 实际）的利润差异
2. 拆解：是量的问题（销量下降）、价的问题（毛利率下降）还是本的问题（成本上升）
3. 针对主要驱动因素制定改善措施

---

### 关键经营指标体系

**盈利能力**

| 指标 | 公式 | 关注点 |
|------|------|--------|
| 毛利率 | 毛利 ÷ 营业额 | 产品/服务的定价能力和成本控制 |
| EBITDA利润率 | EBITDA ÷ 营业额 | 剔除折旧摊销后的经营盈利能力 |
| 净利率 | 净利润 ÷ 营业额 | 最终盈利水平 |
| ROE | 净利润 ÷ 股东权益 | 股东回报率 |
| ROIC | NOPAT ÷ 投入资本 | 资本使用效率 |

**营运效率**

| 指标 | 公式 | 关注点 |
|------|------|--------|
| 应收账款周转天数 | 应收账款 ÷ 日均收入 | 越短越好，过长说明回款慢 |
| 存货周转天数 | 存货 ÷ 日均成本 | 越短越好，过长说明积压 |
| 应付账款周转天数 | 应付账款 ÷ 日均采购额 | 越长越好（占用供应商资金） |
| 现金转换周期 | 应收天数 + 存货天数 - 应付天数 | 越短越好，负数说明商业模式优秀 |

**现金流质量**

| 指标 | 公式 | 关注点 |
|------|------|--------|
| 收现比 | 经营活动现金流 ÷ 净利润 | 长期 < 1 说明利润质量差 |
| 自由现金流 | 经营现金流 - 资本支出 | 真正可支配的现金 |

---

### 杜邦分析

将 ROE 分解为三个驱动因素：

```
ROE = 净利率 × 资产周转率 × 权益乘数
    = (净利润/营业额) × (营业额/总资产) × (总资产/股东权益)
```

**用途**：找出 ROE 变化的根本原因——是盈利能力问题、资产效率问题还是财务杠杆问题。

## 三、预算管理

### 预算编制方法

| 方法 | 适用场景 | 特点 |
|------|---------|------|
| 增量预算 | 业务稳定、变化不大 | 在上年基础上调整，简单但可能固化历史问题 |
| 零基预算 | 业务重组、成本优化 | 从零开始论证每笔支出，工作量大但更合理 |
| 滚动预算 | 业务变化快 | 每月/季度滚动更新未来12个月预算，保持动态 |
| 弹性预算 | 成本控制 | 按不同收入水平编制多套预算，适应量的变化 |

### 预算执行分析

**差异分析**：
```
实际 vs 预算差异 = 量差 + 价差 + 混合差

量差：实际量 vs 预算量的差异
价差：实际单价/单位成本 vs 预算的差异
```

**预算分析报告结构**：
1. 本期实际 vs 预算差异摘要
2. 主要差异项目说明（超预算/低于预算的原因）
3. 全年预测（基于当前趋势）
4. 风险提示和改善建议

## 四、成本管理

### 成本分类

**按与产品的关系**：
- 直接成本：可直接归属到产品/项目的成本（原材料、直接人工）
- 间接成本：需要分摊的成本（制造费用、管理费用）

**按行为**：
- 固定成本：不随产量变化（厂房折旧、管理人员工资）
- 变动成本：随产量变化（原材料、计件工资）
- 阶梯成本：在一定范围内固定，超过临界点跳升（增加一条生产线）

### 成本分摊方法

| 方法 | 适用场景 |
|------|---------|
| 直接分配法 | 辅助部门成本直接分配给生产部门 |
| 阶梯分配法 | 辅助部门按服务顺序逐步分配 |
| 作业成本法（ABC） | 产品多样、间接成本比重大的企业 |

### 作业成本法（ABC）核心逻辑

```
传统方法：间接成本 → 按单一标准（如工时）分摊 → 产品成本
ABC方法：间接成本 → 识别作业 → 找成本动因 → 按实际消耗分配 → 产品成本
```

**优势**：更准确反映产品真实成本，避免高量产品补贴低量产品。

## 五、定价策略

### 常用定价方法

| 方法 | 公式/逻辑 | 适用场景 |
|------|---------|---------|
| 成本加成定价 | 售价 = 成本 × (1 + 加成率) | 制造业、工程项目 |
| 目标毛利率定价 | 售价 = 成本 ÷ (1 - 目标毛利率) | 零售、餐饮 |
| 竞争导向定价 | 参考竞争对手价格 | 同质化产品 |
| 价值导向定价 | 基于客户感知价值 | 差异化产品、SaaS |
| 边际贡献定价 | 售价 > 变动成本即可接单 | 产能过剩时的特殊订单 |

**边际贡献定价的应用**：
- 当有闲置产能时，只要售价 > 变动成本，接单就能增加利润
- 不能长期使用，否则无法覆盖固定成本
- 适用于：淡季促销、清库存、新市场开拓

## 六、投资决策分析

### 常用方法

**净现值法（NPV）**：
```
NPV = Σ [现金流 ÷ (1+折现率)^t] - 初始投资
NPV > 0：投资可行
```

**内部收益率（IRR）**：
- 使 NPV = 0 的折现率
- IRR > 资本成本：投资可行

**回收期法**：
```
静态回收期 = 初始投资 ÷ 年均现金流入
```
- 简单直观，但忽略时间价值和回收期后的现金流

**投资回报率（ROI）**：
```
ROI = (收益 - 投资成本) ÷ 投资成本 × 100%
```

## 七、管理会计报告

### 管理报表 vs 财务报表

| 维度 | 财务报表 | 管理报表 |
|------|---------|---------|
| 目的 | 对外披露，合规 | 内部决策，管理 |
| 频率 | 季度/年度 | 日/周/月 |
| 颗粒度 | 公司整体 | 产品/部门/项目/客户 |
| 准则约束 | 严格遵循会计准则 | 灵活，按管理需要设计 |
| 时间维度 | 历史数据 | 历史+预测 |

### 常见管理报表

**月度经营分析报告结构**：
1. 核心指标仪表盘（营收、毛利率、净利润、现金流）
2. 本月 vs 上月 vs 预算对比
3. 主要差异分析（量/价/本）
4. 分产品/分区域/分客户的盈利分析
5. 风险预警和下月预测

**产品盈利分析**：
- 按产品线计算各自的毛利率和边际贡献
- 识别"利润牛"（高毛利高销量）和"利润黑洞"（低毛利高资源占用）
- 为产品组合优化提供依据

# CPA财务成本管理——成本管理会计部分（第12-20章）

## 第12章 成本管理会计基础

### 成本分类体系

| 分类标准 | 类别 | 说明 |
|---------|------|------|
| 按计入方式 | 直接成本 | 可经济合理地追溯到成本对象 |
| | 间接成本 | 需分配到成本对象 |
| 按经济用途 | 制造成本 | 直接材料+直接人工+制造费用 |
| | 非制造成本 | 销售费用+管理费用+财务费用（期间费用） |
| 按成本性态 | 固定成本 | 总额在相关范围内不随业务量变动 |
| | 变动成本 | 总额随业务量成正比例变动 |
| | 混合成本 | 随业务量变动但不成正比例 |

### 固定成本细分

- **约束性固定成本**：经营能力成本，不能通过当前决策改变（折旧、管理人员工资）
- **酌量性固定成本**：可通过管理决策改变数额（广告费、研发费、培训费）

### 变动成本细分

- **技术性变动成本**（约束性）：由技术关系决定（单位产品材料消耗）
- **酌量性变动成本**：由管理决策决定（销售佣金）

### 混合成本类型

1. **半变动成本**：y = a + bx（初始固定+随业务量正比例变动）
2. **阶梯式成本**：在一定范围内固定，超过后跳跃到新水平
3. **延期变动成本**：一定业务量内固定，超过后正比例增长
4. **非线性成本**：与业务量非直线关系，相关范围内可近似为线性

### 成本管理会计基本原则

1. 成本效益原则：预期收益 > 预期成本时才投入
2. 技术与行为因素原则
3. 不同目的不同成本原则
4. 相关性原则：增量成本才是相关成本

## 第13章 产品成本计算

### 三种基本方法对比

| 方法 | 适用场景 | 成本对象 | 计算期 |
|------|---------|---------|--------|
| 品种法 | 大量大批单步骤生产 | 产品品种 | 定期（月末） |
| 分批法 | 单件小批生产 | 产品批别（订单） | 不定期（与生产周期一致） |
| 分步法 | 大量大批多步骤生产 | 各生产步骤 | 定期（月末） |

### 辅助生产费用分配方法

| 方法 | 核心逻辑 | 特点 |
|------|---------|------|
| 直接分配法 | 不考虑辅助车间互相提供劳务，直接分配给辅助生产以外的受益单位 | 简单但不够精确 |
| 一次交互分配法 | 先在辅助车间之间交互分配，再对外分配 | 两次分配，较精确 |
| 代数分配法 | 用联立方程求解各辅助车间单位成本 | 最精确但计算复杂 |

**直接分配法公式：**
$$辅助生产单位成本 = \frac{辅助生产费用总额}{产品(劳务)总量 - 对其他辅助部门提供的量}$$

### 完工产品与在产品成本分配

| 方法 | 适用条件 |
|------|---------|
| 不计算在产品成本 | 月末在产品数量很少 |
| 在产品按年初数固定 | 各月在产品数量变动不大 |
| 在产品按原材料费用计算 | 原材料占比大且开工时一次投入 |
| **约当产量法** | 通用方法 |
| 在产品按定额成本计算 | 有准确定额且在产品数量稳定 |
| 定额比例法 | 有准确定额且在产品数量变动大 |

**约当产量法核心公式：**
$$单位成本 = \frac{月初在产品成本 + 本月生产费用}{完工产品产量 + 月末在产品约当产量}$$
$$在产品约当产量 = 在产品数量 \times 完工程度$$

**各工序完工程度（按50%平均）：**
$$第i道工序完工程度 = \frac{前面各工序工时定额之和 + 本工序工时定额 \times 50\%}{产品工时总定额}$$

### 联产品成本分配

- **售价法**：按分离点各产品总售价比例分配联合成本
- **可变现净值法**：按（最终售价 - 分离后加工成本）比例分配
- **实物数量法**：按产品数量/重量比例分配

## 第14章 标准成本法

### 标准成本种类

| 分类标准 | 类型 | 特点 |
|---------|------|------|
| 按水平 | 理想标准成本 | 最优条件下最低成本，难以达到 |
| | 正常标准成本 | 效率良好条件下应发生的成本，**广泛使用** |
| 按适用期 | 现行标准成本 | 根据当前条件制定，条件变化时修订 |
| | 基本标准成本 | 基本条件无重大变化则不变动 |

### 标准成本卡结构

$$单位产品标准成本 = 直接材料标准成本 + 直接人工标准成本 + 变动制造费用标准成本 + 固定制造费用标准成本$$

各项 = 用量标准 × 价格标准

### 成本差异分析公式汇总

**变动成本差异通用模型：**
$$成本差异 = 价格差异 + 数量差异$$
$$价格差异 = 实际数量 \times (实际价格 - 标准价格)$$
$$数量差异 = (实际数量 - 标准数量) \times 标准价格$$

| 成本项目 | 价格差异 | 数量差异 |
|---------|---------|---------|
| 直接材料 | 材料价格差异 = AQ×(AP-SP) | 材料数量差异 = (AQ-SQ)×SP |
| 直接人工 | 工资率差异 = AH×(AR-SR) | 效率差异 = (AH-SH)×SR |
| 变动制造费用 | 耗费差异 = AH×(实际分配率-标准分配率) | 效率差异 = (AH-SH)×标准分配率 |

> AQ=实际数量, AP=实际价格, SP=标准价格, SQ=标准数量(实际产量×单位标准用量)
> AH=实际工时, AR=实际工资率, SR=标准工资率, SH=标准工时(实际产量×单位标准工时)

**固定制造费用差异分析：**

| 方法 | 差异分解 |
|------|---------|
| 二因素法 | 耗费差异 = 实际数 - 预算数；能力差异 = (生产能力 - 实际产量标准工时) × 标准分配率 |
| 三因素法 | 耗费差异（同上）；闲置能力差异 = (生产能力 - 实际工时) × 标准分配率；效率差异 = (实际工时 - 实际产量标准工时) × 标准分配率 |

## 第15章 作业成本法与生命周期成本法

### 作业成本法核心逻辑

$$资源 \xrightarrow{资源成本动因} 作业 \xrightarrow{作业成本动因} 产品/服务$$

**vs 传统成本法：** 资源 → 部门 → 产品（单一产量基础分配）

### 作业成本库层级

| 层级 | 含义 | 成本动因举例 |
|------|------|------------|
| 单位级 | 每单位产品执行一次 | 机器工时、直接材料 |
| 批次级 | 每批产品执行一次 | 调试次数、检验次数、订单数 |
| 品种级 | 因某品种存在而发生 | 产品设计次数、零件数量 |
| 生产维持级 | 维护整体生产能力 | 工厂保安、行政管理 |

### 作业成本动因类型

- **业务动因**：以执行次数为基础（假设每次成本相同）
- **持续动因**：以执行时间为基础（每次耗时不同时使用）
- **强度动因**：直接归集实际耗用全部资源（特殊订单）

### 作业成本法计算步骤

1. 将直接成本追溯到产品
2. 将间接成本归集到作业成本库
3. 计算作业分配率 = 作业成本库总成本 ÷ 作业量
4. 将作业成本分配到产品 = 分配率 × 该产品耗用的作业量

### 生命周期成本法

- **产品生命周期成本** = 研发设计 + 生产制造 + 营销推广 + 顾客使用 + 弃置回收各阶段成本
- **资产生命周期成本** = 初始投资成本 + 运行维护成本 + 处置成本
- 核心思想：约80%的产品成本在设计阶段已被锁定

## 第16章 本量利分析

### 基本假设

1. 相关范围假设（期间+业务量）
2. 线性假设（成本与收入均为线性）
3. 产销平衡假设
4. 品种结构不变假设

### 混合成本分解方法

| 方法 | 特点 |
|------|------|
| 高低点法 | b = (最高点成本-最低点成本)÷(最高点业务量-最低点业务量)；a = 总成本 - b×业务量 |
| 直线回归法 | 最小二乘法，最精确 |
| 账户分析法 | 按账户性质直接归类，简单粗糙 |
| 工业工程法 | 技术测定，适用于无历史数据时 |

### 核心公式

$$息税前利润 = 单价 \times 销量 - 单位变动成本 \times 销量 - 固定成本$$
$$= 销量 \times 单位边际贡献 - 固定成本$$
$$= 销售收入 \times 边际贡献率 - 固定成本$$

$$单位边际贡献 = 单价 - 单位变动成本$$
$$边际贡献率 = \frac{单位边际贡献}{单价} = 1 - 变动成本率$$

### 保本分析

$$保本量 = \frac{固定成本}{单位边际贡献}$$
$$保本额 = \frac{固定成本}{边际贡献率}$$

**多品种保本额：**
$$加权平均边际贡献率 = \sum(各产品边际贡献率 \times 各产品销售占比)$$
$$保本销售总额 = \frac{固定成本}{加权平均边际贡献率}$$

### 安全边际与相关指标

$$安全边际 = 实际(预计)销售额 - 保本额$$
$$安全边际率 = \frac{安全边际}{实际(预计)销售额}$$
$$盈亏临界点作业率 + 安全边际率 = 1$$
$$销售息税前利润率 = 安全边际率 \times 边际贡献率$$

### 保利分析

$$保利量 = \frac{固定成本 + 目标利润}{单位边际贡献}$$
$$保利量(税后) = \frac{固定成本 + \frac{税后目标利润}{1-所得税率}}{单位边际贡献}$$

### 敏感分析

$$敏感系数 = \frac{目标值变动百分比}{参数变动百分比}$$

敏感系数绝对值越大，该因素对利润影响越大。正值=同向变动，负值=反向变动。

## 第17章 短期经营决策

### 相关成本与无关成本

**相关成本特征：** ①面向未来 ②各方案间有差异

| 相关成本 | 含义 |
|---------|------|
| 机会成本 | 放弃方案的最大潜在收益 |
| 差量成本 | 两方案预期成本之差 |
| 专属成本 | 可明确归属于某方案的固定成本 |
| 边际成本 | 业务量变动一个单位引起的成本变动 |
| 付现成本 | 需要支付现金的成本 |
| 可避免成本 | 决策改变时可避免的成本 |

| 无关成本 | 含义 |
|---------|------|
| 沉没成本 | 过去已发生、无法改变的成本 |
| 共同成本 | 各方案共同负担的固定成本 |
| 不可避免成本 | 不能通过决策改变的成本（约束性固定成本） |

### 决策规则汇总

| 决策类型 | 判断标准 |
|---------|---------|
| 亏损产品停产 | 边际贡献 > 0 → 不应停产 |
| 自制vs外购 | 比较相关成本（含机会成本、专属成本） |
| 特殊订单 | 边际贡献 > 增加的相关成本（机会成本+专属成本）→ 接受 |
| 约束资源利用 | 优先安排**单位约束资源边际贡献**最大的产品 |
| 出售vs深加工 | 深加工增量收入 > 深加工增量成本 → 深加工 |

**单位约束资源边际贡献：**
$$= \frac{单位产品边际贡献}{单位产品耗用的约束资源量}$$

### 定价方法

- **完全成本加成法**：目标价格 = 单位制造成本 × (1 + 加成率)
- **变动成本加成法**：目标价格 = 单位变动成本 × (1 + 加成率)
- 加成部分须覆盖未计入的成本和目标利润
- **有闲置能力时**：价格底线 = 变动成本（只要 > 变动成本即可接受）

## 第18章 全面预算

### 预算编制方法对比

| 分类维度 | 方法 | 特点 |
|---------|------|------|
| 出发点 | 增量预算法 | 以历史为基础调整，简单但可能延续不合理因素 |
| | 零基预算法 | 从零开始，逐项论证必要性，工作量大但合理 |
| 业务量基础 | 固定预算法 | 单一业务量水平，适应性差 |
| | 弹性预算法 | 多业务量水平，y=a+bx，适应性强 |
| 预算期 | 定期预算法 | 固定会计期间 |
| | 滚动预算法 | 持续滚动，保持固定时间跨度 |

### 弹性预算差异分析

$$固定预算差异 = 作业量差异 + 收入和支出差异$$
- **作业量差异** = 弹性预算数 - 固定预算数（因业务量不同导致）
- **收入和支出差异** = 实际结果 - 弹性预算数（因价格/效率导致）

### 营业预算编制流程

```
销售预算（起点）
  → 生产预算：生产量 = 销售量 + 期末存货 - 期初存货
    → 直接材料预算：采购量 = 生产需用量 + 期末存量 - 期初存量
    → 直接人工预算：人工成本 = 产量 × 单位工时 × 小时工资率
    → 制造费用预算（变动+固定）
  → 产品成本预算（汇总）
  → 销售及管理费用预算
```

### 现金预算结构

1. 可供使用现金 = 期初余额 + 本期现金收入
2. 现金支出合计
3. 现金多余或不足 = (1) - (2)
4. 资金筹措与运用（借款/还款/利息）
5. 期末现金余额

## 第19章 责任会计

### 责任中心类型与业绩评价

| 责任中心 | 控制范围 | 业绩评价指标 |
|---------|---------|------------|
| 成本中心 | 只控制成本 | 预算成本节约额、标准成本差异 |
| 利润中心 | 控制收入和成本 | 边际贡献、可控边际贡献、部门税前经营利润 |
| 投资中心 | 控制收入、成本和投资 | ROI、剩余收益(RI)、EVA |

### 投资中心业绩评价指标

**投资报酬率(ROI)：**
$$ROI = \frac{部门税前经营利润}{部门平均经营资产} = 销售净利率 \times 资产周转率$$

- 优点：综合性强，可分解分析
- 缺点：可能导致次优决策（拒绝高于资本成本但低于现有ROI的项目）

**剩余收益(RI)：**
$$RI = 部门税前经营利润 - 部门平均经营资产 \times 要求的报酬率$$

- 优点：鼓励接受高于资本成本的项目
- 缺点：不便于不同规模部门间比较

### 内部转移价格

| 类型 | 适用条件 | 定价方式 |
|------|---------|---------|
| 市场价格 | 存在完善外部市场 | 中间产品市场价 |
| 协商价格 | 双方协商 | 变动成本 < 协商价 < 市场价 |
| 以成本为基础 | 无外部市场 | 标准成本/标准成本加成/变动成本加成 |
| 双重价格 | 解决内部矛盾 | 供方按市场价、需方按成本价 |

**转移价格下限（供应方）：**
$$最低转移价格 = 单位变动成本 + 单位机会成本$$

## 第20章 业绩评价

### 关键绩效指标法(KPI)

- 从企业战略目标分解出关键成功因素
- 选取可量化的关键绩效指标
- 设定目标值并定期考核
- 特点：目标导向、重点突出、可量化

### 经济增加值(EVA)

$$EVA = 税后净营业利润(NOPAT) - 资本成本$$
$$= 税后净营业利润 - 投入资本总额 \times 加权平均资本成本率(WACC)$$

**调整项目（常见）：**
- 研发费用资本化
- 战略性投资损失加回
- 各种准备金变动调整
- 营业外收支剔除

**EVA的优势：**
- 考虑了全部资本成本（包括权益资本）
- 消除了会计利润的扭曲
- 引导管理者关注长期价值创造

### 平衡计分卡(BSC)

| 维度 | 核心问题 | 典型指标 |
|------|---------|---------|
| 财务维度 | 股东如何看我们？ | ROE、EVA、收入增长率、成本降低率 |
| 客户维度 | 客户如何看我们？ | 客户满意度、市场份额、客户保持率、新客户获取率 |
| 内部流程维度 | 我们必须擅长什么？ | 生产周期、合格率、新产品开发周期、流程效率 |
| 学习与成长维度 | 能否持续改进和创造价值？ | 员工满意度、员工培训时间、信息系统能力、创新能力 |

**BSC四维度因果链：**
$$学习与成长 \rightarrow 内部流程改善 \rightarrow 客户满意度提升 \rightarrow 财务业绩改善$$

**BSC特点：**
- 财务指标与非财务指标结合
- 短期目标与长期目标平衡
- 结果指标与驱动指标并重
- 将战略转化为可操作的衡量指标

### 绩效棱柱模型

五个面（棱面）构成完整的业绩评价框架：

| 棱面 | 核心问题 |
|------|---------|
| 利益相关者满意 | 谁是关键利益相关者？他们的需求和期望是什么？ |
| 利益相关者贡献 | 我们需要利益相关者提供什么？ |
| 战略 | 为满足利益相关者需求，应采取什么战略？ |
| 流程 | 为执行战略，需要什么业务流程？ |
| 能力 | 为运行流程，需要什么能力（人员、技术、基础设施）？ |

**vs BSC的区别**：
- BSC以战略为起点，绩效棱柱以利益相关者为起点
- 绩效棱柱强调双向关系（企业对利益相关者的需求 + 利益相关者对企业的贡献）
- 适用于利益相关者关系复杂的企业

### 标杆管理

**定义**：将本企业的产品、服务和管理实践与最强竞争对手或行业领先者进行对比，找出差距并持续改进的过程。

**四种类型**：

| 类型 | 对标对象 | 特点 |
|------|---------|------|
| 内部标杆 | 企业内部不同部门/分支 | 数据易获取，但视野有限 |
| 竞争性标杆 | 直接竞争对手 | 最相关，但数据难获取 |
| 功能性标杆 | 不同行业的类似功能 | 创新性强，跨行业学习 |
| 通用标杆 | 最佳实践企业（不限行业） | 突破性改进，但适用性需验证 |

**实施步骤**：确定标杆项目 → 选择标杆对象 → 收集数据 → 分析差距 → 制定改进计划 → 实施并监控

## 八、补充财务比率（现金流相关）

| 指标 | 公式 | 含义 |
|------|------|------|
| 现金利息保障倍数 | （经营活动现金流量净额 + 现金利息支出 + 付现所得税）÷ 现金利息支出 | 用现金流衡量偿付利息的能力，比利息保障倍数更可靠 |
| 盈余现金保障倍数 | 经营活动现金流量净额 ÷ 净利润 | 利润的"含金量"，>1 说明利润有现金支撑 |
| 营业现金比率 | 经营活动现金流量净额 ÷ 营业收入 | 每 1 元收入带来多少现金 |
| 全部资产现金回收率 | 经营活动现金流量净额 ÷ 平均资产总额 | 资产创造现金的效率 |
| 每股现金流量 | （经营活动现金流量净额 - 优先股股利）÷ 发行在外普通股平均股数 | 每股能分到多少经营现金 |
| 每股营业现金净流量 | 经营活动现金流量净额 ÷ 普通股股数 | 同上（简化版） |
| 期间费用率 | （销售费用 + 管理费用 + 研发费用 + 财务费用）÷ 营业收入 | 费用控制能力 |
| EBITDA | 净利润 + 利息费用 + 所得税 + 折旧 + 摊销 | 息税折旧摊销前利润，衡量核心经营盈利 |
| 销售成本率 | 销售成本 ÷ 销售收入净额 | = 1 - 毛利率 |
| 股利支付率 | 每股股利 ÷ 每股盈余 | 利润分配比例 |

## 九、财务诊断模型

### Z-Score 破产预警模型（阿尔特曼）

> 类型：公式
> 来源：Altman Z-Score（1968）+ 修正模型
> 置信度：实务经验

[场景：用一个综合判别值预警企业财务困境/破产风险——信用评估、投资尽调、审计持续经营评估高频用]

Z-Score 把多个财务比率加权成一个综合判别值，按区间判断企业陷入财务困境的概率。**按企业类型选不同模型，不能套错。**

**模型一：上市制造业企业（原始 Z 模型）**

$$Z = 1.2 X_1 + 1.4 X_2 + 3.3 X_3 + 0.6 X_4 + 0.999 X_5$$

| 变量 | 含义 | 计算 |
|---|---|---|
| X₁ | 营运资金/总资产 | （流动资产-流动负债）÷ 总资产 |
| X₂ | 留存收益/总资产 | （盈余公积+未分配利润）÷ 总资产 |
| X₃ | 息税前利润/总资产 | EBIT ÷ 总资产 |
| X₄ | 股权市值/总负债 | 股票总市值 ÷ 总负债 |
| X₅ | 营业收入/总资产 | 营业收入 ÷ 总资产（总资产周转率）|

判别区间：
- **Z > 2.99**：财务healthy，破产风险低
- **1.81 ≤ Z ≤ 2.99**：灰色地带，需进一步观察
- **Z < 1.81**：财务困境，破产风险高

**模型二：非上市企业（Z' 模型）**——X₄ 改用"股东权益账面值/总负债"（因无市值）：

$$Z' = 0.717 X_1 + 0.847 X_2 + 3.107 X_3 + 0.420 X_4 + 0.998 X_5$$

判别：Z' > 2.90 安全；1.23 ≤ Z' ≤ 2.90 灰色；Z' < 1.23 困境。

**模型三：非制造业/新兴市场企业（Z'' 模型）**——剔除 X₅（去除行业资产周转差异）：

$$Z'' = 6.56 X_1 + 3.26 X_2 + 6.72 X_3 + 1.05 X_4$$

判别：Z'' > 2.60 安全；1.10 ≤ Z'' ≤ 2.60 灰色；Z'' < 1.10 困境。

⚠️ 使用要点：
- Z-Score 是**预警信号，不是定论**——落入困境区只代表"需重点核查"，不等于一定破产。
- 行业差异大，金融业、房地产业不适用通用模型（资产负债结构特殊）。
- 审计中可作为**持续经营能力评估**的量化辅助（详见 audit_procedures.md 第五阶段持续经营评估）；造假识别中可作为财务困境→造假动机的佐证（详见 financial_fraud.md 造假动机）。

示例：某上市制造企业，营运资金 2000 万、总资产 1 亿、留存收益 1500 万、EBIT 800 万、股票市值 1.2 亿、总负债 6000 万、营业收入 1.5 亿：
X₁=0.20, X₂=0.15, X₃=0.08, X₄=2.0, X₅=1.5
Z = 1.2×0.20 + 1.4×0.15 + 3.3×0.08 + 0.6×2.0 + 0.999×1.5 = 0.24+0.21+0.264+1.2+1.4985 = **3.41 > 2.99，财务healthy**。

### 哈佛分析框架（财报分析顶层方法论）

> 类型：流程
> 来源：哈佛大学 Palepu & Healy 框架
> 置信度：实务经验

[场景：系统分析一家公司财报时的顶层框架——不就数字论数字，先看战略再看会计，最后落到前景]

哈佛框架的核心思想：**财务数字是结果，要从"战略→会计→财务→前景"四层递进理解，不能孤立看比率。**

| 层 | 分析什么 | 核心问题 | 古立特对应模块 |
|---|---|---|---|
| 1 战略分析 | 行业竞争格局、公司竞争优势、商业模式 | 这家公司靠什么赚钱、护城河在哪 | 波特五力（internal_control.md）、行业特有指标（equity_research.md）|
| 2 会计分析 | 会计政策选择、估计的激进/稳健、是否扭曲经济实质 | 报表数字是否真实反映经营 | 财务造假/洗澡识别（financial_fraud.md）、报表勾稽（statement.md）|
| 3 财务分析 | 用比率/杜邦/现金流评估盈利、营运、偿债、成长 | 经营到底好不好 | 杜邦分析、本量利、财务比率（本文件 + financial_ratios 工具）|
| 4 前景分析 | 预测未来业绩、估值 | 公司值多少钱、未来怎样 | DCF/可比公司（investment.md）、guidance 研究（equity_research.md）|

**四层的递进逻辑（不可跳层）：**
1. 先做**战略分析**——不懂商业模式，看比率是盲人摸象（如重资产 vs 轻资产公司的周转率没有可比性）。
2. 再做**会计分析**——数字若被会计政策扭曲，后面的财务分析全是建在沙子上（这一步要警惕盈余管理/造假）。
3. 然后**财务分析**——在数字可信的前提下，用杜邦、现金流、比率诊断经营质量。
4. 最后**前景分析**——基于前三层做预测和估值。

⚠️ 哈佛框架是 `equity_research.md`（上市公司财报研究五阶段工作流）的理论底座——五阶段工作流的"00 画像（战略）→ 02 抽取/03 稽核（会计）→ 04 报告（财务+前景）"本质就是哈佛四层的工程化落地。做深度财报研究时两者配合用。

# 责任中心实战与全面预算管理（企业实务）

> 本章为大型企业经营管理实务的方法提炼，补充第19章责任会计、第18章全面预算的实战落地规则。CPA框架讲"是什么"，本章讲"怎么落地、怎么避坑"。

## 十、五大责任中心实战框架

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

CPA教材常讲三类责任中心（成本/利润/投资）。实务中应拆成**五个层次**，把"收入中心"和"费用中心"单列，便于对不同业务单元差异化考核。

| 责任中心 | 控制范围 | 考核指标 | 适用单元 |
|---------|---------|---------|---------|
| 投资中心 | 收入+成本+利润+投资 | 投资利润率、剩余收益(RI)、EVA | 端到端产品投资与生命周期管理单元、子公司 |
| 利润中心 | 收入+成本 | 边际贡献、可控利润、税前经营利润 | 面向客户作战的区域/事业部 |
| 收入中心 | 仅收入（不对成本负责） | ①销售收入完成率 ②销售款回收平均天数 ③坏账损失发生率 | 销售分公司、销售部 |
| 费用中心 | 期间费用（管理费/财务费） | 费用发生额、费用改进率 | 行政、人事等不直接面向客户的服务部门 |
| 成本中心 | 仅可控成本 | 成本(费用)变动额、成本(费用)变动率 | 生产部门、劳务提供部门 |

**关键判断规则：**
- 收入中心**若有定价权**→对毛收益负责；**若无定价权**→只对销售量和销售结构负责。
- 应当在收入中心考核中引入边际贡献概念，避免单纯追求收入最大而牺牲利润（用作业成本法把收入中心升级为利润中心是趋势）。
- 费用中心分两型：**固定费用中心**（正常必要支出为主）和**管理(随机)费用中心**（支出多少由管理决策定）。后者控制重点在**预算总额审批**。
- 成本中心的"可控成本"必须同时满足三条件：**可预计、可计量、可控制**。技术性成本用弹性预算控制；酌量性成本（研发/广告/培训）控制重点在预算总额审批。
- 投资中心用投资利润率(ROI)易诱发"次优决策"（拒绝高于资本成本但低于现有ROI的项目）→ 应同时考核**剩余收益/EVA**纠偏。

**收入中心三个考核公式：**
```
销售收入目标完成率 = 实际销售收入 ÷ 目标销售收入 × 100%
销售款回收平均天数 = (销售收入 × 回收天数) ÷ 全部销售收入
坏账损失发生率 = 当期坏账损失额 ÷ 当期全部销售收入(或赊销收入) × 100%
```

### 10.1 责任中心 vs 阿米巴：选型判断

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

[场景：企业要"划小经营单元、让业务主官对结果负责"时，选阿米巴式内部结算还是责任中心式授权监控？]

| 维度 | 阿米巴模式 | 责任中心模式 |
|------|-----------|-------------|
| 核心机制 | 强调内部结算，每个单元都是内部利润中心 | 下放经营权 + 加强监控权 |
| 组织取向 | 高度分权，单元自主扩大输入/减少费用 | 集权管理 + 中央平台 + 授权一线作战 |
| 内部交易 | 环环相扣的内部买卖、内部定价 | 以客户为中心，减少内部交易成本 |
| 适用文化 | 精细核算文化 | 以客户为中心、强监控文化 |

**选型规则：**
- 核算单元**不是越小越好**：⚠️ 单元划得越小，内部核算环节越多，内部交易成本越高。应从战略角度综合权衡颗粒度。
- 若企业流程长、做大后反应慢 → 选责任中心：把经营权下放到"听得见炮声"的一线，同时把监控权收归中央平台。
- 把不同系统设为不同层级的利润中心可平衡收入与利润矛盾：**考核作战单元(区域)看利润，考核业务群(BU)看收入**——做大收入要花钱、会压利润，分开考核避免顾此失彼。

### 10.2 利润中心考核的三维度（防短期行为）

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

⚠️ 只考核当期利润会诱导业务主官透支未来、忽视风险。利润中心考核指标应当分三大类平衡长短期：

| 维度 | 含义 | 典型指标 |
|------|------|---------|
| 多产粮食（当期产出） | 有利润的收入、有现金流的利润 | 收入、利润、经营性现金流、资产周转率 |
| 增加土壤肥力（长期能力） | 为长期增长和抗风险蓄力 | 客户关系与满意度、组织能力建设、战略市场份额 |
| 内外合规（守底线） | 减少黑天鹅事件 | 合规遵从率、数据保护合规、内控有效性 |

**规则：**"粮食"不等于收入——应定义为"有利润的收入 + 有现金流的利润"。考核时用**市场占有率**等相对指标定市场目标，不用**销售额**等绝对指标（防止"虽然增长但落后于对手"的假繁荣）。

## 十一、全面预算管理实战

### 11.1 全面预算的"8个常见陷阱"

> 类型：判断表
> 来源：企业财务管理实务
> 置信度：实务经验

[场景：企业推行全面预算时最容易踩的8个坑，每个都给纠偏规则。]

| # | 误区 | 纠偏规则 |
|---|------|---------|
| 1 | 预算是财务部的事，其他部门只配合提供数据 | ⚠️ 全面预算是各部门共同协作的完整控制过程，无主次之分。应由**一把手组织**，财务只做日常运作助手。 |
| 2 | 预算只要保持增长就好 | 要问两件事：①增长够不够（用市场占有率对标对手，不用绝对额）②该不该增长（非战略主航道、到生命周期末端且无竞争力的业务，不应再投入资源）。 |
| 3 | 预算就是领导自上而下发指标，承接即可 | "不听老板的死得快，全听老板的也死得快"。自上而下反映战略诉求，自下而上反映一线市场理解，**两者碰撞融合**才能落地。 |
| 4 | 预算是数字游戏，想变就变 | 变更分两类：①信息遗漏/规则不清导致的返工——应靠**统一的管理核算规则**（收入怎么算、成本怎么算、费用怎么分摊）消除；②市场/政策/战略重大变化导致的——必要且必须。 |
| 5 | 预算只做一年，活在当下 | 只做年度预算会丢掉战略规划(SP)。跨年项目、产品生命周期(3-5年)必须看长期，才能理解当期决策的合理性（如"三年盈亏平衡"承诺）。 |
| 6 | 经营分析是给财务领导看的 | 经营分析是预算闭环的关键手段，应向**全面预算管理小组（组长为最高管理层）**汇报，不能做成财务小闭环。 |
| 7 | 经营分析和预算相对分离 | 经营分析是全面预算闭环的一环——通过它找实际执行差距、纠偏、保障战略落地。 |
| 8 | 历史经验没用，自己很特殊 | 历史数据是企业的"财务大数据"，整理后形成**基线**（如某客户合同写30天回款实际每次60天→基线就是60天）。竞争对手数据用于对标费用/成本占比、找优劣势。 |

### 11.2 全面预算四大落地经验

> 类型：流程
> 来源：企业财务管理实务
> 置信度：实务经验

1. **强大的预算组织保证**：最高层设预算管理委员会（总裁/副总裁/相关部长），下设预算管理部（受CFO直接领导），**半数以上成员从业务线抽调**——业务人员直接参与才能保证预算合理准确。
2. **自上而下为主、自下而上为辅的编制流程**：三级管理（预算管理部 → 预算分会 → 基层预算单位），总部按收入/利润目标倒挤费用额度下发编制原则，经几上几下微调即可执行。理想状态是"只有自上而下，无需自下而上"。
3. **四层次预算控制模式**：**年度预算 → 季度监控 → 月度预警 → 周计划管理**，逐层加密控制频率。
4. **严格的平衡计分卡考核作保障**：`核算 + 考核 = 预算`。只预算不考核，预算就是空中楼阁。

### 11.3 费用预算管控七法

> 类型：规则
> 来源：企业财务管理实务
> 置信度：实务经验

[场景：经营承压、要从费用里"拧水分"又不能伤及战略和客户时的七条管控规则。]

1. **平台组织费用"高压线"**：给职能平台定费用预算基线。太紧→能力建设跟不上；太松→利润扛不住。分部门差异化定线（如流程IT费用占总收入比≤X%）；平台费用要由区域"买单"且**增长不快于收入或毛利增长**；极端时设"费用额零增长"高压线。
2. **弹性预算授予**：上半年只授予全年费用的一部分（如40%），下半年按**收入/毛利完成率孰低**弹性授予——经营计划完不成，费用预算就减少。超预算者述职，严重者取消费用权签资格、停止进人涨薪、按比例扣奖金。
3. **管住权签人手中"一支笔"**：先用差旅/招待制度让开支有章可循，再梳理虚假报销，把违规审批的主管费用权签降级/停权，并对违规报销金额**承担连带赔偿责任**。
4. **战略投入预算单列**：⚠️"再穷不能穷战略"。预研、变革等战略投入应与当期经营预算**分离、单列、专款专用、独立核算**，定期回溯投入节奏，避免被当期经营困难挤占。
5. **客户界面与内部运营费用分开**：⚠️"再省不能省客户"。内部运营费用要"拧水分"提效率；客户界面费用按基线和业务需要合理配置，**节约不归己**（防止把客户的钱挪到内部用）。
6. **人员费用与业务性费用分类管理**：人力费用匹配中长期人力规划（干部培养需3-5年周期，不能像割韭菜），业务性费用直接与当年产出指标挂钩；外包费用科目单列、与产出挂钩弹性管理。
7. **责任中心与资源部门PK结算**：双方签内部结算协议，责任中心拥有预算权，资源部门按明确需求备资源；结算价格通过"价格听证"接受第三方/行管评审，保证客观公允。

### 11.4 计划-预算-核算闭环体系

> 类型：流程
> 来源：企业财务管理实务
> 置信度：实务经验

计划-预算-核算应贯穿三个业务环节：中长期战略规划(SP)、年度业务计划(BP)、日常项目计划(PP)，每个环节都含计划→预算→核算的PDCA闭环。

```
计划是龙头（定方向，做计划的人必须懂业务）
  → 预算是量化（把计划变成可执行的财务数字）
    → 核算是校验（验证预算执行、修正和考核计划）
三者互相促进，关键是做计划的人要懂业务。
```

**年度计划预算十步法：**
```
1. 审视机会，确定产出目标（订货）   2. 细分目标
3. 分析全流程成本找要因 + 建毛利率/效率基线，按利润目标倒挤成本费用
4. 分清成本费用控制目标和责任        5. 按人均效益提升确定人员规划
6. 确定回款及运营资产管理目标        7. 综合平衡（以现金流目标平衡）
8. 汇报评审确定规划和预算           9. 确定预算   10. 制定行动计划
```

**滚动预测分季节重点**（预测对象：订-发-收-回 + 三张报表预测）：
- Q1预测：围绕季度目标，解决眼前问题
- Q2预测：聚焦重点合同/项目，强化竞争优势，动态弹性配置资源
- Q3预测：关注客户需求和投资变化
- Q4预测：关注市场、产业布局，与战略衔接

⚠️ 预算是承诺，预测是配置资源的基础——资源按预测配置，年初预测要增加就得增加资源。

## 十二、战略目标分解到预算（战略落地工具）

> 本章补第18章全面预算与十一实战的上游环节：预算不是凭空编的，是从战略目标层层分解而来。解决"战略清晰但落不了地、预算与战略脱节"的痛点。

### 12.1 战略—预算—业绩闭环框架

> 类型：流程
> 来源：企业全面预算管理实务
> 置信度：实务经验

全面预算是"以战略目标为导向"的资源配置活动，预算在战略目标与执行之间架起桥梁，把抽象战略转成可操作的具体行动。整体闭环：

```
战略目标
  → 长期预算（资本预算 + 战略性支出）
  → 短期预算（经营预算 + 财务预算）
  → 执行预算（战略落地）
  → 业绩评估
  → 反馈（回流到各层预算，循环修正）
```

**战略与预算相辅相成（两个方向）：**
- **预算必须以战略为指导**：战略先做 SWOT 确定长期方向，预算若脱离战略，会只盯短期财务目标，牺牲对技术、员工、客户的长期投入，造成企业缺乏长期发展能力。
- **预算反哺战略**：战略抽象概括，预算把它量化为可执行行动，并通过执行评价检验和修正战略。

⚠️ 关键陷阱：编预算时容易忽略"战略机会点的投入"——战略性支出（预研、变革、市场开拓）应在长期预算里单列，不能被短期经营预算挤掉（呼应十一章 11.3 第4条）。

### 12.2 平衡计分卡战略地图：四维度分解流程

> 类型：流程
> 来源：战略分解工具实务（平衡计分卡/战略地图）
> 置信度：实务经验

平衡计分卡把使命、战略转成财务、客户、内部流程、学习与成长四维度的连贯指标。四维度有明确的**因果驱动逻辑（自下而上支撑）**：

| 维度 | 核心问题 | 在因果链中的角色 |
|------|---------|----------------|
| 财务 | 做什么赚钱、怎么赚钱、赚多少钱 | 其他三维度的最终输出结果 |
| 客户 | 应具备怎样的市场表现、提供何种产品与质量 | 支撑财务目标 |
| 内部流程 | 必须做哪些事、怎么做、形成什么能力 | 支撑财务、客户目标 |
| 学习与成长 | 靠什么样的人、能力、文化来实现 | 支撑上面三维度（一切靠"人"实现） |

**因果链方向**：学习与成长 → 内部流程改善 → 客户满意提升 → 财务业绩改善。

**逐层分解路径（纵向一致、横向协调）：**
```
公司总部平衡计分卡
  → 事业部平衡计分卡
    → 部门平衡计分卡
      → 岗位/个人计分卡（绩效评估表 / 个人绩效承诺 PBC）
```
- **纵向**：下层计分卡必须承接上层的战略目标，逐级分解关联。
- **横向**：同层各部门之间横向协调，避免目标打架。
- **机制**：自上而下规划（反映战略诉求）+ 自下而上执行（反映一线理解），两者碰撞融合（呼应十一章 11.1 第3条）。

**年度预算指标向部门分解的落地表（图-卡-表）：**
- **战略地图（图）**：一张图描述战略目标的因果关系，"一张图胜似千言万语"。
- **计分卡（卡）**：每个战略目标配 衡量指标 / 指标定义计算公式 / 目标值 / 当前水平 / 决策依据。
- **行动计划表（表）**：战略目标 → 具体行动 → 里程碑 → 负责部门 → 资源配置。资源配置直接对接预算（如"研发仪器采购1000万、外部研发合作3000万"）。

### 12.3 平衡计分卡"三原则"与指标筛选三步法

> 类型：规则
> 来源：战略分解工具实务
> 置信度：实务经验

**编制三原则**（把战略转化为衡量指标时必须满足）：
1. **因果关系**：指标之间要能串成"驱动→结果"的逻辑链，不是孤立罗列。
2. **业绩驱动因素**：既要有结果指标（滞后指标，如利润），也要有驱动指标（领先指标，如客户满意度、员工技能）。
3. **与财务指标挂钩**：所有非财务指标最终要能传导到财务结果，否则就是"为考核而考核"。

**指标筛选三步法：**
```
第一步 列举指标：头脑风暴，尽可能全面列出能衡量该战略目标的指标
第二步 筛选指标：用四条标准过滤
第三步 确定指标：汇集并建立与战略目标的对应关系，标明指标类型（结果/驱动）
```

**筛选四标准：**

| 标准 | 判断问题 |
|------|---------|
| 关键性 | 能否有效衡量战略目标？是否驱动所期望的行为？ |
| 衡量性 | 是否有数据基础？结果可测量？测量成本是否低？ |
| 管控性 | 是否不易被被考核人操纵、便于管控？ |
| 聚焦性 | 每个战略目标尽量只设一个结果指标；多个可选时，选最能传达该目标意义的那个 |

### 12.4 战略目标的九大领域（防止只盯财务）

> 类型：判断表
> 来源：战略管理实务
> 置信度：实务经验

战略目标是实施战略想达到的主要成果，应是集体测算 + 结合企业实际的结果。设目标时覆盖九大领域，避免只设财务指标：

| 领域 | 常用衡量方式 |
|------|------------|
| 盈利能力 | 净利润、投资报酬率、每股净利润、销售毛利 |
| 市场 | 市场占有率、销售额或销售量 |
| 生产率 | 设备利用率、单位产品成本 |
| 产品 | 产品线/产品的销售额与盈利、新产品开发完成期 |
| 资金 | 资本构成、新增普通股、现金流量、流动资本、回收期 |
| 生产 | 工作面积、固定费用、生产量 |
| 研究与开发 | 花费货币量、完成的项目 |
| 组织 | 将实行的变革或承担的项目 |
| 人力资源 | 人员流动率、培训人数、培训计划数 |
| 社会责任 | 活动类型、服务天数、财政资助 |

经营计划 = 把战略目标落成"四要素"：**目标（做什么）、责任（谁去做）、时间（何时做）、措施（如何做）**。

### 12.5 业务领先模型（BLM）：战略制定+执行一体化

> 类型：框架
> 来源：战略制定与执行方法论（业务领先模型 Business Leadership Model）
> 置信度：实务经验

BLM 是一套把战略制定和执行打通、最终落到年度预算的方法论，以**领导力**为牵引、**价值观**为基础，中间是"战略规划 + 战略执行"两大模块。

```
            领导力（牵引全程）
   ┌──────────────┬──────────────┐
   战略规划         →         战略执行
 · 市场洞察              · 关键任务
 · 战略意图              · 正式组织
 · 创新焦点              · 人才 / 氛围与文化
 · 业务设计
   └──────── 差距分析 ────────┘
        · 业绩差距  · 机会差距
            价值观（基础）
                ↓
         市场结果 / 年度预算
```

**起点是差距分析（两类差距）：**
- **业绩差距**：业务结果与预期结果之间的**量化差距**（现状分析）——靠改善执行弥补。
- **机会差距**：当前业务结果与"能在市场上取得的结果"之间的差距（长期发展分析）——靠重新评估市场、做新业务设计、构建新组织能力来弥补，需在战略和执行两个层面思考。

**战略规划侧**：市场洞察（了解自身 + 宏观/客户/竞争分析）→ 战略意图（方向和目标）→ 创新焦点 → 业务设计（战略规划到执行的衔接，把战略意图具体化）。
**战略执行侧**：关键任务（可跟踪衡量的标准）→ 正式组织（组织结构、管理制度、考评标准保障关键任务落地）→ 人才/氛围与文化（关键人才与核心岗位匹配，文化激励）。
**收口**：战略执行最终对收入、成本费用、资本性支出进行量化 → 形成**年度预算**。

**BLM vs 平衡计分卡：**
- 两者相通：都把战略转成可执行、可衡量的体系，最终对接预算。
- 区别：BSC 以"战略地图四维度"为分解骨架，偏指标体系与考核；BLM 以"差距分析"为起点、领导力+价值观贯穿，偏战略制定到执行的完整闭环。深度战略落地时两者可配合：BLM 定方向与执行框架，BSC 做指标分解与考核。

## 十三、经济增加值 EVA 与修正经济增加值 REVA（价值衡量）

> 本章扩展第20章 EVA 的简版公式：补全 WACC 计算、NOPAT 调整、三种资产价值基础，并新增 REVA（修正经济增加值）这一以市场价值为基础的衡量指标。解决"会计利润为正但股东财富其实在缩水"的盲区。

### 13.1 为什么需要 EVA/REVA：传统利润指标的盲区

> 类型：判断
> 来源：业绩衡量方法研究（EVA/REVA 指标）
> 置信度：理论探讨

传统的净利润、每股收益、净资产收益率都建立在**账面价值**基础上，存在两个致命盲区：
1. **忽略权益资本的机会成本**：会计上"盈利"的企业，若利润低于股东要求的权益回报，股东财富其实在减少。
2. **报表编制有弹性**：会计利润可被盈余管理扭曲，不一定真实反映经营业绩。

EVA/REVA 的核心思想：**从股东角度定义利润**——只有税后利润超过全部资本（含权益）的机会成本，企业才真正创造了价值。

**企业资产价值的三种衡量基础**（决定了用哪种指标）：

| 价值基础 | 含义 | 对应指标 |
|---------|------|---------|
| 账面价值 | 以历史成本计量的资产价值 | 净利润/ROE 等传统利润指标 |
| 经济价值 | 对账面价值做必要调整后的价值（扣无息流动负债、研发资本化、存货计价调整、商誉调整等） | **EVA** |
| 市场价值 | 企业用资产创造的未来收益现值（含市场对管理层能力的评价） | **REVA** |

### 13.2 EVA 完整计算

> 类型：公式
> 来源：业绩衡量方法研究（EVA 指标）
> 置信度：理论探讨

$$EVA = NOPAT - k_w \times NA$$

- **NOPAT**：调整后的税后净营业利润
- **k_w**：加权平均资本成本（WACC）
- **NA**：公司资产**期初的经济价值**（投入资本总额）

**WACC（加权平均资本成本）：**

$$k_w = \frac{D_M}{D_M + E_M}(1-T)k_D + \frac{E_M}{D_M + E_M}k_E$$

| 符号 | 含义 |
|------|------|
| D_M | 公司负债总额的市场价值 |
| E_M | 公司所有者权益的市场价值 |
| k_D | 负债的税前成本 |
| k_E | 所有者权益（权益资本）的成本 |
| T | 公司边际税率 |

**NOPAT 的调整（从报告期营业净利润出发）：**
1. 加上坏账准备的增加
2. 加上存货计价方法（如后进先出法）下存货准备的增加
3. 加上商誉的摊销
4. 加上净资本化研究开发费用（R&D）的增加
5. 加上其他营业收入（含投资收益）
6. 减去现金营业税

> 实务中对营业利润的调整项目通常 5–15 项，复杂情形可多达上百项。调整目的：剥离会计弹性，使利润更接近"真正创造的财富"。

**以权益为基础的等价算法**（仅当负债与权益的市场价值分别等于其经济价值时，与上式结果一致）：
$$EVA = NI - k_E \times E \quad\text{其中}\quad NI = NOPAT - k_D(1-T)D$$
（NI = 营业净利润减税后利息费用，E 为权益经济价值，D 为负债经济价值）

### 13.3 REVA：以市场价值为基础的修正

> 类型：公式
> 来源：业绩衡量方法研究（修正经济增加值 REVA 指标）
> 置信度：理论探讨

REVA 认为：创造利润的资本总额既不是账面价值、也不是经济价值，而是**市场价值**——因为投资者在期初可按市场价卖出公司、转投同等风险资产获取 WACC 回报，不变现就等于放弃了这个机会成本。

$$REVA_t = NOPAT_t - k_w \times MV_{t-1}$$

- **NOPAT_t**：t 期末调整后的营业净利润
- **MV_(t−1)**：t−1 期末公司资产的**市场总价值** = 权益市场价值 + 调整后负债价值（总负债 − 无息流动负债）

### 13.4 EVA vs REVA：经营评价法 vs 交易评价法

> 类型：判断表
> 来源：业绩衡量方法研究
> 置信度：理论探讨

| 维度 | EVA | REVA |
|------|-----|------|
| 资本价值基础 | 资产的经济价值 | 资产的市场价值 |
| 评价方法 | 经营评价法（只看当期经营） | 交易评价法（期初买入、期末卖出的收益，关注股价变化） |
| 是否反映市场预期修正 | 否 | 是（反映市场对未来收益预测的修正） |
| 指标为正时的含义 | **仍可能**小于资本机会成本（股东财富未必增加） | 一定大于市场价值口径的资本机会成本（**股东财富必定增加**） |
| 计算前提 | 以权益单独计算时，要求负债/权益市场价值=经济价值（少见） | 无此限制，可直接按市场价值算，也可单独按权益算 |

**数值示例**（无负债公司，100 股，市价 25 元/股、经济价值 20 元/股，WACC=10%，NOPAT=225 元）：
- 期初市场价值 = 2500 元，期初经济价值 = 2000 元
- EVA = 225 − 2000×10% = 225 − 200 = **+25 元**（看似为股东创造 25 元）
- REVA = 225 − 2500×10% = 225 − 250 = **−25 元**（按市场价值实际回报仅 8%，低于股东预期的 10%，财富其实缩水）

结论：EVA 为正不代表股东满意，REVA 才真正校验"是否达到市场要求的回报"。

### 13.5 价值驱动逻辑与考核应用

> 类型：规则
> 来源：业绩衡量方法研究
> 置信度：理论探讨

**EVA/REVA 的共同价值（相对传统利润指标）：**
1. **从股东角度定义利润**：考虑全部资本（含权益）的机会成本，揭示"账面盈利却毁损价值"的企业。
2. **把股东财富与决策绑定**：EVA/REVA 持续增长 = 公司市场价值与股东财富持续增加，引导管理层关注长期价值创造而非短期利润。
3. **终结多目标混乱**：用一个统一财务指标串起所有决策，公司只剩一个目标——如何提高 EVA/REVA。
4. **经营者易理解**：从熟悉的经营利润中减去资本成本，概念简单，促使管理者在关心收入的同时管理资产。

**分层考核应用（价值驱动分解）：**
- **高层管理者 → 挂钩 REVA**：高层决定经营战略，控制范围含"实物资产价值 + 未来投资机会价值（市场价值）"，故用市场价值口径的 REVA。
- **部门经理及以下 → 挂钩 EVA**：他们能控制的是现有资产的经济价值，故用经济价值口径的 EVA。

⚠️ 使用要点：
- EVA/REVA 是**价值创造的衡量与激励工具**，实证上未必比 ROE 更能解释短期股价（受市场有效性、会计调整、数据可得性影响），但其"扣除全部资本成本"的思想对内部业绩评价和激励设计很有价值。
- 可用于把社会功能/非经营性损益从经济业绩中剥离（通过对利润做针对性调整），单纯从经济角度评价管理者业绩。
- 与责任会计衔接：投资中心业绩评价的 RI（剩余收益）就是 EVA 的简化形式（详见第19章、十章投资中心指标）。
