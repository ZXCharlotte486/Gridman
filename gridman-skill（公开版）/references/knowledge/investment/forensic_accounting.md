## 第一部分：司法会计鉴定

### 一、基本概念

**定义：** 司法会计鉴定是指在诉讼活动中，鉴定人运用会计学的原理和方法，对涉及案件的财务会计资料及相关材料进行检验、计算、验证和鉴别，并提供鉴定意见的活动。

**性质：** 属于司法鉴定的一种，鉴定意见是法定证据种类之一。

**与审计的区别：**

| 对比项 | 司法会计鉴定 | 财务审计 |
|--------|------------|---------|
| 目的 | 为诉讼提供证据 | 对财务报表发表意见 |
| 委托方 | 法院/检察院/公安/当事人 | 被审计单位/股东 |
| 法律效力 | 鉴定意见是法定证据 | 审计报告不是法定证据 |
| 标准 | 司法鉴定程序通则 + 会计准则 | 审计准则 |
| 范围 | 针对特定争议事项 | 财务报表整体 |
| 鉴定人资格 | 司法鉴定人执业证 | 注册会计师执业证 |

### 二、鉴定程序

#### 2.1 受理阶段

1. 接受委托（法院/检察院/公安机关/当事人委托）
2. 审查委托事项是否属于司法会计鉴定范围
3. 审查鉴定材料是否充分
4. 签订委托协议
5. 确定鉴定时限

**不予受理的情形：**
- 超出本机构业务范围
- 鉴定材料不真实、不完整、不充分
- 鉴定事项超出鉴定人专业能力
- 存在利害关系应回避

#### 2.2 实施阶段

1. 审查鉴定材料（原始凭证、记账凭证、账簿、报表、合同等）
2. 确定鉴定方法
3. 进行检验、计算、验证
4. 形成鉴定意见

**常用鉴定方法：**
- 账证核对法（凭证与账簿核对）
- 账账核对法（总账与明细账核对）
- 账表核对法（账簿与报表核对）
- 账实核对法（账面与实物核对）
- 比较分析法（不同期间/不同主体对比）
- 资金流向追踪法（追踪资金去向）
- 重新计算法（重新计算金额）

#### 2.3 出具鉴定意见

**鉴定意见书内容：**
- 委托方信息
- 鉴定事项
- 鉴定材料
- 鉴定过程
- 分析说明
- 鉴定意见
- 鉴定人签名及资质

**鉴定意见类型：**
- 确定性意见：能够明确得出结论
- 倾向性意见：证据不够充分但有倾向
- 无法鉴定：材料不足或超出能力范围

### 三、常见鉴定事项

| 类型 | 典型事项 |
|------|---------|
| 经济纠纷 | 合同履行金额争议、工程造价争议、投资收益计算、损失金额认定 |
| 刑事案件 | 职务侵占金额认定、挪用资金金额认定、虚开发票金额、非法集资金额 |
| 知识产权 | 侵权损失计算、许可费计算 |
| 破产清算 | 资产负债认定、债权债务确认 |
| 税务争议 | 应纳税额计算、偷逃税金额认定 |

### 四、鉴定人出庭

**出庭义务：** 经法院通知，鉴定人应当出庭作证。无正当理由拒不出庭的，鉴定意见不得作为认定事实的根据。

**出庭内容：**
- 说明鉴定过程和方法
- 回答法官和当事人的质询
- 对鉴定意见进行解释

## 第二部分：验资

### 一、基本概念

**定义：** 验资是注册会计师依法接受委托，对被审验单位注册资本的实收情况或注册资本及实收资本的变更情况进行审验，并出具验资报告的业务。

**法规依据：**
- 《中国注册会计师审计准则第1602号——验资》
- 《公司法》（2023年12月29日修订，2024年7月1日施行）

> **重大变化（2024年新《公司法》）：**
>
> 2024年7月1日起施行的新《公司法》对注册资本制度做出重大调整：
>
> **核心变化：认缴制→限期实缴制**
> - 有限责任公司全体股东认缴的出资额，应当自公司成立之日起**五年内**缴足
> - 股份有限公司发起人应当在公司成立前按照其认购的股份全额缴纳股款
> - 不再允许无限期认缴（此前可约定20年、50年甚至更长）
>
> **存量公司过渡期安排：**
> - 过渡期：2024年7月1日至2027年6月30日（3年）
> - 存量公司出资期限超过5年的，应在过渡期内调整至5年以内
> - 调整后的出资期限自2027年7月1日起计算（即最迟2032年6月30日前缴足）
> - 过渡期内未调整的，登记机关可以依法要求调整
>
> **对验资业务的影响：**
> - 验资需求将显著回升——大量存量公司需要在过渡期内完成实缴
> - 减资验资需求增加——部分公司选择减少注册资本以降低实缴压力
> - 非货币出资评估需求增加——股东可能以实物、知识产权等方式实缴
> - 注册会计师需关注：出资是否真实到位、是否存在"过桥资金"虚假实缴
>
> **仍需强制验资的行业（未变）：**
> - 银行、保险、证券等金融机构
> - 募集设立的股份有限公司
> - 法律法规另有规定的特殊行业

### 二、验资类型

| 类型 | 适用情形 | 审验内容 |
|------|---------|---------|
| 设立验资 | 公司设立时 | 各股东出资是否到位、出资方式是否合规、出资比例是否符合章程 |
| 变更验资 | 增资/减资时 | 新增出资是否到位、减资程序是否合规 |

### 三、出资方式与审验要点

| 出资方式 | 审验要点 |
|---------|---------|
| 货币出资 | 银行进账单/收款凭证、银行询证函确认、资金是否存入公司账户 |
| 实物出资 | 资产评估报告、产权转移手续、实物交接验收记录 |
| 知识产权出资 | 资产评估报告、权属证明、产权转移登记 |
| 土地使用权出资 | 资产评估报告、土地使用权证、变更登记 |

**非货币出资限制：**
- 非货币出资应当评估作价，不得高估或低估
- 法律法规规定不得作为出资的财产除外（如劳务、信用、商誉等不能出资）

### 四、验资程序

1. 了解被审验单位基本情况（章程、股东协议、工商登记等）
2. 获取验资所需资料
3. 审验出资情况
4. 编制验资工作底稿
5. 出具验资报告

**验资报告内容：**
- 标题
- 收件人
- 范围段（说明审验范围）
- 意见段（说明审验结论）
- 说明段（必要的补充说明）
- 注册会计师签名盖章
- 会计师事务所盖章
- 报告日期

### 五、常见问题判断

**Q：认缴制下还需要验资吗？**
2024年7月1日新《公司法》施行后，注册资本必须在5年内缴足（限期实缴制）。这意味着：
- 新设公司：5年内必须实缴到位，实缴时建议验资（证明出资真实性）
- 存量公司：过渡期（至2027.6.30）内需调整出资期限，之后5年内缴足
- 金融机构等特殊行业：仍强制验资
- 实务中，银行贷款、招投标、政府采购等场景仍要求提供验资报告
- **注意**：过渡期内大量公司面临"实缴或减资"的选择，验资和减资审验需求将显著增加

**Q：股东以房产出资，验资时需要什么？**
需要：①资产评估报告（具有证券期货业务资格的评估机构出具）②房产过户到公司名下的产权证明③股东会/董事会关于非货币出资的决议。

**Q：验资报告发现出资不实怎么办？**
注册会计师应当在验资报告中如实反映。如果出资严重不实，可能涉及虚假出资罪（刑事责任）或抽逃出资的民事责任。

**Q：司法会计鉴定和审计报告都能作为法庭证据吗？**
司法会计鉴定意见是法定证据种类（鉴定意见），可直接作为定案依据。审计报告不是法定证据种类，但可以作为书证使用，证明力较弱，法官可能要求补充鉴定。

## 第三部分：大股东资产转移识别框架

> 财务造假往往不是目的，而是手段。真正的目的通常是**通过造假支撑虚假繁荣，以"合法"形式将公司资产转移到大股东个人/家族手中**。
> 当公司最终爆雷时，大股东的个人资产已通过多重安排转移至追索难以触及的地方。

### 一、典型案例：恒大-许家印资产转移工程

#### 1.1 完整转移链条

```
财务造假 → 虚增利润 → 真实分红 → 境外信托 → 离婚切割 → 切断追索
   ↓          ↓          ↓          ↓          ↓
违反CAS 14  违反公司法  违反外汇   规避破产   规避连带
            分红规则   管理      法追索    责任
```

#### 1.2 核心事实

**财务造假层面：**
- 2019 年虚增收入 2139.89 亿元、虚增利润约 407 亿元
- 2020 年虚增收入 3501.57 亿元、虚增利润约 512 亿元
- 合计虚增收入 5641.46 亿元、虚增利润约 920 亿元
- 手法：将未达交付条件的房屋按"已交付"提前确认收入

**资产转移层面：**
- 累计现金分红约 533 亿元，许家印家族通过持股拿走约 500 亿元
- 2023 年 7 月许家印与丁玉梅"技术性离婚"
- 2023 年 9 月许家印被采取强制措施，许智健、许腾鹤被边控
- 通过 BVI/开曼信托结构转移境外资产

**监管处罚（2024 年 3 月）：**
- 恒大地产罚款 41.75 亿元
- 许家印个人罚款 4700 万元 + 终身证券市场禁入
- 时任高管夏海钧、潘大荣等同步处罚

### 二、五大典型转移手法

#### 手法一：超额分红（Cash-out via Dividends）

**操作逻辑：**
1. 通过财务造假虚增利润
2. 利润虚假，但分红是真金白银
3. 大股东通过持股比例拿走绝大部分分红
4. 公司账上累计未分配利润快速消耗

**识别指标：**

| 指标 | 异常阈值 | 说明 |
|------|---------|------|
| 分红率 | > 70% | 远超行业平均（30-50%） |
| 累计分红 / 累计净利润 | > 80% | 几乎把所有利润都分掉 |
| 分红率 / 经营现金流 | > 50% | 现金流不足以支撑分红 |
| 分红 + 大股东减持 | 同时大量进行 | 双重套现 |
| 分红资金来源 | 来自融资而非经营 | "借钱分红" |

**关键判断：**
- 利润是虚的，分红是真的——大股东获得真实现金
- 中小股东获得的分红被股价下跌远远抵消
- 公司因分红失血加速爆雷

**典型案例：**
- 恒大：累计分红 533 亿，许家印家族拿走约 500 亿
- 永煤、华信：爆雷前突击分红

---

#### 手法二：商票套现（Bill Financing Scam）

**操作逻辑：**
1. 强迫供应商接受商业承兑汇票（商票）代替现金付款
2. 商票通过关联方贴现，套出现金
3. 商票到期违约，损失转嫁给上游供应商和接票方
4. 公司账面"应付票据"减少，但实际并未真正付款

**识别指标：**

| 指标 | 异常表现 |
|------|---------|
| 应付票据 / 应付账款比例 | 异常高（行业平均 < 30%，异常 > 60%） |
| 商票贴现规模 | 大量商票通过关联方/特定金融机构贴现 |
| 供应商投诉 | 多家供应商公开追讨欠款 |
| 商票兑付率 | 大幅下降，违约率上升 |
| 商票接收方集中度 | 集中在少数关联方 |

**法规依据：**
- 《票据法》：商票必须有真实交易背景
- 《九民纪要》：恶意签发空头商票构成商业欺诈
- 央行 2021 年商票新规：要求强制披露商票信息

**典型案例：**
- 恒大商票：2021 年商票违约规模超千亿
- 华夏幸福、蓝光发展：商票违约引爆危机

---

#### 手法三：预售款挪用（Pre-sale Fund Diversion）

**操作逻辑（房地产专属）：**
1. 收取购房者首付款和按揭款
2. 按规定应进入"预售款监管账户"专款专用建房
3. 实际通过各种方式挪出，用于拿地、还债、分红
4. 楼盘最终烂尾，业主+银行同时受损

**识别指标：**

| 指标 | 异常表现 |
|------|---------|
| 受限货币资金占比 | 远低于预收款规模（应≥80%） |
| 预收款 / 在建工程投入比 | 严重失衡 |
| 经营现金流流向 | 大量流向投资活动（拿地）和筹资活动（还债、分红） |
| 项目竣工进度 | 严重滞后于预售进度 |
| 监管账户穿透情况 | 通过虚假工程款、关联方走账提取 |

**法规依据：**
- 各地《商品房预售款监管办法》
- 《城市房地产管理法》
- 《刑法》第 224 条合同诈骗罪、第 271 条职务侵占罪

**典型案例：**
- 恒大、融创、阳光城：大规模挪用预售款
- 部分项目业主"强制停贷"维权

---

#### 手法四：境外信托架构（Offshore Trust Structure）

**操作逻辑：**
1. 大股东在境内通过分红、减持等方式获得现金
2. 通过地下钱庄、虚假贸易、境外IPO募资等方式将资金转移境外
3. 在 BVI/开曼/百慕大设立离岸信托
4. 资金注入信托后，法律上属于受托机构，原所有人仅为"受益人"
5. 一旦设立完成，理论上对原所有人的债务追索难以触及信托资产

**信托结构典型形式：**

```
许家印（设立人）
    ↓ 注入资产
BVI 信托（受托人为离岸信托公司）
    ↓ 受益人安排
丁玉梅、子女（受益人，非所有人）
    ↓ 实际控制
开曼/百慕大底层公司
    ↓ 持有
境外房产、现金、股票等资产
```

**识别指标：**

| 指标 | 异常表现 |
|------|---------|
| 大股东境外资产规模 | 远超其境内可见收入 |
| 关联方注册地 | 大量 BVI/开曼/百慕大公司 |
| 跨境资金流动 | 通过虚假贸易、境外借款回流 |
| 上市公司境外子公司 | 与大股东个人架构高度重叠 |
| 离婚/赠与时间点 | 公司爆雷前 1-2 年集中安排 |

**法规约束：**
- 中国《外汇管理条例》：对外投资需登记审批
- CRS（共同申报准则）：境外金融账户信息自动交换
- 中国 2024 年起加强 CRS 信息共享和追溯
- 但信托一旦设立完成，追索仍极其困难

**典型案例：**
- 许家印：通过信托转移境外资产，媒体估算 60-70 亿美元（未官方确认）
- 贾跃亭：FF 上市架构 + 境外信托
- 张兰（俏江南）：CVC 信托案，2022 年新加坡法院刺破信托

#### 手法五：技术性离婚（Strategic Divorce）

**操作逻辑：**
1. 大股东与配偶在公司爆雷前办理离婚
2. 通过财产分割将大量资产划归配偶名下
3. 配偶在离婚后被认定为"独立第三方"
4. 配偶持有的资产理论上不承担前夫/妻的债务
5. 配偶通常已转移至境外，规避境内追索

**识别指标：**

| 指标 | 异常表现 |
|------|---------|
| 离婚时间点 | 公司爆雷/被立案前 6-24 个月 |
| 财产分割比例 | 配偶分得绝大部分（70%+） |
| 配偶所在地 | 离婚后迁居境外 |
| 配偶在公司任职 | 离婚前曾任职公司，离婚后切割 |
| 子女资产安排 | 同步转移给子女信托 |

**法律对抗手段（追索方）：**
- 《民法典》第 1064 条：夫妻共同债务认定（"共债共签"原则）
- 《最高法关于审理涉夫妻债务纠纷案件适用法律有关问题的解释》
- 离婚财产分割可以被认定为恶意逃避债务而被撤销
- 但实操中举证难度极大

**典型案例：**
- 许家印 - 丁玉梅：2023 年 7 月离婚，9 月许家印被边控
- 周立波夫妇、王宝强夫妇：娱乐圈典型案例（性质不同但操作相似）

### 三、综合识别框架

#### 3.1 时间线识别法

观察大股东相关操作的**时间序列异常**：

```
T-3 年：财务数据开始异常（收入虚增、利润畸高）
T-2 年：超额分红、大股东减持开始
T-1 年：商票大量发行、预售款挪用加剧
T-0.5 年：技术性离婚、子女信托设立
T 年：公司爆雷/被立案
T+0：大股东被采取强制措施时，资产已基本转移完毕
```

#### 3.2 资金流穿透法

**问题树：**

1. **公司的钱去哪了？**
   - 分红？→ 谁拿走了？比例多少？
   - 关联交易？→ 交易对方是谁？是否公允？
   - 商票/应付？→ 是否违约？接票方是谁？
   - 预售款？→ 进入监管账户了吗？提取记录是什么？

2. **大股东个人的钱从哪来？**
   - 分红收入是否覆盖其消费/投资？
   - 是否存在不明来源的境外资产？
   - 减持收入流向哪里？
   - 是否通过关联方借款套现？

3. **资金的最终去向？**
   - 境内还是境外？
   - 是否进入信托？
   - 是否登记在配偶/子女名下？
   - 是否用于购置不动产/奢侈品/船舶飞机？

#### 3.3 法律层面的追索路径

**追索方可用工具：**

| 工具 | 法规依据 | 适用情形 |
|------|---------|---------|
| 股东出资责任 | 《公司法》第 28 条 | 出资不实、抽逃出资 |
| 法人人格否认 | 《公司法》第 23 条 | 滥用法人独立地位 |
| 关联交易撤销 | 《公司法》第 22 条 | 不公允关联交易 |
| 破产撤销权 | 《破产法》第 31-33 条 | 临近破产前的财产处置 |
| 个人破产追溯 | 部分地区试点 | 大股东个人破产 |
| 跨境资产追索 | 司法协助条约 | 境外资产追回 |
| 刑事追赃 | 《刑法》第 64 条 | 涉嫌犯罪所得追缴 |

**实际追索的难点：**
- 信托一旦合法设立，追索极其困难（需证明欺诈设立）
- 跨境追索受司法管辖权限制
- 时效问题（破产撤销权 1 年内）
- 举证责任倒置难（普通债权人难以获得证据）

### 四、给审计师/分析师的实务建议

#### 对于审计师

1. **保持职业怀疑**：对超额分红、大额商票、预收款异常等保持高度警惕
2. **关注关键审计事项**：将收入确认、应付票据、预售款监管列为 KAM
3. **执行函证程序**：对预售款监管账户、关联方交易执行充分函证
4. **关注治理层异常**：大股东离婚、子女资产安排等"软信号"
5. **必要时撤出**：发现重大舞弊迹象时及时撤出业务

#### 对于投资者/分析师

1. **看现金流而不是利润**：经营现金流 / 净利润 < 0.8 长期持续是危险信号
2. **看分红而不是声明**：超额分红 + 大股东减持 = 退场信号
3. **看商票而不是合同**：商票规模/违约率反映真实经营状况
4. **看治理而不是公关**：大股东及家族异常操作是先行指标
5. **建立预警清单**：定期跟踪上述五大手法的指标变化

#### 对于监管机构

1. **强化分红约束**：限制亏损/造假企业的分红行为
2. **强化商票披露**：要求实时披露商票信息和违约情况
3. **强化预售监管**：穿透式监管预售款使用
4. **强化跨境合作**：CRS 信息共享、跨境司法协助
5. **强化追溯机制**：拉长破产撤销权时效、建立股东穿透追责制度

### 五、总结：合规外衣下的系统性掏空

**核心警示：**

许家印案的本质不是单点造假，而是**完整的资产转移工程**。每一步单独看可能都"合规"，但整体串起来就是对公司、债权人、中小股东、购房者的系统性掏空。

这种模式的危害比单纯造假更大：
- **更隐蔽**：分散在多个环节，难以被单一审计程序发现
- **更难追溯**：每一步都有合法外衣，事后举证极其困难
- **更具传染性**：一旦被认为"成功"，会被其他企业模仿
- **更难赔偿**：资产已转移境外，国内追索徒劳

**识别这种模式需要：**
- 跳出单一报表视角，看完整的交易链条
- 跳出当期数据，看 3-5 年的时间序列
- 跳出会计科目，看资金的最终流向
- 跳出公司层面，看大股东及家族的整体安排

古立特记住这个教训。在帮用户做尽调、做审计、做投资分析时，永远问一句："**钱最后到了谁手里？**"
