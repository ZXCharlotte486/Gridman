## 一、尽职调查（Due Diligence）

### 尽调范围确认

开始尽调前，先明确：
- 目标公司：名称、行业、商业模式
- 交易类型：平台收购、Add-on、成长期股权、重组、剥离
- 交易规模/复杂度：决定尽调深度
- 已知关注点：客户集中度、监管、环保等
- 时间节点：LOI / 交割目标日期

### 七大尽调工作流

**财务尽调（Financial DD）**
- 收益质量（QoE）：收入和 EBITDA 调整项
- 营运资本分析：正常化 vs 实际
- 债务及类债务项目
- 资本支出（维护性 vs 成长性）
- 税务结构和风险敞口
- 审计历史和会计政策
- 备考调整（run-rate、协同效应）

**商业尽调（Commercial DD）**
- 市场规模和增长（TAM/SAM/SOM）
- 竞争定位和市场份额
- 客户分析：集中度、留存率、NPS
- 定价能力和合同结构
- 销售管道和积压订单
- 销售效率

**法律尽调（Legal DD）**
- 公司架构和组织图
- 重要合同（客户、供应商、合作）
- 诉讼历史和待决索赔
- 知识产权组合和保护
- 监管合规
- 雇佣协议和竞业禁止

**运营尽调（Operational DD）**
- 管理团队评估
- 组织架构和关键人员风险
- IT 系统和基础设施
- 供应链和供应商依赖
- 设施和不动产
- 保险覆盖

**人力资源尽调（HR DD）**
- 组织架构和人员趋势
- 薪酬基准
- 福利和养老金义务
- 关键员工留任风险
- 工会/劳动协议

**IT/技术尽调**（科技型企业）
- 技术栈和架构
- 技术债评估
- 网络安全状况
- 数据隐私合规（GDPR、个人信息保护法）
- 产品路线图和研发投入
- 可扩展性评估

**ESG 尽调**（适用时）
- 环境负债
- 监管合规历史
- ESG 风险和机会

### 行业专项补充

| 行业 | 额外关注点 |
|------|-----------|
| 软件/SaaS | ARR 质量、客户留存分析、SOC2、CAC/LTV |
| 医疗健康 | 监管审批、报销风险、支付方结构 |
| 工业制造 | 设备状况、环境修复、安全记录 |
| 金融服务 | 监管资本、合规历史、信用质量 |
| 消费品 | 品牌健康度、渠道结构、季节性、库存管理 |

### 红旗清单

尽调中发现的问题按严重程度分级：
- **交易终止级**：无法通过谈判解决的根本性问题
- **重大级**：影响估值或交易条款
- **可管理级**：有缓解路径，不影响交易

### 产业政策合规判断

> 类型：流程
> 来源：《产业结构调整指导目录（2024年本）》（发改委令第7号，2024年2月1日起施行）
> 置信度：实务经验

[场景：IPO 尽调/项目投资/并购重组时，判断标的企业的主营业务是否符合产业政策]

**判断规则：**
- 标的业务属于**鼓励类** → 正常推进，可享受信贷/土地/税收等政策支持
- 标的业务属于**限制类** → 禁止新建投资，现有产能需改造升级，影响融资和审批
- 标的业务属于**淘汰类** → 禁止投资，金融机构应停止授信，需限期淘汰
- 目录之外（既不鼓励也不限制）→ 属于**允许类**，按一般规定执行

**查询方式：** 联网访问国家发改委官网查最新版本（目录会修订，不依赖缓存）。当前有效版本为2024年本，网址：https://www.gov.cn/gongbao/2024/issue_11206/202403/content_6937897.html

**IPO 审核重点关注：**
- 发行人主营业务是否属于限制类或淘汰类（属于则构成实质性障碍）
- 发行人是否存在限制类/淘汰类产能（即使非主营，也需披露整改计划和时间表）
- 募投项目是否符合鼓励类方向（影响审核通过率）

⚠️ 本目录不入知识库全文——条目太多且会修订。需要判断具体业务分类时联网查询原文。

### 尽调流程与执行原则（中式创投视角）

> 类型：流程
> 来源：尽职调查白皮书（创投实务）
> 置信度：实务经验

[场景：境内股权/创投项目尽调，前面"七大工作流"是分领域清单，本节是整体作业顺序和心态准则]

**尽调标准作业顺序（不可跳步）：**

```
立项 → 成立工作小组 → 拟定调查计划 → 整理/汇总资料 →
撰写调查报告 → 内部复核 → 递交汇报 → 归档管理 → 参与投资方案设计
```

- 拟定计划**必须**建立在已了解投资目的和目标企业组织架构的基础上，不得先发清单后想目的。
- 尽调报告**必须**通过内部复核后才能提交，未复核的结论不得对外。
- 重大项目尽调周期通常 3-6 个月。

**五种核查方法（应交叉使用，单一方法不足以定论）：**

| 方法 | 抓手 |
|------|------|
| 审阅文件资料 | 工商注册、财报、业务文件、法律合同——发现异常和重大问题 |
| 参考外部信息 | 网络、行业杂志、业内人士、专业数据库 |
| 相关人员访谈 | 各层级各职能人员 + 中介机构 |
| 企业实地调查 | 厂房、土地、设备、产品、存货等实物 |
| 小组内部沟通 | 不同背景成员交叉印证 |

**执行原则（决策心态，写进底稿）：**

- **证伪优先**：站"中立偏疑"立场，按"问题→怀疑→取证"展开，不预设企业是好的。
- **事必躬亲**：关键事实必须亲赴现场核实，不得依赖道听途说或管理层一面之词。
- **突出重点**：先锁定技术/产品/收入的特异性疑点，不眉毛胡子一把抓。
- **以人为本**：在财务、产品、市场之外，重点评判管理团队的能力与**诚信**。
- **横向比较**：与同行业（尤其已上市可比公司）对标，脱离行业的单点数据不下结论。

### 财务与业务尽调核查抓手（红旗触发点）

> 类型：判断表
> 来源：尽职调查白皮书 + 私募股权尽调实务
> 置信度：实务经验
> 关联：financial_fraud.md - 收入造假/资金穿透；7.5 IPO问询预测

[场景：财务/业务尽调时，按抓手逐项核查，命中即标红旗深挖]

| 核查领域 | 抓手与阈值 | 命中后动作 |
|---------|-----------|-----------|
| 供应商集中 | 取前 10/前 5 大供应商，**同一实际控制人合并计算**；单一供应商占比 >50% 视为严重依赖 | 核查是否有备选供应安排、长期供货合同、价格稳定性 |
| 客户集中 | 取前 10/前 5 大客户，同一实控人合并计算；占比过高视为依赖 | 追查销货合同、发票、出库单、银行进账单，必要时**函证** |
| 期末收入异常 | 会计期末销售收入异常增长 | 追查收入确认凭证，判断是否虚开发票、虚增收入 |
| 关联采购/销售 | 高管、核心技术人员、持股 5% 以上股东在供应商/客户中的权益 | 抽查不同时点关联交易价 vs 当时市场公允价，判断是否利润输送/资金转移 |
| 出口收入 | 大量出口 | 函证或走访海关，确认销售真实性和收入确认时点 |
| 存货 | 原材料周转天数异常、实地查残次冷背呆滞 | 评估积压风险和跌价 |
| 利润 vs 现金流背离 | 净利润为正但经营活动现金流量净额很小甚至为负 | 通常因应收账款、其他应收款大幅增长，提示利润含金量低、实际财务状况弱于利润表 |
| 税收优惠依赖 | 税收优惠/财政补贴是否合规、是否持续 | 分析对未来业绩的依赖程度 |

⚠️ 应收账款在收回前不增值反而占用成本；存货因"能通过主营业务成本影响利润"且品种繁杂不易核实，**被操纵可能性最大**——这两项是财务尽调的重点照看对象。

### 现金流量结构判断企业生命周期

> 类型：判断表
> 来源：尽职调查白皮书（现金流量结构分析）
> 置信度：实务经验

[场景：拿到三类现金流净额的正负组合，快速判断企业所处生命周期，用于估值假设和风险定性]

按"经营/投资/筹资"三类现金流净额的正负号组合判断：

| 经营 | 投资 | 筹资 | 判断阶段 | 含义 |
|:---:|:---:|:---:|------|------|
| − | − | + | 初创期 | 尚无造血能力，靠融资投入形成产能、开拓市场 |
| + | − | + | 高速发展期 | 经营回笼现金，但仍需大量追加投资，自身现金流不够还要外部补充 |
| + | + | − | 成熟期 | 销售稳定、进入投资回收期，同时偿还外部资金维持资信 |
| − | + | − | 衰退期 | 市场萎缩、经营现金流入小于流出，被迫收回投资偿债 |

⚠️ 判断阶段后要回看估值假设：对"经营−、筹资+"的初创/衰退期企业用稳定增长 DCF 会失真，应结合阶段重设增长和折现假设。

### 尽调方式与访谈/现场核查抓手

> 类型：流程
> 来源：私募股权投资和尽职调查手册（尽调方式）
> 置信度：实务经验

[场景：执行网络/访谈/现场尽调时，按抓手取证，而非走形式记录]

**网络调查抓手（专挖负面）：**
- 看公司官网**只看可能出问题的地方**（奖惩通知、员工活动、重大事件、丑闻），不看企业自夸。
- 查老板身世判断经营风格——**学者创业**重点查知识产权权属和人事独立性；**销售出身**查研发团队是否稳定、销售是否合规；**资本市场出身**查是否为资本运作做高风险操作。
- 诉讼用公检法系统查：报告期内与供应商、客户均有多起纠纷/诉讼，至少说明不厚道甚至不诚信。

**访谈抓手（按对象设目标）：**
- 访谈应像**有目的的聊天**，不得对着问题逐条念逐句记——会让对象紧张、信息量极少；问题须事先整合到融会贯通，看似不经意实则精心准备。
- 供应商一般都说好话（"只有不好卖的产品，没有买不到的原料"）→ 要从一片大好中查交易是否真实、供应商是否在压力下配合。
- 客户访谈**配合度本身就是信号**：客户不配合甚至不见，除大国企牛或做了亏心事，多半是发行人地位不高；客户配合且评价正面，说明合作互赢、不存在单方依赖。
- 竞争对手不举报、不抹黑就不错，难做理性访谈——改用其宣传资料、招投标文件、公开访谈侧面判断。
- 发行人安排的专家访谈，独立性和真实性大打折扣，资料可采但须独立判断。

**聊天补充对象：** 公司保安/食堂/司机/保洁/前台（问工资社保待遇、对老板评价）、当地出租车司机/餐馆老板（小地方信息更透）、离职员工。

**现场调查抓手（必须亲赴，不得只在空调房看资料）：**
- 外地子分公司**有几家看几家**——先确认是否真实存在、账上设备资产是否虚构。
- 生产车间多去几次：看核心技术应用、设备是否正常运转、车床型号代表的技术水平、旺季是否满负荷、拉货车是否排队。⚠️ 实务中有提前打生产批号造假的例子。
- 仓库：存货是否账实一致，按成品批号更新判断销售情况。
- 细节看管理：厕所是否卫生、消防器材是否完备、迎送车队水平——脏乱的厕所很难配套规范的内控。

⚠️ 铁律：**项目第一眼看到的亮点，就是日后要重点核实的风险点**；"做完了"和"做到位"之间差距巨大，决定结论停在表面还是有内涵。

### 企业价值分析框架与分阶段取舍

> 类型：流程
> 来源：私募股权投资和尽职调查手册（企业价值分析框架）
> 置信度：实务经验

[场景：确定一个项目该把尽调火力集中在哪，避免眉毛胡子一把抓]

价值与风险由四类内部因素 + 三类外部因素交叉验证，**不得孤立看单一指标**：
- 外部：宏观经济环境、行业发展阶段、行业竞争格局。
- 内部：管理团队（根基）→ 战略与业务模式 → 财务指标（表象/结果）；外加股权结构与治理。

**因果链：先有管理团队，才有业务模式，结果体现在财务指标。** 财务指标出问题，根子一定在团队或业务模式上——四者反复交叉验证。

**按项目阶段分配尽调重点（必须取舍）：**

| 阶段 | 核查重点 |
|------|---------|
| 种子/天使 | 团队 + 商业模式 |
| VC | 进一步验证商业模式可行性 + 运营指标 |
| PE | 财务成长性 + 组织能力 + 股东治理 |
| 已上市 | 全面对标，财务规范性要求最高 |

### 从财务报表看核心竞争力（毛利率 + 上下游收付款）

> 类型：判断表
> 来源：私募股权投资和尽职调查手册（财务竞争力）
> 置信度：实务经验

[场景：拿到三表，快速判断企业对上下游的话语权和真实竞争力]

竞争力主要反映在 **毛利率** + **上下游收付款能力**：

| 方向 | 看什么 | 强势信号 | 弱势信号 |
|------|--------|---------|---------|
| 对下游（销售） | 比较收入 vs 预收账款+应收账款+应收票据 | 不打款不发货 → 大额**预收账款**（如茅台） | 回款周期长 + 应收账款高 + 被压质保金（如部分环保工程公司） |
| 对上游（采购） | 比较付款 vs 预付账款+应付票据+应付账款，并对照存货 | 欠供应商的钱远高于存货（如格力存货86亿、应付-预付310多亿） | 被供应商要求预付、账期短 |

⚠️ 上下游都强势（应付高、预收高、存货低）的企业，竞争力最硬（格力、茅台、老干妈）。

### 成长性三层递进分析（速度 → 质量 → 驱动力）

> 类型：流程
> 来源：投资尽职调查（戚威，财务尽调）
> 置信度：实务经验
> 关联：本章 成长速度四模型、企业财务特征五型

[场景：财务尽调判断企业有没有投资价值，按三层逐级筛，前一层不过关就不必看下一层]

成长性拆成三个层层递进的指标，**一票否决式推进**：

1. **成长速度**（现状）：用**收入增速**衡量，不用净利润——净利润受成本/费用/减值/投资损益/税率影响，可调节空间太大。速度**至少高于行业平均**；企业越大要求越低。软件企业参考门槛：收入<2000万年增速≥去年的2~3倍；<6000万≥1~2倍；<1亿≥60%~80%；>1亿≥30%~50%。
2. **成长质量**（现状）：增长须来自**核心业务**（投入资源最多、占营收主体、代表未来方向）。
3. **成长驱动力**（原因）：挖掘是什么在支撑增长、未来能否持续。

⚠️ 速度是质量前提下的速度，质量是速度基础上的质量；速度质量反映现状，驱动力挖掘原因。**缺速度或缺质量，企业都没有投资价值。**

### 成长速度四模型（订单 × 收入 × 回款）

> 类型：判断表
> 来源：投资尽职调查（戚威，成长速度）
> 置信度：实务经验

[场景：拿到订单、收入、回款三组增速，判断企业产业链地位和投资价值]

为什么三个都要看：**订单**金额固定、不受会计政策影响，验证收入真实性和成长性（收入涨而订单不涨=透支未来）；**回款**是现金流，验证真实性 + 反映产业链地位（没回款，收入只是报表数字）。

| 模型 | 订单 | 收入 | 回款 | 判断 |
|------|:---:|:---:|:---:|------|
| 理想型 | 快 | 快 | 快 | 三者趋势统一，竞争力强，**投资目标** |
| 低门槛型 | 快 | 快 | 慢 | 有创新门槛但竞争力不强、产业链地位低 |
| 蓄势待发/垫资型 | 快 | 无增 | 无增 | 初创期（已验证产品、订单产生未交付）=**好早期标的**；成熟期工程类（实施回款周期长、垫资严重）=商业模式差 |
| 透支未来型 | 慢/放缓 | 增 | 增 | 业务遇瓶颈在**吃老本**，只剩财务价值、丧失业务价值 |

⚠️ 跟大企业做生意往往订单大但回款差。利润表风光、靠垫资活着的企业，现金流见真章——能实实在在带来现金的业务才算好。

### 成长质量：高增速 + 高门槛 + 高投入

> 类型：判断表
> 来源：投资尽职调查（戚威，成长质量）
> 置信度：实务经验

[场景：判断增长是否来自有价值的核心业务]

核心业务成长质量好的三标准：

| 标准 | 判断规则 |
|------|---------|
| 高增速 | 核心业务占比应≥30%，增速超传统业务，1~2 年有赶超可能。⚠️ 餐馆主要靠卖餐具增长，要么餐饮没竞争力，要么根本不是餐馆 |
| 高门槛 | 核心业务毛利率应高于行业平均；同模式下越高越好。但毛利率要按行业看（生物医药/半导体天生高毛利），且向下游延伸承接项目而主动降毛利是正常策略，不等于竞争力减弱 |
| 高投入 | 技术类企业核心业务须有足够研发投入；要分清是真研发还是外包定制（结合人均产值、薪酬水平验证） |

**速度优先（牺牲部分质量）仅在满足三条件时成立，且只适合 C 端互联网（电商/在线教育）：**①规模增大能带动产品质量提升；②规模增大能降边际成本；③规模增大能提黏性、降销售成本。**B 端企业不适用**——研发周期长、采购决策多因素、市场分散无规模效应，简单降价换不来市场。

### 企业财务特征五型画像

> 类型：判断表
> 来源：投资尽职调查（戚威，成长驱动力）
> 置信度：实务经验

[场景：按成长驱动力给企业归类，用对应财务特征反向验证创始人陈述]

特征仅在**同行业/同产业链**内可比，跨行业不可比。

| 类型 | 核心驱动 | 收入曲线 | 毛利率/研发 | 关注点 |
|------|---------|---------|-----------|--------|
| 原创技术型 | 自主知识产权、绝对领先 | 前几年低，产品成功后爆发 | 高毛利；研发周期3年+、累计几千万~上亿、占收入≥30% | 研发失败风险、知名机构是否背书、应收占比低且稳、常有预收 |
| 应用技术型 | 通用技术延伸、解决方案 | 前快后平缓 | 毛利低于原创但净利未必低；研发约1年、几百万~上千万、占收入10%~20% | 市场聚焦局限、壁垒在应用转化经验 |
| 成本控制型 | 低附加值外包、管理控本 | 成立即有收入，增速稳无爆发 | 毛利<30%、净利<10% | 门槛在流程管理、稍不注意就亏 |
| 销售驱动型 | 大量销售投入/资源 | 规模增速尚可 | 技术投入低、竞争力弱 | 创始人善营销精包装、爱迎合机构包装成拟上市 |
| 平台模式型 | 商业模式聚集资源、数据为核心 | 成交额增速极快、长期亏损"贴钱" | — | "规模就是一切"，看模式逻辑+执行力+应变 |

⚠️ 真研发判断：创始人背景平平、无外部投资、设立不到3年、30人团队却号称行业顶尖技术——逻辑不成立（技术那么易得门槛必不高；门槛真高这种团队短期拿不到）。研发费用资本化在早期项目少见，一旦出现且核算不规范，基本可判定有粉饰报表动机。

### 财务粉饰与重大财务风险识别（VC 视角）

> 类型：判断表
> 来源：投资尽职调查（戚威，财务风险防范）
> 置信度：实务经验
> 关联：financial_fraud.md - 收入造假手法

[场景：早期项目财务尽调时区分"粉饰"与"造假"，并锁定可能致命的重大风险]

财务粉饰目的是**融资**，故站 VC 角度**只查调增方向**（虚增收入/利润），节税性的调低不在考虑范围。

| 粉饰类型 | 手法 | 判断 |
|---------|------|------|
| 虚增收入 | 不按准则确认（按开票/无验收单确认）；关联方虚增；与代理商私下退货协议或压货 | 审阅期内做法统一、不反复，视为正常不算粉饰；关联方交易占比高则不能证明产品竞争力；私下退货/压货接近造假 |
| 虚增利润 | 混淆成本与费用（项目实施人工计入研发费用）；不当资本化（无形资产/成本挂存货） | 早期项目这类操作成本高，一旦出现且核算不规范，业绩真实性要打问号 |
| 粉饰业务 | 把低门槛业务包装成高门槛（收入分类误导、调整订单内容掩盖真实业务） | 全部合同套解决方案模板、把贸易收入划成解决方案收入 |

**重大财务风险（可能伤及持续经营）：**

| 风险 | 判断标准 |
|------|---------|
| 现金流风险 | 经营净现金流为负、资产负债率高**不必然**是问题；要警惕管理混乱/规划不合理导致的持续紧张且短期无解（应收高企+垫资严重还在大规模扩张招人） |
| 重大税务风险 | 偷漏税金额大到可能担**刑责**才是重大；早期小问题（发票主体不符等）只要非刻意偷逃可包容。但税务有瑕疵的创始人要谨慎评估其风控意识 |
| 重大内控风险 | **可包容**：员工账户收款、现金收款、第三方回款、未签合同直销等普遍瑕疵；**需重点防范**：关联方代收付、实际控制人长期占用企业资金 |

### 管理层盈利预测核查

> 类型：流程
> 来源：投资尽职调查（戚威，盈利预测分析）
> 置信度：实务经验

[场景：融资方提供的财务预测往往是估值直接来源，需评估其可靠性]

⚠️ 几乎没有初创企业按最初预测速度增长——**财务预测的数值结果次要，制作过程才有价值**：它是观察管理层真实想法（市场定位、产品规划、拓展节奏、信心）的窗口；需承担对赌义务时再爱吹的创始人也会谨慎。

核查动作：
- **修正关键财务假设**：用历史值替换乐观假设。⚠️ 新游戏 ARPU 设60元比老游戏42元高43%且无法验证 → 按历史42元计。
- **变更不合规的会计核算方法**：按收款额计收入而非按准则分摊确认 → 调整（除非投资人本就以经营净现金流为衡量口径）。
- **模拟合规性成本**：补足未足额缴纳的税款、社保公积金等，这些会高估实际利润率。
- **盈亏平衡预测**：对贴钱抢市场的企业，核查固定/变动成本拆分、有效区间、单价与单位变动成本假设是否合理，存疑处用敏感性分析框范围。

### 北极星指标（OMTM）拆解业务

> 类型：流程
> 来源：投资尽职调查（戚威，非财务数据分析）
> 置信度：实务经验

[场景：财务数据是结果/滞后指标，需用业务先导指标找增长原因]

北极星指标（One Metric That Matters）= 企业当前阶段最关键、能串联财务与业务的唯一指标。合格的北极星指标应：可明确计量、体现核心价值导向、是营业收入的**先导性指标**。

用法：访谈确认北极星指标 → 拆解为乘法影响因子 → 要求企业按因子提供数据 → 二次拆解分析，找到增长原因和核心竞争力。⚠️ 示例：电竞平台"每日赛事总参与人数 = 每日成功创建赛事数 × 赛事曝光率 × 每日活跃人数 × 人均参与赛事数"，逐因子对比同行析优劣。

### 财务尽调认知流程与重要性阈值

> 类型：流程
> 来源：投资尽职调查（戚威，财务尽调流程）
> 置信度：实务经验

[场景：在有限时间内把最重要的事项查清，避免在小问题上纠结]

**认知流程四步（不得在第一步就急着核真实性）：**
1. 搞清基本财务情况（收入/增速/毛利/费用/利润/应收/存货，分产品）。
2. 厘清钩稽关系：收入涨→应收/存货/销售费用是否同比例涨？账龄是否恶化？重点从业务角度找财务与业务的钩稽。
3. 锁定重要性事项：固定事项（收入/成本/期间费用）；金额>百万或占资产/收入>10%；变动>30%；钩稽异常。
4. 验证真实性：**以钩稽关系验证为主、真实性验证为辅**（伪造银行对账单只能靠函证等外部证据）。系统性造假靠重要性事项抽样测试（抽3项无假 + 20项钩稽合理 → 认定真实）。

**工作流程五步：** 业务访谈挖逻辑 → 财务报表分析 → 明细账+业务数据（科目余额表/序时账/销售合同） → 管理层确认（抽凭/穿行/第三方访谈补证） → 修正盈利预测。

### 财务不规范企业的处理（定量还原，不定性否决）

> 类型：规则
> 来源：投资尽职调查（戚威，如何对待财务不规范的企业）
> 置信度：实务经验

[场景：早期企业财务不规范是常态，不应一刀切否决]

- 早期企业提前确认收入、成本费用归集不准是正常现象，**应定量分析而非定性否决**（提前确认收入比重多少？还原后实际增长如何？）。
- 投资部门与尽调部门割裂会导致一方过分乐观、一方过分悲观——结论必须财务与业务结合。
- ⚠️ 处理"报表不对但说不清哪不对"的项目：不纠结报表数字，抓业务成长性实质——按销售合同台账找核心业务，抽样交付周期推算收入，查工资表+外采推算成本，按比例推算费用，测算真实业绩。若成长性和利润率可接受，至少说明非故意粉饰。

### 创始人识人评价方法

> 类型：判断表
> 来源：投资尽职调查（戚威，对人的评价方法）
> 置信度：实务经验

[场景：评价创始人/团队，落脚点是"是否影响项目成功"，而非贴标签]

**识人三大误区（不得犯）：**
- 过分关注无客观标准的事项（野心/情怀/格局）——定义模糊、无法验证、不能支撑决策。
- 把"防骗"当尽调目的——绝大多数创始人是不完美而非骗子，防骗对正常投资参考意义不大。
- 超越商业边界（看相算卦、纯文学历史心理学评判）——投资人无专业能力评，且对决策无意义。

**评价标准两条件（缺一不可）：**①能通过**具体事件**体现验证，不靠主观表达；②有相对客观标准，不因个人感受而结论摇摆。⚠️ 可由"多次无理由违约"得出"诚信缺失"；但不能由"力排众议坚持"得出"高瞻远瞩"——项目一失败，"坚持"就变成"刚愎自用"。野心/格局这类只能是企业家**自律**的方向，不能拿来衡量别人。

**硬性标准（显性，可核）：**

| 因素 | 重要性 | 判断 |
|------|:---:|------|
| 工作背景（含创业经历） | ★★★★★ | 工作经历与项目的关联性（硬科技几乎必须有行业背景）；销售出身重开拓、技术出身重细节、复合背景兼顾；核实任职真实性（最高职务/管人数/具体项目）；失败创业经历最能体现特点 |
| 学历背景 | ★★★★ | 必要不充分。To B 硬科技几乎是必要条件（厚积薄发，无取巧空间），优异学术背景能吸引人才和客户认可；但不唯学历论 |
| 家庭背景 | ★★★ | 特定情况构成实质影响：父母资源是双刃剑（助力 or 虚假繁荣掩盖问题）；家庭资产影响创业动机；婚姻影响股权稳定。⚠️ 夫妻同任要职的创业难吸引优秀合伙人（团队天然不对等），常以一方退出收场 |
| 年龄 | ★★★ | 非直接限制：年轻团队重点查行业认知深度和资源；年长创始人额外考虑健康、退休预期、创业动力 |

**软性标准（隐性）：** 成功欲望、战略/管理/营销能力、个人品德——无法量化，只能通过具体事件体现。

## 二、估值方法

### 四种核心方法

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| DCF（现金流折现） | 稳定盈利企业 | 理论最严谨，反映内在价值 | 假设敏感性高，终值占比大 |
| 可比公司法（Comps） | 有上市可比公司 | 市场化、直观 | 可比性判断主观 |
| 可比交易法（Precedents） | 有近期并购案例 | 反映控制权溢价 | 案例可得性有限 |
| 资产基础法 | 重资产企业、清算 | 简单直接 | 忽略无形价值和盈利能力 |

实务中通常用"足球场图"（Football Field）展示多种方法的估值区间，取交集作为参考范围。

### 商业模式决定估值方法选择

> 类型：判断表
> 来源：私募股权投资和尽职调查手册（估值）
> 置信度：实务经验

[场景：拿到一个标的，先按商业模式选主估值法，再选辅助法]

| 企业类型 | 主估值法 | 辅助法 |
|---------|---------|--------|
| 重资产型（传统制造） | 净资产法（PB） | 盈利法 |
| 轻资产型（服务业） | 盈利法（PE） | 净资产法 |
| 互联网企业 | 市销率（PS，以用户数/份额为远景） | — |
| 新兴行业/高科技 | 市销率（PS，以市场份额为远景） | — |

⚠️ 各种估值法**均不可单独使用**，至少配合 2~3 种联合研判；绝对值无实战意义，重点是比较（尤其相似企业），跨行业比较无意义。作为投资人应尽可能**悲观**估计——乐观主义是风险和灾难的催化剂。

### 市值量级与市值比较

> 类型：规则
> 来源：私募股权投资和尽职调查手册（市值）
> 置信度：实务经验

[场景：用市值做相对参照，判断标的是否被高估]

市值 = 股价 × 总股本，意义在于**量级比较而非绝对值**。国际量级标准：100 亿美元=优秀成熟大型企业，500 亿美元=国际化超大型，千亿=至高地位。

三种市值比较参照物：① 同股同权跨市场（AH 股比价）；② 同类企业（三一重工 vs 中联重科）；③ 相似业务拆分后比（上海家化 vs 联合利华）。⚠️ 收入在同一量级而市值量级差很多，是高估的**警示信号**（高估低估不直接构成买卖依据，但要警惕）。

### PB / PE 估值前的健康体检调整

> 类型：判断表
> 来源：私募股权投资和尽职调查手册（估值健康体检）
> 置信度：实务经验
> 关联：本文件第一章 财务粉饰识别

[场景：用 PB/PE 前先剔除有问题的净资产/利润，反映真实值]

**PB 法（剔除虚高净资产）：**

| 项目 | 检查 |
|------|------|
| 应收账款 | 周转天数30~60天合理，>90天高度警惕（需90天以上账期赊销=对下游缺谈判力）；坏账准备是否低估（有大量逾期、账龄超1年未100%计提） |
| 库存 | 真实性（獐子岛扇贝式失踪）、跌价准备是否足够（生物/快迭代/时尚库存易出事） |
| 商誉及无形资产 | 价值极不确定，无特别理由说明含金量的应从净资产中**剔除** |
| 固定资产 | 历史成本法 vs 公允价值；折旧方法（加速折旧净值远低于平均法）；原值真实性 |
| 负债/长期投资 | 是否隐瞒债务致净资产虚高；长期股权投资按市场估值还原 |

**PE 法（管理盈利信号）：** 负面信号（业绩不达标时调高当期利润）——调低坏账准备、调低存货跌价、突改折旧方法/年限、减各种准备金、研发费用资本化大幅上升、销售费用突降、最后一月/季销售显著高于前期、客户预收款大降、销售勉强达标/正好不跌；正面信号（业绩远超预期时平滑/隐藏利润，与负面相反）。⚠️ 用 PE 前净利润须剔除非经常性损益。

### 一级市场实战估值门槛（PB / PE / 现金流的买卖纪律）

> 类型：公式
> 来源：私募股权投资和尽职调查手册（估值门槛）
> 置信度：实务经验

[场景：给定企业 ROE/盈利，按"盈利不再增长"的保守假设算可接受的买入价上限和卖出信号]

⚠️ 所有买入门槛都基于"盈利不再增长"的保守假设——因为成长性极难预测且常被高估（巴菲特：极少企业能长期年增15%）。一级市场公司规模更小、更不规范，应用**更严格**标准。

| 方法 | 关系式 | 买入门槛 | 卖出信号 |
|------|--------|---------|---------|
| PB | ROI = ROE / PB | 长期 ROE 20% 对应 3 倍 PB（=20%/6.7%）；ROE 25% 极限 4 倍 PB | PB > 10 倍（此时即便 ROE 25%，当期回报仅 2.5%，低于长期国债） |
| PE | ROI = 1 / PE | 优秀公司基准 16 倍（6.7% 回报）、极限 20 倍 | PE > 40 倍（回报仅 2.5%，不划算） |
| 现金流 | ROI = 三年平均自由现金流 / 估值 | 优秀公司 ROI 不低于 5%、最少 4%（对标优质债 4%~5%） | ROI < 2%（低于货币基金） |

（自由现金流 = 税后净营业利润+折旧摊销−资本支出−营运资本增加 = 经营现金流−资本性支出。推荐用过去 1~5 年的**静态**平均，不赌未来。）

⚠️ PB/PE 须找企业自身至少 5 年（或一个完整周期；新股至少 3 年）的历史最低/最高/平均三档区间，并与同行长交易史企业对比。

### PEG 与格雷厄姆成长股公式

> 类型：公式
> 来源：私募股权投资和尽职调查手册（估值方法）
> 置信度：实务经验

[场景：盈利法的辅助指标，不可单独使用]

- **PEG** = 预测市盈率 / （预测净利润增长率×100）。=1 估值合理，>1 高估，<1 低估。⚠️ 分子分母都含未来盈利预测，出错概率大，实战仅作市盈率辅助。
- **格雷厄姆成长股公式**：价值 = 年收益 × (8.5 + 预期年增长率×2)。年收益用 EPS TTM，增长率用未来3年。⚠️ 须与其他指标结合，不可单独使用。
- **EBITDA 倍数法**：EV/EBITDA（EV = 市值 + 付息负债 − 现金）。并购常用、可快速看偿债能力，但只适合同行业比较——卖企业时可用，**买企业时勿全信**（折旧摊销是实实在在的成本）。

## 三、DCF 模型

### 核心步骤

**Step 1：历史分析（3-5年）**
- 收入增长趋势（CAGR、驱动因素）
- 利润率演变（毛利率、EBIT 利润率、FCF 利润率）
- 资本密集度（D&A 和 CapEx 占收入比）
- 营运资本效率（NWC 变动占收入增量比）

**Step 2：收入预测**
- 基于历史增长率和行业趋势设定假设
- 三情景框架：悲观/基准/乐观
- 增长率应逐步向终值增长率收敛

**Step 3：自由现金流构建**
```
EBIT
(-) 税收（EBIT × 税率）
= NOPAT（税后净营业利润）
(+) D&A（非现金费用）
(-) CapEx（资本支出）
(-) △NWC（营运资本变动）
= 无杠杆自由现金流（UFCF）
```

**Step 4：WACC 计算**
```
股权成本（CAPM）= 无风险利率 + β × 股权风险溢价
税后债务成本 = 税前债务成本 × (1 - 税率)
WACC = 股权成本 × 股权权重 + 税后债务成本 × 债务权重
```

典型 WACC 区间：
- 大型稳定企业：7-9%
- 成长型企业：9-12%
- 高成长/高风险：12-15%

**Step 5：终值计算**

永续增长法（首选）：
```
终值 = 最后一年 FCF × (1 + 终值增长率) / (WACC - 终值增长率)
约束：终值增长率 < WACC，通常不超过长期 GDP 增长率（2-3%）
```

退出倍数法（备选）：
```
终值 = 最后一年 EBITDA × 退出倍数
```

终值健康性检查：终值占企业价值比例通常在 50-70%，超过 75% 说明过度依赖终值假设。

**Step 6：企业价值 → 股权价值桥接**
```
PV（预测期 FCF 之和）
+ PV（终值）
= 企业价值（EV）
(-) 净债务（= 总债务 - 现金）
= 股权价值
÷ 稀释后股数
= 每股隐含价值
```

**Step 7：敏感性分析**

标准三张敏感性表：
1. WACC vs 终值增长率 → 企业价值/每股价值
2. 收入增长率 vs EBIT 利润率 → 企业价值
3. Beta vs 无风险利率 → WACC 变化

敏感性表要求：
- 使用奇数维度（5×5 或 7×7），确保有真正的中心格
- 中心格 = 基准情景，其输出必须等于模型实际结果
- 中心格高亮显示

## 四、LBO 模型

### 核心结构

**来源与用途（Sources & Uses）**
```
用途：
  收购价格（EV）
  + 交易费用（通常 2-4% of EV）
  = 总用途

来源：
  高级债务（Senior Debt）
  + 次级债务/夹层（Sub Debt / Mezz）
  + 股权（Equity Check）
  = 总来源

来源 = 用途（必须平衡）
```

**债务结构典型参数**

| 层级 | 利率 | 摊销 | 优先级 |
|------|------|------|--------|
| 高级有担保（TLB） | SOFR + 300-500bps | 1% 年摊销 | 最高 |
| 高级无担保 | 固定 6-9% | 到期一次性偿还 | 次之 |
| 夹层/PIK | 固定 10-15% | PIK 或到期偿还 | 最低 |

**债务偿还瀑布**
1. 强制摊销（按合同）
2. 现金扫荡（Cash Sweep）：超额现金流按优先级偿还
3. 余额到期偿还

**关键约束**：债务余额不能为负，使用 MAX(0, ...) 函数。

### 回报分析

**核心指标**
- **MOIC**（投资倍数）= 退出股权价值 / 初始股权投入
- **IRR**（内部收益率）= 使 NPV = 0 的折现率

**回报归因分解**
```
总回报 = EBITDA 增长贡献 + 估值倍数扩张/收缩 + 债务偿还贡献 + 费用拖累
```

**敏感性分析标准格式**

进入倍数 vs 退出倍数（同时展示 IRR 和 MOIC）：

| | 退出 6x | 退出 7x | 退出 8x | 退出 9x | 退出 10x |
|---|---------|---------|---------|---------|----------|
| 进入 7x | | | | | |
| 进入 8x | | | | | |
| 进入 9x | | | | | |
| 进入 10x | | | | | |

其他常用敏感性：EBITDA 增长率 vs 退出倍数、杠杆倍数 vs 退出倍数、持有期 vs 退出倍数。

**三情景分析**

| | 乐观 | 基准 | 悲观 |
|---|------|------|------|
| 收入 CAGR | | | |
| 退出 EBITDA 利润率 | | | |
| 退出倍数 | | | |
| MOIC | | | |
| IRR | | | |

**注意事项**
- 现金流符号：投入为负，退出为正
- 管理层跟投和联合投资会改变股权投入金额
- 分红重组（Dividend Recap）会显著影响 IRR
- 交易费用（通常 2-4% of EV）会降低第一天的股权价值

## 五、投委会备忘录（IC Memo）

### 标准结构

**I. 执行摘要**（1页）
- 公司描述、交易逻辑、关键条款
- 推荐意见和核心回报指标
- 前三大风险及缓解措施

**II. 公司概况**（1-2页）
- 业务描述、产品/服务
- 客户群和市场推广
- 竞争定位
- 管理团队

**III. 行业与市场**（1页）
- 市场规模和增长
- 竞争格局
- 长期趋势/顺风
- 监管环境

**IV. 财务分析**（2-3页）
- 历史业绩（收入、EBITDA、利润率、现金流）
- 收益质量调整
- 营运资本分析
- CapEx 需求

**V. 投资逻辑**（1页）
- 为什么这是好投资（3-5个支柱）
- 价值创造杠杆（有机增长、利润率提升、并购、倍数扩张）
- 前100天优先事项

**VI. 交易条款与结构**（1页）
- 企业价值和隐含倍数
- 来源与用途
- 资本结构/杠杆
- 关键法律条款

**VII. 回报分析**（1页）
- 基准、乐观、悲观情景
- 各情景 IRR 和 MOIC
- 驱动回报的关键假设
- 敏感性分析

**VIII. 风险因素**（1页）
- 按严重程度和可能性排序的关键风险
- 每项风险的缓解措施
- 交易终止风险（如有）

**IX. 推荐意见**
- 明确推荐：推进 / 放弃 / 有条件推进
- 关键条件或后续步骤

### 编写原则

- IC 备忘录应客观平衡——同时呈现多空两面
- 不要淡化风险，投委会成员会自己发现；可信度比乐观更重要
- 财务表格必须勾稽——EBITDA 桥接、来源用途平衡、回报数学一致
- 缺少关键信息时，追问而不是假设

## 六、可比公司分析（Comps）

### 关键倍数

| 倍数 | 计算 | 适用场景 |
|------|------|----------|
| EV/EBITDA | 企业价值 / EBITDA | 最常用，跨行业可比 |
| EV/EBIT | 企业价值 / EBIT | 资本密集度差异大时 |
| EV/Revenue | 企业价值 / 收入 | 高增长但未盈利企业 |
| P/E | 市值 / 净利润 | 盈利稳定的成熟企业 |
| P/B | 市值 / 净资产 | 金融、重资产行业 |
| EV/FCF | 企业价值 / 自由现金流 | 现金流质量评估 |

### 可比公司筛选标准

- 行业/子行业相似
- 商业模式相近（B2B vs B2C、产品 vs 服务）
- 规模相近（收入、市值）
- 增长阶段相近
- 地理市场相近

### 数据标准化

- 统一使用 LTM（过去12个月）或 NTM（未来12个月）
- 剔除非经常性项目
- 调整净债务（包括租赁负债、养老金等类债务项目）
- 使用稀释后股数

## 七、IPO 财务准备

### 7.0 中国 A 股全面注册制（2023 年起）

> **政策节点**：2023 年 2 月 17 日证监会发布全面实行股票发行注册制配套规则，2023 年 4 月 10 日全面注册制下首批主板新股上市。**A 股全市场实行注册制**。

**注册制 vs 核准制核心差异**：

| 维度 | 核准制（已退出） | 注册制（现行） |
|------|----------------|---------------|
| 发行决定权 | 证监会实质审核+核准 | 交易所审核+证监会注册（形式审核为主） |
| 审核重点 | 持续盈利能力、合规性 | 信息披露真实、准确、完整 |
| 定价机制 | 23 倍市盈率隐性约束 | 市场化询价 |
| 上市门槛 | 单一财务指标 | 多套上市标准（含未盈利通道） |
| 退市力度 | 弱 | 强（财务类、规范类、交易类、重大违法类） |

**A 股各板块定位与上市标准**：

| 板块 | 定位 | 主要上市标准（以财务指标为例） | 未盈利上市通道 |
|------|------|------------------------------|--------------|
| 主板 | 大型蓝筹、行业龙头、稳定盈利 | 三套标准，最低：近3年净利润累计≥1.5亿元，最近1年≥6,000万元 | 无 |
| 科创板 | "硬科技"——新一代信息技术、高端装备、新材料、新能源、节能环保、生物医药 | 五套标准，覆盖盈利型、未盈利型、红筹型 | **第五套**：预计市值≥40亿+核心产品阶段性成果 |
| 创业板 | "三创四新"——成长型创新创业企业 | 三套标准 | 不再保留未盈利通道（注册制改革后调整） |
| 北交所 | "更早、更小、更新"——专精特新中小企业 | 四套标准，规模门槛最低 | 第四套：预计市值≥15亿+研发占比 |

**港股上市路径**：

| 章节 | 适用 | 关键要求 |
|------|------|---------|
| 主板 | 一般企业 | 三套标准（盈利/市值+收入/市值+收入+现金流） |
| **18A** | 未盈利生物科技 | 预计市值≥15亿港元+核心产品通过临床I期 |
| **18C** | 特专科技公司 | 已商业化≥60亿港元/未商业化≥100亿港元，含五大领域（新一代信息技术/先进硬件软件/先进材料/新能源及节能环保/新食品及农业技术） |
| 19C | 大中华区二次上市 | 已在合资格交易所上市≥5年（创新产业公司≥2年）+市值≥30亿港元 |

**审核重点变化（注册制下）**：

- 信息披露：从"是否披露"转向"是否说清楚"——重大事项的逻辑链条、关键假设、敏感性
- 持续经营能力：仍是关注重点，但不再作为否决性条件（科创板第五套等明确允许未盈利）
- 财务真实性：穿透核查供应商/客户、关联交易、收入确认时点等
- 同业竞争：注册制下从"必须解决"转为"充分披露+解决路径"
- 历史合规：股权清晰、税务合规、社保合规、环保合规

**关键监管文件**：

- 证监会发行注册管理办法（2023 年 2 月发布，沪深主板/科创板/创业板分别立法）
- 沪深北各交易所股票上市规则（2024 年陆续修订）
- 监管规则适用指引——发行类（编号 1-7 号，持续更新）

> 详细规则参见 https://docs.maoyanqing.com/securities/garr/

### 核心要求

| 要求 | 说明 | 常见问题 |
|------|------|----------|
| 会计基础规范 | 严格执行企业会计准则 | 收入确认不规范、关联交易未充分披露 |
| 内部控制健全 | 建立完善的内控体系 | 审批流程缺失、职责分离不到位 |
| 税务合规 | 历史税务问题清理 | 补税、优惠政策不合规 |
| 独立性 | 资产、人员、业务、财务、机构独立 | 与控股股东存在大额往来 |
| 持续盈利能力 | 证明业务可持续 | 依赖单一客户/产品 |

### 审核关注重点

- 收入真实性和可持续性
- 毛利率合理性（与行业对比）
- 关联交易的必要性和公允性
- 税收优惠的合规性和持续性
- 现金流与利润的匹配度
- 重大客户和供应商依赖
- 同业竞争和关联交易解决方案

| 市场 | 报告期 |
|------|--------|
| A 股主板/创业板 | 最近 3 年 1 期 |
| 科创板 | 最近 3 年 1 期（亏损企业可申报） |
| 北交所 | 最近 2 年 1 期 |
| 港股 | 最近 3 个会计年度（或 2 年，视情况） |

### 7.5 IPO 问询预测方法论（发散诊断 + 回复评估）

> 类型：流程
> 来源：实务保荐/审计 IPO 问询经验提炼（古立特化重写）
> 置信度：实务经验
> 关联：regulatory_risk_alerts.md - 监管风险提示/IPO审计红线；penalty_cases.md - IPO问询/处罚

[场景：拟上市企业预判可能被交易所问询的问题、准备回复材料、评估回复质量。]

#### 六条监管主线

所有 IPO 问询归根结底围绕六条主线，预测时用来补漏：

| 主线 | 监管核心关注 |
|------|------------|
| 业务是否真实 | 收入确认、客户真实性、终端销售、合同履约、安装验收 |
| 业绩是否可持续 | 市场空间、行业地位、客户稳定性、在手订单、期后业绩 |
| 利润是否合理 | 成本归集、采购价格公允性、毛利率波动、供应商稳定性 |
| 主体是否清晰独立 | 关联交易、独立性、控制权、历史沿革、同业竞争 |
| 治理是否规范 | 内控、资金流水、财务规范、合规经营、诉讼处罚 |
| 中介是否查到位 | 核查范围、样本选择、函证/访谈/走访、穿行测试、结论依据 |

#### 发散诊断透镜表（信号 → 监管假设 → 问法方向 → 证据需求）

> 类型：判断表
> 来源：IPO审核问询实务

先找项目特异性异常，再归入监管主题。不因为某主题高频就机械提问——没有事实支撑的问题只能做补漏。

| 异常信号 | 监管假设 | 发散问法方向 | 证据需求 |
|---------|---------|------------|---------|
| 收入在期末/Q4集中确认 | 跨期/突击验收/客户配合 | 履约节点、验收实质、期后回款、截止测试 | 合同、发货、安装、验收、发票、回款流水 |
| 收入增长但现金流转弱 | 收入质量/回款可持续性 | 客户付款能力、信用政策变化、逾期、坏账 | 应收账龄、期后回款、客户预算/付款审批、函证 |
| 毛利率显著高于同行或波动大 | 成本归集/定价公允性 | 单价、单位成本、项目毛利、可比公司差异 | 成本明细、采购价格、工时、可比产品、项目台账 |
| 客户集中/单一客户占比高 | 重大依赖/关联关系/收入真实性 | 合作背景、获客方式、终端需求、定价和回款 | 招投标、合同、访谈、工商穿透、终端使用证明 |
| 供应商集中/核心部件外采 | 供应链稳定性/采购公允性/技术独立性 | 是否依赖供应商、替代性、自制进度、采购价格 | 采购合同、比价、供应商访谈、技术分工 |
| 存货/发出商品/合同资产金额大 | 资产真实性/跌价/确认前置不足 | 对应订单、库龄、期后结转、跌价测试 | 订单、盘点、期后发货/验收、可变现净值测算 |
| 募投规模显著大于现有体量 | 产能消化/投资必要性 | 市场容量、在手订单、达产节奏、设备合理性 | 募投可研、产能测算、订单储备、同行扩产 |
| 报告期内股权频繁变动/突击入股 | 利益输送/股份支付/锁定合规 | 入股背景、价格公允、资金来源、穿透核查 | 增资/转让协议、流水、估值报告、访谈 |
| 历史代持/对赌/特殊权利 | 权属争议/控制权稳定 | 形成和解除真实性、恢复条款影响、纠纷风险 | 解除协议、流水、承诺函、律师核查 |
| 研发投入/科创属性叙事过强 | 技术先进性/研发真实性/商业化能力 | 核心技术来源、专利贡献、研发费用归集、产品收入 | 研发项目台账、专利、人员工时、产品收入映射 |

#### 回复质量评估五步闭环

好的问询回复**应当**按以下顺序展开：

```
正面回答问题 → 量化解释原因 → 外部证据佐证 → 中介核查程序 → 补充披露修改
```

缺任何一环都容易被追问。

#### 二轮追问触发条件

当首轮回复出现以下情形，二轮/三轮追问概率极高：

- 只解释不量化（"主要因为市场拓展"但没有数据）
- 自证多第三方证据少（"发行人说明"没有客户/供应商/行业确认）
- 核查程序写得泛（"我们执行了核查"但没说查了什么、查到什么）
- 前后口径不一致（招股书与回复、首轮与二轮、发行人与中介结论矛盾）

#### 预测问题三分法与排序

| 问题类型 | 含义 | 优先级 |
|---------|------|--------|
| 项目特异性 | 由该项目事实直接推导——预测重点 | 最高 |
| 板块高频 | 科创板/创业板/主板/北交所历史问询经验 | 次高 |
| 补漏型 | 六条主线扫描发现、但当前资料证据不足 | 标记"待补充资料" |

每个预测问题标注三个维度：**概率**（高/中/低）× **杀伤力**（影响上市条件/收入利润真实性为高）× **准备难度**（需第三方证据/穿透核查为高）。

### 7.6 IPO 被否/终止典型事由（红旗的"致命子集"）

> 类型：判断表
> 来源：2021 年 IPO 终止/被否案例蒸馏（29 例）+ 监管问询经验
> 置信度：实务经验
> 关联：7.5 IPO问询预测（问询 ≠ 被否）；financial_fraud.md（穿透核查）；penalty_cases.md（处罚先例）

[场景：判断拟上市企业是否存在"可能致命"的发行障碍——区别于 7.5 的"会被问到什么"，本节是"什么会把企业从问询推到被否/劝退"]

**先分清两件事**：被问询 ≠ 被否。绝大多数撤否发生在上会前主动撤回——实际通过率远低于过会率（过会率可达 90%，但叠加大量终止后的实际通过率往往只有六成左右），撤回是主流。真正致命的，是下面这些"问了答不圆"的硬伤。

**致命红旗分类（按出现频率排序）**：

| # | 致命事由 | 典型表现 | 为什么致命 |
|---|---------|---------|-----------|
| 1 | 收入真实性存疑 | 经销/加盟压货、以旧换新调节收入、期末突击确认、第三方回款占比高、总额法↔净额法切换调收入 | 动摇"业务是否真实"根基，近乎一票否决 |
| 2 | 关联交易公允性与独立性 | 关联采购/销售占同类比畸高（70%~99%）、关联业务毛利率高于非关联、向实控人拆出大额资金、关联方代收货款 | 独立性存疑→不符合发行条件 |
| 3 | 毛利率显著异于同行 | 毛利率远高于可比公司且无合理解释、变动趋势与同行背离 | 指向收入虚增或成本不实 |
| 4 | 重大客户/供应商依赖 | 单一客户或供应商占比 >50%、客户与供应商重叠 | 持续经营 + 利益输送双重质疑 |
| 5 | 持续经营能力存疑 | 业绩下滑、阶段性/政策性收入（如限硫令抢装）、产能利用率逐年下降仍募资扩产、依赖政府补贴 | 不符合"持续经营/盈利"要求 |
| 6 | 板块定位不匹配 | 科创属性不足（核心技术外购、研发占比不达标）、创业板"三创四新"勉强、名称含"物联网/科技"但业务名不副实 | 定位是注册制审核硬门槛 |
| 7 | 股权与历史沿革 | 股权代持真实性、突击入股价格公允性、国资转让程序/国资流失、实控人认定与变动、红筹拆除 | 权属不清→发行障碍 |
| 8 | 现金流持续低于净利润 | 经营性现金流净额持续低于净利润甚至为负 | 利润含金量存疑，常与收入不实并发 |
| 9 | 合规硬伤 | 安全生产/环保处罚、商业贿赂嫌疑、虚假宣传（如擅印"教育部门推荐"）、重大未决诉讼、超产无证 | 重大违法→实质性障碍 |
| 10 | 研发费用归集不实 | 非研发支出计入研发、靠归集"凑"科创属性占比 | 既是定位问题也是内控问题 |
| 11 | 资金流水异常 | 实控人与亲属/员工大额往来、账外资金、前财务负责人离职且未提供完整流水 | 指向体外资金循环/虚增 |

**用法**：尽调 / IPO 预审时先扫这 11 类硬伤——命中即标"可能致命，需重点核查或劝退"，区别于 7.5 的常规问询预测。多条并发（如收入真实性 + 毛利率异常 + 现金流背离）时财务造假嫌疑显著上升，交叉 financial_fraud.md 做资金穿透。

⚠️ 这是"被否/劝退"的高频事由，不是"必否"清单——能给出有事实、有第三方证据、有中介核查程序的合理解释，仍可过会。致命的是"答不圆"，不是"有问题"。

## 八、早期/硬科技公司估值

### 传统 DCF 的局限

早期公司现金流为负或高度不确定，传统 DCF 不适用。需要叠加以下方法：

### 情景加权 DCF

不做单一基准预测，构建三个情景并赋予概率权重：

| 情景 | 概率 | 核心假设 |
|------|------|----------|
| 技术突破+市场顺利 | 20-30% | 商业化周期短，市场快速渗透 |
| 基准（渐进发展） | 40-50% | 商业化周期中等，竞争正常 |
| 技术停滞/市场受阻 | 20-30% | 商业化延迟或失败 |

加权估值 = Σ（各情景 DCF × 概率权重）

### 实物期权估值

将技术商业化视为看涨期权：
- 标的资产价值 = 技术成功后的市场规模现值
- 执行价格 = 从当前到商业化所需的总投入
- 期权期限 = 预计商业化时间窗口
- 波动率 = 技术路线的不确定性

实物期权价值 + DCF 现有业务价值 = 完整企业价值

### WACC 的非财务风险溢价

对于供应链高度依赖海外、技术路线竞争激烈的硬科技公司，在标准 CAPM 基础上叠加：

```
Ke = Rf + β × ERP + 供应链风险溢价 + 技术迭代风险溢价 + 合规风险溢价
```

| 风险类型 | 量化思路 | 参考区间 |
|----------|----------|----------|
| 供应链断裂 | 断供概率 × 断供影响（占企业价值比） | 1-5% |
| 技术路线被颠覆 | 技术替代概率 × 资产贬值幅度 | 1-4% |
| 数据主权/合规 | 合规改造成本现值 / 企业价值 | 0.5-3% |

### 关键提示

- 上述溢价的具体数字需基于对特定公司的深度尽调
- 框架的价值在于让决策委员会在同一语言体系下讨论风险，把"地缘政治风险很大"变成可被质疑和修正的具体假设
- 建议由独立的技术专家和行业分析师复核地缘政治和技术路线的评分

## 九、资产评估准则与执业要点

> 本章是"持证资产评估师执业准则"视角，与前面"投行估值"（DCF/可比公司/LBO）是同一件事的两套话语：投行估值服务于投资决策，法定资产评估服务于合规与权属定价。两者方法同源（收益法≈DCF、市场法≈可比公司、成本法≈资产基础法），但评估有法定程序、法定情形、法定报告要求。
> 原文不搬，按需查：MaoDocs 评估板块 https://docs.maoyanqing.com/appraisal/aas/

### 资产评估在财税体系中的定位

> 类型：规则
> 来源：资产评估法（2016）+ 中评协准则体系
> 置信度：准则原文

[场景：搞清评估师和注会、税务师的分工，避免把评估业务当成审计或税务业务来处理]

资产评估是与注册会计师、税务师**平行的独立持证专业**，不是会计或审计的子集：

| 专业 | 持证 | 核心产出 | 解决的问题 |
|---|---|---|---|
| 注册会计师 | CPA | 审计报告 | 财务报表是否公允反映 |
| 资产评估师 | CPV（资产评估师） | 资产评估报告 | 资产/权益值多少钱 |
| 税务师 | CTA | 涉税鉴证/筹划 | 税负是否准确合规 |

⚠️ 评估报告由**资产评估机构**出具、**资产评估师签字**，注会的审计报告不能替代评估报告，反之亦然。涉及法定评估的场景（见下），必须用持证评估机构。

准则体系五层：资产评估法（人大）→ 基本准则（财政部财资〔2017〕43号）→ 职业道德准则 → 执业准则（程序性+实体性）→ 指南与指导意见。

### 法定必须评估的情形（合规红线）

> 类型：判断表
> 来源：资产评估法 + 国资监管规定 + 公司法
> 置信度：准则原文

[场景：判断一笔交易是不是"法律强制要求先评估"，漏评估会导致交易无效或被追责]

| 判断顺序 | 情形 | 是否强制评估 |
|---|---|---|
| 1（国资优先） | 国有资产转让、出售、置换、对外投资、改制 | **强制**，未评估或低于评估值成交无效，可追责 |
| 2 | 以非货币资产出资设立公司 | **强制**（公司法要求作价评估） |
| 3 | 上市公司重大资产重组、发行股份购买资产 | **强制**（证监会要求评估或估值） |
| 4 | 企业破产清算、司法拍卖处置 | **强制**（法院委托评估） |
| 5 | 抵押贷款、担保 | 视债权人要求，通常需要 |
| 6 | 一般民商事主体间的非国资交易 | 不强制，可自愿委托 |

⚠️ 核心红线：**只要一方涉及国有资产，几乎都强制评估**，且成交价不得低于评估结果（确需折价的要履行审批）。这是实务中最容易踩的合规点——国资交易没评估直接定价，交易可被认定无效。

### 三种基本评估方法的选择判断

> 类型：判断表
> 来源：资产评估执业准则——资产评估方法（中评协〔2019〕35号）
> 置信度：准则原文

[场景：拿到一个评估对象，先判断该用哪种方法、为什么不能用某种方法]

三种法定基本方法，对应投行估值的三类：

| 方法 | 原理 | 对应投行估值 | 适用判断 |
|---|---|---|---|
| 收益法 | 预期收益折现 | ≈ DCF | 收益可预测、可量化（经营性资产、企业价值、无形资产） |
| 市场法 | 参照可比交易/可比公司 | ≈ 可比公司法/可比交易法 | 有活跃市场、有可比标的（不动产、设备、上市公司股权） |
| 成本法 | 重置成本 − 各项贬值 | ≈ 资产基础法 | 单项资产、无收益或收益难预测、专用资产（机器设备、在建工程） |

选择规则（执业准则要求）：
1. **应当根据评估对象、价值类型、资料收集情况，分析三种方法的适用性。**
2. **满足条件的应优先考虑两种以上方法**，相互验证，差异大要分析原因。
3. 不能因为"算起来方便"就只用一种。

⚠️ 常见误用：
- 对**持续经营的企业整体**只用成本法（资产加总）→ 漏掉商誉和盈利能力，低估。企业价值应优先收益法/市场法。
- 对**无活跃市场的专用设备**用市场法 → 找不到真正可比标的，结论不可靠，应用成本法。
- 对**早期无收益的技术**用收益法做单一点估计 → 收益假设极不确定，应做情景/期权处理（详见本文件第八章 早期/硬科技估值）。

### 评估与会计的接口（以财务报告为目的的评估）

> 类型：规则
> 来源：以财务报告为目的的评估指南（中评协〔2017〕45号）+ CAS 39 公允价值计量
> 关联：accounting_advanced.md - 公允价值计量；accounting_core.md - 资产减值
> 置信度：准则原文

[场景：会计上要确认公允价值时（企业合并、资产减值、投资性房地产公允模式），评估结论怎么用]

会计准则要求公允价值的地方，常需要评估支撑：

| 会计场景 | 评估对象 | 准则接口 |
|---|---|---|
| 非同一控制企业合并 | 被购买方可辨认资产负债公允价值 | CAS 20，差额计入商誉 |
| 商誉减值测试 | 资产组可收回金额 | CAS 8，需估计未来现金流现值 |
| 投资性房地产公允价值模式 | 房地产公允价值 | CAS 3 |
| 非货币性资产交换 | 换入换出资产公允价值 | CAS 7 |
| 债务重组 | 抵债资产公允价值 | CAS 12 |

⚠️ "以财务报告为目的的评估"有特殊要求：价值类型通常是**公允价值**（不是市场价值），且要与会计准则的计量要求一致。评估师出具的是为会计服务的专项评估，不能直接套用一般市场价值评估的结论。

### 评估与税务的接口

> 类型：规则
> 来源：企业所得税法及实施条例 + 财税〔2009〕59号等
> 关联：tax_core.md - 股权转让/资产划转；tax_advanced.md - 转让定价
> 置信度：准则原文

[场景：资产/股权交易计税时，评估值如何影响计税基础和应纳税额]

| 税务场景 | 评估的作用 |
|---|---|
| 股权转让所得 | 转让价明显偏低且无正当理由的，税务机关按评估/核定价调整（防止平价/低价转让避税） |
| 资产划转、重组 | 特殊性税务处理需以评估确认公允价值，确定计税基础 |
| 非货币资产投资 | 按评估公允价值确认转让所得，可分 5 年递延（财税〔2014〕116号） |
| 关联交易转让定价 | 评估/独立交易价格作为是否公允的判断基准 |
| 房地产、土地增值税清算 | 评估值作为扣除项目/收入确认的依据之一 |

⚠️ 税务上"计税基础"以税法规定为准，评估值是重要参考但不必然等于计税基础——要看具体税种规则。涉及重组、划转的特殊性税务处理，评估是前置条件。

### 新兴评估对象（数据资产、关键核心技术）

> 类型：规则
> 来源：数据资产评估指导意见（中评协〔2023〕17号）、关键核心技术资产评估指导意见（中评协〔2025〕9号）
> 置信度：准则原文

[场景：企业有数据资产入表、或硬科技技术作价时的评估，这是 2023 年后的新领域]

- **数据资产评估（2023〕17号）**：配合"数据资产入表"（财政部数据资产暂行规定，2024.1.1 起），数据资源可作为无形资产或存货入表。评估方法仍是收益法/市场法/成本法三选，但数据资产的特殊性在于：可复制、价值依场景、权属复杂（持有权/加工使用权/经营权分置）。评估前先确认数据权属和合规来源。

> 关联：accounting_core.md - 无形资产确认（数据资产入表的会计处理）；esg.md / financial_bp.md 不涉及，评估归本模块。

## 十、私募基金与管理人出资核查（中基协）

> 本章是"基金管理人登记 + 基金备案"环节，基金业协会对**出资**的穿透核查要点。出资瑕疵且无合理解释将构成登记/备案的实质性障碍。
> 原文不搬，规则会随《私募投资基金登记备案办法》修订，按需查中基协官网。

### 出资人类型穿透核查

> 类型：判断表
> 来源：中基协自律规则 + 私募登记备案实务
> 置信度：实务经验

[场景：判断基金管理人的股东、或基金的投资人是否满足协会出资人门槛]

**基金管理人出资人（穿透核查至实际控制人）：**

| 出资人 | 红线 |
|--------|------|
| 主要出资人（出资比例≥25%）、实际控制人 | 不得从事与私募基金管理相**冲突业务**；不得为资管产品；自然人最近 5 年不得从事冲突业务 |
| 其他出资人（出资比例<25%） | 涉及冲突业务的合计出资比例不得高于 25% |
| 资管产品出资 | 对管理人直接或间接出资比例合计不得高于 25% |

- **冲突业务**定义：民间借贷、民间融资、融资租赁、配资、小额理财/借贷、P2P/P2B、众筹、保理、担保、房地产开发、交易平台等与基金管理相冲突或无关的业务。
- **资管产品**定义：银行非保本理财、证券期货经营机构资管计划、信托计划、保险资管产品、在协会备案的基金等受国务院金融监管机构监管的产品。

**基金投资人（合格投资者标准）：**

投资于单只基金金额**不低于 100 万元**，且满足下列之一：

- 单位：净资产不低于 1,000 万元；
- 个人：金融资产不低于 300 万元，**或**最近三年个人年均收入不低于 50 万元。

⚠️ 以合伙企业等非法人形式投资基金的（依法备案的投资计划除外），**应穿透核查最终投资人**是否合格并合并计算人数。同一投资人在一级/二级/三级各层级都存在的，**每一层出资都应不低于 100 万元**。

### 出资来源核查（严禁代持）

> 类型：规则
> 来源：中基协自律规则
> 置信度：实务经验

- 管理人所有出资人**应当**以**合法自有货币财产**出资，资金来源真实合法、不受制于第三方，股权结构清晰。
- **不得**：以委托资金、债务资金等非自有资金出资；通过委托他人或接受他人委托方式持有股权/份额（代持）；存在循环出资、交叉持股、结构复杂隐瞒关联关系。
- 基金投资人**应当**以真实身份和自有货币资金为自己购买，不存在代缴、代付、代持，不得非法汇集他人资金投资。

⚠️ 参考案例：实际控制人为年迈夫妻、无法提供与注册资本相对应的出资能力材料、且不担任高管——协会认定存在**股权代持嫌疑**，中止登记。出资能力与出资人身份不匹配是代持的首要红旗。

### 初始实缴出资门槛

> 类型：判断表
> 来源：中基协登记备案要求
> 置信度：实务经验

| 主体 | 初始实缴门槛 |
|------|------------|
| 基金管理人（现行反馈口径） | 取孰高：①250 万元或注册资本 25% 的实缴；②覆盖至少 6 个月运营开支 |
| 私募证券基金 | 不低于 1,000 万元 |
| 私募股权基金 | 不低于 2,000 万元（创业投资基金不低于 1,000 万元） |
| 契约型基金备案 | 已认购投资人全部认购款已进入基金托管账户 |
| 公司/合伙型基金备案 | 全部股东/合伙人完成不低于 100 万元初始实缴且已进入基金财产账户 |

例外：管理人及其员工、社保基金、政府引导基金、养老基金、社会公益基金的首轮实缴可按章程/协议约定，可低于 100 万。员工跟投平台实缴需≥100 万元。

**实缴证明材料**：验资报告（盖会所公章）或银行回单（优选，盖银行业务章），记载的出资人/认缴/实缴须与工商登记一致；基金有托管的传资金到账通知书，无托管的传验资证明或银行回单（收款账户名须与基金名一致）。

### 认缴出资差额的出资能力

> 类型：规则
> 来源：中基协登记备案实务
> 置信度：实务经验

对未实缴金额（认缴差额），需提供**足以覆盖**的出资能力证明：

- **自然人**：固定资产（非首套房产权证明或评估材料）、近半年银行流水/金融资产证明、薪资/完税/理财/配偶收入证明等。以夫妻共有财产证明的，须证明夫妻关系并经另一方书面同意。
- **机构**：验资报告、最近年度审计报告；持股平台若自身报表不足以覆盖，可**穿透核查上层股东**出资能力。
- 同一投资人在多只基金出资的，出资能力证明金额**应不低于其对所有已认购基金认缴差额之和**。

⚠️ 管理团队成员的个人资产不能用来证明管理人（机构）的出资能力——主体不能混同。

## 十一、投资协议核心条款（Term Sheet）与控制权安排

> 本章是交易谈判落地环节，前面尽调/估值是"判断要不要投、投多少钱"，本章是"用什么条款保护投资"。参股 vs 控股的条款逻辑完全相反，先定位再套条款。

### 投资人保护性条款清单

> 类型：清单
> 来源：私募股权投资实务
> 置信度：实务经验

[场景：起草/审阅投资条款清单，按参股或控股定位逐项配置]

**判断顺序：先确定投资方是参股还是控股。**
- **参股**：靠合同条款保护——要的保护性条款越多越好。
- **控股**：靠董事会席位 + 公司治理实现控制——应取得董事会多数席位，把下列"重要决定"尽量转为董事会保留事项，多数保护性条款对控股方不适用。

| 条款 | 作用 | 配置要点 |
|------|------|---------|
| 一票否决权 / 保护性条款（Veto / Protective Provisions） | 对增减资、改主营、处置核心 IP/重大资产、对外借款担保投资、关联交易、重大诉讼和解、董事会规模和提名等重大事项一票否决 | 参股方多要；控股方改用董事会多数席位实现 |
| 优先认购权（Pre-emption on issuance） | 增资/发新股时按同等条件优先认购，维持比例 | 参股按持股比例；控股可要求超比例 |
| 优先购买权（Right of First Refusal） | 其他股东向第三方转股时同等条件优先购买 | 同上 |
| 反稀释条款（Anti-Dilution） | 公司不得以低于本轮估值融资/转老股，否则调整投资方估值（控股股东无偿转股或现金补偿降至新低价） | 防低价融资稀释 |
| 估值调整 / 对赌（VAM） | 业绩/上市目标未达成时调整投资价或提前退出 | 见下条法律要点 |
| 回购请求权（Redemption） | 期限内未达业绩或未上市/挂牌/被并购，要求控股股东回购，可约定溢价（弥补资金成本/基础收益） | 优先与股东约定，慎与公司约定 |
| 随售权 / 共同出售权（Tag-Along / Co-sale） | 控股股东转股时，投资方有权按同等条件搭售 | 退出保护 |
| 拖售权 / 强制出售权（Drag-Along） | 投资方出售超 50% 股权时，可要求其他股东同条件一起卖 | 参股方常被创始人要求，可拒或设高转让价底线 |
| 购买/出售期权（Call/Put Option） | 创始人违反竞业等义务时，投资方可选择卖出或收购其股权 | 违约救济 |
| 清算优先权（Liquidation Preference） | 清算时优先于其他股东收回投资款，可约定年化收益 | 下行保护 |
| 最惠国待遇（MFN） | 公司未来给其他投资人的权利自动适用于本方 | 参股方常要 |
| 创始人锁定 | 竞业禁止、全职服务、控制权不得丧失、股权转让限制 | 参股尤其需要锁住核心团队 |

### 对赌（VAM）的法律有效性

> 类型：规则
> 来源：最高人民法院对赌司法判例
> 置信度：准则原文

**判断规则：对赌协议的签约对手决定其效力。**

- 投资方与**标的公司股东**签的对赌（现金/股权补偿、回购）→ 属处分各自财产，**有效**。
- 投资方与**标的公司本身**签的对赌 → 涉及处分公司财产，可能损害其他股东、债权人利益或导致股权不稳定，**可能被认定无效**。
- 与公司约定回购还会触发减少注册资本，程序复杂，**不建议采用**。

⚠️ 硬规则：无论现金补偿、股权补偿还是回购，投资方都**应当**与**标的公司股东**签约并向其主张权利，不要只与公司签。另：股权补偿会改变股权结构，影响稳定性，在 IPO 审核中不易被监管认可。

### 股权比例与控制权判断

> 类型：判断表
> 来源：公司法表决权规则 + 股权架构实务
> 置信度：实务经验
> 关联：尽调中判断实际控制人、控制权稳定性

[场景：尽调或股权设计时，按第一大股东/老大的表决权比例判断控制力]

| 表决权比例 | 控制力 |
|-----------|--------|
| ≥2/3（约 66.7%） | 绝对控股，对股东会**所有**决议事项有决定权（含修改章程、合并分立等特别决议） |
| >50% | 相对控股，对**大部分**普通决议事项有决定权 |
| 34%—49% | 对少数重大事项（需 2/3 通过的特别决议）有一票否决权，但大部分事项无决定权，易失控 |
| 1%—33% | 既无决定权也无否决权 |

**股权架构红旗：**
- 经营团队持股 <50% → 不控股，易失控、创业动力不足；持股 >2/3 才是绝对控股。
- 第一、二大股东差距过小 → 易决策僵局、责任推诿。建议老大老二**至少拉开 10% 差距**，否则用投票权委托、一致行动人协议、持股平台放大老大投票权。
- 缺乏股权动态调整机制 → 早期拍脑袋分配 + 后期新人激励无解，估值越大调整难度越大，**应在早期就设计好调整机制**。

### 核心条件谈判与投资意向书

> 类型：流程
> 来源：私募股权投资和尽职调查手册（核心条件谈判）
> 置信度：实务经验

[场景：达成初步意向后、正式尽调前后的条款谈判]

- **谈判时机**：并购方式（转股/增资扩股）和并购价格这两点，**应在初步意向之后、正式尽调之前**先谈好大致可接受区间；谈判与尽调互动推进。
- **投资意向书（备忘录）效力条款**：排他协商、提供资料、保密、锁定（按约定价买部分股份以保证继续谈判）、费用分担、终止条款。若双方条件都满意可不签意向书，直接进《股权投资协议》。

**谈判要点：**

| 要点 | 做法 |
|------|------|
| 气场相符 | 谈判代表要与企业负责人气场相当 |
| 价值体现 | 想清楚除了钱还能带来什么（产业协同、品牌背书、市场、管理输出） |
| 抓核心问题 | 用尽调挖出的最核心、最致命的 1~2 个问题让对方"心灵一震"，才能赢得尊重和议价空间 |
| 沉得住气 | 用时间拉锯争议价空间（项目从接洽到落地常达 1 年） |
| 谈判场所 | 避免在对方地盘（办公室、对方招待处）谈 |

⚠️ 估值太高影响后续融资，太低让创始人觉得吃亏埋矛盾——把价格谈到势均力敌、对方快不想玩时再略上浮一点。查理·芒格：和品行不良的人很难做成好买卖，别指望合同或尽调能保护自己。

### 买卖协议（SPA / SHA）结构性条款

> 类型：清单
> 来源：私募股权投资和尽职调查手册（股权投资协议）
> 置信度：实务经验
> 关联：本章 投资人保护性条款清单（一票否决/反稀释/对赌等特殊条款）

[场景：起草/审阅正式投资协议 SPA（买卖）与 SHA（股东）的通用结构条款，区别于前述"保护性条款"]

| 条款 | 作用与要点 |
|------|-----------|
| 先决条件（Conditions Precedent） | 必备。交割生效条件（尽调满意、法律意见书满意、买方决议通过、卖方陈述保证仍有效、监管审批）。**应加最后终止日（Long Stop Date）**，避免交易无限期悬空 |
| 交割前义务（承诺条款） | 锁定签约到交割期间卖方/目标公司不得做不利变化：未经同意不得签非常规合同、处置达一定金额资产、大额出资、借款融资、增减资、分红、改章程、发股 |
| 陈述与保证（Reps & Warranties） | 极重要。对目标公司**现状**的保证（法律地位、合法合规、股权无权利负担、财务报表、无未披露负债）；卖方须作的远多于买方。与"交割前义务"（对**未来**的承诺）一起，锁定过去/现状/未来 |
| 违约赔偿（Indemnities） | 必备。创始人股东/转让方与公司承担**连带责任**；但公司对保证/赔偿请求的最高金钱责任应**不高于投资人实际投资额的 100%** |
| 知情权（Information Rights） | 会计年度开始前 30 天报下年度预算/商业计划；年度结束后 90 天报审计报告；季度结束后 45 天报季报；提前书面通知可行使**检查权**（查账簿凭证、约谈员工/审计师/法律顾问） |
| 对价 + 交割 | 对价在交割日或之前支付，可约调整机制（对赌）和分期；交割时一并移交印章、会计凭证、证照，并配合工商变更 |
| 法律适用与争议解决 | 必备。适用法律/管辖地尽量选本国或第三国（香港/新加坡）；争议解决由简到繁：调解 → 仲裁（一裁终局）→ 法院诉讼 |
| 费用与赋税 | 印花税通常各承担 50%，资本利得税由卖方承担 |

⚠️ 履约一般分三段付款：合同生效付一部分 → 卖方交割资产/股权后再付一部分 → 交割后约定期限付尾款，尾款付清协议才算履行完毕。

### 股权结构设计的退出/婚姻/伤亡/磨合机制

> 类型：规则
> 来源：私募股权投资和尽职调查手册（投前管理-股权分配）
> 置信度：实务经验
> 关联：本章 股权比例与控制权判断（已覆盖持股阈值与老大老二差距）

[场景：内部孵化或辅导初创项目时的股权架构设计，补充阈值之外的动态机制]

除持股比例阈值外，下列机制**应在早期就用书面协议定好**（事后一箩筐道理抵不过事前一纸协议）：

- **合伙人磨合**：正式合伙前应有**6 个月以上**具体事情的磨合（体制内外均可，先"恋爱"再"结婚"）；经不起半年磨合的，别奢望合伙 10 年。
- **退出（离职）机制**：进入要"结婚"机制，退出要"离婚"机制；大量股权战争源于缺退出机制。公司法/章程把创业者当投资人，退出机制**必须靠股东协议解决**。
- **婚姻（离婚）机制**：创始人配偶是最大隐名合伙人（婚内财产共有）；可事先约定合理价格回购离婚配偶股权、或分离投票权与分红权，避免公司动荡。⚠️"土豆条款"（配偶完全放弃股权）过于简单粗暴。
- **伤亡机制**：重要股东伤亡若无约定会引发巨大震荡；可约定公司回购、其他股东购买或"钱权分离"保障家属财产权。
- **认缴未实缴控制**：认缴制不等于认而不缴；未实缴过大会加大个人责任、可被除名，建议减资；不建议第三方垫资（经不起审计或构成抽逃出资）。人力驱动型早期公司若无最低注册资本要求，建议注册资本上限 100 万。
- **股东约束机制**：对重要团队/技术骨干施加竞业禁止、保密、知识产权归属、不挖墙角——验证"人力资本出资"是否到位。

### 股权九条生命线（控制权阈值全表）

> 类型：判断表
> 来源：企业融资Ⅱ（廖连中，股权设计）
> 置信度：实务经验
> 关联：本章 股权比例与控制权判断（四档表是决策简版，本表是含监管触发点的全表）

[场景：股权设计或并购持股测算时，按持股/表决权比例快速对应"能做什么、触发什么监管线"]

| 比例 | 性质/触发 | 含义 |
|:---:|------|------|
| 67%（2/3） | 绝对控制权 | 修改章程、合并分立解散、增减资等**特别决议**全部能过 |
| 51% | 相对控制权 | 选举董事、利润分配等**普通决议**能过；但特别决议仍可能被否 |
| 34% | 一票否决权 | 守住 1/3+，可否决所有特别决议（防御性，非进攻性） |
| 30% | 上市公司要约收购线 | 持股达 30% 再增持触发**要约收购义务**；并购举牌的关键卡口 |
| 20% | 重大同业竞争警示线 | 达 20% 推定有重大影响，关联方/同业竞争披露趋严 |
| 10% | 临时会议权 | 可请求召开临时股东会、提起解散公司之诉 |
| 5% | 重大股权变动警示线 | 上市公司持股变动达 5% 须**举牌披露**（权益变动报告） |
| 3% | 临时提案权 | 可在股东会前提出临时提案 |
| 1% | 代位诉讼权 | 可代位起诉董监高（须连续持股 180 天等条件） |

⚠️ 设计控制权时优先锁 **67% / 51% / 34%** 三条主线；做并购测算时优先看 **30%（要约线）/ 5%（举牌线）** 两条监管线——踩线不披露是处罚高发点。

### 股权激励工具选择（期权/限制性股权/虚拟股/业绩股票/TUP）

> 类型：判断表
> 来源：企业融资Ⅱ（廖连中，股权激励）+ 华为虚拟股/TUP 实践
> 置信度：实务经验

[场景：设计员工激励时，按"是否让渡表决权、是否要现金对价、企业阶段"选工具]

| 工具 | 给什么 | 是否稀释表决权 | 适用 |
|------|--------|:---:|------|
| 股票期权 | 未来按约定价买股的权利（赚差价） | 行权后稀释 | 高成长期、股价有上行空间；激励对象先无分红权 |
| 限制性股权 | 满足条件（业绩/上市/年限）才可兑现的股权 | 兑现后稀释 | 成熟型或资金投入要求不高的企业；约束力强、风险小 |
| 虚拟股权 | 只给**分红权+增值收益权**，无所有权/表决权 | **不稀释、不影响决策** | 创始人想保住控制权又要激励核心员工（华为模式）；无须证监审批、不受公开市场股价波动 |
| 业绩股票 | 达成业绩目标后授予的实股 | 稀释 | 有明确可量化业绩目标的成熟企业 |
| TUP（时间单位计划） | 数年内**逐步兑现的分红权+期末股本增值**，到期权益清零 | 不稀释 | 解决新员工/外籍员工"无钱购股、激励不足、离职率高"；类分期付款，留人 |

⚠️ 想保控制权就优先用**虚拟股/TUP**（只分钱不分权）；要绑定长期贡献用**限制性股权/业绩股票**（达标才给）。⚠️ 老员工虚拟股积累过多会造成新老收入失衡——应设**岗位饱和配股上限**（按职级封顶持股数）平衡。

### 股权结构失控反例（平衡股权/一家独大/内部人控制）

> 类型：规则
> 来源：企业融资Ⅱ（廖连中，股权结构反例：1号店/真功夫/雷士/山水水泥）
> 置信度：实务经验

⚠️ 三类畸形股权结构，设计时一票否决：

- **平衡股权（尤其 50:50）最差**：大股东比例过于接近、缺核心股东，决策易僵局、控制权与利益索取权失衡。真功夫两股东各 50%、引入 PE 后仍 47:47，意见一旦不合公司决策瘫痪、最终对簿公堂。**铁律：必须有持股最大的核心股东，贡献与股权比例匹配；老大老二拉开差距（见股权比例与控制权判断）。**
- **股权高度分散→内部人控制**：单一股东均 <10%，小股东"搭便车"不愿监督经理层，所有权与经营权脱节，管理层架空股东。
- **股权高度集中→一家独大**：风险全压在大股东身上，缺制衡。

⚠️ 失控的两个根因与对策：
- **按出资额简单分股**（创始人出钱又出力却成小股东）→ 闹分家、且吓退后续机构投资人。**应按"钱+力+资源"综合定价，而非只看出资比例。**
- **义气让股、忽视股权设计**（雷士吴长江为兄弟义气放弃 51%；1号店用 80% 股权换平安资金，实质让出控制权）→ 创始人沦为打工者被扫地出门。⚠️ 融资前先辨**财务投资人 vs 战略投资人**：战投瞄准的是控制权和产业链布局，AB股/一致行动/有限合伙等控制权工具**只在双方目的一致时才生效**——战投存心要控制权时，这些技术手段挡不住。

## 十二、股权投后评价

> 本章是"投资后回看"环节——对已持股公司的投资过程和效益做审慎性调查并出具评价报告，由会计师事务所或资产评估机构执行。它**不是审计、不是评估**，是独立的咨询性业务。

### 投后评价的定位（与审计的区别）

> 类型：判断表
> 来源：股权投后评价业务基本指引（试行）
> 置信度：准则原文

[场景：承接投后评价业务时，先和委托人说清它不等于审计，数据可靠性不及审计]

| 维度 | 股权投后评价 | 财务报表审计 |
|------|------------|------------|
| 目的 | 分析研判投资过程和投资效益，**不保证**历史财务信息真实公允 | 对财务报表公允性发表意见 |
| 范围 | 财务 + 非财务信息结合，覆盖投前决策、投中实施、投后管理、处置退出 | 限于历史财务信息 |
| 方法 | 按委托双方约定，一般**不执行**函证、盘点、抽凭、信息系统测试 | 执行完整审计程序 |
| 资料可靠性 | **不及审计**，主要依赖持股公司提供资料和管理层陈述 | 高度可靠性验证 |

⚠️ 承接时应在报告中充分提示：评价基于有限时间和有限资料，不保证发现所有问题；并非按审计准则/评估准则出具，不应像审计/评估结论那样依赖。

### 投后评价四阶段框架

> 类型：流程
> 来源：股权投后评价业务基本指引（试行）
> 置信度：准则原文

报告核心由"投资过程评价"和"投资效益评价"两部分构成。

**投资过程评价（按投资生命周期四阶段核查合规性）：**

| 阶段 | 核查要点 |
|------|---------|
| 投前决策 | 可行性研究、尽职调查（财务/法律/行业/技术）、风险评估、资产评估、投资决策与备案——除新设企业外，未做尽调/资产评估须说明原因 |
| 投资实施 | 投资计划/预算审批、投资协议核心条款（出资方式金额时间、董监高席位、对赌、业绩补偿、竞业禁止）、出资状况（含非货币出资作价依据、出资前置条件是否达成）、产权变更（工商/权属变更是否及时） |
| 投后管理 | 公司治理（是否派驻董监高、三会是否有效召开、是否实际参与治理）、经营管理（内控/风控体系）、财务状况（偿债/营运/盈利/成长四类指标） |
| 处置退出 | 退出决策是否合规、退出收益是否达预期、处置款是否及时到账并核算 |

**投资效益评价：**

- **经济效益**：用"**可研达成率**"对标——把投后实际的战略/业务/财务关键指标与投前可行性研究报告的预测数比对，算达成率、列差异、分析原因。
- **投后估值变动**：已退出按退出价 vs 成本算收益；已上市按基准日收盘价；有新一轮融资/转股按最新价；既未上市又近 12 个月无转让/再融资的，参照《非上市公司股权估值指引》用成本法/市场法/收益法评估，必要时另行委托评估机构。
- **社会效益**（委托人要求时）：区域经济、环境保护、科技进步。⚠️ 环保不达标（如被环保部门处罚）可考虑**一票否决**。
- **持续发展评价**：战略、经济效益、技术创新、管理水平四个维度的可持续性，特别关注是否存在内控重大缺陷和重大法律/财务纠纷。

### 投后增值服务六大抓手

> 类型：判断表
> 来源：私募股权投资和尽职调查手册（投前投后管理）
> 置信度：实务经验

[场景：投后管理落地，把"赋能"变成具体动作；前提是创始团队认同且投前已写进协议]

| 领域 | 服务形式 |
|------|---------|
| 市场地位 | 带订单、接入上市公司销售体系、品牌增信 |
| 商业模式 | 研判国家政策、对商业发展提建议 |
| 财务指标 | 可靠融资渠道、后续融资保障 |
| 团队配置 | 提供人脉网络、帮找高级管理人才 |
| 发展阻碍 | 输出管理经验，规范财务/内控/治理 |
| 发展战略 | 战略规划、派驻管理人员建持续改进机制 |

⚠️ 实务铁律：① 让投资团队和协同企业关键经办人**参与持股平台做小股东**，本身就是最好的投后管理和风控；② 能带来**订单提升**的协同才最扎实；③ 企业能真正带给标的的价值**一般比说的少**，别轻易承诺，免得损伤个人信誉；④ 投后管理人员收获与职业风险严重不对等（见效慢、量化难），做不好更尴尬。
⚠️ 反向视角：被投企业加入大平台前**极少反向尽调平台公司**（实力不对等 + 光环效应），但应至少做到——多维度调查平台实控人/创始人的人品口碑商誉，并最好先从 1~2 个项目合作试合作，否则被合并后拿走财务人事控制权、文化理念不合时，想离开比离婚还难。

### 私募股权退出路径对比

> 类型：判断表
> 来源：私募股权投资和尽职调查手册（退出方式）
> 置信度：实务经验

[场景：设计退出方案时按回报、效率、确定性权衡四条路径]

退出是"募投管退"中最基础最重要的环节。

| 路径 | 回报 | 优势 | 弊端 |
|------|------|------|------|
| IPO | 最高（常达投资额几倍至几十倍，>10倍） | 收益最大、三方利益最大化、提升 PE 知名度 | 门槛高、耗时长、机会成本高、面临政策与行情风险 |
| 兼并收购 | 较低（近年≤5倍） | 高效灵活、交割后一次性全退、缓解流动性压力 | 实力买家有限、收益低于IPO、管理层可能反对 |
| 股份回购 | 确定 | 过程简单、资本安全有保障 | 错失企业业绩爆发的潜在收益 |
| 清算 | 亏损 | 止损强制性强 | 部分/全部投资损失、市场负面影响 |

⚠️ 退出方案应在投资时就设计，与估值、交易结构一并考量——退出未实现前，所有盈利都只是账面浮盈。

## 十三、并购重组交易结构设计与典型案例

> 本章是"上市公司层面的并购重组"——区别于前面"PE 投一家未上市公司"。落脚点是**交易结构怎么搭、用什么支付、控制权怎么保、监管怎么过**。案例只提炼可复用的结构判断与教训，不复述流水账。
> 来源：上交所《上市公司并购重组典型案例汇编》+ 企业融资Ⅱ（廖连中）。

### 并购重组方式分类与适用

> 类型：判断表
> 来源：企业融资Ⅱ（并购分类）+ 上交所案例
> 置信度：实务经验

[场景：拿到一个并购需求，先定"用哪种方式"，再设计结构]

| 方式 | 含义 | 适用判断 |
|------|------|---------|
| 协议收购 | 与目标股东协商签约受让股权 | 股权集中、卖方明确、最常见 |
| 要约收购 | 向**全体股东**公开发出收购要约 | 收购上市公司控制权、私有化退市（持股将触及 30% 要约线时强制） |
| 竞价收购 | 通过公开竞价/挂牌受让 | 国资标的、产权交易所挂牌、LP 份额公开转让 |
| 吸收合并（A 吸 A） | 一方发股换股，另一方注销并入存续方 | 两家实力相当（常为上市公司）整合、解决同业竞争 |
| 资产置换 | 以资产换资产，差额补现金/股份 | 置出劣质主业、置入优质资产；可冲抵对价减少发股 |
| 债务重组 | 调整债权债务条件 | 标的高负债、需先减债再整合 |
| 分拆/重组上市（借壳） | 子公司分拆独立上市 / 借壳注入 | 集团内资产证券化、绕开排队 IPO |

⚠️ 选择顺序：先看**是否涉及上市公司控制权变化**（变→要约/借壳监管最严）、**对方是否上市公司**（是→优先吸收合并）、**是否国资**（是→强制评估+挂牌）。

### 并购支付工具选择决策

> 类型：判断表
> 来源：上交所并购重组典型案例（闻泰/洛阳钼业/华电国际等）
> 置信度：实务经验
> 关联：本章 跨境并购交易结构、市场化债转股、A吸A换股吸收合并

[场景：交易作价确定后，选支付工具——核心权衡是"现金压力 vs 控制权稀释 vs 卖方诉求 vs 估值分歧"]

| 支付工具 | 何时选 | 代价/约束 |
|---------|--------|----------|
| 现金 | 卖方要现金、追求交割快、不愿稀释控制权、跨境卖方诉求复杂 | 占用资金/加杠杆，财务风险大；程序相对简单，海外收购常用 |
| 发行股份 | 标的大、现金不足、想绑定卖方长期利益 | 稀释控制权；A 股发股底价为市场参考价的 80%（2023 年前为 90%） |
| 定向可转债 | 股价低于每股净资产无法发股（国资）、想给投资人"保底息+股价上行期权" | 需设强制转股/分红/资产注入等措施夯实转股；2023 年起可 100% 用定向可转债 |
| 资产置换 | 既要置出劣质资产又要置入优质资产 | 可**冲抵对价、减少发股、避免过度摊薄** |
| 承债式 | 标的负债是收购对价的一部分 | 需承接并重组债务 |
| 综合证券 | 单一工具无法平衡各方诉求 | 现金+股份+可转债组合，结构复杂 |
| Earn-out（或有支付） | 估值分歧大、标的未来不确定性高（跨境/集成电路/生物医药）、想缓解短期现金压力 | 见下条专项 |

⚠️ 决策主线：**现金保控制权但加杠杆，发股缓现金但稀释控制权**。控制权集中度低的收购方（如闻泰科技实控人仅约 30%）应**先债后股、随股价上涨逐步缩小发股规模**，把稀释降到最低。

### 跨境并购交易结构设计（蛇吞象）

> 类型：流程
> 来源：上交所案例（闻泰科技收购安世半导体 267 亿、洛阳钼业、长电科技、锦江卢浮）
> 置信度：实务经验

[场景：收购规模远超自身净资产的跨境"蛇吞象"，设计可落地、控制权可保、风险可控的结构]

可复用的结构要点：

- **两步走/分步收购**：先以现金参与上层基金 LP 份额公开竞拍取得先机（锁定标的），再用发股+配套融资+境外贷款收购其余股权。分步降低一次性资金与控制权冲击。
- **多层 SPV + 出资比例设计保控制权**：通过境内外多层特殊目的公司持有标的，在各层用出资/借款拆分把战投/产业基金的话语权压到不夺取收购方控制权的水平，并控制各层董事会多数席位。
- **股权+债权混合融资**：上市公司层面 + 标的层面分别融资；战投出资 + 战投借款（后续债转股）+ 银行并购贷 + 配套募资 + 境外银团贷款多渠道并用。
- **境外银团贷款优于内保外贷**：以**标的子公司**为保证人、用标的子公司股权质押担保，收购方自身不提供担保、不质押自身股权——规避自身股权被质押风险、避免资金出海汇兑损失。
- **差异化定价**：并购基金中 GP（牵头/管理贡献）与 LP、单一最大 LP（按公开竞拍价）与其他小额 LP（折让协商价）分别定价，用对价体现不同出资人价值贡献，平衡各方。
- **分期支付控风险**：如签约后付 10%、配套资金到账后付 90%，用付款节奏把控交割风险。

⚠️ 收购**有限合伙 LP 份额**而非直接收底层股权，有 5 类特殊风险须先查：可转让性（合伙协议是否限制转让）、控制性（LP 通常只有重大影响，可能只能权益法甚至按金融资产核算）、承接义务（管理费/超额收益）、合伙期限届满清算风险、新入伙 LP 对入伙前债务的连带责任。
⚠️ 跨境并购审批多（境内反垄断/发改/商务/外汇 + 境外多国反垄断与外资审批），应**协议签署后各项审批并联同步推进**，并对久拖不决的审批做预案。

### Earn-out（或有支付）机制设计

> 类型：规则
> 来源：上交所案例（洛阳钼业以 LME 铜/钴价为触发条件）
> 置信度：实务经验

[场景：买卖双方估值分歧大、标的未来不确定性高时，用后付机制对冲风险]

Earn-out = 按标的未来业绩/约定条件的达标程度调整支付款的合约条款。四要素：**支付总额、衡量指标、时间期限、支付安排**。

- **衡量指标**：财务性（收入/净利/EBITDA）适合经营性标的；非财务性适合发展阶段标的；**外部市场指标**（如大宗商品价格）适合对冲跨境/周期性风险。
- **三大作用**：①事前筛选优质标的、防炒作（延期支付倒逼卖方对未来负责）；②捆绑双方利益、降不确定性；③后付属性缓解收购方短期现金压力（经济低迷、融资难时尤其有效）。

⚠️ Earn-out 条款设置难度大、抬高谈判成本，在市场化程度不高的 A 股推广较难；但能缓解**业绩承诺随意性和业绩补偿兑现难**，是比刚性业绩对赌更柔性的工具。

### PE 基金参与上市公司并购（模式与风险）

> 类型：判断表
> 来源：上交所案例（长电科技引入产业基金/芯电半导体；歌力思、西王食品、青岛海尔+KKR）
> 置信度：实务经验
> 关联：本章 跨境并购交易结构

[场景：上市公司借基金资金/资源做并购时的模式选择与风控]

两类合作模式：

| 模式 | 操作 | 特点 |
|------|------|------|
| 大股东/关联方 + 基金 | 大股东与基金设 SPV 先收标的，择机以股份/现金注入上市公司 | 兼具融资 + 缩短上市公司直接收购的审核；海外收购多见 |
| 上市公司 + 基金 | 上市公司作参股方/LP（控制力较高，引战投）或控股方引基金（偏融资功能） | 视上市公司角色定控制力 |

基金核心诉求是**保收益 + 退出渠道**，上市公司核心诉求是**完成并购 + 保控制权**，靠条款平衡（如给上市公司优先收购权/回购义务 + 给基金向大股东和第三方出售权；约定锁定期、年化收益率、债转股权利）。

⚠️ 三条风控铁律：
- **合理估值防利益输送**：警惕大股东与基金合谋低买高卖、标的经基金转手数月估值翻倍；设信息隔离防内幕交易，别把上市公司当"提款机"。
- **平衡收购权与退出渠道**：上市公司对注入资产无决策权时，应限制注入资产的行业/盈利能力。
- **看重基金的战略整合能力**，而非只看资金实力（KKR 协助海尔整合 GE 家电是正例）。

### 市场化债转股交易结构（定向可转债）

> 类型：规则
> 来源：上交所案例（华电国际定向可转债债转股）
> 置信度：实务经验

[场景：火电/周期性国企股价长期低于每股净资产、又要降杠杆做市场化债转股]

矛盾：国资发股价**原则上不低于最近一期经审计每股净资产**，但股价低于净资产时直接发股会让投资人账面浮亏，常规债转股不适用。

解法：**交易对价以定向可转债为主（如 98%）+ 少量股份（2%）**——股份发行价和转股价都定为每股净资产（合规），同时压低股份占比减少投资人当期浮亏。

⚠️ 定向可转债债转股的关键是**成功转股**（转股才算真正降杠杆），须用组合措施夯实转股确定性：参考公开可转债的阶梯利率结构（突出"股性"、低利率）、承诺高比例分红提振信心、注入资产增厚每股收益、设**强制转股条款**（发行满一定期限后股价连续高于转股价即可强制转股）。

### A 吸 A 换股吸收合并要点

> 类型：判断表
> 来源：上交所案例（宝钢吸并武钢、中航电子吸并中航机电、南北车、中远中海）
> 置信度：实务经验

[场景：两家上市公司（尤其同一实控人下）整合、解决同业竞争时的换股吸收合并方案]

| 要素 | 要点 |
|------|------|
| 合并方选择 | 通常由**实力更强/股权控制比例更高**的一方作存续方（节省整体交易成本，如宝钢吸武钢、选股权控制比例更高的中航电子存续） |
| 换股价格 | 以董事会决议公告日前 20/60/120 个交易日均价为市场参考价，不低于其 90%（2023 年起降至 80%）；可在底价之上按可比估值/溢价折价**谈判调整**，吸并方与被吸并方定价区间不要求一致 |
| 换股比例 | = 被吸并方换股价 ÷ 吸并方换股价；除权除息外原则不调整 |
| 现金选择权/异议股东收购请求权 | 由大股东（非上市公司本身）提供的现金退出机制；价格**多 ≤ 换股价**（换股股东承担经营/市场风险应享更高收益；现金选择权类认沽期权，其价值体现在与换股价价差） |
| 价格调整机制 | 多为**单边下调**（避免市价低于现金选择权形成倒挂套利）；股价上行时投资人可自行二级市场卖出，无需上调 |
| 库存股处理 | 回购用途已定为注销的库存股**不参与换股**；用途为"出售"的回购股理论上可参与，无明确先例 |
| 跨沪深交易所 | 被吸并方先在原交易所终止上市，再到存续方所在交易所换股（需跨市场转登记） |

⚠️ 换股吸收合并是吸并方/被吸并方/双方大股东/中小股东多方博弈结果；现金选择权定价过高会留套利空间、抬高成本，且中小股东为博高现金选择权更可能投反对票，反而不利交易达成。

### 国资并购重组特殊规则

> 类型：规则
> 来源：上交所案例（宝钢武钢、中交设计、华电国际）+ 国资监管规定
> 置信度：实务经验
> 关联：本文件第九章 法定必须评估的情形（国资强制评估）

⚠️ 涉及国有资产的并购重组三条硬规则：
- **发股/换股价原则上不低于每股净资产**（国有上市公司发行新股价不低于净资产水平）。例外：宝钢吸并武钢用**市场化定价**（前 20 日均价的 90%）突破了净资产限制，但**须取得国资委支持**——理由是两家均为 A 股上市公司、二级市场价公允、同一上市公司所有股东同价换股、不引入新股东、不造成国有资产流失。
- **必经评估并备案**：国资交易强制评估，成交价不得低于评估值，未评估直接定价可致交易无效（详见第九章）。
- **可用无偿划转 + 集团更名**整合控股股东层面（如宝钢集团更名中国宝武、武钢集团股权无偿划转入），改善"一股独大"、便捷间接减持获取运营资金。

### 重组上市（借壳）的 IPO 等同条件与分拆/同业竞争论证

> 类型：规则
> 来源：上交所案例（中交设计分拆借壳祁连山）
> 置信度：实务经验
> 关联：第七章 IPO 财务准备/被否事由

[场景：判断借壳/分拆上市方案能否过关]

- **重组上市≈IPO 门槛**：标的资产须符合 IPO 发行条件、板块定位、交易所《重大资产重组审核规则》的财务指标；多个标的原则上**模拟合并**财务数据判断是否达标。差异：标的交易后是上市公司子公司，可为有限公司或股份公司（IPO 要求拟上市主体须为股份公司）。
- **资产置换+托管减摊薄**：置出资产先内部整合（"人随资产走"安置员工）→ 用资产置换冲抵对价、减少发股、避免股东权益过度摊薄 → 置出资产可托管给有运营经验的关联上市公司（发挥协同、解决接手方无经营经验问题）。
- **同业竞争"30% 口径"论证**：分拆/重组中存在重叠业务时，论证报告期内集团下属企业从事同类业务的收入、毛利占置入资产主营收入、毛利比例**未超 30%**即不构成重大不利影响；对确实重叠的存量业务（如在手未完工总包合同）给**明确可行的解决方案**（不再独立开展、调整业务模式、组联合体分工）。
- **争取股东支持**：股权分散时提前与重要自然人股东沟通、办路演引战投增持，争取股东大会赞成票。

### 跨境换股回归 A 股（三步走）

> 类型：流程
> 来源：上交所案例（首旅酒店收购如家、私有化回归）
> 置信度：实务经验

[场景：境外上市的中概股私有化回归 A 股 / A 股收购境外上市标的]

三步走结构：**①现金收购非主要股东股权实现私有化**（境外设 SPV、银行贷款支付）→ **②向主要股东/管理层发行股份购买其余股权**（绑定团队、降低现金需求）→ **③配套募集资金**（缓解偿债压力）。

- 私有化与 A 股证券化**同步进行**：无需先借壳找壳，私有化同时注入 A 股上市公司。
- 向管理团队/战略合作方支付**股份**（而非现金）→ 锁定核心团队、形成利益联盟。
- 发股购买境外上市公司资产**不以评估报告为依据**，按市值/净资产/可比公司估值定价。

⚠️ 跨境换股曾长期受限（10 号文要求作支付的股权须为境外上市公司股权）；2024 年《外国投资者对上市公司战略投资管理办法》修订后放宽：允许以**境外非上市公司股份**作支付对价、取消定向发行的 10% 持股比例下限、明确**自然人可作战投**、锁定期由 3 年降至 12 个月。

### 并购择时与杠杆收购财务风险化解

> 类型：规则
> 来源：上交所案例（洛阳钼业持币静守、长电科技杠杆风险）
> 置信度：实务经验

[场景：决定何时出手并购、以及高杠杆并购的财务风险防范]

**择时（持币静守，伺机而动）**：行业周期底部（卖方被迫出售、优质标的折价）是收购良机；但前提是**自身现金流充裕 + 银行授信充足 + 标的符合自身战略定位**——洛阳钼业在矿业周期底部、账面充裕现金+百亿授信下抄底境外矿产。理性并购的本质是**为交易做足准备**（找真实商业理由、想清整合计划、备好可行融资方案），而非追热点。

⚠️ 杠杆收购（尤其标的尚亏损）的财务风险化解三招：
- **评估杠杆资金风险敞口**：杠杆债务主要靠并购后标的现金流偿还；若标的仍亏损（如长电收购时星科金朋亏损），收购方自身盈利又薄，风险敞口覆盖不住，资产负债率会快速恶化（长电从 63% 升至 77%）。
- **预留整合期资金，别在并购初期就把杠杆用满**：负债率高企会让后续外部借款更贵更难，反过来拖累标的整合。
- **设计杠杆退出/化解路径**：如把战投借款安排**债转股**，缓解纯现金偿债冲击、争取市值成长缓冲时间、避免控制权变更——但市值不及预期时仍可能被迫增发致控制权旁落（长电终致原大股东失去第一大股东地位）。
- **标的先行债务重组**：交割前用过桥贷款置换标的高息债、发永续证券降资产负债率，减轻整合期债务负担。

## 三表联动模型（3-Statement Model）

### 概述

三表联动模型是投行、PE、FP&A 的基础工具：利润表、资产负债表、现金流量表通过内在逻辑相互勾稽，改变一个假设，三张表同步联动。

### 搭建顺序

```
假设表（Assumptions）
    ↓
利润表（Income Statement）
    ↓
资产负债表（Balance Sheet）
    ↓
现金流量表（Cash Flow Statement）
    ↓
校验表（Checks）
```

### 假设表（驱动因子）

| 类别 | 典型假设 |
|------|----------|
| 收入 | 收入增长率、产品单价、销量 |
| 成本 | 毛利率、COGS 占收入比 |
| 费用 | 销售费用率、管理费用率、研发费用率 |
| 营运资本 | DSO（应收周转天数）、DIO（存货周转天数）、DPO（应付周转天数） |
| 资本支出 | CapEx 占收入比、折旧年限 |
| 融资 | 利率、还款计划、股权融资 |
| 税务 | 有效税率 |

### 利润表搭建

```
营业收入
- 营业成本
= 毛利
- 销售费用
- 管理费用
- 研发费用
= EBITDA
- 折旧与摊销
= EBIT（营业利润）
- 财务费用（利息）
+ 投资收益
= 利润总额（EBT）
- 所得税费用
= 净利润
```

**规则：** 预测期每个科目必须由假设表驱动（公式引用），不能硬编码数字。

### 资产负债表搭建

**资产侧：**
- 货币资金 = 现金流量表期末现金（联动）
- 应收账款 = 收入 × DSO ÷ 365
- 存货 = 营业成本 × DIO ÷ 365
- 固定资产净值 = 期初 + CapEx - 折旧

**负债侧：**
- 应付账款 = 营业成本 × DPO ÷ 365
- 短期借款 / 长期借款 = 按还款计划
- 应付利息 = 借款余额 × 利率（可能产生循环引用）

**权益侧：**
- 未分配利润 = 期初 + 净利润 - 分红

### 现金流量表搭建

**经营活动：**
```
净利润
+ 折旧与摊销（加回非现金）
+ 股份支付（加回非现金）
- 应收账款增加（营运资本变动）
- 存货增加
+ 应付账款增加
= 经营活动现金流量净额
```

**投资活动：**
```
- 购建固定资产（CapEx）
- 长期投资
= 投资活动现金流量净额
```

**筹资活动：**
```
+ 借款收到的现金
- 偿还借款
- 分配股利
+ 吸收投资
= 筹资活动现金流量净额
```

**期末现金 = 期初现金 + 经营 + 投资 + 筹资**

### 核心校验规则（必须全部通过）

| 校验项 | 公式 | 期望结果 |
|--------|------|----------|
| 资产负债表平衡 | 资产 - 负债 - 权益 | = 0 |
| 现金勾稽 | 现金流量表期末现金 - 资产负债表货币资金 | = 0 |
| 净利润联动 | 利润表净利润 - 现金流量表起始净利润 | = 0 |
| 未分配利润滚动 | 期初未分配利润 + 净利润 - 分红 - 期末未分配利润 | = 0 |

**如果校验不通过：**
1. 先检查资产负债表是否平衡
2. 再检查现金是否勾稽
3. 最后检查未分配利润滚动
4. 常见原因：漏了某个营运资本变动项、折旧没加回、利息循环引用未收敛

### 循环引用处理

利息费用 → 净利润 → 现金 → 借款余额 → 利息费用（循环）

**解决方法：**
- Excel：开启迭代计算（文件 → 选项 → 公式 → 启用迭代计算，最大迭代次数 100）
- 加一个"断路器"开关：在假设表放一个 TRUE/FALSE 单元格，FALSE 时利息用上期余额计算（打断循环）

### 情景分析

| 情景 | 说明 |
|------|------|
| 基准情景（Base） | 管理层指引或市场一致预期 |
| 乐观情景（Upside） | 收入增长超预期、毛利率扩张 |
| 悲观情景（Downside） | 收入下滑、毛利率压缩 |

用 CHOOSE 函数或 INDEX/MATCH 实现情景切换，一个下拉框控制三张表同时变化。

**情景校验：** 乐观 > 基准 > 悲观（净利润、EBITDA、自由现金流、各项利润率）。如果不满足这个层级关系，说明假设设置有逻辑问题。
