## 第1章 财务管理概述

### 财务管理目标

| 目标 | 含义 | 局限性 |
|------|------|--------|
| 利润最大化 | 利润越多越好 | 未考虑时间、资本额、风险 |
| 每股收益最大化 | 利润/股数 | 未考虑时间、风险；不同公司不可比 |
| **股东财富最大化**（本教材采纳） | 股东权益市场价值最大化 | 综合考虑时间、风险、资本 |

- 股东财富 = 股东权益的市场价值
- 股东权益市场增加值 = 股东权益市场价值 − 股东投资资本
- 股价最大化 ≈ 股东财富最大化（前提：资本市场有效、投资资本不变）
- 公司价值最大化 ≈ 股东财富最大化（前提：债务价值不变）

### 代理问题与协调

| 代理关系 | 冲突表现 | 协调措施 |
|----------|----------|----------|
| 股东 vs 经营者 | 道德风险（不尽力）、逆向选择（损害股东利益） | 监督（审计、治理）+ 激励（股票期权、奖金） |
| 股东 vs 债权人 | 投资高风险项目、发行新债稀释旧债 | 限制性条款、拒绝合作/提前收回贷款 |

### 金融市场类型

| 分类标准 | 类型 | 说明 |
|----------|------|------|
| 期限 | 货币市场（≤1年）/ 资本市场（>1年） | 短期流动性 vs 长期融通 |
| 属性 | 债务市场 / 股权市场 | 固定收益 vs 剩余权益 |
| 发行 | 一级市场（发行）/ 二级市场（流通） | 筹资 vs 交易 |
| 程序 | 场内（交易所）/ 场外（OTC） | 集中竞价 vs 协商定价 |

**有效市场假说三个层次**：弱式（历史价格）、半强式（公开信息）、强式（所有信息）

## 第2章 财务报表分析和财务预测

### 财务比率分析体系

**短期偿债能力比率**

| 指标 | 公式 | 判断标准 |
|------|------|----------|
| 营运资本 | 流动资产 − 流动负债 | 越多越安全，但过多说明资金闲置 |
| 流动比率 | 流动资产 ÷ 流动负债 | 无统一标准，行业差异大；营业周期越短合理值越低 |
| 速动比率 | 速动资产 ÷ 流动负债 | 速动资产 = 货币资金+交易性金融资产+应收款项 |
| 现金比率 | (货币资金+交易性金融资产) ÷ 流动负债 | 最保守指标 |
| 现金流量比率 | 经营活动现金流量净额 ÷ 流动负债 | 流量视角，更具说服力 |

- 营运资本配置比率 = 营运资本 ÷ 流动资产 = 1 − 1/流动比率
- 表外增强因素：银行授信额度、可快速变现的非流动资产、偿债声誉
- 表外削弱因素：担保或有负债、未决诉讼

**长期偿债能力比率**

| 指标 | 公式 | 说明 |
|------|------|------|
| 资产负债率 | 总负债 ÷ 总资产 | 越低越安全，但过低说明未充分利用杠杆 |
| 产权比率 | 总负债 ÷ 股东权益 | = 资产负债率 ÷ (1−资产负债率) |
| 权益乘数 | 总资产 ÷ 股东权益 | = 1 + 产权比率 = 1/(1−资产负债率) |
| 长期资本负债率 | 非流动负债 ÷ (非流动负债+股东权益) | 反映长期资本结构 |
| 利息保障倍数 | EBIT ÷ 利息费用 | <1表示经营收益不能覆盖利息；越大越安全 |
| 现金流量利息保障倍数 | 经营活动现金流量净额 ÷ 利息费用 | 比利息保障倍数更可靠 |
| 现金流量与负债比率 | 经营活动现金流量净额 ÷ 负债总额 | 流量视角的长期偿债能力 |
| 资产有息负债率 | 有息负债 ÷ 资产总额 | 衡量付息债务压力，比资产负债率更聚焦 |
| 有形净值债务率 | 负债总额 ÷（股东权益 - 无形资产净值） | 剔除无形资产后的真实偿债能力 |
| 偿债保障比率 | 负债总额 ÷ 经营活动现金流量净额 | 用当前现金流还清全部负债需要几年 |

**营运能力比率**

| 指标 | 公式 | 说明 |
|------|------|------|
| 应收账款周转率 | 营业收入 ÷ 平均应收账款 | 周转天数 = 365/周转率 |
| 存货周转率 | 营业收入(或营业成本) ÷ 平均存货 | 评估管理业绩用成本，分解周转率用收入 |
| 流动资产周转率 | 营业收入 ÷ 平均流动资产 | |
| 非流动资产周转率 | 营业收入 ÷ 平均非流动资产 | |
| 总资产周转率 | 营业收入 ÷ 平均总资产 | 综合营运效率 |

**盈利能力比率**

| 指标 | 公式 |
|------|------|
| 营业净利率 | 净利润 ÷ 营业收入 |
| 总资产净利率(ROA) | 净利润 ÷ 平均总资产 |
| 权益净利率(ROE) | 净利润 ÷ 平均股东权益 |

**市价比率**

| 指标 | 公式 | 说明 |
|------|------|------|
| 市盈率(P/E) | 每股市价 ÷ 每股收益 | 反映市场对盈利的定价 |
| 市净率(P/B) | 每股市价 ÷ 每股净资产 | 反映市场对净资产的溢价 |
| 市销率(P/S) | 每股市价 ÷ 每股营业收入 | 适用于亏损企业 |

### 传统杜邦分析体系

$$\text{权益净利率(ROE)} = \text{营业净利率} \times \text{总资产周转次数} \times \text{权益乘数}$$

$$= \frac{\text{净利润}}{\text{营业收入}} \times \frac{\text{营业收入}}{\text{总资产}} \times \frac{\text{总资产}}{\text{股东权益}}$$

### 改进的杜邦分析体系（管理用财务报表）

**核心思想**：区分经营活动与金融活动

| 管理用报表项目 | 公式 |
|----------------|------|
| 净经营资产 | 经营资产 − 经营负债 |
| 净负债 | 金融负债 − 金融资产 |
| 净经营资产 | 净负债 + 股东权益 |
| 税后经营净利润 | 净利润 + 税后利息费用 |
| 税后利息费用 | 利息费用 ×（1 − 税率） |

**改进的核心公式**：

$$\text{权益净利率} = \text{净经营资产净利率} + (\text{净经营资产净利率} - \text{税后利息率}) \times \text{净财务杠杆}$$

其中：
- 净经营资产净利率 = 税后经营净利润 ÷ 净经营资产
- 税后利息率 = 税后利息费用 ÷ 净负债
- 净财务杠杆 = 净负债 ÷ 股东权益
- 经营差异率 = 净经营资产净利率 − 税后利息率
- 杠杆贡献率 = 经营差异率 × 净财务杠杆

### 财务预测——销售百分比法

**步骤**：
1. 确定经营资产和经营负债的销售百分比
2. 预计各项经营资产和经营负债 = 预计营业收入 × 销售百分比
3. 融资总需求 = 预计净经营资产 − 基期净经营资产
4. 外部融资额 = 融资总需求 − 可动用金融资产 − 增加留存收益

$$\text{增加留存收益} = \text{预计营业收入} \times \text{营业净利率} \times (1 - \text{股利支付率})$$

### 可持续增长率

**定义**：不增发新股/回购股票，不改变经营效率和财务政策时的最大销售增长率。

**根据期初股东权益**：

$$g = \text{营业净利率} \times \text{期末总资产周转次数} \times \text{期末总资产/期初权益} \times \text{利润留存率}$$

**根据期末股东权益**：

$$g = \frac{\text{营业净利率} \times \text{总资产周转次数} \times \text{权益乘数} \times \text{利润留存率}}{1 - \text{营业净利率} \times \text{总资产周转次数} \times \text{权益乘数} \times \text{利润留存率}}$$

**内含增长率**（仅靠留存收益增长，外部融资=0）：

$$\text{内含增长率} = \frac{\text{净利润/净经营资产} \times \text{利润留存率}}{1 - \text{净利润/净经营资产} \times \text{利润留存率}}$$

## 第3章 价值评估基础

### 货币时间价值核心公式

| 类型 | 终值(F) | 现值(P) |
|------|---------|---------|
| 复利 | $F = P(1+i)^n$ | $P = F(1+i)^{-n}$ |
| 普通年金 | $F = A \cdot \frac{(1+i)^n - 1}{i}$ | $P = A \cdot \frac{1-(1+i)^{-n}}{i}$ |
| 预付年金 | $F = A[(F/A,i,n+1)-1]$ | $P = A[(P/A,i,n-1)+1]$ |
| 递延年金(递延m期) | 同普通年金 | $P = A(P/A,i,n)(P/F,i,m)$ |
| 永续年金 | 无终值 | $P = A/i$ |

**有效年利率**：$i_{eff} = (1 + r/m)^m - 1$（r为报价利率，m为年复利次数）

**连续复利**：$i_{eff} = e^r - 1$

### 风险与报酬

**单项资产风险度量**：
- 期望报酬率：$\bar{K} = \sum P_i K_i$
- 方差：$\sigma^2 = \sum P_i(K_i - \bar{K})^2$
- 标准差：$\sigma = \sqrt{\sigma^2}$
- 变异系数（标准离差率）：$CV = \sigma / \bar{K}$

### 投资组合理论

- 组合期望报酬率：$\bar{R}_p = \sum w_i \bar{R}_i$（加权平均）
- 两资产组合标准差：$\sigma_p = \sqrt{w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\rho_{12}\sigma_1\sigma_2}$
- 相关系数ρ=1时无分散效应；ρ<1时有分散效应；ρ=−1时可完全消除风险

### 资本资产定价模型（CAPM）

$$R_i = R_f + \beta_i(R_m - R_f)$$

- $R_f$：无风险利率
- $R_m$：市场组合期望报酬率
- $R_m - R_f$：市场风险溢价
- $\beta_i$：第i项资产的系统风险系数

**β系数**：$\beta_i = \frac{Cov(R_i, R_m)}{\sigma_m^2} = \frac{\rho_{im}\sigma_i}{\sigma_m}$

**资本市场线（CML）**：描述有效组合的期望报酬率与总风险（标准差）的线性关系

**证券市场线（SML）**：描述所有资产/组合的期望报酬率与系统风险（β）的线性关系，即CAPM

### 债券估值

$$V_B = \sum_{t=1}^{n}\frac{I}{(1+r)^t} + \frac{M}{(1+r)^n} = I \cdot (P/A,r,n) + M \cdot (P/F,r,n)$$

### 股票估值

**零增长模型**：$P_0 = D/r_s$

**固定增长模型（Gordon模型）**：$P_0 = \frac{D_1}{r_s - g} = \frac{D_0(1+g)}{r_s - g}$

**两阶段模型**：高速增长期逐年折现 + 稳定期用Gordon模型折现

## 第4章 资本成本

### 债务资本成本（税后）

$$r_d(1-T)$$

估计方法：
1. **到期收益率法**：使债券市价等于未来现金流现值的折现率
2. **可比公司法**：参照同风险等级公司债券的到期收益率
3. **风险调整法**：$r_d = r_f + \text{企业信用风险补偿率}$

### 权益资本成本

| 方法 | 公式 |
|------|------|
| CAPM | $r_s = r_f + \beta(r_m - r_f)$ |
| 股利增长模型 | $r_s = D_1/P_0 + g$ |
| 债券收益率加风险溢价 | $r_s = r_d + RP_s$（RP通常3%~5%） |

### 加权平均资本成本（WACC）

$$WACC = \sum w_j \cdot r_j = w_d \cdot r_d(1-T) + w_p \cdot r_p + w_s \cdot r_s$$

权重确定方法：账面价值权重、市场价值权重（推荐）、目标资本结构权重

## 第5章 投资项目资本预算

### 项目评价方法

| 方法 | 公式/规则 | 决策标准 |
|------|-----------|----------|
| **净现值(NPV)** | $NPV = \sum\frac{CF_t}{(1+r)^t} - C_0$ | NPV > 0 接受 |
| **内含报酬率(IRR)** | 令NPV=0的折现率 | IRR > 资本成本 接受 |
| **现值指数(PI)** | 未来现金流现值 ÷ 初始投资 | PI > 1 接受 |
| **回收期** | 累计现金流=0的时间 | 越短越好（辅助指标） |
| **折现回收期** | 累计折现现金流=0的时间 | 越短越好（辅助指标） |

### 项目现金流量估计原则

- **增量原则**：只考虑有无该项目的差异现金流
- **税后原则**：使用税后现金流
- **沉没成本不考虑**
- **机会成本要考虑**
- **关联效应要考虑**（如蚕食效应）

**营业现金流量**：

$$OCF = \text{税后营业收入} - \text{税后付现成本} + \text{折旧抵税}$$
$$= \text{收入}(1-T) - \text{付现成本}(1-T) + \text{折旧} \times T$$
$$= (\text{收入} - \text{付现成本} - \text{折旧})(1-T) + \text{折旧}$$
$$= \text{税后净利润} + \text{折旧}$$

### 项目折现率估计

**使用类比法估计项目β**：

当项目的系统风险与公司整体不同时，需要单独估计项目折现率：

1. 找到经营该类业务的上市可比公司
2. 卸载可比公司的财务杠杆：$\beta_{资产} = \frac{\beta_{权益}}{1 + (1-T) \times (D/E)}$
3. 加载目标公司的财务杠杆：$\beta_{权益(目标)} = \beta_{资产} \times [1 + (1-T) \times (D/E)_{目标}]$
4. 用CAPM计算项目权益资本成本，再计算项目WACC

**项目特有风险调整**：
- 如果项目风险高于公司平均风险，折现率上调1%~3%
- 如果项目风险低于公司平均风险，折现率下调1%~3%
- 调整幅度依赖主观判断，缺乏理论依据

### 互斥项目决策

- 寿命相同：直接比较NPV
- 寿命不同：**共同年限法**（最小公倍数）或**等额年金法**（NPV÷年金现值系数）

### 通货膨胀处理

- 名义现金流用名义折现率折现
- 实际现金流用实际折现率折现
- $(1+\text{名义利率}) = (1+\text{实际利率})(1+\text{通胀率})$

## 第6章 期权价值评估

### 期权基本概念

| 要素 | 看涨期权(Call) | 看跌期权(Put) |
|------|---------------|---------------|
| 权利 | 买入标的资产 | 卖出标的资产 |
| 到期日价值 | Max(S−X, 0) | Max(X−S, 0) |
| 内在价值 | Max(S−X, 0) | Max(X−S, 0) |
| 时间溢价 | 期权价值 − 内在价值 | 期权价值 − 内在价值 |

**看涨-看跌平价关系**（欧式）：$C - P = S - PV(X)$

**影响期权价值的因素**：

| 因素 | 看涨期权 | 看跌期权 |
|------|----------|----------|
| 股价↑ | ↑ | ↓ |
| 执行价格↑ | ↓ | ↑ |
| 到期时间↑ | ↑ | ↑ |
| 波动率↑ | ↑ | ↑ |
| 无风险利率↑ | ↑ | ↓ |
| 股利↑ | ↓ | ↑ |

### 二叉树模型

**单期模型核心公式**：

$$C_0 = \frac{(1+r-d)}{(u-d)} \cdot \frac{C_u}{1+r} + \frac{(u-1-r)}{(u-d)} \cdot \frac{C_d}{1+r}$$

- 套期保值比率（Delta）：$H = \frac{C_u - C_d}{S_0(u-d)}$
- 上行概率（风险中性）：$p = \frac{1+r-d}{u-d}$
- 期权价值 = 购买股票支出 − 借款 = $H \cdot S_0 - B$

**多期模型**：从后向前逐级推进

- $u = e^{\sigma\sqrt{t/n}}$，$d = 1/u$

### BS模型

$$C_0 = S_0 N(d_1) - Xe^{-r_c t} N(d_2)$$

$$d_1 = \frac{\ln(S_0/X) + (r_c + \sigma^2/2)t}{\sigma\sqrt{t}}$$

$$d_2 = d_1 - \sigma\sqrt{t}$$

- $N(d)$：标准正态分布累积概率
- $r_c$：连续复利无风险利率
- $\sigma$：股票报酬率标准差（连续复利）

### 实物期权

| 类型 | 含义 | 对应金融期权 |
|------|------|-------------|
| 扩张期权 | 未来扩大投资规模的权利 | 看涨期权 |
| 放弃期权 | 中途放弃项目获得残值的权利 | 看跌期权 |
| 延迟期权 | 推迟投资等待更多信息的权利 | 看涨期权 |

项目价值 = NPV（传统）+ 实物期权价值

## 第7章 企业价值评估

### 现金流折现法

| 模型 | 现金流定义 | 折现率 |
|------|-----------|--------|
| **FCFF（实体自由现金流）** | 税后经营净利润 + 折旧 − 资本支出 − 营运资本增加 | WACC |
| **FCFE（股权自由现金流）** | FCFF − 利息(1−T) − 偿还本金 + 新借债务 | 权益资本成本 |

**FCFF详细公式**：

$$FCFF = \text{税后经营净利润} - \text{净经营资产增加}$$
$$= \text{税后经营净利润} \times (1 - \text{净经营资产增长率/净经营资产净利率})$$

**两阶段模型**：

$$V = \sum_{t=1}^{n}\frac{FCFF_t}{(1+WACC)^t} + \frac{FCFF_{n+1}/(WACC-g)}{(1+WACC)^n}$$

### 相对价值法

| 指标 | 公式 | 适用 | 驱动因素 |
|------|------|------|----------|
| **市盈率(P/E)** | 股价/EPS | 盈利稳定企业 | 增长率、股利支付率、风险 |
| **市净率(P/B)** | 股价/每股净资产 | 重资产企业 | ROE、增长率、风险 |
| **市销率(P/S)** | 股价/每股营业收入 | 亏损企业 | 营业净利率、增长率 |
| **EV/EBITDA** | 企业价值/息税折旧摊销前利润 | 资本密集型 | 消除资本结构差异 |

**本股价值 = 可比公司平均乘数 × 目标公司对应指标**

修正方法：修正市盈率 = 实际P/E ÷ (预期增长率×100)

## 第8章 资本结构

### MM理论

| 命题 | 无税MM | 有税MM |
|------|--------|--------|
| 命题I（价值） | $V_L = V_U$（资本结构无关） | $V_L = V_U + T \times D$（负债有税盾价值） |
| 命题II（资本成本） | $r_s = r_0 + (r_0-r_d)(D/E)$ | $r_s = r_0 + (r_0-r_d)(1-T)(D/E)$ |

- $r_0$：无杠杆权益成本 = 无负债公司的WACC
- 有税时：WACC随负债增加而降低

### 权衡理论

$$V_L = V_U + PV(\text{税盾}) - PV(\text{财务困境成本})$$

最优资本结构：税盾边际收益 = 财务困境边际成本

### 优序融资理论

融资顺序：内部融资（留存收益）→ 债务融资 → 股权融资

原因：信息不对称导致外部融资成本高于内部融资

### 杠杆系数

| 杠杆 | 公式 | 含义 |
|------|------|------|
| **经营杠杆系数(DOL)** | $\frac{\Delta EBIT/EBIT}{\Delta Q/Q} = \frac{Q(P-V)}{Q(P-V)-F} = \frac{M}{EBIT}$ | 销量变动→EBIT变动的放大倍数 |
| **财务杠杆系数(DFL)** | $\frac{\Delta EPS/EPS}{\Delta EBIT/EBIT} = \frac{EBIT}{EBIT-I}$ | EBIT变动→EPS变动的放大倍数 |
| **联合杠杆系数(DTL)** | $DOL \times DFL = \frac{M}{EBIT-I}$ | 销量变动→EPS变动的总放大倍数 |

- M = 边际贡献 = 收入 − 变动成本
- F = 固定成本，I = 利息费用

## 第9章 长期筹资

### 主要筹资方式比较

| 方式 | 优点 | 缺点 |
|------|------|------|
| 普通股 | 无固定股利负担、无到期日、增强信誉 | 资本成本高、分散控制权、信息披露 |
| 优先股 | 无表决权不稀释控制、固定股利可税前扣除（部分） | 资本成本高于债务、股利不可抵税 |
| 长期借款 | 成本低（利息抵税）、速度快 | 限制条件多、财务风险大 |
| 公司债券 | 成本低于股权、不稀释控制权 | 财务风险大、限制条件 |

### 可转换债券

- 转换比率 = 债券面值 ÷ 转换价格
- 价值下限 = Max(纯债券价值, 转换价值)
- 实质：债券 + 看涨期权（认股权证）

### 认股权证

- 实质：长期看涨期权
- 筹资时附带发行，降低债券利率
- 行权时公司增发新股（与普通期权的区别）

### 租赁决策

| 类型 | 经营租赁 | 融资租赁 |
|------|----------|----------|
| 期限 | 短期 | 接近资产寿命 |
| 维护 | 出租人 | 承租人 |
| 期满 | 退还 | 留购/续租/退还 |
| 实质 | 租用 | 分期购买 |

**融资租赁决策**：比较"借款购买"与"租赁"的现金流现值，选择成本低者

**租赁净现值（NAL）**：
$$NAL = \text{租赁相对于借款购买的净现值}$$
$$= \text{避免的资产购置支出} - \text{租赁付款现值} - \text{丧失的折旧抵税现值} + \text{避免的维护费现值}$$

- NAL > 0：租赁有利
- NAL < 0：借款购买有利
- 折现率 = 有担保借款的税后利率（因为租赁付款的风险类似于有担保债务）

### 优先股资本成本

$$r_p = \frac{D_p}{P_0(1-f)}$$

- $D_p$：优先股每年股利
- $P_0$：优先股发行价格
- $f$：发行费用率
- 优先股股利不可税前扣除，无需税后调整

## 第10章 股利分配

### 股利理论

| 理论 | 核心观点 |
|------|----------|
| **股利无关论（MM）** | 在完美市场中，股利政策不影响公司价值 |
| **税差理论** | 资本利得税率<股利所得税率，少发股利有利 |
| **信号理论** | 股利传递公司前景信息，增加股利=利好信号 |
| **代理理论** | 股利减少管理层可支配现金，降低代理成本 |

### 四种股利政策

| 政策 | 特点 | 适用 |
|------|------|------|
| 剩余股利政策 | 先满足投资需要，剩余利润分配 | 初创期/高增长期 |
| 固定股利政策 | 每股股利固定不变 | 成熟稳定企业 |
| 固定股利支付率 | 按固定比例分配利润 | 较少采用 |
| 低正常股利加额外股利 | 维持低水平+好年景额外发放 | 盈利波动大的企业 |

### 股票分割与回购

- **股票分割**：增加股数、降低每股价格，不改变股东权益总额
- **股票回购**：减少流通股数，提高EPS和股价；可替代现金股利（税收优势）

## 第11章 营运资本管理

### 营运资本策略

| 策略 | 流动资产水平 | 短期融资比例 | 风险 | 收益 |
|------|-------------|-------------|------|------|
| 激进型 | 低 | 高 | 高 | 高 |
| 保守型 | 高 | 低 | 低 | 低 |
| 适中型 | 中 | 中 | 中 | 中 |

**易变现率** = (股东权益 + 长期债务 + 自发性流动负债 − 长期资产) ÷ 经营流动资产
- 易变现率 > 1：保守型
- 易变现率 < 1：激进型

### 现金管理模型

**鲍莫尔模型**（确定性模型）：

$$C^* = \sqrt{\frac{2TF}{K}}$$

- C*：最佳现金持有量
- T：一定期间现金总需求量
- F：每次转换固定成本
- K：持有现金的机会成本率

**米勒-奥尔模型**（随机模型）：

$$R = \sqrt[3]{\frac{3b\sigma^2}{4i}} + L$$

- R：回归线（目标现金余额）
- H = 3R − 2L（上限）
- L：下限（管理层设定）
- b：每次转换固定成本
- σ²：每日现金流量方差
- i：日利率

### 应收账款信用政策

**信用政策五要素**：信用标准、信用条件（信用期限、折扣期限、现金折扣）、收账政策

**信用决策**：比较放宽信用的边际收益与边际成本

- 边际收益 = 增加的边际贡献
- 边际成本 = 增加的应收账款机会成本 + 坏账损失 + 收账费用 + 现金折扣成本

$$\text{应收账款机会成本} = \frac{\text{年赊销额}}{360} \times \text{平均收现期} \times \text{变动成本率} \times \text{资本成本}$$

### 存货管理——经济订货量(EOQ)模型

$$Q^* = \sqrt{\frac{2KD}{K_c}}$$

- Q*：经济订货批量
- K：每次订货固定成本
- D：年需求量
- $K_c$：单位存货年储存成本

**相关总成本**：$TC = \sqrt{2KDK_c}$

**最佳订货次数**：$N = D/Q^*$

**再订货点**：$R = L \times d$（L=交货时间，d=日需求量）

**考虑保险储备**：$R = L \times d + B$（B=保险储备量）

### 短期债务管理

**商业信用成本**：

放弃现金折扣的信用成本（年化）：
$$\text{放弃折扣成本} = \frac{\text{折扣百分比}}{1 - \text{折扣百分比}} \times \frac{365}{\text{信用期} - \text{折扣期}}$$

例：条件"2/10, n/30"（10天内付款享2%折扣，30天为信用期）：
$$= \frac{2\%}{1-2\%} \times \frac{365}{30-10} = 37.24\%$$

**决策规则**：如果短期借款利率 < 放弃折扣成本 → 应借款享受折扣

**短期借款实际利率**：

| 方式 | 实际利率 |
|------|----------|
| 收款法（到期付息） | 名义利率 |
| 贴现法（预扣利息） | 名义利率 ÷ (1 − 名义利率) |
| 补偿性余额 | 名义利率 ÷ (1 − 补偿性余额比例) |

### 资本结构决策方法

**每股收益无差别点法（EBIT-EPS分析）**：

$$\frac{(EBIT^* - I_1)(1-T) - D_1}{N_1} = \frac{(EBIT^* - I_2)(1-T) - D_2}{N_2}$$

- $EBIT^*$：每股收益无差别点的息税前利润
- 当预期EBIT > $EBIT^*$时，选择负债筹资方案（财务杠杆有利）
- 当预期EBIT < $EBIT^*$时，选择股权筹资方案
