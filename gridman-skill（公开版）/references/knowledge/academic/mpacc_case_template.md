## 一、三件套整体规范（评奖底线）

MPAcc 案例**不是**论文，**不是**分析报告，**不是**咨询报告。它是**教学产品**。三件套各有职能，缺一不可。

| 文件 | 给谁看 | 核心职能 | 篇幅 |
|---|---|---|---|
| 案例封面 | 评审 + 教学组织者 | 案例元数据、教学目标、关键词索引 | 1-2 页 |
| 案例正文 | 学生 | 故事 + 数据 + 启发思考题（**不给答案**） | 8000-15000 字 |
| 案例说明书 | 教师 | 答案要点 + 理论依据 + 教学组织建议 | 5000-12000 字 |

**铁律（违反即不合规）：**
1. 正文只提问题，绝不给答案——答案在说明书里
2. 说明书的题目必须**严格复刻**正文的「启发思考题」顺序与文字
3. 每道题答案 = `出题目的` + `理论依据` + `案例分析` 三段式不可少
4. 三件套都不得脱离案例真实数据，编造数据等于自毁
5. 中英文摘要必须平行（不是机翻，专业术语对应）

## 二、封面规范（评奖第一关）

封面是一份**结构化合同**，字段顺序固定，缺一即扣分。

### 标准字段顺序（严格复刻）

```
# 案例名称：[修辞标题：核心冲突 + 主体]

专业领域/方向：[管理会计 / 财务会计 / 财务管理 / 审计 / 税务 / 商业伦理与会计职业道德]

适用课程：《[课程名]》[第X章 内容定位]

选用课程：《[课程名]》

编写目的：[3-5 段，说明案例要解决的教学痛点 + 4 个核心教学目标]

知识点：[3-5 个，逗号分隔]

关 键 词：[3-5 个，分号分隔]   ← 注意"关 键 词"中间有空格（行业惯例）

案例摘要：[300-500 字，覆盖背景 + 主角 + 核心冲突 + 解决路径 + 教学价值]

英文摘要：[英译案例摘要，全文必备]
```

### 字段写作要点

- **专业领域** 决定后面套哪个理论库：
  - 管理会计 → 阿米巴/平衡计分卡/预算/CLV/ABC
  - 财务会计 → CAS 准则、收入确认、所得税、合并报表
  - 财务管理 → DCF/LBO/分拆/资本运作/估值
  - 审计 → 风险评估/重要性/函证/截止/KAM
  - 商业伦理 → ESG/合法性/三重底线/利益相关者
- **编写目的** 必须含 4 个排比点（"理解 XX / 掌握 XX / 探讨 XX / 思考 XX"），与教指委课程大纲建立逻辑联系
- **知识点** 是给评委看的索引；**关键词** 是给检索用的标签；二者重合度约 70%
- **中英文摘要严格平行**——英文不是机翻，专业术语要用学界惯用词（例如「财务洗澡」= Big Bath，「关键审计事项」= Key Audit Matters / KAM）

### 封面禁忌
- ✗ 缺英文摘要（非选项，必须有）
- ✗ 编写目的写成"本案例旨在分析..."（要写成给学员的承诺式：「帮助学员……」「引导学员……」「使学员……」）
- ✗ 摘要里出现"我们""作者"等第一人称
- ✗ 摘要超过 500 字（评审一眼扫不完）

## 三、标题修辞规则（决定第一印象）

5 篇获奖案例标题修辞共 5 种套路，**任选一种或叠加**：

### 套路 1：双关字 / 谐音
- "经营有**'稻'**：金源环宇智破阿米巴经营下的会计难题"（稻盛和夫的"稻"）
- "收钱不认收入：五粮液 2025 年报'**涂改液**'..."（五粮液谐音）

### 套路 2：古诗化用
- "**树木丛生，百草丰茂**——海尔智家 ESG 的成长之路"（曹操《观沧海》）
- "**老树发新芽**——客户关系会计引领澳度智能家居重生"

### 套路 3：数字冲击
- "**3 年连拆 5，剑指 12 家**，微创医疗的'分拆大业'是战略融资还是变相圈钱？"

### 套路 4：设问引争议
- "...是**战略融资还是变相圈钱**？"
- "...是**创新还是套路**？"

### 套路 5：双标题（**最常用、最稳**）

```
[文学修辞主标题]：[企业名 + 核心议题/会计审计动作 + 学科背景]
```

示例：
- "收钱不认收入：五粮液 2025 年报'监管商品'模式下的收入确认口径切换与财务洗澡识别"
- "W 公司财务体系赋能科技创新效果研究——基于实地调研的案例分析"

### 标题禁忌
- ✗ 纯学术标题（"基于 XX 理论的 XX 研究"）
- ✗ 报告语气（"关于 XX 公司 XX 问题的分析"）
- ✗ 标题里没有企业名

## 四、正文写作公式

### 整体结构（5 篇平均得出）

```
1. 引言（设疑导入）        ── 抛反常现象/数据矛盾，500-1000 字
2. 案例背景               ── 行业、政策、公司、股权、组织
3. 案例情节（主体）        ── 按时间或逻辑展开 3-6 个情节段
4. 关键转折/决策点         ── 每段都给数据表、流程图、附录
5. 尾声/未解之问           ── 留悬念，不下结论
6. 启发思考题             ── 4-7 题（5 题最稳）
7. 参考文献               ── 中英文文献 8-30 篇
8. 附录（关键）           ── 6-13 个附件
```

### 引言写作公式（不写学术腔）

✗ **错误范式**：「本案例研究...本文以...为对象...」

✓ **正确范式 1（金源环宇）**：新闻特写式
> 2021 年 4 月 12 日下午 5 点，3 月份的月度经营分析会和第一季度经营分析会已经进行到了尾声，金源环宇董事长崔征手上拿着 2021 年第一季度的财务报表，看着部门负责人的汇报，不禁回想起以往的第一季度......

✓ **正确范式 2（五粮液）**：矛盾陈述式
> 2026 年 4 月 30 日晚间，五粮液 2025 年度报告挂上巨潮资讯网。报告的封面有一个细节耐人寻味：法定代表人一栏签署的是"华涛（代）"——一个"代"字，暗示着公司治理层正经历非常时期......

✓ **正确范式 3（微创医疗）**：直接点冲突
> 在一众公司当中，存在一位分拆上市的"资深玩家"：微创医疗。从孵化到拆分上市，从产品研发到资本运筹，短短几年内，微创医疗已经成功拆分旗下五家子公司上市......

### 数据呈现规则

- **每 1500 字至少配 1 张表/图**——5 篇案例平均 8-15 个表/图
- 表标号格式：「表 X-Y」（章节-序号，**跨章节连续**）
- 图标号格式：「图 X-Y」（同上）
- **所有表/图都要在正文中被点名引用**（"如表 3-1 所示"，"详见图 4-2"）
- 数据来源必须**脚注**或表/图下方备注

### 启发思考题设计规则（评奖核心）

**数量**：4-7 题，**5 题最稳**。

**梯度**：从「描述」→「分析」→「评价」→「创造」（布卢姆分类六层取四层）。

**结构**：每题 1-2 个问号；**最后 1 题必为开放性**（无标准答案，启发批判思维）。

✓ **5 题标准梯度模板**（套用任何案例）：

```
1. 描述层：[企业] 是一家什么样的企业？面临什么困境？
2. 解释层：[企业] 为什么要采取 [措施]？背后理论是什么？
3. 应用层：在 [理论] 框架下 [企业] 是如何做的？
4. 分析层：[企业] 应用了哪些 [工具]？效果如何？请用 [指标] 量化分析。
5. 评价/创造层（开放）：你认为 [冲突点 A] 还是 [冲突点 B]？谈谈你的看法。
```

### 附录设计原则（隐形评分项）

**附录是案例从"故事"升级为"教案"的关键**。学生分组讨论时人手一份附录数据，才能真做分析。**没有附录的案例，等于没有教学功能**。

**必备附录类型**（按案例性质选 6-13 个）：

| 案例类型 | 必备附录 |
|---|---|
| 管理会计 | 部门职责表、KPI 指标体系全表、月度会流程、财务数据明细 |
| 财务会计 | 三大报表附注摘录、关键科目明细、会计政策说明 |
| 财务管理 | 行业对比表、估值模型参数、年报关键数据、股权结构图 |
| 审计 | 审计程序清单、底稿模板、风险评估矩阵、KAM 摘录 |
| ESG/合规 | 披露框架对照、利益相关方矩阵、监管文件原文 |
| 投行/分拆 | 招股书摘录、各期 IPO 节奏表、事件研究 CAR 表 |

附录尽量**复刻真实表格格式**，连边框、字号都接近原企业内部文件——这就是"真实感"评分的来源。

## 五、说明书写作公式（教师手册）

### 整体结构（七章式，严格按序）

```
一、本案例要解决的关键问题（教学目标）
   1. 适用课程
   2. 适用对象
   3. 教学目的（含 3-4 个排比目标）

二、案例讨论的准备工作
   （一）启发思考题（复刻正文）
   （二）相关理论
   （三）行业背景
   （四）制度背景

三、案例分析要点
   （一）需要学员识别的关键问题（思考题清单 + 答案三段式）
   （二）解决问题的可供选择方案及其评价
   （三）推荐解决问题的方案及具体措施

四、教学组织方式
   （一）问题清单及提问顺序
   （二）课时分配
   （三）讨论方式
   （四）课堂讨论总结要点

五、关键要点（案例要点 + 知识点 + 课程思政）

六、建议课堂计划（含板书设计）

七、参考文献
```

### 每道思考题的答案三段式（核心格式）

正文里的 5 道题，说明书每题必须**严格按三段式作答**：

```markdown
【启发思考题】X. [严格复刻正文原题]

【出题目的】[1-3 句，说明这道题考察什么能力/知识点]

【理论依据】[逐条列出本题用到的理论，每个理论：定义 + 提出者 + 文献]
（1）XX 理论：[定义]（[作者], [年份]）
（2）XX 模型：[公式 + 计算示例]

【案例分析】[严格依据正文数据 + 理论框架展开，分点论述]
1. [分析维度 1]
2. [分析维度 2]
...
```

### 教学组织建议（评奖加分项）

**必含 5 个要素**：

#### 1. 课时分配（精确到分钟，总时长 90-120 分钟）

```
课堂前言：       5 分钟    教师介绍案例主题
案例回顾：       5-10 分钟  随机提问回顾要点
案例分析与讨论：  60-90 分钟 每题 10-15 分钟
案例总结：       10-15 分钟 教师归纳 + 课程思政
```

#### 2. 讨论方式

- **小组讨论**（最常用）：4-6 人一组，每组讨论后选代表发言
- **角色扮演 / 三方对抗**（适合争议性案例）：质疑方 + 公司方 + 鉴证方
- **翻转课堂**（课前预习 + 课堂对抗）：发预习单 + 课前完成关键比率计算

#### 3. 课堂提问逻辑

按思考题顺序，**每题给 2-3 个追问点**（让讨论能展开）。

#### 4. 课程思政（教指委硬性要求，2020 年起）

把案例与下列主题之一挂钩：
- "绿水青山就是金山银山"（适用 ESG / 双碳 / 环保）
- "共同富裕"（适用收入分配 / 社会责任）
- "科技自立自强"（适用研发 / 创新 / 高端制造）
- "中国式现代化"（适用国企改革 / 制度建设）
- "职业道德底线"（适用审计 / 财务造假 / 合规）

每个案例至少嵌入 **1-2 个**思政点。

#### 5. 板书设计

流程图、关系图，**画在说明书里**（用 mermaid 或 ASCII 画都可）。

## 六、理论库（按专业领域分类，写新案例直接挑用）

### 财务会计类
- **会计假设与信息质量要求**：CAS 总则
- **资产确认条件**：CAS 6（含合并报表与个别报表的客户关系资产差异）
- **CAS 14 收入确认五步法 + 控制权转移模型**
- **应付客户对价 / 可变对价**：2017 版收入准则
- **递延所得税**：CAS 18（暂时性差异、内部交易未实现利润）
- **金融工具三分类**：CAS 22（SPPI 测试 + 业务模式）
- **预期信用损失（ECL）三阶段模型**
- **政府补助会计**：CAS 16（总额法 vs 净额法）

### 管理会计类
- **目标管理理论**（德鲁克）
- **资源配置理论**
- **业财融合理论**
- **平衡计分卡**（卡普兰，4 维度：财务/客户/内部流程/学习成长）
- **客户终身价值（CLV）模型**：保留模型 / 迁移模型
- **客户利润贡献分析法**（含 ABC 分层）
- **阿米巴经营 / 稻盛哲学**：会计七原则 + 经营十二条
- **预算管理**（全面预算、滚动预测、权变管理）
- **人才激励理论 / 股权激励**
- **MECE / 议题树**

### 财务管理类
- **融资需求理论**（Allen & McConnell, 1998）
- **核心化战略理论**（Aron, 1991）
- **管理层激励理论**（Baker, 1998）
- **超额收益理论**（Schipper & Smith, 1986; Nanda, 1991）
- **大股东侵占理论 / 空心化风险**
- **DCF / LBO / 可比公司估值**
- **事件研究法**（CAR 计算公式见下文）
- **杜邦分析体系**

### ESG 与商业伦理类
- **ESG 内涵**（联合国全球契约 2004）
- **合法性理论**（韦伯）
- **声誉理论**（Grund, 1996）
- **三重底线理论**（Elkington, 1997：经济/环境/社会）
- **利益相关者理论**（Freeman, 1984）
- **可持续发展理论**（《我们共同的未来》, 1987）

### 财务报表分析类
- **四大能力分析**（盈利/偿债/营运/成长）
- **现金流与利润勾稽 / 收现比**
- **财务洗澡 vs 财务造假辨析框架**（核心见下文）
- **关键审计事项（KAM）解读**
- **季度/月度数据畸形识别**

### 审计与监管类
- **CSA 1141 舞弊准则**
- **CSA 1301 审计证据 / 1313 重要性 / 1320 重要性水平**
- **关联方审计 / 截止测试 / 函证**
- **会计监管风险提示第 1-9 号**

### 公司治理与制度环境类
- **地方国资双重剩余索取人理论**（既是所有者又是征税者）
- **官员任期与会计选择的时间错配**（"新官不理旧账"逻辑同构）
- **借周期出清的合理化外衣**

## 七、关键工具公式（写案例直接套）

### 工具 1：财务洗澡 vs 财务造假辨析框架

| 维度 | 财务造假 | 财务洗澡 |
|---|---|---|
| 合规性 | 违法违规 | 形式合规 |
| 操纵方向 | 通常虚增 | 通常虚减 |
| 动机 | 维持股价、满足融资 | 甩包袱、做低基数 |
| 审计意见 | 事后可能追溯为非标 | 通常标准无保留 |
| **现金流特征**（第一性判据） | 利润高但现金流差 | **利润低但现金流好** |
| 持续性 | 需持续造假维持 | 一次性释放后自然恢复 |

### 工具 2：财务洗澡红旗信号体系（11 项 + 阈值）

| # | 红旗信号 | 阈值 |
|---|---|---|
| 1 | 收入跌幅 / 行业跌幅 | > 3 倍为红色 |
| 2 | 收现比（经营现金流 ÷ 净利润） | > 3.0 为红色 |
| 3 | 销售收现 ÷ 营业收入 | > 2.0 为异常 |
| 4 | 合同负债增 + 收入降 | 合同负债 +>10% 且 收入 −>40% |
| 5 | 收入口径变更但不追溯 | 收入降 >30% 未重述 |
| 6 | 新增大额过渡科目 | 占总资产 >5% 为红色 |
| 7 | 季度利润分布畸形 | 最高/最低 > 10 倍 |
| 8 | 内部未实现利润变动 | 同比增 >300% 为红色 |
| 9 | 治理信号：高管被查 | 定性触发 |
| 10 | 治理信号：管理层更换 | 定性触发 |
| 11 | 措辞信号 | "基于谨慎性原则调整"等 |

### 工具 3：事件研究法 CAR 计算（用于分拆/并购等案例）

**个股收益率**：
$$R_{it} = \frac{P_{it}}{P_{it-1}} - 1$$

**市场收益率**：
$$R_{mt} = \frac{MP_{it}}{MP_{it-1}} - 1$$

**标准市场模型**（用估计期数据回归）：
$$R_{it} = \alpha_i R_{mt} + \beta_i$$

**超额收益率**：
$$AR_{it} = R_{it} - (\alpha_i R_{mt} + \beta_i)$$

**累计超额收益率**：
$$CAR_{it} = \sum AR_{it}$$

**窗口期建议**：[-1,1] 短窗口 / [-3,3] 中窗口 / [-10,10] 长窗口
**估计期建议**：[-130,-11]（120 个交易日）

### 工具 4：客户终身价值（CLV）保留模型

$$CLV_i = \sum_{t=0}^{\infty} \frac{(p_{it} - c_{it}) r_{it}}{(1+d)^t} - AC_i$$

其中：
- $p_{it}$ = 客户 i 在 t 期支付价格
- $c_{it}$ = 服务客户 i 的直接成本
- $r_{it}$ = 客户 i 在 t 期保留率
- $d$ = 资本成本/折现率
- $AC_i$ = 客户获取成本

### 工具 5：财务洗澡四组勾稽（递减瀑布）

真洗澡的典型指纹是「**账面收入跌幅 > 实物销量跌幅 > 流转税跌幅 > 实缴税费跌幅**」的递减瀑布——因为流转税按真实出货计征，跟着真经营走，跌不下来。

## 九、撰写新 MPAcc 案例的标准流程

```
Step 1: 选题（真实事件 + 教学价值）
   ↓
Step 2: 确定专业领域 + 课程定位 + 套用哪些理论
   ↓
Step 3: 写双标题（修辞 + 冲突）
   ↓
Step 4: 收集真实数据（年报 / 公告 / 监管文件 / 访谈）
   ↓
Step 5: 设计 5 道思考题（梯度 + 最后 1 题开放）
   ↓
Step 6: 写正文（引言设疑 → 背景 → 情节 → 尾声 → 思考题）
   ↓
Step 7: 设计 6-13 个附录（学生可独立分析）
   ↓
Step 8: 写说明书（思考题三段式 + 教学组织 + 课程思政）
   ↓
Step 9: 写封面（最后写，因为字段需提炼前两件套精华）
   ↓
Step 10: 三件套交叉校对（题目对应 / 数据一致 / 中英文摘要平行）
```

**为什么封面最后写？** 因为封面要凝练正文+说明书的核心。先写完两件套，封面就是"摘要+索引"，半小时搞定。反过来先写封面，常常被自己挖的坑卡住。

```
【出题目的】考察学员对 CAS 14 收入准则核心规则（控制权转移五项迹象）

（2）控制权转移五项迹象（收款权利、法定所有权、实物转移、风险报酬、

3. 不追溯的可比性损害——2024 旧口径数据未重述
4. 披露的充分性——附注披露简略，难以还原"沿用旧口径下的收入"
```

- 知识目标：掌握 CAS 14 + 财务洗澡识别框架 + 递延所得税
