## 第一章 房地产开发企业

> 注意区分：本章是**房地产开发企业**（建房卖房，开发产品是存货）的会计。
> 持有出租/增值的"投资性房地产"是另一回事，详见：accounting_core.md - 第五章 投资性房地产。

### 1.1 行业特殊性总览

> 类型：判断表
> 来源：行业实务
> 置信度：实务经验

房地产开发区别于一般工商企业的四个核心特征，决定了它的会计和税务都特殊：

| 特征         | 含义                                             | 引出的会计/税务难点                  |
| ------------ | ------------------------------------------------ | ------------------------------------ |
| 开发周期长   | 拿地到交付通常 2-4 年，跨多个会计期间            | 借款费用资本化、成本跨期归集         |
| 预售制度     | 未完工先收款（预售款）                           | 收入确认时点、合同负债、三项税金预缴 |
| 成本归集复杂 | 一个项目多业态、分期开发、共同成本需分摊         | 成本对象划分、共同成本分摊方法       |
| 税负重且特殊 | 土地增值税是房开独有、增值税预缴、所得税预计毛利 | 三税联动、清算时点判断               |

### 1.2 开发成本归集与分摊

> 类型：规则
> 来源：企业会计准则 + 房开企业会计核算实务
> 置信度：实务经验

房地产开发成本通过"开发成本"科目归集，完工后结转"开发产品"（存货），销售时结转"主营业务成本"。

**六大成本项目：**

| 成本项               | 内容                                               |
| -------------------- | -------------------------------------------------- |
| 土地征用及拆迁补偿费 | 土地出让金、契税、拆迁补偿、市政配套费             |
| 前期工程费           | 规划设计、勘察、可行性研究、三通一平               |
| 建筑安装工程费       | 建安合同造价（最大头）                             |
| 基础设施费           | 小区道路、供水供电、绿化、管网                     |
| 公共配套设施费       | 不可有偿转让的会所、物业用房、变电站               |
| 开发间接费           | 项目部直接组织管理发生的费用（工资、周转房摊销等） |

**成本对象划分**——成本归集的第一步，应当在开工前确定。划分标准（按可售性 + 功能 + 开发时间）：

- 能单独立项、单独核算、单独销售的 → 独立成本对象（如分期项目的每一期、不同业态的住宅/商业/车位）

**共同成本分摊方法**（一个成本对象内多业态，或多个成本对象共担的成本）：

| 分摊对象               | 应当采用的分摊方法                                         |
| ---------------------- | ---------------------------------------------------------- |
| 土地成本               | 占地面积法（按各业态占地面积比例）                         |
| 单独作为成本对象的配套 | 建筑面积法                                                 |
| 借款费用               | 直接归属的直接计入，无法直接归属的按建筑面积法或预算造价法 |
| 其他共同成本           | 建筑面积法（最常用）                                       |

⚠️ 例外：占地相对独立的，土地成本可直接归集到对应成本对象，不参与分摊。

**会计分录——开发成本归集与结转：**

发生开发支出：

```
借：开发成本——XX项目——土地征用费/前期工程费/建安费/...    XX
  贷：银行存款 / 应付账款 / 应付职工薪酬                    XX
```

项目完工，结转开发产品：

```
借：开发产品——XX项目                                      XX
  贷：开发成本——XX项目                                    XX
```

销售并满足收入确认条件，结转成本：

```
借：主营业务成本                                          XX
  贷：开发产品——XX项目                                    XX
```

### 1.3 预售与收入确认

> 类型：规则
> 来源：CAS 14（2017）+ 房开实务
> 关联：accounting_core.md - 第十七章 收入确认五步法（通用规则主存放）
> 置信度：准则原文

[场景：房地产开发企业销售商品房]

**核心判断：时段确认 还是 时点确认？** 按 CAS 14 三条件判断是否"在某一时段内履行履约义务"。中国大陆商品房销售**绝大多数为时点确认**——因为商品房通常不满足"履约过程中所产出商品具有不可替代用途"或"有权就累计已完成部分收取款项"的时段条件。

⚠️ 关键结论：**收到预售款不确认收入**。预售款是合同负债，要到控制权转移（通常是交房、办理交付手续）时点才确认收入。

**判断顺序：**

| 判断顺序    | 情形                                                            | 收入确认                 |
| ----------- | --------------------------------------------------------------- | ------------------------ |
| 1（排除项） | 仅签合同、收预售款，房屋未交付                                  | 不确认收入，挂"合同负债" |
| 2           | 房屋竣工验收 + 交付业主（控制权转移）                           | 时点确认收入             |
| 3（特殊）   | 定制化开发、客户主导设计且不可移作他用 + 有权收取已完工部分款项 | 可能时段确认（罕见）     |

**会计分录——预售到确认收入全流程：**

预售收款时，全额挂合同负债（未到纳税义务发生时点，先不做价税分离）：

```
借：银行存款                              1,090,000
  贷：合同负债                            1,090,000
```

按 3% 预征率预缴增值税（预缴额 = 1,090,000 ÷ (1+9%) × 3% = 30,000）：

```
借：应交税费——预交增值税                   30,000
  贷：银行存款                              30,000
```

交房、确认收入（此时价税分离，确认销项税额）：

```
借：合同负债                              1,090,000
  贷：主营业务收入                          1,000,000
      应交税费——应交增值税（销项税额）         90,000
```

纳税义务发生后，预交增值税结转抵减应交增值税：

```
借：应交税费——未交增值税                   30,000
  贷：应交税费——预交增值税                   30,000
```

⚠️ 易错点：预售时**不要**按 9% 做价税分离挂销项（90,000）——预售未到纳税义务发生时点，只按 3% 预征率预缴（30,000），全额挂合同负债。价税分离与销项确认要等交房时点。

> 预缴增值税的完整规则（一般计税/简易计税/土地价款抵减）详见 1.5。

### 1.4 借款费用资本化（房开特殊）

> 类型：规则
> 来源：CAS 17
> 关联：accounting_core.md - 第十一章 借款费用（通用规则主存放）
> 置信度：准则原文

房地产开发产品属于"经过相当长时间达到可销售状态的存货"，**应当**将符合条件的借款费用资本化计入开发成本。

- 资本化期间：开发活动开始 至 达到可销售状态（通常竣工备案）
- 非正常中断超过 3 个月 → 暂停资本化
- 达到可销售状态后发生的借款费用 → 费用化计入财务费用

⚠️ 易错点：项目分期开发时，先完工的一期已达可销售状态后，对应借款费用应停止资本化，不能继续往后续期的成本里塞。

### 1.5 增值税处理（预缴 + 一般/简易计税）

> 类型：规则
> 来源：财税〔2016〕36号 + 国家税务总局公告2016年第18号
> 置信度：准则原文

[场景：房地产开发企业销售自行开发的房地产项目]

**计税方法选择：**

| 项目类型                               | 计税方法     | 税率/征收率           |
| -------------------------------------- | ------------ | --------------------- |
| 一般纳税人 + 一般项目                  | 一般计税     | 销项 9%，可抵进项     |
| 一般纳税人 + 老项目（2016.4.30前开工） | 可选简易计税 | 5% 征收率，不可抵进项 |
| 小规模纳税人                           | 简易计税     | 5% 征收率             |

**预缴：** 采取预收款方式销售的，收到预收款时**应当**按 **3% 预征率**预缴增值税。

预缴增值税 = 预收款 ÷ (1 + 9%或5%) × 3%

**一般计税的土地价款抵减：** 一般计税方法下，销售额 = (全部价款 − 当期允许扣除的土地价款) ÷ (1 + 9%)。土地价款可以从销售额中扣除——这是房开增值税的独有规则。

**会计分录——预缴增值税：**

```
借：应交税费——预交增值税              XX
  贷：银行存款                        XX
```

纳税义务发生（交房）时，预交增值税结转：

```
借：应交税费——未交增值税              XX
  贷：应交税费——预交增值税              XX
```

### 1.6 土地增值税（房开独有，预征 + 清算）

> 类型：规则
> 来源：《土地增值税暂行条例》及实施细则
> 置信度：准则原文

土地增值税是房地产开发**独有**的税种，按增值额（转让收入 − 扣除项目）四级超率累进征收。

**四级超率累进税率：**

| 增值额占扣除项目比例 | 税率 | 速算扣除系数 |
| -------------------- | ---- | ------------ |
| ≤ 50%               | 30%  | 0%           |
| 50%（不含）~ 100%    | 40%  | 5%           |
| 100%（不含）~ 200%   | 50%  | 15%          |
| > 200%               | 60%  | 35%          |

土地增值税 = 增值额 × 适用税率 − 扣除项目金额 × 速算扣除系数

**扣除项目（五项）：**

1. 取得土地使用权所支付的金额
2. 开发成本（土地征用、前期、建安、基础设施、配套、开发间接费）
3. 开发费用（管理+销售+财务费用，按规定比例计算扣除）
4. 与转让相关的税金
5. **财政部规定的加计扣除**：房开企业可就"土地成本 + 开发成本"**加计 20% 扣除**（这是房开独有的优惠，鼓励开发）

**预征与清算：**

- **预征**：项目未达清算条件前，按预征率（各省不同，通常 2%-3.5%）预缴
- **清算**：达到清算条件时，重新按四级累进算出应纳税额，多退少补

**应当清算的情形（强制清算）：**

- 已竣工验收的项目，已转让面积占可售面积 ≥ 85%
- 已转让面积虽未达 85%，但剩余可售面积已出租或自用
- 取得销售（预售）许可证满 3 年仍未销售完毕

**会计分录——土地增值税：**

预征（计入税金及附加）：

```
借：税金及附加                        XX
  贷：应交税费——应交土地增值税          XX
借：应交税费——应交土地增值税          XX
  贷：银行存款                        XX
```

⚠️ 清算补缴跨年时，若收入已在以前年度确认，补缴的土增税通过"以前年度损益调整"或当期"税金及附加"处理，需结合具体情况判断。

### 1.7 企业所得税（预计计税毛利率）

> 类型：规则
> 来源：国家税务总局令第31号（房开企业所得税处理办法）
> 置信度：准则原文

[场景：房开企业预售阶段的企业所得税]

预售收入虽未达会计收入确认条件，但税法要求**按预计计税毛利率先行计算应纳税所得额**预缴企业所得税——这是房开所得税的核心特殊点（税会差异）。

- 预售阶段：预售收入 × 预计计税毛利率 → 计入当期应纳税所得额
- 预计计税毛利率：由各省税务机关确定，一般不低于 15%（经济适用房等保障性住房可低至 3%）
- 完工结转收入后：按实际毛利与已计税预计毛利的差额，调整当期应纳税所得额

⚠️ 这是典型的税会差异——会计不确认收入，税务先征税。形成的差异通过纳税调整处理，不通过递延所得税（因为是时间性的预缴，最终随收入确认转回）。

### 1.8 审计与造假关注点

> 类型：判断表
> 来源：实务 + 监管案例
> 关联：financial_fraud.md（造假手法主存放）、audit.md（审计程序主存放）
> 置信度：实务经验

房开企业审计/尽调的行业特有红旗：

| 关注点          | 风险信号                                                       |
| --------------- | -------------------------------------------------------------- |
| 收入确认时点    | 提前确认（未交房先确认）或递延确认（已交房不确认）调节利润     |
| 成本归集分摊    | 在售期与未售期之间挪移成本、多分摊已售部分降低当期利润递延纳税 |
| 合同负债余额    | 预售款长期挂账不结转，可能隐瞒已交付收入                       |
| 土增税清算      | 已达清算条件不清算、少计增值额                                 |
| 关联方拿地/工程 | 高价从关联方采购建安服务转移利润                               |
| 项目公司表外    | 通过明股实债、合作开发出表隐藏负债                             |

> 房开行业财务造假的完整手法与案例详见：financial_fraud.md。

## 第二章 建筑施工企业

### 2.1 行业特殊性总览

> 类型：判断表
> 来源：行业实务
> 置信度：实务经验

建筑施工与一般制造业的核心差异，决定其会计税务特殊点：

| 特征               | 含义                     | 引出的难点                       |
| ------------------ | ------------------------ | -------------------------------- |
| 单项目周期长、跨期 | 一个工程跨多个会计期间   | 时段确认收入、履约进度估计       |
| 异地施工           | 项目所在地≠机构所在地   | 增值税跨地区预缴、所得税分摊     |
| 总分包链条         | 总包—专业分包—劳务分包 | 分包额扣除、农民工工资、连带责任 |
| 结算滞后于施工     | 干活在前、业主结算在后   | 合同资产/合同负债、质保金        |
| 资质与挂靠         | 资质决定承接资格         | 挂靠经营的收入归属与税务风险     |

### 2.2 建造合同收入确认

> 类型：规则
> 来源：CAS 14（2017）
> 关联：accounting_core.md - 第十七章 收入确认五步法（通用规则与合同结算分录主存放）
> 置信度：准则原文

[场景：建筑施工企业承接工程施工]

建筑施工**绝大多数为时段确认**——施工形成的在建工程通常满足"企业履约创造的资产由客户控制"（建在客户的土地上、客户随施工进展取得控制）。这与房地产开发（多为时点确认）正好相反，是两章最大的区别。

⚠️ 关键：满足时段条件时，**应当**按履约进度确认收入，不能等竣工结算才一次确认。

**履约进度方法（投入法为主）：**

| 方法                     | 计算                                    | 适用           |
| ------------------------ | --------------------------------------- | -------------- |
| 投入法（成本法，最常用） | 累计实际成本 ÷ 预计总成本              | 建筑施工主流   |
| 产出法                   | 已完工程量 ÷ 总工程量（形象进度/测量） | 工程量易计量时 |

⚠️ 例外：履约进度**不能合理确定**时，已发生成本预计能收回的 → 按已发生成本金额确认收入（不确认毛利），直到能合理确定为止（详见 regulatory_risk_alerts.md 收入相关问题）。

⚠️ 投入法易错点：未耗用的材料（已采购堆在工地未使用）、分包未执行部分，**不计入**已发生成本，否则高估履约进度提前确认收入（典型造假手法）。

> 完整的合同结算/合同资产/主营业务成本分录见 accounting_core.md 第十七章，本章不复制。

### 2.3 合同资产/合同负债与"已结算"列报

> 类型：判断表
> 来源：CAS 14 + 列报实务
> 置信度：准则原文

建筑业用"合同结算"科目核算累计已办理结算与累计已确认收入的差额，期末按余额方向列报：

| 判断顺序 | 情形                                               | 列报项目                 |
| -------- | -------------------------------------------------- | ------------------------ |
| 1        | 累计已确认收入 > 累计已结算（干得多、结算少）      | 合同资产（已完工未结算） |
| 2        | 累计已结算 > 累计已确认收入（结算多、干得少/预收） | 合同负债（已结算未完工） |

⚠️ 区分"合同资产"与"应收账款"：已办理结算、对方无条件应付的 → 应收账款；已确认收入但尚未办理结算（收款还需经过结算审批）→ 合同资产。

### 2.4 质保金（质量保证金）

> 类型：规则
> 来源：CAS 14 + 实务
> 置信度：实务经验

工程结算时业主按合同比例（通常 3%-5%）扣留质保金，缺陷责任期满后支付。

- 质保金**应当**全额确认收入（它是交易价格的一部分，不因暂扣而减少收入）
- 暂扣部分在符合无条件收款权前列"合同资产"，符合后转"应收账款"
- 若质保金包含重大融资成分（缺陷责任期长，如 2 年以上）→ 考虑融资成分影响

### 2.5 增值税（跨地区预缴是核心特殊点）

> 类型：规则
> 来源：财税〔2016〕36号 + 国家税务总局公告2016年第17号
> 置信度：准则原文

[场景：建筑企业提供建筑服务]

**计税方法：**

| 项目类型                              | 计税方法     | 税率/征收率 | 销售额扣除 |
| ------------------------------------- | ------------ | ----------- | ---------- |
| 一般纳税人 + 一般项目                 | 一般计税     | 9%          | —         |
| 老项目（2016.4.30前）/甲供工程/清包工 | 可选简易计税 | 3%          | 可扣分包款 |
| 小规模纳税人                          | 简易计税     | 3%          | 可扣分包款 |

**跨地区预缴（建筑业独有重点）：** 跨县市区提供建筑服务，**应当**在**建筑服务发生地**预缴增值税，向**机构所在地**申报。

预缴税款 = (全部价款 − 支付的分包款) ÷ (1 + 9%或3%) × 预征率

- 一般计税预征率 **2%**
- 简易计税预征率 **3%**

⚠️ 分包款扣除需取得合法分包发票，否则不得扣除。

**会计分录——跨地区预缴：**

```
借：应交税费——预交增值税                XX
  贷：银行存款                          XX
```

## 第三章 金融业（银行/证券/保险）

### 3.1 行业特殊性总览

> 类型：判断表
> 来源：金融企业会计制度 + CAS 22/25 + 监管规则
> 置信度：实务经验

金融业是受监管最强、会计最特殊的行业，三个子行业各不相同：

| 子行业 | 核心资产/收入 | 会计特殊点 | 监管约束 |
|--------|---------------|------------|----------|
| 银行 | 贷款、利息净收入 | 贷款减值（五级分类+ECL）、实际利率法 | 资本充足率、拨备覆盖率 |
| 证券 | 自营、经纪、资管 | 客户资金不入表、多业务线收入确认 | 净资本、风险准备金 |
| 保险 | 保费、保险合同负债 | 新保险合同准则（CSM）、准备金 | 偿付能力充足率 |

⚠️ 金融业普遍**双重报告**：会计口径（CAS）+ 监管口径（银保监/证监）。两者并存、互不替代，差异通过附注或监管报表衔接。

### 3.2 银行业

> 类型：规则
> 来源：CAS 22 + 商业银行风险监管核心指标
> 关联：accounting_core.md - 第十三章 金融工具（ECL 三阶段主存放）
> 置信度：准则原文

**贷款五级分类 vs 会计 ECL（核心特殊点）：**

银行贷款有两套并行的"风险刻度"：

| 维度 | 五级分类（监管口径） | ECL 三阶段（会计口径） |
|------|---------------------|------------------------|
| 分级 | 正常 / 关注 / 次级 / 可疑 / 损失 | 第一阶段 / 第二阶段 / 第三阶段 |
| 目的 | 监管风险监测、拨备监管要求 | 财务报表减值计量 |
| 减值 | 监管拨备（拨备覆盖率、拨贷比） | 12个月 / 整个存续期预期信用损失 |

⚠️ 两者大致对应（正常/关注→一二阶段，次级及以下→第三阶段）但不完全等同。拨备计提**应当**按会计 ECL 计量，同时满足监管最低要求（拨备覆盖率、拨贷比孰高）。

> ECL 三阶段完整模型详见：accounting_core.md - 第十三章。

**利息收入：** 按实际利率法确认。第三阶段（已减值）贷款的利息按**摊余成本净额**（账面余额−减值准备）乘实际利率计算，不再按合同本金。

**会计分录——计提贷款损失准备：**
```
借：信用减值损失                          XX
  贷：贷款损失准备                         XX
```

**关键监管指标（孰低/孰高约束）：**

| 指标 | 监管要求 | 含义 |
|------|----------|------|
| 资本充足率 | 核心一级 ≥ 7.5%、一级 ≥ 8.5%、总 ≥ 10.5% | 资本对风险加权资产的覆盖 |
| 拨备覆盖率 | ≥ 120%-150% | 贷款损失准备 ÷ 不良贷款 |
| 拨贷比 | ≥ 1.5%-2.5% | 贷款损失准备 ÷ 贷款总额 |

> 银行经营分析与估值（净息差、不良生成率）详见：equity_research.md - 银行。

### 3.3 证券业

> 类型：判断表
> 来源：证券公司会计核算 + CAS 14
> 置信度：实务经验

**客户资金不入表（核心特殊点）：** 客户交易结算资金实行**第三方存管**，属于客户所有，证券公司**不确认**为自身资产和负债（只在表外核算或作为代理负债），仅自有资金入表。

⚠️ 这是证券业最易错点：千万不要把客户保证金当成公司的货币资金。

**各业务线收入确认：**

| 业务 | 收入 | 确认 |
|------|------|------|
| 经纪 | 佣金、手续费 | 交易完成时点确认 |
| 自营 | 投资收益、公允价值变动 | 按金融工具准则 |
| 资管 | 管理费、业绩报酬 | 管理费时段确认；业绩报酬待可变对价不确定性消除时确认 |
| 两融/股票质押 | 利息净收入 | 实际利率法时段确认 |
| 投行 | 承销、保荐费 | 履约义务完成（发行成功）时点确认 |

**风险准备金/净资本：** 按规定计提一般风险准备、交易风险准备；受净资本监管约束（净资本/各项风险资本准备 ≥ 100% 等）。

### 3.4 保险业

> 类型：规则
> 来源：CAS 25（2020 新保险合同准则，2026 全面施行）
> 关联：cas_application_cases.md - 浮动收费法
> 置信度：准则原文

**新保险合同准则（CSM 核心特殊点）：** 2020 修订的 CAS 25 已全面施行，颠覆了"收到保费即确认收入"的旧做法。

- 保险合同负债 = **履约现金流**（未来现金流现值 + 非金融风险调整）+ **合同服务边际（CSM）**
- 合同服务边际（CSM）代表未赚取的利润，**应当**在保险责任期间内**逐期摊销**确认为保险服务收入——而不是收到保费时一次确认
- 亏损合同的亏损立即计入当期损益（不递延）

⚠️ 与旧准则最大区别：保费收入不再是"保费流入"，而是按提供保险服务的进度逐期确认。投资成分（如分红险中的储蓄部分）要从保险收入中**分拆剔除**。

**主要准备金（旧口径仍用于理解）：**
- 未到期责任准备金：已收保费中对应未来保险期间的部分
- 未决赔款准备金：已发生但未结案赔案的预估

**分保（再保险）：** 分出业务确认"分出保费"、"摊回赔付/费用"；分入业务确认"分保费收入"。

**偿付能力充足率（监管约束）：** 核心偿付能力充足率、综合偿付能力充足率（详见 equity_research.md 保险部分的约束阈值）。

### 3.5 审计与关注点

> 类型：判断表
> 来源：实务 + 监管
> 关联：audit.md、financial_fraud.md
> 置信度：实务经验

| 子行业 | 关注点 | 风险信号 |
|--------|--------|----------|
| 银行 | 贷款减值充分性 | 不良分类不实（次级藏在关注类）、拨备少提虚增利润、借新还旧掩盖不良 |
| 银行 | 利息收入 | 第三阶段贷款仍按本金计息、虚增利息收入 |
| 证券 | 客户资金 | 挪用客户保证金、客户资金与自有资金混同 |
| 证券 | 自营估值 | 难以估值的金融资产公允价值操纵 |
| 保险 | 准备金 | 准备金计提不足（未决赔款低估）虚增利润、CSM 摊销节奏操纵 |
| 保险 | 保费收入 | 投资成分未分拆，虚增保险收入规模 |

> 金融机构舞弊手法详见：financial_fraud.md；银行/保险经营估值详见：equity_research.md。

## 第四章 农业企业

### 4.1 行业特殊性总览

> 类型：判断表
> 来源：CAS 5 + 农业税收政策
> 置信度：实务经验

农业企业的核心特殊点是"生物资产"这一独有资产类别（CAS 5 专门规范）+ 重度税收优惠：

| 特征               | 含义                                 | 引出的难点                         |
| ------------------ | ------------------------------------ | ---------------------------------- |
| 生物资产           | 活的动物和植物，会自然生长/繁殖/退化 | CAS 5 三分类、收获与计量           |
| 生产周期受自然约束 | 种植养殖周期不可压缩                 | 跨期成本归集、消耗性生物资产计价   |
| 自然灾害风险       | 旱涝病虫害导致减值                   | 生物资产减值、保险理赔             |
| 重税收优惠         | 国家扶持农业                         | 农产品免增值税、农林牧渔所得税减免 |

### 4.2 生物资产三分类

> 类型：判断表
> 来源：CAS 5 第三条
> 置信度：准则原文

[场景：判断一项生物资产归入哪一类，决定后续计量方式]

| 判断顺序 | 类别           | 定义                                                | 典型                             | 后续计量                                           |
| -------- | -------------- | --------------------------------------------------- | -------------------------------- | -------------------------------------------------- |
| 1        | 公益性生物资产 | 以防护、环境保护为目的                              | 防风固沙林、水土保持林           | 不计提折旧/减值（除非有确凿证据）                  |
| 2        | 生产性生物资产 | 为产出农产品、提供劳务或出租而持有，可多次/长期使用 | 产蛋鸡、奶牛、果树、橡胶树、役畜 | 类似固定资产：达到成熟后计提折旧                   |
| 3        | 消耗性生物资产 | 为出售而持有，或在将来收获为农产品                  | 育肥猪、商品鱼、大田作物、用材林 | 类似存货：收获时结转成本，期末成本与可变现净值孰低 |

⚠️ 同一物种因持有目的不同分类不同：种猪（产仔）= 生产性；育肥猪（卖肉）= 消耗性。判断看**持有目的**，不看物种。

### 4.3 生物资产的计量

> 类型：规则
> 来源：CAS 5
> 置信度：准则原文

**初始计量：** 一律按**成本**初始计量（CAS 5 以成本模式为主，公允价值模式仅在有活跃市场且能可靠取得公允价值时采用，实务罕见）。

**消耗性生物资产：**

- 自行栽培/营造/繁殖的，按成本（材料+人工+应分摊间接费）归集
- 郁闭前的林木类按营造费用资本化，郁闭后发生的管护费用费用化
- 收获为农产品时，按账面价值结转为农产品（存货）成本
- 期末按"成本与可变现净值孰低"计提跌价准备

**生产性生物资产：**

- 达到预定生产经营目的（成熟）前，发生的支出资本化计入成本
- 成熟后**应当**计提折旧（与固定资产逻辑一致，可用年限平均法/工作量法/产量法）
- 郁闭/成熟后的后续支出费用化

**会计分录——生产性生物资产成熟并计提折旧：**

达到成熟（如果树进入产果期）：

```
借：生产性生物资产——成熟生产性生物资产      XX
  贷：生产性生物资产——未成熟生产性生物资产    XX
```

计提折旧：

```
借：农业生产成本 / 管理费用                  XX
  贷：生产性生物资产累计折旧                  XX
```

**会计分录——消耗性生物资产收获为农产品：**

```
借：农产品（存货）                          XX
  贷：消耗性生物资产                         XX
```

### 4.4 生物资产减值

> 类型：规则
> 来源：CAS 5
> 置信度：准则原文

| 类别           | 减值规则                                                                                        |
| -------------- | ----------------------------------------------------------------------------------------------- |
| 消耗性生物资产 | 期末成本 > 可变现净值 → 计提"存货跌价准备"，价值回升可转回                                     |
| 生产性生物资产 | 可收回金额 < 账面价值 → 计提"生产性生物资产减值准备"，**一经计提不得转回**（同固定资产） |

⚠️ 自然灾害（旱涝、病虫害、疫情扑杀）导致生物资产毁损 → 按账面价值转"营业外支出"，收到保险赔偿冲减损失或计入其他收益。

### 4.5 农业税收优惠

> 类型：规则
> 来源：增值税暂行条例 + 企业所得税法第27条
> 置信度：准则原文

| 税种       | 优惠                                                                                                             |
| ---------- | ---------------------------------------------------------------------------------------------------------------- |
| 增值税     | 农业生产者销售自产农产品**免征增值税**；农业生产资料（部分）免税；纳税人购进农产品可按规定计算抵扣进项     |
| 企业所得税 | 从事农林牧渔项目（蔬菜/谷物/中药材种植、牲畜家禽饲养等）所得**免征**；远海捕捞、部分项目**减半征收** |

⚠️ 易错点：免税的是"农业生产者销售**自产**农产品"。购进再销售（贸易）不享受自产免税；深加工产品（如果汁、罐头）超出初级农产品范围，不再免税。

⚠️ 所得税减免针对"项目所得"，需分项目核算；兼营非优惠项目的应分别核算，否则不得享受。

### 4.6 审计与关注点

> 类型：判断表
> 来源：实务
> 关联：audit.md、financial_fraud.md
> 置信度：实务经验

| 关注点         | 风险信号                                                                    |
| -------------- | --------------------------------------------------------------------------- |
| 生物资产存在性 | "存栏不实"——账上牲畜/林木数量与实际不符（獐子岛扇贝、绿大地苗木经典造假） |
| 计量与折旧     | 生产性生物资产成熟时点判断、折旧是否充分                                    |
| 减值充分性     | 自然灾害/疫情后减值是否足额                                                 |
| 自产认定       | 虚构"自产"骗免税，实为外购贸易                                              |
| 现金交易       | 农业收购多现金、难取证，关注收入真实性                                      |

> 农业行业造假手法（生物资产盘点难、易虚增）详见：financial_fraud.md。

### 5.2 影视作品的成本与摊销

> 类型：规则
> 来源：CAS 6 无形资产 / 存货准则 + 影视行业惯例
> 置信度：实务经验

影视作品的会计处理取决于持有目的：

| 持有目的              | 归类                                           | 后续计量              |
| --------------------- | ---------------------------------------------- | --------------------- |
| 自制对外发行/销售版权 | 存货（库存商品——影视作品）或无形资产（版权） | 按收入比例法摊销      |

**制作成本归集：** 编剧、导演、演员片酬、拍摄、后期制作等直接成本资本化计入"影视作品成本"，发行宣传费一般费用化。

**摊销方法（收入比例法为主）：** 当期摊销额 = 影视作品成本 × (当期收入 ÷ 预计总收入)。

⚠️ 预计总收入需谨慎估计，高估总收入会少摊成本、虚增当期利润。预计无法收回的成本**应当**及时计提减值。

### 6.2 库存商品成本核算（售价金额核算法）

> 类型：规则
> 来源：存货准则 + 零售行业惯例
> 置信度：实务经验

[场景：商超等 SKU 海量、难以逐笔核算进价的零售企业]

售价金额核算法：平时按**售价**记库存商品，月末用"进销差价率"倒算已售成本。

- 进销差价率 = 期初+本期进销差价合计 ÷ 期初+本期售价金额合计
- 本期已销商品应分摊的进销差价 = 本期销售收入（售价）× 进销差价率
- 本期销售成本 = 本期销售收入 − 本期已销商品分摊的进销差价

**示例：** 进销差价率 25%，本月销售收入 800,000 元 → 分摊进销差价 = 800,000 × 25% = 200,000 元 → 销售成本 = 800,000 − 200,000 = 600,000 元。

### 6.3 储值卡、积分与会员

> 类型：规则
> 来源：CAS 14 + 财税〔2016〕53号
> 关联：cas_application_cases.md - 案例4 储值卡增值税
> 置信度：准则原文

**储值卡（预付卡）：** 售卡收到的钱是预收款，**不确认收入**，挂"合同负债"；客户实际消费时确认收入。

⚠️ 增值税：售卡时不缴增值税（不确定适用税率），实际消费时按商品税率计税（详见 cas_application_cases.md 案例4）。

```
售卡：
借：银行存款                              XX
  贷：合同负债——储值卡                     XX
消费核销：
借：合同负债——储值卡                      XX
  贷：主营业务收入 / 应交税费——销项税额      XX
```

**奖励积分（重大权利）：** 销售时赠送的积分若构成"重大权利"，**应当**将部分交易价格按单独售价比例分摊至积分，递延为合同负债，待积分兑换或失效时确认收入。

**会员费：** 一次性会员费按服务期间分摊确认（时段），不在收款时一次确认。

### 6.4 供应商返利与通道费

> 类型：判断表
> 来源：实务 + 税务
> 置信度：实务经验

| 收取项目             | 性质判断     | 处理                                                    |
| -------------------- | ------------ | ------------------------------------------------------- |
| 与销售量挂钩的返利   | 冲减采购成本 | 冲减"库存商品/主营业务成本"，按平销返利开红字发票冲进项 |
| 进场费/通道费/上架费 | 提供服务取得 | 确认为收入（其他业务收入），缴增值税                    |

⚠️ 平销返利不得作为收入，**应当**冲减进项税额，否则少缴增值税（税务稽查高频点）。

### 6.5 门店租赁

> 类型：规则
> 关联：accounting_core.md - 第十四章 租赁
> 置信度：准则原文

执行新租赁准则，门店经营租赁**应当**确认"使用权资产"和"租赁负债"。零售企业门店多，租赁负债影响资产负债率显著。

### 8.2 SaaS 订阅收入确认

> 类型：规则
> 来源：CAS 14
> 关联：accounting_core.md - 第十七章 收入确认五步法
> 置信度：准则原文

[场景：SaaS 企业按年/月收取订阅费]

SaaS 订阅服务是"在一段时间内提供服务"，**应当**按服务期间**时段确认**收入，预收的年费挂合同负债逐月结转。

```
预收年费 120,000（含税，简化）：
借：银行存款                              120,000
  贷：合同负债                            113,207
      应交税费——销项税额（6%）             6,793
每月确认（1/12）：
借：合同负债                              9,434
  贷：主营业务收入                         9,434
```

⚠️ 一次性实施费/接入费：若不能独立于订阅服务、无单独价值 → 并入订阅服务在服务期内分摊，不在收款时一次确认。

⚠️ 易错点：把预收的年度订阅费在收款当期一次确认收入 = 提前确认，是 SaaS 企业虚增收入的典型手法。

### 8.3 平台型企业：主要责任人 vs 代理人

> 类型：判断表
> 来源：CAS 14
> 关联：cas_application_cases.md - 主要责任人代理人判断
> 置信度：准则原文

[场景：电商/外卖/出行等平台，判断按总额还是净额确认收入]

| 判断顺序 | 关键判断                                | 结论                         |
| -------- | --------------------------------------- | ---------------------------- |
| 1        | 平台是否在商品/服务转移给客户前控制它？ | 控制 → 主要责任人（总额法） |
| 2        | 平台仅撮合、不承担存货风险、不主导定价  | 代理人（净额法，只确认佣金） |

⚠️ 自营电商 = 主要责任人按总额（GMV）确认；纯撮合平台 = 代理人只按佣金（抽成）净额确认。混淆总额/净额是平台公司虚增收入的高频手法（瑞幸、部分电商案例）。

#### 8.5.1 跨境电商收入确认（总额法 vs 净额法）

> 类型：判断表
> 来源：CAS 14 + 主要责任人/代理人判断（详见 cas_application_cases.md）
> 置信度：准则原文

跨境电商收入确认的核心难点是"卖家在平台代收代付下，按总额还是净额确认"。判断顺序：

| 判断顺序 | 业务模式 | 判断关键 | 收入确认 |
|---|---|---|---|
| 1 | 自营 FBA / 自营独立站 | 卖家承担存货风险、定价、退货——主要责任人 | **总额法**：按消费者支付的售价确认收入；佣金/FBA费/广告费等作为销售费用列支 |
| 2 | 平台代理（如收取代销佣金的小卖家） | 仅撮合不承担风险——代理人 | **净额法**：仅按佣金净额确认收入（罕见，跨境电商主流是自营总额法） |
| 3 | 转售型商业模式 | 先采购后转售 | 总额法 |

⚠️ 易错点：自营 FBA 是**总额法**，不能因为"亚马逊代收货款"就当成代理人按净额确认。判断看是否承担存货风险与定价权，不看是否代收款。

> 主要责任人/代理人完整判断详见：cas_application_cases.md。

#### 8.5.2 出口退税会计处理（跨境电商与一般贸易共通）

> 类型：规则
> 来源：财税〔2012〕39号 + 跨境电商9810/9710监管方式
> 置信度：准则原文

出口退税的会计处理对跨境电商和一般出口贸易**共通**，核心是"免抵退"机制。

**生产企业出口（"免抵退"全流程分录示例）：**

某出口企业出口货物销售额（FOB 价折人民币）1,000,000 元，退税率 13%，征税率 13%。

确认收入：
```
借：应收账款 / 银行存款                  1,000,000
  贷：主营业务收入                       1,000,000
```

⚠️ 出口环节免征增值税，不计销项税额。

计算应退税额（简化情形：征退税率一致）= 1,000,000 × 13% = 130,000 元，确认应收出口退税款：
```
借：应收出口退税款（增值税）              130,000
  贷：应交税费——应交增值税（出口退税）       130,000
```

收到退税款：
```
借：银行存款                            130,000
  贷：应收出口退税款（增值税）              130,000
```

⚠️ 征退税率不一致（征 13% 退 9%）时，差额部分计入"主营业务成本"——这部分不能退、是企业成本。

**外贸企业出口（"先征后退"，分录略简）：** 采购时进项税额留底→出口时按 FOB 价×退税率计算应退税额。

#### 8.5.3 海外仓存货核算（FBA仓/海外仓）

> 类型：规则
> 来源：CAS 1 存货
> 置信度：实务经验

跨境电商的 FBA 仓/自建海外仓存货是企业自有存货——**应当**确认在企业账面，不因物理上在境外、由平台代管而出表。

| 关注点 | 处理 |
|--------|------|
| 入仓时点 | 头程物流到达 FBA 仓且亚马逊确认收货 → 转"库存商品（海外仓）" |
| 头程成本 | 海运/空运/快递到 FBA 仓的运费、关税、清关费 → 资本化计入存货成本 |
| 仓储费 | 月度仓储费、长期仓储附加费 → 费用化（销售费用） |
| 存货跌价 | 长库龄、滞销 SKU 按"成本与可变现净值孰低"计提跌价准备；移除/销毁的存货按账面价值结转损失 |

⚠️ 易错点：把头程物流费费用化是常见错误。头程是"使存货达到目前场所和状态所发生的支出"，**应当**资本化。

#### 8.5.4 多币种核算与汇兑损益

> 类型：规则
> 来源：CAS 19 外币折算
> 关联：accounting_advanced.md - 外币折算
> 置信度：准则原文

跨境电商收入以外币（美元/欧元/日元等）结算，会计核算采用记账本位币（人民币）：

- 销售收款：按交易日即期汇率折算
- 月末未结汇外币应收/银行存款：按月末即期汇率调整，差额计入"财务费用——汇兑损益"
- 平台账户结汇/提现：按实际结汇汇率，差额计入财务费用

> 外币折算完整规则详见：accounting_advanced.md。

#### 8.5.5 跨境监管方式与会计/税务对应

> 类型：判断表
> 来源：海关总署 + 财税〔2018〕103号
> 置信度：实务经验

跨境电商出口的海关监管方式直接影响税务处理：

| 监管方式 | 含义 | 税务处理 |
|----------|------|----------|
| 9810 | 跨境电商 B2B 出口海外仓 | 视同自营出口，可申请出口退税 |
| 9710 | 跨境电商 B2B 直接出口 | 视同自营出口，可申请出口退税 |
| 9610 | 跨境电商 B2C 直邮零售出口 | 满足条件免征增值税、消费税；所得税核定征收 |
| 1210 | 跨境电商保税备货零售出口 | 同 9610 |

⚠️ 9610/1210 享受"无票免税"政策（无购进发票也可免税），但要求企业纳入跨境电商综合试验区且电子订单等数据完整。

#### 8.5.6 审计与税务关注点

> 类型：判断表
> 来源：实务 + 监管
> 关联：tax_risk_indicators.md、financial_fraud.md
> 置信度：实务经验

| 关注点 | 风险信号 |
|--------|----------|
| 收入完整性 | 平台后台 GMV 与账面收入差异（漏报、不入账少缴税） |
| 总额/净额混淆 | 自营按净额（佣金）申报、虚减收入；或代理按总额申报、虚增规模 |

## 第九章 石油天然气与采掘业

### 9.1 行业特殊性总览

> 类型：判断表
> 来源：CAS 27 石油天然气开采 + 财企〔2012〕16号
> 置信度：实务经验

| 特征 | 含义 | 引出的难点 |
|------|------|------------|
| 资源消耗性 | 储量越采越少 | 油气资产折耗（产量法）、矿区权益 |
| 高危行业 | 安全投入强制 | 安全生产费（专项储备） |
| 环境责任重 | 开采后须修复 | 弃置义务/环境恢复准备金 |
| 勘探高风险 | 探井可能无果 | 勘探支出资本化/费用化判断 |

### 9.2 油气资产与折耗

> 类型：规则
> 来源：CAS 27
> 置信度：准则原文

油气资产分"矿区权益"和"油气井及相关设施"。

- **折耗方法：应当采用产量法**（按当期产量占探明储量/探明已开发储量的比例），不是直线法——因为资源随开采递减
- 矿区权益按探明经济可采储量折耗；油气井及设施按探明已开发储量折耗
- 探明储量发生重大变化时，折耗率未来适用法调整

**勘探支出：** 探井支出先暂时资本化，钻探结果确定后——发现探明经济可采储量则转油气资产，否则费用化计入当期损益。

### 9.3 安全生产费（专项储备，采掘业独有）

> 类型：规则
> 来源：财企〔2012〕16号
> 置信度：准则原文

[场景：煤炭、非煤矿山、危险品等高危行业按规定提取安全生产费]

高危行业**应当**按产量或营业收入的规定标准提取安全生产费，是采掘业独有的特殊处理：

提取时（计入成本，形成所有者权益项下"专项储备"）：
```
借：生产成本 / 制造费用                    XX
  贷：专项储备——安全生产费                  XX
```

使用时——费用性支出，直接冲减专项储备：
```
借：专项储备——安全生产费                  XX
  贷：银行存款                            XX
```

使用时——形成固定资产的，先按在建工程归集，完工转固定资产，同时按形成固定资产的成本冲减专项储备并确认等额累计折旧（该固定资产以后不再计提折旧）：
```
借：专项储备——安全生产费                  XX
  贷：累计折旧                            XX
```

⚠️ 专项储备是所有者权益项目（在资产负债表"专项储备"列示），不是负债、不是费用准备。提取计入成本但同时增加权益，是一种特殊的"成本与权益同增"处理。

### 9.4 弃置义务

> 类型：规则
> 来源：CAS 13 + CAS 4
> 关联：accounting_core.md - 第三章 固定资产（弃置费用主存放）
> 置信度：准则原文

油气设施、矿山等的弃置（拆除、环境恢复）义务，**应当**在取得资产时按弃置费用现值计入资产成本，同时确认"预计负债"，后续按实际利率法计提利息（财务费用）逐期调增负债。

> 弃置费用初始计量分录详见：accounting_core.md - 第三章。

## 第十二章 PPP项目（政府和社会资本合作）

### 12.1 行业特殊性总览

> 类型：规则
> 来源：CAS 解释第14号（PPP 项目合同）
> 关联：government_npo_accounting.md - 3.9 PPP项目资产（政府方/政府会计视角）
> 置信度：准则原文

[场景：社会资本方参与 PPP/BOT 项目（建造-运营-移交）的会计处理]

⚠️ 先分清主体——PPP 有两个会计主体、两套准则，不要混淆：

| 主体 | 适用准则 | 资产确认 | 主存放 |
|------|----------|----------|--------|
| **社会资本方（企业）** ← 本章 | 企业会计准则 CAS解释14号 | 不确认固定资产，确认金融资产/无形资产 | 本文件第十二章 |
| **政府方** | 政府会计准则 | 确认"PPP项目资产/PPP项目净资产"，平行记账 | government_npo_accounting.md - 3.9 |

本章只讲**社会资本方**视角。政府方（双特征+双控制、PPP项目资产折旧冲减净资产）详见：government_npo_accounting.md - 3.9 PPP项目资产。

PPP 项目合同会计（社会资本方）的核心是：对项目资产**不确认为自己的固定资产**（因为最终要移交政府），而是按取得对价的性质分两种模式确认。

### 12.2 两种确认模式（核心判断）

> 类型：判断表
> 来源：CAS 解释第14号
> 置信度：准则原文

| 判断顺序 | 取得对价的性质 | 确认模式 | 入账科目 |
|---|---|---|---|
| 1 | 有权收取**确定金额**的货币（政府付费/可行性缺口补助保底） | 金融资产模式 | 长期应收款（按摊余成本/实际利率法） |
| 2 | 有权向用户**收费**（收费金额不确定，取决于使用量） | 无形资产模式 | 无形资产（特许经营权，按受益期摊销） |
| 3 | 部分确定金额 + 部分收费权 | 混合模式 | 长期应收款 + 无形资产，分别确认 |

### 12.3 建造期与运营期

> 类型：规则
> 来源：CAS 解释第14号 + CAS 14
> 置信度：准则原文

- **建造期**：社会资本方提供建造服务，**应当**按建造合同（时段确认）确认建造收入和成本，同时按上述模式确认金融资产或无形资产
- **运营期**：金融资产模式按实际利率法确认利息收入 + 收回本金；无形资产模式确认运营服务收入 + 无形资产摊销

⚠️ 关键：社会资本方**不将 PPP 项目资产确认为自己的固定资产**——这是 PPP 会计与一般自建资产最大的区别。建造的是"未来收费权/收款权"，不是自有资产。

> 建造合同时段确认详见：本文件第二章 + accounting_core.md 第十七章。


## 第十三章 信托与资产管理

> 类型：判断表
> 来源：CAS 33 合并 + CAS 22 + 资管行业实务
> 置信度：实务经验

| 关注点 | 处理要点 |
|--------|----------|
| 自有资金 vs 受托资产 | 受托管理的信托计划/资管产品资产**不入管理人表**（代客理财，资产归委托人/投资者）；仅管理费、自有跟投份额入表 |
| 资管产品净值核算 | 产品层面按净值法核算（公允价值计量金融资产），定期估值披露净值 |
| 结构化主体合并 | 管理人是否将资管产品/SPV 纳入合并报表——按 CAS 33 控制三要素判断（权力+可变回报+用权力影响回报），不只看持有份额比例 |
| 管理费/业绩报酬 | 管理费时段确认；业绩报酬（超额收益分成）待不确定性消除（如产品到期/开放日）才确认 |

⚠️ 易错点：管理人持有少量份额但实质控制产品（主导投资决策+承担可变回报）→ 可能需合并；纯通道、被动管理 → 通常不合并。判断看实质控制，不看份额。

| 预付卡/会员储值 | 预收挂合同负债，消费时确认（详见第六章零售 6.3） |
| 会员积分 | 构成重大权利的递延（详见第六章 6.3） |
| 加盟费（特许经营） | 初始加盟费按履约义务完成确认（不是收款时一次确认）；持续特许权使用费按加盟商销售额时段确认 |

⚠️ 加盟费易错点：初始加盟费若只是"授予品牌使用权"而后续无重大履约，可在授权时确认；若包含开业筹备、培训、选址等持续服务，应在服务期内分摊，不能签约就全确认。

## 第十六章 物流与仓储

> 类型：规则
> 来源：CAS 14
> 置信度：实务经验

| 业务 | 处理要点 |
|------|----------|
| 运输服务 | 按运输进度（里程/时间）**时段确认**收入；跨期在途运输按进度拆分（同第十章 10.3） |
| 仓储服务 | 按仓储期间时段确认；预收仓储费挂合同负债 |
| 合同履约成本 | 为履行合同发生的、预期收回的成本（如不能直接确认收入的前置成本）资本化为"合同履约成本"，按收入确认进度摊销 |
| 免费期 | 合同含免费仓储/运输期的，将免费期作为合同对价的组成部分，在整个合同期内分摊确认 |

## 第十七章 体育俱乐部

> 类型：规则
> 来源：CAS 6 无形资产 + 行业实务
> 置信度：实务经验

| 关注点 | 处理要点 |
|--------|----------|
| 球员注册权 | 转会引进的球员注册权确认为"无形资产"，按合同期摊销 |
| 转会费 | 引进付出的转会费资本化（无形资产）；出售球员收到的转会费，处置时确认损益（处置价 − 账面净值） |
| 青训支出 | 自培球员的青训支出通常**费用化**（不满足无形资产资本化条件，未来经济利益不确定） |
| 赛事/转播/赞助收入 | 按赛事进行/合同期时段或时点确认 |

⚠️ 自培球员与外购球员的不对称：外购球员有明确成本可资本化，自培球员青训费用化、账面无球员资产——导致俱乐部账面资产无法反映自培球星的真实价值（类似自创商誉不入表）。
