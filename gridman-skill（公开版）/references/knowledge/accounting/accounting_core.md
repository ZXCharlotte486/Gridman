## 第一章 总论

### 一、会计基本假设

| 假设 | 含义 | 实务要点 |
|------|------|----------|
| 会计主体 | 确认、计量和报告的空间范围 | 会计主体≠法律主体；企业集团作为会计主体编合并报表；基金虽非法律主体但属会计主体 |
| 持续经营 | 企业将按当前规模和状态继续经营，不会停业或大规模削减业务 | 持续经营→历史成本+折旧；非持续经营→按破产清算处理；封闭式基金等有限寿命主体不影响持续经营假设 |
| 会计分期 | 将持续经营活动划分为连续、间隔相同的期间 | 分为年度和中期（月/季/半年）；因会计分期才有折旧、摊销、权责发生制 |
| 货币计量 | 以货币为主要计量单位 | 非财务信息（战略、研发能力等）可在报告中补充披露 |

### 二、会计基础

- **权责发生制**：当期已实现的收入和已发生/应负担的费用，无论款项是否收付，均作为当期收入和费用
- **收付实现制**：以收到或支付现金为确认依据。我国行政事业单位预算会计采用收付实现制，财务会计采用权责发生制

### 三、会计信息质量要求（8项）

| 要求 | 核心含义 | 关键判断 |
|------|----------|----------|
| 可靠性 | 以实际发生的交易为依据，如实反映 | 信息应完整、中立、无偏；不得虚构交易 |
| 相关性 | 与使用者决策需要相关 | 具有反馈价值和预测价值；以可靠性为基础 |
| 可理解性 | 清晰明了，便于理解和使用 | 假定使用者具有一定专业知识；复杂信息仍应充分披露 |
| 可比性 | 同一企业不同时期可比；不同企业同期可比 | 不得随意变更会计政策；变更应在附注说明 |
| 实质重于形式 | 按经济实质确认计量，不仅以法律形式为依据 | 如：保留法定所有权但已转移控制权→确认收入 |
| 重要性 | 省略或错报会影响使用者决策判断 | 从性质和金额两方面判断；标准一经确定不得随意变更 |
| 谨慎性 | 不高估资产或收益，不低估负债或费用 | 计提减值准备、确认预计负债；不允许设置秘密准备 |
| 及时性 | 及时确认、计量和报告 | 及时性与可靠性需权衡，以决策有用性为判断标准 |

### 四、会计要素

#### （一）六大要素定义与确认条件

| 要素 | 定义 | 确认条件 |
|------|------|----------|
| 资产 | 过去交易形成的、企业拥有或控制的、预期会带来经济利益的资源 | ①经济利益很可能流入；②成本或价值能可靠计量 |
| 负债 | 过去交易形成的、预期会导致经济利益流出的现时义务 | ①经济利益很可能流出；②流出金额能可靠计量 |
| 所有者权益 | 资产扣除负债后由所有者享有的剩余权益 | 依赖资产和负债的确认；金额取决于资产和负债的计量 |
| 收入 | 日常活动中形成的、导致所有者权益增加的、与所有者投入资本无关的经济利益总流入 | 履行了合同中的履约义务，客户取得商品控制权时确认 |
| 费用 | 日常活动中发生的、导致所有者权益减少的、与向所有者分配利润无关的经济利益总流出 | ①经济利益很可能流出；②导致资产减少或负债增加；③流出额能可靠计量 |
| 利润 | 一定会计期间的经营成果 | 依赖收入、费用、利得、损失的确认和计量 |

**收入 vs 利得**：收入来自日常活动；利得来自非日常活动（计入当期损益或其他综合收益）
**费用 vs 损失**：费用来自日常活动；损失来自非日常活动

**负债中的义务类型**：
- 法定义务：因合同、法规等产生（如应付账款、借款、应交税费）
- 推定义务：因以往习惯做法、公开承诺等产生（如售后保修）

#### （二）所有者权益来源构成

- 股本（实收资本）
- 资本公积（股本溢价/资本溢价、其他资本公积）
- 其他综合收益
- 盈余公积
- 未分配利润
- 一般风险准备（商业银行等金融企业）

#### （三）利润构成

利润 = 收入 - 费用 + 利得 - 损失

其中：收入-费用 = 日常活动业绩；利得-损失 = 非日常活动业绩

### 五、计量属性（5种）

| 属性 | 含义 | 典型应用 |
|------|------|----------|
| 历史成本 | 取得时实际支付的现金或等价物 | 一般计量基础；固定资产、存货初始计量 |
| 重置成本 | 按当前市场条件重新取得同样资产所需支付的金额 | 盘盈固定资产、盘盈存货 |
| 可变现净值 | 估计售价 - 至完工成本 - 销售费用 - 相关税费 | 存货期末计量 |
| 现值 | 未来现金流量以恰当折现率折现后的价值 | 固定资产弃置费用、分期收款销售 |
| 公允价值 | 市场参与者在有序交易中出售资产/转移负债的价格（脱手价格） | 金融工具、投资性房地产（公允价值模式）、非货币性资产交换 |

**应用原则**：一般采用历史成本；采用其他属性时应保证金额能够取得并可靠计量。公允价值的引入是适度、谨慎和有条件的。

**注意**：数据资源当前尚不具备公允价值计量条件，不得以评估值直接作为后续计量依据。

### 六、财务报告

**构成**：财务报表（报表本身 + 附注）+ 其他应披露的相关信息

**报表至少包括**：资产负债表、利润表、现金流量表（小企业可不编）、所有者权益变动表（全面执行企业会计准则的企业）

### 七、可持续信息披露（ESG）

**定义**：企业环境、社会和治理方面的可持续议题相关风险、机遇和影响的信息

**重要性原则（双重重要性）**：
- 财务重要性：可持续风险和机遇是否会给企业带来重要的当期或预期财务影响
- 影响重要性：企业活动是否对经济、社会和环境产生了重要影响

**四个核心要素**：治理、战略、风险和机遇管理、指标和目标

**信息质量要求**：可靠性、相关性、可比性、可验证性、可理解性、及时性

**与财务报表的关系**：
- 报告期间与财务报表一致
- 报告主体范围与财务报表主体一致（合并主体时与合并范围一致）
- 所用数据和假设应尽可能与财务报表一致
- 以货币计量的可持续信息使用与财务报表一致的币种

> 本章为理论章节，无会计分录。

## 第二章 存货

### 一、存货的定义与范围

**定义**：企业在日常活动中持有以备出售的产成品或商品、处在生产过程中的在产品、在生产过程或提供劳务过程中耗用的材料物料等。

**具体包括**：原材料、在产品、半成品、产成品、商品、周转材料（包装物、低值易耗品等）

**不包括**：
- 为建造固定资产储备的材料（不符合存货定义）
- 消耗性生物资产
- 通过建造等服务合同归集的合同履约成本（按收入准则处理）

**确认条件**：①经济利益很可能流入；②成本能够可靠计量

**数据资源**：日常活动中持有、最终目的用于出售的数据资源，符合条件的确认为存货。

### 二、存货的初始计量

存货按**成本**计量。成本 = 采购成本 + 加工成本 + 其他成本

#### （一）外购存货的采购成本

采购成本 = 购买价款 + 相关税费（进口关税、消费税、不能抵扣的增值税等）+ 运输费 + 装卸费 + 保险费 + 其他可归属费用

**采购过程中的损耗处理**：
- 合理途中损耗 → 计入采购成本
- 供货方/运输方赔款 → 冲减采购成本
- 意外灾害损失 → 待处理财产损溢 → 营业外支出
- 应由相关人员赔偿 → 其他应收款
- 管理原因损失 → 管理费用

**商品流通企业进货费用**：
- 原则：计入所购商品成本
- 实务：可先归集，期末按存销情况分摊（已销→主营业务成本；未售→期末存货成本）
- 金额较小：可直接计入当期销售费用

#### （二）加工取得存货的成本

加工成本 = 直接人工 + 制造费用

**制造费用**：生产部门管理人员薪酬、折旧费、办公费、水电费、机物料消耗、劳动保护费、季节性/修理期间停工损失、固定资产日常修理费用等

**成本分配方法**：品种法、分批法、分步法

**完工与在产品分配方法**：不计算在产品成本法、固定成本法、所耗直接材料成本法、约当产量比例法、定额成本法、定额比例法

#### （三）其他方式取得存货

| 方式 | 成本确定 |
|------|----------|
| 投资者投入 | 合同/协议约定价值（不公允时按公允价值，差额计入资本公积） |
| 非货币性资产交换 | 按非货币性资产交换准则 |
| 债务重组 | 按债务重组准则 |
| 企业合并 | 按企业合并准则 |
| 盘盈 | 重置成本 |

#### （四）不计入存货成本的费用（计入当期损益）

1. 非正常消耗的直接材料、直接人工及制造费用
2. 入库后的仓储费用（但生产过程中为达到下一阶段所必需的仓储费用计入成本）
3. 不能归属于使存货达到目前场所和状态的其他支出
4. 采购用于广告营销的特定商品 → 取得时计入销售费用

### 三、发出存货的计量方法

| 方法 | 特点 | 适用 |
|------|------|------|
| 先进先出法 | 先购入先发出；物价上升时高估利润和库存 | 可随时结转，但收发频繁时工作量大 |
| 移动加权平均法 | 每次进货后重新计算单位成本 | 及时了解成本，但计算量大 |
| 月末一次加权平均法 | 月末一次计算平均单价 | 简化计算，但平时无法提供单价 |
| 个别计价法 | 逐一辨认各批次实际成本 | 不能替代使用的存货、贵重物品；最准确 |

**注意**：不允许采用后进先出法。性质和用途相似的存货应采用相同方法。

### 四、期末存货计量

#### （一）原则：成本与可变现净值孰低

- 成本 < 可变现净值 → 按成本计量
- 成本 > 可变现净值 → 按可变现净值计量，差额计提存货跌价准备

#### （二）可变现净值的确定

**直接用于出售的存货**（产成品、商品、用于出售的材料）：
> 可变现净值 = 估计售价 - 估计的销售费用 - 相关税费

**需要加工的材料存货**：
> 可变现净值 = 所生产产成品的估计售价 - 至完工估计成本 - 估计的销售费用 - 相关税费

#### （三）估计售价的确定

| 情形 | 估计售价 |
|------|----------|
| 有合同且数量=持有量 | 合同价格 |
| 持有量 > 合同数量 | 合同部分用合同价；超出部分用市场价 |
| 持有量 < 合同数量 | 实际持有部分用合同价（亏损合同另按或有事项处理） |
| 无合同 | 一般市场销售价格 |
| 用于出售的材料 | 材料市场价格（有合同按合同价） |

#### （四）材料存货的期末计量

- 产成品可变现净值 > 产成品成本 → 材料按成本计量
- 产成品可变现净值 < 产成品成本 → 材料按可变现净值计量

#### （五）计提方法

1. **通常按单个存货项目计提**
2. 数量繁多、单价较低 → 可按存货类别计提
3. 同一地区、相同用途、难以分开计量 → 可合并计提
4. 同一项存货部分有合同、部分无合同 → 分别确定，不得相互抵销

#### （六）跌价准备转回

- 条件：以前减记价值的影响因素已消失
- 限额：在原已计提金额内转回，以冲减至零为限
- 对应关系：转回与计提的存货项目/类别应直接对应

**会计分录——存货跌价准备的转回**：
```
借：存货跌价准备                         XX
  贷：资产减值损失——存货减值损失           XX
```

#### （七）跌价准备结转

销售存货时同时结转已计提的跌价准备（实际按账面价值结转成本）。按类别计提的，按转出成本占该类别总成本的比例结转。

### 五、存货清查盘点

| 情形 | 处理 |
|------|------|
| 盘亏/毁损——计量差错和管理不善 | 扣除残料/赔偿后净损失 → 管理费用 |
| 盘亏/毁损——自然灾害等非常原因 | 扣除处置收入/赔偿后净损失 → 营业外支出 |
| 盘亏/毁损——重大会计差错 | 按会计政策变更和差错更正准则处理 |
| 盘盈——自然溢余 | 调整库存数量和单位成本 |
| 盘盈——重大会计差错 | 按差错更正准则处理 |
| 盘盈——其他原因（日常收发计量不重大差错） | 按重置成本冲减管理费用或计入营业外收入 |

**注意**：盘亏/毁损中不能抵扣的增值税进项税额应予转出。期末"待处理财产损溢"应无余额。

**会计分录——存货盘亏或毁损**：

转入待处理：
```
借：待处理财产损溢                       XX
  贷：原材料/库存商品等                    XX
```

批准处理后——管理不善等原因：
```

    其他应收款                           XX（保险/过失人赔偿）
    原材料                               XX（残料价值）
  贷：待处理财产损溢                      XX
```

批准处理后——自然灾害等非常原因：
```

    其他应收款                           XX（保险/过失人赔偿）
    原材料                               XX（残料价值）

## 第三章 固定资产

### 一、定义与确认

**定义**：同时具有以下特征的有形资产：①为生产商品、提供劳务、出租或经营管理而持有；②使用寿命超过一个会计年度

**确认条件**：①经济利益很可能流入；②成本能够可靠计量

**关键判断**：
- "出租"指机器设备类出租，不含建筑物出租（建筑物出租属投资性房地产）
- 工具、备品备件等数量多单价低的通常确认为存货，但符合固定资产定义的（如高价周转件）应确认为固定资产
- 各组成部分使用寿命不同或折旧方法不同的，应分别确认为单项固定资产（如飞机引擎）
- 因安全或环保要求购入的设备，虽不能直接带来经济利益，但有助于从其他资产获取利益，应确认为固定资产

### 二、初始计量

固定资产按**取得成本**计量 = 达到预定可使用状态前的一切合理、必要支出

#### （一）外购固定资产

成本 = 购买价款 + 相关税费 + 运输费 + 装卸费 + 安装费 + 专业人员服务费等（不含可抵扣增值税进项税额）

- 不需安装：购入即达到预定可使用状态 → 直接记"固定资产"
- 需要安装：先记"在建工程"，安装完毕达到预定可使用状态后转入"固定资产"
- 一笔款项购入多项资产：按各项公允价值比例分配总成本

**会计分录——外购需要安装的固定资产**：

步骤一：支付设备价款
```
借：在建工程——XX设备                    XX（购买价款）
    应交税费——应交增值税（进项税额）      XX
  贷：银行存款                            XX
```

步骤二：领用材料、支付安装工人工资
```

#### （二）分期付款购入固定资产（融资性质）

**判断标准**：付款期限超过正常信用条件（如分期付款且期限较长）→ 具有融资性质

**计量规则**：
- 固定资产成本 = 各期付款额的**现值之和**（非付款总额）
- 折现率 = 反映当前市场货币时间价值和延期付款债务特定风险的利率（实质为供货方必要报酬率）
- 总价款与现值的差额 = 未确认融资费用

**未确认融资费用的分摊**：
- 达到预定可使用状态**前**：符合资本化条件的 → 计入固定资产成本（通过在建工程）
- 达到预定可使用状态**后**：计入财务费用（当期损益）
- 分摊方法：实际利率法

#### （三）自行建造固定资产

成本 = 达到预定可使用状态前的必要支出（工程物资、人工、税费、资本化借款费用、间接费用等）

**试运行销售处理（重要）**：
- 试运行产出的产品/副产品 → 符合条件确认为存货
- 试运行销售收入和成本 → **分别**计入当期损益
- **不得**将试运行销售净额冲减固定资产成本
- 属于日常活动 → 营业收入/营业成本；非日常活动 → 资产处置收益

**自营建造**：
- 工程物资按实际成本核算，领用时转入在建工程
- 建设期间物资盘亏/报废净损失 → 计入工程成本；盘盈/处置净收益 → 冲减工程成本
- 工程完工后物资盘亏/报废 → 营业外支出；盘盈 → 营业外收入
- 已达到预定可使用状态但未办竣工决算 → 按暂估价值转入固定资产并计提折旧，决算后调整暂估价值但**不调整已提折旧**

**安全生产费形成固定资产**：
- 通过在建工程归集 → 确认固定资产
- 同时冲减"专项储备"并确认等额累计折旧
- 该固定资产以后期间**不再计提折旧**

**出包建造**：
- 成本 = 建筑工程支出 + 安装工程支出 + 待摊支出
- 待摊支出包括：管理费、可行性研究费、临时设施费、监理费、资本化借款费用等
- 土地出让金**不计入**在建工程成本 → 确认为无形资产
- 待摊支出分配率 = 累计待摊支出 ÷（建筑工程支出 + 安装工程支出 + 在安装设备支出）× 100%

| 非货币性资产交换 | 按非货币性资产交换准则 |
| 债务重组 | 按债务重组准则 |
| 企业合并 | 按企业合并准则 |

#### （五）存在弃置义务的固定资产

**适用范围**：特殊行业的特定固定资产（如核电站、矿井等），根据法律/行政法规/国际公约承担的环境保护和生态恢复义务

**初始计量**：
- 弃置费用按**现值**计入固定资产成本，同时确认等额预计负债
- 固定资产成本 = 建造成本 + 弃置费用现值

**后续处理**：
- 每年按预计负债摊余成本 × 实际利率计算利息费用 → 计入财务费用，增加预计负债
- 弃置费用变动调整规则：
  - 预计负债减少 → 以固定资产账面价值为限扣减固定资产成本（超出部分计入当期损益）
  - 预计负债增加 → 增加固定资产成本
- 调整后的固定资产在剩余使用年限内计提折旧
- 使用寿命结束后，预计负债的后续变动 → 计入当期损益

**注意**：一般工商企业的报废清理费用**不属于**弃置费用，应在发生时作为处置费用处理

### 三、固定资产的后续计量

#### （一）固定资产折旧

**定义**：在使用寿命内，按确定方法对应计折旧额进行的系统分摊

**影响因素**：
1. 固定资产原价（成本）
2. 使用寿命（考虑：预计生产能力、有形损耗、无形损耗、法律限制）
3. 预计净残值（使用寿命终了时处置金额 - 处置费用）
4. 已计提减值准备累计金额

**计提范围**：
- 所有固定资产均应计提折旧，**除外**：已提足折旧仍继续使用的、单独计价入账的土地
- 当月增加 → 当月不提，下月起提；当月减少 → 当月照提，下月起停
- 闲置/退出活跃使用 → 仍应计提折旧（除非已提足）
- 划分为持有待售 → 停止计提折旧
- 提足折旧后不再提；提前报废不补提
- 暂估入账的 → 按暂估价值计提折旧，决算后调整原价但**不调整已提折旧**

**应计折旧额** = 原价 - 预计净残值 - 已计提减值准备累计金额

**折旧方法**：

| 方法 | 公式 | 特点 |
|------|------|------|
| 年限平均法 | 年折旧率 = (1-预计净残值率) ÷ 预计使用寿命 | 各期折旧额相等 |
| 工作量法 | 单位工作量折旧额 = 原价×(1-净残值率) ÷ 预计总工作量 | 按实际使用量计提 |
| 双倍余额递减法 | 年折旧率 = 2÷预计使用寿命×100%；折旧额 = 期初净值×年折旧率 | 最后两年改为直线法（扣除净残值后平均） |
| 年数总和法 | 年折旧率 = 尚可使用寿命÷年数总和；折旧额 = (原价-净残值)×年折旧率 | 逐年递减 |

**重要规则**：
- 不应以收入为基础计提折旧（收入受投入、生产、销售等因素影响，与经济利益消耗方式无关）
- 构成固定资产的重要部件成本占比重大的 → 应单独计提折旧
- 折旧方法一经确定不得随意变更

**折旧计入科目**：

| 用途 | 计入科目 |
|------|----------|
| 基本生产车间 | 制造费用 |
| 管理部门 | 管理费用 |
| 销售部门 | 销售费用 |
| 自行建造固定资产使用 | 在建工程 |
| 经营租出 | 其他业务成本 |

**会计分录——固定资产折旧**：
```

    管理费用                             XX（管理部门）
    销售费用                             XX（销售部门）

    其他业务成本                         XX（经营租出）

```

**使用寿命、预计净残值和折旧方法的复核**：
- 至少每年年度终了进行复核
- 有确凿证据表明与原估计有差异 → 调整（作为会计估计变更处理）
- 经济环境、技术环境变化可能导致使用寿命缩短、净残值减少、折旧方法不再适用

## 第四章 无形资产

### 一、定义与基本特征

**定义**：企业拥有或者控制的没有实物形态的可辨认非货币性资产

**三大特征**：

| 特征 | 要点 |
|------|------|
| 不具有实物形态 | 表现为权利、技术或获取超额利润的综合能力；存在于实物载体中不改变其无形性质 |
| 具有可辨认性 | 区别于商誉的关键；满足条件之一：①能从企业分离/划分出来单独出售转让；②产生于合同性权利或其他法定权利 |
| 属于非货币性资产 | 不直接表现为固定货币数额，区别于应收款项等 |

**商誉不属于无形资产**：商誉与企业整体价值联系，不具有可辨认性，不能单独辨认和独立确认

**不确认为无形资产的项目**：
- 客户关系、人力资源（无法控制其带来的未来经济利益）
- 内部产生的品牌、报刊名、刊头、客户名单（不能与整个业务开发成本区分）

### 二、无形资产的内容

| 类别 | 说明 | 法定期限 |
|------|------|----------|
| 专利权 | 发明创造的专有权利（发明、实用新型、外观设计） | 发明20年、实用新型10年、外观设计15年（自申请日起） |
| 非专利技术 | 不享有法律保护的技术和诀窍，以自我保密维持独占性 | 无法定期限 |
| 商标权 | 在指定商品/产品上使用特定名称或图案的权利 | 有效期10年（自核准注册日起），可续展（每次10年） |
| 著作权 | 作者对文学、科学和艺术作品的权利 | 署名权等不受限；财产权为作者终生+死后50年 |
| 特许权 | 经营或销售特定商品的权利（政府授权或企业间合同） | 按合同/授权确定 |
| 土地使用权 | 在一定期间内对国有土地的开发、利用、经营权利 | 按出让合同确定 |

### 三、确认条件

同时满足：
1. **经济利益很可能流入企业**：应有明确证据支持，对各种经济因素作出合理估计
2. **成本能够可靠计量**：内部产生的品牌、报刊名等因成本无法可靠计量不确认

**无形项目支出的处理**：除符合无形资产确认条件的部分和企业合并中构成商誉的部分外，均应于发生时计入当期损益

- 引入新产品的宣传广告费、管理费用及其他间接费用

- 非必不可少的经营活动损益

- 按相应准则确定

#### （四）政府补助取得

- 按公允价值计量；公允价值不能可靠取得的按名义金额计量

#### （五）土地使用权

**一般规则**：按取得时支付的价款及相关税费确认为无形资产，与地上建筑物**分别核算**（分别摊销和折旧）

**例外**：
- 房地产开发企业用于建造对外出售房屋 → 计入房屋建筑物成本
- 外购房屋建筑物无法合理分配土地和建筑物价值 → 全部作为固定资产
- 改变用途用于出租或增值 → 转为投资性房地产

**会计分录——土地使用权及自行建造厂房**：

取得土地使用权：
```
借：无形资产——土地使用权                 XX
  贷：银行存款                            XX
```

在土地上自行建造厂房：
```
借：在建工程                             XX
  贷：工程物资                            XX
      应付职工薪酬                        XX
      银行存款                            XX
```

厂房达到预定可使用状态：
```
借：固定资产                             XX
  贷：在建工程                            XX
```

每年分别摊销土地使用权和计提厂房折旧：
```
借：制造费用（土地摊销）                  XX（土地成本÷使用年限）
    制造费用（厂房折旧）                  XX（厂房成本÷使用年限）
  贷：累计摊销                            XX
      累计折旧                            XX
```

#### （六）企业合并取得

| 合并类型 | 计量基础 |
|----------|----------|
| 同一控制下吸收合并 | 被合并方在最终控制方合并报表中的账面价值 |
| 同一控制下控股合并 | 合并报表中按最终控制方合并报表账面价值 |
| 非同一控制下 | 购买日公允价值（包括被购买方原未确认但满足条件的无形资产） |

**非同一控制下确认条件**（满足之一）：
- 源于合同性权利或其他法定权利
- 能够从被购买方中分离/划分出来，单独或与相关合同/资产/负债一起用于出售、转移等

### 五、内部研究开发支出

#### （一）研究阶段与开发阶段的划分

| | 研究阶段 | 开发阶段 |
|---|----------|----------|
| 定义 | 为获取新技术和知识进行的有计划调研 | 将研究成果应用于某项计划/设计，生产新的或实质性改进的产品 |
| 特点 | 计划性、探索性 | 针对性、形成成果可能性较大 |
| 目标 | 不具体，不具针对性 | 针对具体目标、产品、工艺 |
| 风险 | 成功率低，风险大 | 成功率较高，风险较小 |
| 结果 | 研究报告等基础性成果 | 具体的新技术、新产品 |
| 会计处理 | **全部费用化**（管理费用——研究费用） | 符合条件的**资本化**，不符合的费用化 |

**无法区分研究阶段和开发阶段** → 全部费用化

#### （二）开发阶段支出资本化的五个条件（同时满足）

1. **技术可行性**：完成该无形资产以使其能够使用或出售在技术上具有可行性
2. **完成意图**：具有完成该无形资产并使用或出售的意图（有书面决策文件）
3. **经济利益方式**：能证明产品存在市场/无形资产自身存在市场/内部使用有用性
4. **资源支持**：有足够的技术、财务资源和其他资源支持完成开发
5. **支出可靠计量**：归属于开发阶段的支出能够可靠地计量（单独核算）

#### （三）内部开发无形资产的成本构成

**计入成本**：开发阶段满足资本化条件后至达到预定用途前的支出
- 耗费的材料、劳务成本
- 注册费
- 使用的其他专利权和特许权的摊销
- 可资本化的利息支出

**不计入成本**：
- 销售费用、管理费用等间接费用
- 达到预定用途前的可辨认无效和初始运作损失
- 为运行该无形资产发生的培训支出

**重要规则**：资本化条件之前已费用化的支出**不再调整**（即使后续证明可资本化）

#### （四）会计处理

**会计分录——内部研究开发支出**：

发生研发支出：
```
借：研发支出——费用化支出                 XX
    研发支出——资本化支出                 XX
  贷：原材料                              XX
      应付职工薪酬                        XX
      银行存款                            XX
```

期末费用化支出转入损益：
```
借：管理费用——研究费用                   XX
  贷：研发支出——费用化支出                XX
```

达到预定用途：
```
借：无形资产                             XX
  贷：研发支出——资本化支出                XX
```

### 六、无形资产的后续计量

#### （一）使用寿命的判断

**使用寿命有限**：能预见为企业带来经济利益的期限 → 在使用寿命内摊销
**使用寿命不确定**：无法预见期限 → 不摊销，但每年至少进行减值测试

**确定使用寿命应考虑的因素**：
- 产品寿命周期、类似资产使用寿命信息
- 技术/工艺现状及未来发展
- 产品/服务市场需求
- 竞争者预期行动
- 维护支出及企业支付能力
- 控制期限、法律/类似限制
- 与其他资产使用寿命的关联性
- 数据资源还应关注：业务模式、权利限制、更新频率和时效性、技术迭代、同类竞品

**使用寿命确定规则**：
- 有合同/法定期限 → 不超过该期限
- 企业预期使用期限短于法定期限 → 按预期使用期限
- 续约不需付出重大成本时 → 续约期可包括在使用寿命中
- 无明确合同/法律规定 → 综合各方面情况确定；仍无法确定的 → 使用寿命不确定

**复核**：
- 使用寿命有限的 → 至少每年年末复核使用寿命及摊销方法
- 使用寿命不确定的 → 每个会计期间复核；有证据表明有限的 → 估计使用寿命，按会计估计变更处理

#### （二）使用寿命有限的无形资产

**应摊销金额** = 成本 - 残值 - 已计提减值准备

**摊销期**：自可供使用时起至终止确认时止

**摊销方法**：直线法、产量法等（根据经济利益预期消耗方式确定）
- 无法可靠确定消耗方式 → 采用直线法
- 不应以收入为基础摊销（极有限例外：合同规定收入总额达到固定金额时失效；有确凿证据表明收入与经济利益消耗高度相关）
- 车流量法摊销高速公路经营权 → 不属于以收入为基础的方法

**摊销计入科目**：
- 一般 → 当期损益（管理费用）
- 专门用于生产产品 → 相关产品成本（制造费用）
- 划分为持有待售 → 不摊销

**会计分录——无形资产摊销**：
```
借：制造费用                             XX（用于产品生产的专利技术）
    管理费用                             XX（商标权等）
  贷：累计摊销                            XX
```

**残值确定**：
- 一般视为零
- 例外：有第三方承诺购买；或存在活跃市场且该市场在使用寿命结束时可能仍存在
- 残值高于账面价值 → 不再摊销，直至残值降至低于账面价值时恢复

#### （三）使用寿命不确定的无形资产

- 持有期间**不摊销**
- 至少每年年度终了进行**减值测试**

#### （四）无形资产减值

- 使用寿命有限 → 出现减值迹象时进行减值测试
- 使用寿命不确定 → 无论是否有减值迹象，每年都应进行减值测试
- 可收回金额 < 账面价值 → 确认资产减值损失
- 无形资产减值准备一经计提**不得转回**
- 确认减值后 → 在剩余使用寿命内按调整后账面价值重新摊销

**会计分录——无形资产减值**：
```
借：资产减值损失                         XX（账面价值-可收回金额）
  贷：无形资产减值准备                    XX
```

### 七、无形资产的处置

#### （一）出售

- 划分为持有待售类别 → 按持有待售准则处理
- 未划分为持有待售 → 出售价款 - 账面价值 - 相关税费 = 资产处置损益

#### （二）报废

- 不能再为企业带来经济利益（被新技术替代、超过法律保护期等）→ 转销
- 账面价值转作当期损益（营业外支出）

**会计分录——无形资产报废**：
```
借：累计摊销                             XX
    无形资产减值准备                      XX（如有）
    营业外支出                           XX（差额）
  贷：无形资产                            XX
```

### 八、数据资产会计处理（财会〔2023〕32号）

> 《企业数据资源相关会计处理暂行规定》2023年8月发布，2024年1月1日起施行。首次将数据资源纳入企业会计核算体系。

#### （一）数据资源的分类确认

| 持有目的 | 确认为 | 列报位置 |
|----------|--------|----------|
| 日常活动中持有、最终目的用于出售 | **存货** | 存货项下单独列示"数据资源" |
| 用于提供服务/内部使用/对外许可 | **无形资产** | 无形资产项下单独列示"数据资源" |
| 正在开发中的数据资源 | **开发支出** | 开发支出项下单独列示 |

#### （二）初始计量

**外购数据资源成本** = 购买价款 + 相关税费 + 脱敏/清洗/标注/整合/分析/可视化等加工支出 + 权属鉴证/质量评估/登记结算/安全管理费用

**内部开发数据资源**：适用研发支出资本化规则（同无形资产第四章），满足资本化五条件的计入无形资产，否则费用化。

**注意**：
- 数据资源当前尚不具备公允价值计量条件，不得以评估值直接作为入账价值
- 企业内部产生的数据，如果无法可靠计量成本，不得确认为资产

#### （三）后续计量

- 确认为**无形资产**的数据资源：按使用寿命判断规则摊销（需额外关注更新频率、时效性、技术迭代、同类竞品）
- 确认为**存货**的数据资源：按成本与可变现净值孰低计量
- 使用寿命不确定的数据资源：不摊销，每年减值测试

#### （四）披露要求

- 在资产负债表中单独列示"数据资源"（存货项下或无形资产项下）
- 附注中披露：数据资源的确认条件、计量方法、摊销政策、减值情况
- 鼓励披露：数据资源的来源、应用场景、安全保障措施

## 第五章 投资性房地产

### 第一节 投资性房地产的特征与范围

**定义：** 为赚取租金或资本增值（或两者兼有）而持有的房地产。

**与其他房地产的区分：**

| 类别 | 特征 | 核算方式 |
|------|------|----------|
| 投资性房地产 | 赚取租金或资本增值 | 按投资性房地产准则 |
| 自用房地产 | 生产商品/提供劳务/经营管理 | 固定资产或无形资产 |
| 存货性房地产 | 房地产开发企业销售或为销售而开发 | 存货 |

**投资性房地产的范围：**

1. **已出租的土地使用权** — 以经营租赁方式出租的土地使用权
   - 租入再转租的土地使用权 → 不属于投资性房地产
2. **持有并准备增值后转让的土地使用权**
   - 闲置土地（国家认定）→ 不属于投资性房地产
3. **已出租的建筑物** — 企业拥有产权的、以经营租赁方式出租的建筑物
   - 租入再转租的建筑物 → 不属于投资性房地产
   - 空置建筑物：董事会书面决议明确用于经营出租且持有意图短期内不变 → 视为投资性房地产
   - 辅助服务不重大 → 确认为投资性房地产

**不属于投资性房地产的情形：**
- 出租给本企业职工居住的宿舍（自用性质）
- 企业自行经营的旅馆饭店（经营场所）
- 房地产开发企业用于开发后出售的土地使用权（即使决定待增值后转让）

**部分自用/部分出租：** 不同用途部分能够单独计量和出售的 → 分别确认

### 第二节 投资性房地产的确认和初始计量

**确认条件：**
1. 经济利益很可能流入企业
2. 成本能够可靠计量

**确认时点：**
- 已出租的土地使用权/建筑物 → 租赁期开始日
- 持有并准备增值后转让的土地使用权 → 停止自用、准备增值后转让的日期

**初始计量 — 按成本：**

| 取得方式 | 初始成本构成 |
|----------|-------------|
| 外购 | 购买价款 + 相关税费 + 直接归属支出 |
| 自行建造 | 达到预定可使用状态前的必要支出（非正常损失不计入） |
| 非投资性房地产转换 | 成本模式：转换日账面价值；公允价值模式：转换日公允价值 |

**会计分录——外购投资性房地产（部分出租、部分自用）**：
```
借：投资性房地产                         XX（出租部分）
    固定资产                             XX（自用部分）
  贷：银行存款                            XX
```

**后续支出：**
- 资本化：满足确认条件的改建、扩建、装修支出 → 计入投资性房地产成本
  - 再开发期间继续作为投资性房地产，不计提折旧或摊销
- 费用化：日常维护支出 → 计入当期损益（其他业务成本）

**会计分录——投资性房地产改扩建（成本模式）**：

转入改扩建：
```
借：投资性房地产——XX（在建）             XX（账面价值）
    投资性房地产累计折旧                  XX
  贷：投资性房地产——XX                    XX（原价）
```

发生改扩建支出：
```
借：投资性房地产——XX（在建）             XX
  贷：银行存款等                          XX
```

改扩建完工：
```
借：投资性房地产——XX                     XX（新入账价值）
  贷：投资性房地产——XX（在建）            XX
```

### 第三节 投资性房地产的后续计量

**核心规则：** 同一企业只能采用一种模式，不得同时采用两种。

#### 成本模式

- 按期计提折旧或摊销
- 存在减值迹象 → 计提减值准备，已计提的不得转回

**会计分录——成本模式后续计量**：

按期计提折旧/摊销：
```
借：其他业务成本                         XX
  贷：投资性房地产累计折旧（摊销）         XX
```

取得租金收入：
```
借：银行存款                             XX
  贷：其他业务收入                        XX
```

#### 公允价值模式

**适用条件（同时满足）：**
1. 所在地有活跃的房地产交易市场
2. 能够取得同类或类似房地产的市场价格及相关信息

**核算特点：**
- 不计提折旧或摊销
- 资产负债表日按公允价值计量
- 公允价值变动 → 计入"公允价值变动损益"

**公允价值确定方法：**
1. 活跃市场同类/类似房地产现行市场价格
2. 最近交易价格（考虑交易情况、日期、区域等因素调整）
3. 预计未来租金收益和现金流量现值

**会计分录——公允价值模式后续计量**：

资产负债表日公允价值上升：
```
借：投资性房地产——公允价值变动            XX
  贷：公允价值变动损益                    XX
```

资产负债表日公允价值下降：
```
借：公允价值变动损益                     XX
  贷：投资性房地产——公允价值变动           XX
```

#### 计量模式变更

- 成本模式 → 公允价值模式：作为会计政策变更，差额调整期初留存收益
- 公允价值模式 → 成本模式：**不允许**

### 第四节 投资性房地产的转换和处置

#### 转换形式与转换日

| 转换方向 | 转换日 |
|----------|--------|
| 投资性房地产 → 自用房地产 | 房地产达到自用状态日 |
| 投资性房地产 → 存货 | 租赁期届满 + 董事会书面决议重新开发用于对外销售日 |
| 存货 → 投资性房地产 | 租赁期开始日（或董事会书面决议日） |
| 自用房地产 → 投资性房地产 | 租赁期开始日（或董事会书面决议日） |

#### 转换的会计处理

**成本模式下的转换：** 按账面价值对应结转，不产生损益。

**公允价值模式下 — 投资性房地产转为非投资性房地产：**
- 以转换日公允价值作为新资产账面价值
- 公允价值与原账面价值的差额 → 计入当期损益（公允价值变动损益）

**公允价值模式下 — 非投资性房地产转为投资性房地产：**
- 以转换日公允价值入账
- 公允价值 < 账面价值 → 差额计入"公允价值变动损益"（借方）
- 公允价值 > 账面价值 → 差额计入"其他综合收益"（贷方）
- 处置时，原计入其他综合收益的部分转入当期损益

**会计分录——投资性房地产转换**：

成本模式下投资性房地产转为自用房地产：
```
借：固定资产                             XX（原价）
    投资性房地产累计折旧                  XX
  贷：投资性房地产                        XX（原价）
      累计折旧                            XX
```

公允价值模式下投资性房地产转为自用房地产：
```

## 第六章 长期股权投资与合营安排

### 第一节 基本概念

#### 股权投资的分类

| 影响程度 | 投资类型 | 核算依据 |
|----------|----------|----------|
| 控制 | 对子公司投资 | 长期股权投资准则（成本法） |
| 共同控制 | 对合营企业投资 | 长期股权投资准则（权益法） |
| 重大影响 | 对联营企业投资 | 长期股权投资准则（权益法） |
| 无重大影响 | 金融资产 | 金融工具确认和计量准则 |

#### 重大影响的判断

- 持股20%~50% → 一般认为具有重大影响（除非有反证）
- 持股<20% → 一般认为不具有重大影响（除非能明确证明）

**判断重大影响的情形：**
1. 在董事会或类似权力机构中派有代表
2. 参与财务和经营政策制定过程
3. 与被投资单位之间发生重要交易
4. 向被投资单位派出管理人员
5. 向被投资单位提供关键技术资料

**关键原则：** 重大影响是"参与决策的权力"而非"正在行使的权力"，判断核心是是否具备参与并施加重大影响的权力。

#### 共同控制与合营企业

- 共同控制：相关活动必须经过分享控制权的参与方一致同意后才能决策
- 合营企业：合营方仅对该安排的净资产享有权利

### 第二节 长期股权投资的初始计量

#### 对联营企业、合营企业投资的初始计量

| 取得方式 | 初始投资成本 |
|----------|-------------|
| 支付现金 | 实际支付的购买价款（含直接相关费用、税金等，不含已宣告未发放的股利） |
| 发行权益性证券 | 所发行权益性证券的公允价值 |
| 债务重组/非货币性资产交换 | 按相应准则确定 |

- 发行权益性证券的手续费、佣金 → 冲减资本公积（股本溢价），不足冲减的依次冲减盈余公积和未分配利润

**会计分录——以发行权益性证券取得联营/合营企业投资**：

  贷：股本                                XX（面值）
      资本公积——股本溢价                  XX（差额）
```

发行费用：

  贷：银行存款                            XX

#### 对子公司投资的初始计量

**同一控制下控股合并：**
- 初始投资成本 = 被合并方所有者权益在最终控制方合并财务报表中的账面价值 × 持股比例
- 该"账面价值"是从最终控制方角度持续计算至合并日的价值（非被合并方个别报表账面价值）
- 初始投资成本与支付对价账面价值的差额 → 调整资本公积（资本溢价/股本溢价），不足冲减的依次冲减盈余公积和未分配利润
- 合并中发生的审计、法律等中介费用 → 计入当期管理费用
- 支付对价按账面价值结转，不确认损益

- 直接相关费用 → 计入管理费用

- 以库存商品作为对价 → 按公允价值确认收入并结转成本

### 第四节 长期股权投资核算方法的转换及处置

#### 转换总览

| 转换方向 | 原因 | 处理原则 |
|----------|------|----------|
| 成本法 → 权益法 | 处置部分股权丧失控制但仍有重大影响/共同控制 | 剩余股权追溯调整为权益法 |
| 公允价值/权益法 → 成本法 | 追加投资取得控制 | 按子公司投资初始计量规定处理 |
| 公允价值 → 权益法 | 追加投资取得重大影响/共同控制 | 原公允价值 + 新对价公允价值 = 初始投资成本 |
| 权益法 → 公允价值 | 处置部分股权丧失重大影响/共同控制 | 剩余股权按公允价值重新计量，差额计入损益 |
| 成本法 → 公允价值 | 处置部分股权丧失控制且无重大影响 | 剩余股权按公允价值重新计量，差额计入损益 |

#### 成本法转权益法的关键处理

**个别财务报表：**
1. 按处置比例结转长期股权投资成本，差额计入当期损益
2. 剩余股权比较成本与原投资时应享有可辨认净资产公允价值份额
3. 追溯调整：原取得投资后至转换日被投资单位净损益中应享有的份额
   - 取得投资至处置当期期初 → 调整期初留存收益
   - 处置当期期初至处置日 → 调整当期损益
4. 其他所有者权益变动 → 计入其他综合收益或资本公积

**合并财务报表：**
- 剩余股权按丧失控制权日的公允价值重新计量
- 处置对价 + 剩余股权公允价值 - 原持股比例应享有的自购买日持续计算的净资产份额 = 投资收益
- 与原子公司相关的其他综合收益 → 采用与被投资单位直接处置相关资产/负债相同的基础处理

#### 权益法转公允价值计量

- 剩余股权按公允价值重新计量，差额计入当期损益
- 原计入其他综合收益的 → 采用与被投资单位直接处置相关资产/负债相同的基础处理
- 原计入资本公积（其他资本公积）的 → 全部转入当期损益

#### 长期股权投资的处置

- 出售所得价款与账面价值的差额 → 确认处置损益
- 权益法下处置：原计入其他综合收益/资本公积的按比例结转（终止权益法的全部结转）
- 同一控制下合并取得的子公司处置并丧失控制权：原调整的资本公积在个别报表中不得转出至损益或留存收益

**会计分录——分步取得股权形成非同一控制下控股合并（原持有指定为FVOCI的权益工具）**：
```
借：长期股权投资——投资成本               XX（原股权公允价值+新增投资成本）
  贷：其他权益工具投资——成本              XX
      其他权益工具投资——公允价值变动       XX
      银行存款                            XX

借：其他综合收益                         XX（原计入OCI的累计变动）
  贷：利润分配——未分配利润                XX
```

### 第五节 合营安排

#### 合营安排的认定

**定义：** 由两个或两个以上的参与方共同控制的安排。

**共同控制的判断步骤：**
1. 判断是否所有参与方或参与方组合集体控制该安排
2. 判断相关活动的决策是否必须经过这些参与方一致同意

**关键判断规则：**
- 一致同意：不要求主动提出议案的能力，只要具备否决权即可
- 存在多个参与方组合均能满足最低投票权比例 → 不构成共同控制（除非约定明确指定特定组合）
- 仅享有保护性权利的参与方 → 不享有共同控制
- 争议解决机制（如仲裁）的存在 → 不妨碍共同控制的判断

#### 合营安排的分类

| 类型 | 特征 | 合营方的会计处理 |
|------|------|-----------------|
| 共同经营 | 合营方享有资产且承担负债 | 确认自身份额的资产、负债、收入、费用 |
| 合营企业 | 合营方仅对净资产享有权利 | 按权益法核算长期股权投资 |

**参与方分类：**
- 合营方：对合营安排享有共同控制的参与方
- 非合营方：对合营安排不享有共同控制的参与方

## 第七章 资产减值

### 第一节 资产减值概述

**适用范围（CAS 8号）：**
- 固定资产、在建工程、无形资产、商誉、长期股权投资、投资性房地产（成本模式）、使用权资产、生产性生物资产等
- **不适用**：存货（CAS 1）、金融资产（CAS 22）、投资性房地产（公允价值模式）、递延所得税资产（CAS 18）

**核心原则：** 资产减值损失一经确认，在以后会计期间不得转回（区别于存货跌价准备可转回）

### 第二节 资产可收回金额的计量

**可收回金额 = max（公允价值 - 处置费用，预计未来现金流量现值）**

#### 一、公允价值减去处置费用后的净额

确定顺序：
1. 公平交易中资产销售协议价格 - 处置费用
2. 资产存在活跃市场 → 市场价格 - 处置费用
3. 无销售协议和活跃市场 → 以可获取的最佳信息为基础估计

**处置费用：** 与资产处置直接相关的增量成本（法律费用、搬运费、使资产达到可销售状态的直接费用等），不包括财务费用和所得税费用

#### 二、预计未来现金流量现值

**（一）预计未来现金流量**

预计基础：
- 以经管理层批准的最近财务预算/预测为基础，最多涵盖5年（除非能证明更长期间合理）
- 超出预算期的现金流量：以稳定或递减增长率为基础（除非能证明递增合理）
- 增长率不应超过所处行业/国家/地区的长期平均增长率

应包括的内容：
1. 资产持续使用过程中预计产生的现金流入
2. 为实现现金流入所必需的预计现金流出（含达到预定可使用状态的支出）
3. 资产使用寿命结束时处置所收到/支付的净现金流量

应考虑的因素：
- **以当前状况为基础**：不包括尚未承诺的重组或资产改良的影响
- 已承诺重组：应反映重组节约的费用和带来的利益
- **不包括**：筹资活动和所得税收付产生的现金流量
- 通货膨胀因素：与折现率保持一致基础
- 内部转移价格：应调整为公平交易价格

预计方法：
- **传统法**：单一的每期最可能现金流量 + 单一折现率
- **期望现金流量法**：各种可能情况下的现金流量 × 发生概率加权

**（二）折现率**

- 应为反映当前市场货币时间价值和资产特定风险的**税前利率**
- 如已在现金流量中调整了特定风险 → 折现率不再考虑该风险
- 确定方法：以资产市场利率为依据；无法获得时用替代利率（加权平均资金成本、增量借款利率等调整后确定）
- 通常使用单一折现率；对风险差异或利率期间结构敏感的 → 不同期间采用不同折现率

**（三）外币未来现金流量现值**

步骤：
1. 以结算货币为基础预计未来现金流量
2. 按该货币适用的折现率计算现值
3. 按计算日即期汇率折算为记账本位币

### 第三节 资产减值损失的确认与计量

**确认原则：**
- 可收回金额 < 账面价值 → 将账面价值减记至可收回金额
- 减记金额确认为资产减值损失，计入当期损益
- 同时计提资产减值准备（资产的备抵项目）

**后续影响：**
- 减值后的折旧/摊销以新的账面价值为基础重新计算
- 资产减值损失一经确认，以后会计期间**不得转回**

**会计分录——计提资产减值损失**：
```
借：资产减值损失——XX减值损失             XX（账面价值-可收回金额）
  贷：固定资产减值准备                    XX
     /无形资产减值准备                    XX
     /商誉减值准备                        XX
```
> 注：长期资产减值损失一经确认，在以后会计期间不得转回

### 第四节 资产组的认定及减值处理

#### 一、资产组的认定

**定义：** 企业可以认定的最小资产组合，其产生的现金流入应当基本上独立于其他资产或资产组。

**认定要点：**
- 关键因素：能否独立产生现金流入
- 几项资产组合的产品存在活跃市场 → 即使仅供内部使用，也表明能独立创造现金流入
- 应考虑管理层对生产经营的管理/监控方式和资产使用处置的决策方式
- 资产组一经确定，各会计期间保持一致，不得随意变更（重组、变更用途等除外，需附注说明）

#### 二、资产组减值测试

**账面价值确定基础：**
- 应与可收回金额的确定方式一致
- 包括可直接归属于或可合理分摊至资产组的资产账面价值
- 通常不包括已确认负债的账面价值（除非不考虑该负债就无法确定可收回金额）

**减值损失分摊顺序：**
1. 首先抵减分摊至资产组中商誉的账面价值
2. 然后按其他各项资产账面价值比重按比例抵减

**分摊限制：** 抵减后各资产账面价值不得低于以下三者最高者：
- 该资产的公允价值减去处置费用后的净额（如可确定）
- 该资产预计未来现金流量的现值（如可确定）
- 零

未能分摊的减值损失 → 按其他资产账面价值比重再次分摊

**会计分录——资产组减值的分摊**：

> 分摊顺序：先抵减商誉账面价值→再按比例抵减其他资产账面价值

```
借：资产减值损失——商誉减值损失           XX
    资产减值损失——固定资产减值损失        XX
    资产减值损失——无形资产减值损失        XX
  贷：商誉减值准备                        XX
      固定资产减值准备                    XX
      无形资产减值准备                    XX
```

#### 三、总部资产的减值测试

**特征：** 难以脱离其他资产或资产组产生独立现金流入，账面价值难以完全归属于某一资产组。

**处理方法：**

（1）总部资产能按合理一致基础分摊至资产组的部分：
- 将总部资产账面价值分摊至资产组
- 比较资产组账面价值（含分摊的总部资产）与可收回金额

（2）总部资产难以按合理一致基础分摊的部分：
- 步骤一：不考虑总部资产，对资产组进行减值测试
- 步骤二：认定包含该总部资产的最小资产组组合
- 步骤三：比较资产组组合账面价值（含总部资产）与可收回金额

**分摊方法示例：** 按各资产组账面价值×剩余使用寿命加权计算分摊比例

### 第五节 商誉减值测试与处理

#### 一、基本要求

- 商誉至少应当在每年年度终了进行减值测试
- 商誉应结合相关资产组或资产组组合进行减值测试
- 自购买日起按合理方法分摊至相关资产组/资产组组合
- 相关资产组/组合应能从企业合并协同效应中受益
- 不应大于经营分部

#### 二、商誉减值测试方法

**步骤：**
1. 首先对不包含商誉的资产组/组合进行减值测试
2. 再对包含商誉的资产组/组合进行减值测试
3. 减值损失首先抵减商誉账面价值，再按比例抵减其他资产

**少数股东权益的商誉处理：**
- 合并报表中仅确认母公司份额的商誉，少数股东的商誉未确认
- 减值测试时：调整资产组账面价值，将归属于少数股东的商誉包括在内
- 归属于少数股东的商誉 = (合并成本 ÷ 持股比例 - 子公司可辨认净资产公允价值) × 少数股东持股比例
- 计算出的商誉减值损失中，仅确认归属于母公司的部分

**处置含商誉资产组中的经营：**
- 相关商誉应包括在处置经营的账面价值中确定处置损益
- 按处置经营与资产组剩余部分的价值比例分摊商誉

#### 四、应交税费

##### （一）增值税

**一般纳税人核算体系：**

| 明细科目 | 用途 |
|----------|------|
| 应交增值税 | 核算进项税额、销项税额等 |
| 未交增值税 | 月末转入的当月未交/多交增值税 |
| 预交增值税 | 预缴的增值税 |
| 待抵扣进项税额 | 待抵扣的进项税额 |

**"应交增值税"专栏设置：**
- 借方：进项税额、已交税金、减免税款、转出未交增值税、销项税额抵减
- 贷方：销项税额、出口退税、进项税额转出、转出多交增值税

**关键业务处理：**

| 业务 | 处理规则 |
|------|----------|
| 购销业务 | 价税分离：价款→成本/收入，税额→进项/销项 |
| 小规模纳税人 | 进项税不得抵扣，计入成本；销售额=含税额÷(1+征收率) |
| 视同应税交易 | 自产/委托加工货物用于集体福利、个人消费、无偿转让等→计算销项税额 |
| 进项税额转出 | 非正常损失、改变用途→借记相关科目，贷记"进项税额转出" |
| 差额征税 | 全部价款扣除支付给第三方费用后的余额为销售额 |
| 月末结转 | 应交未交→转出未交增值税；多交→转出多交增值税 |
| 交纳增值税 | 当月交当月→已交税金；交以前期→未交增值税 |
| 预缴增值税 | 月末转入"未交增值税" |
| 减免增值税 | 借记"减免税款"，贷记"其他收益" |

### 第二节 非流动负债

#### 一、长期借款

- 期限在一年以上（不含一年）
- 按实际收到款项入账，与面值差额记入"利息调整"
- 资产负债表日：按摊余成本和实际利率计算利息费用
- 利息费用计入：在建工程（资本化）/ 财务费用（费用化）/ 制造费用

**会计分录——长期借款**：

借入时：
```
借：银行存款                             XX
  贷：长期借款——本金                      XX
```

计提利息（资本化期间）：
```
借：在建工程                             XX
  贷：长期借款——应计利息                  XX（到期一次还本付息）
     /应付利息                            XX（分期付息）
```

计提利息（费用化期间）：
```
借：财务费用                             XX
  贷：长期借款——应计利息                  XX
     /应付利息                            XX
```

归还本息：
```

    长期借款——应计利息                   XX
  贷：银行存款                            XX
```

#### 二、公司债券

**发行方式：**

| 方式 | 条件 | 实质 |
|------|------|------|
| 面值发行 | 票面利率 = 市场利率 | — |
| 溢价发行 | 票面利率 > 市场利率 | 溢价是以后各期多付利息的事先补偿 |
| 折价发行 | 票面利率 < 市场利率 | 折价是以后各期少付利息的预先补偿 |

**核算要点：**
- 面值记入"应付债券——面值"，差额记入"应付债券——利息调整"
- 利息调整采用**实际利率法**在存续期间摊销
- 实际利率：将未来现金流量折现为当前账面价值的利率
- 每期利息费用 = 期初摊余成本 × 实际利率
- 每期摊销的利息调整 = 票面利息 - 利息费用（溢价）或 利息费用 - 票面利息（折价）

**会计分录——应付债券**：

发行债券：
```
借：银行存款                             XX（实际收到金额）
    应付债券——利息调整                   XX（折价发行时）
  贷：应付债券——面值                      XX
      应付债券——利息调整                  XX（溢价发行时）
```

计提利息（实际利率法）：
```
借：在建工程/财务费用                    XX（期初摊余成本×实际利率）
  贷：应付利息                            XX（面值×票面利率）
      应付债券——利息调整                  XX（差额，折价摊销）
```

#### 三、长期应付款

- 除长期借款和应付债券以外的其他长期应付款项
- 延期支付超过正常信用条件、具有融资性质的购买资产：
  - 资产成本 = 购买价款的现值
  - 实际支付价款与现值的差额 → "未确认融资费用"
  - 未确认融资费用在信用期间内按实际利率法摊销

#### 四、地方政府专项债券

**需承担偿还义务的：**
- 确认为负债（长期应付款——专项债券）
- 设置"本金""应计利息"明细科目
- 收到资金：借记银行存款，贷记长期应付款——专项债券——本金
- 计提利息：借记在建工程/财务费用，贷记长期应付款——专项债券——应计利息

**不需承担偿还义务的：** 结合具体约定按有关准则处理

## 第九章 职工薪酬

### 第一节 职工和职工薪酬的范围及分类

#### 一、职工的范围

| 类别 | 说明 |
|------|------|
| 与企业订立劳动合同的人员 | 含全职、兼职和临时职工（固定期限/无固定期限/完成一定工作为期限） |
| 未订立合同但由企业正式任命的人员 | 如独立董事、外部监事 |
| 虽未订立合同也未正式任命，但提供类似服务的人员 | 如通过劳务中介公司向企业提供服务的人员 |

#### 二、职工薪酬的分类

**定义：** 企业为获得职工提供的服务或终止劳动合同关系而给予的各种形式的报酬或补偿。包括提供给职工配偶、子女、受赡养人、已故员工遗属等的福利。

| 类别 | 定义 | 主要内容 |
|------|------|----------|
| 短期薪酬 | 职工提供服务的年度报告期间结束后12个月内需全部支付的（辞退福利除外） | 工资奖金津贴补贴、职工福利费、社会保险费、住房公积金、工会经费、职工教育经费、短期带薪缺勤、短期利润分享计划、非货币性福利 |
| 离职后福利 | 职工退休或解除劳动关系后提供的报酬和福利（短期薪酬和辞退福利除外） | 设定提存计划、设定受益计划 |
| 辞退福利 | 劳动合同到期前解除劳动关系或鼓励自愿裁减给予的补偿 | 企业决定辞退的补偿、鼓励自愿接受裁减的补偿 |
| 其他长期职工福利 | 除上述三类以外的所有职工薪酬 | 长期带薪缺勤、长期残疾福利、长期利润分享计划 |

**短期带薪缺勤：** 年休假、病假、短期伤残、婚假、产假、丧假、探亲假等

**离职后福利计划分类：**
- 设定提存计划：向独立基金缴存固定费用后，企业不再承担进一步支付义务
- 设定受益计划：除设定提存计划以外的离职后福利计划（企业承担精算风险和投资风险）

### 第二节 短期薪酬的确认与计量

**总原则：** 在职工提供服务的会计期间，将实际发生的短期薪酬确认为负债，计入当期损益或相关资产成本。

#### 一、货币性短期薪酬

**会计分录——短期薪酬的计提与发放**：

计提时：
```
借：生产成本                             XX（生产工人）
    制造费用                             XX（车间管理人员）
    管理费用                             XX（行政管理人员）
    销售费用                             XX（销售人员）
    在建工程                             XX（工程人员）
    研发支出                             XX（研发人员）
  贷：应付职工薪酬——工资                  XX
      应付职工薪酬——职工基本医疗保险费     XX
      应付职工薪酬——住房公积金            XX
      应付职工薪酬——工会经费              XX
      应付职工薪酬——职工教育经费          XX
```

发放时：
```
借：应付职工薪酬                         XX
  贷：银行存款                            XX
      应交税费——应交个人所得税            XX
```

**计提基础：**
- 工资奖金津贴补贴：根据职工提供服务情况和工资标准
- 职工福利费：实际发生时根据实际发生额计入
- 社会保险费和住房公积金：按国务院/地方政府规定的基准和比例
- 工会经费和职工教育经费：按国家规定的计提标准

#### 二、带薪缺勤

| 类型 | 特征 | 会计处理 |
|------|------|----------|
| 累积带薪缺勤 | 权利可结转下期使用 | 职工提供服务增加权利时确认负债，以累积未行使权利增加的预期支付金额计量 |
| 非累积带薪缺勤 | 权利不能结转，未用完作废 | 职工实际缺勤时确认（通常已包含在每期工资中，无需额外处理） |

**累积带薪缺勤要点：**
- 离职时能获得现金支付的 → 确认全部累积未使用权利的金额
- 离职时不能获得现金支付的 → 根据预期将在下期使用的追加金额预计
- 休假扣除顺序为后进先出（先扣当年权利，再扣上年结转）

**会计分录——累积带薪缺勤**：
```
借：管理费用/生产成本等                  XX
  贷：应付职工薪酬——累积带薪缺勤          XX
```

#### 三、短期利润分享计划

**确认条件（同时满足）：**
1. 企业因过去事项导致现在具有支付职工薪酬的法定义务或推定义务
2. 因利润分享计划所产生的应付职工薪酬义务能够可靠估计

**可靠估计的三种情形：**
- 财务报告批准报出前已确定应支付金额
- 计划正式条款中包括确定金额的方式
- 过去惯例提供了确定推定义务金额的明显证据

**计量要点：**
- 应反映职工因离职而没有得到利润分享计划支付的可能性
- 业绩跨期的利润分享计划：按服务期间比例确认
- 12个月内不需全部支付的 → 适用其他长期职工福利规定
- 企业提取的奖金计划 → 比照利润分享计划处理

**会计分录——短期利润分享计划**：
```
借：生产成本/管理费用等                  XX
  贷：应付职工薪酬——利润分享计划          XX
```

#### 四、非货币性福利

**计量原则：** 按公允价值计量；公允价值不能可靠取得的，按成本计量。

**（一）以自产产品或外购商品发放给职工：**

| 情形 | 计量基础 | 处理要点 |
|------|----------|----------|
| 自产产品 | 公允价值+相关税费 | 视同销售：确认收入、结转成本、计算销项税额 |
| 外购商品 | 公允价值+相关税费 | 进项税额转出（计入成本费用） |

处理流程：先通过"应付职工薪酬——非货币性福利"归集，再实际发放。

**会计分录——非货币性福利（以自产产品发放）**：

计提时：
```
借：生产成本/管理费用等                  XX（含税公允价值）
  贷：应付职工薪酬——非货币性福利          XX
```

实际发放时：
```
借：应付职工薪酬——非货币性福利           XX
  贷：主营业务收入                        XX
      应交税费——应交增值税（销项税额）     XX
```

同时结转成本：
```
借：主营业务成本                         XX
  贷：库存商品                            XX
```

**（二）向职工提供企业支付了补贴的商品或服务：**

| 情形 | 处理方式 |
|------|----------|
| 合同规定职工购得住房后须服务一定年限（提前离开退回差价） | 差额作为长期待摊费用，在服务年限内平均摊销 |
| 合同未规定必须服务年限 | 差额直接计入出售当期损益或资产成本 |

**会计分录——将自有房屋无偿提供给职工使用**：
```
借：生产成本/管理费用等                  XX
  贷：应付职工薪酬——非货币性福利          XX

借：应付职工薪酬——非货币性福利           XX
  贷：累计折旧                            XX
```

**会计分录——租赁住房提供给职工使用**：
```
借：生产成本/管理费用等                  XX
  贷：应付职工薪酬——非货币性福利          XX

借：应付职工薪酬——非货币性福利           XX
  贷：银行存款（或其他应付款）             XX
```

### 第三节 离职后福利的确认与计量

#### 一、设定提存计划

- 向独立基金缴存固定费用后，企业不再承担进一步支付义务
- 会计处理简单：每期义务=该期间将要提存的金额
- 不需要精算假设，通常不存在精算利得或损失
- 预期12个月内不能全部支付的 → 以折现后金额计量

**会计分录——设定提存计划**：
```
借：生产成本/管理费用等                  XX
  贷：应付职工薪酬——设定提存计划          XX

借：应付职工薪酬——设定提存计划           XX
  贷：银行存款                            XX
```

#### 二、设定受益计划

**特征：** 企业承担精算风险和投资风险；费用≠当期应付提存金。

**判断标准（属于设定受益计划）：**
- 计划福利公式不仅与提存金相关，且资产不足时要求企业进一步提存
- 对提存金的特定回报作出担保

**核算四步骤：**

| 步骤 | 内容 |
|------|------|
| 步骤一 | 确定设定受益义务现值和当期服务成本（预期累计福利单位法+精算假设+折现） |
| 步骤二 | 确定设定受益计划净负债或净资产（义务现值 - 计划资产公允价值） |
| 步骤三 | 确定计入当期损益的金额（服务成本 + 利息净额） |
| 步骤四 | 确定计入其他综合收益的金额（重新计量，后续不重分类至损益） |

**关键规则：**
- 存在盈余时：净资产 = min(盈余, 资产上限)
- 资产上限 = 企业可从计划退款或减少未来缴存而获得的经济利益现值
- 服务成本包括：当期服务成本、过去服务成本、结算利得或损失
- 折现率：与义务期限和币种匹配的国债或高质量公司债券市场收益率
- 计划终止时：其他综合收益中的金额全部结转至未分配利润

### 第四节 辞退福利的确认与计量

#### 一、辞退福利与其他福利的区分

| 对比项 | 辞退福利 | 正常退休养老金 |
|--------|----------|----------------|
| 引发补偿的事项 | 辞退（终止雇佣） | 职工在职时提供的服务 |
| 确认时点 | 辞退计划满足负债确认条件时 | 职工提供服务的会计期间 |
| 费用归属 | 一次计入当期管理费用 | 在服务期间分期确认 |

**职工内部退休计划：**
- 正式退休前 → 比照辞退福利处理
- 正式退休后 → 按离职后福利处理

#### 二、确认时点

以下两者孰早日确认辞退福利负债并计入当期损益：
1. 企业不能单方面撤回解除劳动关系计划或裁减建议所提供的辞退福利时
2. 企业确认涉及支付辞退福利的重组相关成本或费用时（需同时满足：有详细正式的重组计划 + 已对外公告）

#### 三、计量规则

| 情形 | 计量方法 |
|------|----------|
| 职工没有选择权 | 按计划条款规定的拟解除人数×每一职位辞退补偿 |
| 自愿接受裁减（人数不确定） | 预计将接受裁减的职工数量×每一职位辞退补偿 |
| 预期12个月内完全支付 | 适用短期薪酬规定 |
| 预期12个月内不能完全支付 | 选择恰当折现率，以折现后金额计量 |

#### 四、确认要点

- 所有辞退福利均一次计入费用（管理费用），**不计入资产成本**
- 分期/分阶段实施的计划：每期/每阶段满足确认条件时分别确认
- 内部退休计划：将自停止服务日至正常退休日期间拟支付的工资和社保费等一次确认为应付职工薪酬，计入当期管理费用

**会计分录——辞退福利**：

确认辞退福利：
```
借：管理费用                             XX
  贷：应付职工薪酬——辞退福利              XX
```

实际支付时：
```
借：应付职工薪酬——辞退福利               XX
  贷：银行存款                            XX
```

## 第十章 股份支付

### 一、股份支付概述

股份支付是指企业为获取职工和其他方提供服务而授予权益工具或承担以权益工具为基础确定的负债的交易。

**四个环节**：授予日 → 等待期（可行权日）→ 行权日 → 出售日

**两大类型**：

| 类型 | 结算方式 | 常用工具 | 计量基础 |
|------|----------|----------|----------|
| 权益结算 | 授予权益工具（股份/期权） | 限制性股票、股票期权 | 授予日公允价值（锁定不变） |
| 现金结算 | 支付现金（金额与股价挂钩） | 模拟股票、现金股票增值权 | 每个资产负债表日重新计量公允价值 |

### 二、可行权条件

| 条件类型 | 内容 | 对公允价值的影响 | 对可行权数量估计的影响 |
|----------|------|------------------|------------------------|
| 服务期限条件 | 职工须服务满规定期限 | 不影响 | 影响（按预计离职率调整） |
| 非市场业绩条件 | 如净利润增长率达标 | 不影响 | 影响（按是否达标调整） |
| 市场条件 | 如股价达到某水平 | 影响（计入授予日公允价值） | 不影响（即使未达标也确认费用） |
| 非可行权条件 | 如统一账户条款 | 影响（计入授予日公允价值） | 不影响 |

### 三、条款修改规则

- **有利修改**：在原费用基础上，额外确认权益工具公允价值增加额（修改日前后公允价值之差），在修改日至新可行权日期间分摊
- **不利修改**：视同未发生，继续按原条款确认费用
- **取消或结算**：作为加速可行权处理，将剩余等待期应确认金额立即计入当期损益

### 四、权益结算的股份支付分录

授予日：不做账务处理（立即可行权的除外）

等待期内每个资产负债表日：
```
借：管理费用等（当期应确认金额）
  贷：资本公积——其他资本公积
```

当期费用 = 权益工具数量 × 授予日公允价值 × 累计等待期/总等待期 × (1-预计离职率) - 前期累计已确认金额

行权日：
```
借：银行存款（行权价 × 股数）
    资本公积——其他资本公积（等待期累计确认额）
  贷：股本（面值 × 股数）
      资本公积——股本溢价（差额）
```

### 五、现金结算的股份支付分录

等待期内每个资产负债表日：
```
借：管理费用等（当期应确认金额）
  贷：应付职工薪酬——股份支付
```

当期费用 = 权益工具数量 × 当日公允价值 × 累计等待期/总等待期 × (1-预计离职率) - 前期累计已确认金额

可行权日后（公允价值变动）：
```
借：公允价值变动损益
  贷：应付职工薪酬——股份支付
```
（或反向分录，视公允价值升降而定）

行权日（支付现金）：
```
借：应付职工薪酬——股份支付
  贷：银行存款
```

### 六、集团股份支付

母公司（结算企业）以自身权益工具结算，子公司（接受服务企业）无结算义务：

结算企业（母公司）个别报表：
```
借：管理费用（本公司员工部分）
    长期股权投资——子公司（子公司员工部分）
  贷：资本公积——其他资本公积
```

接受服务企业（子公司）个别报表：
```
借：管理费用
  贷：资本公积——其他资本公积
```

合并报表抵销分录：
```
借：资本公积——其他资本公积
  贷：长期股权投资——子公司
```

## 第十一章 借款费用

### 一、借款费用概述

**借款费用的组成**：
- 借款利息费用（含折价/溢价摊销）
- 因外币借款而发生的汇兑差额
- 辅助费用（手续费、佣金等）

**借款分类**：
- 专门借款：为购建或生产符合资本化条件的资产而专门借入
- 一般借款：除专门借款之外的借款

**符合资本化条件的资产**：需要经过相当长时间（通常一年以上）的购建或生产活动才能达到预定可使用或可销售状态的固定资产、投资性房地产、存货等。

### 二、资本化的三个时点

| 时点 | 条件 |
|------|------|
| 开始资本化 | 同时满足：①资产支出已发生 ②借款费用已发生 ③购建/生产活动已开始 |
| 暂停资本化 | 非正常中断且连续超过3个月（正常中断不暂停） |
| 停止资本化 | 资产达到预定可使用或可销售状态 |

### 三、专门借款资本化金额

**公式**：资本化金额 = 专门借款当期实际发生的利息费用 - 尚未动用的借款资金存入银行取得的利息收入或投资收益

专门借款利息资本化：
```
借：在建工程（资本化金额）
    应收利息/银行存款（闲置资金收益）
  贷：应付利息（票面利息）
```

若为折价/溢价发行债券：
```
借：在建工程（实际利率法计算的利息 - 闲置资金收益）
    应收利息/银行存款（闲置资金收益）
  贷：应付债券——应计利息（票面利息）
      应付债券——利息调整（差额，或借方）
```

### 四、一般借款资本化金额

**公式**：
- 一般借款资本化金额 = 累计资产支出超过专门借款部分的资产支出加权平均数 × 所占用一般借款的资本化率
- 资本化率 = 所占用一般借款加权平均利率 = 所占用一般借款当期实际发生的利息之和 ÷ 所占用一般借款本金加权平均数

一般借款利息资本化：
```
借：在建工程（一般借款资本化金额）
    财务费用（一般借款费用化金额）
  贷：应付利息（一般借款当期应付利息）
```

注意：每期资本化金额不得超过当期一般借款实际发生的利息费用。

### 五、外币专门借款汇兑差额资本化

外币专门借款本金及利息的汇兑差额，在资本化期间内应予资本化，计入符合资本化条件的资产成本。

```
借：在建工程（汇兑差额，资本化期间）
  贷：长期借款——外币借款（汇率变动产生的差额）
```

或（费用化期间）：
```
借：财务费用——汇兑差额
  贷：长期借款——外币借款
```

## 第十二章 或有事项

### 一、或有事项概述

或有事项是指过去的交易或事项形成的，其结果须由某些未来事项的发生或不发生才能决定的不确定事项。

**概率判断标准**：

| 概率表述 | 对应比例 |
|----------|----------|
| 基本确定 | 大于95%但小于100% |
| 很可能 | 大于50%但小于或等于95% |
| 可能 | 大于5%但小于或等于50% |
| 极小可能 | 大于0但小于或等于5% |

### 二、预计负债的确认条件（三条件同时满足）

1. 该义务是企业承担的**现时义务**
2. 履行该义务**很可能**导致经济利益流出企业
3. 该义务的金额能够**可靠计量**

### 三、未决诉讼确认预计负债

确认预计负债：
```
借：营业外支出（或管理费用等）
  贷：预计负债
```

实际支付赔偿时：
```
借：预计负债
  贷：银行存款
```

如有第三方补偿（如保险赔偿），基本确定能收到时单独确认资产：
```
借：其他应收款
  贷：营业外支出
```

### 四、产品质量保证

确认产品质量保证费用：
```
借：销售费用——产品质量保证
  贷：预计负债——产品质量保证
```

实际发生保修支出时：
```
借：预计负债——产品质量保证
  贷：银行存款/原材料/应付职工薪酬等
```

### 五、亏损合同

亏损合同是指履行合同义务不可避免会发生的成本超过预期经济利益的合同。

**处理规则**：
- 待执行合同变为亏损合同时，如有标的资产，先对标的资产进行减值测试
- 如无标的资产（或减值后仍有剩余亏损），将超出部分确认为预计负债

```
借：营业外支出（或资产减值损失）
  贷：预计负债（或存货跌价准备等）
```

**注意**：退出合同的成本（违约金）如低于继续履行的损失，按较低者确认。

## 第十三章 金融工具

### 一、金融资产三分类

| 分类 | 业务模式 | SPPI测试 | 计量方式 | 科目 |
|------|----------|----------|----------|------|
| 以摊余成本计量 | 收取合同现金流量 | 通过 | 摊余成本 | 债权投资 |
| 以公允价值计量且变动计入其他综合收益（FVOCI） | 既收取现金流量又出售 | 通过 | 公允价值 | 其他债权投资 |
| 以公允价值计量且变动计入当期损益（FVTPL） | 其他（含交易目的） | 不要求 | 公允价值 | 交易性金融资产 |

**SPPI测试**：合同现金流量是否仅为对本金和以未偿付本金金额为基础的利息的支付（Solely Payments of Principal and Interest）。

**权益工具投资的特殊指定**：非交易性权益工具投资可不可撤销地指定为FVOCI → 科目"其他权益工具投资"，公允价值变动计入其他综合收益，处置时不转损益。

### 二、债权投资（摊余成本计量）

购入（含交易费用）：
```
借：债权投资——成本（面值）
    债权投资——利息调整（差额，或贷方）
  贷：银行存款（实际支付价款）
```

期末计息（实际利率法）：
```
借：应收利息（面值 × 票面利率，分期付息）
    或 债权投资——应计利息（到期一次还本付息）
  贷：投资收益（期初摊余成本 × 实际利率）
      债权投资——利息调整（差额，或借方）
```

到期收回本金：
```
借：银行存款
  贷：债权投资——成本
      债权投资——应计利息（到期一次还本付息时）
```

### 三、其他债权投资（FVOCI债权类）

购入：
```
借：其他债权投资——成本（面值）
    其他债权投资——利息调整（差额，或贷方）
  贷：银行存款
```

期末计息：
```
借：应收利息（面值 × 票面利率）
  贷：投资收益（摊余成本 × 实际利率）
      其他债权投资——利息调整（差额）
```

公允价值变动：
```
借：其他债权投资——公允价值变动
  贷：其他综合收益——公允价值变动
```
（或反向）

出售：
```
借：银行存款
  贷：其他债权投资——成本
      其他债权投资——利息调整
      其他债权投资——公允价值变动
      投资收益（差额）
```

同时将其他综合收益转入投资收益：
```
借：其他综合收益——公允价值变动
  贷：投资收益
```
（或反向）

### 四、交易性金融资产（FVTPL）

购入（交易费用计入当期损益）：
```
借：交易性金融资产——成本（公允价值）
    投资收益（交易费用）
  贷：银行存款
```

公允价值变动：
```
借：交易性金融资产——公允价值变动
  贷：公允价值变动损益
```
（或反向）

出售：
```
借：银行存款
  贷：交易性金融资产——成本
      交易性金融资产——公允价值变动（或借方）
      投资收益（差额）
```

### 五、其他权益工具投资（FVOCI权益类，不可撤销指定）

购入（含交易费用）：
```
借：其他权益工具投资——成本（公允价值 + 交易费用）
  贷：银行存款
```

公允价值变动：
```
借：其他权益工具投资——公允价值变动
  贷：其他综合收益——公允价值变动
```
（或反向）

出售（其他综合收益不转损益，转入留存收益）：
```
借：银行存款
  贷：其他权益工具投资——成本
      其他权益工具投资——公允价值变动（或借方）
      盈余公积（差额的10%）
      利润分配——未分配利润（差额的90%）
```

同时结转其他综合收益：
```
借：其他综合收益——公允价值变动
  贷：盈余公积
      利润分配——未分配利润
```
（或反向）

### 六、复合金融工具（可转换公司债券）

**拆分原则**：发行方应将可转换债券拆分为负债成分和权益成分。
- 负债成分 = 未来现金流量按同期限同条件不含转换权的债券利率折现的现值
- 权益成分 = 发行价格 - 负债成分公允价值

发行时：
```
借：银行存款（实际收到金额）
  贷：应付债券——可转换公司债券（面值）（负债成分面值）
      应付债券——可转换公司债券（利息调整）（差额，或借方）
      其他权益工具（权益成分）
```

期末计息（按负债成分的实际利率）：
```
借：财务费用/在建工程（负债成分期初摊余成本 × 实际利率）
  贷：应付利息（面值 × 票面利率）
      应付债券——可转换公司债券（利息调整）（差额）
```

转股时：
```
借：应付债券——可转换公司债券（面值）（转股部分账面价值）
    应付债券——可转换公司债券（利息调整）（对应余额）
    其他权益工具（转股部分对应的权益成分）
  贷：股本（转换的股数 × 面值）
      资本公积——股本溢价（差额）
```

### 七、金融负债与权益工具的区分

**核心判断标准**：

| 判断要素 | 金融负债 | 权益工具 |
|----------|----------|----------|
| 是否有交付现金或其他金融资产的合同义务 | 有 | 无 |
| 是否以自身权益工具结算 | 非衍生：交付可变数量自身权益工具；衍生：以股份换股份以外的方式结算 | 非衍生：交付固定数量自身权益工具；衍生：以固定数量自身权益工具交换固定金额现金 |
| 是否有强制付息/还本义务 | 有（如普通债券） | 无（如永续债可递延付息） |

**永续债的判断**：关注是否有到期日、能否无条件递延付息、是否有强制赎回条款。

### 八、预期信用损失三阶段模型

| 阶段 | 信用风险状况 | 损失准备计量基础 | 利息收入计算基础 |
|------|-------------|------------------|------------------|
| 第一阶段 | 初始确认后信用风险未显著增加 | 未来12个月预期信用损失 | 账面余额（总额） |
| 第二阶段 | 信用风险显著增加但未发生信用减值 | 整个存续期预期信用损失 | 账面余额（总额） |
| 第三阶段 | 已发生信用减值 | 整个存续期预期信用损失 | 摊余成本（净额） |

计提减值：
```
借：信用减值损失
  贷：债权投资减值准备 / 坏账准备 / 其他综合收益——信用减值准备
```

**简化处理**：应收账款、合同资产、租赁应收款可选择始终按整个存续期预期信用损失计量。

### 九、套期会计基本类型

| 套期类型 | 被套期风险 | 套期工具利得/损失处理 | 被套期项目处理 |
|----------|-----------|----------------------|----------------|
| 公允价值套期 | 已确认资产/负债或确定承诺的公允价值变动 | 计入当期损益 | 因被套期风险的公允价值变动计入当期损益，调整账面价值 |
| 现金流量套期 | 已确认资产/负债或预期交易的现金流量变动 | 有效部分计入其他综合收益（套期储备），无效部分计入当期损益 | 套期储备在被套期现金流量影响损益时转出 |
| 境外经营净投资套期 | 境外经营净投资的外汇风险 | 有效部分计入其他综合收益，无效部分计入当期损益 | 处置境外经营时从其他综合收益转入当期损益 |

## 第十四章 租赁

### 一、承租人会计处理

**初始确认**（短期租赁和低价值资产租赁除外）：

确认使用权资产和租赁负债：
```
借：使用权资产（租赁负债现值 + 初始直接费用 + 预计复原成本等）
    租赁负债——未确认融资费用（差额）
  贷：租赁负债——租赁付款额（未折现的租赁付款总额）
      银行存款（初始直接费用）
      预计负债（复原成本现值，如有）
```

**后续计量——计提利息**：
```
借：财务费用（租赁负债期初余额 × 折现率）
  贷：租赁负债——未确认融资费用
```

**后续计量——支付租金**：
```
借：租赁负债——租赁付款额
    应交税费——应交增值税（进项税额）
  贷：银行存款
```

**后续计量——计提折旧**：
```
借：管理费用/制造费用等
  贷：使用权资产累计折旧
```

折旧期间：租赁期与使用权资产剩余使用寿命孰短（能合理确定将取得所有权的，按剩余使用寿命折旧）。

### 二、出租人分类标准

| 分类 | 判断标准 | 实质 |
|------|----------|------|
| 融资租赁 | 实质上转移了与标的资产所有权有关的几乎全部风险和报酬 | 类似分期购买 |
| 经营租赁 | 未转移几乎全部风险和报酬 | 正常出租 |

**融资租赁的判断指标**（满足一项或多项通常分类为融资租赁）：
1. 租赁期届满时资产所有权转移给承租人
2. 承租人有购买选择权且预计将行使（优惠购买选择权）
3. 租赁期占资产使用寿命的大部分
4. 租赁付款额现值几乎相当于资产公允价值
5. 标的资产性质特殊，不作较大改造只有承租人才能使用

### 三、短期租赁和低价值资产租赁（简化处理）

承租人可选择不确认使用权资产和租赁负债，将租赁付款额在租赁期内按直线法或其他系统合理方法计入当期损益：

```
借：管理费用/制造费用等
  贷：银行存款
```

- 短期租赁：租赁期不超过12个月且不包含购买选择权
- 低价值资产租赁：单项标的资产为全新时价值较低（通常不超过5万元人民币，准则未规定具体金额）

## 第十五章 持有待售的非流动资产、处置组和终止经营

### 一、持有待售的分类条件

非流动资产或处置组同时满足以下条件时，应划分为持有待售类别：
1. **可立即出售**：在当前状况下仅根据出售此类资产或处置组的惯常条款即可立即出售
2. **出售极可能发生**：
   - 已作出出售决议（如董事会或类似机构已批准）
   - 已签订不可撤销的转让协议或已确定买方
   - 预计出售将在一年内完成

### 二、计量规则

- 划分为持有待售时：按账面价值与公允价值减去出售费用后的净额孰低计量
- 账面价值高于公允价值减去出售费用后净额的差额，确认为资产减值损失
- 持有待售的非流动资产**不计提折旧或摊销**
- 后续公允价值减去出售费用后净额增加的，可转回原已确认的减值损失（以原已确认的减值损失为限）

```
划分为持有待售时（如需减值）：
借：资产减值损失
  贷：持有待售资产减值准备
```

**注意**：持有待售类别的资产在资产负债表中单独列示为"持有待售资产"，相关负债列示为"持有待售负债"。

## 第十六章 所有者权益

### 一、实收资本（股本）增减变动

**资本公积转增资本**：
```
借：资本公积——股本溢价
  贷：实收资本/股本
```

**盈余公积转增资本**：
```
借：盈余公积
  贷：实收资本/股本
```

**回购注销库存股**：

回购时：
```
借：库存股（回购价 × 股数）
  贷：银行存款
```

注销时：
```
借：股本（面值 × 股数）
    资本公积——股本溢价（差额先冲）
    盈余公积（不足部分冲）
    利润分配——未分配利润（仍不足部分冲）
  贷：库存股
```

注销时回购价低于面值的：
```
借：股本（面值 × 股数）
  贷：库存股（回购价 × 股数）
      资本公积——股本溢价（差额）
```

### 二、其他权益工具（永续债）

**发行永续债**：
```
借：银行存款（实际收到金额）
  贷：其他权益工具——永续债（面值）
      其他权益工具——永续债（利息调整）（溢价部分，或借方折价）
```

**计提利息（计入利润分配，不计入财务费用）**：
```
借：利润分配——应付永续债利息
  贷：应付利息
```

**支付利息**：
```
借：应付利息
  贷：银行存款
```

**赎回永续债**：
```
借：其他权益工具——永续债（账面价值）
  贷：银行存款（赎回价格）
      资本公积——股本溢价（差额，或借方）
```

### 三、盈余公积

**提取法定盈余公积**（税后利润的10%，累计达注册资本50%时可不再提取）：
```
借：利润分配——提取法定盈余公积
  贷：盈余公积——法定盈余公积
```

**提取任意盈余公积**：
```
借：利润分配——提取任意盈余公积
  贷：盈余公积——任意盈余公积
```

**盈余公积弥补亏损**：
```
借：盈余公积
  贷：利润分配——盈余公积补亏
```

### 四、利润分配全流程

年末结转本年利润：
```
借：本年利润
  贷：利润分配——未分配利润
```
（亏损则反向）

提取盈余公积：
```
借：利润分配——提取法定盈余公积
    利润分配——提取任意盈余公积
  贷：盈余公积——法定盈余公积
      盈余公积——任意盈余公积
```

宣告分配现金股利：
```
借：利润分配——应付现金股利
  贷：应付股利
```

结转利润分配明细：
```
借：利润分配——未分配利润
  贷：利润分配——提取法定盈余公积
      利润分配——提取任意盈余公积
      利润分配——应付现金股利
```

### 五、其他综合收益分类

| 类别 | 后续能否转入损益 | 典型项目 |
|------|------------------|----------|
| 以后会计期间不能重分类进损益 | 否 | ①重新计量设定受益计划变动额 ②其他权益工具投资公允价值变动 ③自用房地产转为公允价值模式的投资性房地产时评估增值 |
| 以后会计期间满足条件时将重分类进损益 | 是 | ①其他债权投资公允价值变动 ②现金流量套期储备 ③外币报表折算差额 ④权益法下被投资单位其他综合收益变动中本企业享有的份额 |

## 第十七章 收入、费用和利润

### 一、收入确认五步法

**第一步：识别与客户订立的合同**

合同成立的五个条件（同时满足）：
1. 合同各方已批准该合同并承诺将履行各自义务
2. 能够识别每一方与所转让商品相关的权利
3. 能够识别与所转让商品相关的支付条款
4. 合同具有商业实质
5. 因向客户转让商品而有权取得的对价很可能收回

**第二步：识别合同中的单项履约义务**

可明确区分商品的两个条件（同时满足）：
1. 客户能够从该商品本身或与其他易于获得的资源一起使用中受益
2. 企业向客户转让该商品的承诺与合同中其他承诺可单独区分

不可单独区分的情形：
- 企业提供重大整合服务
- 商品之间高度关联或高度依赖
- 一项或多项商品对合同中其他商品进行重大修改或定制

**第三步：确定交易价格**

交易价格的确定需考虑：
1. **可变对价**：按期望值或最可能发生金额估计，受"限制"约束（极可能不会发生重大转回）
2. **重大融资成分**：调整交易价格以反映货币时间价值（合同开始日与付款日间隔超过一年时考虑）
3. **非现金对价**：按公允价值计量
4. **应付客户对价**：冲减交易价格（除非取得可明确区分的商品）

**第四步：将交易价格分摊至各单项履约义务**

- 按各单项履约义务所承诺商品的**单独售价的相对比例**分摊
- 单独售价无法直接观察时，采用市场调整法、成本加成法或余值法估计
- 折扣分摊：通常按比例分摊至所有履约义务；有证据表明折扣仅与部分履约义务相关时，分摊至相关部分

**第五步：履行各单项履约义务时确认收入**

在某一**时段**内确认收入（满足以下任一条件）：
1. 客户在企业履约的同时即取得并消耗企业履约所带来的经济利益
2. 客户能够控制企业履约过程中在建的商品
3. 企业履约过程中所产出的商品具有不可替代用途，且企业在整个合同期间内有权就累计至今已完成的履约部分收取款项

不满足上述任一条件的，在某一**时点**确认收入（控制权转移时点）。

### 二、基本收入确认分录

**在某一时点确认收入（如商品销售）**：
```
借：银行存款/应收账款/合同资产
    应交税费——应交增值税（销项税额）（贷方，含税时）
  贷：主营业务收入
      应交税费——应交增值税（销项税额）
```

同时结转成本：
```
借：主营业务成本
  贷：库存商品
```

**在某一时段内确认收入（如建造合同）**：
```
借：合同结算——收入结转
  贷：主营业务收入（履约进度 × 交易价格 - 前期已确认收入）
```

```
借：主营业务成本（履约进度 × 预计总成本 - 前期已确认成本）
  贷：合同履约成本
```

### 三、合同资产、合同负债与应收账款的区分

| 项目 | 含义 | 收款权利的性质 |
|------|------|----------------|
| 应收账款 | 企业已履行履约义务，仅取决于时间流逝即有权收款 | 无条件收款权（仅时间条件） |
| 合同资产 | 企业已履行履约义务，但收款权取决于时间流逝之外的其他条件 | 有条件收款权（如需对方验收） |
| 合同负债 | 客户已付款或已到期应付，但企业尚未履行履约义务 | 企业的预收款/履约义务 |

合同资产转为应收账款（条件满足时）：
```
借：应收账款
  贷：合同资产
```

收到客户预付款：
```
借：银行存款
  贷：合同负债
```

后续确认收入时：
```
借：合同负债
  贷：主营业务收入
```

### 四、利润公式

```
营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 财务费用
+ 其他收益 + 投资收益（-投资损失）+ 净敞口套期收益（-净敞口套期损失）
+ 公允价值变动收益（-公允价值变动损失）+ 信用减值损失（-信用减值收益）
+ 资产减值损失（-资产减值收益）+ 资产处置收益（-资产处置损失）
= 营业利润

营业利润 + 营业外收入 - 营业外支出
= 利润总额

利润总额 - 所得税费用
= 净利润
```

**其中**：
- 营业收入 = 主营业务收入 + 其他业务收入
- 营业成本 = 主营业务成本 + 其他业务成本
- 所得税费用 = 当期所得税 + 递延所得税费用（或 - 递延所得税收益）
