## 一、审计处罚四维缺陷模型

会计师事务所被处罚的审计程序缺陷归纳为四个维度：

### 维度 A：函证程序缺陷（约占 35%）

| 编号 | 缺陷 | 违反准则 | 后果 |
|------|------|---------|------|
| A1 | 未对全部银行账户函证（含零余额/已注销） | CSA 1312 第17条 | 警示函 |
| A2 | 未回函项目未实施替代程序 | CSA 1312 第21条 | 警示函 |
| A3 | 未对询证函保持控制（回函由被审计单位转交） | CSA 1312 第17-18条 | 警示函/行政处罚 |
| A4 | 未对回函异常保持警觉（地址不一致/印章异常） | CSA 1312 第19条 + CSA 1141 第13条 | 警示函/行政处罚 |

### 维度 B：审计证据缺陷（约占 25%）

| 编号 | 缺陷 | 违反准则 | 案例 |
|------|------|---------|------|
| B1 | 未获取充分适当的审计证据 | CSA 1301 第10、15条 | 大华所〔2024〕313号 |
| B2 | 异常交易未追查至底层证据 | CSA 1301 第15条 + CSA 1141 第13-14条 | 普华永道恒大案 |
| B3 | 依赖管理层声明替代审计程序 | CSA 1301 第10、15条 | 多案——管理层声明不能替代实质性程序 |
| B4 | 收入确认舞弊风险应对不足 | CSA 1141 第14条 | 普华永道恒大案 |

### 维度 C：职业判断缺陷（约占 20%）

| 编号 | 缺陷 | 核心要点 |
|------|------|---------|
| C1 | 未对重大异常形成恰当职业判断 | 毛利率异常/收入与产能不匹配/期末集中确认 |
| C2 | 未保持职业怀疑 | "已执行常规程序"不是免责理由 |
| C3 | 会计估计合理性未充分评估 | 坏账/跌价/商誉减值 |
| C4 | 持续经营能力评估不充分 | CSA 1324 |

### 维度 D：底稿与质控缺陷（约占 20%）

| 编号 | 缺陷 | 严重程度 |
|------|------|---------|
| D1 | 底稿不完整/记录与执行不一致 | 叠加性——强化其他维度处罚 |
| D2 | 质控复核流于形式 | 苏亚金诚案 |
| D3 | 伪造/篡改/毁损底稿 | **最严重**——必然行政处罚+暂停业务（天职国际〔2024〕78号） |
| D4 | 档案管理不合规 | 轻微 |

### 处罚升级路径

```
单维度缺陷（较轻）→ 警示函
多维度缺陷（一般）→ 警示函或责令改正
严重缺陷+故意行为 → 行政处罚+暂停业务+CPA罚款
连续审计失败（多年未发现造假）→ 高额罚款+暂停业务+市场禁入
```

## 三、合规红线启发式（8 条）

### H1：实质重于形式
交易经济实质与法律形式不一致，且影响信披真实性 → 已触发红线。
先例：江苏舜天〔2024〕41号——合同齐全、资金流水完整，仍认定虚增收入。

### H2：全部银行账户函证
未对全部银行账户（含零余额/已注销）函证 → 已触发最低程序红线。

### H3：关联方资金占用标尺
占净资产 >5% 且未披露 → 监管关注显著提高。>30% 且有主观故意 → 从严。
先例：金洲慈航〔2024〕1号——51.15%，终身禁入。

### H4：持续造假年限
3年以上持续造假 → 近年案例均严惩。
先例：江苏舜天13年、美尚生态9年。

### H5：收入造假"无商业实质"判定
上下游同一方控制 + 资金回流 + 不承担风险 + 缺乏商业合理性 → 触发红线。

### H6：审计"未勤勉尽责"最低门槛
"签字但未关注到明显异常"即构成未勤勉尽责。不需证明"参与造假"。
先例：钱仁勇〔2024〕6号——"不知悉、未参与"被驳回。

### H7：主动纠正的正面作用
立案调查前主动更正/追回资金/补缴税款 → 正面因素。立案后效果有限。

### H8：形式合规≠实质合规
"走了流程"（董事会签字、审批）不构成免责。必须有实质性独立判断证据。
先例：恒大〔2024〕49号——"消极不履责同样是未勤勉尽责"。

## 五、财务造假典型手法库

| 手法 | 典型案例 | 识别信号 |
|------|---------|---------|
| 提前确认收入 | 恒大〔2024〕49号 | 期末集中确认、收入与交付不匹配 |
| 无商业实质自循环 | 江苏舜天〔2024〕41号 | 上下游同一控制、资金闭环、不承担风险 |
| 虚构交易+资金闭环 | 东方金钰 | 壳公司、多层中转、资金回流 |
| 隐瞒担保/资金占用 | 三盛教育〔2023〕11号 | 未履行决策程序、占净资产比例高 |
| 伪造单据 | 华锐风电〔2015〕66号 | 签署单据与实物交付不匹配 |
| 虚增服务收入 | 金洲慈航〔2024〕1号 | 无实际服务内容、资金占用 |
| IPO 阶段造假 | 思尔芯、紫晶存储 | 上市前虚增、申报期现场检查暴露 |
| 跨期调节虚构业绩 | 金通灵 | 收入费用跨期、关联方配合、连续多年 |

**通用识别信号：**
- 收入增长与产能/人员/现金流不匹配
- 毛利率显著高于行业且不可解释
- 应收增速远超收入增速
- 关联交易占比高且对手集中
- 频繁变更会计估计或政策
- 期末集中确认大量收入

## 六、问询函编制规范

### 双角色架构

| 角色 | 职责 | 视角 |
|------|------|------|
| 审核专家（主导） | 准则适用性、信披完整性、内控有效性、会计估计合理性 | 以准则为标尺，关注"是否合规" |
| 舞弊专员（协同） | 舞弊动机（压力/机会/合理化）、收入操纵、隐性关联交易 | 以舞弊三角+处罚先例为参照，关注"是否存在异常信号" |

### 问询问题标准结构

```
【核查发现】
[公司自身财务数据异常，用表格或逐条列举]

【问询事项】
（1）请说明/论证/量化/提供……（底稿要求：①……；②……）
（2）请说明/论证/量化/提供……（底稿要求：③……；④……）

【准则/法规依据】（如适用）
【参照先例】（如适用）
```

### 核心原则

1. **基于公司自身数据异常发问，不做同业直接对比**
2. **解释责任交予企业，不前置定性结论**
3. **每个问题附带底稿要求**

### 禁止行为

- 不以同业数字对比作为问询逻辑起点
- 不在问题中预设结论（如"你公司通过XX操纵利润"）
- 不要求企业自证"未违规"——应要求"说明业务逻辑和会计处理依据"
- 不编造底稿编号或数据来源

## 七、IPO审核被否原因归类（2021年29家被否案例）

> 类型：判断 + 实操
> 来源：2021年IPO终止被否案例分析及IPO上市重点关注问题总结（Wind/证监会/沪深交易所）
> 置信度：高（基于发审委/上市委公开问询）

### 审核动态背景（2020 vs 2021）

- 各板块被否家数由9家增至29家（创业板8、科创板8、上证主板7、深证主板6）
- 2021年终止家数大幅上升，大量企业上会前撤回或过会后终止
- **实际通过率**（审核通过/(通过+终止，含被否)）：2020年86.93% → 2021年仅 **62.33%**，大幅下降
- 启示：过会率（90.65%）远高于实际通过率，审核实质门槛显著提高

### 被否原因14类归类（按问询高频排序）

**1. 持续经营能力存疑**
- 阶段性/政策红利收入（限硫令前船东抢装、补贴依赖）随政策实施难持续
- 行业政策不利变化（光伏标杆电价/补贴下降、带量采购降价、农药监管升级）
- 案例：杨瑞新材、天地环保（船舶脱硫抢装）、运高新能（光伏电价补贴）、菲仕技术（新能源车业务下滑+在建工程无投入）、林华医疗（带量采购）、世佳科技（农药监管）

**2. 收入确认真实性与准确性**
- 经销/加盟模式压货提前确认收入、以旧换新调节收入、终验法vs工期进度法差错更正、总额法vs净额法、放宽信用政策调节收入、跨期确认
- 案例：梦金园（加盟压货+以旧换新）、才府玻璃（向非直接用户压货+放宽信用）、派特罗尔（终验法改工期进度法、重大会计差错）、丁点儿（经销占比93%）、世佳（新客户突击放量）、涧光特种（验收时点）、老铺黄金（联营模式净额改总额法）

**3. 毛利率异常**
- 显著高于或变动趋势不同于同行业且不可解释
- 案例：正和科技（涂装毛利40.78%）、才府玻璃、林华医疗（留置针高毛利）、老铺黄金（单店平效远超同行）、世佳科技

**4. 大额关联交易与独立性**
- 关联采购/销售占比畸高、对单一客户重大依赖、客户供应商重叠、关联方代收货款
- 案例：杨瑞新材（关联采购98%+脱硫委托关联电厂）、鲁华泓锦（向中石化采购>70%且为第一大客户）、垦丰种业（KWS授权+合资公司并表+关联代收货款）、派特罗尔（中石油系客户供应商重叠）、正和科技（陕汽商用车）、速达股份（郑煤机）

**5. 资金往来与银行流水**
- 实控人与员工/亲属大额资金往来、第三方回款占比高、第三方代付货款、股东间无息无期限借款
- 案例：鲁华泓锦（实控人与亲属员工往来）、华泰永创（股东向实控人借款+第三方代付）、汇川物联（财务经理离职+流水未提供全）、九恒条码（实控人配偶与供应商客户往来）、吉凯基因（第三方回款>50%）

**6. 内控有效性/会计基础薄弱**
- 重大会计差错更正暴露内控缺陷、预计合同成本内控缺失、资金管理内控缺陷
- 案例：派特罗尔（会计差错更正影响业绩重大）、华泰永创（资金拆借未履决策程序）

**7. 历史沿革与股权问题**
- 股权代持、出资瑕疵/虚假出资、国有资产流失、实际控制人认定及变动、业绩对赌/股份回售特殊条款
- 案例：华南装饰（出资来源+国资转让公允性）、涧光特种（集体企业改制+股权代持）、灿星文化（共同控制人+红筹拆除+实控人认定）、速达股份（一致行动规避实控人认定）、九恒条码（对赌回售条款金融负债/权益判断）

**8. 商誉减值合理性**
- 收购后短期全额计提减值、承诺期不减值期满后大额计提、商誉减值追溯调整
- 案例：林华医疗（收购北京悦通短期全额计提商誉减值）、灿星文化（梦响强音19.68亿商誉、2020年追溯调整2016年末减值）

**9. 板块定位/科创属性/创业板定位匹配**
- 技术先进性不足、核心技术来源于外部授权/合作研发构成技术依赖、研发投入/费用归集不实
- 案例：吉凯基因（技术壁垒+主流靶点）、海和药物（核心产品授权引进、对药物所技术依赖）、汇川物联（物联网定位+专利相关性）、珈创生物（检测通用技术+研发费用归集）、赛赫智能（研发投入占比）、鸿基节能（创业板定位）

**10. 安全生产与环保违法**
- 超产能生产、危废处置违法、生态安全处罚导致停产
- 案例：德纳化学（EO超产+响水爆炸影响）、鲁华泓锦（生产/生态安全处罚）、康鹏科技（关联方污染环境罪+多起安全环保事故停产）

**11. 资质合规与经营合法**
- 经营资质续期风险、食品安全资质、招投标合规/商业贿赂、虚假宣传
- 案例：珈创生物（CMA/BSL-2资质2022到期）、丁点儿（食品生产许可）、华夏万卷（"教育部门推荐"虚假宣传+著作权诉讼）、华南装饰（招投标合规+商业贿赂）、林华医疗（高销售费用率商业贿赂疑虑）

**12. 业绩下滑与盈利能力**
- 收入上涨但扣非净利润大幅下降、毛利率下降、盈利预测不达70%
- 案例：垦丰种业（净利润大幅下降）、才府玻璃（前次申报盈利预测不及70%）

**13. 经营性现金流持续低于净利润**
- 案例：鸿基节能（经营现金流持续低于净利润且曾为负）、九恒条码（现金流与净利润不匹配、用电量与收入不匹配）

**14. 股份支付与入股价格公允性**
- 申报前突击入股、国资/混改员工持股价格公允性、入股价格与股份支付
- 案例：杨瑞新材/天地环保（浙能迈领国资员工持股评估）、华泰永创（建龙集团申报前12个月入股）、才府玻璃（自然人申报前入股规避锁定期）、德纳化学（九鼎入股价格及资金来源）

### IPO重点关注问题（保荐/申报实操要点）

**提前关注的四大问题**：
1. **银行流水**：实控人、董监高、关键人员、关联方的资金往来核查，警惕流水最终流入客户/供应商（利益输送、虚增收入）
2. **业务独立性**：关联交易占比、单一客户依赖、客户供应商重叠、关联方代收代付
3. **历史沿革合规**：合法是基础、合理是保障——出资、股权代持、改制、国资程序
4. **合法经营**：安全生产、环保、食品/经营资质、招投标、商业贿赂

**理念红线**：
- 注册制不是降低上市要求，实质对企业提出更高要求
- 财务问题可"以时间换空间"解决，但法律问题处理不当会断送整个上市进程
- 不要在监管面前自作聪明；信息披露真实、准确、完整是底线

**经验总结**：选择适合自己的中介机构、尊重中介专业判断、上市是系统工程、上市是持久战、对实控人（老板）人品的考验
